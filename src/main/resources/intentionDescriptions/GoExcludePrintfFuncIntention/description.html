Exclui funções ou métodos de serem identificados como funções de formatação semelhantes a fmt.Printf:
  • Não será verificado se verbos de formatação incorretos, como %s, %d, %v e outros, são usados na string de formato
  • Os verbos de formatação na string de formato não serão realçados
  • A interpolação (dobradura de código) da string formatada final será desativada
