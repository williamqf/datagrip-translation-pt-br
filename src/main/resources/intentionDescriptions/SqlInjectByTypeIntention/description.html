Injeta um dialeto e o associa ao tipo SQL selecionado. Aplica este dialeto a todas as ocorrências do mesmo tipo (por exemplo, query).

No exemplo Before, você pode invocar a ação de intenção Injetar por tipo no valor 'SELECT * FROM foo' e, em seguida, injetar este tipo de objeto e associá-lo a qualquer dialeto SQL. O tipo de objeto deste valor é query (especificado por q query DEFAULT 'SELECT * FROM foo'). Portanto, da próxima vez que você usar o tipo query em seu código, o dialeto selecionado será injetado automaticamente.
