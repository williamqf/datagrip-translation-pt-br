Substitui chamadas de construtor por chamadas a um método de fábrica.

Como os métodos de fábrica podem criar objetos de qualquer tipo, essa substituição é especialmente útil quando uma nova instância da classe envolvente não é desejada.

Por exemplo, quando a classe atual é estendida e objetos de uma subclasse devem ser criados em seu lugar. Ou quando o cache é introduzido e objetos existentes devem ser retornados.
