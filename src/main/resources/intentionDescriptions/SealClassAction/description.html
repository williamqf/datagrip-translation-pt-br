Torna uma classe final se ela não possuir subclasses.

Se houver subclasses, a intenção converte a classe em uma classe sealed e fornece a lista de suas subclasses na cláusula permits (para subclasses em outras unidades de compilação). Se as subclasses não estiverem marcadas como sealed / non-sealed / final, o modificador non-sealed será aplicado automaticamente a elas.
