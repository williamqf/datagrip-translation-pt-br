Marca a função como uma função de formatação semelhante a fmt.Printf:
  • Verificará se a string de formato utiliza verbos de formatação incorretos, como %s, %d, %v e outros
  • Realçará os verbos de formatação na string de formato
  • Habilitará a interpolação da string formatada final (code folding) quando a funcionalidade de dobramento de código estiver ativa.
