Divide condições de conjunção ou disjunção em várias instruções if.

Condições de conjunção são divididas em dois if aninhados, e condições de disjunção são divididas em duas instruções if sequenciais contendo blocos duplicados.

Dividir uma condição em várias instruções if pode ser útil para melhorar a legibilidade quando as condições se tornam muito extensas ou para facilitar a edição quando as condições precisam ser alteradas.
