Atualiza várias dependências no arquivo go.mod selecionado.

As seguintes opções estão disponíveis:

  • Atualizar todas as dependências para a última versão de patch.
  • Atualizar todas as dependências para a última versão.
  • Atualizar as dependências diretas para a última versão de patch.
  • Atualizar as dependências diretas para a última versão.

Dependências diretas são módulos importados explicitamente nos arquivos de origem do projeto.

Para mais detalhes sobre comandos de atualização de dependências, consulte o go-get em go.dev.
