Substitui nomes de colunas por referências posicionais.

Alguns DBMS podem referenciar colunas em ORDER BY e GROUP BY por meio de números sequenciais. Exceto por Cassandra, ClickHouse, Hive, Spark e HSQLDB, todos os dialetos suportam ORDER BY. MySQL, Maria, PostgreSQL (e seus subdialetos), Exasol, SQLite e Snowflake suportam GROUP BY.

Exemplo (PostgreSQL):

CREATE TABLE tab(a INT, b INT);
SELECT * FROM tab ORDER BY 1;

Você pode usar esta ação de intenção para substituir o 1 na consulta SELECT pelo nome da coluna. Portanto, após aplicar a ação de intenção, a consulta ficará da seguinte forma:

SELECT * FROM tab ORDER BY a;
