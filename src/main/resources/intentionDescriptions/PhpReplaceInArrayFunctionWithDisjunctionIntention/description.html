Expande a chamada in_array($needle, $haystack, $strict) para a disjunção correspondente.

Se a disjunção resultante fizer parte de uma expressão lógica, ela será colocada entre parênteses. Se o $haystack contiver um elemento com efeito colateral, esse elemento será colocado na primeira expressão de igualdade. Se o $haystack contiver vários elementos com efeitos colaterais, esta intenção não será sugerida.
