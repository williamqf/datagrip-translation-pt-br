Gera código boilerplate para tratamento de erros idiomático.

Você pode invocar o template em chamadas que retornam error ou em qualquer expressão do tipo error. O template introduz novas variáveis, atribui a expressão a elas e verifica se o erro não é nil usando o seguinte código: 

if err != nil {
  return value1, value2, ...
}

Cada value será a variável err ou o valor padrão para o tipo de retorno correspondente da função envolvente.
