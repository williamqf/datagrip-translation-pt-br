Gera código boilerplate para tratamento de erros idiomático.

Pode ser chamado em variáveis e expressões do tipo error.

  • Substitui a variável err pela seguinte construção:

    if err != nil {
    return value1, value2, ...
  }

  • Substitui a expressão expr pela seguinte construção:

    if err := expr; err != nil {
    return value1, value2, ...
  }

Cada value é a variável err ou o valor padrão para o tipo de retorno correspondente da função circundante. Você pode chamar este template em expressões com múltiplos valores.
