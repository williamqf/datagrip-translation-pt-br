Realize commits parciais

É possível dividir alterações feitas em um mesmo arquivo entre diferentes commits. Na janela de ferramentas Commit &shortcut:ActivateCommitToolWindow;, clique em Diff. Marque a caixa de seleção ao lado de cada bloco de código modificado ou novo que você deseja realizar o commit e clique em Commit. As alterações não selecionadas permanecerão na lista de alterações atual.
