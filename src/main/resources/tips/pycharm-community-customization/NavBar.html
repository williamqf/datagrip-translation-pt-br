Barra de Navegação

A barra de navegação é uma alternativa rápida para a janela de ferramentas Project, onde você pode percorrer a estrutura do seu projeto, abrir arquivos específicos e saltar para elementos de código específicos no arquivo atual.

Pressione &shortcut;ShowNavBar; para mostrar a barra de navegação e use as teclas de seta para alternar entre arquivos e pastas.
