OPyCharmCommunity permite configurar os interpretadores Python em vários estágios de desenvolvimento:

  • Quando um projeto está apenas sendo criado (Arquivo | Novo Projeto, ou Criar Novo Projeto na tela de Boas-vindas).

  • Em um projeto já existente, use o widget Interpretador Python na Barra de Status ou selecione Projeto | Interpretador Python nas Configurações do projeto.
