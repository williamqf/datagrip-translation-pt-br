Live templates (Modelos dinâmicos)

Use modelos dinâmicos para inserir construções de código frequentes.

Quando você seleciona um modelo dinâmico na lista de sugestões, o modelo se expande para a variante definida na página Editor | Live Templates (Editor | Modelos dinâmicos) da caixa de diálogo Configuração (Settings) (&shortcut:ShowSettings;).

Use Tab para mover entre os campos de modelo.
