Adicione dicionários personalizados para verificação ortográfica

Quando algumas palavras estiverem faltando nos dicionários pré-definidos, você pode criar o seu próprio. Um dicionário personalizado é um arquivo de texto .dic que contém cada palavra em uma nova linha.

Para adicionar novos dicionários, abra as configurações pressionando &shortcut:ShowSettings; e navegue para Editor | Idiomas Naturais | Ortografia.
