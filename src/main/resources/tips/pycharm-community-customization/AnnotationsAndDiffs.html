Veja as anotações do Git Blame

As anotações do Git mostram informações detalhadas sobre a origem de cada linha de código (clique com o botão direito na margem vertical e selecione Comentar com Git Blame).

Clique com o botão direito em uma anotação e escolha Mostrar Diferença para analisar as diferenças entre a versão atual e a versão anterior do arquivo.
