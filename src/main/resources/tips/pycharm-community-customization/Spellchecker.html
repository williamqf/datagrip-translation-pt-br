Verificação ortográfica e gramatical

As verificações ortográfica e gramatical são habilitadas por padrão. Essas verificações são fornecidas pelo plugin Grazie, que é fornecido com o IDE. Para configurar as configurações de revisão, abra as configurações pressionando &shortcut:ShowSettings; e navegue até Editor | Natural Languages | Grammar.

Todos os textos, incluindo comentários, strings textuais e literais, são inspecionados em relação aos dicionários predefinidos ou personalizados. Os erros ortográficos são destacados para que você possa escolher a palavra correta, aceitar a ortografia atual ou desabilitar a inspeção.

Para corrigir um erro ortográfico ou gramatical, pressione &shortcut:ShowIntentionActions; para ver a lista de ações disponíveis.
