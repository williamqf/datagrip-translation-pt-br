Realçar ocorrências dentro de um arquivo

Pressione &shortcut:HighlightUsagesInFile; (Editar | Buscar ocorrências | Realçar ocorrências no arquivo) para realçar rapidamente ocorrências de uma variável específica no arquivo atual.

Pressione &shortcut:FindNext; e &shortcut:FindPrevious; para navegar pelas ocorrências realçadas. Pressione &shortcut:EditorEscape; para remover o realce.
