Mova instruções para cima e para baixo

As ações Code | Move Statement Up/Down são úteis para reorganizar linhas de código, por exemplo, para aproximar uma declaração de variável do uso da variável.

Selecione um fragmento de código e pressione &shortcut:MoveStatementUp; ou &shortcut:MoveStatementDown;.

Quando nada é selecionado no editor, a linha na posição do cursor será movida.
