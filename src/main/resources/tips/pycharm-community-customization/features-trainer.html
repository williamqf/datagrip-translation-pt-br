Tente cursos de treinamento interativos

O PyCharm Community inclui um IDE Features Trainer (Treinador de recursos da IDE) integrado. Ele fornece cursos interativos de treinamento para ajudar você a aprender a executar algumas tarefas comuns, como refatoração de código, navegação entre projetos, depuração e trabalho com controle de versão.

Verifique se o plugin IDE Features Trainer está ativado na guia Instalado da página Plugins nas configurações da IDE.

Para abrir a janela da ferramenta Learn e começar os exercícios, selecione View | Tool Windows | Learn (Exibir | Janelas de ferramentas | Aprender) ou Help | Learn IDE Features (Ajuda | Aprender recursos da IDE) no menu principal. A janela da ferramenta lista os tutoriais disponíveis.
