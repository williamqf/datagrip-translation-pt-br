Formatação do código

Para formatar o código de acordo com as configurações de estilo de código atuais, selecione " Código | Formatar código " no menu principal ou pressione &shortcut:ReformatCode; .

Você também pode usar " Código | Otimizar importações " para organizar instruções de importação e remover importações não utilizadas de acordo com as configurações de estilo de código do projeto.
