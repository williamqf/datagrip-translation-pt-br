Navegando entre interfaces e tipos

Você pode ir rapidamente para uma interface ou para todos os tipos que implementam essa interface (no diretório Interno) e navegar entre eles.

Para ir para uma interface, selecione um tipo e pressione &shortcut:GotoSuperMethod;. Para ver todos os métodos que implementam essa interface, selecione a interface e pressione &shortcut:GotoSuperMethod;.
