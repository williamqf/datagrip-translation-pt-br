Refatoração: Extração de interface

Você pode criar uma interface baseada em um tipo selecionado. A caixa de diálogo "Extrair Interface" lista os métodos e especificações que podem ser usadas para criar novas especificações de métodos.

Para criar uma interface, posicione o cursor no tipo, interface ou método que será usado como base para a nova interface. Selecione "Refatoração | Extrair | Interface". Na caixa de diálogo "Extrair Interface", selecione os membros que serão extraídos com a interface e clique em "Refatorar".
