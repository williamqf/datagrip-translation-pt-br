Switcher de navegação

Para alternar entre arquivos abertos e janelas de ferramentas, utilize o switcher &shortcut:Switcher;. Para manter o pop-up do switcher aberto, mantenha pressionada a tecla Control. Para navegar, utilize Tab e Shift+Tab.

Para fechar uma guia de editor ou ocultar uma janela de ferramentas, pressione Delete ou BackSpace.
