Tratamento de erro com verificação de 'nil'

O modelo de pós-conclusão rr cria código para verificar se a variável de erro não é nil e encerra a função, retornando um valor padrão. Você pode usar o modelo de pós-conclusão rr para tratamento de erro. Ele combina a funcionalidade dos 2 modelos já disponíveis (rre e rrv).
