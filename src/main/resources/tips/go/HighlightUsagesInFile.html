Destacando as ocorrências no arquivo

&shortcut:HighlightUsagesInFile; (Editar | Encontrar usos | Destacar usos do arquivo) pressione isso para destacar rapidamente as ocorrências de uma variável específica no arquivo atual.

&shortcut:FindNext; e &shortcut:FindPrevious; pressione para navegar entre as ocorrências destacadas. &shortcut:EditorEscape; pressione isso para remover o destaque.
