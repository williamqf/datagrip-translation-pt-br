Pesquisa global

Shift Pressione 2 vezes para pesquisar por arquivos em todo o projeto, ações, símbolos, elementos da interface do usuário, branches do Git e comentários. Pressione Shift 2 vezes novamente para pesquisar também por itens fora do escopo do projeto.

Para refinar os resultados da pesquisa, use as abas ou use diretamente os atalhos necessários (como &shortcut:GotoClass; para classes, &shortcut:GotoFile; para arquivos, &shortcut:GotoSymbol; para símbolos e &shortcut:GotoAction; para ações).
