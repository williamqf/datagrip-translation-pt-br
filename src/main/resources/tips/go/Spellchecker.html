Verificação de ortografia e gramática

A verificação de ortografia e gramática está habilitada por padrão. Essas verificações são fornecidas pelo plug-in Grazie, integrado ao IDE. Para configurar as opções de correção, pressione &shortcut:ShowSettings; e vá para "Editor | Linguagem natural | Gramática".

Todo o texto, incluindo comentários, strings do corpo e literais, são verificados usando dicionários predefinidos ou personalizados. Os erros de digitação são destacados para que você possa selecionar a palavra correta, aceitar a grafia atual ou desativar a inspeção.

Para corrigir erros de ortografia ou gramática, pressione &shortcut:ShowIntentionActions; para exibir uma lista de ações disponíveis.
