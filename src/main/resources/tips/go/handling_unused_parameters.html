Quick-Fix: remover o nome do parâmetro

Para parâmetros não utilizados, o GoLand realça quando tanto o nome quanto o tipo do parâmetro podem ser removidos. Se o tipo for utilizado para implementar a especificação do método, somente o nome será realçado.

No último caso, clique no nome e pressione &shortcut:ShowIntentionActions; para selecionar uma das seguintes correções rápidas: "remover o nome do parâmetro" ou "renomear para _". A mesma ação também é fornecida para funções utilizadas como um tipo de função.
