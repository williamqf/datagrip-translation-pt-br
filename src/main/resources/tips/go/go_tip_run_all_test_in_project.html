Executar todos os testes no projeto

Para executar todos os testes em um projeto, crie um diretório de configuração de execução/depuração.

Clique em "Executar | Editar configuração de execução". Clique no botão "Adicionar nova configuração" e então selecione "Teste Go". No campo "Diretório" informe o caminho do diretório que contém os arquivos de aplicação e de teste. Certifique-se de que o "Formato de teste" é "Diretório". Clique em "Aplicar" e em "OK".

Para executar a configuração criada, clique em "Executar | <go_test_configuration_name>".
