Experimente o curso de treinamento interativo

O GoLand vem com um instrutor de recursos de IDE integrado. Ele oferece cursos de treinamento interativos para ajudá-lo a aprender como realizar tarefas comuns, como refatoração de código, navegação no projeto e trabalhar com sistemas de controle de versão.

Certifique-se de que o plugin IDE Features Trainer esteja habilitado na guia "Instalado" da página "Plugins" nas configurações do IDE.

Para iniciar os exercícios abrindo a ferramenta "Learn", selecione "Exibir | Janelas de Ferramentas | Learn" ou "Ajuda | Aprenda os recursos do IDE" no menu principal. Tutoriais disponíveis são listados na janela de ferramentas.
