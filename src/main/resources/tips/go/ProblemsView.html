Exibir todos os problemas de código em um arquivo

Não há necessidade de rolar o arquivo para baixo para verificar todos os problemas detectados. Todos os problemas são agrupados na janela de ferramentas "Issues" exclusiva. Nessa janela de ferramentas, você pode verificar os problemas, aplicar as correções sugeridas e corrigi-los livremente.

Selecione "Exibir | Janelas de Ferramentas | Issues" no menu principal ou clique no widget "Inspeção" no canto superior direito do editor para abrir a janela de ferramentas.
