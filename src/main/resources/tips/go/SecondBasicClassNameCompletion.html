Complementação de código

A complementação de código sugere opções disponíveis na posição atual de cursor com base na entrada. Pressionar &shortcut:CodeCompletion; duas vezes encontra todas as funções apropriadas que aceitem o valor como primeiro argumento.

Por exemplo, digitar "/hello/world" e pressionar &shortcut:CodeCompletion; duas vezes mostra uma lista de funções que aceitam string como primeiro argumento.
