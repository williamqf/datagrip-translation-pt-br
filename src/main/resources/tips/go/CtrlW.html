Expansão de seleção de código

Para expandir a seleção, pressione &shortcut:EditorSelectWord;. Cada vez que você pressiona &shortcut:EditorSelectWord;, a seleção se expande para outras áreas do código.

Por exemplo, se um nome de método for selecionado, a seleção poderá ser expandida para sua expressão de chamada, para a declaração inteira ou para um bloco abrangente.
