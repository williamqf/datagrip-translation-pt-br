Completamento de código compatível com tipo

O preenchimento de código "compatível com tipo" analisa o tipo esperado de toda a expressão e ajuda a encontrar métodos e variáveis aplicáveis no contexto atual. Este preenchimento de código funciona após a palavra-chave retornar, argumentos das chamadas de método, atribuições etc. Você pode pressionar &shortcut:SmartTypeCompletion; (código | preenchimento de código | compatível com o tipo) para restringir a lista de preenchimentos.
