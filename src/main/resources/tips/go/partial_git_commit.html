Commit parcial das modificações

Para dividir as modificações feitas no mesmo arquivo em diferentes commits, clique em "Diff" na janela de ferramentas "Commit" &shortcut:ActivateCommitToolWindow;. Selecione as caixas de seleção ao lado de cada bloco de código modificado ou novo para incluir no commit e clique em "Commit". As modificações não selecionadas permanecerão na lista atual de modificações.
