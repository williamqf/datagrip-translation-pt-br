Aninhamento de arquivos '_test'

Para uma melhor organização do projeto, agrupe os arquivos usando regras de aninhamento. Por exemplo, todos os arquivos _test podem ser filhos do arquivo principal.

Para definir uma regra, clique no ícone do "Menu de opções" na janela de ferramentas Projeto e selecione Aninhamento de arquivos. Em seguida, insira a regra no campo Sufixo de arquivos filhos. Ex: _test.go
