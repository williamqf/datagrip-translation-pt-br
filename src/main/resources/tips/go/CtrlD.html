Duplicar Bloco ou Linha de Código

Pressione &shortcut:EditorDuplicate; no editor para duplicar o bloco de código selecionado ou, caso nenhum bloco esteja selecionado, duplicar a linha atual.

Para inserir uma nova linha entre a parte duplicada e o texto original, pressione &shortcut:GotoAction; e digite "Duplicate Entire Lines". Você pode atribuir um atalho para essa ação nas definições do item de menu "Mapeamento de Teclas" &shortcut:ShowSettings;.
