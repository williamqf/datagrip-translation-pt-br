Exibir anotações do Git Blame

Anotações do git exibem informações detalhadas sobre a origem de cada linha de código (Clique com o botão direito na sarjeta e selecione "Anotar com Git Blame" ).

Clique com o botão direito na anotação e selecione "Mostrar diff" para verificar a diferença entre a versão atual do arquivo e a versão anterior.
