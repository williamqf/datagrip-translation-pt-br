**Navegar entre os erros**

Pressione GotoNextError; / GotoPreviousError; para navegar entre os erros de sintaxe destacados.

Pressione PreviousOccurence; / NextOccurence; para navegar entre as mensagens de erro ou os resultados da pesquisa.

Para pular os avisos, clique com o botão direito na barra lateral de Verificação / Barra de Marcadores e selecione "Ir apenas para problemas de alta prioridade".
