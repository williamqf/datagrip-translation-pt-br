Exibe todos os estilos de tag

Para checar rapidamente todos os estilos aplicados em uma tag HTML no momento, selecione "Exibir estilos aplicados à tag " no menu de contexto. Quando o WebStorm abrir a janela de ferramentas "Estilos CSS ", você pode clicar no botão da barra de ferramentas no painel e ir para a definição da tag e suas propriedades no código-fonte.
