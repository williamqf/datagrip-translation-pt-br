Exibir a estrutura de arquivos

&shortcut:FileStructurePopup; (Navegar | Estrutura de arquivos) pode ser usado para navegar no arquivo atual rapidamente.

A estrutura de arquivos lista os membros da class atual. Para navegar para um elemento, selecione-o e pressione &shortcut:EditorEnter; ou &shortcut:EditSource;.

Começar a digitar o nome do item para localizar facilmente itens na lista
