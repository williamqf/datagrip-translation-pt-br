Pesquisar em qualquer lugar

Pressione Shift duas vezes para pesquisar arquivos, ações, símbolos, elementos da IU, ramificações do Git, comentários no projeto inteiro. Pressione Shift duas vezes novamente para pesquisar também itens não incluídos no projeto.

Use as guias ou os atalhos necessários diretamente para filtrar os resultados da pesquisa (por exemplo: &shortcut:GotoClass; para classes, &shortcut:GotoFile; para arquivos, &shortcut:GotoSymbol; para símbolos e &shortcut:GotoAction; para ações).
