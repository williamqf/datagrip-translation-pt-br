Controle de versão

O WebStorm pode ser integrado imediatamente aos sistemas de controle de versão Git, Mercurial, Subversion e Perforce.

O controle de versão do aplicativo permite não apenas a colaboração como também a eliminação do risco de armazenamento de toda a base de códigos localmente. Pressione &shortcut:ActivateVersionControlToolWindow; para exibir o histórico de alterações do projeto e &shortcut:CheckinProject; para enviar as alterações locais ao repositório.
