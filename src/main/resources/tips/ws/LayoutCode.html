Formatação de código

Para formatar o código de acordo com as configurações de estilo de código atuais, selecione " Código | Formatar código " no menu principal ou pressione &shortcut:ReformatCode;.

Você também pode usar " Código | Otimizar importação " para organizar as declarações de importação de acordo com as configurações de estilo de código do projeto e remover importações não usadas.
