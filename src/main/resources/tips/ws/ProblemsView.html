Exibir todos os problemas de código no arquivo

Não é necessário rolar o arquivo para baixo para verificar todos os problemas detectados. Todos os problemas são centralizados na janela de ferramentas dedicada "Problemas". Nessa janela de ferramentas, você pode verificar livremente os problemas, aplicar as correções sugeridas e corrigir os problemas.

Selecione "Exibir | Janela de ferramentas | Problemas" no menu principal ou clique no widget "Inspeção" no canto superior direito do editor para abrir a janela de ferramentas.
