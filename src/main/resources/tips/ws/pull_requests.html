Webhook | Gerenciando Pull Requests

Gerencie os Pull Requests recebidos do GitHub diretamente do WebStorm. Basta selecionar "Git | GitHub | Mostrar Pull Requests" a partir do menu principal. Com o WebStorm, você pode atribuir e mesclar pull requests, ver timelines e comentários em linha, enviar comentários e revisões, e aprovar alterações sem precisar sair do seu IDE.
