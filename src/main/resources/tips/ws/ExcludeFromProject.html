Exclusão de arquivos

Você pode excluir arquivos arbitrários de seu projeto para que eles sejam ignorados pela indexação, inspeções e finalizações de código.

Na caixa de ferramentas "Project", clique com o botão direito no arquivo que você deseja excluir e selecione "Override File Type" no menu de contexto. Depois, selecione "Plain Text" na lista "Select File Type".

Se necessário, você pode sempre reverter o arquivo ao tipo original selecionando "Revert File Type Override" no menu de contexto.
