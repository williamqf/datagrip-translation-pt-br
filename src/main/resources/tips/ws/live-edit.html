Live Edit

Ao usar o Live Edit em uma sessão de depuração, você pode monitorar alterações no código sem precisar atualizar a página. Você sabia que é possível usar o Live Edit mesmo sem a extensão JetBrains IDE Support para o Chrome? Abra a caixa de diálogo "Configurações" &shortcut:ShowSettings;, vá até "Criar, executar e implantar | Depurador | Live Edit" e marque a caixa de seleção "Intervalo de atualização do aplicativo no Chrome".
