Adicionar Descrição ao Ponto de Parada

Para facilitar as buscas futuras, você pode adicionar uma descrição simples ao ponto de parada.

Pressione &shortcut:ViewBreakpoints; para visualizar todos os pontos de parada existentes. Em seguida, clique com o botão direito do mouse sobre o ponto de parada que deseja adicionar a descrição e selecione "Editar Descrição":

A descrição adicionada será útil para buscas futuras de pontos de parada:
