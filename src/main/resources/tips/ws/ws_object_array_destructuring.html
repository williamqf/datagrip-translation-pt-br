Desestruturação

Ao utilizar a desestruturação, é possível extrair facilmente valores de matrizes ou objetos para variáveis. Esse recurso tem uma sintaxe muito reduzida e é bastante utilizado quando é necessário passar dados no aplicativo.

No WebStorm, é possível invocar a desestruturação utilizando a ação de intenção (&shortcut:ShowIntentionActions;). A ação "Substituir por desestruturação de objeto/matriz" remove a atribuição original. Já para manter a atribuição, utilize "Introduzir desestruturação de objeto/matriz".
