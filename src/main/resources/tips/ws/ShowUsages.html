Exibir usos

Lista todas as ocorrências de uma classe, método ou variável em todo o projeto e permite alternar para o item selecionado rapidamente. Para isso, posicione o cursor no símbolo e pressione &shortcut:ShowUsages; (Editar | Pesquisar usos | Exibir usos).

Para alternar para um uso, selecione a ocorrência desejada na lista e pressione &shortcut:EditorEnter;.
