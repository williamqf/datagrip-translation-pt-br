Executando Funções da IDE a partir do Terminal

Quando um comando é realçado no terminal, não o execute imediatamente. Em vez disso, pressione Control+Enter (Windows e Linux) / Control+Enter (macOS) para habilitar a função correspondente da IDE.

Por exemplo, experimente git log para mostrar a aba "Log" na janela de ferramentas Git, ou execute "Show Diff" em vários nomes de ficheiros para abrir no visualizador Diff.
