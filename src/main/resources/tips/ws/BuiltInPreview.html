Pré-visualização integrada

A pré-visualização integrada do WebStorm mostra a saída do código HTML em uma guia separada do editor. O WebStorm salvaguarda as alterações feitas no arquivo HTML ou nos arquivos CSS ou JavaScript vinculados e recarrega automaticamente a visualização.

Para abrir a pré-visualização, coloque o cursor sobre o código HTML, abra o pop-up do navegador e clique no ícone do WebStorm.
