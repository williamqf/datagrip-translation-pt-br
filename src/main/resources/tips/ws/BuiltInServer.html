Alterar a porta do servidor Web incorporado

Para alterar a porta do servidor Web incorporado, navegue até Configurações (&shortcut:ShowSettings;) | Construir, Executar, Implantar | Depurador e defina um novo valor para o campo Porta do servidor incorporado .

Ao visualizar uma prévia de arquivos HTML no navegador &shortcut:WebOpenInAction; , você verá que eles estão sendo executados na nova porta.
