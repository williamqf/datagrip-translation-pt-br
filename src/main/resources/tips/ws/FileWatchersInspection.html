File Watchers

A inspeção "File Watcher disponível " é chamada para todos os arquivos reconhecidos como alvos para um File Watcher definido (SASS, LESS, SCSS, TypeScript, Jade, Dart2JS, CoffeeScript). Se não houver um File Watcher definido aplicável para o projeto atual, o WebStorm sugerirá adicionar um File Watcher definido.
