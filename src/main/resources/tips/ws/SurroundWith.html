Delinear parte do código

Para delinear rapidamente um bloco de código usando uma construção válida, selecione o bloco de código no editor e pressione &shortcut:SurroundWith; ( Código | Delinear).

A lista de opções ou delimitadores disponíveis varia de acordo com o contexto e o idioma. Por exemplo, essa ação pode fazer coisas como envolver um bloco de html com tags
