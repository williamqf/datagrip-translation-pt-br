Controle de Versão

DataSpell prove uma integração pronta para uso com os seguintes sistemas de controle de versão: Git, Mercurial, Subversion e Perforce.

Controle a versão do seu aplicativo para colaborar nele, assim como eliminar os riscos de armazenar todo o seu código localmente. Pressione &shortcut:ActivateVersionControlToolWindow; para ver o histórico das alterações no projeto ou pressione &shortcut:CheckinProject; para enviar suas alterações locais para o repositório.
