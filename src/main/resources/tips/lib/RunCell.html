Executando as células

Use os seguintes atalhos inteligentes para executar rapidamente as células de código:

  • Ctrl + Enter: Executa a célula atual.

  • Shift + Enter: Executa a célula atual e seleciona a célula abaixo dela.

  • Ctrl + Shift + Alt + Enter: Executa todas as células de código no seu notebook.

Você pode encontrar mais opções no menu Executar.
