Experimente nossos cursos de treinamento interativos

O PyCharm contém um IDE Features Trainer integrado, que oferece cursos de treinamento interativos para ajudá-lo a aprender como realizar tarefas comuns, como refatoração de código, movimentação de projetos e utilização de sistemas de controle de versão.

Verifique se o plugin IDE Features Trainer está ativado na guia "Instalados" da página "Plugins" nas configurações do IDE.

Para abrir a janela de ferramentas "Educação" e iniciar as lições, selecione "Exibir | Janela de Ferramentas | Educação" ou "Ajuda | Aprenda os recursos do IDE" no menu principal. A janela de ferramentas listará os tutoriais disponíveis.
