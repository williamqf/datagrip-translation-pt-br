Exibir arquivos usados recentemente

&shortcut:RecentFiles; ( View | Recent Files ) para exibir uma lista dos arquivos abertos recentemente.

Também é possível exibir os resultados de pesquisa de uso recente. Para fazer isso, use o mesmo atalho &shortcut:RecentFiles; na janela de ferramentas "Search " e selecione o resultado de pesquisa pretendido no pop-up "Recent Usage Searches ".
