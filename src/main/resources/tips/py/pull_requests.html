Gestão de requisições pull

Você consegue gerenciar requisições pull recebidas do GitHub diretamente do PyCharm. Para isso, selecione "Git | GitHub | Exibir requisições pull" no menu principal. Usando o PyCharm, você pode designar e fazer merge de requisições pull, visualizar linhas do tempo e comentários in-line, enviar comentários e revisões e aprovar alterações sem sair do IDE.
