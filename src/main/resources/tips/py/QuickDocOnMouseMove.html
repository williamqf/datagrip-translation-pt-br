Posicione o cursor para visualizar a documentação rápida

Não é necessário chamar a documentação rápida explicitamente com &shortcut:QuickJavaDoc;. Posicione o cursor do mouse sobre o símbolo de interesse para que a janela pop-up de documentação rápida seja exibida automaticamente.

Para habilitar este recurso, marque a caixa de seleção "Exibir documentação rápida ao pairar o mouse" na página "Editor | Edição de código" da caixa de diálogo "Configurações" (&shortcut:ShowSettings;).
