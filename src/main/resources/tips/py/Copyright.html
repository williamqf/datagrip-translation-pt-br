Adicionar um cabeçalho de copyright

No PyCharm, você pode configurar cabeçalhos de copyright diferentes para cada conjunto de arquivos. Você pode usar variáveis para adicionar conveniência. Por exemplo, usar $today.year manterá o ano atualizado.

Na caixa de diálogo "Configurações" (&shortcut:ShowSettings;), em "Editor | Copyright | Perfis de copyright", você pode criar um novo perfil e usar variáveis para incluir no texto de copyright.
