Conclusão de código posterior

A conclusão de código posterior é aplicada da direita para a esquerda, evitando que o cursor se mova para trás durante a codificação. Digite um ponto . após o fragmento de código a ser alterado e selecione a opção desejada.

Você também pode pressionar &shortcut:InsertLiveTemplate; para exibir uma lista de candidatos que incluem o template posterior.
