Adicionar células a um notebook Jupyter

Você pode adicionar novas células de uma das seguintes maneiras:

  • No menu "Células", selecione "Inserir célula acima" ou "Inserir célula abaixo".

  • Clique no botão "+" na barra de ferramentas do notebook.

  • Passe o mouse sobre o limite superior de uma célula para ver comandos inline para adicionar uma célula de código ou uma célula Markdown.
