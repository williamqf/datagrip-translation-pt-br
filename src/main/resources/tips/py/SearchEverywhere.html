Busca rápida

Pressione Shift duas vezes para procurar por arquivos, ações, símbolos, elementos de interface, branches Git e comentários em todo o projeto. Pressione Shift duas vezes novamente para procurar por itens que não estão no projeto.

Use as guias ou execute diretamente os atalhos necessários (para classes &shortcut:GotoClass;, para arquivos &shortcut:GotoFile;, para símbolos &shortcut:GotoSymbol; e para ações &shortcut:GotoAction;) para diminuir o escopo da pesquisa.
