Barra de navegação

A Barra de navegação é uma substituta para a janela de ferramentas do "Projeto". Ela exibe a estrutura do projeto, permite abrir arquivos específicos e navegar para elementos específicos do código no arquivo atual.

Pressione &shortcut:ShowNavBar; para exibir a Barra de navegação e navegue por arquivos e pastas usando as teclas de direção.
