Indo de um Comentário para uma Tarefa

Você pode ir de um comentário para uma tarefa no rastreador de tarefa diretamente. Para isso, defina os padrões de navegação de tarefas em "Configurações | Controle de Versão | Navegação de Tarefas".

Quando você for confirmar commits de alterações de lista, digite uma mensagem de confirmação que corresponda a algum dos padrões de navegação de tarefas. Depois, quando você estiver vendo a lista de alterações no repositório, os comentários que corresponderem serão transformados em hiperlinks.

Clique no hiperlink para ir diretamente para a tarefa no rastreador de tarefas.
