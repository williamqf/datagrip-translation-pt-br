Switcher de Navegação

Para navegar entre os arquivos abertos e as janelas de ferramenta, use o Switcher &shortcut:Switcher;. Mantenha a tecla Control pressionada para manter a janela pop-up do Switcher aberta. Para navegar, use as teclas Tab e Shift+Tab.

Para fechar uma guia do editor ou ocultar uma janela de ferramenta, pressione Delete ou BackSpace.
