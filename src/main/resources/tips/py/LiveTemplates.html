Live Templates

Live Templates permitem que você insira construções de código frequentemente usadas. Por exemplo, digite main no editor e pressione &shortcut:EditorTab;.

Como você selecionou um Live Template da lista de candidatos, o template será expandido para as variáveis definidas na página "Editor | Live Templates" do diálogo "Settings" (&shortcut:ShowSettings;).

Use Tab para navegar entre os campos de template.
