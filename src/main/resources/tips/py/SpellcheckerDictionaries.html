Adicionar dicionário de verificação ortográfica personalizado

Se o dicionário predefinido não contém uma palavra necessária, você pode criar seu próprio. Um dicionário personalizado é um arquivo de texto .dic, no qual cada palavra é listada em uma nova linha.

Para adicionar um novo dicionário, pressione &shortcut:ShowSettings; para abrir as configurações e navegue até "Editor | Linguagens naturais | Ortografia".
