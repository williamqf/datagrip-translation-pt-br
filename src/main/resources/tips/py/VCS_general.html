Controle de versão

O PyCharm pode se integrar rapidamente com os sistemas de controle de versão Git, Mercurial, Subversion e Perforce.

O controle de versão permite colaboração e elimina o risco de armazenar toda a base de código localmente. Pressione &shortcut:ActivateVersionControlToolWindow; para ver o histórico de alterações do projeto e pressione &shortcut:CheckinProject; para enviar alterações locais ao repositório.
