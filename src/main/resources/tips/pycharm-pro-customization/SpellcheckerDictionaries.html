Adicione dicionários personalizados para verificação ortográfica

Quando algumas palavras estiverem faltando nos dicionários pré-definidos, você poderá criar os seus próprios. Um dicionário personalizado é um arquivo de texto .dic contendo cada palavra em uma nova linha.

Para adicionar novos dicionários, abra as configurações pressionando &shortcut:MostreConfigurações; e navegue até Editor | Linguagens Naturais | Ortografia.
