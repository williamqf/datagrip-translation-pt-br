Realizar commit de alterações parcialmente

Você pode dividir alterações feitas no mesmo arquivo entre diferentes commits. Na janela de ferramenta Commit &shortcut:ActivateCommitToolWindow;, clique Diff. Selecione a caixa de seleção ao lado de cada pedaço de código modificado ou novo que você deseja efetuar commit e clique em Commit. As alterações não selecionadas permanecerão na changelist atual.
