Visualizar tabela de conteúdos em arquivos Markdown

Você pode pré-visualizar o esboço dos cabeçalhos Markdown na janela de ferramentas Structure. Para abri-la, selecione Window | Tool Windows | Structure no menu principal.

Clique em para navegar do cabeçalho selecionado no editor para o item correspondente na janela de ferramentas Structure.

Da mesma forma, clique em para navegar do item selecionado na estrutura do notebook para a célula Markdown correspondente.
