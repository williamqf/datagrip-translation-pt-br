Cheque ortografia e gramática

Verificações de ortografia e gramática estão habilitadas por padrão. Essas verificações são fornecidas pelo plugin Grazie, que é empacotado com o IDE. Para configurar as definições de revisão, abra as definições pressionando &shortcut:ShowSettings; e navegue para Editor | Natural Languages | Grammar.

Todos os textos, incluindo comentários, strings textuais e literais, são inspecionados contra os dicionários pré-definidos ou personalizados. Erros de digitação são destacados para que você possa escolher a palavra correta, aceitar a ortografia atual ou desabilitar a inspeção.

Para corrigir um erro de ortografia ou gramática, pressione &shortcut:ShowIntentionActions; para ver a lista de ações disponíveis.
