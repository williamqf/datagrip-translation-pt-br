Veja arquivos recentes

Pressione &shortcut:RecentFiles; (View | Recent Files) para ver a lista de arquivos abertos recentemente.

Você também pode exibir os resultados das pesquisas de uso realizadas recentemente. Para fazer isso, use o mesmo atalho &shortcut:RecentFiles; na janela de ferramentas Find e selecione o resultado de pesquisa desejado no pop-up Recent Find Usages.
