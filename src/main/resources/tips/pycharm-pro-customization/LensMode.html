Modo de lente

Para pré-visualizar o código sem rolar até ele, passe o mouse sobre um aviso, barra de erro ou uma seção do código-fonte na barra de rolagem e verá uma lente:

Para desativar a lente, clique com o botão direito do mouse na barra de erro na barra de rolagem e desmarque a caixa de seleção Mostrar lente de código ao passar o mouse pela barra de rolagem.
