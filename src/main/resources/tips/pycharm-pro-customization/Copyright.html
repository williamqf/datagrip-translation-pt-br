Adicionar avisos de copyright

No PyCharm, você pode configurar diferentes avisos de copyright para diferentes conjuntos de arquivos. Por conveniência, você pode usar variáveis. Por exemplo, use $today.year para manter o ano atualizado.

Na caixa de diálogo Configurações (&shortcut:ShowSettings;), selecione Editor | Copyright | Perfis de Copyright, crie um novo perfil e adicione o texto para o seu aviso de copyright usando variáveis.
