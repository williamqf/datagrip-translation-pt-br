Controle de Versão

O PyCharm fornece integração pronta para uso com os seguintes sistemas de controle de versão: Git, Mercurial, Subversion e Perforce.

Faça o controle de versão do seu aplicativo para colaborar com outras pessoas e elimine os riscos de armazenar toda a sua base de código localmente. Pressione &shortcut:ActivateVersionControlToolWindow; para exibir o histórico das alterações no projeto ou pressione &shortcut:CheckinProject; para enviar as alterações locais para o repositório.
