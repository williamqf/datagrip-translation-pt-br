Mover instruções para cima e para baixo

As ações Código | Mover Instrução Para Cima/Para Baixo são úteis para reorganizar linhas de código, por exemplo para mover uma declaração de variável mais perto do seu uso.

Selecione um fragmento de código e pressione &shortcut:MoveStatementUp; ou &shortcut:MoveStatementDown;.

Quando nada for selecionado no editor, a linha na posição do cursor será movida.
