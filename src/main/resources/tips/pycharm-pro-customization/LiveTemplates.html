Modelos de preenchimento automático

Use os modelos de preenchimento automático para inserir construções de código frequentes. Por exemplo, digite main no editor e pressione &shortcut:EditorTab;.

Quando você seleciona um modelo de preenchimento automático da lista de sugestões, o modelo se expande para a variante definida na página Editor | Modelos de preenchimento automático da caixa de diálogo Configurações (&shortcut:ShowSettings;).

Use Tab para mover-se entre os campos do modelo.
