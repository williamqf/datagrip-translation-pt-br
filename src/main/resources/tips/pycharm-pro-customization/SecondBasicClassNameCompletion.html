Preenchimento de código

O preenchimento de código sugere as opções que podem ser acessadas da posição atual do cursor conforme você digita. Pressione &shortcut:CodeCompletion; duas vezes para estender a lista de sugestões.

Se você usar o preenchimento automático para descobrir um nome de classe e esta classe ainda não tiver sido importada, a instrução de importação será gerada automaticamente.
