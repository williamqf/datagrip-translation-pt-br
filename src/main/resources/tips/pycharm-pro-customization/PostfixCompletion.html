Completação de código pós-fixo

A complementação de código pós-fixo é aplicada da direita para esquerda para evitar saltos de cursor para trás ao codificar. Digite um ponto . após o fragmento de código que deseja alterar e selecione a opção desejada.

Para ver a lista de sugestões com modelos pós-fixos, você também pode pressionar &shortcut:InsertLiveTemplate;.
