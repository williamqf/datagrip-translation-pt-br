Experimente cursos de treinamento interativos

O PyCharm inclui a ferramenta embutida Treinador de Componentes do IDE. Este oferece cursos de treinamento interativos para ajudá-lo a aprender a executar algumas tarefas comuns, como refatoração de código, navegação por projetos, depuração e como trabalhar com controle de versão.

Verifique se o plugin Treinador de Componentes do IDE foi habilitado na guia Instalado da página Plugins nas configurações do IDE.

Para abrir a janela de ferramentas Aprenda e começar seus exercícios, selecione Visualizar | Janelas de ferramentas | Aprenda ou Ajuda | Aprenda os componentes do IDE no menu principal. A janela de ferramentas lista os tutoriais disponíveis.
