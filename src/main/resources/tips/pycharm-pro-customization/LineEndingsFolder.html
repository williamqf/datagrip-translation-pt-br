Alterar os separadores de linha

Você não precisa abrir um arquivo no editor para alterar seu estilo de separador de linha. Use a janela de ferramentas Project: selecione um ou mais arquivos ou pastas, selecione File | File Properties| Line Separators no menu principal e depois escolha o estilo desejado para o final da linha.

Para um diretório, o novo separador de linha se aplica recursivamente.
