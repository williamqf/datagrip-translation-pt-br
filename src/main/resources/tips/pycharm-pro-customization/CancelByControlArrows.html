Selecionar sugestões de preenchimento

Quando a opção Mostrar sugestões enquanto digita está habilitada para preenchimento de código, você pode usar as ações Pesquisar para cima e Pesquisar para baixo para fechar a lista de sugestões e mover o cursor para cima ou para baixo no editor.

No Windows, use os atalhos Ctrl+Down e Ctrl+Up.

No macOS, os atalhos não são configurados por padrão, então você pode atribuir qualquer combinação de teclas que seja conveniente para você em Configurações | Mapa de teclas.
