Conclusão de código

Você pode aceitar a seleção atual no código de lista de sugestões com a tecla de ponto (.), vírgula (,), ponto e vírgula (;), espaço e alguns outros caracteres.

O nome selecionado é digitado automaticamente no editor seguido pelo caractere digitado.

Para usar esse recurso, verifique se o Inserir sugestão selecionada pressionando espaço, ponto ou outras teclas sensíveis ao contexto checkbox está marcado na página Editor | Geral | Conclusão de Código da caixa de diálogo Configurações &shortcut:ShowSettings;.
