Gerencie solicitações de recebimento

Gerencie as solicitações de recebimento do GitHub diretamente do PyCharm: no menu principal, selecione Git | GitHub | Ver solicitações de recebimento. O PyCharm permite atribuir e mesclar solicitações de recebimento, ver o cronograma e comentários integrados, enviar comentários e revisões e aceitar alterações sem sair do IDE.
