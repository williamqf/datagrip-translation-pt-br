Adicionar uma célula em um notebook Jupyter

Você pode adicionar uma nova célula de uma das seguintes maneiras:

  • Selecione Adicionar Célula Acima ou Adicionar Célula Abaixo no menu Célula.

  • Clique em + na barra de ferramentas do notebook.

  • Passe o cursor do mouse sobre a borda superior de uma célula para ver os comandos embutidos de adicionar uma célula de código e uma célula de redução.
