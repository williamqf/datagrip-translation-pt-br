Buscar e substituir

Pressione &shortcut:Find; para visualizar a barra de busca. Pressione &shortcut:Replace; para adicionar outro campo onde você pode digitar a string de substituição.

Na caixa de diálogo Buscar em arquivos, você pode alternar para substituir pressionando &shortcut:ReplaceInPath;. Da mesma forma, pressione &shortcut:FindInPath; para ocultar o campo Substituir por e alternar para a busca regular.
