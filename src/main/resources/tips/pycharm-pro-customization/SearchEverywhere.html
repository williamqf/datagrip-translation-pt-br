Pesquise em todos os lugares

Pressione Shift duas vezes para pesquisar arquivos, ações, símbolos, elementos da interface do usuário, branches do Git e comentários em todo o seu projeto. Pressionar o Shift duas vezes novamente estenderá a pesquisa para itens fora do projeto.

Use guias ou atalhos diretos &shortcut:GotoClass; para classes, &shortcut:GotoFile; para arquivos, &shortcut:GotoSymbol; para símbolos e &shortcut:GotoAction; para ações para restringir seus resultados de pesquisa.
