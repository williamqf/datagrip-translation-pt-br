Pesquisar em todos os lugares

Pressione Shift duas vezes para pesquisar arquivos, ações, símbolos, elementos da interface do usuário, ramificações do Git e comentários do projeto inteiro. Pressione novamente Shift duas vezes para pesquisar também itens que não estão no projeto.

Use a tabulação ou diretamente o atalho desejado (&shortcut:GotoClass; para classes, &shortcut:GotoFile; para arquivos; &shortcut:GotoSymbol; para símbolos e &shortcut:GotoAction; para ações) para refinar a pesquisa.
