Gerenciamento de Pull Requests

É possível gerenciar pull requests recebidos do GitHub diretamente pelo Aqua. Para isso, selecione a opção "Git | GitHub | Mostrar Pull Requests" no menu principal. O Aqua permite atribuir e mesclar pull requests, exibir cronogramas e comentários em linha, enviar comentários e revisões e aprovar alterações sem sair do IDE.
