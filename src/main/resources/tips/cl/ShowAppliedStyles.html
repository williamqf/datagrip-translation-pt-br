Mostrar todos os estilos de tag

Para verificar rapidamente todos os estilos aplicados atualmente a uma tag de HTML, selecione "Mostrar estilos aplicados na tag" no menu de contexto. Quando o CLion abrir a janela de ferramentas "Estilos CSS", você poderá clicar na barra de ferramentas do painel para navegar até as definições de tags e propriedades que estão no código-fonte.
