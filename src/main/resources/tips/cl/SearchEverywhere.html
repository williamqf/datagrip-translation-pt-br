Pesquise em qualquer lugar

Pressione Shift duas vezes para pesquisar arquivos, ações, símbolos, elementos da interface do usuário, ramificações do Git e comentários em todo o projeto. Pressione Shift duas vezes novamente para pesquisar também itens que não estão incluídos no projeto.

Para refinar os resultados da pesquisa, utilize as guias ou use diretamente o atalho necessário (&shortcut:GotoClass; para classes, &shortcut:GotoFile; para arquivos, &shortcut:GotoSymbol; para símbolos e &shortcut:GotoAction; para ações).
