Pressione &shortcut:Generate; em qualquer lugar do seu código ou selecione Code | Generate. O menu irá mostrar todas as opções disponíveis para geração de código. Use esse menu para gerar rapidamente construtores/destrutores, getters/setters, uma variedade de operadores, métodos implementados/sobrepostos.

As opções de teste e classe de protótipos também serão exibidas quando o menu Generate for chamado a partir de um arquivo de Teste do Google.
