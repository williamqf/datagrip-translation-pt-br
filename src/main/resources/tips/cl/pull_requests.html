Gerenciamento de solicitação de pull

É possível gerenciar diretamente as solicitações de pull recebidas do GitHub a partir do CLion. Para isso, selecione "Git | GitHub | Exibir solicitações de pull" no menu principal. O CLion permite atribuir e mesclar solicitações de pull, exibir cronogramas e comentários embutidos, enviar comentários e revisões, além de aprovar alterações sem precisar sair do IDE.
