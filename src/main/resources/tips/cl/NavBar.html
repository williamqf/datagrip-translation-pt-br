Barra de Navegação

A barra de navegação substitui a janela de ferramentas "Projeto ". Ela permite que você visualize a estrutura do projeto, abra arquivos específicos ou vá para elementos de código particulares no arquivo atual.

Pressione &shortcut:ShowNavBar; para exibir a barra de navegação e use as teclas de direção para alternar entre arquivos e pastas.
