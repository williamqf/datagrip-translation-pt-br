Alterar o Foco nas Janelas de Ferramentas

Pressione &shortcut:EditorEscape; na Janela de Ferramentas para mover o foco para o Editor.

Pressione &shortcut:HideActiveWindow; para mover o foco para o Editor e ocultar a Janela de Ferramentas ativa ou a última ativa.

Pressione &shortcut:JumpToLastWindow; para mover o foco do Editor para a última Janela de Ferramentas com foco.
