Adicionar um dicionário personalizado de verificação ortográfica

Caso o dicionário predefinido não contenha uma palavra necessária, você poderá criar a sua própria palavra. Um dicionário personalizado é um arquivo de texto .dic no qual cada palavra está registrada em uma nova linha.

Para adicionar um novo dicionário, pressione &shortcut:ShowSettings; para abrir as configurações e acesse "Editor | Linguagem natural | Ortografia".
