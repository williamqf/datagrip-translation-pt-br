Busca e Localização

Pressione &shortcut:Find; para exibir a barra de pesquisa. Pressione &shortcut:Replace; para adicionar outro campo onde é possível inserir uma sequência de substituição.

Na caixa de diálogo Search within files (Buscar nos arquivos), pressione &shortcut:ReplaceInPath; para alternar para a substituição. Da mesma forma, pressione &shortcut:FindInPath; para ocultar o campo Replace (Substituir) e alternar para a pesquisa normal.
