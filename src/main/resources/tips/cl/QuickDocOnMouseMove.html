Passe o cursor para exibir o Quick Doc

Não é necessário chamar o Quick Doc explicitamente com &shortcut:QuickJavaDoc;. O pop-up do Quick Doc será exibido automaticamente quando você passar o cursor do mouse sobre o símbolo desejado.

Para habilitar esta funcionalidade, vá para a página "Editor | Code Style" na caixa de diálogo "Settings" (&shortcut:ShowSettings;) e marque a caixa de seleção "Hover to show quick doc".
