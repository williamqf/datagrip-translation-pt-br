Commit parcial das alterações

Para dividir as alterações feitas no mesmo arquivo em commits diferentes, clique em "Diferenças" na janela de ferramentas "Commit" &shortcut:ActivateCommitToolWindow;. Selecione a caixa de seleção ao lado de cada pedaço de código alterado ou novo a ser commitado e clique em "Commit". As alterações não selecionadas permanecerão na lista de alterações atual.
