Executando recursos de IDE pelo terminal

Se um comando estiver destacado no terminal, não o execute imediatamente. Em vez disso, pressione Control+Enter (Windows e Linux) / Control+Enter (macOS) para habilitar o recurso correspondente de IDE.

Por exemplo, experimente git log para exibir a guia "Log" na janela de ferramentas Git ou execute "Mostrar diferenças" em vários nomes de arquivo para abri-los no visualizador de diferenças.
