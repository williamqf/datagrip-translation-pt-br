Puxar para cima o membro e empurrar o membro para baixo são refatorações que ajudam a mover os membros da classe para cima e para baixo na hierarquia, ou seja, eles permitem que você puxe os membros da classe para cima, para a classe base, ou os extraia para subclasses.

Para invocar esta refatoração, selecione a classe que deseja modificar e, no menu principal ou de contexto, chame Refatoração | Puxar para cima o membro ou Refatoração | Empurrar o membro para baixo.
