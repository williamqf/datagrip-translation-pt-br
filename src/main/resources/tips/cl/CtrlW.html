Expandir Seleção de Código

Para expandir uma seleção, pressione &shortcut:EditorSelectWord;. Cada vez que pressionar &shortcut:EditorSelectWord;, a seleção se expande para outras regiões do código.

Por exemplo, se um nome de método estiver selecionado, você pode expandir a seleção para a expressão que chama o método, a declaração inteira, o bloco que o contém, etc.
