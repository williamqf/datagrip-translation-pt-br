Exibindo Anotações Git Blame

As anotações Git exibem informações detalhadas sobre a origem de cada linha de código (clique com o botão direito no gutter e selecione "Anotar com Git Blame").

Clique com o botão direito na anotação e selecione "Mostrar Diferença" para ver a diferença entre a versão atual do arquivo e a versão anterior.
