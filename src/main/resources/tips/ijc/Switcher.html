Switcher de navegação

Para navegar entre arquivos abertos e janelas de ferramentas, use o switcher &shortcut:Switcher;. Mantenha pressionado o atalho Control para abrir a janela do switcher. Use os atalhos Tab e Shift+Tab para navegar.

Para fechar abas do editor ou ocultar janelas de ferramentas, pressione Delete ou BackSpace.
