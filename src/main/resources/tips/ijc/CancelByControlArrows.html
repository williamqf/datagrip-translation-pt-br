Seleção de Preenchimento Automático

Se a opção "Mostrar sugestoes enquanto digita" estiver habilitada para preenchimento automático de códigos, as ações "Pesquisa para baixo" e "Pesquisa para cima" podem ser usadas para ocultar a lista de sugestões e mover o cursor para baixo ou para cima no editor.

No Windows, use os atalhos &shortcut:EditorLookupDown; e &shortcut:EditorLookupUp;.

No macOS, o atalho não é configurado por padrão. Portanto, você pode atribuir quaisquer combinações de teclas conforme sua preferência em "Configurações | Mapeamentos de Teclas".
