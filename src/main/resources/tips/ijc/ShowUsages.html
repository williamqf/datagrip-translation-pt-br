Mostrar usos

Lista todos os usos de uma classe, método ou variável em todo o projeto e permite que você navegue rapidamente até a ocorrência selecionada. Para fazer isto, coloque o cursor no símbolo e pressione &shortcut:ShowUsages; (Editar | Localizar usos | Mostrar usos).

Para navegar até um uso, selecione o uso desejado na lista e pressione &shortcut:EditorEnter;.
