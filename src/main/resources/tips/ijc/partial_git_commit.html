Cometimento parcial de alterações

Para dividir as alterações feitas no mesmo arquivo em diferentes commits, clique em "Diffs" na janela de ferramentas "Commit" &shortcut:ActivateCommitToolWindow;. Selecione o checkbox ao lado de cada pedaço de código modificado ou novo para commitar e clique em "Commit". As alterações não selecionadas permanecerão na lista de alterações atual.
