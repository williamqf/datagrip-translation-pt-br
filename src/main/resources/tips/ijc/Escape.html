Alterar o foco em uma janela de ferramentas

Na janela de ferramentas, pressione &shortcut:EditorEscape; para mover o foco para o editor.

Pressione &shortcut:HideActiveWindow; para mover o foco para o editor e ocultar a janela de ferramentas atualmente ativa ou que estava ativa recentemente.

Pressione &shortcut:JumpToLastWindow; para mover o foco do editor para a janela de ferramentas que recebeu o foco por último.
