Realçar as ocorrências em uso nos arquivos

&shortcut:HighlightUsagesInFile; (Editar | Buscar Ocorrências | Realçar as ocorrências em uso nos arquivos) pressiona para rapidamente realçar as ocorrências em uso de uma determinada variável no arquivo atual.

&shortcut:FindNext; e &shortcut:FindPrevious; para avançar para as ocorrências realçadas. &shortcut:EditorEscape; para remover o realce.
