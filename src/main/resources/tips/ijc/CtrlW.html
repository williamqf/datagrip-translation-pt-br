Expanda a seleção de código

Para expandir a seleção, pressione &shortcut:EditorSelectWord;. Cada vez que você pressionar &shortcut:EditorSelectWord;, a seleção se expandirá para outras áreas do código.

Por exemplo, se um nome de método for selecionado, você pode expandir a seleção para a expressão que chama o método, a declaração inteira, o bloco incluído, etc.
