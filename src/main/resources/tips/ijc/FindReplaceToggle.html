Pesquisa e substituição

Pressione &shortcut:Find; para mostrar a barra de pesquisa. Pressione &shortcut:Replace; para abrir outro campo para inserir a string de substituição.

No diálogo " Localizar arquivo ", pressione &shortcut:ReplaceInPath; para substituição. Da mesma forma, pressione &shortcut:FindInPath; para alternar para a pesquisa regular, ocultando o campo Substituir
