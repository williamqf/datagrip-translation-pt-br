Controle de pull requests

Você pode gerenciar pull requests do GitHub diretamente do IntelliJIDEACommunity. Para isso, no menu principal, selecione Git | GitHub | Exibir pull requests . Usando o IntelliJ IDEA Community, você pode atribuir e mesclar pull requests, visualizar o cronograma e comentários em linha, enviar comentários e reviews e aprovar alterações sem sair do IDE.
