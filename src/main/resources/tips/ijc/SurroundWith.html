Envolvendo blocos de código

Para envolver rapidamente um bloco de código com um constructo válido, selecione o bloco de código no editor e pressione &shortcut:SurroundWith; (Código | Envolver).

A lista de opções ou wrappers disponíveis varia de acordo com o contexto e a linguagem. Por exemplo, você pode realizar ações como envolver um bloco html com tags.
