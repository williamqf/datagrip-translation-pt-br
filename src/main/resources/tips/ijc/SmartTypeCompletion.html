Preenchimento de código do tipo de correspondência

A finalização de código de "tipo de correspondência" analisa o tipo esperado de uma expressão inteira e ajuda a localizar métodos e variáveis aplicáveis no contexto atual. Essa finalização de código funciona após a palavra-chave return, em atribuições, em listas de argumentos de chamada de método, etc. Pressione &shortcut:SmartTypeCompletion; ( Código | Preenchimento de Código | Tipo de Correspondência) para reduzir a lista de finalizações.
