Adicionar Aviso de Direitos Autorais

No IntelliJIDEACommunity, você pode configurar um aviso de direitos autorais diferente para cada conjunto de arquivos. Você pode usar variáveis para facilitar. Por exemplo, usar $today.year mantém o ano atualizado.

Na caixa de diálogo "Settings" (&shortcut:ShowSettings;), selecione "Editor | Copyright | Copyright Profiles" para criar um novo perfil ou adicionar texto de aviso de direitos autorais com variáveis.
