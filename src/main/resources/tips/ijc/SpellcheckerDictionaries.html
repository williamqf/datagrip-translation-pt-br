Adicionar dicionário de verificação ortográfica personalizado

Se houver palavras faltando no dicionário predefinido, você pode criar as suas próprias. O dicionário personalizado é um arquivo de texto .dic com cada palavra em uma nova linha.

Para adicionar um novo dicionário, pressione &shortcut:ShowSettings; para abrir as configurações e vá até "Editor | Linguagem natural | Ortografia".
