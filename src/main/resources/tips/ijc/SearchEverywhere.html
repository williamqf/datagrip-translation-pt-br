Pesquisa global

Pressione Shift duas vezes para pesquisar por arquivos, ações, símbolos, elementos da interface do usuário, comandos do Git e comentários em todo o projeto. Pressione Shift duas vezes novamente para pesquisar também por itens fora do projeto.

Para refinar os resultados da pesquisa, use as guias ou use diretamente os atalhos necessários (para classes, &shortcut:GotoClass; para arquivos, &shortcut:GotoFile; para símbolos, &shortcut:GotoSymbol; e para ações, &shortcut:GotoAction;).
