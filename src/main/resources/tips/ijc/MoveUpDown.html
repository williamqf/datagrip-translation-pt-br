Movendo blocos de código para cima ou para baixo

A ação "Code | Mover bloco para cima/Mover bloco para baixo" é útil para reorganizar suas linhas de código, como aproximar declarações de variáveis aonde elas são usadas.

Selecione um fragmento de código e pressione &shortcut:MoveStatementUp; ou &shortcut:MoveStatementDown;.

Se nada for selecionado no editor, o bloco da linha onde está o cursor será movido.
