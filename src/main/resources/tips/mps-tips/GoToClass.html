Para abrir um nó do editor rapidamente, pressione &shortcut:GotoClass; (Navegar | Ir para o Nó Raiz) e comece digitando o nome do nó (Corcundas de camelo permitidas). Selecione o nó a partir de uma lista suspensa que aparece.

Você pode abrir qualquer arquivo em seu projeto de forma similar usando &shortcut:GotoFile; (Navegar | Ir para Arquivo)
