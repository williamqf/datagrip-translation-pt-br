Consoles de banco de dados, arquivos scratch e buffers scratch

Consoles de banco de dados são arquivos SQL que já estão conectados a uma fonte de dados. Quando você cria uma fonte de dados, um console de banco de dados é criado automaticamente. Para abrir o console, clique com o botão direito na fonte de dados e selecione Navigation | Jump to Query Console | Open Default Console. Para criar um novo console, clique com o botão direito na fonte de dados e selecione New | Query Console.

Arquivos scratch são arquivos SQL que não estão associados a uma fonte de dados. Refira-se a arquivos scratch como notas ou rascunhos temporários para algumas ideias de código fora do contexto do projeto. Para criar um arquivo scratch, navegue até File | New | Scratch File (ou pressione &shortcut:NewScratchFile;).

Buffers scratch são arquivos de texto simples sem nenhum recurso de assistência à codificação. Os buffers scratch podem ser usados para listas de tarefas e notas simples para você mesmo. Não há nenhum item de menu dedicado para a ação de criar um novo buffer scratch, mas você pode usar o pop-up Find Action (&shortcut:GotoAction;) e executar a ação New Scratch Buffer.
