Arquivos SQL como fonte de dados DDL

Arquivos SQL que contêm declarações CREATE podem ser usados como fontes de dados. Como resultado, você pode criar diagramas, referenciar todas as tabelas, colunas e outros objetos definidos nesses arquivos no editor.

Para criar uma fonte de dados DDL, arraste o arquivo DDL para o Database Explorer (Exibir | Janelas de Ferramentas | Database Explorer) do seu gerenciador de arquivos.
