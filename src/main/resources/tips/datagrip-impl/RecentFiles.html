Visualizar arquivos recentes

Se você acidentalmente fechou um console de consulta, um script SQL ou qualquer outra aba, pressione &shortcut:ArquivosRecentes;. O diálogo Arquivos Recentes exibe uma lista de arquivos editados recentemente.

Pressione &shortcut:ArquivosRecentes; (Ver | Arquivos Recentes) para visualizar a lista de arquivos abertos recentemente.
