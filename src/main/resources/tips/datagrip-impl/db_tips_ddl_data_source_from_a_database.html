Despejar para a Fonte de Dados DDL

A fonte de dados DDL é uma fonte de dados virtual com um esquema baseado em um número de scripts SQL. Armazenar esses arquivos no Sistema de Controle de Versão (VCS) é uma maneira de manter seu banco de dados no VCS.

Para criar uma fonte de dados DDL, clique com o botão direito do mouse em uma fonte de dados no Database Explorer e selecione Import/Export | Dump to DDL Data Source.
