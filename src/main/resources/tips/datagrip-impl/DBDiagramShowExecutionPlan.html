Plano de execução de sentença

Você pode visualizar um conjunto de etapas que foram usadas para acessar dados em um banco de dados.

Para criar um plano de execução, clique com o botão direito do mouse em uma consulta no console de consulta e selecione Explicar o Plano. Se desejar criar um diagrama para a consulta, clique no ícone Mostrar Visualização.
