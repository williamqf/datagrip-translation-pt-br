Buscar e substituir

Pressione &shortcut:Find; para exibir a barra de pesquisa. Pressione &shortcut:Replace; para adicionar outro campo onde você pode digitar a string de substituição.

Na caixa de diálogo Find in Files, você pode alternar para substituir pressionando &shortcut:ReplaceInPath;. Da mesma forma, pressione &shortcut:FindInPath; para ocultar o campo Replace with e alternar para a pesquisa regular.
