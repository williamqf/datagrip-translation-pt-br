DDL, dados e local do objeto do banco de dados no Database Explorer

Pressione &shortcut:GotoDeclaration; para abrir a DDL do objeto selecionado. Por exemplo, se você pressionar este atalho no nome de uma tabela em uma instrução SELECT, verá o DDL dessa tabela (a instrução CREATE TABLE).

Pressione &shortcut:EditSource; para abrir os dados do objeto. Por exemplo, se você pressionar este atalho no nome de uma tabela em uma instrução SELECT, verá o conteúdo da tabela.

Pressione Alt+Shift+B para navegar até o objeto na janela de ferramentas Database Explorer .
