Modelos dinâmicos

Use modelos dinâmicos para inserir construções de código frequentes. Por exemplo, digite sel no editor.

Ao selecionar um modelo dinâmico na lista de sugestões, o modelo se expande para a variante definida na página Editor | Modelos Dinâmicos da caixa de diálogo Configurações/Preferências (&shortcut:ShowSettings;).

Use Tab para navegar entre os campos do modelo.
