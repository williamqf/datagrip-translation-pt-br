Sincronização de estados de banco de dados

Para sincronizar o estado do banco de dados automaticamente, no Database Explorer (View | Tool Windows | Database Explorer), clique no botão Data Source Properties () e selecione a fonte de dados que deseja alterar. Na guia Options, marque a caixa de seleção Auto sync em Introspection.

Se a caixa de seleção Auto sync for desmarcada, a exibição da fonte de dados no Database Explorer (View | Tool Windows | Database Explorer) é sincronizada com o estado real do banco de dados somente quando você clica no ícone Refresh ou pressiona &shortcut:Refresh;.
