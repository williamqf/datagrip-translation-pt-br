Visualizar arquivos lado a lado

Você pode visualizar múltiplos arquivos lado a lado no editor. Clique com o botão direito na aba do editor desejada e selecione como você quer dividir o editor (Dividir Para a Direita ou Dividir Para Baixo).

Alternativamente, arraste uma aba para qualquer área do editor para ativar o modo de tela dividida.
