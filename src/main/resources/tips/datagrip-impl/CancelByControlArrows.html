Selecione sugestões de preenchimento

Quando a opção Mostrar sugestões conforme você digita está habilitada para preenchimento de código, você pode usar as ações Pesquisar acima e Pesquisar abaixo para fechar a lista de sugestões e mover o cursor para cima ou para baixo no editor.

No macOS, os atalhos não são configurados por padrão, portanto, você pode atribuir qualquer combinação de teclas conveniente para você em Preferências | Mapa de Teclas.
