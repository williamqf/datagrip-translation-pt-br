Pesquisa em Todo Lugar

Pressione Shift duas vezes para pesquisar arquivos, ações, símbolos, elementos de IU, branches do Git e comentários em seu projeto. Pressionando Shift duas vezes novamente, você estenderá a pesquisa para itens que não são do projeto.

Use as abas ou os atalhos diretos &shortcut:GotoClass; para objetos do banco de dados, &shortcut:GotoFile; para arquivos, &shortcut:GotoSymbol; para símbolos, e &shortcut:GotoAction; para ações para restringir os resultados da pesquisa.
