Realçar usos dentro de um arquivo

Pressione &shortcut:HighlightUsagesInFile; (Editar | Localizar Usos | Realçar Usos no Arquivo) para realçar rapidamente os usos de uma determinada variável no arquivo atual.

Pressione &shortcut:FindNext; e &shortcut:FindPrevious; para navegar pelos usos realçados. Pressione &shortcut:EditorEscape; para remover o realce.
