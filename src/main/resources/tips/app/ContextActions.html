Melhore o código com as ações de contexto

Pressione &shortcut:ShowIntentionActions; no editor para corrigir um erro ou aviso realçado, melhorar ou otimizar uma construção de código. Para algumas ações de intenção, você pode abrir uma prévia ou uma descrição da ação pressionando &shortcut:QuickJavaDoc; (Visualizar | Documentação rápida).
