Exibir anotações de culpado Git

As anotações Git mostram informações detalhadas na origem de cada linha de código (Clique com o botão direito do mouse no gutter e selecione Annotate with Git Blame).

Clique com o botão direito em uma anotação e escolha Show Diff para revisar as diferenças entre a versão atual e a versão anterior do arquivo.
