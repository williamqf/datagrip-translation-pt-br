Gerenciar requisições pull

Gerencie requisições pull do GitHub diretamente no IntelliJIDEACommunity: no menu principal, selecione Git | GitHub | Visualizar requisições pull. O IntelliJIDEACommunity permite que você atribua e mescle requisições pull, visualize o cronograma e comentários embutidos, envie comentários e revisões e aceite mudanças sem sair do IDE.
