Verifique a ortografia e a gramática

As verificações de ortografia e gramática são ativadas por padrão. Essas verificações são fornecidas pelo plugin Grazie, que é fornecido com o IDE. Para configurar as configurações de revisão, abra as configurações pressionando &shortcut:ShowSettings; e navegue até Editor | Idiomas Naturais | Gramática.

Todos os textos, incluindo comentários, strings de texto e literais, são inspecionados em relação aos dicionários predefinidos ou personalizados. Os erros de digitação são destacados para que você possa escolher a palavra correta, aceitar a ortografia atual ou desativar a inspeção.

Para corrigir um erro de ortografia ou gramática, pressione &shortcut:ShowIntentionActions; para ver a lista de ações disponíveis.
