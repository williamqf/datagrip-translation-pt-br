Adicionar avisos de direitos autorais

No IntelliJIDEACommunity, você pode configurar diferentes avisos de direitos autorais para diferentes conjuntos de arquivos. Por conveniência, você pode usar variáveis. Por exemplo, use $today.year para manter o ano atualizado.

No diálogo Configurações/Preferências (&shortcut:ShowSettings;), selecione Editor | Direitos Autorais | Perfis de Direitos Autorais, crie um novo perfil e adicione o texto para o seu aviso de direitos autorais usando variáveis.
