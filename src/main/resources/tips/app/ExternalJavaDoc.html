Documentação externa

Para abrir seu navegador com a página de documentação para o elemento no cursor, pressione &shortcut:ExternalJavaDoc; ou selecione Ferramentas | Navegadores da Web e visualização no menu principal.

Certifique-se de definir um navegador padrão nas configurações. Para verificar o navegador padrão, abra as configurações pressionando &shortcut:ShowSettings; e navegue até Ferramentas | Navegadores da Web e visualização.
