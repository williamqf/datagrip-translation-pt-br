Pesquisar em Todos os Locais

Aperte Shift duas vezes para pesquisar arquivos, ações, símbolos, elementos da interface do usuário, branches do Git e comentários em todo o seu projeto. Ao pressionar duas vezes Shift novamente, estenderá a pesquisa para itens que não são do projeto.

Use as guias ou atalhos diretos &shortcut:GotoClass; para classes, &shortcut:GotoFile; para arquivos, &shortcut:GotoSymbol; para símbolos e &shortcut:GotoAction; para filtrações nas ações para restringir os resultados da sua pesquisa.
