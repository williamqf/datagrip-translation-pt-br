Selecionar sugestões de conclusão

Quando a opção Mostrar sugestões conforme você digita está ativada para conclusão de código, você pode usar as ações Lookup Para cima e Lookup Para baixo para fechar a lista de sugestões e mover o cursor para cima ou para baixo no editor.

No Windows, use os atalhos Ctrl+Down e Ctrl+Up.

No macOS, os atalhos não são configurados por padrão, portanto, você pode atribuir qualquer combinação de teclas conveniente para você em Preferências | Mapa de teclas.
