Crie construções de código com autocompletar

Você pode criar construções de código usando autocompletar de instruções. Comece a digitar uma declaração de método, uma chamada de método ou uma instrução como se, faça -enquanto, tente -pegue, ou retorne. Pressione &shortcut:EditorCompleteStatement; para completar a instrução em uma construção sintaticamente correta.
