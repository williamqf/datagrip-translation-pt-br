Expandir seleção de código

Pressione &shortcut:EditorSelectWord; para expandir a seleção de código. Cada vez que você pressionar &shortcut:EditorSelectWord;, a seleção se expande para outras áreas do código.

Por exemplo, a seleção se expande de um nome de método para a expressão que chama esse método, depois para toda a declaração, depois para o bloco container e assim por diante.
