Mover declarações para cima e para baixo

As ações Código | Mover Declaração para Cima/Baixo são úteis para reorganizar linhas de código, por exemplo, para aproximar uma declaração de variável ao uso da variável.

Selecione um fragmento de código e pressione &shortcut:MoveStatementUp; ou &shortcut:MoveStatementDown;.

Quando nada for selecionado no editor, a linha na posição do cursor será movida.
