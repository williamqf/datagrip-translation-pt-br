Após digitar a parte relevante de uma declaração ou comando, é possível pressionar &shortcut:EditorCompleteStatement; para inserir automaticamente elementos de sintaxe necessários (parêntesis de fechamento, chaves, ponto e vírgulas, e etc.) e ir para a posição na qual você pode continuar digitando.

Por exemplo, se você digitou

public void Foo(string input/*caret*/

você pode pressionar &shortcut:EditorCompleteStatement; para obter

public void Foo(string input) { /*caret*/ }
