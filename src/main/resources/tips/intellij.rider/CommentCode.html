Para comentar ou descomentar uma linha de código, pressione &shortcut:CommentByLineComment;. A linha atual será comentada/descomentada e o cursor se moverá para a próxima linha.

Alternativamente, você pode pressionar &shortcut:CommentByBlockComment; para colocar a seleção dentro de um comentário de bloco ou descomentar todo o comentário de bloco se o cursor estiver dentro dele.
