Mover sentenças para cima e para baixo

As ações Código | Mover Sentença para Cima/Baixo são úteis para reorganizar linhas de código, por exemplo para aproximar a declaração de uma variável do uso da variável.

Selecione um fragmento de código e pressione &shortcut:MoveStatementUp; ou &shortcut:MoveStatementDown;.

Quando nada for selecionado no editor, a linha da posição do cursor será movida.
