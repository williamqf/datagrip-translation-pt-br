Pesquisar em todos os lugares

Pressione Shift duas vezes para pesquisar por arquivos, ações, símbolos, elementos de IU, branches do Git e comentários em todo o seu projeto. Pressionar novamente o duplo Shift estenderá a pesquisa para itens fora do projeto.

Use as teclas ou atalhos diretos &shortcut:GotoClass; para classes, &shortcut:GotoFile; para arquivos, &shortcut:GotoSymbol; para símbolos e &shortcut:GotoAction; para ações para refinar os resultados da sua pesquisa.
