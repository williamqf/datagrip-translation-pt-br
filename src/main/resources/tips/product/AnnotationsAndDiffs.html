Veja anotações com Git blame

Anotações Git exibem informações detalhadas sobre a origem de cada linha de código (clique com o botão direito na sarjeta e selecione Anote com Git Blame).

Clique com o botão direito em uma anotação e escolha Exibir Diferença para revisar as diferenças entre a versão atual e a anterior do arquivo.
