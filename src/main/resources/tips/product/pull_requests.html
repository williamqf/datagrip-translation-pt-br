Gerencie requisições pull

Gerencie requisições pull recebidas do GitHub diretamente do IntelliJIDEA: no menu principal selecione Git | GitHub | Ver requisições pull. O IntelliJIDEA permite que você atribua e mescle requisições pull, visualize a linha do tempo e os comentários in-line, envie comentários e revisões, e aceite alterações sem sair do IDE.
