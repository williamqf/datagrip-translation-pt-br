Melhore o código com ações contextuais

Pressione &shortcut:ShowIntentionActions; no editor para corrigir um erro ou aviso destacado, melhorar ou otimizar uma construção de código. Para algumas ações de intenção, você pode abrir uma prévia ou descrição de ação pressionando &shortcut:QuickJavaDoc; (Exibir | Documentação Rápida).
