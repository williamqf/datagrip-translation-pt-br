Adicionar uma célula em um notebook Jupyter

Você pode adicionar uma nova célula seguindo um dos métodos abaixo:

  • Selecione Adicionar célula acima ou Adicionar célula abaixo do menu Célula.

  • Clique em + na barra de ferramentas do notebook.

  • Passe o mouse sobre a borda superior de uma célula para exibir os comandos incorporados para adicionar uma célula de código e uma célula Markdown.
