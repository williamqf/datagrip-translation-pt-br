Monitore o uso de recursos

Se você precisar monitorar métricas como memória, CPU ou uso de threads dentro de seu aplicativo, certifique-se de usar Gráficos ao Vivo de CPU e Memória. Esta ferramenta é útil para entender os requisitos do seu aplicativo e como ele reage a determinados eventos em termos de consumo de recursos.

No menu principal, selecione Exibir | Janelas de ferramenta | Profiler, clique com o botão direito do mouse no processo desejado e selecione Gráficos ao Vivo de CPU e Memória.
