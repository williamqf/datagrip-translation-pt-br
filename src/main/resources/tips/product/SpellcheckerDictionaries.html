Adicione dicionários personalizados para verificar ortografia

Quando algumas palavras estiverem faltando nos dicionários pré-definidos, você pode criar as suas próprias. Um dicionário personalizado é um arquivo de texto .dic contendo cada palavra em uma nova linha.

Para adicionar novos dicionários, abra as configurações pressionando &shortcut:ShowSettings; e navegue até Editor | Linguagens Naturais | Ortografia.
