Visualizar tabela de conteúdo em arquivos Markdown

Você pode pré-visualizar o contorno dos títulos do Markdown na janela de ferramentas Estrutura. Para abri-la, selecione Janela | Janelas de ferramentas | Estrutura no menu principal.

Clique em para navegar do título selecionado no editor para o item correspondente na janela de ferramentas Estrutura.

Da mesma forma, clique em para navegar do item selecionado na estrutura do caderno para a célula Markdown correspondente.
