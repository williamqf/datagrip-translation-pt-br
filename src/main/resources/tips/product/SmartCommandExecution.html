Execute funcionalidades da IDE pelo terminal

Se você vir um comando destacado no terminal, não o execute de imediato. Em vez disso, pressione Control+Enter (para Windows e Linux) / Control+Enter (para macOS) para ativar a funcionalidade correspondente da IDE.

Por exemplo, tente git log para visualizar a aba Log da janela da ferramenta Git ou diff com os nomes dos arquivos para abri-los no visualizador de diff.
