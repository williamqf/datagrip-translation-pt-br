Selecionar sugestões de preenchimento

Quando a opção Mostrar sugestões enquanto você digita está ativada para preenchimento de código, é possível usar as ações Pesquisar para baixo e Pesquisar para cima para fechar a lista de sugestões e mover o cursor para cima ou para baixo no editor.

No Windows, use os atalhos &shortcut:EditorLookupDown; e &shortcut:EditorLookupUp;.

No macOS, os atalhos não são configurados por padrão; configure qualquer combinação de teclas que seja conveniente para você em Configurações | Atribuição de teclas.
