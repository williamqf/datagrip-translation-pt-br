Conclusão de código pós-fixado

A conclusão de código pós-fixado é aplicada da direita para a esquerda para evitar saltos de cursor para trás durante a codificação. Digite um ponto . após o fragmento de código que você deseja alterar e selecione a opção desejada.

Para ver a lista de sugestões com modelos pós-fixados, você também pode pressionar &shortcut:InsertLiveTemplate;.
