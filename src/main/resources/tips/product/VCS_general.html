Controle de Versão

O IntelliJ IDEA proporciona integração pronta para uso com os seguintes sistemas de controle de versão: Git, Mercurial, Subversion, e Perforce.

Versões da sua aplicação para colaboração nela, e também para eliminar os riscos de salvar toda a sua base de código localmente. Pressione &shortcut:ActivateVersionControlToolWindow; para ver o histórico das alterações no projeto, ou pressione &shortcut:CheckinProject; para enviar suas alterações locais para o repositório.
