DDL, Dados e Localização do Database Explorer de Objetos de Banco de Dados

Pressione &shortcut:GotoDeclaration; para abrir a DDL dos objetos selecionados. Por exemplo, pressione este atalho no nome da tabela de uma declaração SELECT para exibir a DDL da tabela (declaração CREATE TABLE).

Pressione &shortcut:EditSource; para abrir os dados do objeto. Por exemplo, pressione este atalho no nome da tabela de uma declaração SELECT para exibir o conteúdo da tabela.

Pressione Alt+Shift+B para navegar para o objeto na janela de ferramentas "Database Explorer ".
