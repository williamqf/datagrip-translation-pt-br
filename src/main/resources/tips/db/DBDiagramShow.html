Diagrama

Você pode criar diagramas de fonte de dados, esquema ou tabela.

Database Explorer (Visualizar | Janela de Ferramentas | Database Explorer) clique com o botão direito do mouse no objeto do banco de dados e selecione "Diagrama | Mostrar Visualização". Para exibir o diagrama em uma janela pop-up, selecione "Diagrama | Mostrar Pop-up de Visualização".
