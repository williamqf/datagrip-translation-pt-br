Diminuir os títulos das abas

Por padrão, a opção "Diminuir os títulos das abas" está habilitada. O nome do conjunto de dados exibido no título da guia é reduzido para 36 caracteres.

Para desabilitar esse recurso, clique com o botão direito do mouse na guia do console de consulta e selecione "Diminuir os títulos das abas".
