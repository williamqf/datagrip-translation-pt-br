Sincronizar o estado do Banco de Dados

Para sincronizar automaticamente o estado do Banco de Dados, clique no botão "Data Source Properties " () no Database Explorer (View | Tool Windows | Database Explorer) e selecione o data source que você deseja modificar. Na guia "Options ", marque o checkbox "Auto-sync " na seção "Introspection ".

Se o checkbox "Auto-sync " estiver desmarcado, a visualização do data source no Database Explorer (View | Tool Windows | Database Explorer) será sincronizada com o estado real do Banco de Dados apenas quando você clicar no ícone "Refresh " ou pressionar &shortcut:Refresh;.
