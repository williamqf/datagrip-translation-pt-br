Mostrar Usos

Exiba uma lista de todos os usos de uma tabela, coluna ou chave em todo o projeto e navegue rapidamente até o item selecionado. Para fazer isso, posicione o cursor no símbolo e pressione &shortcut:ShowUsages; ( Edición | Buscar Usos | Mostrar Usos).

Para navegar para um uso, selecione o uso pretendido na lista e pressione &shortcut:EditorEnter; .
