Consoles de Banco de Dados, Arquivos temporários e buffers temporários

Consoles de Banco de Dados são arquivos SQL pré-atribuídos a um banco de dados. Os consoles de dados são criados automaticamente quando uma fonte de dados é criada. Para abrir um console, clique com o botão direito do mouse na fonte de dados e selecione Navegação | Ir para o Console de Consulta | Abrir Console Padrão. Para criar um novo console, clique com o botão direito do mouse na fonte de dados e selecione Novo | Console de Consulta.

Arquivos Temporários são arquivos SQL que não pertencem a uma fonte de dados. Arquivos temporários são usados como rascunhos ou notas temporárias para ideias de código fora do escopo do projeto. Para criar um arquivo temporário, navegue até Arquivo | Novo | Arquivo Temporário (ou pressione &shortcut:NewScratchFile;).

Buffers Temporários são arquivos de texto simples que não oferecem suporte a recursos de assistência de codificação. Os buffers temporários podem ser usados para listas de tarefas simples ou notas. Não há um item de menu específico para a ação de criar um novo buffer temporário, mas você pode executar a ação Novo Buffer Temporário usando o popup Localizar Ação (&shortcut:GotoAction;).
