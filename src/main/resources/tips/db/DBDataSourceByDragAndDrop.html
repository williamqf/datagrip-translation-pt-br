Arquivo SQL como fonte de dados DDL

Arquivos SQL que contêm sentenças CREATE podem ser usados como fontes de dados. Graças a isso, todos as tabelas, colunas e outros objetos definidos em tais arquivos podem ser referenciados no editor ao criar diagramas.

Para criar uma fonte de dados DDL, arraste o arquivo DDL do gerenciador de arquivos para o Navegador de banco de dados (Exibir | Janelas de ferramentas | Navegador de banco de dados).
