Live Templates

Os modelos ativos permitem inserir construções de código usadas com frequência. Por exemplo, tente digitar sel no editor.

Quando você seleciona um modelo ativo na lista de candidatas, o modelo é expandido para as variáveis definidas na página "Configurações" do diálogo (&shortcut:ShowSettings;) em "Editor | Live Templates".

Você pode usar a tecla Tab para navegar entre os campos do modelo.
