Dumping em Banco de Dados do Data Source DDL

Um banco de dados DDL é um banco de dados virtual com esquema com base em diversos scripts SQL. Você pode armazenar esses arquivos no sistema de controle de versão (VCS) para gerenciar os bancos de dados no VCS.

Para criar um banco de dados DDL, clique com o botão direito do mouse no banco de dados em "Database Explorer" e selecione "Importar/Exportar | Dumping em DDL".
