Exibição de vários arquivos lado a lado

No editor, é possível exibir vários arquivos lado a lado. Para isso, clique com o botão direito do mouse na guia do editor desejada e selecione como dividir o editor (por exemplo, "Split Vertically" ou "Split Horizontally").

Outra opção é habilitar o modo de divisão de tela arrastando uma guia para qualquer área do editor.
