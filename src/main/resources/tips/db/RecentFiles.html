Exibindo arquivos usados recentemente

Caso tenha fechado acidentalmente uma aba como Query Console ou SQL Script, pressione &shortcut:RecentFiles;. O diálogo "Arquivos recentes" exibirá uma lista dos arquivos editados recentemente.

&shortcut:RecentFiles; (Exibir | Arquivos usados recentemente) exibirá uma lista dos arquivos abertos recentemente.
