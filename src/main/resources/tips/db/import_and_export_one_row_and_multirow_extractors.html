Extratores de Dados para Inserção de Linha Única e Múltiplas Linhas no SQL

O extrator One-Row copia as colunas em uma string delimitada por vírgulas. Ele pode ser usado para colar valores múltiplos em uma cláusula IN de forma agrupada.

O extrator SQL-Insert-Multirow gera uma única instrução INSERT para inserir novas linhas múltiplas.
