Pesquisar em qualquer lugar

Pressione Shift 2 vezes para pesquisar em todo o projeto por arquivos, ações, símbolos, elementos da interface do usuário, branches do Git e comentários. Pressione Shift novamente 2 vezes para pesquisar por itens que não estão no projeto.

Para refinar os resultados da pesquisa, use as abas ou utilize diretamente o atalho necessário (para objetos de banco de dados, &shortcut:GotoClass; ; para arquivos &shortcut:GotoFile; ; para símbolos, &shortcut:GotoSymbol; ; e para ações, &shortcut:GotoAction;).
