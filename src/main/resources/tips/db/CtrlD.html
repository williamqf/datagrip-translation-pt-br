Duplicar bloco ou linha de código

Pressione &shortcut:EditorDuplicate; no editor para duplicar o bloco de código selecionado ou, se nenhum bloco estiver selecionado, para duplicar a linha atual.

Se quiser adicionar uma nova linha entre a duplicata e o texto original, pressione &shortcut:GotoAction; e digite "Duplicate Entire Lines". Um atalho pode ser atribuído a essa ação no menu do item "Mapeamento de teclas" &shortcut:ShowSettings;.
