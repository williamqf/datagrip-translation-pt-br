Edição simultânea de várias linhas

&shortcut:EditorToggleColumnMode; pressione para alternar o modo Seleção de colunas. Este modo permite que você faça as mesmas alterações em várias linhas. Por exemplo, é útil quando você precisa exibir um texto como uma lista e quebrar várias linhas dentro da mesma tag. Use o mesmo atalho para desativar esse modo.
