Destacando usos do arquivo

Pressione &shortcut:HighlightUsagesInFile; ( Editar | Encontrar usos | Destacar usos no arquivo ) para destacar rapidamente os usos de uma variável específica no arquivo atual.

Pressione &shortcut:FindNext; e &shortcut:FindPrevious; para navegar pelos usos destacados. Pressione &shortcut:EditorEscape; para limpar os destaques.
