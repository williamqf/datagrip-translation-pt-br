Adicionando um dicionário personalizado de verificação ortográfica

Se houver palavras faltando nos dicionários pré-definidos, você pode criar seu próprio. Dicionários personalizados são arquivos de texto .dic com uma palavra em cada linha separada por quebra de linha.

Para adicionar um novo dicionário, pressione &shortcut:ShowSettings; para abrir as configurações e acesse "Editor | Linguagem natural | Ortografia".
