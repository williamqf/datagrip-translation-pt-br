Gerenciamento de pull requests

Você pode gerenciar pull requests do GitHub recebidas diretamente no IntelliJ IDEA. Para isso, selecione "Git | GitHub | Exibir pull requests" no menu principal. Com o IntelliJ IDEA, você pode atribuir e mesclar pull requests, exibir linhas do tempo e comentários in-line, enviar comentários e revisões, e aprovar alterações sem sair do IDE.
