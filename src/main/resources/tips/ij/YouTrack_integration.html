Integração da IDE com o YouTrack

Você poderá gerenciar projetos de forma eficiente usando o YouTrack. Se você possui uma instância do YouTrack, é possível usar o plugin YouTrack Integration para operar tarefas diretamente no IntelliJIDEA.

Equipes com até 10 membros podem usar o YouTrack de graça por tempo indeterminado. Experimente o YouTrack.
