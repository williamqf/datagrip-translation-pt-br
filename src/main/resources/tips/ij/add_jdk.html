Adicionar JDK ao projeto

É possível adicionar a versão e o fornecedor necessários do JDK diretamente do IDE. Vá para "Arquivo | Estrutura do projeto | Projeto", expanda a lista "SDK" e selecione "Baixar JDK". Essa opção também está disponível para um novo projeto no assistente "Novo projeto" (Arquivo | Novo | Projeto).
