Mover para Declaração

Para mover de um ponto de uso para a declaração de uma classe, método ou variável, posicione o cursor no símbolo e pressione &shortcut:GotoDeclaration; (Navegação | Declaração ou Uso). Você também pode pressionar a tecla Control (Windows e Linux) / Command (macOS) enquanto clica no ponto de uso para mover para a declaração.
