Adicionando Avisos de Copyright

O IntelliJ IDEA permite configurar um aviso de copyright diferente para cada conjunto de arquivos. É possível usar variáveis para maior comodidade. Por exemplo, $today.year manterá o ano atualizado.

No diálogo "Configurações" (&shortcut:ShowSettings;), selecione "Editor | Direitos Autorais | Perfil de Direitos Autorais" para criar um novo perfil ou adicionar texto de direitos autorais usando variáveis.
