Exibir anotações de Git Blame

As anotações do Git fornecem informações detalhadas sobre a origem de cada linha de código (clique com o botão direito na calha e selecione "Anotar com Git Blame").

Você pode revisar as diferenças entre a versão atual do arquivo e uma versão anterior clicando com o botão direito na anotação e selecionando "Exibir diferença".
