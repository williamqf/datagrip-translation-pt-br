Busca em todos os lugares

Pressione Shift duas vezes para pesquisar em todo o projeto por arquivos, ações, símbolos, elementos da IU, branches Git e comentários. Pressione Shift duas vezes novamente para pesquisar também por itens que não estão no projeto.

Use as guias para refinar os resultados da busca ou use diretamente o atalho necessário (para classes, &shortcut:GotoClass; para arquivos, &shortcut:GotoFile; para símbolos, &shortcut:GotoSymbol; para ações, &shortcut:GotoAction;).
