Criar Diagrama de Diferença

Se o projeto estiver sob controle de versão, é possível criar um diagrama UML que reflita as alterações locais e visualize as relações entre os componentes alterados.

Pressione &shortcut:UML.ShowChanges; e selecione a lista de alterações necessária para criar o diagrama. Clique duas vezes nos nós do diagrama para verificar as alterações na janela de diferença.
