Melhore o código usando ações contextuais

No editor, pressione &shortcut:ShowIntentionActions; para corrigir erros e alertas destacados, melhorar a estrutura de código ou otimizá-lo. Para algumas ações contextuais, você pode pressionar &shortcut:QuickJavaDoc; (Exibir | Documento rápido) para abrir uma prévia ou uma descrição da ação.
