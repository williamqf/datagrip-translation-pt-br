Expandindo a seleção de código

Para expandir uma seleção, pressione &shortcut:EditorSelectWord; . Cada vez que &shortcut:EditorSelectWord; é pressionado, a seleção é expandida para outros escopos do código.

Por exemplo, se o nome de um método for selecionado, é possível expandir a seleção para a expressão que chama o método, o próprio statement, o bloco que o inclui, etc.
