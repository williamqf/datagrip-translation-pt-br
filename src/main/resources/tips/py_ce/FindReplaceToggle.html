Pesquisar e Substituir

Ctrl + F exibe uma barra de pesquisa. Pressione Ctrl + H para adicionar outro campo onde você pode digitar a string de substituição.

Você pode alternar para a substituição no diálogo Find in Path pressionando Ctrl + Alt + Shift + R. De forma similar, você pode pressionar Ctrl + Alt + Shift + F para ocultar o campo Substituir e alternar para a pesquisa regular.
