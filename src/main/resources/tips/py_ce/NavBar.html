Barra de Navegação

A Barra de Navegação é um substituto para a ferramenta "Projeto". Use-a para verificar a estrutura do projeto, abrir um arquivo específico ou navegar para um elemento de código específico no arquivo atual.

Pressione Ctrl + F12 para habilitar a Barra de Navegação. Use as setas do teclado para navegar pelos arquivos e pastas.
