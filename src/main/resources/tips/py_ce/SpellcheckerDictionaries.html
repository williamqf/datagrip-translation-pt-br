Adicionando um Dicionário de Verificação Ortográfica Personalizado

Se há uma palavra ausente no dicionário pré-definido, você pode criar seu próprio dicionário. Um dicionário personalizado é um arquivo texto .dic, no qual cada palavra é registrada separada por uma quebra de linha.

Para adicionar um novo dicionário, pressione &atalho:ShowSettings; para abrir as configurações e vá para "Editor | Linguagen Natural | Ortografia".
