Visualize anotações do Git Blame

As anotações do Git exibem informações detalhadas sobre a origem de cada linha de código (clique com o botão direito do mouse na sarjeta e selecione "Anotar com Git Blame").

Clique com o botão direito do mouse nas anotações e selecione "Mostrar diff" para verificar as diferenças entre as versões atuais e anteriores do arquivo.
