Visualizar estrutura de arquivos

Você pode usar &shortcut:FileStructurePopup; (Navegar | Estrutura de arquivos) para navegar rapidamente pelo arquivo atual.

A estrutura de arquivos lista os membros da classe atual. Para navegar até um elemento, use &shortcut:EditorEnter; ou &shortcut:EditSource; após selecionar o elemento.

Para localizar facilmente um item na lista, comece a digitar o nome do item.
