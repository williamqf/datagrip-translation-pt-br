Envolva Blocos de Código

Para envolver rapidamente um bloco de código com um construtor válido, selecione o bloco de código alvo no editor e pressione &shortcut:SurroundWith; (Código | Envolver).

A lista de opções ou encapsuladores disponíveis varia de acordo com o contexto e idioma. Por exemplo, você pode executar operações como envolver blocos html em tags.
