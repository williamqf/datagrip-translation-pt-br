Destacar ocorrências no arquivo

Pressione &shortcut:HighlightUsagesInFile; (menu Editar | Encontrar usos | Destacar ocorrências do arquivo) para destacar rapidamente ocorrências de uma variável específica no arquivo atual.

Pressione &shortcut:FindNext; e &shortcut:FindPrevious; para mover entre as ocorrências destacadas. Pressione &shortcut:EditorEscape; para remover o destaque.
