Pesquisa em qualquer lugar

Pressione Shift duas vezes para pesquisar por arquivos, ações, símbolos, elementos de IU, branches do Git e comentários em todo o projeto. Pressione Shift duas vezes novamente para também pesquisar itens que não estão no projeto.

Use as guias ou diretamente os atalhos necessários (para classes, &shortcut:GotoClass; para arquivos, &shortcut:GotoFile; para símbolos, &shortcut:GotoSymbol; para ações, &shortcut:GotoAction;) para restringir os resultados da pesquisa.
