Verificação ortográfica e gramatical

A verificação ortográfica e gramatical é habilitada por padrão. Essas verificações são fornecidas pelo plugin Grazie que é integrado ao IDE. Para configurar as opções de correção, pressione &shortcut:ShowSettings; e navegue até "Editor | Linguagens naturais | Gramática".

Todo o texto, incluindo comentários, strings do corpo e literais, é inspecionado usando dicionários predefinidos e/ou personalizados. Os erros de digitação são destacados permitindo que você selecione a palavra correta, aceite a grafia atual ou desabilite a inspeção.

Para corrigir erros de ortografia ou gramática, pressione &shortcut:ShowIntentionActions; para exibir uma lista de ações disponíveis.
