Em PyCharmCommunity, é possível configurar o interpretador Python em diferentes estágios do desenvolvimento, da seguinte forma:

  • Durante a criação do projeto ("Arquivo | Novo projeto", ou no menu "Novo projeto" natela de boas-vindas).

  • Se o projeto já existe, use owidget "Python Interpreter" nabarra de status ou nasconfigurações do projeto, selecionando "Projeto | Python Interpreter".
