Gerenciamento de Pull Request

Você pode gerenciar pull requests recebidos do GitHub diretamente no PyCharm Community. Para fazer isso, no menu principal, selecione Git | GitHub | Mostrar Pull Requests. Com o PyCharm Community, você pode atribuir e mesclar pull requests, visualizar o cronograma e os comentários incorporados, enviar comentários e revisões e aprovar alterações diretamente no IDE.
