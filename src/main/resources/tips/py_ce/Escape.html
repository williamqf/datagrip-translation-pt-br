Alterar o foco na janela de ferramentas

Pressione &shortcut:EditorEscape; em uma janela de ferramentas para mover o foco para o editor.

Pressione &shortcut:HideActiveWindow; para mover o foco para o editor e ocultar a janela de ferramentas atualmente ativa ou a última que esteve ativa.

Pressione &shortcut:JumpToLastWindow; para mover o foco do editor para a última janela de ferramentas na qual o foco estava.
