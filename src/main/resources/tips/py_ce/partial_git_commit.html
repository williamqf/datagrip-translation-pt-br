Commit parcial de alterações

Para dividir alterações feitas no mesmo arquivo entre commits diferentes, clique em "Diff" na janela de ferramentas "Commit" &shortcut:ActivateCommitToolWindow;. Marque as caixas de seleção ao lado de cada parte do código modificado ou novo que você deseja fazer commit e pressione "Commit". As alterações não selecionadas permanecerão na lista atualizada de alterações.
