Você pode personalizar a estrutura das instruções import contidas no código. É possível dividir as importações e agrupá-las por origem. Você também pode deixá-las conforme estão, sem qualquer estrutura.

Para selecionar, acesse "Configurações" (&shortcut:ShowSettings;), selecione "Editor | Estilo do código | Python" e marque a caixa de seleção correspondente em "Estrutura de instruções import "from"".
