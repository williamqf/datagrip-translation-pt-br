Adicionar cabeçalho de direitos autorais

O PyCharmCommunity pode configurar cabeçalhos de direitos autorais diferentes para cada conjunto de arquivos. Você pode usar variáveis para facilitar seu uso. Por exemplo, $today.year sempre mantém o ano atual.

Na caixa de diálogo "Configurações" (&shortcut:ShowSettings;), escolha "Editor | Direitos autorais | Perfil de direitos autorais" para criar um novo perfil ou adicionar texto de cabeçalho de direitos autorais usando variáveis.
