Modo lente

Para pré-visualizar o código sem rolá-lo para sua posição real, passe o ponteiro do mouse sobre uma parte do código fonte que tenha um aviso, uma faixa de erro ou uma barra de rolagem. Uma lente será exibida:

Para desativar as lentes, clique com o botão direito do mouse em uma faixa de erro na barra de rolagem e desmarque a caixa de seleção "Exibir lentes de código na posição do mouse na barra de rolagem ".
