Mover declarações para cima/para baixo

A ação "Código | Mover declaração para cima/para baixo" é útil para reorganizar linhas de código, como aproximar uma declaração de variável a um local em que a variável é utilizada.

Selecione um fragmento de código e pressione &shortcut:MoveStatementUp; ou &shortcut:MoveStatementDown;.

Caso não haja nenhuma seleção no editor, a linha na posição do cursor será movida.
