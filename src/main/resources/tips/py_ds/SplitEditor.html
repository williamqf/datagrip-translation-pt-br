Exibir arquivos lado a lado

No editor, é possível exibir vários arquivos lado a lado. Para isso, clique com o botão direito na guia do editor desejado e selecione como o editor deve ser dividido ("Dividir à direita" ou "Dividir abaixo").

Ou, arraste a guia para qualquer lugar do editor para habilitar o modo de tela dividida.
