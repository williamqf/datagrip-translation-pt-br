Gestão de Versão

O DataSpell integra-se prontamente aos sistemas de gestão de versão Git, Mercurial, Subversion e Perforce.

A gestão de versão da aplicação não só permite a colaboração como também elimina o risco de armazenamento local de toda a base de código. Pressionando &shortcut:ActivateVersionControlToolWindow; é exibido o histórico de alterações do projeto e pressionando &shortcut:CheckinProject; as alterações locais são enviadas para o repositório.
