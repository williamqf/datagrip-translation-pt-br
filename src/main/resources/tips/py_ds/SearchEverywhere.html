Pesquisa em qualquer lugar

Pressione Shift duas vezes para pesquisar arquivos, ações, símbolos, elementos da IU, ramificações Git e comentários em todo o projeto. Pressione Shift duas vezes novamente para pesquisar também itens não incluídos no projeto.

Use as abas para filtrar os resultados da pesquisa ou use diretamente os atalhos necessários (para classes, &shortcut:GotoClass; para arquivos, &shortcut:GotoFile; para símbolos, &shortcut:GotoSymbol; para ações, &shortcut:GotoAction;).
