Uso de cores para fontes de dados

É possível usar cores para diferenciar o banco de dados de seus elementos. Clique com o botão direito do mouse no elemento e selecione "Ferramentas | Definir cor" para definir uma cor para a fonte de dados ou seu objeto. na janela de ferramentas "Banco de dados" (Janela | Janelas de ferramentas | Banco de dados).
