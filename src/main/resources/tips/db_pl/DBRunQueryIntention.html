Manipulação de arquivos SQL

Para abrir um arquivo SQL no IDE, clique duas vezes sobre ele. Para executar uma consulta a partir deste arquivo, invoque a ação de intenção (Option+Enter no macOS, Alt+Enter no Windows e no Linux) e selecione "Executar consulta no console". Na lista "Sessões", selecione um console existente ou crie um novo.

Esteja ciente de que abrir um novo console de consulta origina uma nova conexão com a fonte de dados.
