Abra a tabela no editor de dados

Ao clicar duas vezes em uma tabela na janela de ferramentas Banco de Dados (Janela | Janela de Ferramentas | Banco de Dados), a tabela será aberta no editor de dados.

No editor de dados, você pode criar e excluir novas entradas, alterar entradas existentes, filtrar dados por colunas específicas, exibir dados em um visualizador separado e comparar duas tabelas.
