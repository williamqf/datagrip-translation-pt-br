Se você tiver classes Scala no seu projeto, você pode extrair métodos Spark delas para seu caderno Zeppelin.

  1. No caderno do Zeppelin, selecione os fragmentos de código do método Spark.
  2. Clique com o botão direito do mouse no código selecionado e, no menu de contexto, selecione Refatorar | Extrair para método Spark .
  3. Na caixa de diálogo aberta, especifique a classe Scala (você pode selecioná-la na árvore do projeto).
  4. Declare o método e clique em Refatorar . O código do método selecionado será adicionado à classe Scala de destino.
