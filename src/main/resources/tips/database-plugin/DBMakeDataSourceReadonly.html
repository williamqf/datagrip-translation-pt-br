Habilitar modo somente-leitura para conexões

Para proteger seu banco de dados contra modificações acidentais, você pode ativar o modo somente leitura para uma conexão.

Para habilitar o modo somente leitura para uma conexão, clique em Arquivo | Fontes de dados e selecione a fonte de dados necessária na lista Fontes de Dados. Na guia Opções, marque a caixa de seleção Somente leitura.
