Visualizador de diferenças para objetos do banco de dados

Seu IDE tem um visualizador de diferenças para objetos do banco de dados. Por exemplo, na janela de ferramentas Database, você pode selecionar duas tabelas, clicar com o botão direito do mouse na seleção e escolher Comparar. A diferença entre estas tabelas estará disponível na caixa de diálogo separada do visualizador de diferenças.
