Trabalhando com arquivos SQL

Clique duas vezes em um arquivo SQL para abri-lo na IDE. Para executar uma consulta a partir deste arquivo, chame ações de intenção (Option+Enter para macOS, Alt+Enter para Windows e Linux) e selecione Executar consulta no console. Na lista Sessões, selecione um console existente ou crie um novo.

Observe que um novo console de consulta significa uma nova conexão com uma fonte de dados.
