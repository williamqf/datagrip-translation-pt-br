Esqueça os schemas em cache e sincronize a fonte de dados novamente

Se você não vê objetos abaixo do nível do schema, não consegue encontrar alterações em objetos, experimenta tabelas quebradas ou qualquer outro problema de visualização, tente limpar o cache e sincronizar a fonte de dados novamente.

Para limpar o cache, clique com o botão direito em uma fonte de dados e navegue até Diagnóstico | Esquecer Schemas em Cache.
