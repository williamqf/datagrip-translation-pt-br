Para executar as funções do IDE no terminal

Se um comando no terminal for destacado, não o execute imediatamente. Pressione Control+Enter (Windows e Linux)/Control+Enter (macOS) para ativar as funções do IDE correspondentes.

Por exemplo, tente git log para exibir a guia Log na janela de ferramentas Git ou execute "Diferença" para abrir vários nomes de arquivo no visualizador de diferenças.
