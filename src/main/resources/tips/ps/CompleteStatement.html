Criação de construções de código por autocompletar

É possível criar construções de código usando o recurso Autocompletar Declaração. Comece a digitar uma declaração de método, uma chamada de método ou uma declaração como if, do-while, try-catch ou return. Pressione &shortcut:EditorCompleteStatement; para que a declaração seja autocompletada, tornando-se uma construção com sintaxe correta.
