Destacar ocorrências dentro do arquivo

&shortcut:HighlightUsagesInFile; (Editar | Encontrar Usos | Destacar Usos no Arquivo) pressiona rapidamente para destacar as ocorrências de uso de uma variável específica contida no arquivo atual.

&shortcut:FindNext; e &shortcut:FindPrevious; pressiona para navegar até as ocorrências destacadas. &shortcut:EditorEscape; pressiona para remover os destaques.
