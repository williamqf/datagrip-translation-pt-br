Barra de navegação

A barra de navegação substitui a janela de ferramentas "Projeto". Nela, você pode conferir a estrutura do projeto, abrir arquivos específicos e ir para elementos de código específicos presentes no arquivo atual.

Pressione &shortcut:ShowNavBar; para exibir a barra de navegação; em seguida, use as teclas de direção para alternar entre arquivos e pastas.
