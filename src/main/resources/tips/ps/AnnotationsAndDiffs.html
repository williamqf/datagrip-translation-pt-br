Exibir anotações do Git Blame

As anotações do Git mostram informações detalhadas sobre a origem de cada linha do código (clique com o botão direito na sarjeta e selecione "Anote no Git Blame").

Você pode clicar com o botão direito em uma anotação e selecionar "Mostrar diferença" para ver a diferença entre a versão atual e a versão anterior do arquivo.
