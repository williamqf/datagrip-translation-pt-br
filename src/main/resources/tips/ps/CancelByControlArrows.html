Seleção de candidatos para conclusão

Quando a opção Exibir candidatos durante a digitação está habilitada na conclusão de código, as ações Busca para cima e Busca para baixo podem ser usadas para fechar a lista de candidatos e mover o cursor para cima ou para baixo no editor.

No Windows, use os atalhos &shortcut:EditorLookupDown; e &shortcut:EditorLookupUp;.

No macOS, os atalhos não são configurados por padrão, logo, você deve atribuir uma combinação de teclas de acordo com sua preferência em Configurações | Mapeamento de teclas.
