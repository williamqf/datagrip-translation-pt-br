Configurações de Cor nos Arquivos CSS

No PHPStorm, é possível definir cores nos arquivos CSS de forma fácil. Na Lateral do Editor, há um ícone de cor correspondente à propriedade color . Ao clicar neste ícone, é possível selecionar a cor desejada usando o Seletor de Cores.
