Encapsulamento de locais de código

Para encapsular rapidamente um bloco de código em um constructo válido, selecione o bloco de código no editor e pressione &shortcut:SurroundWith; (Código | Encapsular).

A lista de opções ou delimitadores disponíveis varia de acordo com o contexto e a linguagem. Por exemplo, é possível realizar uma operação como encapsular um bloco html com tags.
