Preenchimento de Código Pós-fixo

O preenchimento de código pós-fixo é aplicado da direita para a esquerda, evitando que o cursor se mova para trás durante a programação. Após o fragmento de código a ser modificado insira um ponto . e selecione a opção desejada.

Também é possível utilizar o atalho &shortcut:InsertLiveTemplate; para que uma lista de sugestões contendo templates pós-fixos seja exibida.
