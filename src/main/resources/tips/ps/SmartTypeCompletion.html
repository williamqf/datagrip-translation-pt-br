Conclusão de Código de Correspondência de Tipo

A conclusão de código de "correspondência de tipo" analisa o tipo previsto de uma expressão inteira e ajuda a encontrar métodos e variáveis aplicáveis no contexto atual. Esta conclusão de código funciona após a palavra-chave return, atribuições, listas de argumentos de chamadas de método, etc. O atalho &shortcut:SmartTypeCompletion; (ou Código | Conclusão de Código | Correspondência de Tipo) pode ser utilizado para filtrar a lista de conclusão.
