Controle de solicitações pull

É possível controlar as solicitações pull recebidas do GitHub diretamente a partir do PhpStorm. Para isso, selecione " Git | GitHub | Exibir solicitações pull " no menu principal. Ao utilizar o PhpStorm, é possível atribuir e mesclar solicitações pull, visualizar linhas do tempo e comentários em linha, enviar comentários e revisões e aprovar alterações sem sair do IDE.
