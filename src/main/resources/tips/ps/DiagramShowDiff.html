Contruindo Diagrama de Diferenças

Se o projeto está sob controle de versão, você pode criar um diagrama UML que reflita as alterações locais e visualize os relacionamentos entre os componentes alterados.

&shortcut:UML.ShowChanges; pressionando para selecionar as alterações que você deseja no diagrama. Faça duplo clique no nó de diagrama verifique as alterações no diálogo de diferenças.
