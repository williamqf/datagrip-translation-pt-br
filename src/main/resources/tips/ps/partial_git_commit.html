Commit parcial de alterações

Para dividir as alterações feitas no mesmo arquivo entre diferentes commits, clique em "Diferença" na janela de ferramentas "Commit"&shortcut:ActivateCommitToolWindow;. Selecione as caixas de seleção ao lado de cada pedaço de código alterado ou novo que deseja fazer commit e clique em "Commit". As alterações não selecionadas permanecerão na lista de alterações atual.
