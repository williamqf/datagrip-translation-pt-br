Mover declaração para cima/baixo

A ação "Código | Mover declaração para cima/baixo " é útil para reorganizar linhas de código, como aproximar uma declaração de variável a um uso desta variável.

Selecione o fragmento de código e pressione &shortcut:MoveStatementUp; ou &shortcut:MoveStatementDown; .

Se nenhuma seleção for feita no editor, a linha na posição do cursor será movida.
