Expandir Seleção de Códigos

Para expandir a seleção, pressione &shortcut:EditorSelectWord;. A cada vez que você pressionar &shortcut:EditorSelectWord;, a seleção será expandida para outras áreas do código.

Por exemplo, se um nome de método estiver selecionado, você poderá expandir a seleção para a expressão que chama o método, para todo o statement ou para o bloco includente.
