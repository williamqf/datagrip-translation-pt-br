Navegue de um comentário até uma issue

Você pode navegar diretamente de um comentário para uma issue do seu rastreador de issues. Para fazer isso, defina os padrões de Issue Navigation em "Configurações | Controle de Versão | Navegação de Issues ".

Ao confirmar uma lista de alterações, digite uma mensagem de confirmação que corresponda a um dos padrões de Issue Navigation. Mais tarde, ao visualizar a lista de alterações em seu repositório, os comentários qualificados serão transformados em hiperlinks.

Clicar no hiperlink navegará diretamente para a issue no seu rastreador de issues.
