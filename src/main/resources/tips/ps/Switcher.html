Navegação Switcher

Para navegar entre os arquivos e janelas de ferramentas abertas, use o Switcher &shortcut:Switcher;. Para manter o Switcher aberto, continue pressionando Control. Para navegar, use Tab (avançar) e Shift+Tab (voltar).

Para fechar uma guia de edição ou ocultar uma janela de ferramentas, pressione Delete ou BackSpace.
