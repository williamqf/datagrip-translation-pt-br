Adicionar um dicionário de verificação ortográfica personalizado

Caso existam palavras faltantes no dicionário predefinido, você pode criar as suas próprias palavras. Um dicionário personalizado é um arquivo de texto .dic com cada palavra registrada em uma nova linha.

Para adicionar um novo dicionário, pressione &shortcut:ShowSettings; para abrir as configurações e vá até "Editor | Linguagem natural | Ortografia".
