Documentação externa

Para abrir a página de documentação do elemento na posição do cursor no navegador, pressione &shortcut:ExternalJavaDoc; ou selecione "Ferramentas | Navegador da web e visualização" no menu principal.

Verifique se o navegador padrão está configurado. Para verificar o navegador padrão, pressione &shortcut:ShowSettings; para abrir as configurações e vá para "Ferramentas | Navegador da web e visualização".
