Mostrar estrutura de arquivos

&shortcut:FileStructurePopup; (mover | estrutura de arquivos) permite navegar rapidamente no arquivo atual.

A estrutura do arquivo lista os membros da classe atual. Para mover para um elemento, selecione-o e pressione &shortcut:EditorEnter; ou &shortcut:EditSource;.

Para encontrar facilmente itens em uma lista, comece a digitar o nome do item.
