Pesquisar em qualquer lugar

Pressione Shift duas vezes para pesquisar arquivos, ações, símbolos, elementos da IU, branches do Git e comentários em todo o projeto. Pressione Shift duas vezes novamente para pesquisar também em itens que não estão incluídos no projeto.

Para refinar os resultados da pesquisa, use as guias ou digite diretamente o atalho necessário (para classes, &shortcut:GotoClass; para arquivos, &shortcut:GotoFile; para símbolos, &shortcut:GotoSymbol; para ações, &shortcut:GotoAction;).
