Dica de URL Path inlay

Clicar na dica de inlay acima de cada ação do controlador abre uma lista de ações que podem ser usadas na URL.

  • Vá para declaração ou usos: Vai para a declaração no arquivo routes.rb.

  • Gera requisição em cliente HTTP: Cria uma nova requisição em um cliente HTTP.

  • Mostra todos os endpoints do módulo: Mostra uma visão agregada da API do módulo na janela de ferramenta Endpoints.
