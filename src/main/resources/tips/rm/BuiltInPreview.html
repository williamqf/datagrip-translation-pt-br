Prévia integrada

A Visualização incorporada do RubyMine exibe a saída do código HTML em uma guia de edição independente. O RubyMine salva alterações feitas no arquivo HTML ou em arquivos CSS e JavaScript vinculados, e recarrega a visualização automaticamente.

Para abrir a visualização, posicione o cursor no código HTML e clique no ícone do RubyMine na janela pop-up do navegador.
