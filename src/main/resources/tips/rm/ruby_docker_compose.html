Docker Compose como interpretador remoto

Para configurar o Docker Compose como interpretador remoto, abra o diálogo de Configurações &shortcut:ShowSettings;, navegue até a página Idioma e estrutura | SDK e Gem do Ruby. Clique no botão e selecione Novo remoto. Na caixa de diálogo acionada, selecione Docker Compose e especifique as configurações necessárias.
