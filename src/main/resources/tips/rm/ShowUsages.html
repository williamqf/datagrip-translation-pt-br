Exibição dos locais de uso

Exibe uma lista com todos os locais de uso de uma classe, método ou variável no projeto inteiro, permitindo que você navegue rapidamente até o item selecionado. Para isso, posicione o cursor no símbolo e pressione &shortcut:ShowUsages; (Editar | Pesquisar locais de uso | Exibir locais de uso).

Para navegar até um local de uso, selecione o local de uso desejado na lista e pressione &shortcut:EditorEnter;.
