Debug usando uma configuração Rails predefinida

Para os projetos Rails, o RubyMine cria uma configuração de execução/depuração para o development automaticamente, usando o ambiente Rails correspondente. Para iniciar a depuração do seu aplicativo usando a configuração de execução/depuração do development, pressione Control duas vezes para exibir o popup "Executar qualquer coisa". Depois, comece a digitar development e pressione Shift após selecioná-la na lista (o título do diálogo mudará para "Debug"). Por fim, pressione &shortcut:EditorEnter;.
