Compartilhamento de configurações de execução/depuração

As configurações de execução/depuração podem ser salvas como arquivo XML e compartilhadas com os colegas de equipe usando um sistema de controle de versão. Ao editar a configuração de execução/depuração desejada, ative a opção "Salvar como arquivo de projeto" e especifique onde o arquivo de configuração deverá ser salvo.
