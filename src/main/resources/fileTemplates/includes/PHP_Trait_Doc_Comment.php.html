Este é um modelo padrão para comentários PHPDocs a serem usados com traits PHP. Ele inclui um trecho de código que pode ser incluído em um template de arquivo (guia Templates) usando a diretiva #parse.
O modelo é editável. Além de texto estático, código e comentários, você pode usar variáveis predefinidas que se expandirão para seus valores correspondentes assim como macros.
As variáveis predefinidas assumem os seguintes valores:
${NAME} O nome do trait.
${NAMESPACE} O nome totalmente qualificado da classe trait, sem uma barra inicial.
${DS} O cifrão de dólar é avaliado como o caractere '$' comum.
${DATE} A data atual do sistema
${TIME} A hora atual do sistema
${YEAR} O ano atual
${MONTH} O mês atual
${MONTH_NAME_SHORT} As três primeiras letras do nome do mês atual. Ex: Jan, Fev, etc.
${MONTH_NAME_FULL} O nome completo do mês atual. Ex: Janeiro, Fevereiro, etc.
${DAY} O dia do mês atual
${HOUR} A hora atual
${MINUTE} O minuto atual
${SECOND} O segundo atual
${PROJECT_NAME} O nome do projeto atual
${USER} O usuário atual
${CARET} Armazena a posição para a qual o cursor deve ser movido após inserir um comentário.
