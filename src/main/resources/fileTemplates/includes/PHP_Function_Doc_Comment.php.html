Modelo embutido para comentários PHPDoc para funções de PHP. Ele contém um trecho de código que pode ser incluído em um template de arquivo (guia Templates) utilizando a diretiva #parse.
O template é editável. Além do texto estático, código e comentários, você pode usar variáveis predefinidas que serão expandidas para os valores correspondentes, como macros.
Variáveis predefinidas recebem os seguintes valores:
${NAME} Nome da função.
${TYPE_HINT} Dica de tipo de retorno da função. Se o tipo de retorno não for inferido, será avaliado como 'void'.
${PARAM_DOC}

Comentário PHPDoc do parâmetro.

Gerado como uma série de linhas '* @param tipo nome'. Avalia como string vazia se não houver parâmetros.

${THROWS_DOC}

comentário doc de exceção.

Gerado como uma série de linhas '* @throws tipo'. Avalia como string vazia se nenhuma exceção for lançada.

${STATIC} Assume o valor 'static' se a função/método for estático, ou uma string vazia caso contrário. Por exemplo: 
      #if (${STATIC} == "static") * @static
      #end
${DS} Símbolo do dólar, avaliado como o caractere '$' comum.
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${MONTH_NAME_SHORT} As três primeiras letras do nome do mês atual. Ex: Jan, Fev, etc.
${MONTH_NAME_FULL} Nome completo do mês atual. Ex: Janeiro, Fevereiro, etc.
${DAY} Dia do mês atual
${HOUR} Hora atual
${MINUTE} Minuto atual
${SECOND} Segundo atual
${PROJECT_NAME} Nome do projeto atual
${USER} Usuário atual
${CARET} Armazena a posição para onde o cursor deve
