Um modelo embutido para cabeçalhos em arquivos PHP. Ele contém um trecho de código que pode ser incluído em templates de arquivo (guia "Templates") com o auxílio da diretiva #parse.
O template é editável. Além de textos estáticos, códigos e comentários, você pode utilizar também variáveis pré-definidas, que serão expandidas para o valor correspondente, como uma macro.
As variáveis pré-definidas assumem os seguintes valores:
${FILE_NAME} O nome do arquivo atual
${USER} O nome de login do usuário atual no sistema
${DATE} A data atual do sistema
${TIME} A hora atual do sistema
${YEAR} O ano atual
${MONTH} O mês atual
${MONTH_NAME_SHORT} Os três primeiros caracteres do nome do mês atual. Ex.: Jan, Fev, etc.
${MONTH_NAME_FULL} O nome completo do mês atual. Ex.: Janeiro, Fevereiro, etc.
${DAY} O dia do mês atual
${HOUR} A hora atual
${MINUTE} O minuto atual
${SECOND} O segundo atual
${PRODUCT_NAME} O nome da IDE atual
${PROJECT_NAME} O nome do projeto atual
