Contém um fragmento de código que pode ser incluído em templates de arquivo (aba Templates) com a ajuda da diretiva #parse.
Este template integrado é editável. Além de texto estático, código e comentários, você também pode usar variáveis predefinidas que serão expandidas para os valores correspondentes, como macros.
As variáveis predefinidas aceitam os seguintes valores:
${PACKAGE_NAME}   O nome do pacote no qual o novo arquivo é criado
${USER}   Nome de login do sistema do usuário atual
${DATE}   A data atual do sistema
${TIME}   A hora atual do sistema
${YEAR}   O ano atual
${MONTH}   O mês atual
${MONTH_NAME_SHORT}   As primeiras 3 letras do nome do mês atual. Ex: Jan, Feb, etc.
${MONTH_NAME_FULL}   O nome completo do mês atual. Ex: January, February, etc.
${DAY}   O dia atual do mês
${DAY_NAME_SHORT}   As primeiras 3 letras do nome do dia atual. Ex: Mon, Tue, etc.
${DAY_NAME_FULL}   O nome completo do dia atual. Ex: Monday, Tuesday, etc.
${HOUR}   A hora atual
${MINUTE}   O minuto atual
${PROJECT_NAME}   O nome do projeto atual
