Neste template, você pode usar a diretiva #parse para especificar fragmentos de código a serem incluídos nos templates de arquivo (aba Arquivos). Um template pode ser compartilhado por vários templates de arquivo. Além de instruções RBS, expressões e comentários, você também pode usar variáveis predefinidas, que serão expandidas para seus respectivos valores como macros.
As variáveis predefinidas aceitam os seguintes valores:
${FILE_NAME} Nome do arquivo atual
${USER} Nome de login do sistema do usuário atual
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${DAY} Dia atual do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${PRODUCT_NAME} Nome do IDE atual
${PROJECT_NAME} Nome do projeto atual
