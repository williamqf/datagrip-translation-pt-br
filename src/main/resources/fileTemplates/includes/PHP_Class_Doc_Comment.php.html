Modelo interno para comentários PHPDoc para uso em classes PHP. Contém um snippet de código que pode ser incluído num modelo de ficheiro (guia Modelos) com a ajuda da diretiva #parse.
O modelo é editável. Para além de texto estáticos, códigos e comentários, poderá usar variáveis predefinidas que são expandidas para o valor correspondente, tal como as macros.
As variáveis predefinidas recebem os seguintes valores:
${NAME} Nome da classe.
${NAMESPACE} Nome completamente qualificado do namespace da classe, sem o slash inicial.
${DS} O símbolo do dólar, avalia para o caracter "$" normal.
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${MONTH_NAME_SHORT} As primeiras 3 letras do nome do mês atual. Ex.: Jan, Fev, etc.
${MONTH_NAME_FULL} O nome completo do mês atual. Ex.: Janeiro, Fevereiro, etc.
${DAY} Dia do mês atual
${HOUR} Hora atual
${MINUTE} Minuto atual
${SECOND} Segundo atual
${PROJECT_NAME} Nome do projeto atual
${USER} Usuário atual
${CARET} Armazena a posição para onde o cursor de texto deve ser movido depois que o comentário for adicionado.
