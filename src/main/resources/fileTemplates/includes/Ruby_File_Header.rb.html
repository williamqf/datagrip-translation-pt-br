Neste modelo, você pode especificar um fragmento de código para ser incluído em modelos de arquivo (guia Arquivos) com a ajuda da diretiva #parse. Um modelo pode ser compartilhado por vários modelos de arquivo. Junto com instruções Ruby, expressões e comentários, você também pode usar variáveis predefinidas que serão então expandidas como macros para os valores correspondentes.
Variáveis predefinidas assumem os seguintes valores:
${FILE_NAME} Nome atual do arquivo
${USER} Nome atual de login do usuário do sistema
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${DAY} Dia atual do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${PRODUCT_NAME} Nome atual do IDE
${PROJECT_NAME} Nome atual do projeto
