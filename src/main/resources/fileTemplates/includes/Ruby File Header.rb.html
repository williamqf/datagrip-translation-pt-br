Neste template, você pode especificar fragmentos de código a serem incluídos em templates de arquivo (aba Arquivos) usando a diretiva #parse. Um único template pode ser compartilhado por vários templates de arquivo. Além de instruções, expressões e comentários Ruby, você pode usar variáveis predefinidas que serão expandidas para seus valores correspondentes, como macros.
As variáveis predefinidas aceitam os seguintes valores:
${FILE_NAME} Nome do arquivo atual
${USER} Nome de login do sistema do usuário atual
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${DAY} Dia atual do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${PRODUCT_NAME} Nome do IDE atual
${PROJECT_NAME} Nome do projeto atual
