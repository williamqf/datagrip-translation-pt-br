Um modelo integrado para comentários PHPDoc de uma função PHP. Ele contém um fragmento de código que pode ser incluído em modelos de arquivo (aba Templates) com a ajuda da diretiva #parse.
O modelo é editável. Além de texto estático, código e comentários, você também pode usar variáveis predefinidas, que serão expandidas para os valores correspondentes como macros.
As variáveis predefinidas aceitam os seguintes valores:
${NAME}   Nome da função.
${TYPE_HINT}   Dica de tipo (type hint) para o valor de retorno da função. Avalia como 'void' se nenhum tipo de retorno for inferido.
${PARAM_DOC}  

Comentários PHPDoc para parâmetros.

Gerado como uma série de linhas '* @param tipo nome'. Avalia como conteúdo vazio se não houver parâmetros.

${THROWS_DOC}  

Comentários de documentação para exceções.

Gerado como uma série de linhas '* @throws tipo'. Avalia como conteúdo vazio se nenhuma exceção for lançada.

${STATIC}   Assume o valor 'static' se a função/método for estático, caso contrário, uma string vazia. Exemplo: 
      #if (${STATIC} == "static") * @static
      #end
${DS}   Sinal de cifrão, avalia como um caractere '$' comum.
${DATE}   Data atual do sistema
${TIME}   Hora atual do sistema
${YEAR}   Ano atual
${MONTH}   Mês atual
${MONTH_NAME_SHORT}   As primeiras 3 letras do nome do mês atual. Exemplo: Jan, Fev, etc.
${MONTH_NAME_FULL}   Nome completo do mês atual. Exemplo: Janeiro, Fevereiro, etc.
${DAY}   Dia do mês atual
${HOUR}   Hora atual
${MINUTE}   Minuto atual
${SECOND}   Segundo atual
${PROJECT_NAME}   Nome do projeto atual
${USER}   Usuário atual
${CARET}   Marca a posição para a qual o cursor de texto deve se mover após adicionar o comentário.
