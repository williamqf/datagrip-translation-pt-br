Modelo integrado para comentários PHPDoc a serem usados com propriedades de classe PHP. Contém trechos de código que podem ser incluídos em modelos de arquivo (guia "Modelos") usando a diretiva #parse.
O modelo é editável. Além de texto estático, código e comentários, você pode usar variáveis predefinidas, que serão expandidas para os valores correspondentes assim como macros.
Variáveis predefinidas obtêm os seguintes valores:
${NAME} Nome da propriedade.
${CLASS_NAME} Nome da classe.
${STATIC} Assume o valor 'static' se a propriedade for estática, caso contrário, uma string vazia.
${NAMESPACE} Nome totalmente qualificado do namespace da propriedade, sem a barra inicial.
${DS} O cifrão é avaliado como o caractere "$" comum.
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${MONTH_NAME_SHORT} Os primeiros 3 caracteres do nome do mês atual. Por exemplo: Jan, Fev etc.
${MONTH_NAME_FULL} O nome completo do mês atual. Por exemplo: Janeiro, Fevereiro etc.
${DAY} Dia do mês atual
${HOUR} Hora atual
${MINUTE} Minuto atual
${SECOND} Segundo atual
${PROJECT_NAME} O nome do projeto atual
${USER} O usuário atual
${CARET} Armazena a posição para onde o cursor deve ser movido após a adição do comentário.
