Contém um fragmento de código que pode ser incluído em um modelo de arquivo (aba Modelos) usando a diretiva #parse.
O modelo é editável. Além de texto estático, código e comentários, você também pode usar variáveis predefinidas, que serão expandidas em valores correspondentes assim como macros.
As variáveis predefinidas aceitam os seguintes valores:
${FILE_NAME} Nome do arquivo atual
${USER} Nome de login de sistema do usuário atual
${USER_NAME} Nome completo do usuário atual
${ORGANIZATION_NAME} Nome da organização do projeto Xcode
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${MONTH_NAME_SHORT} As primeiras 3 letras do nome do mês atual. Ex: Jan, Fev, etc.
${MONTH_NAME_FULL} Nome completo do mês atual. Ex: Janeiro, Fevereiro, etc.
${DAY} Dia atual do mês
${DAY_NAME_SHORT} As primeiras 3 letras do nome do dia da semana atual. Ex: Seg, Ter, etc.
${DAY_NAME_FULL} Nome completo do dia da semana atual. Ex: Segunda-feira, Terça-feira, etc.
${HOUR} Hora atual
${MINUTE} Minuto atual
${PROJECT_NAME} Nome do projeto atual
${COMPILER_VERSION_MAJOR} Versão principal (major) do compilador Swift
${COMPILER_VERSION_MINOR} Versão secundária (minor) do compilador Swift
${PREFER_EXPLICIT_RETURN} O valor da opção Preferir retorno explícito (Preferências | Editor | Estilo de Código | Swift | Geração de Código)
