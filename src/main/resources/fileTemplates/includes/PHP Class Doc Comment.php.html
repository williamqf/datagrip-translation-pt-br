Um modelo integrado para comentários PHPDoc de uma classe PHP. Ele contém um trecho de código que pode ser incluído em modelos de arquivo (aba Modelos) com a ajuda da diretiva #parse.
O modelo é editável. Além de texto estático, código e comentários, você também pode usar variáveis predefinidas que serão expandidas para seus valores correspondentes, como macros.
As variáveis predefinidas aceitam os seguintes valores:
${NAME} Nome da classe.
${NAMESPACE} O nome totalmente qualificado do namespace da classe, sem a barra inicial.
${DS} O cifrão, avaliado como um caractere '$' comum.
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${MONTH_NAME_SHORT} As primeiras 3 letras do nome do mês atual. Ex: Jan, Fev, etc.
${MONTH_NAME_FULL} O nome completo do mês atual. Ex: Janeiro, Fevereiro, etc.
${DAY} Dia do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${SECOND} Segundo atual
${PROJECT_NAME} Nome do projeto atual
${USER} Usuário atual
${CARET} Marca a posição para a qual o cursor de texto deve ser movido após a inserção do comentário.
