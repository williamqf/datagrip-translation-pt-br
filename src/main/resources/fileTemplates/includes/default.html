Use a diretiva #parse para especificar fragmentos de código que deseja incluir em um modelo de arquivo (guia Modelos).
Um modelo pode ser compartilhado por vários modelos de arquivo.

Além de texto estático, código e comentários, você também pode usar variáveis predefinidas, que serão expandidas para seus valores correspondentes como se fossem macros.
As variáveis predefinidas aceitam os seguintes valores:
${PACKAGE_NAME} O nome do pacote no qual o novo arquivo é criado
${USER} Nome de login de sistema do usuário atual
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${MONTH_NAME_SHORT} As primeiras 3 letras do nome do mês atual (Jan, Feb, etc.)
${MONTH_NAME_FULL} Nome completo do mês atual (January, February, etc.)
${DAY} Dia atual do mês
${DAY_NAME_SHORT} As primeiras 3 letras do nome do dia atual (Mon, Tue, etc.)
${DAY_NAME_FULL} Nome completo do dia da semana atual (Monday, Tuesday, etc.)
${HOUR} Hora atual
${MINUTE} Minuto atual
${PROJECT_NAME} Nome do projeto atual
