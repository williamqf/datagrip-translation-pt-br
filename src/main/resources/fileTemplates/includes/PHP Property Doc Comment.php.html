Um template integrado para comentários PHPDoc de propriedades de classe PHP. Ele contém um fragmento de código que pode ser incluído em templates de arquivo (aba Templates) com a ajuda da diretiva #parse.
O template é editável. Além de texto estático, código e comentários, você também pode usar variáveis predefinidas que serão expandidas como macros para seus valores correspondentes.
As variáveis predefinidas aceitam os seguintes valores:
${NAME} O nome da propriedade.
${CLASS_NAME} O nome da classe.
${STATIC} Assume o valor 'static' se a propriedade for estática, caso contrário, uma string vazia.
${NAMESPACE} O nome totalmente qualificado (FQN) do namespace da propriedade, sem a barra invertida inicial.
${DS} O símbolo de cifrão, avaliado como o caractere '$' comum.
${DATE} A data atual do sistema
${TIME} A hora atual do sistema
${YEAR} O ano atual
${MONTH} O mês atual
${MONTH_NAME_SHORT} As primeiras 3 letras do nome do mês atual. Ex: Jan, Fev, etc.
${MONTH_NAME_FULL} O nome completo do mês atual. Ex: Janeiro, Fevereiro, etc.
${DAY} O dia atual do mês
${HOUR} A hora atual
${MINUTE} O minuto atual
${SECOND} O segundo atual
${PROJECT_NAME} O nome do projeto atual
${USER} O usuário atual
${CARET} Marca a posição para onde o cursor de texto deve ser movido após a inserção do comentário.
