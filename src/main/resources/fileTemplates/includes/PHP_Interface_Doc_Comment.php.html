Um template embutido para comentários PHPDoc para uso em interfaces no PHP. Ele contém trechos de código que podem ser incluídos em templates de arquivos (abas "Templates") com a ajuda da diretiva #parse.
O template é editável. Além de texto estático, código e comentários, você pode usar variáveis predefinidas, que serão expandidas para seus valores correspondentes assim como macros.
As variáveis predefinidas assumem os seguintes valores:
${NAME} O nome da interface.
${NAMESPACE} O nome completo do namespace da interface sem a barra inicial.
${DS} O símbolo do dólar, avaliado para o caracter normal '$'.
${DATE} A data atual do sistema
${TIME} A hora atual do sistema
${YEAR} O ano atual
${MONTH} O mês atual
${MONTH_NAME_SHORT} Os 3 primeiros caracteres do nome do mês atual. Ex: Jan, Fev etc..
${MONTH_NAME_FULL} O nome completo do mês atual. Ex: Janeiro, Fevereiro etc.
${DAY} O dia do mês atual
${HOUR} A hora atual
${MINUTE} O minuto atual
${SECOND} O segundo atual
${PROJECT_NAME} O nome do projeto atual
${USER} O usuário atual
${CARET} Contém a localização para onde o cursor deve ser movido após o comentário ter sido adicionado.
