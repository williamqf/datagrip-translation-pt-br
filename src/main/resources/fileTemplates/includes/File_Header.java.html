Contém um fragmento de código que pode ser incluído em modelos de arquivos (a guia Modelos) com a ajuda da diretiva #parse.
Este modelo incorporado é editável. Junto com texto estático, código e comentários, você também pode usar as variáveis predefinidas que serão expandidas como macros nos valores correspondentes.
As variáveis predefinidas assumem os seguintes valores:
${PACKAGE_NAME} Nome do pacote no qual o novo arquivo é criado
${USER} Nome de login do sistema do usuário atual
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${MONTH_NAME_SHORT} As 3 primeiras letras do nome do mês atual. Exemplo: Jan, Fev, etc.
${MONTH_NAME_FULL} Nome completo do mês atual. Exemplo: Janeiro, Fevereiro, etc.
${DAY} Dia atual do mês
${DAY_NAME_SHORT} As 3 primeiras letras do nome do dia atual. Exemplo: Seg, Ter, etc.
${DAY_NAME_FULL} Nome completo do dia atual. Exemplo: Segunda-feira, Terça-feira, etc.
${HOUR} Hora atual
${MINUTE} Minuto atual
${PROJECT_NAME} Nome do projeto atual
