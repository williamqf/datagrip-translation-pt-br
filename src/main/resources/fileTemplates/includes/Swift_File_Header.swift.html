O #parse diretiva permite incluir nos File Templates (na aba Templates) um pedaço de código.
Este template é editável. É possível usar junto dos textos estáticos, código e comentários, variáveis predefinidas que serão expandidas para seus valores correspondentes, assim como macros.
As variáveis predefinidas utilizam os seguintes valores:
${FILE_NAME} O nome do arquivo atual
${USER} Nome de login do sistema do usuário atual
${USER_NAME} Nome completo do usuário atual
${ORGANIZATION_NAME} Nome da organização do projeto Xcode
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${MONTH_NAME_SHORT} As três primeiras letras do nome do mês atual. Ex: Jan, Fev e etc
${MONTH_NAME_FULL} O nome completo do mês atual. Ex: Janeiro, Fevereiro e etc
${DAY} O dia do mês atual
${DAY_NAME_SHORT} As três primeiras letras do nome do dia da semana atual. Ex: Seg, Ter e etc
${DAY_NAME_FULL} O nome completo do dia da semana atual. Ex: Segunda-feira, Terça-feira e etc.
${HOUR} Hora atual
${MINUTE} Minuto atual
${PROJECT_NAME} O nome do projeto atual
${COMPILER_VERSION_MAJOR} A versão principal do compilador Swift
${COMPILER_VERSION_MINOR} A versão secundária do compilador Swift
${PREFER_EXPLICIT_RETURN} Valor da opção Preferir retorno explícito (Preferências | Editor | Estilo do Código | Swift | Geração de Código)
