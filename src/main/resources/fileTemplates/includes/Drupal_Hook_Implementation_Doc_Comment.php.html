Um modelo integrado para comentários PHPDoc utilizados em implementações de hooks do Drupal. Ele contém trechos de código que podem ser incluídos em modelos de arquivo (guia "Modelos") com a ajuda da diretiva #parse.
O modelo é editável. Além de texto estático, código e comentários, você também pode usar variáveis predefinidas que serão expandidas para os valores correspondentes como macros.
Variáveis predefinidas recebem os seguintes valores:
${HOOK_NAME} Nome do hook.
