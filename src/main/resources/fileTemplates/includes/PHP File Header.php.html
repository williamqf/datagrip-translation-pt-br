Um modelo integrado para um cabeçalho de arquivo PHP. Ele contém um trecho de código que pode ser incluído em modelos de arquivo (aba Templates) com a ajuda da diretiva #parse.
O modelo é editável. Além de texto estático, código e comentários, você também pode usar variáveis predefinidas, que serão expandidas para os valores correspondentes como macros.
As variáveis predefinidas aceitam os seguintes valores:
${FILE_NAME} Nome do arquivo atual
${USER} Nome de login do usuário atual no sistema
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${MONTH_NAME_SHORT} As primeiras 3 letras do nome do mês atual. Ex: Jan, Fev, etc.
${MONTH_NAME_FULL} O nome completo do mês atual. Ex: Janeiro, Fevereiro, etc.
${DAY} Dia atual do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${SECOND} Segundo atual
${PRODUCT_NAME} Nome da IDE atual
${PROJECT_NAME} Nome do projeto atual
