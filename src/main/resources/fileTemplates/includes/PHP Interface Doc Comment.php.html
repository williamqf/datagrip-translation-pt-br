Um template integrado para comentários PHPDoc de uma interface PHP. Ele contém um fragmento de código que pode ser incluído em templates de arquivo (aba Templates) com a ajuda da diretiva #parse.
O template é editável. Além de texto estático, código e comentários, você também pode usar variáveis predefinidas que serão expandidas para os valores correspondentes, funcionando como macros.
As variáveis predefinidas aceitam os seguintes valores:
${NAME} Nome da interface.
${NAMESPACE} O nome totalmente qualificado (fully qualified name) do namespace da interface, sem a barra invertida inicial.
${DS} Símbolo de cifrão, avaliado como o caractere "$" literal.
${DATE} Data atual do sistema.
${TIME} Hora atual do sistema.
${YEAR} Ano atual.
${MONTH} Mês atual.
${MONTH_NAME_SHORT} As primeiras 3 letras do nome do mês atual. Ex: Jan, Feb, etc.
${MONTH_NAME_FULL} Nome completo do mês atual. Ex: January, February, etc.
${DAY} Dia atual do mês.
${HOUR} Hora atual.
${MINUTE} Minuto atual.
${SECOND} Segundo atual.
${PROJECT_NAME} Nome do projeto atual.
${USER} Usuário atual do sistema.
${CARET} Marca a posição para onde o cursor de texto será movido após a inserção do comentário.
