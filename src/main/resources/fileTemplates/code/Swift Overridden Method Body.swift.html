Preenche o corpo do método quando um método sobrescrito é gerado (por exemplo, usando a ação Código | Gerar | Sobrescrever Métodos).
O template é editável. Além de expressões Swift e comentários, você pode usar variáveis predefinidas que serão expandidas para os valores correspondentes.
As variáveis predefinidas aceitam os seguintes valores:
${CALL_SUPER} Chamada ao método super. Para métodos Void: super.methodName() com ou sem parâmetros; para outros métodos: return super.methodName() com ou sem parâmetros.
${RETURN_TYPE} O tipo de retorno do método criado
${DEFAULT_RETURN_VALUE} O valor retornado por padrão pelo método
${SIGNATURE} A assinatura do método
Todas as variáveis predefinidas do template Cabeçalho de arquivo Swift (guia Incluir) também estão disponíveis (por exemplo, ${USER}, ${DATE}, etc.).
