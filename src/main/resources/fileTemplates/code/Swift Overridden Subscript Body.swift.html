Preenche o corpo de um subscript sobrescrito quando ele é gerado (por exemplo, usando a ação Código | Gerar | Sobrescrever Métodos).
O modelo é editável. Além de expressões Swift e comentários, você também pode usar variáveis predefinidas que serão expandidas para seus valores correspondentes.
As variáveis predefinidas assumirão os seguintes valores:
${CALL_SUPER} Chamada do super subscript - return super[] com parâmetros
${RETURN_TYPE} O tipo de retorno do subscript criado
${DEFAULT_RETURN_VALUE} O valor retornado por padrão pelo subscript
${SIGNATURE} A assinatura do subscript
${HAS_SET} "true" quando o subscript tem um setter
${HAS_SUPER_SET} "true" quando o subscript sobrescrito tem um setter
${GETTER_EFFECTS} Efeitos do getter (async, throws) para o getter da propriedade
${GET_ATTRIBUTES} Atributos para o getter do subscript
${SET_ATTRIBUTES} Atributos para o setter do subscript
Todas as variáveis predefinidas no modelo Cabeçalho de Arquivo Swift (aba Inclui) também estão disponíveis (por exemplo, ${USER}, ${DATE}, etc.).
