Este modelo integrado é usado para criar componentes de função React, por exemplo, através da refatoração Extrair componente. Você pode personalizar este modelo, por exemplo, definir componentes como funções de seta ou alterar as definições de tipos de prop. Você também pode usar variáveis predefinidas. Essas variáveis serão substituídas pelos valores correspondentes quando o modelo for aplicado. Este modelo é usado em TypeScript e Flow.
Variáveis predefinidas assumirão os seguintes valores:
${NAME} nome do componente
${COMPONENT_BODY} corpo do componente
${PROP_TYPES} definição de tipo de props
