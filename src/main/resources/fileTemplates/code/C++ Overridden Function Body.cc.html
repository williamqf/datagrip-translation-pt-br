Cria o corpo de uma função sobrescrita quando ela é gerada (por exemplo, usando a ação Código | Sobrescrever Métodos).
Ao editar o template, você pode usar expressões C++, comentários e as seguintes variáveis predefinidas:
${CALL_SUPER} Chamada da função base: para funções void — BaseClass::functionName(), com ou sem parâmetros; para outras funções — return BaseClass::functionName(), com ou sem parâmetros.
${RETURN_TYPE} O tipo de retorno da função.
${DEFAULT_RETURN_VALUE} O valor de retorno padrão da função.
Todas as variáveis predefinidas do template Cabeçalho de Arquivo C (aba Includes) também estão disponíveis (por exemplo, ${USER} e ${DATE}).
