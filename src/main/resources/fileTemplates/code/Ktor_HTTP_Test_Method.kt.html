Este é um modelo para criar métodos de teste HTTP para o Ktor em uma classe de teste do JUnit 4.
As variáveis predefinidas assumem os seguintes valores:
${NAME} Nome do método que será criado
${HTTP_METHOD} Método HTTP sendo usado (por exemplo, GET, POST).
${ROUTE} Rota relativa ao serviço atual que precisa ser testada
${CONFIGURE_SERVER} Um método de extensão de rota para configurar a árvore de rotas para a parte do servidor que precisa ser testada
${BODY} Corpo do método gerado
