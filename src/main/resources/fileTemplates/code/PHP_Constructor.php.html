Um template embutido para um construtor de uma classe PHP.
Variáveis predefinidas assumem os seguintes valores:
${PARAM_LIST} Uma lista de parâmetros do construtor separados por vírgulas, uma para cada propriedade selecionada.
${PARAM_DOC} Os comentários PHPDoc dos parâmetros.
Gerado como uma série de linhas '* @param type name'. Se não houver parâmetros, é avaliado como um conteúdo vazio.
${THROWS_DOC} Os comentários PHPDoc das exceções.
Gerado como uma série de linhas '* @throws type'. Se nenhuma exceção for gerada, é avaliado como um conteúdo vazio.
${CLASS_NAME} O nome da classe.
${BODY} O corpo do construtor que inclui a atribuição '$this->property=$property'.
