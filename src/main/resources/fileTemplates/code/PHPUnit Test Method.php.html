Um template integrado para um método de teste PHPUnit. Por exemplo, ele é usado quando a caixa de seleção para gerar métodos de teste é selecionada na caixa de diálogo "Novo Teste".
As variáveis predefinidas aceitam os seguintes valores:
${NAME} O nome do método.
${CAPITALIZED_NAME} O nome do método com o primeiro caractere em maiúsculo.
${TESTED_NAME} O nome da classe que está sendo testada.
