Cria o corpo de um método sobreposto -isEqual: quando ele é gerado, por exemplo, ao chamar a ação Código | Sobrepor Método ou Código | Gerar | -isEqual: e -hash.
O modelo é editável. Junto com expressões e comentários do Objective-C, você pode usar variáveis predefinidas que serão expandidas posteriormente para os valores correspondentes.
As variáveis predefinidas terão os seguintes valores:
${CALL_SUPER} uma chamada ao super método, para métodos void — [super methodName] com ou sem parâmetros; para outros métodos — return [super methodName] com ou sem parâmetros
${RETURN_TYPE} o tipo de retorno do método criado
${DEFAULT_RETURN_VALUE} o valor retornado pelo método por padrão
${CONTAINING_CLASS} o nome da classe na qual o método é sobreposto
${CUSTOM_CODE} específico do tipo isEqualToType: o nome do método de comparação, se gerado
Todas as variáveis predefinidas do modelo Cabeçalho do Arquivo C (a guia Inclusões) também estão disponíveis (${USER},${DATE}, etc.)
