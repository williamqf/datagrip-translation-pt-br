Substitui uma expressão de literal de string (por exemplo, String s = "Hello";) por uma expressão de código Java internacionalizada (por exemplo, String s = myResourceBundle.getString("hello");).
Este template incorporado é editável. Junto com as expressões e comentários Java, pode também utilizar variáveis predefinidas que serão então expandidas nos valores correspondentes.
As variáveis predefinidas assumem os seguintes valores:
${RESOURCE_BUNDLE} Expressão do tipo java.util.ResourceBundle, cuja instância está disponível neste contexto.
${PROPERTY_KEY} Nome da chave de propriedade especificada no ficheiro de propriedades correspondente. Normalmente, é o valor utilizado para o parâmetro do método ResourceBundle.getString().
${PROPERTY_VALUE} Valor da propriedade definido no ficheiro de propriedades correspondente. Este é o valor do literal da string Java original.
