Este modelo integrado é usado para criar componentes funcionais do React (por exemplo, por meio da refatoração Extrair Componente). Você pode personalizar este modelo, por exemplo, definindo o componente como uma arrow function ou alterando a definição dos tipos de prop. Você também pode usar variáveis predefinidas. Essas variáveis serão substituídas pelos valores correspondentes quando o modelo for aplicado. Este modelo é usado em TypeScript e Flow.
As variáveis predefinidas assumirão os seguintes valores:
${NAME} Nome do componente
${COMPONENT_BODY} Corpo do componente
${PROP_TYPES} Definição de tipos das props
