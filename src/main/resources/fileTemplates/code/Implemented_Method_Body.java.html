Preenche o corpo de um método implementado quando ele é gerado, por exemplo, quando as ações de intenção Implementar Classe Abstrata ou Implementar Método Abstrato são aplicadas ou quando a função Código | Implementar Métodos... é chamada.
Este template integrado é editável. Junto com expressões e comentários Java, você também pode usar as variáveis predefinidas que então serão expandidas nos valores correspondentes.
As variáveis predefinidas recebem os seguintes valores:
${RETURN_TYPE} Tipo de retorno do método criado
${DEFAULT_RETURN_VALUE} Valor retornado pelo método padrão
${METHOD_NAME} Nome do método implementado
${CLASS_NAME} Nome qualificado da classe na qual o método é implementado
${SIMPLE_CLASS_NAME} Nome não qualificado da classe na qual o método é implementado
Todas as variáveis predefinidas do template Cabeçalho do Arquivo (guia Includes) também estão disponíveis (por exemplo ${USER}, ${DATE}, etc.).
