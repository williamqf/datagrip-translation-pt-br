Cria definições de etapas do Cucumber para arquivos Java.
As variáveis predefinidas aceitam os seguintes valores:
${STEP_KEYWORD} A anotação da definição de etapa. Normalmente, a palavra-chave é usada para invocar etapas (Given, Then, And, etc.).
${STEP_REGEXP} A expressão regular gerada para aceitar a etapa.
${METHOD_NAME} O nome do método gerado.
${PARAMETERS} Os parâmetros do método gerado.
${BODY} O corpo do método gerado.
