Cria o corpo de um método implementado -description quando ele é gerado para uma classe que não tem variáveis de instância, por exemplo, quando a ação Código | Gerar | -description é chamada.
O modelo é editável. Junto com expressões e comentários de Objective-C, você pode usar variáveis predefinidas que serão então expandidas para os valores correspondentes.
As variáveis predefinidas assumirão os seguintes valores:
${RETURN_TYPE} o tipo de retorno do método criado
${DEFAULT_RETURN_VALUE} o valor retornado pelo método por padrão
${CONTAINING_CLASS} o nome da classe em que o método é implementado
Todas as variáveis predefinidas do modelo Cabeçalho de Arquivo C (a guia Inclusões) também estão disponíveis (${USER}, ${DATE}, etc.)
