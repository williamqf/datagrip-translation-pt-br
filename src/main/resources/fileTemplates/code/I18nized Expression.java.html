Substitui expressões de literais de string (como String s = "Hello";) por expressões de código Java internacionalizadas (como String s = myResourceBundle.getString("hello");).
Este modelo integrado é editável. Além de expressões e comentários Java, você também pode usar variáveis predefinidas que serão expandidas para os valores correspondentes.
As variáveis predefinidas aceitam os seguintes valores:
${RESOURCE_BUNDLE} Uma expressão do tipo java.util.ResourceBundle; instâncias deste tipo estão disponíveis neste contexto.
${PROPERTY_KEY} O nome da chave da propriedade especificada no arquivo de propriedades correspondente. Geralmente, é o valor usado como parâmetro para o método ResourceBundle.getString().
${PROPERTY_VALUE} O valor da propriedade definido no arquivo de propriedades correspondente. Este é o valor original do literal de string Java.
