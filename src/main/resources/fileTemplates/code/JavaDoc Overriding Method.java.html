Define o template usado para gerar JavaDoc para métodos sobrescritos, por exemplo, quando a ação de intenção Adicionar Javadoc é aplicada.

Além de expressões Java e comentários, você também pode usar variáveis predefinidas, que serão expandidas para seus valores correspondentes como macros.

Usando a diretiva #parse, você pode incluir templates da aba Inclusões. Para incluir um template, especifique o nome completo do template como um parâmetro entre aspas (por exemplo, #parse("File Header.java")).
As variáveis predefinidas aceitam os seguintes valores:
${PARAMS}   Lista de nomes de parâmetros
${TYPE_PARAMS}   Lista de nomes de parâmetros de tipo
${THROWS}   Lista de exceções lançadas
${RETURN_TYPE}   O tipo retornado pelo método
${PARAMS_INHERITED}   Lista de parâmetros com a descrição do super método
${ELEMENT_NAME}   Nome do membro da classe que possui o JavaDoc gerado
${CONTAINING_CLASS}   Nome da classe pai
${SUPER_METHOD}   Nome totalmente qualificado do super método
${PACKAGE_NAME}   Nome do pacote no qual o novo enum será criado
${NAME}   O nome do novo enum que você especificou no diálogo Criar Nova Classe
${USER}   Nome de login do sistema do usuário atual
${DATE}   Data atual do sistema
${TIME}   Hora atual do sistema
${YEAR}   Ano atual
${MONTH}   Mês atual
${MONTH_NAME_SHORT}   As primeiras 3 letras do nome do mês atual (Jan, Fev, etc.)
${MONTH_NAME_FULL}   Nome completo do mês atual (Janeiro, Fevereiro, etc.)
${DAY}   Dia do mês atual
${HOUR}   Hora atual
${MINUTE}   Minuto atual
${PROJECT_NAME}   Nome do projeto atual
