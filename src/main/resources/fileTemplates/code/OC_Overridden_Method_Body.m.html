Cria o corpo de um método substituído quando ele é gerado, por exemplo, ao chamar a ação Código | Substituir Método.
O modelo é editável. Juntamente com expressões e comentários Objective-C, você também pode usar variáveis predefinidas que serão expandidas para os valores correspondentes.
As variáveis predefinidas receberão os seguintes valores:
${CALL_SUPER} uma chamada de supermétodo, para métodos void — [super methodName] com ou sem parâmetros; para outros métodos — return [super methodName] com ou sem parâmetros
${RETURN_TYPE} o tipo de retorno do método criado
${DEFAULT_RETURN_VALUE} o valor retornado pelo método por padrão
${CONTAINING_CLASS} o nome da classe em que o método é substituído
Todas as variáveis predefinidas do modelo Cabeçalho de Arquivo C (a guia Inclusões) também estão disponíveis (por exemplo, ${USER}, ${DATE}, etc.)
