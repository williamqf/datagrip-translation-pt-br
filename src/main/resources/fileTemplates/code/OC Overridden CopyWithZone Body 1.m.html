Cria o corpo do método -copyWithZone: sobrescrito ao gerá-lo (por exemplo, ao chamar a ação Código | Gerar | -copyWithZone:).
O template é editável. Além de expressões e comentários Objective-C, você também pode usar variáveis predefinidas, que serão expandidas para seus valores correspondentes.
As variáveis predefinidas aceitam os seguintes valores:
${CALL_SUPER} A chamada do método super; para métodos void — [super methodName], com ou sem parâmetros; para outros métodos — return [super methodName], com ou sem parâmetros
${RETURN_TYPE} O tipo de retorno do método criado
${DEFAULT_RETURN_VALUE} O valor retornado por padrão pelo método
${CONTAINING_CLASS} O nome da classe na qual o método é sobrescrito
${CUSTOM_CODE} O corpo do método gerado
Todas as variáveis predefinidas do template Cabeçalho de arquivo C (aba Incluir) também estão disponíveis (${USER}, ${DATE}, etc.)
