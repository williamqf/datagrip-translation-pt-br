Um template embutido para o corpo de métodos PHP sobrepostos.
Variáveis predefinidas assumem os seguintes valores:
${NAME} Nome do método.
${PARAM_LIST} Uma lista de parâmetros separada por vírgulas como a lista de parâmetros do método original.
${RETURN} Se o método original tiver uma declaração de return , esta variável conterá uma string 'return', caso contrário, ela conterá uma string vazia.
