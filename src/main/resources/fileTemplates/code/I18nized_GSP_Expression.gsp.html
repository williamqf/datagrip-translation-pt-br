Este é um modelo interno usado para substituir texto dentro do GSP. Por exemplo:
<html> <body> Hello! </body> </html>
Para uso de tags customizadas internacionalizadas de GSP, por exemplo:
<html> <body> <g:message code="HelloKey"/> </body> </html> .

O modelo é editável. Assim como expressões permitidas em um contexto GSP, você pode usar variáveis pré-definidas que irão expandir para os seus respectivos valores.

As variáveis pré-definidas recebem os seguintes valores:
${PROPERTY_KEY} O nome da chave da propriedade conforme especificado no arquivo de propriedades correspondente.
${PROPERTY_VALUE} O valor da propriedade conforme especificado no arquivo de propriedades correspondente. Este é o valor de seleção de texto GSP original.
