Preencha o corpo do método quando um método implementado for gerado (exemplo: usando a ação Código | Gerar | Implementar métodos).
O template é editável. Além das expressões e comentários Swift, você pode usar variáveis predefinidas, que serão expandidas para seus respectivos valores.
As variáveis predefinidas recebem os seguintes valores:
${RETURN_TYPE} Tipo de retorno do método criado
${DEFAULT_RETURN_VALUE} Valor padrão sugerido para o valor de retorno (pode ser nulo)
${SIGNATURE} Assinatura do método
Todas as variáveis predefinidas no template Cabeçalho de Arquivo Swift (guia Inclusões) também estão disponíveis (por exemplo ${USER}, ${DATE} etc.).
