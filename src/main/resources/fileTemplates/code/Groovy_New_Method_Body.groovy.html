Este é um modelo embutido usado para preencher o corpo de um método Groovy sempre que ele for gerado pelo programa, por exemplo, ao usar a ação de intenção Create method from usage.
O modelo é editável. Junto com expressões e comentários Groovy, você também pode usar variáveis predefinidas que serão expandidas para os valores correspondentes.
As variáveis predefinidas terão os seguintes valores:
${RETURN_TYPE} um tipo de retorno de um método criado
${DEFAULT_RETURN_VALUE} um valor retornado pelo método por padrão
${METHOD_NAME} nome do método criado
${CLASS_NAME} nome qualificado da classe onde o método é criado
${SIMPLE_CLASS_NAME} nome não qualificado da classe onde o método é implementado
