Este modelo integrado é usado para criar componentes de classe React, por exemplo, através da refatoração Extrair componente. Você pode customizar o modelo, por exemplo, usar Component ao invés de PureComponent ou alterar as definições dos tipos de prop. Você também pode usar variáveis predefinidas. Essas variáveis serão substituídas pelos valores correspondentes quando o modelo for aplicado.
Variáveis predefinidas assumirão os seguintes valores:
${NAME} nome do componente
${COMPONENT_BODY} corpo do componente
${PROP_TYPES} definição dos tipos de prop (usando módulo prop-types)
${PROP_TYPES} valores de prop padrão a serem usados pelo React quando o valor de prop não for fornecido
