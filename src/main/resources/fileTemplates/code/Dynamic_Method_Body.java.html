Este é um template embutido usado para preencher o corpo de um método dinâmico FreeMarker sempre que este é gerado programaticamente.
Este template pode ser editado. Além das expressões e comentários Java, você também pode usar variáveis predefinidas, que serão expandidas no seu valor correspondente.
As variáveis predefinidas recebem os seguintes valores:
${RETURN_TYPE} O tipo de retorno do método criado
${DEFAULT_RETURN_VALUE} O valor que o método retorna por padrão
${METHOD_NAME} O nome do método criado
${CLASS_NAME} O nome qualificado da classe na qual o método é criado
