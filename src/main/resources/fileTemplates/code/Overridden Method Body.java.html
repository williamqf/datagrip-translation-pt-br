Preenche o corpo do método sempre que um método sobrescrito é gerado pelo programa (por exemplo, ao chamar a função Código | Sobrescrever Métodos...).
Este template integrado é editável. Além de expressões Java e comentários, você também pode usar variáveis predefinidas que serão expandidas para seus respectivos valores.
As variáveis predefinidas aceitam os seguintes valores:
${CALL_SUPER}   Chamada do método super; para métodos void - super.method_name();, com ou sem parâmetros; para outros métodos - return super.method_name();, com ou sem parâmetros
${PLAIN_CALL_SUPER}   Chamada do método super, super.method_name();, com ou sem parâmetros
${RETURN_TYPE}   O tipo de retorno do método criado
${DEFAULT_RETURN_VALUE}   O valor retornado pelo método por padrão
${METHOD_NAME}   O nome do método sobrescrito
${CLASS_NAME}   O nome qualificado da classe na qual o método é sobrescrito
${SIMPLE_CLASS_NAME}   O nome não qualificado da classe na qual o método é implementado
Todas as variáveis predefinidas do template Cabeçalho de Arquivo (aba Includes) também estão disponíveis. (Por exemplo: ${USER}, ${DATE}, etc.)
