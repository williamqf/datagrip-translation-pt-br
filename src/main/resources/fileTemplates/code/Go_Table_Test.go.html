Go Table Test é um modelo embutido que pode ser usado para criar uma função de teste de tabela Go em um arquivo de teste. Este modelo funciona apenas em arquivos de teste com sufixo _test. Para usar o modelo, clique em Código | Gerar e selecione Teste de tabela.
Lista de variáveis predefinidas
${NAME} Nome da função de teste de tabela
${PARAM_NAME} Nome do parâmetro da função
${PARAM_TYPE} Tipo do parâmetro da função
