Preenche o corpo de um subscript implementado quando ele é gerado (por exemplo, por meio da ação Código | Gerar | Implementar Métodos).
O modelo é editável. Além de expressões Swift e comentários, você também pode usar variáveis predefinidas que serão expandidas para seus valores correspondentes.
As variáveis predefinidas aceitam os seguintes valores:
${RETURN_TYPE} O tipo de retorno do subscript criado
${DEFAULT_RETURN_VALUE} O valor de retorno padrão sugerido (pode estar vazio)
${SIGNATURE} A assinatura do subscript
${HAS_SET} "true" se o subscript tiver um setter
${GETTER_EFFECTS} Efeitos do getter (async, throws) para o getter da propriedade
${GET_ATTRIBUTES} Atributos para o getter do subscript
${SET_ATTRIBUTES} Atributos para o setter do subscript
Todas as variáveis predefinidas no modelo Cabeçalho de Arquivo Swift (aba Incluir) também estão disponíveis (por exemplo, ${USER}, ${DATE}, etc.).
