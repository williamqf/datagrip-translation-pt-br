Preenche o corpo do bloco catch quando gerado, p. ex. quando a função Código | Envolver com... é aplicada.
Este modelo embutido é editável. Junto com as expressões e comentários Java, você também pode usar as variáveis predefinidas que serão depois expandidas para os valores correspondentes.
As variáveis predefinidas assumem os seguintes valores:
${EXCEPTION}Nome da variável Exception especificada como parâmetro catch
${EXCEPTION_TYPE}Tipo do parâmetro catch
