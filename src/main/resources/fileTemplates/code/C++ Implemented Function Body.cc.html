Cria o corpo de uma função implementada quando ela é gerada (por exemplo, ao usar a ação Code | Implement Methods).
Ao editar o template, você pode usar expressões C++, comentários e as seguintes variáveis predefinidas:
${RETURN_TYPE} O tipo de retorno da função.
${DEFAULT_RETURN_VALUE} O valor de retorno padrão da função.
Todas as variáveis predefinidas do template C File Header (aba Includes) também estão disponíveis (por exemplo, ${USER} e ${DATE}).
