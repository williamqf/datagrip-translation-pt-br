Preencha o corpo da função Kotlin sempre que isso for gerado pelo programa por exemplo, ao usar a ação de intenção Criar Função a partir do Uso.
Este template é editável. Assim como expressões e comentários Kotlin, você pode usar variáveis pré-definidas, as quais serão expandidas para o valor correspondente.
As variáveis pré-definidas recebem os seguintes valores:
${RETURN_TYPE} Tipo de retorno da função criada
${FUNCTION_NAME} Nome da função criada
${CLASS_NAME} Nome qualificado da classe onde a função será criada
${SIMPLE_CLASS_NAME} Nome não qualificado da classe onde a função será implementada
