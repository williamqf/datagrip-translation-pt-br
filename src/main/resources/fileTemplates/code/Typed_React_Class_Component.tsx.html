Este template embutido é utilizado para a criação de componentes de classe React, por exemplo, mediante a refatoração "Extrair componente". Você pode customizar o template, por exemplo, utilizar "Componente" em vez de "PureComponent" ou alterar as definições dos tipos de "prop". Você pode também utilizar variáveis predefinidas. Essas variáveis serão substituídas pelos valores correspondentes quando o template é aplicado. Esse template é utilizado em TypeScript e Flow.
Variáveis predefinidas anzunehmen os seguintes valores:
${NAME} nome do componente
${COMPONENT_BODY} corpo do componente
${PROP_TYPES} definição de tipo de props
${PROP_TYPES} valores padrão de props a serem usados pelo React quando o valor de prop não for fornecido
