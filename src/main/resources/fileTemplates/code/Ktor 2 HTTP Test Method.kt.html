Este é um modelo para criar métodos de teste HTTP para Ktor 2.x.x em classes de teste JUnit 4.
As variáveis predefinidas aceitarão os seguintes valores:
${NAME} O nome do método criado
${HTTP_METHOD} O método HTTP utilizado (GET, POST, etc.).
${ROUTE} A rota relativa ao serviço atual que precisa ser testado
${CONFIGURE_SERVER} Um método de extensão de rota para configurar a árvore de roteamento para a parte do servidor que precisa ser testada
${BODY} O corpo do método gerado
