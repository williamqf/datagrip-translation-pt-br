Go Benchmark é um template integrado que pode ser usado para criar funções de benchmark do Go em arquivos de teste. Este template funciona apenas em arquivos de teste com o sufixo _test. Para usar o template, clique em Código | Gerar e selecione Benchmark.
Lista de variáveis predefinidas
${NAME} O nome da função de benchmark
${PARAM_NAME} O nome do parâmetro da função
${PARAM_TYPE} O tipo do parâmetro da função
