Define um template usado para gerar JavaDoc para os métodos, por exemplo
quando a ação de intenção Adicionar Javadoc é aplicada.

Junto com expressões Java e comentários, você também pode usar as variáveis predefinidas
que serão expandidas como macros para os valores correspondentes.

Usando a diretiva #parse, você pode incluir templates a partir da aba Incluir. Para incluir um template, especifique o nome completo do template como parâmetro entre aspas (por exemplo, #parse("File Header.java")).
As variáveis predefinidas assumem os seguintes valores:
${PARAMS} Lista dos nomes dos parâmetros
${TYPE_PARAMS} Lista dos nomes dos parâmetros de tipo
${THROWS} Lista das exceções lançadas
${RETURN_TYPE} Tipo retornado pelo método
${ELEMENT_NAME} Nome do membro da classe que é o dono do JavaDoc gerado
${CONTAINING_CLASS} Nome da classe pai
${PACKAGE_NAME} Nome do pacote no qual uma nova enumeração é criada
${NAME} Nome da nova enumeração especificada por você na caixa de diálogo Criar Nova Classe
${USER} Nome de login do usuário atual no sistema
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${MONTH_NAME_SHORT} As primeiras 3 letras do nome do mês atual (Jan, Fev, e assim por diante)
${MONTH_NAME_FULL} Nome completo do mês atual (Janeiro, Fevereiro, e assim por diante)
${DAY} Dia atual do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${PROJECT_NAME} Nome do projeto atual
