Este modelo integrado é usado para preencher o corpo do método quando um método é gerado (por exemplo, por meio da ação de intenção Criar método a partir do uso).
O modelo é editável. Além de expressões TypeScript e comentários, você também pode usar variáveis predefinidas. Ao aplicar o modelo, essas variáveis serão substituídas pelos valores correspondentes.
As variáveis predefinidas aceitam os seguintes valores:
${RETURN_TYPE} O tipo de retorno do método criado
${DEFAULT_RETURN_VALUE} O valor retornado por padrão pelo método
${METHOD_NAME} O nome do método criado
${SEMICOLON} Um ponto e vírgula ou vazio, dependendo das opções de estilo de código
Todas as variáveis predefinidas do modelo Cabeçalho de arquivo (aba Incluir) também estão disponíveis. (Por exemplo: ${USER}, ${DATE}, etc.)
