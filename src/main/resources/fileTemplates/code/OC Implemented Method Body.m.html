Cria o corpo de um método implementado quando este é gerado (por exemplo, ao invocar a ação Código | Implementar Métodos).
O modelo é editável. Além de expressões Objective-C e comentários, você também pode usar variáveis predefinidas que serão expandidas para os valores correspondentes.
As variáveis predefinidas aceitam os seguintes valores:
${RETURN_TYPE} O tipo de retorno do método criado
${DEFAULT_RETURN_VALUE} O valor retornado por padrão pelo método
${CONTAINING_CLASS} O nome da classe na qual o método foi implementado
Todas as variáveis predefinidas do modelo Cabeçalho de arquivo C (aba Includes) também estão disponíveis (${USER}, ${DATE}, etc.)
