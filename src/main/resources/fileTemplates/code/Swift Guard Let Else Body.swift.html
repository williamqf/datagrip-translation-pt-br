Preenche o bloco else após desempacotar um valor opcional com a instrução guard. Este template é editável. Além de expressões e comentários Swift, você pode usar variáveis pré-definidas que serão expandidas para os valores correspondentes.
As variáveis pré-definidas assumirão os seguintes valores:
${RETURN_TYPE} O tipo de retorno da função envolvente (se houver)
${DEFAULT_RETURN_VALUE} O valor de retorno padrão sugerido (pode estar vazio)
Todas as variáveis pré-definidas no template Swift File Header (aba Includes) também estão disponíveis (por exemplo, ${USER}, ${DATE}, etc.).
