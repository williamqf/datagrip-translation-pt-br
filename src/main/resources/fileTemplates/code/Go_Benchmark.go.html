Go Benchmark é um template interno que pode ser utilizado para criar funções de benchmark Go no arquivo de teste. Este template funciona apenas em arquivos de teste com o sufixo _test. Para usar este template, clique em Código | Gerar e selecione Benchmark.
Lista de variáveis predefinidas
${NAME} Nome da função de benchmark
${PARAM_NAME} Nome do parâmetro da função
${PARAM_TYPE} Tipo do parâmetro da função
