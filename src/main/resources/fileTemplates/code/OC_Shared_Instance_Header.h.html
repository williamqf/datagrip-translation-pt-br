Gera uma declaração do método +sharedInstance na seção de cabeçalho de classe, por exemplo, ao chamar a ação Code (Código) | Generate (Gerar) | +sharedInstance.
O modelo é editável. Junto com expressões e comentários Objective-C, você pode usar variáveis predefinidas que serão então expandidas para os valores correspondentes.
As variáveis predefinidas assumirão os seguintes valores:
${$CLASS_TYPE} o nome da classe em que o método é implementado
Todas as variáveis predefinidas do modelo C File Header (Cabeçalho de arquivo C) (a guia Includes (Inclui)) também estão disponíveis (${USER}, ${DATE}, etc.)
