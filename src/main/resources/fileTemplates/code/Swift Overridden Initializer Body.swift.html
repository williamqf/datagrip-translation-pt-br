Preenche o corpo de um inicializador sobrescrito quando este é gerado (por exemplo, usando a ação Código | Gerar | Sobrescrever Métodos).
O modelo é editável. Além de expressões Swift e comentários, você também pode usar variáveis predefinidas, que serão expandidas para os valores correspondentes.
As variáveis predefinidas aceitam os seguintes valores:
${CALL_SUPER} Chamada do inicializador da superclasse - super.init() com ou sem parâmetros.
${SIGNATURE} A assinatura do método
Todas as variáveis predefinidas no modelo Cabeçalho de Arquivo Swift (aba Incluir) também estão disponíveis (por exemplo, ${USER}, ${DATE}, etc.).
