Cria uma definição de passo no Cucumber para um arquivo Java.
Variáveis pré-definidas recebem os seguintes valores:
${STEP_KEYWORD} Anoção de definição de passo. Geralmente, a palavra-chave é utilizada para invocar um passo (Given, Then, And, etc.).
${STEP_REGEXP} Expressão regular gerada para aceitar o passo.
${METHOD_NAME} Nome do método gerado.
${PARAMETERS} Parâmetros do método gerado.
${BODY} Corpo do método gerado.
