Template integrado para métodos de teste de unidade do Codeception usado quando, por exemplo, as caixas de seleção para métodos gerados são selecionadas na caixa de diálogo 'Novo Teste'.
As variáveis predefinidas assumirão os seguintes valores:
${NAME} O nome do método.
${CAPITALIZED_NAME} O nome do método com o primeiro caractere em maiúsculo.
${TESTED_NAME} O nome da classe que está sendo testada.
