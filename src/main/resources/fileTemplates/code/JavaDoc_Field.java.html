Define o template usado para gerar JavaDoc para os campos, por exemplo, quando a ação de intenção Adicionar Javadoc é aplicada.

Junto com expressões e comentários Java, você também pode usar as variáveis predefinidas que serão então expandidas como macros para os valores correspondentes.

Utilizando a directiva #parse, você pode incluir templates a partir da guia Inclusões. Para incluir um template, especifique o nome completo do template como um parâmetro entre aspas (por exemplo, #parse("File Header.java")).
As variáveis predefinidas assumem os seguintes valores:
${ELEMENT_NAME} Nome do membro da classe que é proprietário do JavaDoc gerado
${CONTAINING_CLASS} Nome da classe pai
${PACKAGE_NAME} Nome do pacote no qual um novo enum é criado
${NAME} Nome do novo enum especificado por você na caixa de diálogo Criar nova classe
${USER} Nome de login do sistema do usuário atual
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${MONTH_NAME_SHORT} Primeiras 3 letras do nome do mês atual (Jan, Fev, etc.)
${MONTH_NAME_FULL} Nome completo do mês atual (Janeiro, Fevereiro, etc.)
${DAY} Dia atual do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${PROJECT_NAME} Nome do projeto atual
