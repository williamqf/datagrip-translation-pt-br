Este modelo embutido é usado para criar componentes de funções React, por exemplo, por meio da refatoração da Extract Component. Você pode personalizar o modelo, por exemplo, definir componentes como funções de seta ou alterar as definições dos tipos de prop. Você também pode usar variáveis pré-definidas. Essas variáveis serão substituídas pelos valores correspondentes quando o modelo for aplicado.
As variáveis pré-definidas terão os seguintes valores:
${NAME} nome do componente
${HAS_PROPS} true se o componente tiver props
${COMPONENT_BODY} corpo do componente
${PROP_TYPES} definição de tipos de propriedade (usando o módulo prop-types)
