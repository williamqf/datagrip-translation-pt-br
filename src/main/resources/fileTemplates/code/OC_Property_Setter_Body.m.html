Cria o corpo do método setter sobrecarregado para uma propriedade quando gerado, por exemplo, ao chamar a ação Código | Sobrecarregar Método.
O modelo é editável. Junto com expressões e comentários Objective-C, você pode usar variáveis predefinidas que serão expandidas para os valores correspondentes.
As variáveis predefinidas assumirão os seguintes valores:
${CALL_SUPER} chamada de método super, para métodos void — [super methodName] com ou sem parâmetros; para outros métodos — return [super methodName] com ou sem parâmetros
${RETURN_TYPE} o tipo de retorno do método criado
${DEFAULT_RETURN_VALUE} o valor retornado pelo método por padrão
${CONTAINING_CLASS} o nome da classe onde o método é sobrecarregado
${IVAR_IS_AVAILABLE} "true" quando a propriedade tem uma variável de instância
${IVAR} o nome da variável de instância
${PARAM} o nome de um parâmetro setter
${SETTER_SEMANTICS} no modo ARC: "copyArc" quando o atributo copy é definido, "assign" caso contrário
no modo não-ARC: "copy" quando o atributo copy é definido, "retain" quando o atributo retain é definido, "assign" caso contrário
Todas as variáveis predefinidas do modelo Cabeçalho do Arquivo C (a guia Inclusões) também estão disponíveis (${USER}, ${DATE}, etc.)
