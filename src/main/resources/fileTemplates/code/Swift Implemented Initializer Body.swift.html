Preenche o corpo de inicializadores implementados quando eles são gerados (por exemplo, usando a ação Código | Gerar | Implementar Métodos).
O template é editável. Além de expressões Swift e comentários, você também pode usar variáveis predefinidas, que serão expandidas em seus valores correspondentes.
As variáveis predefinidas aceitam os seguintes valores:
${SIGNATURE} A assinatura do inicializador
Todas as variáveis predefinidas no template Cabeçalho de Arquivo Swift (aba Incluir) também estão disponíveis (por exemplo, ${USER}, ${DATE}, etc.).
