Este template integrado é usado para criar componentes de classe React (por exemplo, por meio da refatoração Extrair Componente). Você pode personalizar este template, por exemplo, usando Component em vez de PureComponent ou alterando as definições de tipo de prop. Você também pode usar variáveis predefinidas. Essas variáveis serão substituídas pelos valores correspondentes quando o template for aplicado. Este template é usado em TypeScript e Flow.
As variáveis predefinidas assumirão os seguintes valores:
${NAME} Nome do componente
${COMPONENT_BODY} Corpo do componente
${PROP_TYPES} Definições de tipos de props
${PROP_TYPES} Valores de prop padrão que o React usará quando nenhum valor de prop for fornecido.
