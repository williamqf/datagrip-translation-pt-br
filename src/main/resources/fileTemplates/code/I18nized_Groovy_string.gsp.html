Este é um modelo incorporado usado para substituir texto em GSP. Exemplo:
<html> <body> Olá! </body> </html>
Quando usado com uma tag personalizada de GSP internacionalizada Exemplo:
<html> <body> <g:message code="HelloKey"/> </body> </html>

Os modelos podem ser editados. Além das expressões que podem ser usadas em um contexto GSP, também pode usar variáveis predefinidas que serão expandidas para seus respectivos valores.

Variáveis predefinidas recebem os seguintes valores:
${PROPERTY_KEY} O nome da chave da propriedade conforme especificado no arquivo de propriedades correspondente.
${PROPERTY_VALUE} O valor da propriedade conforme especificado no arquivo de propriedades correspondente. Este é o valor de seleção do texto original do GSP.
