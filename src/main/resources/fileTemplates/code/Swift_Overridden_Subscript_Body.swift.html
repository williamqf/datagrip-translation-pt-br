Preencha o corpo do Subscript que será gerado quando um Subscript que está sendo sobreescrito for gerado utilizando ações como "Código | Gerar | Substituir Métodos"
Este template é editável. Além de expressões e comentários Swift, você também poderá utilizar variáveis predefinidas, que serão expandidas para seus respectivos valores.
As variáveis predefinidas recebem os seguintes valores:
${CALL_SUPER} Chamada do Subscript super - return super[] com argumentos
${RETURN_TYPE} Tipo de retorno do Subscript que será criado
${DEFAULT_RETURN_VALUE} Valor que será retornado por padrão pelo Subscript
${SIGNATURE} Assinatura do Subscript
${HAS_SET} "true" se o Subscript possuir um setter
${HAS_SUPER_SET} "true" se o Subscript que está sendo sobreescrito possuir um setter
${GETTER_EFFECTS} Efeitos getter para o getter da propriedade (async, throws)
${GET_ATTRIBUTES} Atributos do getter do Subscript
${SET_ATTRIBUTES} Atributos do setter do Subscript
Todas as variáveis predefinidas do template Cabeçalho de Arquivo Swift (guia Incluir) também poderão ser utilizadas (por exemplo, ${USER}, ${DATE}).
