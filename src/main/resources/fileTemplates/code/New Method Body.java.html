Preenche o corpo do método quando ele é gerado (por exemplo, ao aplicar a ação de intenção Criar método a partir do uso).
Este template integrado é editável. Além de expressões Java e comentários, você pode usar variáveis predefinidas que serão expandidas para os valores correspondentes.
As variáveis predefinidas aceitam os seguintes valores:
${RETURN_TYPE}   O tipo de retorno do método criado
${DEFAULT_RETURN_VALUE}   O valor retornado por padrão pelo método
${METHOD_NAME}   O nome do método criado
${CLASS_NAME}   O nome qualificado da classe onde o método é criado
${SIMPLE_CLASS_NAME}   O nome não qualificado da classe onde o método é implementado
Todas as variáveis predefinidas do template de Cabeçalho de arquivo (aba Incluir) também estão disponíveis. (Por exemplo: ${USER}, ${DATE}, etc.)
