Preenche o corpo das propriedades implementadas quando elas são geradas (por exemplo, usando a ação Código | Gerar | Implementar métodos).
O modelo é editável. Além de expressões Swift e comentários, você também pode usar variáveis predefinidas que serão expandidas em seus valores correspondentes.
As variáveis predefinidas aceitam os seguintes valores:
${RETURN_TYPE} O tipo da propriedade criada
${DEFAULT_RETURN_VALUE} O valor de retorno padrão sugerido (pode estar vazio)
${SIGNATURE} A assinatura da propriedade
${HAS_SET} "true" se a propriedade tiver um setter
${GETTER_EFFECTS} Efeitos do getter (async, throws) para o getter da propriedade
${GET_ATTRIBUTES} Os atributos do getter da propriedade
${SET_ATTRIBUTES} Os atributos do setter da propriedade
Todas as variáveis predefinidas no modelo Swift File Header (aba Inclui) também estão disponíveis (por exemplo, ${USER}, ${DATE}, etc.).
