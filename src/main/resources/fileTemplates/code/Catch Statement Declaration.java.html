Preenche o parâmetro catch ao ser gerado, por exemplo, ao chamar a função Código | Envolver com....
Este modelo integrado é editável. Por exemplo, você pode definir o parâmetro gerado como final para que um nome predefinido seja sempre usado, independentemente do nome qualificado da exceção capturada.
As variáveis predefinidas aceitam os seguintes valores:
${EXCEPTION} O nome da variável Exception atribuída como o parâmetro catch derivado do tipo de exceção capturada e, se o nome qualificado terminar com Exception.
${EXCEPTION_TYPE} O tipo do parâmetro catch.
