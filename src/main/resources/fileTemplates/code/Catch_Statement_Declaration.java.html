Preenche o parâmetro catch quando ele é gerado, por exemplo, quando a função Código | Envolver com... é chamada.
Esse template embutido é editável. Por exemplo, você pode fazer o parâmetro gerado ser final e sempre usar o nome pré-definido, a despeito do nome completo da exceção capturada.
Variáveis pré-definidas assumem os seguintes valores:
${EXCEPTION} Nome da variável Exception especificada como um parâmetro catch derivado do tipo da exceção capturada, e, se o nome completo termina com Exception.
${EXCEPTION_TYPE} Tipo do parâmetro catch.
