Um modelo integrado para um método setter PHP. Por exemplo, é usado quando a correção rápida "Adicionar getter e setter" é chamada.
As variáveis predefinidas assumem os seguintes valores:
${STATIC}   Se a propriedade for estática, assume o valor 'static', caso contrário, uma string vazia.
${NAME}   Nome do método no estilo CamelCase, sem o prefixo "set" e com o primeiro caractere em maiúsculo. Por exemplo, o nome da propriedade '_my_property' é convertido para 'MyProperty'.
${CLASS_NAME}   O nome da classe que contém o método.
${SCALAR_TYPE_HINT}   Contém o type hint escalar se o nível de linguagem for PHP 7.0 ou superior.
${FIELD_NAME}   O nome original da propriedade permanece inalterado.
${PARAM_NAME}   O nome do parâmetro do setter. Quase idêntico ao ${FIELD_NAME}, mas remove o primeiro sublinhado (se houver). Por exemplo, o nome da propriedade '_myProperty' é convertido para 'myProperty'.
${TYPE_HINT}   Type hint da propriedade ou uma string vazia se não for especificado.
${VOID_RETURN_TYPE}   Indica se o tipo de retorno void está disponível para o nível de linguagem PHP do projeto.
${DS}   Símbolo de cifrão, avaliado como o caractere '$' comum.
