Um template integrado para o corpo de um método PHP sobrescrito.
As variáveis predefinidas aceitam os seguintes valores:
${NAME} O nome do método.
${PARAM_LIST} Uma lista de parâmetros separados por vírgula, semelhante à do método original.
${RETURN} Se o método original tiver uma instrução return, esta variável conterá a string 'return'; caso contrário, conterá uma string vazia.
