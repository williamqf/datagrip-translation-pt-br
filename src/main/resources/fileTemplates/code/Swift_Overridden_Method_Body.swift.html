Preencha o corpo do método quando métodos sobrepostos forem gerados (por exemplo, usando a ação Código | Gerar | Sobrepor métodos).
O modelo é editável. Além de expressões e comentários Swift, você também pode usar variáveis predefinidas que serão expandidas para os valores correspondentes.
Variáveis predefinidas recebem os seguintes valores:
${CALL_SUPER} Chamada do método super, para métodos Void: super.methodName() com ou sem parâmetros; para outros métodos: return super.methodName() com ou sem parâmetros.
${RETURN_TYPE} Tipo de retorno do método criado
${DEFAULT_RETURN_VALUE} Valor retornado por padrão pelo método
${SIGNATURE} Assinatura do método
Todas as variáveis predefinidas no modelo Cabeçalho do arquivo Swift (guia Inclusões) também podem ser usadas (por exemplo, ${USER}, ${DATE}, etc.).
