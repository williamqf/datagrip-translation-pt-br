Preenche o corpo de um método quando ele é gerado, por exemplo, quando a ação de intenção Criar método a partir do uso for aplicada.
Este modelo interno é editável. Junto com expressões e comentários Java, você também pode usar as variáveis predefinidas que serão então expandidas para os valores correspondentes.
As variáveis predefinidas assumem os seguintes valores:
${RETURN_TYPE} Tipo de retorno do método criado
${DEFAULT_RETURN_VALUE} Valor retornado pelo método por padrão
${METHOD_NAME} Nome do método criado
${CLASS_NAME} Nome qualificado da classe na qual o método é criado
${SIMPLE_CLASS_NAME} Nome não qualificado da classe na qual o método é implementado
Todas as variáveis predefinidas do modelo Cabeçalho do Arquivo (guia Inclusões) também estão disponíveis (por exemplo: ${USER}, ${DATE}, etc.)
