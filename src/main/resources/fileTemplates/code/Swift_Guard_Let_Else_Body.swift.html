Desempacote um valor opcional com uma declaração guard e preencha o bloco else. Este modelo é editável. Além das expressões e comentários Swift, você pode usar variáveis predefinidas que serão expandidas para seus valores correspondentes.
As variáveis predefinidas recebem os seguintes valores:
${RETURN_TYPE} O tipo de retorno da função circundante (se houver)
${DEFAULT_RETURN_VALUE} O valor padrão sugerido para o retorno (pode ser vazio)
Todas as variáveis predefinidas no modelo Cabeçalho do Arquivo Swift (guia Inclusões) também estão disponíveis (por exemplo, ${USER}, ${DATE} e assim por diante).
