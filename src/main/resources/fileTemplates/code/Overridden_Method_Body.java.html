Preenche o corpo de um método sobrescrito sempre que este for gerado pelo programa, por exemplo, quando for chamada a função Código | Sobrescrever Método....
É possível editar este modelo interno. Juntamente com expressões e comentários Java, você também pode usar variáveis predefinidas que serão então expandidas nos valores correspondentes.
As variáveis predefinidas assumem os seguintes valores:
${CALL_SUPER} Chamada do método super. Para métodos void, super.nome_do_método(); com ou sem parâmetros. Para outros métodos, return super.nome_do_método(); com ou sem parâmetros.
${PLAIN_CALL_SUPER} Chamada do método super, super.nome_do_método(); com ou sem parâmetros.
${RETURN_TYPE} Tipo de retorno do método criado
${DEFAULT_RETURN_VALUE} Valor retornado pelo método por padrão
${METHOD_NAME} Nome do método sobrescrito
${CLASS_NAME} Nome qualificado da classe na qual o método é sobrescrito
${SIMPLE_CLASS_NAME} Nome não qualificado da classe na qual o método é implementado
Todas as variáveis predefinidas do modelo Cabeçalho do Arquivo (guia Inclusões) também estão disponíveis (por exemplo, ${USER}, ${DATE}, etc.)
