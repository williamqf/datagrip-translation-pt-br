Um modelo integrado para um método setter PHP. Por exemplo, ele é usado quando a correção rápida "Add getter and setter" é chamada.
As variáveis predefinidas aceitam os seguintes valores:
${NAME} O nome do método no estilo CamelCase, sem o prefixo "set", com o primeiro caractere em maiúscula. Por exemplo, o nome da propriedade "_my_property" é convertido para "MyProperty".
${CLASS_NAME} O nome da classe contentora.
${SCALAR_TYPE_HINT} Contém o type hint escalar se o nível de linguagem for PHP 7.0 ou superior.
${RETURN_TYPE} Indica se o nível da linguagem é PHP 7.0 ou superior, portanto, um tipo de retorno pode ser especificado para a função/método.
${FIELD_NAME} O nome original da propriedade.
${PARAM_NAME} O nome do parâmetro para o setter. Quase idêntico a ${FIELD_NAME}, mas remove o primeiro sublinhado (se houver). Por exemplo, o nome da propriedade "_myProperty" é convertido para "myProperty".
${TYPE_HINT} O type hint da propriedade ou uma string vazia se não for especificado.
${DS} O símbolo de cifrão, avaliado como o caractere "$" comum.
