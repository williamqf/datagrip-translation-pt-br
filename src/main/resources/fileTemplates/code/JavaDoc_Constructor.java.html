Define o template utilizado para gerar JavaDoc para construtores, por exemplo,
quando a ação de intenção Adicionar Javadoc é aplicada.

Juntamente com expressões Java e comentários, você também pode usar as variáveis predefinidas,
que serão expandidas como macros para os valores correspondentes.

Usando a diretiva #parse, você pode incluir templates da guia Inclusões. Para incluir um template, especifique o nome completo do template como um parâmetro entre aspas (por exemplo, #parse("File Header.java")).
Variáveis predefinidas assumem os seguintes valores:
${PARAMS} Lista de nomes de parâmetros
${TYPE_PARAMS} Lista de nomes de parâmetros de tipo
${THROWS} Lista de exceções lançadas
${CONTAINING_CLASS} Nome da classe pai
${PACKAGE_NAME} Nome do pacote no qual uma nova enumeração é criada
${NAME} Nome da nova enumeração especificada por você na caixa de diálogo Criar Nova Classe
${USER} Nome de login do sistema do usuário atual
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${MONTH_NAME_SHORT} Primeiras 3 letras do nome do mês atual (Jan, Fev e assim por diante)
${MONTH_NAME_FULL} Nome completo do mês atual (Janeiro, Fevereiro e assim por diante)
${DAY} Dia atual do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${PROJECT_NAME} Nome do projeto atual
