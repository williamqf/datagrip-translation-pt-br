Substitui texto no JSP, exemplo:
<html><body>
Hello!
</body></html>
com uma tag customizável JSP internacionalizada, exemplo:
<html><body>
<bean:messagekey="HelloKey"/>
</body></html>
Este modelo embutido é editável. Juntamente com expressões permitidas em um contexto JSP, você também pode usar variáveis predefinidas que serão então expandidas com os valores correspondentes.
Variáveis predefinidas recebem os seguintes valores:
${PROPERTY_KEY} Nome da chave da propriedade especificado no arquivo de propriedades correspondente.
${PROPERTY_VALUE} Valor da propriedade especificado no arquivo de propriedades correspondente. Este é um valor de seleção de texto JSP original.
