Cria o corpo de um método -copyWithZone: substituído quando é gerado, por exemplo, ao chamar a ação Código | Gerar | -copyWithZone:.
O modelo é editável. Junto com expressões e comentários Objective-C, você pode usar variáveis predefinidas que então serão expandidas para os valores correspondentes.
As variáveis predefinidas assumirão os seguintes valores:
${CALL_SUPER} uma chamada de método super, para métodos void — [super methodName] com ou sem parâmetros; para outros métodos — return [super methodName] com ou sem parâmetros
${RETURN_TYPE} o tipo de retorno do método criado
${DEFAULT_RETURN_VALUE} o valor retornado pelo método por padrão
${CONTAINING_CLASS} o nome da classe onde o método é substituído
${CUSTOM_CODE} o corpo do método gerado
Todas as variáveis predefinidas do modelo Cabeçalho do Arquivo C (guia Inclusões) também estão disponíveis (${USER}, ${DATE}, etc.)
