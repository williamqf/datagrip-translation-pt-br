Template embutido para os métodos de teste de unidade do Codeception usado, por exemplo, quando a caixa de seleção para gerar métodos é selecionada na caixa de diálogo 'Novo teste'.
As variáveis predefinidas recebem os seguintes valores:
${NAME} Nome do método.
${CAPITALIZED_NAME} Nome do método começando com letra maiúscula.
${TESTED_NAME} Nome da classe testada.
