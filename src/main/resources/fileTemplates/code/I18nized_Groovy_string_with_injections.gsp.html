Esta é uma macro interna, usada para substituição de texto em GSP. Por exemplo:
<html> <body> Olá ${'mundo'}! </body> </html>
Se usarmos uma tag personalizada de GSP internacionalizada, ficaria assim:
<html> <body> <g:message code="HelloKey" args="['mundo']"/> </body> </html> .

O template é editável. Além das expressões autorizadas no contexto do GSP, você pode usar uma variável pré-definida que será expandida para o valor correspondente.

Variáveis pré-definidas recebem os seguintes valores:
${PROPERTY_KEY} O nome da chave de propriedade, conforme especificado no arquivo de propriedades correspondente.
${PROPERTY_VALUE} O valor da propriedade, conforme especificado no arquivo de propriedades correspondente. Este é o valor selecionado do texto GSP original.
${ARGS_KEY} Os argumentos
