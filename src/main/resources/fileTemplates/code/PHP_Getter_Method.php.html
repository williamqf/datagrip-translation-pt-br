Um template interno para métodos getters PHP. É usado, por exemplo, quando a correção rápida "Adicionar getters e setters" é invocada.
As variáveis predefinidas recebem os seguintes valores:
${STATIC} Se a propriedade for estática, recebe o valor 'static', caso contrário é uma string vazia.
${GET_OR_IS} Prefixo 'get/is'. Se uma propriedade tiver um tipo booleano (dica de tipo), é avaliada como 'is', caso contrário é avaliada como a string 'get'.
${NAME} Nome do método no estilo CamelCase sem o prefixo "get", a primeira letra em maiúsculas. Por exemplo, o nome da propriedade '_my_property' é convertido para 'MyProperty'.
${CLASS_NAME} Nome da classe que contém.
${RETURN_TYPE} Contém a dica de tipo de retorno se o nível de linguagem for PHP 7.0 ou superior.
${FIELD_NAME} O nome da propriedade original preservado como está.
${TYPE_HINT} A dica de tipo da propriedade, ou uma string vazia se não for especificada.
${DS} O cifrão é avaliado como o caractere '$' normal.
