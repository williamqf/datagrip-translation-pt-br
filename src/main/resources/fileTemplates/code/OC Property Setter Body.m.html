Cria o corpo de um método setter sobrescrito para uma propriedade quando gerado (por exemplo, ao chamar a ação Código | Sobrescrever Métodos).
O modelo é editável. Além de expressões e comentários em Objective-C, você também pode usar variáveis predefinidas que serão expandidas para seus valores correspondentes.
As variáveis predefinidas aceitam os seguintes valores:
${CALL_SUPER} Chamada do método super: para métodos void — [super methodName], com ou sem parâmetros; para outros métodos — return [super methodName], com ou sem parâmetros
${RETURN_TYPE} O tipo de retorno do método criado
${DEFAULT_RETURN_VALUE} O valor retornado por padrão pelo método
${CONTAINING_CLASS} O nome da classe na qual o método está sendo sobrescrito
${IVAR_IS_AVAILABLE} "true" quando a propriedade possui uma variável de instância
${IVAR} O nome da variável de instância
${PARAM} O nome do parâmetro do setter
${SETTER_SEMANTICS} No modo ARC: "copyArc" se o atributo copy estiver definido, caso contrário, "assign"
No modo não ARC: "copy" se o atributo copy estiver definido, "retain" se o atributo retain estiver definido, caso contrário, "assign"
Todas as variáveis predefinidas do modelo Cabeçalho de Arquivo C (aba Includes) também estão disponíveis (${USER}, ${DATE}, etc.).
