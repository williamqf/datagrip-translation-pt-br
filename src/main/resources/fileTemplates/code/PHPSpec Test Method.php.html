Um modelo integrado para métodos de teste PHPSpec, usado quando a caixa de seleção para gerar métodos é selecionada, por exemplo, na caixa de diálogo "Novo Teste".
As variáveis predefinidas aceitarão os seguintes valores:
${NAME} O nome do método.
${CAPITALIZED_NAME} O nome do método com a primeira letra em maiúsculo.
${TESTED_NAME} O nome da classe que está sendo testada.
