Modelo básico para o método de teste funcional do Codeception utilizado, por exemplo, quando a caixa de seleção para gerar métodos é selecionada na caixa de diálogo "Novo Teste".
As variáveis pré-definidas recebem os seguintes valores:
${NAME} Nome do método.
${CAPITALIZED_NAME} Nome do método com a primeira letra maiúscula.
${TESTED_NAME} Nome da classe testada.
