Modelo interno para métodos setter PHP. Por exemplo, é usado quando a correção rápida "Adicionar getters e setters" é invocada.
Variáveis predefinidas assumem os seguintes valores:
${STATIC} Assumirá o valor 'static' se a propriedade for estática, caso contrário, assumirá uma string vazia.
${NAME} Nome do método em estilo CamelCase sem o prefixo "set" e com a primeira letra em maiúscula. Por exemplo, o nome da propriedade '_my_property' é convertido para 'MyProperty'.
${CLASS_NAME} Nome da classe que contém.
${SCALAR_TYPE_HINT} Contém a sugestão de tipo escalar se o nível da linguagem for PHP 7.0 ou posterior
${FIELD_NAME} Nome da propriedade original permanece inalterado.
${PARAM_NAME} Nome do parâmetro do setter. É quase igual ao ${FIELD_NAME}, porém o sublinhado inicial é removido, se houver. Por exemplo, o nome da propriedade '_myProperty' é convertido para 'myProperty'.
${TYPE_HINT} Sugestão de tipo para a propriedade. É uma string vazia se não for especificada.
${VOID_RETURN_TYPE} Indica se o tipo de retorno void pode ser usado no nível da linguagem PHP do projeto.
${DS} O cifrão é avaliado como um caractere '$' comum.
