Este modelo interno é usado para preencher o corpo de um método quando gerado, por exemplo, através da ação de intenção Criar método pelo uso.
O modelo é editável. Além de comentários e expressões TypeScript, você também pode usar variáveis pré-definidas. Estas variáveis serão substituídas por seus valores correspondentes quando o modelo for aplicado.
Variáveis pré-definidas terão os seguintes valores:
${RETURN_TYPE} tipo de retorno do método criado
${DEFAULT_RETURN_VALUE} valor retornado pelo método por padrão
${METHOD_NAME} nome do método criado
${SEMICOLON} ponto e vírgula ou vazio, com base nas opções de estilo de código
Todas as variáveis pré-definidas do modelo Cabeçalho do Arquivo (Guia Inclusões) também estão disponíveis (ex: ${USER}, ${DATE}, etc.)
