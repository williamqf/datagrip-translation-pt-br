Um template integrado para um método de teste funcional do Codeception, usado quando, por exemplo, a caixa de seleção para métodos gerados é selecionada na caixa de diálogo "Novo Teste".
As variáveis predefinidas recebem os seguintes valores:
${NAME}   O nome do método.
${CAPITALIZED_NAME}   O nome do método com o primeiro caractere em maiúsculo.
${TESTED_NAME}   O nome da classe testada.
