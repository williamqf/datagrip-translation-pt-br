Cria o corpo do método -description sobrescrito ao ser gerado (por exemplo, ao chamar a ação Código | Gerar | -description).
O template é editável. Além de expressões Objective-C e comentários, você pode usar variáveis predefinidas que serão expandidas para seus respectivos valores.
As variáveis predefinidas assumem os seguintes valores:
${CALL_SUPER} Chamada do super método; para métodos void — [super methodName], com ou sem parâmetros; para outros métodos — return [super methodName], com ou sem parâmetros
${RETURN_TYPE} O tipo de retorno do método criado
${DEFAULT_RETURN_VALUE} O valor retornado por padrão pelo método
${CONTAINING_CLASS} O nome da classe na qual o método está sendo sobrescrito
${CUSTOM_CODE} O corpo do método gerado
Todas as variáveis predefinidas no template Cabeçalho de arquivo C (aba Includes) também estão disponíveis (${USER}, ${DATE}, etc.)
