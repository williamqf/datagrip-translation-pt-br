Define o template usado para gerar JavaDoc para campos, por exemplo, ao aplicar a ação de intenção Adicionar Javadoc.

Além de expressões Java e comentários, você também pode usar variáveis predefinidas que serão expandidas para seus valores correspondentes como macros.

Ao usar a diretiva #parse, é possível incluir templates da aba Inclusões. Para incluir um template, especifique o nome completo do template como um parâmetro entre aspas (por exemplo, #parse("File Header.java")).
As variáveis predefinidas aceitam os seguintes valores:
${ELEMENT_NAME} Nome do membro da classe que é o proprietário do JavaDoc gerado
${CONTAINING_CLASS} Nome da classe pai
${PACKAGE_NAME} O nome do pacote no qual a nova enumeração será criada
${NAME} O nome da nova enumeração que você especificou na caixa de diálogo Criar Nova Classe
${USER} Nome de login de sistema do usuário atual
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${MONTH_NAME_SHORT} As primeiras 3 letras do nome do mês atual (Jan, Fev, etc.)
${MONTH_NAME_FULL} Nome completo do mês atual (Janeiro, Fevereiro, etc.)
${DAY} Dia do mês atual
${HOUR} Hora atual
${MINUTE} Minuto atual
${PROJECT_NAME} Nome do projeto atual
