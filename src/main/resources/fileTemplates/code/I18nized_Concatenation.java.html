Substitui uma concatenação de strings (por exemplo: String s = "Hello " + name;) por uma expressão de código Java internacionalizada (por exemplo: String s = MessageFormat.format(​myResourceBundle.getString(​"hello.0"), name);).
Este é um modelo integrado que pode ser editado. Além das expressões Java e de comentários, também é possível utilizar as variáveis predefinidas que serão então expandidas para os valores correspondentes.
As variáveis predefinidas recebem os seguintes valores:
${RESOURCE_BUNDLE} Expressão do tipo java.util.ResourceBundle, cuja instância está disponível neste contexto.
${PARAMETERS} Variáveis usadas na concatenação de strings, geralmente passadas como argumentos para a expressão internacionalizada.
${PROPERTY_KEY} Nome da chave de propriedade definida no arquivo de propriedades correspondente. Geralmente é o valor usado para o parâmetro do método ResourceBundle.getString().
${PROPERTY_VALUE} Valor da propriedade definido no arquivo de propriedades correspondente. É o valor literal original da string Java.
