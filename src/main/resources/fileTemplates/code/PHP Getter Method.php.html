Um modelo integrado para um método getter PHP. Por exemplo, ele é utilizado quando a correção rápida (quick fix) "Add getter and setter" é acionada.
As variáveis predefinidas aceitam os seguintes valores:
${STATIC}   Assume o valor 'static' se a propriedade for estática; caso contrário, uma string vazia.
${GET_OR_IS}   Prefixo 'get/is'. Se uma propriedade possuir um tipo booleano (type hint), avalia para 'is'; caso contrário, avalia para a string 'get'.
${NAME}   O nome do método no estilo CamelCase, sem o prefixo "get", com o primeiro caractere em maiúscula. Por exemplo, o nome da propriedade '_my_property' é convertido para 'MyProperty'.
${CLASS_NAME}   O nome da classe de contenção.
${RETURN_TYPE}   Contém o type hint de retorno se o nível da linguagem for PHP 7.0 ou superior.
${FIELD_NAME}   O nome original da propriedade, mantido como está.
${TYPE_HINT}   O type hint da propriedade ou uma string vazia se não for especificado.
${DS}   O sinal de cifrão, avaliado como um caractere '$' comum.
