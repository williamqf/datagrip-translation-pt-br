Gera a ramificação default para expressões ou instruções switch usadas em ações de correção rápida, como Inserir ramificação 'default'.
Este template integrado é editável. Por exemplo, você pode alterar a exceção lançada ou chamar um framework de log em seu lugar.
As variáveis predefinidas aceitam os seguintes valores:
${EXPRESSION} Texto da expressão seletora do switch
${EXPRESSION_TYPE} Tipo da expressão seletora do switch
