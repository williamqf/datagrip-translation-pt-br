Define o modelo usado para gerar JavaDoc para métodos, por exemplo, ao aplicar a ação de intenção Adicionar Javadoc.

Além de expressões Java e comentários, você pode usar variáveis predefinidas, que serão expandidas para os valores correspondentes como macros.

Usando a diretiva #parse, você pode incluir modelos da aba Includes. Para incluir um modelo, especifique o nome completo do modelo como um parâmetro entre aspas (por exemplo, #parse("File Header.java")).
As variáveis predefinidas aceitam os seguintes valores:
${PARAMS} Lista de nomes de parâmetros
${TYPE_PARAMS} Lista de nomes de parâmetros de tipo
${THROWS} Lista de exceções lançadas
${RETURN_TYPE} O tipo retornado pelo método
${ELEMENT_NAME} O nome do membro da classe que é o proprietário do JavaDoc gerado
${CONTAINING_CLASS} O nome da classe pai
${PACKAGE_NAME} O nome do pacote no qual a nova enumeração é criada
${NAME} O nome da nova enumeração que você especificou no diálogo Criar Nova Classe
${USER} Nome de login de sistema do usuário atual
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${MONTH_NAME_SHORT} As primeiras 3 letras do nome do mês atual (Jan, Fev, etc.)
${MONTH_NAME_FULL} O nome completo do mês atual (Janeiro, Fevereiro, etc.)
${DAY} Dia do mês atual
${HOUR} Hora atual
${MINUTE} Minuto atual
${PROJECT_NAME} Nome do projeto atual
