Este template embutido é usado para preenchimento do corpo de um método sobreposto toda vez que ele é gerado, por exemplo, quando a função Código | Sobrescrever método... é chamada.
O template é editável. Junto com expressões e comentários JavaScript, você também pode usar variáveis predefinidas. Estas variáveis serão substituídas pelos valores correspondentes quando o template for aplicado.
Variaveis predefinidas terão os seguintes valores:
${CHAMADA_SUPER} uma chamada de método super, para métodos void - super.nome_do_metodo(); com ou sem parâmetros; para outros métodos - return super.nome_do_metodo(); com ou sem parâmetros
${CHAMADA_PLANA_SUPER} uma chamada de método super, super.nome_do_metodo(); com ou sem parâmetros;
${TIPO_RETORNO} tipo de retorno de um método criado
${VALOR_RETORNO_PADRAO} valor retornado pelo método por padrão
${NOME_DO_METODO} nome do método que está sendo sobreposto
Todas as variáveis predefinidas do template Cabeçalho do Arquivo (aba Inclusões) também estão disponíveis (por exemplo: ${USUARIO}, ${DATA}, etc.)
