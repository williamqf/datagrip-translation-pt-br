Gera o branch padrão para a expressão ou declaração switch usado em ações de correção rápida como Inserir branch "padrão".
Este template nativo é editável. Por exemplo, você pode alterar a exceção lançada ou chamar um framework de registro.
Variáveis predefinidas assumem os seguintes valores:
${EXPRESSÃO} Texto da expressão do seletor do switch
${TIPO_EXPRESSÃO} Tipo de expressão do seletor do switch
