Preencha o corpo do construtor implementado quando ele foi gerado por uma ação como código | gerar | implementar métodos.
O modelo é editável. Você pode usar expressões e comentários do Swift além de variáveis predefinidas que serão expandidas para os valores correspondentes.
As variáveis predefinidas recebem estes valores:
${SIGNATURE} Assinatura do construtor
Todas as variáveis predefinidas do modelo Cabeçalho de arquivo Swift (guia Incluir) também estão disponíveis (por exemplo: ${USER}, ${DATE} etc.).
