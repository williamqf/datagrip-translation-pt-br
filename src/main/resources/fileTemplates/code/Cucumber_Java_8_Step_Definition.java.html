Cria uma definição de passo de Cucumber para arquivo Java.
As variáveis pré-definidas assumem os seguintes valores:
${STEP_KEYWORD} A annotation para definição do passo. Geralmente, as palavras-chave são usadas para chamar passos (Given, Then, And, etc.)
${STEP_REGEXP} A expressão regular gerada para aceitação do passo.
${PARAMETERS} Os parâmetros para o método gerado.
${BODY} O corpo do método gerado.
