Modelo embutido para métodos setter da classe PHP. Por exemplo, quando a correção rápida "adicionar getter e setter" é chamada.
Variáveis predefinidas recebem os seguintes valores:
${NAME} Nome do método no estilo CamelCase, sem o prefixo "set" e com a primeira letra maiúscula. Por exemplo, o nome da propriedade '_my_property' é convertido em 'MyProperty'.
${CLASS_NAME} Nome da classe que inclui.
${SCALAR_TYPE_HINT} Inclui dicas de tipo escalar se o nível da linguagem for PHP 7.0 ou superior.
${RETURN_TYPE} Indica se o nível da linguagem é PHP 7.0 ou superior e, portanto, o tipo de retorno pode ser especificado para a função/método.
${FIELD_NAME} Nome da propriedade original mantido como está.
${PARAM_NAME} Nome do parâmetro do setter. Quase igual ao ${FIELD_NAME}, mas remove o sublinhado inicial (se houver). Por exemplo, o nome da propriedade '_myProperty' é convertido em 'myProperty'.
${TYPE_HINT} Dicas de tipo da propriedade. Se não for especificado, será uma string vazia.
${DS} O cifrão do dólar é avaliado como o caractere '$' comum.
