Este é um modelo integrado usado para substituir texto em GSP, por exemplo:
<html> <body> Hello! </body> </html>
pelo uso de tags personalizadas GSP internacionalizadas, por exemplo:
<html> <body> <g:message code="HelloKey"/> </body> </html>.

O modelo é editável. Além das expressões permitidas no contexto GSP, você pode usar variáveis predefinidas que serão expandidas para os valores correspondentes.

As variáveis predefinidas aceitarão os seguintes valores:
${PROPERTY_KEY} O nome da chave de propriedade especificada no arquivo de propriedades correspondente.
${PROPERTY_VALUE} O valor da propriedade especificado no arquivo de propriedades correspondente. Este é o valor da seleção de texto GSP original.
