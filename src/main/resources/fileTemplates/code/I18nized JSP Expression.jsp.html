Substitui o texto em JSP, por exemplo:
<html><body>
Hello!
</body></html>
por tags personalizadas de JSP internacionalizadas, por exemplo:
<html><body>
<bean:messagekey="HelloKey"/>
</body></html>
Este modelo integrado é editável. Além das expressões permitidas no contexto JSP, você pode usar variáveis predefinidas que serão expandidas para os valores correspondentes.
As variáveis predefinidas aceitam os seguintes valores:
${PROPERTY_KEY} O nome da chave da propriedade especificada no arquivo de propriedades correspondente.
${PROPERTY_VALUE} O valor da propriedade especificado no arquivo de propriedades correspondente. Este é o valor da seleção de texto JSP original.
