Um modelo integrado para um construtor de classe PHP.
Variáveis predefinidas aceitam os seguintes valores:
${PARAM_LIST} Lista de parâmetros do construtor separados por vírgula, correspondente a cada propriedade selecionada.
${PARAM_DOC} Comentários PHPDoc para os parâmetros.
Gerado como uma série de linhas '* @param tipo nome'. Se não houver parâmetros, será avaliado como conteúdo vazio.
${THROWS_DOC} Comentários PHPDoc para exceções.
Gerado como uma série de linhas '* @throws tipo'. Se nenhuma exceção for lançada, será avaliado como conteúdo vazio.
${CLASS_NAME} O nome da classe.
${BODY} O corpo do construtor contendo as atribuições '$this->property=$property;'.
