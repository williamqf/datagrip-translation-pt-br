Cria o corpo de um método -init substituído quando é gerado, por exemplo, ao chamar o Código | Substituir Método ou Código | Gerar | -initWith.
O modelo é editável. Juntamente com expressões e comentários de Objective-C, você também pode usar variáveis predefinidas que então serão expandidas para os valores correspondentes.
As variáveis predefinidas assumirão os seguintes valores:
${CALL_SUPER} uma chamada ao super método, para os métodos void — [super methodName] com ou sem parâmetros; para outros métodos — return [super methodName] com ou sem parâmetros
${RETURN_TYPE} o tipo de retorno do método criado
${DEFAULT_RETURN_VALUE} o valor retornado pelo método por padrão
${CONTAINING_CLASS} o nome da classe em que o método é substituído
${CUSTOM_CODE} o corpo do método
Todas as variáveis predefinidas do modelo Cabeçalho de Arquivo C (a guia Inclusões) também estão disponíveis (${USER}, ${DATE}, etc.)
