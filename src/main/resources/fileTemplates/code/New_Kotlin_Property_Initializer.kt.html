Preencha o inicializador de uma propriedade Kotlin sempre que ela for gerada pelo programa (por exemplo, quando a ação de intenção Criar Propriedade pelo Uso for usada). Esse template é editável. Além das expressões e comentários em Kotlin, é possível usar variáveis predefinidas que serão expandidas para seus respectivos valores.
As variáveis predefinidas assumem os seguintes valores:
${RETURN_TYPE} Tipo da propriedade que está sendo criada
${PROPERTY_NAME} Nome da propriedade que está sendo criada
${CLASS_NAME} Nome qualificado da classe na qual a propriedade está sendo criada
${SIMPLE_CLASS_NAME} Nome não qualificado da classe na qual a propriedade está sendo criada
