Substitui a concatenação de strings (ex: String s = "Olá " + name;) por uma expressão de código Java internacionalizada (ex: String s = MessageFormat.format(​myResourceBundle.getString(​"hello.0"), name);).
Este template integrado é editável. Além de expressões e comentários Java, você também pode usar variáveis predefinidas, que serão expandidas para seus valores correspondentes.
As variáveis predefinidas aceitam os seguintes valores:
${RESOURCE_BUNDLE} Uma expressão do tipo java.util.ResourceBundle, cujas instâncias estão disponíveis neste contexto.
${PARAMETERS} Variáveis usadas na concatenação de strings, geralmente passadas como argumentos para a expressão internacionalizada.
${PROPERTY_KEY} O nome da chave de propriedade definida no arquivo de propriedades correspondente. Geralmente, é o valor usado para o parâmetro do método ResourceBundle.getString().
${PROPERTY_VALUE} O valor da propriedade definido no arquivo de propriedades correspondente. Este é o valor original do literal de string Java.
