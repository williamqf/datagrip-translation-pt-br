Este template integrado é usado para preencher o corpo do método sempre que um método sobrescrito for gerado (por exemplo, ao utilizar a funcionalidade Código | Sobrescrever Métodos...).
O template é editável. Além de expressões JavaScript e comentários, você também pode usar variáveis predefinidas. Essas variáveis serão substituídas pelos valores correspondentes quando o template for aplicado.
As variáveis predefinidas aceitam os seguintes valores:
${CALL_SUPER} Chamada do método super; para métodos void - super.method_name(); com ou sem parâmetros; para outros métodos - return super.method_name(); com ou sem parâmetros
${PLAIN_CALL_SUPER} Chamada do método super, super.method_name(); com ou sem parâmetros;
${RETURN_TYPE} O tipo de retorno do método criado
${DEFAULT_RETURN_VALUE} O valor retornado por padrão pelo método
${METHOD_NAME} O nome do método que está sendo sobrescrito
Todas as variáveis predefinidas do template de Cabeçalho do Arquivo (aba Incluir) também estão disponíveis. (Por exemplo: ${USER}, ${DATE}, etc.)
