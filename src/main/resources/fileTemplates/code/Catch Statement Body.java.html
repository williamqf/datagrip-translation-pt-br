Preenche o corpo do bloco catch quando ele é gerado, por exemplo, ao aplicar a função Código | Envolver com....
Este template integrado é editável. Além de expressões Java e comentários, você também pode usar variáveis predefinidas que serão expandidas para os valores correspondentes.
As variáveis predefinidas aceitam os seguintes valores:
${EXCEPTION} O nome da variável Exception especificada como parâmetro do catch
${EXCEPTION_TYPE} O tipo do parâmetro do catch
