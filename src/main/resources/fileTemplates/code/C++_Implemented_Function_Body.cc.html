Cria o corpo de uma função implementada quando ela é gerada, por exemplo, usando a ação Código | Implementar Métodos.
Ao editar o modelo, você pode usar expressões C++, comentários e as seguintes variáveis predefinidas:
${RETURN_TYPE} o tipo de retorno da função.
${DEFAULT_RETURN_VALUE} o valor de retorno padrão da função.
Todas as variáveis predefinidas do modelo Cabeçalho do Arquivo C (a guia Inclusões), como ${USER} e ${DATE}, também estão disponíveis.
