Este é um modelo integrado usado para preencher o corpo do método sempre que um método Groovy for gerado pelo programa (por exemplo, ao usar a ação de intenção Criar método a partir do uso).
O modelo é editável. Além de expressões Groovy e comentários, você também pode usar variáveis predefinidas que serão expandidas para os valores correspondentes.
As variáveis predefinidas aceitam os seguintes valores:
${RETURN_TYPE}   O tipo de retorno do método criado
${DEFAULT_RETURN_VALUE}   O valor retornado por padrão pelo método
${METHOD_NAME}   O nome do método criado
${CLASS_NAME}   O nome qualificado da classe na qual o método é criado
${SIMPLE_CLASS_NAME}   O nome não qualificado da classe na qual o método é implementado
