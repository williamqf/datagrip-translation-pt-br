Gera a declaração do método +sharedInstance na seção de cabeçalho da classe, por exemplo, ao chamar a ação Código | Gerar | +sharedInstance.
O modelo é editável. Além de expressões Objective-C e comentários, você também pode usar variáveis predefinidas que serão expandidas para os valores correspondentes.
As variáveis predefinidas aceitam os seguintes valores:
${$CLASS_TYPE} O nome da classe na qual o método é implementado
Todas as variáveis predefinidas no modelo Cabeçalho de Arquivo C (aba Incluir) também estão disponíveis (${USER}, ${DATE}, etc.)
