Um template embutido para um arquivo ERB. Ele é usado toda vez que você cria um novo arquivo ERB selecionando Novo | Arquivo ERB no menu pop-up numa das views do projeto. Juntamente com comandos de Ruby, HTML e comentários, você também pode usar as variáveis predefinidas listadas abaixo. Cada variável é expandida em macros para o valor correspondente. Por meio da diretriz #parse, você pode incluir templates da guia Inclusions especificando o nome completo do template desejado como um parâmetro entre aspas.
Variáveis predefinidas tomam os seguintes valores:
${FILE_NAME} Nome do arquivo atual
${USER} Nome do usuário de login no sistema
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${DAY} Dia atual do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${PRODUCT_NAME} Nome do IDE atual
${PROJECT_NAME} Nome do projeto atual
