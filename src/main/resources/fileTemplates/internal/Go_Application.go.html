Aplicação Go é um modelo padrão usado toda vez que um novo arquivo Go é criado. Ao contrário do modelo Arquivo Go, o Aplicativo Go possui uma função principal predefinida. Para usar esse modelo, abra o Arquivo | Novo, selecione Arquivo Go na categoria Go e clique em Aplicativo simples.

O modelo é editável. Nele é possível usar as seguintes variáveis predefinidas. Essas variáveis são expandidas posteriormente no editor para seus respectivos valores.

Também é possível especificar variáveis personalizadas. As variáveis personalizadas usam o formato ${NOME_DA_VARIÁVEL}, em que NOME_DA_VARIÁVEL é o nome da variável (como ${NOME_DA_MINHA_FUNÇÃO_PERSONALIZADA}). Antes de o IDE usar variáveis personalizadas para criar um novo arquivo, uma caixa de diálogo é exibida, em que é possível definir os valores das variáveis personalizadas no modelo.

Use a diretiva #parse para incluir modelos da guia Incluir. Para incluir um modelo, especifique o nome completo do modelo entre aspas como parâmetro (como #parse("File Header")).
Lista de variáveis predefinidas
${GO_PACKAGE_NAME_WITH_TEST_SUFFIX} Nome do pacote em que um novo arquivo de teste é criado. O arquivo deve ter o sufixo _test. O nome usa o formato pacote_teste.
${GO_PACKAGE_NAME} Nome do pacote em que um novo arquivo é criado.
${DATE} Atual data do sistema.
${DAY} Dia do mês atual.
${DAY_NAME_SHORT} As três primeiras letras do dia da semana atual (como Seg, Ter, etc.).
${DAY_NAME_FULL} Nome completo do dia da semana atual (como segunda-feira, terça-feira etc.).
${DIR_PATH} Caminho do diretório em que o novo arquivo é armazenado. Esse caminho é relativo à raiz do projeto.
${DS} Cifrão ($). Essa variável é usada para escapar o caractere de cifrão para que ele não seja interpretado como o prefixo de uma variável de modelo.
${FILE_NAME} Nome do novo arquivo.
${HOUR} Hora atual.
${MINUTE} Minuto atual.
${SECOND} Segundo atual.
${MONTH} Mês atual.
${MONTH_NAME_SHORT} As três primeiras letras do mês atual (como jan, fev etc.).
${MONTH_NAME_FULL} Nome completo do mês atual (como janeiro, fevereiro etc.).
${NAME} Nome da nova entidade (como arquivo, tipo, interface etc.).
