Modelo embutido para um novo interface PHP. O mesmo é utilizado sempre que se cria um novo interface PHP ao escolher Novo | Classe PHP no menu pop-up em qualquer projeto.
O modelo pode ser editado. Além das expressões e comentários em PHP, você também pode utilizar as seguintes variáveis pré-definidas listadas abaixo. Cada variável é expandida para seu respectivo valor da mesma forma que macros.
Usando diretivas #parse, é possível incluir modelos localizados na guia Includes, especificando o nome completo do modelo desejado entre aspas como parâmetro.
As variáveis pré-definidas assumem os seguintes valores.:
${FILE_NAME} Nome do arquivo atual.
${USER} Nome de usuário atual no sistema.
${DATE} Data atual do sistema.
${TIME} Hora atual do sistema.
${YEAR} Ano atual.
${MONTH} Mês atual.
${DAY} Dia do mês atual.
${HOUR} Hora atual.
${MINUTE} Minuto atual.
${PRODUCT_NAME} Nome do IDE atual.
${DS} O cifrão é avaliado como o caractere '$' comum.
${PROJECT_NAME} Nome do projeto atual.
