Aplicável a novas classes Java criadas ao invocar Novo | Classe Java | Classe na janela de ferramentas Projeto.
Este modelo integrado é editável. Além de expressões Java e comentários, você também pode usar variáveis predefinidas (listadas abaixo), que serão expandidas para seus valores correspondentes como macros.

Você também pode especificar variáveis personalizadas. Variáveis personalizadas usam o seguinte formato: ${VARIABLE_NAME}, onde VARIABLE_NAME é o nome da variável (por exemplo, ${MY_CUSTOM_FUNCTION_NAME}). Antes que a IDE crie um novo arquivo usando variáveis personalizadas, um diálogo será exibido para que você possa definir os valores das variáveis personalizadas no modelo.

Usando a diretiva #parse, você pode incluir modelos da aba Incluir. Para incluir um modelo, especifique o nome completo do modelo entre aspas como um parâmetro (por exemplo, #parse("File Header.java")).
As variáveis predefinidas aceitam os seguintes valores:
${PACKAGE_NAME} O nome do pacote no qual a nova classe é criada
${NAME} O nome da nova classe que você especificou no diálogo Criar Nova Classe
${USER} O nome de login de sistema do usuário atual
${DATE} A data atual do sistema
${TIME} A hora atual do sistema
${YEAR} O ano atual
${MONTH} O mês atual
${MONTH_NAME_SHORT} As primeiras 3 letras do nome do mês atual (Jan, Feb, etc.)
${MONTH_NAME_FULL} O nome completo do mês atual (January, February, etc.)
${DAY} O dia atual do mês
${HOUR} A hora atual
${MINUTE} O minuto atual
${PROJECT_NAME} O nome do projeto atual
