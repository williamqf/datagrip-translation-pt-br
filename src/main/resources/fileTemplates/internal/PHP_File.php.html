Modelo padrão para arquivos PHP. É usado sempre que você cria um novo arquivo PHP selecionando Novo | Arquivo PHP no menu pop-up em uma das visualizações de projeto.
O modelo é editável. Além de expressões, instruções e comentários PHP, você também pode usar as variáveis predefinidas descritas abaixo. Cada variável se expande para seu valor correspondente como uma macro.
Você pode usar a diretiva #parse para incluir modelos da guia Inclusões especificando o nome completo do modelo desejado como um parâmetro entre aspas.
As variáveis predefinidas assumem os seguintes valores:
${FILE_NAME} O nome do arquivo atual
${USER} O nome de login do sistema do usuário atual
${DATE} A data atual do sistema
${TIME} A hora atual do sistema
${YEAR} O ano atual
${MONTH} O mês atual
${DAY} O dia do mês atual
${HOUR} A hora atual
${MINUTE} O minuto atual
${PRODUCT_NAME} O nome do IDE atual
${PROJECT_NAME} O nome do projeto atual
