Um template embutido para um arquivo Liquid. É utilizado toda vez que você cria um novo arquivo Liquid selecionando Novo | Arquivo Liquid no menu de contexto em uma das vistas do projeto. Além de usar comandos, expressões e comentários do Liquid, você também pode usar as variáveis predefinidas listadas a seguir. Cada variável é expandida como uma macro para o valor correspondente. Por meio da diretiva #parse, é possível incluir template da guia Inclusões especificando o nome completo do template desejado como um parâmetro entre aspas.
As variáveis predefinidas assumem os seguintes valores:
${FILE_NAME} Nome do arquivo atual
${USER} Nome de login do sistema do usuário atual
${DATE} Data do sistema atual
${TIME} Hora do sistema atual
${YEAR} Ano atual
${MONTH} Mês atual
${DAY} Dia atual do mês
${HOUR} Hora Atual
${MINUTE} Minuto atual
${PRODUCT_NAME} Nome do IDE atual
${PROJECT_NAME} Nome do projeto atual
