Um template interno para um módulo Ruby. Ele é usado sempre que você cria um novo módulo Ruby selecionando Novo | Arquivo/Classe Ruby a partir do menu popup em uma das visualizações do projeto. Junto com instruções Ruby, expressões e comentários, você também pode usar as variáveis pré-definidas listadas abaixo. Toda variável é expandida, como macros, para o valor correspondente. Por meio da diretiva #parse, você pode incluir templates da aba Inclusões especificando o nome completo do template desejado como parâmetro entre aspas.
As variáveis pré-definidas recebem os seguintes valores:
${FILE_NAME} Nome do arquivo atual
${USER} Nome de login do usuário atual no sistema
${DATE} Data atual do sistema
${TIME} Horário atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${DAY} Dia atual do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${PRODUCT_NAME} Nome atual do IDE
${PROJECT_NAME} Nome do projeto atual
