Um modelo padrão para PHP Traits. Este é usado sempre que você criar um novo PHP Trait, selecionando Novo | PHP Classe a partir do menu de contexto em qualquer um dos modos de exibição de projeto.
O modelo é editável. Além de instruções, expressões e comentários do PHP, você também pode usar as variáveis predefinidas listadas abaixo. Cada variável será expandida para o seu valor correspondente, assim como uma macro.
O direcionamento #parse permite incluir modelos localizados na aba Incluir, especificando o nome completo do modelo desejado como um parâmetro entre aspas.
As variáveis predefinidas recebem os seguintes valores:
${FILE_NAME} Nome do arquivo atual
${USER} Nome de acesso do usuário atual
${DATE} Data atual do sistema
${TIME} Horário atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${DAY} Dia do mês atual
${HOUR} Hora atual
${MINUTE} Minuto atual
${PRODUCT_NAME} Nome do IDE atual
${DS} O cifrão é avaliado como o caractere '$' comum.
${PROJECT_NAME} Nome do projeto atual
