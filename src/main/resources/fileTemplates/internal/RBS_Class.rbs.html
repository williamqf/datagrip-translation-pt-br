Um modelo pré-definido para uma classe de RBS. Ele é usado toda vez que você cria uma nova classe de RBS, selecionando Novo | Arquivo/Classe de RBS no menu pop-up em uma das visualizações do projeto. Junto com as declarações, expressões e comentários de RBS, você também pode usar as variáveis pré-definidas abaixo. Cada variável é expandida como macros, no valor correspondente. Por meio da diretiva #parse, você pode incluir modelos da guia Inclusões, especificando o nome completo do modelo desejado como um parâmetro entre aspas.
As variáveis pré-definidas assumem os seguintes valores:
${FILE_NAME} Nome do arquivo atual
${USER} Nome do login do usuário atual
${DATE} Data atual do sistema
${TIME} Horário atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${DAY} Dia atual do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${PRODUCT_NAME} Nome do IDE atual
${PROJECT_NAME} Nome do projeto atual
