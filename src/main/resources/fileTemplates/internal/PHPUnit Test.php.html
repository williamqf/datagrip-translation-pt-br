Um template integrado para testes PHPUnit para a versão 5 ou anterior. Ele é usado sempre que você cria um novo teste PHPUnit. Selecione Novo | Teste PHP no menu de contexto de uma das visualizações do projeto e escolha 'PHPUnit < 6' como a biblioteca de testes.
O template é editável. Além de instruções, expressões e comentários PHP, você também pode usar as variáveis predefinidas listadas abaixo. Cada variável é expandida para o valor correspondente como uma macro.
Através da diretiva #parse, é possível incluir templates da aba Includes especificando o nome completo do template desejado como um parâmetro entre aspas.
As variáveis predefinidas aceitam os seguintes valores:
${TESTED_NAME} Nome da classe que está sendo testada
${TESTED_NAMESPACE} Namespace da classe que está sendo testada
${NAME} Nome da classe de teste
${NAMESPACE} Namespace da classe de teste
${FILE_NAME} Nome do arquivo atual
${USER} Nome de login do usuário atual no sistema
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${DAY} Dia do mês atual
${HOUR} Hora atual
${MINUTE} Minuto atual
${PRODUCT_NAME} Nome do IDE atual
${DS} Símbolo de cifrão, avaliado como um caractere '$' comum.
${PROJECT_NAME} Nome do projeto atual
