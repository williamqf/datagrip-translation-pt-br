Aplica-se ao novo arquivo package-info.java criado ao chamar Novo | package-info.java na janela de ferramentas Projeto.

Este modelo integrado é editável. Além de expressões Java e comentários, você também pode usar variáveis predefinidas (listadas abaixo), que serão expandidas para seus respectivos valores como macros.

Você também pode especificar variáveis personalizadas. Variáveis personalizadas usam o seguinte formato: ${VARIABLE_NAME}, onde VARIABLE_NAME é o nome da variável (por exemplo, ${MY_CUSTOM_FUNCTION_NAME}). Antes que a IDE crie um novo arquivo usando variáveis personalizadas, será exibida uma caixa de diálogo onde você poderá definir os valores para as variáveis personalizadas no modelo.

Ao usar a diretiva #parse, você pode incluir modelos da aba Incluir. Para incluir um modelo, especifique o nome completo do modelo como um parâmetro entre aspas (por exemplo, #parse("File Header.java")).
As variáveis predefinidas aceitam os seguintes valores:
${PACKAGE_NAME}   O nome do pacote no qual o novo arquivo package-info.java é criado
${USER}   O nome de login de sistema do usuário atual
${DATE}   A data atual do sistema
${TIME}   A hora atual do sistema
${YEAR}   O ano atual
${MONTH}   O mês atual
${MONTH_NAME_SHORT}   As primeiras 3 letras do nome do mês atual (Jan, Feb, etc.)
${MONTH_NAME_FULL}   O nome completo do mês atual (Janeiro, Fevereiro, etc.)
${DAY}   O dia atual do mês
${HOUR}   A hora atual
${MINUTE}   O minuto atual
${PROJECT_NAME}   O nome do projeto atual
