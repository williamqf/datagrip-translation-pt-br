Aplicável a novos enums Java criados ao invocar Novo | Classe Java | Enum na janela de ferramentas Projeto.

Este template integrado é editável. Além de expressões Java e comentários, você também pode usar variáveis predefinidas (listadas abaixo), que serão expandidas para seus valores correspondentes como macros.

Você também pode especificar variáveis personalizadas. Variáveis personalizadas usam o formato: ${VARIABLE_NAME}, onde VARIABLE_NAME é o nome da variável (por exemplo, ${MY_CUSTOM_FUNCTION_NAME}). Antes que a IDE crie um novo arquivo usando variáveis personalizadas, um diálogo será exibido para que você defina os valores dessas variáveis no template.

Ao usar a diretiva #parse, você pode incluir templates da aba Incluir. Para incluir um template, especifique o nome completo do template como um parâmetro entre aspas (por exemplo, #parse("File Header.java")).
As variáveis predefinidas aceitam os seguintes valores:
${PACKAGE_NAME} O nome do pacote no qual o novo enum é criado
${NAME} O nome do novo enum que você especificou no diálogo Criar Nova Classe
${USER} O nome de login de sistema do usuário atual
${DATE} A data atual do sistema
${TIME} A hora atual do sistema
${YEAR} O ano atual
${MONTH} O mês atual
${MONTH_NAME_SHORT} As primeiras 3 letras do nome do mês atual (Jan, Feb, etc.)
${MONTH_NAME_FULL} O nome completo do mês atual (Janeiro, Fevereiro, etc.)
${DAY} O dia atual do mês
${HOUR} A hora atual
${MINUTE} O minuto atual
${PROJECT_NAME} O nome do projeto atual
