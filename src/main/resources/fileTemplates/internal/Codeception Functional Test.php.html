Um template integrado para testes funcionais do Codeception. Ele é usado sempre que você cria um novo teste funcional do Codeception ao selecionar Novo | Teste PHP no menu de contexto de uma das visualizações do projeto e escolher 'Codeception Functional' como a biblioteca de teste.
O template é editável. Além de instruções, expressões e comentários PHP, você também pode usar as variáveis predefinidas listadas abaixo. Cada variável é expandida para o valor correspondente, assim como uma macro.
Por meio da diretiva #parse, é possível incluir templates da aba Includes especificando o nome completo do template desejado como um parâmetro entre aspas.
As variáveis predefinidas aceitam os seguintes valores:
${TESTED_NAME} Nome da classe sendo testada
${TESTED_NAMESPACE} Namespace da classe sendo testada
${NAME} Nome da classe de teste
${NAMESPACE} Namespace da classe de teste
${FILE_NAME} Nome do arquivo atual
${USER} Nome de login do usuário atual no sistema
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${DAY} Dia atual do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${PRODUCT_NAME} Nome do IDE atual
${DS} Símbolo de cifrão, que resolve para o caractere '$' literal.
${PROJECT_NAME} Nome do projeto atual
