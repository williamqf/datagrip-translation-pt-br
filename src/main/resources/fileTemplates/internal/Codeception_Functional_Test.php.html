Modelo integrado para testes funcionais do Codeception. É utilizado sempre que você cria um novo teste funcional Codeception (selecione Novo | Teste PHP no menu pop-up em um dos visualizadores de projeto e selecione "Codeception Functional" como a biblioteca de testes).
O modelo é editável. Além de expressões, sentenças e comentários do PHP, você também pode usar as variáveis predefinidas listadas abaixo. Cada variável é expandida para seu valor correspondente, da mesma forma que uma macro.
Usando a diretiva #parse você pode incluir modelos na guia Incluir especificando o nome completo do modelo desejado como um parâmetro entre aspas.
As variáveis predefinidas recebem os seguintes valores:
${TESTED_NAME} O nome da classe testada.
${TESTED_NAMESPACE} O namespace da classe testada.
${NAME} O nome da classe de teste.
${NAMESPACE} O namespace da classe de teste.
${FILE_NAME} O nome do arquivo atual.
${USER} O login do sistema do usuário atual.
${DATE} A data do sistema atual.
${TIME} A hora do sistema atual.
${YEAR} O ano atual.
${MONTH} O mês atual.
${DAY} O dia do mês atual.
${HOUR} A hora atual.
${MINUTE} O minuto atual.
${PRODUCT_NAME} O nome do IDE atual.
${DS} O símbolo do dólar é avaliado para o caractere '$' comum.
${PROJECT_NAME} O nome do projeto atual.
