Modelo integrado para testes de PHPUnit (versão 5 ou anterior). Ele é usado sempre que você cria um novo teste PHPUnit (no menu suspenso de um dos Project views, selecione Novo | Teste PHP e escolha "PHPUnit < 6" como biblioteca de teste).
Os modelos podem ser editados. Além de expressões, declarações e comentários de PHP, você também pode usar as variáveis predefinidas listadas abaixo. Cada variável é expandida para o valor correspondente, assim como em uma macro.
Usando a diretiva #parse, você pode incluir modelos na guia Inclusões especificando o nome completo do modelo desejado entre aspas como um parâmetro.
As variáveis predefinidas recebem os seguintes valores:
${TESTED_NAME} Nome da classe testada
${TESTED_NAMESPACE} Namespace da classe testada
${NAME} Nome da classe de teste
${NAMESPACE} Namespace da classe de teste
${FILE_NAME} Nome do arquivo atual
${USER} Nome de login do sistema do usuário atual
${DATE} Data atual do sistema
${TIME} Horário atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${DAY} Dia do mês atual
${HOUR} Hora atual
${MINUTE} Minuto atual
${PRODUCT_NAME} Nome do IDE atual
${DS} O cifrão de dólar é avaliado como o caractere "$" comum.
${PROJECT_NAME} Nome do projeto atual
