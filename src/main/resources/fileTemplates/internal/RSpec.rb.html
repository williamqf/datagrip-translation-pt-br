Um modelo integrado para testes RSpec. Ele é usado sempre que você cria um novo teste selecionando Novo | Teste Ruby no menu de contexto de uma das visualizações do projeto. O modelo fica disponível quando a pasta Raiz de Origem de Teste está selecionada. Além de instruções, expressões e comentários Ruby, você também pode usar as variáveis predefinidas listadas abaixo. Cada variável é expandida para o valor correspondente como uma macro. Através da diretiva #parse, os modelos da aba Includes podem ser incluídos especificando o nome completo do modelo desejado como um parâmetro entre aspas.
As variáveis predefinidas aceitam os seguintes valores:
${FILE_NAME} Nome do arquivo atual
${USER} Nome de login do sistema do usuário atual
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${DAY} Dia atual do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${PRODUCT_NAME} Nome do IDE atual
${PROJECT_NAME} Nome do projeto atual
