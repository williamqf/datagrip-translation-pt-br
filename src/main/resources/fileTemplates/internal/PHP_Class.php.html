Modelo embutido para uma classe PHP. É utilizado sempre que você cria uma nova classe PHP, selecionando "Novo | Classe PHP" no menu pop-up de uma das vistas do projeto.
O modelo é editável. Além de expressões, declarações PHP e comentários, você também pode usar as variáveis predefinidas a seguir. Cada variável é expandida, como uma macro, para o seu valor correspondente.
Usando a diretiva #parse você pode incluir os modelos encontrados na aba "Incluir", especificando como parâmetro o nome completo do modelo desejado, entre aspas.
Variáveis predefinidas assumem os seguintes valores:
${FILE_NAME} Nome do arquivo atual
${USER} Login do usuário atual no sistema
${DATE} Data atual do sistema
${TIME} Horário atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${DAY} Dia do mês atual
${HOUR} Hora atual
${MINUTE} Minuto atual
${PRODUCT_NAME} Nome da IDE atual
${PROJECT_NAME} Nome do projeto atual
${DS} O cifrão é avaliado como o caractere "$" comum.
