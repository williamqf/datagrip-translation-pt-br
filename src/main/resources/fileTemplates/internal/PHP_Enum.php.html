Esta é uma predefinição para enumerações em PHP. Ela será usada sempre que for criado um novo tipo de enumeração em PHP, seja no menu popup Novo | Classes PHP ou em uma das visualizações de projeto.
Esta predefinição pode ser editada. Além das instruções, expressões e comentários de PHP, é possível usar também as variáveis predefinidas listadas abaixo. Cada variável será expandida para o valor correspondente, como uma macro.
Usando a diretiva #parse, é possível incluir as predefinições encontradas em Incluir guias especificando o nome completo da predefinição necessária como parâmetro entre aspas.
Variáveis predefinidas abaixo:
${FILE_NAME} O nome do arquivo atual
${USER} O atual usuário fez login no sistema
${DATE} A data atual do sistema
${TIME} A hora atual do sistema
${YEAR} O ano atual
${MONTH} O mês atual
${DAY} O dia atual do mês
${HOUR} A hora atual
${MINUTE} O minuto atual
${PRODUCT_NAME} O nome do IDE atual
${DS} O cifrão dólar é avaliado como o caractere '$' normal.
${PROJECT_NAME} O nome do atual projeto
