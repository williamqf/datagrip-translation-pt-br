Um modelo embutido para o arquivo principal do plugin do WordPress. É usado sempre que um novo projeto do tipo plugin do WordPress é criado.
O modelo é editável. Além de declarações, expressões e comentários em PHP, você também pode usar as variáveis predefinidas listadas abaixo. Cada variável é expandida para o seu valor correspondente, assim como acontece com as macros.
Usando a diretiva #parse é possível incluir modelos da guia Includes, especificando o nome completo do modelo desejado como parâmetro entre aspas.
As variáveis predefinidas assumem os seguintes valores:
${PLUGIN_NAME} O nome do plugin atual derivado do nome do diretório
${FILE_NAME} O nome do arquivo atual
${USER} O nome de login do usuário atual no sistema
${DATE} A data atual do sistema
${TIME} A hora atual do sistema
${YEAR} O ano atual
${MONTH} O mês atual
${DAY} O dia do mês atual
${HOUR} A hora atual
${MINUTE} O minuto atual
${PRODUCT_NAME} O nome do IDE atual
${PROJECT_NAME} O nome do projeto atual
${DS} O símbolo do dólar é avaliado como o caractere "$" comum.
