Arquivo Go é um template embutido usado cada vez que você cria um novo arquivo Go. Para usar este template, vá para Arquivo | Novo , selecione Arquivo Go na categoria Go, e então clique em Arquivo vazio .

O template é editável. Você pode usar as variáveis pré-definidas listadas abaixo. Estas variáveis serão expandidas mais tarde no editor para os valores correspondentes.

Você também pode especificar variáveis customizadas. Variáveis customizadas usam o seguinte formato: ${VARIABLE_NAME}, onde VARIABLE_NAME é o nome da variável (ex: ${MY_CUSTOM_FUNCTION_NAME}). Antes de o IDE usar variáveis customizadas para criar um novo arquivo, você verá uma caixa de diálogo onde poderá definir os valores das variáveis customizadas no template.

Ao usar a diretiva #parse, você pode incluir templates que estão na aba Incluir. Para incluir um template, especifique o nome completo do template entre aspas como um parâmetro (ex: #parse("File Header")).
Lista de variáveis pré-definidas
${GO_PACKAGE_NAME_WITH_TEST_SUFFIX} O nome do pacote onde o novo arquivo de teste será criado. O arquivo deve ter o sufixo _test. O nome usa o seguinte formato: package_test.
${GO_PACKAGE_NAME} O nome do pacote onde o novo arquivo será criado.
${DATE} A data atual do sistema.
${DAY} O dia do mês atual.
${DAY_NAME_SHORT} As primeiras 3 letras do nome do dia atual (ex: Seg, Ter, etc).
${DAY_NAME_FULL} O nome completo do dia atual (ex: Segunda-feira, Terça-feira, etc).
${DIR_PATH} O caminho para o diretório onde o novo arquivo será armazenado. Este caminho é relativo à raiz do projeto.
${DS} O símbolo do dólar ($). Esta variável é usada para escapar o caractere dólar para que ele não seja tratado como um prefixo de variável de template.
${FILE_NAME} O nome do novo arquivo.
${HOUR} A hora atual.
${MINUTE} O minuto atual.
${SECOND} O segundo atual.
${MONTH} O mês atual.
${MONTH_NAME_SHORT} As primeiras 3 letras do nome do mês atual (ex: Jan, Fev, etc).
${MONTH_NAME_FULL} O nome completo do mês atual (ex: Janeiro, Fevereiro, etc).
${NAME} O nome da nova entidade (arquivo, tipo, interface, etc).
