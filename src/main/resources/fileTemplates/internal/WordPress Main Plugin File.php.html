Um modelo integrado para o arquivo principal de um plugin do WordPress. Ele é usado sempre que um novo projeto do tipo WordPress Plugin é criado.
O modelo é editável. Além de instruções, expressões e comentários PHP, você também pode usar as variáveis predefinidas listadas abaixo. Cada variável é expandida para o valor correspondente, como uma macro.
Usando a diretiva #parse, você pode incluir modelos da aba Includes especificando o nome completo do modelo desejado como um parâmetro entre aspas.
As variáveis predefinidas aceitam os seguintes valores:
${PLUGIN_NAME}   Nome atual do plugin derivado do nome do diretório
${FILE_NAME}   Nome do arquivo atual
${USER}   Nome de login de sistema do usuário atual
${DATE}   Data atual do sistema
${TIME}   Hora atual do sistema
${YEAR}   Ano atual
${MONTH}   Mês atual
${DAY}   Dia atual do mês
${HOUR}   Hora atual
${MINUTE}   Minuto atual
${PRODUCT_NAME}   Nome da IDE atual
${PROJECT_NAME}   Nome do projeto atual
${DS}   Símbolo de cifrão, avaliado como um caractere '$' comum.
