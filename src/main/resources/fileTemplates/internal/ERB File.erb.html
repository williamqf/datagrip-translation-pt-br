Um modelo integrado para um arquivo ERB. Ele é usado sempre que você cria um novo arquivo ERB selecionando Novo | Arquivo ERB no menu de contexto de uma das visualizações de projeto. Além de instruções, expressões e comentários Ruby e HTML, você também pode usar as variáveis predefinidas listadas abaixo. Cada variável é expandida para o valor correspondente como uma macro. Através da diretiva #parse, é possível incluir modelos da aba Includes especificando o nome completo do modelo desejado como um parâmetro entre aspas.
As variáveis predefinidas aceitam os seguintes valores:
${FILE_NAME} Nome do arquivo atual
${USER} Nome de login de sistema do usuário atual
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${DAY} Dia atual do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${PRODUCT_NAME} Nome do IDE atual
${PROJECT_NAME} Nome do projeto atual
