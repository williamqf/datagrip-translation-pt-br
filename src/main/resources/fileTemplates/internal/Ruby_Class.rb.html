Um template embutido para uma classe Ruby. Ele é usado toda vez que você cria uma nova classe Ruby selecionando Novo | Arquivo Ruby/Classe no menu pop-up em uma das visualizações do projeto. Junto com declarações Ruby, expressões e comentários, você pode também usar as variáveis predefinidas listadas abaixo. Toda variável é expandida como macros para o valor correspondente. Através da diretriz #parse, você pode incluir templates da aba Includes especificando o nome completo do template desejado como um parâmetro entre aspas.
As variáveis predefinidas assumem os seguintes valores:
${FILE_NAME} Nome do arquivo atual
${USER} Nome do login do usuário no sistema
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${DAY} Dia atual do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${PRODUCT_NAME} Nome do IDE atual
${PROJECT_NAME} Nome do projeto atual
