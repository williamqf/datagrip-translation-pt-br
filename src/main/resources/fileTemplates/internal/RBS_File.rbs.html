Um modelo interno para um arquivo RBS. Ele é usado toda vez que você cria um novo arquivo RBS selecionando Novo | Arquivo/Classe RBS no menu pop-up de uma das exibições de projeto. Junto com instruções, expressões e comentários RBS, você também pode usar as variáveis predefinidas listadas abaixo. Cada variável é expandida como macros para o valor correspondente. Por meio da diretiva #parse, você pode incluir modelos da guia Inclusões especificando o nome completo do modelo desejado como um parâmetro entre aspas.

As variáveis predefinidas assumem os seguintes valores:
${FILE_NAME} Nome do arquivo atual
${USER} Nome de login do sistema do usuário atual
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${DAY} Dia atual do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${PRODUCT_NAME} Nome do IDE atual
${PROJECT_NAME} Nome do projeto atual
