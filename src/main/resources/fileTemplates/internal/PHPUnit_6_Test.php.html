Um modelo integrado para testes (versão 6 ou superior) do PHPUnit.
Usado sempre que você cria um novo teste do PHPUnit (selecionando Novo | Teste PHP no menu de contexto de uma das visualizações do projeto e selecionando 'PHPUnit' como biblioteca de teste).
Este modelo é editável. Além de expressões, instruções e comentários do PHP, você também pode usar as variáveis predefinidas listadas abaixo. Cada variável será expandida para seu valor correspondente, como uma macro.
Usando a diretiva #parse, você pode incluir um modelo na guia Inclusões especificando o nome completo do modelo desejado como parâmetro entre aspas.
As variáveis predefinidas assumem os seguintes valores:
${TESTED_NAME} O nome da classe testada
${TESTED_NAMESPACE} O namespace da classe testada
${NAME} O nome da classe de teste
${NAMESPACE} O namespace da classe de teste
${FILE_NAME} O nome do arquivo atual
${USER} O nome de logon do usuário atual no sistema
${DATE} A data atual do sistema
${TIME} A hora atual do sistema
${YEAR} O ano atual
${MONTH} O mês atual
${DAY} O dia do mês atual
${HOUR} A hora atual
${MINUTE} O minuto atual
${PRODUCT_NAME} O nome do IDE atual
${DS} O símbolo de cifrão é avaliado como o caractere '$' comum.
${PROJECT_NAME} O nome do projeto atual
