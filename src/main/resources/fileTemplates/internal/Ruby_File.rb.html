Um modelo embutido para um arquivo Ruby. Ele é usado sempre que você cria um novo arquivo Ruby selecionando Novo | Arquivo/Classe Ruby a partir do menu suspenso em uma das visualizações de projeto.
Variáveis predefinidas tomam os seguintes valores:
${FILE_NAME} Nome do arquivo atual
${USER} Nome de login do sistema do usuário atual
${DATE} Data do sistema atual
${TIME} Hora do sistema atual
${YEAR} Ano atual
${MONTH} Mês atual
${DAY} Dia atual do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${PRODUCT_NAME} Nome do IDE atual
${PROJECT_NAME} Nome do projeto atual
