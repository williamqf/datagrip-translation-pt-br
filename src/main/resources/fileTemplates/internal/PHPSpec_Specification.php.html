Modelo interno para especificações do PHPSpec. Usado sempre que uma nova especificação PHPSpec é criada (selecionando Novo | Teste PHP no menu pop-up em uma das visualizações de projeto e escolhendo 'PHPSpec' como a biblioteca de teste).
Este modelo é editável. Além de expressões, comandos e comentários PHP, você também pode usar as variáveis pré-definidas descritas a seguir. Cada variável será expandida para seu valor correspondente, assim como uma macro.
O uso da diretiva #parse permite incluir modelos na guia Incluir especificando o nome completo do modelo desejado entre aspas como um parâmetro.
As variáveis pré-definidas assumem os seguintes valores:
${TESTED_NAME} Nome da classe que está sendo testada
${TESTED_NAMESPACE} Namespace da classe que está sendo testada
${NAME} Nome da classe de teste
${NAMESPACE} Namespace da classe de teste
${FILE_NAME} Nome do arquivo atual
${USER} Nome de login do sistema do usuário atual
${DATE} Data do sistema atual
${TIME} Hora do sistema atual
${YEAR} Ano atual
${MONTH} Mês atual
${DAY} Dia do mês atual
${HOUR} Hora atual
${MINUTE} Minuto atual
${PRODUCT_NAME} Nome do IDE atual
${DS} O cifrão de dólar, avaliado como o caractere '$' comum.
${PROJECT_NAME} Nome do projeto atual
