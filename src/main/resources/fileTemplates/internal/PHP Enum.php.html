Este é um template integrado para Enumerações (Enums) PHP. Ele é usado sempre que você cria um novo Enum PHP selecionando Novo | Classe PHP no menu de contexto de uma das visualizações do projeto.
O template é editável. Além de instruções, expressões e comentários PHP, você também pode usar as variáveis predefinidas listadas abaixo. Cada variável é expandida para o valor correspondente, assim como uma macro.
Através da diretiva #parse, você pode incluir templates da aba Includes especificando o nome completo do template desejado como um parâmetro entre aspas.
As variáveis predefinidas aceitam os seguintes valores:
${FILE_NAME} Nome do arquivo atual
${USER} Nome de login do usuário atual no sistema
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${DAY} Dia atual do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${PRODUCT_NAME} Nome da IDE atual
${DS} Símbolo de cifrão, avaliado como um caractere "$" literal.
${PROJECT_NAME} Nome do projeto atual
