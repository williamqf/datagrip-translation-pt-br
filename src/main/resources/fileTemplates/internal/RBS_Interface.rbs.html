Um modelo embutido para uma interface RBS. Ele é utilizado sempre que você cria uma nova interface RBS selecionando Novo | Arquivo/Classe RBS no menu pop-up em uma das views de projeto. Junto com declarações RBS, expressões e comentários, você pode também usar as variáveis pré-definidas listadas abaixo. Cada variável é expandida como macros no valor correspondente. Por meio da directiva #parse, você pode incluir modelos a partir da guia Incluir especificando o nome completo do modelo desejado como um parâmetro entre aspas.
Variáveis pré-definidas assumem os seguintes valores:
${FILE_NAME} Nome do arquivo atual
${USER} Nome de login do sistema do usuário atual
${DATE} Data do sistema atual
${TIME} Horário do sistema atual
${YEAR} Ano atual
${MONTH} Mês atual
${DAY} Dia atual do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${PRODUCT_NAME} Nome do IDE atual
${PROJECT_NAME} Nome do projeto atual
