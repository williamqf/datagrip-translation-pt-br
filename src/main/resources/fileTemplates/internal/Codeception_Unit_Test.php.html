Modelo básico para testes unitários do Codeception. É usado sempre que se cria um novo teste unitário do Codeception (na view de um dos projetos, selecione Novo | Teste PHP no menu e escolha 'Codeception Unit' como biblioteca de teste).
Este modelo é editável. Além de expressões, instruções PHP e comentários, você também pode utilizar as variáveis predefinidas listadas abaixo. Cada variável é expandida para o valor correspondente como uma macro.
Você pode incluir modelos localizados na aba "Incluir" utilizando a diretiva #parse para especificar o nome do modelo necessário como um parâmetro entre aspas.
As variáveis predefinidas assumem os seguintes valores:
${TESTED_NAME} Nome da classe testada
${TESTED_NAMESPACE} Namespace da classe testada
${NAME} Nome da classe de teste
${NAMESPACE} Namespace da classe de teste
${FILE_NAME} Nome do arquivo atual
${USER} Nome de login atual do usuário no sistema
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${DAY} Dia do mês atual
${HOUR} Hora atual
${MINUTE} Minuto atual
${PRODUCT_NAME} Nome atual do IDE
${DS} O sinal de cifrão é resolvido para um caractere '$' comum.
${PROJECT_NAME} Nome do projeto atual
