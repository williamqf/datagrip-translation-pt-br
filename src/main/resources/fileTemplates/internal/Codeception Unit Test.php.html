Um modelo integrado para um teste unitário do Codeception. Ele é usado toda vez que você cria um novo teste unitário do Codeception ao selecionar Novo | Teste PHP no menu de contexto de uma das visualizações de projeto e, em seguida, selecionar 'Codeception Unit' como a biblioteca de testes.
O modelo é editável. Além de instruções PHP, expressões e comentários, você também pode usar as variáveis predefinidas listadas abaixo. Cada variável é expandida para o valor correspondente, assim como uma macro.
Através da diretiva #parse, modelos da aba Includes podem ser incluídos especificando o nome completo do modelo desejado como um parâmetro entre aspas.
As variáveis predefinidas aceitam os seguintes valores:
${TESTED_NAME} O nome da classe que está sendo testada
${TESTED_NAMESPACE} O namespace da classe que está sendo testada
${NAME} O nome da classe de teste
${NAMESPACE} O namespace da classe de teste
${FILE_NAME} O nome do arquivo atual
${USER} O nome de login do sistema do usuário atual
${DATE} A data atual do sistema
${TIME} A hora atual do sistema
${YEAR} O ano atual
${MONTH} O mês atual
${DAY} O dia atual do mês
${HOUR} A hora atual
${MINUTE} O minuto atual
${PRODUCT_NAME} O nome do IDE atual
${DS} O cifrão, avaliado como um caractere '$' comum.
${PROJECT_NAME} O nome do projeto atual
