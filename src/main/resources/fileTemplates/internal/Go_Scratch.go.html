Go Rascunho é um modelo interno usado sempre que um novo arquivo temporário Go é criado. Arquivos temporários são arquivos totalmente funcionais, executáveis e podem ser depurados, que você pode usar para elaborar seu código como rascunho. Para criar um arquivo temporário, clique em File | New | Scratch File. Na caixa de diálogo New Scratch File, encontre e selecione Go.

O modelo é editável. Neste modelo, você pode usar as variáveis pré-definidas abaixo. Estas variáveis serão expandidas posteriormente no editor, com os valores correspondentes.

Você também pode especificar variáveis personalizadas. As variáveis personalizadas usam este formato: ${VARIABLE_NAME}, onde VARIABLE_NAME é o nome da variável (por exemplo, ${MY_CUSTOM_FUNCTION_NAME}). Antes que o IDE crie um arquivo novo usando as variáveis personalizadas, uma caixa de diálogo será exibida, permitindo que você defina os valores das variáveis personalizadas deste modelo.

Usando a diretiva #parse, você pode incluir modelos na guia Incluir. Para incluir um modelo, especifique o nome completo do modelo entre aspas como parâmetro (por exemplo, #parse("File Header")).
Lista de Variáveis Pré-definidas
${GO_PACKAGE_NAME_WITH_TEST_SUFFIX} O nome do pacote no qual o novo arquivo de teste será criado. O arquivo deve ter o sufixo _test. O nome terá este formato: package_test.
${GO_PACKAGE_NAME} O nome do pacote no qual o novo arquivo será criado.
${DATE} A data atual do sistema.
${DAY} O dia do mês atual.
${DAY_NAME_SHORT} As primeiras 3 letras do nome do dia atual (por exemplo, Seg, Ter etc.).
${DAY_NAME_FULL} O nome completo do dia atual (por exemplo, Segunda-feira, Terça-feira etc.).
${DIR_PATH} O caminho do diretório onde o novo arquivo será armazenado. Este caminho é relativo à raiz do projeto.
${DS} O símbolo do dólar ($). Esta variável é usada para escapar o caractere do dólar para evitar que seja tratado como um prefixo de variáveis de modelo.
${FILE_NAME} O nome do novo arquivo.
${HOUR} A hora atual.
${MINUTE} O minuto atual.
${SECOND} O segundo atual.
${MONTH} O mês atual.
${MONTH_NAME_SHORT} As primeiras 3 letras do nome do mês atual (por exemplo, Jan, Fev etc.).
${MONTH_NAME_FULL} O nome completo do mês atual (por exemplo, Janeiro, Fevereiro etc.).
