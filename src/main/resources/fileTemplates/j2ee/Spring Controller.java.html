Aplica-se a novos controladores Spring criados ao invocar Novo | Componente Spring | Controller na janela de ferramentas Projeto.
Este modelo integrado é editável. Além de expressões e comentários Java, você também pode usar variáveis predefinidas (listadas abaixo), que serão expandidas para os valores correspondentes como macros.

Você também pode especificar variáveis personalizadas. As variáveis personalizadas usam o seguinte formato: ${VARIABLE_NAME}, onde VARIABLE_NAME é o nome da variável (por exemplo, ${MY_CUSTOM_FUNCTION_NAME}). Antes que a IDE crie um novo arquivo usando variáveis personalizadas, uma caixa de diálogo será exibida para que você possa definir os valores das variáveis personalizadas no modelo.

Ao usar a diretiva #parse, você pode incluir modelos da aba Includes. Para incluir um modelo, especifique o nome completo do modelo como um parâmetro entre aspas (por exemplo, #parse("File Header.java")).
As variáveis predefinidas aceitam os seguintes valores:
${PACKAGE_NAME} Nome do pacote no qual a nova classe é criada
${NAME} O nome da nova classe que você especificou na caixa de diálogo Criar Nova Classe
${USER} Nome de login de sistema do usuário atual
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${MONTH_NAME_SHORT} As primeiras 3 letras do nome do mês atual (Jan, Feb, etc.)
${MONTH_NAME_FULL} Nome completo do mês atual (January, February, etc.)
${DAY} Dia atual do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${PROJECT_NAME} Nome do projeto atual
