Este é um modelo integrado usado sempre que você cria uma nova Anotação de Escopo do Guice.
Ao usar a diretiva #parse, você pode incluir modelos da aba Incluir especificando o nome completo do modelo desejado como um parâmetro entre aspas. Por exemplo:
#parse("File Header.java")
As variáveis predefinidas aceitarão os seguintes valores:
${PACKAGE_NAME} O nome do pacote no qual a nova classe é criada
${NAME} O nome da nova classe especificado na caixa de diálogo Criar Nova Classe
${USER} Nome de login do usuário atual no sistema
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${MONTH_NAME_SHORT} As primeiras 3 letras do nome do mês atual. Por exemplo: Jan, Fev, etc.
${MONTH_NAME_FULL} O nome completo do mês atual. Por exemplo: Janeiro, Fevereiro, etc.
${DAY} Dia atual do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${PROJECT_NAME} O nome do projeto atual
