Aplica-se a novos Controller Advices do Spring criados ao invocar Novo | Componente Spring | Controller Advice na janela de ferramenta Projeto.
Este template integrado é editável. Além de expressões Java e comentários, você também pode usar variáveis predefinidas (listadas abaixo), que serão expandidas para seus respectivos valores como macros.

Você também pode especificar variáveis personalizadas. Variáveis personalizadas usam o seguinte formato: ${VARIABLE_NAME}, onde VARIABLE_NAME é o nome da variável (por exemplo, ${MY_CUSTOM_FUNCTION_NAME}). Antes que a IDE crie um novo arquivo usando variáveis personalizadas, será exibido um diálogo onde você poderá definir os valores para elas no template.

Usando a diretiva #parse, você pode incluir templates da aba Inclusões. Para incluir um template, especifique o nome completo do template como um parâmetro entre aspas (por exemplo, #parse("File Header.java")).
As variáveis predefinidas aceitam os seguintes valores:
${PACKAGE_NAME} Nome do pacote onde a nova classe será criada
${NAME} O nome da nova classe que você especificou no diálogo Criar Nova Classe
${USER} Nome de login do usuário atual no sistema
${DATE} Data atual do sistema
${TIME} Hora atual do sistema
${YEAR} Ano atual
${MONTH} Mês atual
${MONTH_NAME_SHORT} As primeiras 3 letras do nome do mês atual (Jan, Feb, etc.)
${MONTH_NAME_FULL} Nome completo do mês atual (January, February, etc.)
${DAY} Dia atual do mês
${HOUR} Hora atual
${MINUTE} Minuto atual
${PROJECT_NAME} Nome do projeto atual
