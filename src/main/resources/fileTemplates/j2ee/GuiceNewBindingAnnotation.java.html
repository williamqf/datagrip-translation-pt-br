Este é o template integrado usado sempre que você cria uma nova anotação de binding do Guice.
Usando a diretiva #parse, você pode incluir um template da aba Includes especificando o nome completo do template desejado como um parâmetro entre aspas. Exemplo:
#parse("File Header.java")
As variáveis predefinidas assumirão os seguintes valores:
${PACKAGE_NAME} O nome do pacote no qual a nova classe é criada
${NAME} O nome da nova classe especificado no diálogo Criar Nova Classe
${USER} Nome de login do usuário atual no sistema
${DATE} A data atual do sistema
${TIME} A hora atual do sistema
${YEAR} O ano atual
${MONTH} O mês atual
${MONTH_NAME_SHORT} As primeiras 3 letras do nome do mês atual. Exemplo: Jan, Fev, etc.
${MONTH_NAME_FULL} O nome completo do mês atual. Exemplo: Janeiro, Fevereiro, etc.
${DAY} O dia atual do mês
${HOUR} A hora atual
${MINUTE} O minuto atual
${PROJECT_NAME} O nome do projeto atual
