Denunciar tipo de parâmetro de método de repositório incorreto.

Exemplo:


  public class Pessoa {
    private int id;
    @Column
    private String sobrenome;

    @Column
    private int idade;
  }

  public interface RepositorioCliente extends Repositório<Pessoa, Integer> {
    List<Pessoa> findByAddress(int sobrenome);  // Deve ser do tipo 'String'
    List<Pessoa> deleteByAge(String str);   // Deve ser do tipo 'int'
  }
