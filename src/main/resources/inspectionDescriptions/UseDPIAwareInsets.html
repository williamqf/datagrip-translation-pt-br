Relata usos de java.awt.Insets e JBUI.insetsXyz() que podem ser simplificados.

As instâncias de Insets não são sensíveis a DPI e podem causar problemas no layout da UI.

A correção rápida realiza a substituição por JBUI.insets() ou simplifica a expressão.

Exemplo:


// Incorreto:
Insets insets1 = new Insets(1, 2, 3, 4);
Insets insets2 = new Insets(1, 2, 1, 2);
Insets insets3 = new Insets(1, 0, 0, 0);

// Correto:
Insets insets1 = JBUI.insets(1, 2, 3, 4);
Insets insets2 = JBUI.insets(1, 2);
Insets insets3 = JBUI.insetsTop(1);
