Relata chamadas para métodos de asserção de igualdade do PHPUnit (como assertEquals(), assertNotEquals(), assertSame(), etc.) onde o argumento do resultado esperado não é um literal, mas o argumento do resultado real é um literal. Essas chamadas funcionam corretamente para asserções que passam, mas podem produzir relatórios de erro confusos se os argumentos esperado e real forem diferentes.

Para mais detalhes, consulte assertEquals.
