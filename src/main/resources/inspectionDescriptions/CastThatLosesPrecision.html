Relata operações de cast entre tipos numéricos primitivos que podem resultar em perda de precisão.

Tais casts não são necessariamente um problema, mas se ocorrer uma perda de precisão inesperada, pode ser difícil rastrear bugs.

Exemplo:


  int a = 420;
  byte b = (byte) a;

Use a opção Ignorar casts de int para char para ignorar casts de int para char. Esse tipo de cast é frequentemente usado ao implementar operações de E/S, pois o método read() da classe java.io.Reader retorna um int.

Use a opção Ignorar casts de int 128-255 para byte para ignorar casts de valores constantes (128-255) de int para byte. Tais valores sofrerão overflow para números negativos que ainda cabem dentro de um byte.
