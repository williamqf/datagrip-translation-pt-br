Relata os seguintes problemas em comentários Javadoc:
  • Javadoc ausente onde é necessário
  • Tags obrigatórias ausentes
  • Tags inválidas ou incompletas
  • Descrição do Javadoc ausente ou incompleta

Exemplo:


  class Main {
  }

Após aplicar a correção rápida:


  /**
   *
   */
  class Main {
  }

Exemplo:


  String id(String value) {
    return value;
  }

Após aplicar a correção rápida:


  /**
   * @param value
   * @return
   */
  String id(String value) {
    return value;
  }
Use as abas e as opções de inspeção correspondentes para definir o escopo de visibilidade onde o Javadoc é exigido, bem como as tags Javadoc necessárias para classes, métodos, campos e classes internas.
