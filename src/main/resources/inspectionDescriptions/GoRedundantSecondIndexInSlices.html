Relata um segundo índice (limite superior) redundante em expressões de slice.

Geralmente, o segundo índice é opcional. Removê-lo pode melhorar a legibilidade do código.

Exemplo:

var a []int
a = a[0:len(a)] // `a[0:len(a)]` é o mesmo que `a[0:]`

Você pode aplicar a correção rápida Remover índice redundante para esses casos. Após aplicar a correção rápida, o código ficará assim:

var a []int
a = a[0:]
