Relata qualquer chamada para super(A, B) que não atenda aos seguintes requisitos:

  • B é uma instância de A
  • B é uma subclasse de A

Exemplo:


class Figure:
    def color(self):
        pass


class Rectangle(Figure):
    def color(self):
        pass


class Square(Figure):
    def color(self):
        return super(Rectangle, self).color() # Square não é uma instância ou subclasse de Rectangle

Como correção, você pode tornar Square uma subclasse da classe Rectangle.
