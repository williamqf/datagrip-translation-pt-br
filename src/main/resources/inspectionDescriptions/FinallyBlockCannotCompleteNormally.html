Relata as instruções return, throw, break, continue e yield usadas dentro de um bloco finally. Essas instruções fazem com que o bloco finally seja concluído de forma abrupta em vez de ser concluído normalmente. Quaisquer exceções lançadas nos blocos try e catch da mesma instrução try-catch serão suprimidas.

Exemplo:


  void x() {
    try {
      throw new RuntimeException();
    } finally {
      // Se bar() retornar true, a RuntimeException será suprimida
      if (bar()) return;
    }
  }
