Relata classes utilitárias que não são final nem abstract.

Uma classe utilitária declara todos os campos e métodos como static. Torná-la final evita que ela seja estendida acidentalmente.

Exemplo:


  public class UtilityClass {
    public static void foo() {}
  }

Após aplicar a correção rápida:


  public final class UtilityClass {
    public static void foo() {}
  }
