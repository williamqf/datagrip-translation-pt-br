Relata tentativas de realizar verificações por reflexão da presença de anotações que não estão disponíveis em tempo de execução.

O uso de Class.isAnnotationPresent() para testar anotações com política de retenção definida como SOURCE ou CLASS (padrão) sempre resultará em um resultado negativo. Este erro é facilmente ignorado.

Exemplo:


  {
    getClass().isAnnotationPresent(SourceAnnotation.class); // sempre false
  }

  @Retention(RetentionPolicy.SOURCE)
  @interface SourceAnnotation {}
