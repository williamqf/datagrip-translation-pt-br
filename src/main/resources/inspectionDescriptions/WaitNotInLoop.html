Relata chamadas wait() que não são feitas dentro de um loop.

wait() é geralmente usado para suspender uma thread até que alguma condição se torne verdadeira. Uma thread pode ser acordada por diferentes motivos, portanto, a condição deve ser verificada após o retorno da chamada wait(). Usar um loop é uma maneira fácil de realizar essa verificação.

Exemplo:


  class BoundedCounter {
    private int count;
    synchronized void inc() throws InterruptedException {
      if (count >= 10) wait();
      ++count;
    }
  }

O código adequado deve ser semelhante a:


  class BoundedCounter {
    private int count;
    synchronized void inc() throws InterruptedException {
      while (count >= 10) wait();
      ++count;
    }
  }
