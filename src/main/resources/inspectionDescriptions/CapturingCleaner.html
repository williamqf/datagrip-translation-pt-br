Relata um Runnable passado para Cleaner.register() que captura a referência sendo registrada. Se a referência for capturada, ela nunca será alcançável por phantom e a ação de limpeza nunca será chamada.

Possíveis fontes deste problema:

  • Lambdas que usam métodos, campos não estáticos ou o próprio this
  • Classes internas não estáticas (anônimas ou não) sempre capturam esta referência no Java 18 e versões anteriores
  • Referências de método de instância
  • Acesso a membros não estáticos da classe externa a partir de uma classe interna não estática

Exemplo de código que será relatado: 


  int fileDescriptor;
  Cleaner.Cleanable cleanable = Cleaner.create().register(this, () -> {
    System.out.println("adsad");
    //this é capturado por meio de fileDescriptor
    fileDescriptor = 0;
  });

Esta inspeção relata apenas se o nível de linguagem do projeto ou módulo for 9 ou superior.
