Relata null atribuído a uma variável Optional ou retornado de um método que retorna Optional.

É recomendado usar Optional.empty() (ou Optional.absent() para o Guava) para representar valores vazios.

Exemplo:


  Optional<Integer> foo(boolean flag) {
    return flag ? Optional.of(42) : null;
  }

Após aplicar o ajuste rápido:


  Optional<Integer> foo(boolean flag) {
    return flag ? Optional.of(42) : Optional.empty();
  }

Configuração da inspeção:

Use a opção Relatar comparações de Optional com null para também relatar comparações como optional == null. Embora isso possa estar correto em casos raros (por exemplo, campos opcionais com inicialização tardia), variáveis opcionais geralmente não são nulas e provavelmente o que se pretendia era optional.isPresent().
