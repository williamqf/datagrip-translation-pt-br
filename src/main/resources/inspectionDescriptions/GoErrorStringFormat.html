Relata problemas de formatação em strings de erro.

Exemplo:

err := fmt.Errorf("Cannot read the file!")
log.Printf("Reading %s: %v", file, err)

De acordo com as Error Strings no github.com/golang, as strings de erro não devem começar com letra maiúscula ou terminar com pontuação, pois podem aparecer em outros contextos.

Para corrigir a formatação, a correção rápida Corrigir formatação da string de erro pode ser aplicada. Após aplicar a correção rápida, a string de erro ficará assim:

err := fmt.Errorf("cannot read the file")
log.Printf("Reading %s: %v", file, err)
