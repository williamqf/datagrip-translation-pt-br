Relata chamadas para MutableCollection.addAll / += em que o argumento é uma chamada para Iterable.map/filter.

Este código aloca um objeto List adicional, o que pode ser evitado usando mapTo/filterTo.

A correção rápida substitui a chamada por mapTo/filterTo.

Exemplo:


    coll1.addAll(coll2.map { transform(it) })
    coll1 += coll2.map { transform(it) }

Após a aplicação da correção rápida:


    coll2.mapTo(coll1) { transform(it) }
