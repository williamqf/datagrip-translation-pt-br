Relata nomes de receptor como me, this, self ou nomes que diferem de outros nomes de receptor para este tipo.

Exemplo:

func (self *MeterSnapshot) Rate5() float64 { return math.Float64frombits(self.rate5) }

De acordo com Receiver Names no github.com/golang, você não deve usar nomes genéricos como "me", "this" ou "self". Esses identificadores são típicos em linguagens orientadas a objetos e podem conferir um significado especial ao método.
