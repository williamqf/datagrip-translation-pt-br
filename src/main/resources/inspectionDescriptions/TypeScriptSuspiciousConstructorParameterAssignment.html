Relata erros comuns em código TypeScript onde um campo de classe é declarado como um parâmetro do construtor e, em seguida, uma atribuição é feita a esse parâmetro.
Nesses casos, o campo correspondente não recebe a atribuição, apenas o valor do parâmetro local é alterado. 

class Foo {
  constructor(private p: number) {
    p = 1; // deve ser this.p = 1;
  }
}
