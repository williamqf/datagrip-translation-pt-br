Relata casos em que um operador spread redundante é usado em chamadas de funções da família arrayOf.

Use a correção rápida "Remover operador spread redundante" para limpar o código.

Exemplo:


  fun foo(vararg s: String) { }

  fun bar(ss: Array<String>) {
      foo(*arrayOf("abc"))       // Em ambas as chamadas de 'foo', a criação do array
      foo(*arrayOf(*ss, "zzz"))  // e o spread subsequente são redundantes
  }

Após aplicar a correção rápida:


  fun foo(vararg s: String) { }

  fun bar(ss: Array<String>) {
      foo("abc")
      foo(*ss, "zzz")
  }
