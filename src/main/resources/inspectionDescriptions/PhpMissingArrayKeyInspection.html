Relata arrays que não fornecem todas as chaves/entradas obrigatórias definidas no formato do array (array shape).

Esta inspeção ajuda a identificar cenários em que a instanciação de um array pode causar problemas em tempo de execução devido à ausência de entradas obrigatórias, garantindo melhor consistência do código e conformidade com a estrutura definida.

Snippet de código incorporado: 


      function connect(#[ArrayShape([
          'path' => 'string',
          'host' => [
              'name' => 'string',
              'port' => 'int'
          ]
      ])] array $options = []) { };
      connect(['host' => ['name' => 'localhost', 'port' => 1234]]); // chave 'path' ausente
