Relata chamadas de função com o tipo de resultado Deferred cujo valor de retorno não é utilizado.

Se o valor de retorno Deferred não for utilizado, o local da chamada não aguardará a conclusão desta função.

Exemplo:


  fun calcEverythingAsync() = CompletableDeferred(42)

  fun usage() {
      calcEverythingAsync()
  }
A chamada calcEverythingAsync() não é utilizada aqui e não é aguardada com await(), o que é suspeito.
