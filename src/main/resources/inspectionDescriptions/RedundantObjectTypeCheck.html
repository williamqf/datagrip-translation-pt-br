Relata verificações de tipo is não idiomáticas em objetos.

Recomenda-se substituir tais verificações por comparações de referência.

Exemplo:


  object Foo

  fun foo(arg: Any) = when {
      arg is Foo -> ...
      arg !is Foo -> ...
  }

Após a aplicação da correção rápida:


  object Foo

  fun foo(arg: Any) = when {
      arg === Foo -> ...
      arg !== Foo -> ...
  }
