Reporta sequências de escape Unicode desnecessárias. As sequências de escape Unicode são desnecessárias quando a codificação do arquivo pode processar o caractere sem a necessidade de escape. Esta inspeção não reporta caracteres de controle Unicode, exceto quebras de linha e tabulações horizontais.

Exemplo:


  String s = "\u0078";

Após a aplicação da correção rápida:


  String s = "x";
