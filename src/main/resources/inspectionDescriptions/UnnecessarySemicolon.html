Relata quaisquer ponto e vírgulas desnecessários, incluindo ponto e vírgulas usados entre membros de classe, dentro de instruções de bloco ou após definições de classe.

Embora esses ponto e vírgulas sejam válidos em Java, eles são redundantes e podem ser removidos.

Exemplo:


  class C {
    ;
    void m() throws Exception {
        try (AutoCloseable r1 = createAutoCloseable();) {
          ;
        }
    }
    ;
  }

Após a aplicação da correção rápida:


  class C {
    void m() throws Exception {
      try (AutoCloseable r1 = createAutoCloseable()) {
      }
    }
  }
