Relata rótulos redundantes que não podem ser referenciados pelas instruções 'break', 'continue' ou 'return'.

Tais rótulos podem ser removidos com segurança.

Exemplo:


fun foo(c: Boolean) {
    b@ while (true) a@ {
        while (true) {
            break@b
        }
    }
}

Após a aplicação do ajuste rápido:


fun foo(c: Boolean) {
    b@ while (true) {
        while (true) {
            break@b
        }
    }
}
