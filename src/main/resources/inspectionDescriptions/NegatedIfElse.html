Relata instruções if que contêm um ramo else e uma condição negada.

Inverter a ordem dos ramos if e else geralmente aumenta a clareza de tais instruções.

A instrução if atual pode ser invertida por meio de uma correção rápida.

Exemplo:


    void m(Object o1, Object o2) {
        if (o1 != o2) {
            System.out.println(1);
        }
        else {
            System.out.println(2);
        }
    }

Após aplicar a correção rápida:


    void m(Object o1, Object o2) {
        if (o1 == o2) {
            System.out.println(2);
        } else {
            System.out.println(1);
        }
    }

Use a opção Ignorar comparações '!= null' para ignorar comparações na forma != null.

Use a opção Ignorar comparações '!= 0' para ignorar comparações na forma != 0.
