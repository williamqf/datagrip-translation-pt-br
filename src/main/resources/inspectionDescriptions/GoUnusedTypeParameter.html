Relata parâmetros de tipo não utilizados. 
func main() {
 printAll(
  42,
  "bird",
 )
}

func printAll[I int, S string](
 i I,
 s string,
) {
 fmt.Println(i)
 fmt.Println(s)
}

A função printAll possui dois parâmetros de tipo, I e S, mas apenas o primeiro é utilizado. Portanto, S string ficará esmaecido.
