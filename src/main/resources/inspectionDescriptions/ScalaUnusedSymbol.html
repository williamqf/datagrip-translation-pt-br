Relata as declarações que não são utilizadas.

O quick-fix remove a declaração não utilizada.

Nas Opções abaixo, você pode controlar se a inspeção deve funcionar para declarações públicas e/ou locais. No caso de declarações locais, além de habilitar e desabilitar a inspeção, você também pode torná-la dependente da existência de uma das três opções do compilador: -Wunused:locals, -Wunused:linted, or -Xlint:unused.

Exemplo:


  def test(): Int = {
    val unused = 3
    3
  }

Após o quick-fix ser aplicado:


  def test(): Int = {
    3
  }
