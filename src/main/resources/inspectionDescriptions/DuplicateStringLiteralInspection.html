Relata literais de string duplicados em todo o projeto que não foram alterados. Duas correções rápidas são fornecidas. Uma delas introduz uma constante para a string duplicada e a utiliza em todo o projeto, enquanto a outra exibe todos os locais onde o literal de string específico foi duplicado.

Exemplo:


  class C1 { String CONST1 = "duplicate string"; }
  class C2 { String CONST2 = "duplicate string"; }

Configuração da inspeção:

  • Use o campo Comprimento mínimo da string para definir o tamanho mínimo da string necessário para detectar duplicatas.
  • Use a opção Ignorar expressões @PropertyKey para ignorar strings passadas como argumentos para métodos anotados com org.jetbrains.annotations.PropertyKey.
