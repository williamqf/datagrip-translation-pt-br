Relata campos definidos em subclasses de DelayedInit que são inicializados somente quando o método main é executado.

Exemplo:


  class Test {
    println(MyApp.initializedLater) // imprime 0
  }
  object MyApp extends DelayedInit {
    val initializedLater = 3
    override def delayedInit(body: => Unit): Unit = {
      new Test
      corpo
    }
  }
