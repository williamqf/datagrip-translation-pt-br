Relata definições de método com sintaxe de procedimento.

Não é recomendado aos desenvolvedores utilizar sintaxe similar a procedimentos para métodos. É inconsistente, pode levar a erros e será removido no Scala 3.

A correção rápida converte a definição de sintaxe de procedimento para sintaxe de função.

Exemplo:


  class Test {
    def method() { }
  }

Após a correção rápida ser aplicada:


  class Test {
    def method(): Unit = { }
  }

Dica: Você pode utilizar Analisar / Executar Inspeção por Nome (Ctrl+Alt+Shift+I) para aplicar esta inspeção ao projeto inteiro.
