Relata cláusulas else redundantes em instruções if que contêm um return.

Exemplo:


  if someCode() {
      fmt.Println("branch A")
      return nil
  } else {
      fmt.Println("branch B")
  }

Após a aplicação da correção rápida:


  if someCode() {
      fmt.Println("branch A")
      return nil
  }
  fmt.Println("branch B")
