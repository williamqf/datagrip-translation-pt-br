Relata a instanciação de objetos java.util.Map cujo tipo de chave é uma classe enum. Tais objetos java.util.Map podem ser substituídos por objetos java.util.EnumMap.

A implementação java.util.EnumMap pode ser muito mais eficiente, pois a estrutura de dados subjacente é um array simples.

Exemplo:


  Map<MyEnum, String> myEnums = new HashMap<>();

Após aplicar a correção rápida:


  Map<MyEnum, String> myEnums = new EnumMap<>(MyEnum.class);
