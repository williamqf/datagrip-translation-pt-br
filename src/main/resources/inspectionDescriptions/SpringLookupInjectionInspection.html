Relata referências de bean incorretas no parâmetro value da anotação @Lookup.

Exemplo:


  @Component public class FooBean {...}
  @Component public class OtherBean {...}

  @Component public class MyComponent {
  @Lookup("fooBean")
  public FooBean fooBean() {...}

  @Lookup("fooBean") // relata "O bean deve ser do tipo 'OtherBean'"
  public OtherBean otherBean() {...}

  @Lookup("unknown") // relata "Não foi possível resolver o bean 'unknown'"
  public OtherBean fooBean() {...}
  }
