Relata classes permitidas que não estendem uma classe base selada (sealed).

O Groovy não exige que todas as classes permitidas estejam disponíveis no momento da compilação e sejam compiladas junto com a classe base. O compilador não avisa o usuário ao processar subclasses permitidas que não realizam a extensão, mas isso contradiz a natureza das classes sealed.

Exemplo:


  class A permits B {} // B é relatada
  class B {}
