Relata recursos JNDI que não são fechados com segurança. Os recursos JNDI relatados por esta inspeção incluem javax.naming.InitialContext e javax.naming.NamingEnumeration.

Por padrão, esta inspeção assume que o recurso pode ser fechado por qualquer método com 'close' ou 'cleanup' em seu nome.

Exemplo:


  Object findObject(Properties properties, String name) throws NamingException {
    Context context = new InitialContext(properties); // o contexto não foi fechado
    return context.lookup(name);
  }

Configure a inspeção usando as seguintes opções:

  • Se é permitido abrir recursos JNDI dentro de blocos try. Este estilo é menos desejável por ser mais verboso do que abrir o recurso antes do bloco try.
  • Se o recurso pode ser fechado por qualquer chamada de método que o passe como um argumento.
