Relata instruções if que podem ser simplificadas em uma única atribuição ou em uma instrução return ou assert.

Exemplo:


  if (foo()) {
    return true;
  } else {
    return false;
  }

Após a aplicação da correção rápida:


  return foo();

Configuração da inspeção:

Use a opção Ignorar instruções 'if' encadeadas se desejar ocultar avisos para instruções if encadeadas.

Por exemplo, no código a seguir, o aviso será ocultado, mas a correção rápida ainda estará disponível:


  if (condition1) return true;
  if (condition2) return false;
  return true;

Observe que, se a condição tiver efeitos colaterais, substituir if (isTrue()) assert false; por assert isTrue(); pode alterar a semântica do programa quando o assert estiver desativado. Se desejar ocultar avisos para instruções if que contêm apenas uma instrução assert em seu corpo, use a opção Ignorar instruções 'if' com 'assert' trivial.
