Relata as seguintes violações da especificação EJB:
  • Tipo de propriedade injetada inválido
  • A referência EJB é inconsistente com o `ejb-jar.xml`

Exemplo:


@LocalBean
@Stateless
public class ExampleBean {

  private AnotherBean anotherBean;

  @Inject
  public void getAnotherBean() { // Erro: a injeção deve ser realizada em um campo ou método setter
    return anotherBean;
  }

  ...
}
