Relata expressões dinâmicas onde expressões adiadas são esperadas ou expressões adiadas onde expressões dinâmicas são esperadas.

Exemplo:



<jsp:useBean id="exampleBean" scope="request" type="java.util.Date"/>

<h:outputText id="${exampleBean.day}" value="#{exampleBean.day}" /> // OK
<h:outputText id="#{exampleBean.day}" value="${exampleBean.day}" /> // Erro: o atributo `id` requer uma expressão dinâmica, `value` - adiada
