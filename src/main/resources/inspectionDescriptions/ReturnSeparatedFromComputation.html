Relata instruções return que retornam variáveis locais cujo valor é calculado em outro local no mesmo método.

A correção rápida incorpora a variável retornada movendo a instrução return para o local onde o valor da variável é calculado. Quando o valor de retorno não puder ser incorporado diretamente na instrução return, a correção rápida tenta mover a instrução return o mais próximo possível de onde o valor de retorno é calculado.

Exemplo:


  int n = -1;
  for (int i = 0; i < a.length; i++) {
    if (a[i] == b) {
      n = i;
      break;
    }
  }
  return n;

Após aplicar a correção rápida:


  int n = -1;
  for (int i = 0; i < a.length; i++) {
    if (a[i] == b) {
      return i;
    }
  }
  return n;
