Relata métodos declarados na mesma classe que possuem o mesmo nome e o mesmo número de parâmetros. Tais sobrecargas podem ser muito confusas, pois pode não estar claro qual sobrecarga está sendo chamada.

Exemplo:


  class Main {
    public static void execute(Runnable r) {}
    public static <T> void execute(RunnableFuture<T> c) {}
  }

Use a opção para ignorar métodos sobrecarregados com tipos de parâmetros que sejam claramente incompatíveis.
