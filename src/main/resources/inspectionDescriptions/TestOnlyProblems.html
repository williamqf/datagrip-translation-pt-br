Relata métodos e classes anotados com @TestOnly e @VisibleForTesting que são usados no código de produção. Também relata casos onde @TestOnly e @VisibleForTesting são aplicados ao mesmo elemento.

Os problemas não são relatados se o método ou classe for referenciado a partir de:

  • Código sob a pasta Test Sources
  • Classes de teste (JUnit/TestNG)
  • Outro método anotado com @TestOnly

Exemplo (no código de produção):


  @TestOnly
  fun foo() { ... }

  fun main () {
    foo()
  }
