Relata classes que implementam Serializable e não declaram um campo serialVersionUID.

Sem um campo serialVersionUID, quaisquer alterações feitas na classe tornarão as versões serializadas anteriormente ilegíveis.

Exemplo:


  class Main implements Serializable {
  }

Após aplicar a correção rápida:


  class Main implements Serializable {
      private static final long serialVersionUID = -1446398935944895849L;
  }
Ao usar um nível de linguagem do JDK 14 ou superior, a correção rápida também adicionará a anotação java.io.Serial.

Configure a inspeção usando as seguintes opções:

  • Liste as classes cujos sucessores esta inspeção não deve relatar. Isso se aplica a classes que herdam Serializable de uma superclasse, mas não se destinam à serialização.
  • Se as classes anônimas Serializable devem ser ignoradas.
