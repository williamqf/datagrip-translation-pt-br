Relata literais de string entre aspas duplas que podem ser convertidos em literais de string brutos (raw), bem como literais de string brutos que podem ser convertidos em literais de string entre aspas duplas.

Exemplo:


  var s = "Hello\n   \"World\""

Após a aplicação da correção rápida:


  var s = `Hello
   "World"`

Existem dois pontos a serem considerados ao converter strings entre aspas duplas em strings brutas:

  • De acordo com a especificação da linguagem Go, caracteres de retorno de carro ('\r') são descartados de valores de string brutos. Como não há como exibir um retorno de carro em uma string bruta, descartamos o caractere de retorno de carro ao converter uma string entre aspas duplas em uma string bruta.
  • O caractere de crase ('`') não pode estar presente em um literal de string bruto, pois não há como escapá-lo. Portanto, uma string entre aspas duplas contendo um caractere de crase não pode ser convertida em uma string bruta e resultará em um erro de sintaxe. O usuário pode decidir desfazer a conversão ou usar alguma forma de concatenação para preservar a crase da string entre aspas duplas. Por exemplo, "`ab``" produziria `ab``, o que é sintaticamente incorreto. O usuário pode desfazer a operação ou alterar a string resultante para `ab` + "`".
