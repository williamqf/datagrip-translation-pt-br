Relata declarações de classes e objetos com corpos vazios.

Use a correção rápida 'Remover corpo de classe vazio redundante' para limpar o código.

Exemplo:


  class EmptyA() {} // <== corpo vazio

  class EmptyB {
      companion object {} // <== corpo vazio
  }

  fun emptyC() {
     object {} // <== objeto anônimo, ok (não relatado)
  }

Após aplicar a correção rápida:


  class EmptyA()

  class EmptyB {
      companion object
  }

  fun emptyC() {
     object {}
  }
