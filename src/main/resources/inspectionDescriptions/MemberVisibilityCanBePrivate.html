Relata declarações que podem ser definidas como private para seguir o princípio do encapsulamento.

Exemplo:


class Service(val url: String) {
    fun connect(): URLConnection = URL(url).openConnection()
}

Após aplicar a correção rápida (considerando que url não é utilizada fora da classe Service):


class Service(private val url: String) {
    fun connect(): URLConnection = URL(url).openConnection()
}
