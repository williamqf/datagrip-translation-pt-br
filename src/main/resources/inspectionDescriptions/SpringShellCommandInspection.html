Relata declarações incorretas de comandos do Spring Shell:

  • Descrição do comando ausente
  • O comando não está dentro de org.springframework.shell.standard.@ShellComponent

Exemplo:


// Todos os métodos serão relatados porque o @ShellComponent está ausente
class MathCommands {

  @ShellMethod("")
  public int sum(int a, int b) { // O método será relatado porque a descrição está vazia
    return a + b;
  }

  @ShellMethod
  public int mul(int a, int b) { // O método será relatado porque a descrição está ausente
    return a * b;
  }
}
