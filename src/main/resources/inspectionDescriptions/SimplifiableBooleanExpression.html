Relata expressões booleanas que podem ser simplificadas.

Exemplo:


  void f(boolean foo, boolean bar) {
    boolean b = !(foo ^ bar);
  }

Após a aplicação da correção rápida:


  void f(boolean foo, boolean bar) {
    boolean b = foo == bar;
  }

Exemplo:


  void f(boolean foo, boolean bar) {
    boolean b = (foo && bar) || !foo;
  }

Após a aplicação da correção rápida:


  void f(boolean foo, boolean bar) {
    boolean b = !foo || bar;
  }
