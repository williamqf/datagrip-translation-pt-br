Relata o uso dos decoradores @classmethod ou @staticmethod em métodos fora de uma classe.

Exemplo:


class State(object):

    @classmethod
    def my_state(cls, name):
        cls.name = name


@classmethod
def change_state(self):
    pass

O método change_state não deve usar o decorador @classmethod ou deve ser movido para a declaração da classe State.

Se a ação Remover decorador for aplicada, o código se tornará:


class State(object):

    @classmethod
    def my_state(cls, name):
        cls.name = name


def change_state(self):
    pass
