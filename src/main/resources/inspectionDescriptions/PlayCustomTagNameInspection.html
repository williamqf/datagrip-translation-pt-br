Inspeciona tags personalizados inexistentes.

Você pode criar tags personalizadas para seu aplicativo usando arquivos de modelo simples armazenados no diretório app/views/tags. O nome do arquivo do modelo é usado como nome da tag. Por exemplo, para criar a tag hello, basta criar o arquivo app/views/tags/hello.html. Faça referência direta à tag: #{hello /}.

Esta inspeção reportará a referência #{hello /} se app/views/tags/hello.html não existir.
