Relata valores numéricos retornados de métodos que não estão em conformidade com o intervalo de retorno declarado para o método. Você pode usar várias anotações para declarar o intervalo de retorno de um método:
  • org.jetbrains.annotations.Range do pacote de anotações da JetBrains (especifica 'from' e 'to')
  • org.checkerframework.common.value.qual.IntRange do pacote de anotações do Checker Framework (especifica 'from' e 'to')
  • org.checkerframework.checker.index.qual.GTENegativeOne do pacote de anotações do Checker Framework (o intervalo é '>= -1')
  • org.checkerframework.checker.index.qual.NonNegative do pacote de anotações do Checker Framework (o intervalo é '>= 0')
  • org.checkerframework.checker.index.qual.Positive do pacote de anotações do Checker Framework (o intervalo é '> 0')
  • javax.annotation.Nonnegative do pacote de anotações JSR 305 (o intervalo é '>= 0')
  • javax.validation.constraints.Min (especifica o valor mínimo)
  • javax.validation.constraints.Max (especifica o valor máximo)

Exemplo:


  @Range(from = 0, to = Integer.MAX_VALUE) int getValue() {
    // Aviso: -1 está fora do intervalo declarado
    return -1;
  }
