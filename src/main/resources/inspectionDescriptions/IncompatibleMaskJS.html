Reporta expressões de máscara bit a bit que avaliam obrigatoriamente como true ou false. A expressão possui o formato (var & constant1) == constant2 ou (var | constant1) == constant2, onde constant1 e constant2 são constantes de máscara de bits incompatíveis.

Exemplo:


// Máscara incompatível: como o último byte na máscara é zero,
// 0x1200 é possível, mas 0x1234 é impossível
if ((mask & 0xFF00) == 0x1234) {...}
