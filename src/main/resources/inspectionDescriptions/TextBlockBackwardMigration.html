Relata blocos de texto que podem ser substituídos por literais de string regulares. Esta inspeção ajuda no downgrade para compatibilidade com versões anteriores do Java.

Exemplo:


  Object obj = engine.eval("""
                           function hello() {
                             print('"Hello, world"');
                           }

                           hello();
                           """);

Após a aplicação da correção rápida:


  Object obj = engine.eval("function hello() {\n" +
                           "  print('\"Hello, world\"');\n" +
                           "}\n" +
                           "\n" +
                           "hello();\n");

Novidade na versão 2019.3
