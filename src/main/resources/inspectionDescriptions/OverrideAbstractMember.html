Reporta funções que sobrescrevem membros abstratos, mas que não têm um modificador de sobrescrita.

Scala não exige que um desenvolvedor use override em métodos que implementam membros abstratos. No entanto, é uma boa ideia marcar esses métodos com override para ser notificado pelo compilador quando a relação de sobrescrita for quebrada (por exemplo, por meio de uma refatoração subsequente).

Exemplo:


  trait Base {
    def fun(): Int
  }

  class Impl extends Base {
    def fun(): Int = 42
  }

Após a correção rápida ser aplicada:


  trait Base {
    def fun(): Int
  }

  class Impl extends Base {
    override def fun(): Int = 42
  }
