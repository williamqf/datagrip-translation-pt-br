Relata pacotes não resolvidos na anotação @ComponentScan e nos atributos de alias correspondentes da meta-anotação @ComponentScan.

Exemplo:


  @ComponentScan(basePackages = {
     "com.my.company",
     "com.unknown" // Relata "Não é possível resolver o pacote 'unknown'"
  }) +
  @Configuration +
  public class MyConfiguration {}
