Relata o uso de código de estilo pseudofuncional do Guava quando a Java Stream API está disponível.

Embora a Guava Iterable API forneça funcionalidades semelhantes à Java Stream API, ela é ligeiramente diferente e pode carecer de alguns recursos. Em particular, variantes de stream especializadas para tipos primitivos, como IntStream, têm melhor desempenho que as variantes genéricas.

Exemplo:


List<String> transformedIterable = Iterables.transform(someList, someTransformFunction);//Aviso: código de estilo pseudofuncional

Após a aplicação da correção rápida:

List<String> transformedIterable = someList.stream().map(someTransformFunction).collect(Collectors.toList());

Nota: A semântica do código pode mudar; por exemplo, o Iterable.transform do Guava produz um iterável com avaliação preguiçosa (lazy), mas a substituição realiza uma avaliação imediata (eager).
