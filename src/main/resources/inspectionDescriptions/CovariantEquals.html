Relata métodos equals() que aceitam um tipo de argumento diferente de java.lang.Object, caso a classe que os contém não possua outra sobrecarga de equals() que aceite java.lang.Object como seu tipo de argumento.

Uma versão covariante de equals() não sobrescreve o método Object.equals(Object). Isso pode resultar em comportamento inesperado em tempo de execução. Por exemplo, se a classe for usada para construir uma classe de coleção padrão, que espera que o método Object.equals(Object) seja sobrescrevido.

Exemplo:


  class Foo {
      public boolean equals(Foo foo) {  // aviso
          return false;
      }
  }
  class Bar {
      public boolean equals(Bar bar) {  // nenhum aviso aqui
          return false;
      }
      @Override
      public boolean equals(Object obj) {
          return false;
      }
  }
