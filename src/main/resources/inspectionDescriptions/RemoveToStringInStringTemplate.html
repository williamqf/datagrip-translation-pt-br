Relata chamadas a toString() em templates de string que podem ser removidas com segurança.

Exemplo:

  fun foo(a: Int, b: Int) = a + b

  fun test(): String {
      return "Foo: ${foo(0, 4).toString()}" // 'toString()' é redundante
  }

Após a aplicação do quick fix:

  fun foo(a: Int, b: Int) = a + b

  fun test(): String {
      return "Foo: ${foo(0, 4)}"
  }
