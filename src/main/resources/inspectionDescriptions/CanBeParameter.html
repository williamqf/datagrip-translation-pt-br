Relata parâmetros do construtor primário nos quais val ou var podem ser removidos.

Declarar propriedades de classe no construtor aumenta o consumo de memória. Se os valores dos parâmetros forem usados apenas no construtor, eles podem ser omitidos.

Observe que os objetos referenciados podem ser coletados pelo garbage collector antecipadamente.

Exemplo:


  class Task(val name: String) {
      init {
          print("Task created: $name")
      }
  }

A correção rápida remove a palavra-chave val ou var extra:


  class Task(name: String) {
      init {
          print("Task created: $name")
      }
  }
