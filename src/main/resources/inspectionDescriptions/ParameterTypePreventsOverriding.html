Relata tipos de parâmetro de métodos de subclasse que possuem o mesmo nome que os tipos de parâmetro do supermétodo correspondente, mas pertencem a pacotes diferentes. Nesses casos, o método da subclasse não pode sobrescrever o supermétodo.

Exemplo:


public class A {
 public void method(Object o) {}
}

public class B extends A {
 public void method(Object o) {} // aviso no tipo do parâmetro
 class Object {}
}

Após a aplicação da correção rápida:


public class A {
 public void method(Object o) {}
}

public class B extends A {
 public void method(java.lang.Object o) {} // novo tipo de parâmetro
 class Object {}
}
