Relata padrões de formato de data e hora inválidos.

Os seguintes erros são relatados:

  • Letras de padrão não suportadas, como "TT"
  • Uso de caracteres reservados, como "#"
  • Uso incorreto de preenchimento (padding)
  • Colchetes desbalanceados
  • Número incorreto de letras de padrão consecutivas

Exemplo:


  DateTimeFormatter.ofPattern("[][]]"); // ']' de fechamento sem '[' de abertura
  DateTimeFormatter.ofPattern("TT"); // letra de padrão 'T' inválida
  DateTimeFormatter.ofPattern("{"); // caractere reservado '{' utilizado
  DateTimeFormatter.ofPattern("MMMMMM"); // excesso de letras de padrão 'M' consecutivas
