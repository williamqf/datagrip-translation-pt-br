Relata strings de um único caractere usadas como argumentos em chamadas de String.indexOf() e String.lastIndexOf().

Sugere substituir tais literais de string por literais de caractere equivalentes por meio de uma correção rápida, proporcionando uma pequena melhoria de desempenho.

Exemplo:


  return s.indexOf("x");

Após a aplicação da correção rápida:


  return s.indexOf('x');
