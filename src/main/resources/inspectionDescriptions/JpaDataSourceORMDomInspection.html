Relata descritores XML de ORM de persistência que não podem ser resolvidos na fonte de dados configurada e sugere a atualização da fonte de dados ou a atribuição de uma fonte de dados diferente.

Exemplo de tabela de banco de dados:


  CREATE TABLE DATA_TABLE (
    ID INTEGER,
    DATA VARCHAR(100)
  );

Exemplo de arquivo JSP:


  <?xml version="1.0" encoding="UTF-8"?>
  <entity-mappings xmlns="http://java.sun.com/xml/ns/persistence/orm" version="1.0">
    <entity name="JavaEntity" class="JavaEntity">
        <table name="UNKNOWN_TABLE"/> // Erro: tabela não resolvida
        <attributes>
          <id name="id">
            <column name="ID" nullable="false"/>
          </id>
          <basic name="UNKNOWN_COLUMN"> // Erro: coluna não resolvida
            <column name="UNKNOWN_COLUMN" length="12"/>
          </basic>
        </attributes>
    </entity>
  </entity-mappings>

Observação: as fontes de dados podem ser configuradas por meio da ação `Atribuir fonte de dados`.
