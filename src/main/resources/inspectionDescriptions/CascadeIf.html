Relata instruções if com três ou mais ramificações que podem ser substituídas por uma expressão when com um sujeito.

Exemplo:


  fun translateNumber(n: Int): String {
    return if (n == 1) {
      "one"
    } else if (n == 2) {
      "two"
    } else {
      "???"
    }
  }

A correção rápida converte a expressão if em when:


  fun translateNumber(n: Int): String {
    return when (n) {
      1 -> {
        "one"
      }
      2 -> {
        "two"
      }
      else -> {
        "???"
      }
    }
  }
