Relata operadores ternários aninhados. Operadores ternários aninhados podem ser difíceis de ler e, geralmente, é melhor usar pelo menos uma instrução if em seu lugar.

Exemplo:

min = x < y ? (x < z ? x : z) : (y < z ? y : z)

Após a aplicação da correção rápida, o resultado será assim: 
min = if x < y
  x < z ? x : z
else
  y < z ? y : z
end
