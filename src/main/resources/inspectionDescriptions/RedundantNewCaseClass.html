Relata o uso de new para instanciar classes de casos e sugere removê-lo.

O método apply é fornecido automaticamente para classes de casos.

Exemplo:


  case class Book(isbn: String)
  val guide = new Book("978-0345391803")

Após a aplicação da correção rápida:


  case class Book(isbn: String)
  val guide = Book("978-0345391803")
