Reporta caracteres latinos não básicos em literais e comentários. A inspeção fornece uma correção rápida para substituir tais caracteres por escapes Unicode em literais e por entidades HTML em comentários.

Exemplo:


  // © 2021
  char c = '©';
  String s = "Áî";

Após aplicar a correção rápida:


  // &copy; 2021
  char c = '\u00a9';
  String s = "\u00c1\u00ee";
Por padrão, esta inspeção não realça no editor e fornece apenas correções.
