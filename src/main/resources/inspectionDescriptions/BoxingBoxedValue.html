Relata o boxing de valores já encapsulados.

Esta é uma operação redundante, pois qualquer valor já encapsulado será primeiro submetido ao unboxing automático e, em seguida, encapsulado novamente. Se realizado em loops internos, esse tipo de código pode causar problemas de desempenho.

Exemplo:


  Integer value = 1;
  method(Integer.valueOf(value));

Após a aplicação da correção rápida:


  Integer value = 1;
  method(value);
