Relata chamadas para String.startsWith() e String.endsWith() onde literais de string de um único caractere são passados como argumentos.

Uma correção rápida é sugerida para substituir tais chamadas pelo String.charAt(), que possui uma implementação mais eficiente.

No entanto, o ganho de desempenho dessa alteração é mínimo e reduz a legibilidade do código devido à verificação adicional de comprimento diferente de zero; recomenda-se aplicar esta correção rápida apenas dentro de loops intensos.

Esta inspeção é destinada ao Java ME e outros ambientes com recursos altamente limitados. A aplicação dos resultados desta inspeção sem critério pode afetar negativamente a clareza e o design do código.

Exemplo:


  boolean startsWithX(String s) {
    return s.startsWith("x");
  }

Após a aplicação da correção rápida:


  boolean startsWithX(String s) {
    return !s.isEmpty() && s.charAt(0) == 'x';
  }
