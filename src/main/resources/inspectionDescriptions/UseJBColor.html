Relata o uso de java.awt.Color.

Isso pode não detectar temas "escuros" (por exemplo, o "Darcula" integrado), o que pode resultar em uma aparência desagradável da UI.

A correção rápida substitui o uso por JBColor, que permite definir variantes de cores "escuras".

Exemplo:


// Incorreto:
Color darkGreen = new Color(12, 58, 27);
Color blue = Color.BLUE;

// Correto:
Color darkGreen = new JBColor(12, 58, 27);
Color blue = JBColor.BLUE;
Color green = new JSColor(new Color(12, 58, 27), new Color(27, 112, 39));
