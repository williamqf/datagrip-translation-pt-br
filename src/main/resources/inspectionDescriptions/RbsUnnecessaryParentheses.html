Relata parênteses desnecessários em tipos de função. Funções sem parâmetros não precisam de parênteses.

Uma correção rápida pode ser usada para remover os parênteses desnecessários.

Exemplo:

class Example
  # parênteses desnecessários no tipo de função
  def call: () -> untyped
end

Após a aplicação da correção rápida:

class Example
  def call: -> untyped
end
