Relata casos em que a chamada dinâmica do construtor poderia ser substituída por uma chamada de método de fábrica.

Exemplo:


  import akka.actor._
  
  class Foo extends AbstractActor
  
  object Foo {
    def props(): Props = Props(new Foo())
  }
  
  Props(classOf[Foo])

Depois que o quick-fix é aplicado:


  import akka.actor._
  
  class Foo extends AbstractActor
  
  object Foo {
    def props(): Props = Props(new Foo())
  }
  
  Foo.props()
