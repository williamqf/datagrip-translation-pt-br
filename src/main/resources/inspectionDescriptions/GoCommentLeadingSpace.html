Relata comentários que não possuem um espaço inicial.

Observe que esta inspeção só funciona se você selecionar Adicionar espaço inicial aos comentários na aba Outros em Editor | Estilo de Código | Go.

Comentários com espaços iniciais são mais fáceis de ler, pois a primeira palavra é separada do marcador de comentário por um espaço.

Exemplo:

//imprime o JSON recebido da função createJSON
func printJSON(output []byte) {
	fmt.Println(string(output))
}
