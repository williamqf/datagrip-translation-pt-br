Denunciar utilização de retorno rotulado na última expressão em lambdas.

Este tipo de retorno é desnecessário e pode ser removido com segurança.

Exemplo:


fun foo() {
  listOf(1,2,3).find {
    return@find true
  }
}

Após a aplicação da correção rápida:


fun foo() {
  listOf(1,2,3).find {
    true
  }
}
