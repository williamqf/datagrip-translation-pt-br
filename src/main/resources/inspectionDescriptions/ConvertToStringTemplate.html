Relata concatenações de string que podem ser convertidas em templates de string.

O uso de templates de string é recomendado, pois torna o código mais fácil de ler.

Exemplo:


  fun example() {
      val capitals = mapOf("France" to "Paris", "Spain" to "Madrid")
      for ((country, capital) in capitals) {
          print(capital + " is a capital of " + country)
      }
  }

Após a aplicação da correção rápida:


  fun example() {
      val capitals = mapOf("France" to "Paris", "Spain" to "Madrid")
      for ((country, capital) in capitals) {
          print("$capital is a capital of $country")
      }
  }
