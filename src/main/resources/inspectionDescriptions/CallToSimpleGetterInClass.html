Relata chamadas a getters de propriedade simples de dentro da própria classe da propriedade.

Um getter de propriedade simples é definido como aquele que apenas retorna o valor de um campo, sem realizar outros cálculos. Tais chamadas de getter simples podem ser transformadas em inline com segurança usando a correção rápida. Para manter a clareza do código, alguns padrões de codificação também sugerem não utilizar getters simples internamente.

Exemplo:


  public class Salient {
    private String name;

    public String getName() {
      return name;
    }

    @Override
    public String toString() {
      return getName();
    }
  }

Após a aplicação da correção rápida:


  public class Salient {
    private String name;

    public String getName() {
      return name;
    }

    @Override
    public String toString() {
      return name;
    }
  }

Configure a inspeção usando as seguintes opções:

  • Se deve relatar apenas chamadas de getter em this, em vez de objetos do mesmo tipo passados como parâmetros.
  • Se deve ignorar getters que não sejam private.
