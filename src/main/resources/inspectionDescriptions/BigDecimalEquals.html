Relata chamadas equals() que comparam dois números java.math.BigDecimal. Isso geralmente é um erro, pois dois números java.math.BigDecimal são considerados iguais apenas se o valor e a escala forem iguais.

Exemplo:


  if (new BigDecimal("2.0").equals(
    new BigDecimal("2.00"))) {} // false

Após a aplicação do quick fix:


  if (new BigDecimal("2.0").compareTo(
    new BigDecimal("2.00")) == 0) {} // true
