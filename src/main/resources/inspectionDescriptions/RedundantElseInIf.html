Relata um else redundante em instruções if que possuem um return.

Exemplo:


  fun foo(arg: Boolean): Int {
      if (arg) return 0
      else { // Este else é redundante, o código entre chaves pode simplesmente ser movido para a esquerda
          someCode()
      }
  }

Após aplicar a correção rápida:


  fun foo(arg: Boolean): Int {
      if (arg) return 0
      someCode()
  }
