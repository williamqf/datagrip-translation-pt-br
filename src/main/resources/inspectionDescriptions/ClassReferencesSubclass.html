Relata classes que contêm uma referência a uma de suas subclasses. Tais referências podem ser confusas e violar várias regras de design orientado a objetos.

Exemplo:


  class Entity {
    // Aviso: a classe referencia sua subclasse
    void compare(SimpleEntity entity) {
      ...
    }
  }
  class SimpleEntity extends Entity {
    ...
  }
