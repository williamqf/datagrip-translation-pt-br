Relata problemas que podem ser causados por extensões EL não padrão: expressões JSF EL fora de atributos, expressões EL não padrão, etc.

Exemplo:



<f:view>
  <h:form>
    <f:loadBundle var="example"/>
    <h:commandButton action="${example.size()}"/> // Aviso: a chamada de método é uma extensão não padrão
  </h:form>
</f:view>
