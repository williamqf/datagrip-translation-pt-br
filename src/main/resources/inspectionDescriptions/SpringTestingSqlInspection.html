Relata referências de arquivo não resolvidas no atributo scripts da anotação @Sql e nos atributos de alias correspondentes na meta-anotação @Sql.

Exemplo:


  @SqlGroup({
     @Sql("not-found"), // Relata "Não foi possível resolver o arquivo 'not-found'"
     @Sql("found.sql")
  })
  public class MyTestWithSqlData {...}
