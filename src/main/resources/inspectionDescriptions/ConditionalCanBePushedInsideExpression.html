Relata expressões condicionais com ramos then e else que são suficientemente semelhantes para que a expressão possa ser movida para dentro. Esta ação encurta o código.

Exemplo:


  double g(int a, int b) {
    return a == b ? Math.cos(0) : Math.cos(1);
  }

Após aplicar a correção rápida:


  double g(int a, int b) {
    return Math.cos(a == b ? 0 : 1);
  }

Novidade na versão 2017.2
