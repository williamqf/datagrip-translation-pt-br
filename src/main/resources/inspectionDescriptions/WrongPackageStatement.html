Detecta instruções package que não correspondem à estrutura de diretórios do projeto. Além disso, relata classes sem uma instrução package que não estejam localizadas diretamente no diretório raiz de código-fonte.

Embora a linguagem Java não exija isso estritamente, é recomendável manter as classes do pacote com.example.myapp no diretório com/example/myapp diretamente sob a raiz de código-fonte. Caso contrário, isso pode confundir quem lê o código e fazer com que algumas ferramentas não funcionem corretamente.
