Relata chamadas a métodos native dentro de blocos ou métodos synchronized.

Se possível, recomenda-se manter as chamadas a métodos native fora de contextos sincronizados, pois tais chamadas induzem trocas de contexto dispendiosas e podem causar problemas de desempenho.

Exemplo:


  native void nativeMethod();

  void example(){
    synchronized (lock){
      nativeMethod();//aviso
    }
  }
