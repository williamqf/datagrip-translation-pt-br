Relata qualificadores super desnecessários em chamadas de método e referências de campo.

Quando um campo ou método de uma superclasse não está oculto ou sobrescrito na classe de chamada, o qualificador super não é necessário.

Exemplo:


  class Foo {
    void foo() {}
  }

  class Bar extends Foo {
    void bar() {
      super.foo();
    }
  }

Após a aplicação da correção rápida:


  class Foo {
    void foo() {}
  }

  class Bar extends Foo {
    void bar() {
      foo();
    }
  }

Use as configurações de inspeção para ignorar qualificadores que ajudam a distinguir o acesso a membros da superclasse de membros de classes externas com o mesmo nome.

Consulte também as seguintes inspeções:

  • Java | Visibilidade | Acesso a campo herdado se assemelha ao acesso a elementos no código circundante
  • Java | Visibilidade | Chamada de método herdado se assemelha a uma chamada de método local
