Informa as definições com uma anotação @targetName que sobrescrevem definições sem uma anotação @targetName.

Veja A anotação @targetName.

A correção rápida remove a anotação @targetName.

Exemplo:


  class A:
    def foo: Int = 1

  class B extends A:
    @targetName("overriddenFoo")
    override def foo: Int = 2

Após a correção rápida ser aplicada:


  class A:
    def foo: Int = 1

  class B extends A:
    override def foo: Int = 2
