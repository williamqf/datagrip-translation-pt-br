Relata violações das convenções de nomenclatura PEP8.

Exemplo:


class mammalia(object):
    extremities = 4

    def feeds(self):
        print("milk")

Neste trecho de código, a IDE sugere renomear mammalia para Mammalia. Após a aplicação do ajuste rápido, o código é alterado para:


class Mammalia(object):
    extremities = 4

    def feeds(self):
        print("milk")
