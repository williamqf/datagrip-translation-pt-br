Relata situações em que valores NULL são inseridos em colunas que aceitam apenas valores NOT NULL.

Exemplo (Microsoft SQL Server):

CREATE TABLE br2 (
id INT NOT NULL,
col1 NVARCHAR (20) NOT NULL,
col2 NVARCHAR (20) NOT NULL,
);
--
INSERT INTO br2 (id, col1, col2)
VALUES (1, NULL, NULL);

Você não pode inserir valores NULL em col1 e col2, pois elas são definidas como NOT NULL. Se você executar o script como está, receberá uma mensagem de erro. Para corrigir este código, substitua o NULL na parte VALUES por valores (por exemplo, 42 e 'bird').

INSERT INTO br2 (id, col1, col2)
VALUES (1, 42, 'bird');
