Relata parâmetros de tipo que possuem o mesmo nome de um tipo visível no escopo atual. Esses nomes de parâmetro podem ser confusos.

Exemplo:


  abstract class MyList<T> extends AbstractList<T> {
    private List<T> elements;
    // O parâmetro de tipo 'T' oculta o parâmetro de tipo 'T'
    public <T> T[] toArray(T[] array) {
        return elements.toArray(array);
    }
}
