Relata referências inválidas dentro da anotação org.springframework.shell.standard.ShellMethodAvailability do Spring Shell.

Exemplo:


@ShellComponent
public class MyCommands {

    @ShellMethod("sum")
    @ShellMethodAvailability("sumAvailability") // O valor da anotação será realçado como um símbolo não resolvido
    public int sum(int a, int b) {
        return a + b;
    }

    @ShellMethodAvailability("sumMethod") // O valor da anotação será realçado como um símbolo não resolvido
    public Availability isSumAvailable() {
        return Availability.available();
    }
}
