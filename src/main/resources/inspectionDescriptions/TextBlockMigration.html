Relata concatenações de String que podem ser simplificadas substituindo-as por blocos de texto.

Requisitos:

  • \n ocorre duas ou mais vezes.
  • Os blocos de texto não estão concatenados.

Use a opção Reportar literais de string únicos para realçar literais únicos que contenham quebras de linha. A correção rápida permanece disponível mesmo se esta opção estiver desativada.

Exemplo:


  String html = "<html>\n" +
                "    <body>\n" +
                "        <p>Hello, world</p>\n" +
                "    </body>\n" +
                "</html>\n";

Após aplicar a correção rápida:


  String html = """
                <html>
                    <body>
                        <p>Hello, world</p>
                    </body>
                </html>
                """;
