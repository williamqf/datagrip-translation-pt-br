Relata operações de deslocamento em que o valor do deslocamento é uma constante fora do intervalo razoável.

Operações de deslocamento de inteiros fora do intervalo 0..31 e operações de deslocamento de long fora do intervalo 0..63 são relatadas. Deslocar por um valor negativo ou excessivamente grande quase certamente indica um erro de codificação.

Exemplo:


  int shiftSize = 32;
  // Aviso: o deslocamento de 32 bits é equivalente ao deslocamento de 0 bits, portanto não há deslocamento.
  int mask = (1 << shiftSize) - 1;
