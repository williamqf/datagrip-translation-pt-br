Relata expressões não utilizadas, o que geralmente indica que o código está incompleto ou que uma instrução return está ausente.

Se a expressão não utilizada for a última instrução em uma função e seu tipo corresponder ao tipo de retorno da função, uma correção rápida adicionará automaticamente a palavra-chave return.

Exemplo:


fun example(): Int {
    val x = 10
    x // Aviso: a expressão não é utilizada
}

Após a aplicação da correção rápida:


fun example(): Int {
    val x = 10
    return x
}
