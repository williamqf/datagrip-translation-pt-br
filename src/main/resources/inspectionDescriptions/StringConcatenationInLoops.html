Relata a concatenação de String dentro de loops.

Como cada concatenação de String copia a string inteira, geralmente é preferível substituí-la por uma chamada explícita a StringBuilder.append() ou StringBuffer.append().

Exemplo:


  String str = "";
  for(int i=0; i<10; i++) {
    str += i;
  }

Após aplicar a correção rápida:


  String str = "";
  StringBuilder strBuilder = new StringBuilder(str);
  for(int i = 0; i<10; i++) {
    strBuilder.append(i);
  }
  str = strBuilder.toString();

Às vezes, uma variável String pode ser convertida em um StringBuilder ou um novo StringBuilder pode ser introduzido por meio desta correção rápida. Deve-se ter cuidado quando o código original trata valores null especificamente, pois a substituição pode alterar a semântica. Se valores null forem possíveis, recomenda-se uma correção segura para nulos (null-safe), gerando as verificações de nulo necessárias. Além disso, não há garantia de que a substituição automática sempre resultará em um melhor desempenho.
