Relata referências a métodos e campos static por meio de uma instância de classe em vez da própria classe.

Embora a especificação da linguagem PHP permita referenciar membros static por meio de variáveis de instância, isso pode tornar o código confuso, pois os leitores podem acreditar que o método depende da instância.

A correção rápida substitui a variável de instância pelo nome da classe.

Exemplo:


  class StaticClass {
    public static function someStaticFunction(): int
    {
        return 3;
    }

  }

  $var = new StaticClass
  $var::someStaticFunction();

Após aplicar a correção rápida:


  StaticClass::someStaticFunction()
