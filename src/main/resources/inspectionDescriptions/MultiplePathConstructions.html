Relata quando múltiplos java.nio.file.Path consecutivos podem ter sua construção (java.nio.file.Paths.get ou java.nio.file.Path.of) substituída por uma única variável java.nio.file.Path.

Exemplo:


if (Files.isRegularFile(Path.of(fileName))) {
  try(InputStream is = Files.newInputStream(Path.of(fileName))) {
    // algum código
  }
  catch (IOException e) {
    throw new UncheckedIOException(e);
  }
}

Após aplicação da correção rápida:


Path path = Path.of(fileName);
if (Files.isRegularFile(path)) {
  try(InputStream is = Files.newInputStream(path)) {
    // algum código
  }
  catch (IOException e) {
    throw new UncheckedIOException(e);
  }
}

Essa inspeção só é relatada quando o nível da linguagem do projeto ou módulo é 7 ou superior.
