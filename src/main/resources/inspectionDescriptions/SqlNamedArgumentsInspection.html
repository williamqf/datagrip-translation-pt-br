Relata argumentos usados sem nomes em chamadas de rotina. Por padrão, esta inspeção está desabilitada.

Para obter mais detalhes sobre a diferença entre parâmetros nomeados e não nomeados, consulte Binding Parameters by Name (Named Parameters) no docs.microsoft.com.

Exemplo (Microsoft SQL Server):

CREATE FUNCTION foo(n INT, m INT) RETURNS INT AS
BEGIN
    RETURN n + m;
END;

CREATE PROCEDURE test AS
BEGIN
    foo n = 1, m = 2;

--- A chamada a seguir não possui nomes de parâmetros e será destacada
    foo 1, 2;
END;

Os parâmetros 1, 2 na chamada foo 1, 2; são destacados porque não possuem nomes.
