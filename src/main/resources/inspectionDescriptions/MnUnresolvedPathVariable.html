Relata parâmetros @PathVariable que são declarados na assinatura do método, mas estão ausentes no caminho da URL, ou que são declarados no caminho da URL, mas estão ausentes na assinatura do método. A correção rápida adiciona o parâmetro ausente.

Exemplo:


  @Get("/path/{myVariable}/")
  public String handler(@PathVariable String name_is_not_equal_to_myVariable) {
    return "...";
  }

Após a aplicação da correção rápida, o resultado será o seguinte:


  @Get("/path/{myVariable}/")
  public String handler(@PathVariable String myVariable) {
    return "...";
  }
