Relata expressões lambda que podem ser substituídas por classes anônimas.

A expansão de uma expressão lambda para uma classe anônima pode ser útil se for necessário implementar métodos adicionais dentro da classe anônima.

Exemplo:


  s -> System.out.println(s)

Após a aplicação da correção rápida:

new Consumer<String>() {
  @Override
  public void accept(String s) {
    System.out.println(s);
  }
}
