Relata quaisquer referências a classes internas que sejam qualificadas desnecessariamente pelo nome da classe envolvente.

Essa qualificação pode ser removida com segurança, o que às vezes adiciona uma importação para a classe interna.

Exemplo:


  class X {
     X.Y foo;
     class Y{}
  }

Após a aplicação da correção rápida:


  class X {
     Y foo;
     class Y{}
  }

Use a opção Ignorar referências que exigem importação para ignorar as referências a classes internas onde a remoção da qualificação adicionaria uma importação.
