Relata chamadas de containsAll() em java.util.List.

A complexidade de tempo desta chamada de método é O(n·m), onde n é o número de elementos na lista em que o método é chamado e m é o número de elementos na coleção passada como parâmetro para o método. Esta pode ser uma operação dispendiosa quando a lista é grande.

A correção rápida envolverá a lista em um new java.util.HashSet<>(), pois o tempo necessário para criar um java.util.HashSet a partir de uma java.util.List e executar containsAll() no java.util.HashSet é O(n+m).

Exemplo:

  public boolean check(List<String> list, Collection<String> collection) {
    // Complexidade O(n·m)
    return list.containsAll(collection);
  }

Após a aplicação da correção rápida:

  public boolean check(List<String> list, Collection<String> collection) {
    // Complexidade O(n+m)
    return new HashSet<>(list).containsAll(collection);
  }
