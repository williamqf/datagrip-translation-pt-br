Relatório de arquivos incorretos JPDL <process-definition>:

  • Referência a uma classe não-instanciável
  • Referência a uma classe que não estende a classe desejada
  • Referência a uma classe com escopo inadequado
  • Valor vazio de tag ou atributo
  • Valor de tag ou atributo que não corresponde ao padrão desejado (por exemplo, identificador Java)
  • Tag não contém uma sub-tag ou atributo obrigatório
  • Tag define um objeto com nome duplicado
