Relata palavras-chave else redundantes em instruções if—else e cadeias de instruções.

A palavra-chave else é redundante quando todas as ramificações anteriores terminam com uma instrução return, throw, break ou continue. Nesse caso, as instruções da ramificação else podem ser colocadas após a instrução if e a palavra-chave else pode ser removida.

Exemplo:


  if (name == null) {
      throw new IllegalArgumentException();
  } else {
      System.out.println(name);
  }

Após a aplicação da correção rápida:


  if (name == null) {
      throw new IllegalArgumentException();
  }
  System.out.println(name);

Desative a opção Relatar quando não houver instruções após a instrução 'if' para ignorar casos em que a instrução if—else seja a última instrução em um bloco de código.
