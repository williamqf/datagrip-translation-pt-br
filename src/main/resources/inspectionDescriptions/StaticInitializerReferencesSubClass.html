Relata classes que referenciam suas subclasses em inicializadores static ou campos static.

Tais referências podem causar um deadlock no nível da JVM em ambientes multithread (ou seja, quando uma thread tenta carregar a superclasse e outra tenta carregar a subclasse simultaneamente).

Exemplo:


  class Parent {
    static final Child field = new Child();
  }
  class Child extends Parent { }
