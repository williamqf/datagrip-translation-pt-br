Reporta declarações inconsistentes de aliases de classe e módulo. Classes não podem ser usadas como aliases de módulo, e módulos não podem ser usados como aliases de classe.

Uma classe usada como alias de módulo, ou um módulo usado como alias de classe, resultará em um InconsistentClassModuleAliasError.

Uma correção rápida está disponível para navegar até a declaração conflitante.

Exemplo: 


class User
end

# A classe 'User' é usada como destino do alias de módulo
module Customer = User
