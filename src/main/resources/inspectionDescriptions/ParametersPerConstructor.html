Relata construtores cujo número de parâmetros excede o limite máximo especificado. Tais objetos são difíceis de instanciar, especialmente se alguns dos parâmetros forem opcionais. Um construtor com excesso de parâmetros pode indicar a necessidade de refatoração. Por exemplo, considere a aplicação do padrão Builder.

Exemplo:


  public BankAccount(long accountNumber,
                     String owner,
                     double balance,
                     double interestRate) {
    // inicialização dos campos
  }

Configuração da inspeção:

  • Use o campo Limite de parâmetros para especificar o número máximo de parâmetros permitido em um construtor.
  • Use a lista Ignorar construtores com as seguintes visibilidades para especificar se a inspeção deve ignorar construtores com visibilidades específicas.
