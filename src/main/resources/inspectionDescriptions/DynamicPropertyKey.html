Relata literais não string que são usados como chave de propriedade do bundle.

Isso é difícil de manter. É melhor passar strings de linguagem natural já resolvidas.

Exemplo:


  val key =
    if (???) "property.key.1"
    else "property.key.2"

  Bundle.message(key)

Use em vez disso:


  val string =
    if (???) Bundle.message("property.key.1")
    else Bundle.message("property.key.2")

  string
