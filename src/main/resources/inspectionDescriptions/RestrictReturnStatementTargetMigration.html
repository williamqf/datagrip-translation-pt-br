Relata rótulos que não apontam para uma função.

É proibido declarar rótulos de destino que não representem uma função.

A correção rápida remove o rótulo.

Exemplo:


  fun testValLabelInReturn() {
      L@ val fn = { return@L }
      fn()
  }

Após a aplicação da correção rápida:


  fun testValLabelInReturn() {
      L@ val fn = { return }
      fn()
  }

Esta inspeção relata apenas se o nível de linguagem do projeto ou módulo for 1.4 ou superior.
