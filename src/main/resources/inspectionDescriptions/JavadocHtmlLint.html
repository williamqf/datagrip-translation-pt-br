Relata os mesmos problemas de HTML em comentários Javadoc que o DocLint relata desde o Java 8.

Esta inspeção detecta os seguintes problemas:

  • Tags de fechamento automático, não fechadas, desconhecidas, mal posicionadas ou vazias
  • Atributos desconhecidos ou incorretos
  • Texto mal posicionado

Exemplo:


/**
 * Tag desconhecida: List<String>

 * Atributo incorreto ou vazio: <a href=""></a>
 * Tag de fechamento automático: <br/>
 * ...
 */
