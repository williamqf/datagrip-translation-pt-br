Relata instruções if, when e try que podem ser convertidas em uma expressão ao extrair a instrução return ou uma atribuição.

Exemplo:


  fun foo(arg: Int): String {
      when (arg) {
          0 -> return "Zero"
          1 -> return "One"
          else -> return "Multiple"
      }
  }
Após a aplicação da correção rápida: 

  fun foo(arg: Int): String {
      return when (arg) {
          0 -> "Zero"
          1 -> "One"
          else -> "Multiple"
      }
  }

Se você deseja que esta inspeção realce códigos mais complexos com branches de múltiplas instruções, desmarque a opção "Relatar apenas se cada branch for uma única instrução".
