Relata hashes que usam strings como chaves. É preferível usar símbolos em vez de strings como chaves de hash.

Exemplo:

# Má prática
{ 'foo' => 1, 'bar' => 2 }
Uma correção rápida está disponível para converter todas as chaves de string em símbolos. Após a aplicação da correção rápida, o resultado será o seguinte: 
# Boa prática
{ foo: 1, bar: 2 }
