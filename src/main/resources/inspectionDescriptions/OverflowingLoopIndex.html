Relata loops que não podem ser concluídos sem overflow do índice ou loops que não realizam nenhuma iteração. Isso geralmente ocorre devido a um erro na operação de atualização.

Exemplo:


  void foo(int s) {
    for (int i = s; i > 12; i++) { // i-- deveria estar aqui
      System.out.println(i);
    }
  }
