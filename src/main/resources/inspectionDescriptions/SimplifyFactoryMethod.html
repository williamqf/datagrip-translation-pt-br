Relata casos em que a invocação dinâmica pode ser substituída por uma invocação do construtor.

Exemplo:


  import akka.actor._
  
  class Foo(foo: String, bar: Int) extends AbstractActor
  
  object Foo {
    def props(foo: String, bar: Int): Props = Props(classOf[Foo], foo, bar)
  }
  
  Foo.props("foo", 42)

Após a correção rápida ser aplicada:


  import akka.actor._
  
  class Foo(foo: String, bar: Int) extends AbstractActor
  
  object Foo {
    def props(foo: String, bar: Int): Props = Props(new Foo(foo, bar))
  }
  
  Foo.props("foo", 42)
