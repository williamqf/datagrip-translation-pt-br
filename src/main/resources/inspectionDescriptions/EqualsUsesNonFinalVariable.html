Relata implementações de equals() que acessam variáveis não final. Esse acesso pode fazer com que equals() retorne resultados diferentes em diferentes momentos do ciclo de vida do objeto, o que pode causar problemas ao usar classes de coleção padrão.

Exemplo:


  public class Person {
    private String lastName;

    @Override
    public boolean equals(Object obj) {
      ...
      Person other = (Person) obj;
      if (lastName == null) {
        if (!lastName.equals(other.lastName)) {
        return false;
      ...
      }
    }
  }
