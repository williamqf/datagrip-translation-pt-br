Relata métodos createOptionsPanel() em implementações de inspeção que podem ser convertidos automaticamente para getOptionsPane().

A criação de controles de opção de inspeção por meio de createOptionsPanel() foi depreciada em favor da descrição de controle declarativa getOptionsPane(). Esta inspeção verifica as implementações de createOptionsPanel() e sugere automaticamente a conversão para a nova API se elas usarem o InspectionOptionsPanel ou seus descendentes e forem suficientemente simples.

Esta inspeção suporta atualmente apenas Java e Kotlin. Não é possível converter códigos que utilizam métodos Kotlin como apply ou also. Tente fazer o inline deles.
