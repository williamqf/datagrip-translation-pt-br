Relata chamadas a setters de propriedades simples de dentro da classe da propriedade.

Um setter de propriedade simples é definido como aquele que apenas atribui o valor de seu parâmetro a um campo, sem realizar outros cálculos. Tais chamadas de setter simples podem ser incorporadas (inlined) com segurança. Para manter a clareza do código, alguns padrões de codificação também recomendam não usar setters simples.

Exemplo:


  class Foo {
  private int index;
    public Foo(int idx) {
       setIndex(idx);
    }
    public void setIndex(int idx) {
       index = idx;
    }
  }

Após a aplicação da correção rápida:


  class Foo {
    private int index;
    public Foo(int idx) {
      index = idx;
    }
    public void setIndex(int idx) {
      index = idx;
    }
  }

Configure a inspeção usando as seguintes opções:

  • Se deve relatar apenas chamadas de setter em this, e não em objetos do mesmo tipo passados como parâmetros.
  • Se deve ignorar setters que não sejam private.
