Sugere a substituição de for(Entry<?,?> entry : map.entrySet()) {...} ou map.entrySet().forEach(entry -> ...) por map.forEach((key, value) -> ...).

Exemplo:


  void print(Map<String, Integer> map) {
    map.entrySet().forEach(entry -> {
      String str = entry.getKey();
      System.out.println(str + ":" + entry.getValue());
    });
  }

Após a aplicação da correção rápida:


  void print(Map<String, Integer> map) {
    map.forEach((str, value) -> System.out.println(str + ":" + value));
  }

Quando a opção Não relatar loops estiver ativada, apenas os casos de entrySet().forEach() serão relatados. No entanto, a ação de correção rápida também está disponível para loops for.

Novidade na versão 2017.1
