Relata setters de propriedade redundantes.

Um setter é considerado redundante em um dos seguintes casos:

  1. O setter não possui corpo. A visibilidade do acessor não é alterada, a declaração não é external e não possui anotações. 
    
  var myPropWithRedundantSetter: Int = 0
      set // redundante

  var myPropA: Int = 0
      private set // OK - a visibilidade da propriedade foi alterada para private

  var myPropB: Int = 0
      external set // OK - não implementado no Kotlin (external)

  var myPropC: Int = 0
      @Inject set // OK - o acessor possui anotação
  2. O corpo do setter é um bloco com uma única instrução que atribui o parâmetro ao backing field. 
    
  var prop: Int = 0
      set(value) { // redundante
          field = value
      }
