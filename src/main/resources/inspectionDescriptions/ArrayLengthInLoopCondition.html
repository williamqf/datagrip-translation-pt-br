Relata o acesso à propriedade .length de um array na parte condicional de instruções de loop. Em ambientes com recursos altamente restritos, tais chamadas podem afetar negativamente o desempenho.

Esta inspeção é destinada ao Java ME e outros ambientes com recursos altamente restritos. Aplicar os resultados desta inspeção sem critério pode afetar negativamente a clareza e o design do código.

Exemplo:


  void foo(Object[] x) {
    for (int i = 0; i < x.length; i++) { /**/ }
  }
