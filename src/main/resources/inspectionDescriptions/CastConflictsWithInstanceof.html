Relata expressões de cast de tipo que são precedidas por uma verificação instanceof para um tipo diferente.

Embora isso possa ser intencional, tal construção é muito provavelmente um erro e resultará em uma java.lang.ClassCastException em tempo de execução.

Exemplo:


  class Main {
    int whenCharSequenceCastToNumber(Object o){
      if (o instanceof CharSequence) {
        return ((Number) o).intValue();
      }
      return 0;
    }

    int earlyReturnWhenNotCharSequence(Object o){
      if (!(o instanceof CharSequence)) return 0;
      return ((Number)o).intValue();
    }
  }
