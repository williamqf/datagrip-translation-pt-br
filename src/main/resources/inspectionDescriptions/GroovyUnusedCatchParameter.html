Relata parâmetros catch que não são usados nos blocos correspondentes. Esta inspeção não relata nenhum parâmetro catch chamado "ignore" ou "ignored".

Exemplo:


  try {
      def arr = new int[3]
      arr[5] = 5
  } catch(Exception ex) {
      println('Catching the exception')
  }

O parâmetro ex aqui nunca é usado no bloco catch.

Após a aplicação da correção rápida:


  try {
      def arr = new int[3]
      arr[5] = 5
  } catch(Exception ignored) {
      println('Catching the exception')
  }
