Relata quaisquer chamadas a métodos sobrescritos da classe atual durante a construção do objeto. Isso ocorre se a construção do objeto for interna a:
  • Construtores
  • Inicializadores de instância não static
  • Inicializadores de campo não static
  • clone()
  • readObject()
  • readObjectNoData()

Tais chamadas podem causar erros sutis, pois não há garantia de que o objeto tenha sido inicializado antes da chamada do método.

Exemplo:


  abstract class Parent {
    void someMethod() { }
  }

  class Child extends Parent {
    Child() {
      someMethod();
    }

    @Override
    void someMethod() { }
  }

Esta inspeção compartilha sua funcionalidade com as seguintes inspeções:

  • Método abstrato chamado durante a construção do objeto
  • Método sobrescrevível chamado durante a construção do objeto

Apenas uma inspeção deve ser ativada por vez para evitar avisos duplicados.
