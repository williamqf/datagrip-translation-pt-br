Relata declarações de tipo redundantes em literais compostos.

Exemplo:

nums := [][]int{[]int{1}, []int{2}}

Este é um slice de slices do tipo int. Nesse caso, você pode usar uma definição mais curta. Você pode corrigir este código manualmente ou usar a correção rápida Remover tipo redundante. Após aplicar a correção rápida, o código ficará assim:

nums := [][]int{{1},{2}}

Para mais informações sobre literais compostos, consulte a Go Language Specification: Composite Literals em go.dev.
