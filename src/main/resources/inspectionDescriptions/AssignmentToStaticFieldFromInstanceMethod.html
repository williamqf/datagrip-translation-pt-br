Relata atribuições ou modificações de campos static de dentro de métodos de instância.

Embora seja sintaticamente correto, esse tipo de atribuição é difícil de ser realizado com segurança e geralmente é o resultado de marcar um campo como static de forma não intencional.

Exemplo:


  class Counter {
    private static int count = 0;

    void increment() {
      // Alerta: atualizando um campo static
      // a partir de um método de instância
      count++;
    }
  }
