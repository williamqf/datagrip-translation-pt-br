Reporta chamadas como s.substring(0, x) que podem ser substituídas por s.take(x).

O uso de take() torna o código mais simples.

A correção rápida substitui a chamada substring por take().

Exemplo:


  fun foo(s: String) {
      s.substring(0, 10)
  }

Após aplicar a correção rápida:


  fun foo(s: String) {
      s.take(10)
  }
