Relata implementações de Iterator.next() que não podem lançar java.util.NoSuchElementException.

Tal implementação viola o contrato de java.util.Iterator e pode resultar em erros difíceis de detectar se o iterador for usado de forma não padrão.

Exemplo:


  class Numbers implements Iterator<Integer> {
    @Override
    public Integer next() { // aviso
        if (hasNext()) {
            return generateNext();
        } else {
            return null; // em vez disso, deve lançar NoSuchElementException
        }
    }

    ...
  }
