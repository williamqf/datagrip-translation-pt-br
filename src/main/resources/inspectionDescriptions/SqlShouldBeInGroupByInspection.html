Relata colunas que não estão na cláusula GROUP BY ou em uma chamada de função de agregação.

Exemplo (Microsoft SQL Server):

CREATE TABLE t1 (a INT, b INT);
SELECT a, b FROM t1 GROUP BY a;

Se você executar a consulta SELECT, receberá um erro porque o Microsoft SQL Server exige que a coluna b esteja no GROUP BY ou seja usada dentro de uma função de agregação. Os dois exemplos a seguir corrigem o erro.

SELECT a, b FROM t1 GROUP BY a, b;
SELECT a, max(b) max_b FROM t1 GROUP BY a;
