Relata alocações de arrays grandes que não são verificadas para java.lang.OutOfMemoryError. Em ambientes com restrição de memória, as alocações de grandes objetos de dados provavelmente devem ser verificadas quanto à exaustão de memória.

Esta inspeção destina-se ao Java ME e a outros ambientes com recursos altamente limitados. A aplicação dos resultados desta inspeção sem a devida consideração pode afetar negativamente a clareza e o design do código.

Use a opção para especificar o número máximo de elementos permitidos em alocações de arrays não verificadas.
