Relata chamadas de inclusão (include, prepend e extend) que referenciam classes. Chamadas de inclusão são permitidas apenas para referenciar módulos ou interfaces.

Uma correção rápida pode ser usada para remover chamadas de inclusão incorretas.

Exemplo:

class Example
  # Classe 'Object' usada na chamada 'include'
  include Object
end

Após a aplicação da correção rápida:

class Example
end
