Relata parênteses vazios redundantes em entradas de anotação.

Use a correção rápida 'Remover parênteses desnecessários' para limpar o código.

Exemplo:


  annotation class MyAnnotationA
  annotation class MyAnnotationB(val x: Int)
  annotation class MyAnnotationC(val x: Int = 10) // valor padrão presente

  @MyAnnotationA() // <== parênteses redundantes
  fun testA() {
  }

  @MyAnnotationB() // <== argumento ausente, parênteses são obrigatórios
  fun testB() {
  }

  @MyAnnotationC() // <== parênteses redundantes
  fun testC() {
  }
