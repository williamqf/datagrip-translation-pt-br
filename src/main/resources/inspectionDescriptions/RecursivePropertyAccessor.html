Relata chamadas recursivas de acessadores de propriedades que podem resultar em StackOverflowError. Essas chamadas são frequentemente confundidas com o acesso ao campo de suporte (backing field).

Exemplo:


  var counter: Int = 0
      set(value) {
          counter = if (value < 0) 0 else value
      }

Após a aplicação da correção rápida:


  var counter: Int = 0
      set(value) {
          field = if (value < 0) 0 else value
      }
