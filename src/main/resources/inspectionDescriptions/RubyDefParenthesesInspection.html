Relata definições de métodos com estilo incorreto de parênteses. Ao contrário das definições de métodos sem parâmetros, os parênteses devem ser usados em definições de métodos com parâmetros.

Exemplo:

# Prática ruim
def example_method(); end
def example_method_with_parameters param1, param2; end
Uma correção rápida pode ser usada para remover ou adicionar os parênteses, respectivamente. Após aplicar a correção rápida, o resultado será o seguinte: 
# Prática recomendada
def example_method; end
def example_method_with_parameters(param1, param2); end
