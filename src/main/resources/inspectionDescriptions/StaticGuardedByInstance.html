Relata anotações @GuardedBy em campos ou métodos static onde o guard é um campo não estático ou this.

Proteger um elemento static com um elemento não estático pode levar a uma concorrência excessiva, pois várias threads podem realizar o bloqueio em diferentes contextos de objeto para acessar o campo protegido simultaneamente.

Exemplo:


    private ReadWriteLock lock = new ReentrantReadWriteLock();

    @GuardedBy("lock")
    public static void bar() {
        // ...
    }

As anotações @GuardedBy suportadas são:

  • net.jcip.annotations.GuardedBy
  • javax.annotation.concurrent.GuardedBy
  • org.apache.http.annotation.GuardedBy
  • com.android.annotations.concurrency.GuardedBy
  • androidx.annotation.GuardedBy
  • com.google.errorprone.annotations.concurrent.GuardedBy
