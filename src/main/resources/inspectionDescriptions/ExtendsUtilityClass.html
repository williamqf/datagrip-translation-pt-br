Relata classes que estendem classes utilitárias.

Uma classe utilitária é uma classe não vazia na qual todos os campos e métodos são static. Estender uma classe utilitária também permite a instanciação acidental de objetos da classe utilitária, já que o construtor não pode ser definido como private para permitir a extensão.

Configuração da inspeção:

  • Use a opção Ignorar se a classe de sobreposição for uma classe utilitária para ignorar qualquer classe que estenda uma classe utilitária, mas que também seja uma classe utilitária.
