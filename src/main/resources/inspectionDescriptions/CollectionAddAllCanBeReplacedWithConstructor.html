Relata chamadas Collection.addAll() e Map.putAll() feitas imediatamente após a instanciação de uma coleção usando um construtor sem argumentos.

Tais construções podem ser substituídas por uma única chamada a um construtor parametrizado, simplificando o código. Além disso, para certas coleções, a substituição pode ser mais eficiente.

Exemplo:


  Set<String> set = new HashSet<>();
  set.addAll(Arrays.asList("alpha", "beta", "gamma"));

Após a aplicação do ajuste rápido (quick fix):


  Set<String> set = new HashSet<>(Arrays.asList("alpha", "beta", "gamma"));

As classes de coleção do JDK são suportadas por padrão. Além disso, você pode especificar outras classes usando o painel Classes a serem verificadas.
