Relata variáveis que excedem o comprimento de caracteres definido.

Exemplo (Microsoft SQL Server):

CREATE PROCEDURE test() AS
BEGIN
DECLARE myVarOk VARCHAR(5) = 'abcde';
DECLARE myVarExceeded VARCHAR(5) = 'abcde12345';

SET myVarOk = 'xyz';
SET myVarExceeded = '123456789';
END;

A variável myVarExceeded é definida como VARCHAR(5), mas ambos os valores atribuídos ('abcde12345' e '123456789') excedem esse limite. Você pode truncar o valor atribuído ou aumentar o comprimento definido. Para aumentar o comprimento, use a correção rápida Aumentar comprimento do tipo.

Após aplicar a correção rápida:

CREATE PROCEDURE test() AS
BEGIN
DECLARE myVarOk VARCHAR(5) = 'abcde';
DECLARE myVarExceeded VARCHAR(10) = 'abcde12345';

SET myVarOk = 'xyz';
SET myVarExceeded = '123456789';
END;
