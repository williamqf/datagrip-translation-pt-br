Relata chamadas de função de biblioteca que podem ser substituídas por uma forma simplificada.

O uso da função correspondente torna o código mais simples.

A correção rápida substitui as chamadas de função umas pelas outras.

Exemplo:


  fun test(list: List<Any>) {
      list.filter { it is String }
  }

Após aplicar a correção rápida:


  fun test(list: List<Any>) {
      list.filterIsInstance<String>()
  }
