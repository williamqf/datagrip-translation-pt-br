Relata variáveis de instância que são lidas antes de serem inicializadas.

Esta inspeção ignora verificações de igualdade com null.

Exemplo:


  class Foo {
    int bar;

    Foo() {
      System.out.println(bar);
    }
  }

Observe que esta inspeção usa um algoritmo de fluxo de dados muito conservador e pode relatar incorretamente variáveis de instância como não inicializadas. Variáveis relatadas como inicializadas serão sempre inicializadas.

Use a opção Ignorar se houver anotação para especificar anotações especiais. A inspeção ignorará os campos anotados com uma dessas anotações.

Use a opção Ignorar campos primitivos para ignorar campos primitivos não inicializados.
