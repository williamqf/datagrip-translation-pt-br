Relata loops for que iteram sobre coleções ou arrays e podem ser substituídos automaticamente por um loop for aprimorado (sintaxe de iteração foreach).

Exemplo:


  for (Iterator<String> iterator = list.iterator(); iterator.hasNext(); ) {
    String item = iterator.next();
    System.out.println(item);
  }

Após a aplicação da correção rápida:


  for (String item : list) {
    System.out.println(item);
  }

Use a opção Relatar loops 'java.util.List' indexados para encontrar loops que envolvem chamadas list.get(index). Esses loops geralmente podem ser substituídos por um loop for aprimorado, a menos que modifiquem a lista subjacente durante o processo, por exemplo, chamando list.remove(index). Nesse caso, o loop for aprimorado pode lançar uma ConcurrentModificationException. Em certas situações, o loop list.get(index) também pode ser ligeiramente mais rápido.

Use a opção Não relatar iteração em coleções sem tipo definido para ignorar coleções que não possuem parâmetros de tipo. Isso evita a criação de variáveis de loop for aprimorado do tipo java.lang.Object e a inserção de casts onde a variável de loop é utilizada.
