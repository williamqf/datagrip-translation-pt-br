Reporta expressões if-then que podem ser convertidas em uma expressão Elvis (?:).

Exemplo:


  fun maybeFoo(): String? = "foo"

  var foo = maybeFoo()
  val bar = if (foo == null) "hello" else foo

A correção rápida converte a expressão if-then em uma expressão Elvis (?:):


  fun maybeFoo(): String? = "foo"

  var foo = maybeFoo()
  val bar = foo ?: "hello"
