Relata parâmetros de métodos cujos nomes são muito curtos, muito longos ou não seguem o padrão de expressão regular especificado.

Exemplo: void fooBar(int X) deve ser relatado se a inspeção estiver habilitada com as configurações padrão na qual um nome de parâmetro deve começar com uma letra minúscula.

Configure a inspeção:

Use os campos na seção Opções para especificar o comprimento mínimo, comprimento máximo e uma expressão regular esperada para nomes de parâmetros de método. Especifique 0 para não verificar o comprimento dos nomes.

Expressões regulares devem ser especificadas no formato padrão java.util.regex.
