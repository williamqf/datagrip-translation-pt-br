Relata chamadas para toString() que utilizam a implementação padrão em java.lang.Object.

A implementação padrão raramente é intencional e pode ser um uso indevido.

Esta inspeção ignora chamadas toString() em objetos de tipos java.lang.Object, interfaces ou classes abstratas.

Exemplo:


  class Bar {
      void foo1(Bar bar) {
          String s = bar.toString(); // aviso
          /* ... */
      }

      void foo2(Object obj) {
          String s = obj.toString(); // sem aviso aqui
          /* ... */
      }
  }
