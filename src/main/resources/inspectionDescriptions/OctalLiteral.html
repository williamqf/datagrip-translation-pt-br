Relata literais de números inteiros octais. Alguns padrões de codificação proíbem o uso de literais octais porque eles podem ser facilmente confundidos com literais decimais.

Exemplo:


  int i = 015;
  int j = 0_777;

Esta inspeção possui duas correções rápidas diferentes. Após aplicar a correção rápida Converter literal octal para literal decimal, o código se torna:


  int i = 13;
  int j = 511;

Após aplicar a correção rápida Remover zero à esquerda para tornar decimal, o código se torna:


  int i = 15;
  int j = 777;
