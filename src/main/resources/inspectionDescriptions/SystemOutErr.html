Relata o uso de System.out ou System.err.

Tais instruções são geralmente usadas para depuração temporária e devem ser removidas do código de produção ou substituídas por um utilitário de registro de log mais robusto.

A correção rápida fornecida suporta SLF4J e Log4j 2. Ela substitui as chamadas de System.out e System.err por chamadas de log.

Exemplo:


  public static void test(Object o) {
    System.out.println("Test: "+ o);
  }

Após aplicar a correção rápida:


  private static final Logger log = LoggerFactory.getLogger(Test.class);

  public static void test(Object o) {
      log.info("Test: {}", o);;
  }

Utilize a opção Método de log para 'System.err' para especificar o método usado para registrar mensagens para chamadas 'System.err'. Utilize a opção Método de log para 'System.out' para especificar o método usado para registrar mensagens para chamadas 'System.out'.
