Relata dependências depreciadas. O uso de versões depreciadas pode causar vulnerabilidades de segurança, problemas de incompatibilidade, ineficiência e outros riscos.

Para resolver o problema, você pode seguir as instruções de migração no aviso de depreciação fornecido ou usar uma dependência alternativa.

As correções rápidas disponíveis podem sugerir a leitura do aviso de depreciação.

Para mais detalhes, consulte a depreciação de módulos no go.dev.
Os dados de depreciação são fornecidos pelo comando go list -m -u.
