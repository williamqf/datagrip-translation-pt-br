Relata métodos de teste JUnit com múltiplas exceções na cláusula throws. Tais cláusulas são desnecessariamente verbosas. Como os métodos de teste não serão chamados por outros códigos do projeto, não há necessidade de tratar essas exceções individualmente.

Exemplo:


  @Test
  public void testReflection() throws NoSuchMethodException,
      InvocationTargetException, IllegalAccessException {
    String result = (String) String.class.getMethod("trim")
        .invoke(" hello ");
    assertEquals("hello", result);
  }

A correção rápida substitui as declarações de exceção por uma única exceção:


  @Test
  public void testReflection() throws Exception {
    String result = (String) String.class.getMethod("trim")
        .invoke(" hello ");
    assertEquals("hello", result);
  }
