Relata verificações de null seguidas por chamadas de método que retornam obrigatoriamente false ao passar null (por exemplo, Class.isInstance).

Nesses casos, a chamada do método sempre retorna false, portanto essa verificação parece redundante.

Exemplo:


  if (x != null && myClass.isInstance(x)) { ... }

Após a aplicação da correção rápida:


  if (myClass.isInstance(x)) { ... }
