Relata anotações do Spring Web Services incorretas.

Exemplo:


  // 1. @PayloadRoot em classes que não são de endpoint...
  public class HolidayEndpoint {
	@PayloadRoot(namespace = NAMESPACE_URI, localPart = "HolidayRequest") // @PayloadRoot é suportado em classes @Endpoint
	public void handleHolidayRequest(@RequestPayload Element holidayRequest) throws Exception {...}
  }

  // 2. @XPathParam em métodos @PayLoad
  @Endpoint
  public class AirlineEndpoint {
	  @Namespace(prefix = "m", uri = MESSAGES_NAMESPACE) @ResponsePayload
	  public GetFlightsResponse getFlights(@XPathParam("//m:sc") String serviceClassString) {...} // @XPathParam é suportado em métodos @PayLoad
  }
