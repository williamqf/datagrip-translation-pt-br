Relata chamadas a Thread.yield().

O comportamento de yield() é indeterminístico e dependente da plataforma, sendo raramente apropriado usar este método. Seu uso deve ser combinado com profiling e benchmarking detalhados para garantir que o efeito desejado seja realmente alcançado.

Exemplo:


  public static void main(String[] args) {
    Runnable r = () -> {
      for (int i = 0; i < 10; i++) {
        System.out.println(i);
        Thread.yield();
      }
    };
    new Thread(r).start();
    new Thread(r).start();
  }
