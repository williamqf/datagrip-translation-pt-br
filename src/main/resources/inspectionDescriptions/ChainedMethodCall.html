Relata uma chamada de método cujo alvo é outra chamada de método. A correção rápida sugere a introdução de uma variável local.

Exemplo:


  class X {
    int foo(File f) {
      return f.getName().length();
    }
  }

Após a aplicação da correção rápida:


  class X {
    int foo(File f) {
      final String name = f.getName();
      return name.length();
    }
  }

Use as opções de inspeção para alternar os avisos para o seguinte:

  • Chamadas de método encadeadas em inicializadores de campo, como private final int i = new Random().nextInt();

  • Chamadas de método encadeadas operando no mesmo tipo, por exemplo, new StringBuilder().append("x: ").append(new X()).append("y: ").append(new Y()).toString();.
