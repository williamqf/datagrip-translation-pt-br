Relata anotações @Suppress que podem ser removidas com segurança, pois os diagnósticos do compilador associados não se aplicam mais neste contexto.

Exemplo:


  fun doSmth(@Suppress("UNUSED_PARAMETER") used: Int) {
    println(used)
  }

Após aplicar a correção rápida:


  fun doSmth(used: Int) {
    println(used)
  }
