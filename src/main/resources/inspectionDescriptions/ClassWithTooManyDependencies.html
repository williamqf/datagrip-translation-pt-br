Relata classes que dependem diretamente de muitas outras classes no projeto.

Qualquer modificação nas dependências de tais classes pode exigir alterações na própria classe, tornando-a propensa a instabilidade.

Apenas classes de nível superior são relatadas.

Use o campo Número máximo de dependências para especificar o número máximo de dependências permitidas para uma classe.

Disponível apenas em Código | Inspecionar código ou Código | Analisar código | Executar inspeção por nome e não é relatado no editor.
