Relata nomes exportados que começam com o nome do pacote. Esta inspeção não relata tais nomes no pacote main.

Exemplo:

package myPackage

func MyPackageGetIP() {

}

O nome MyPackageGetIP será destacado porque começa com o nome do pacote.

De acordo com Package Names no github.com/golang, todas as referências a nomes em um pacote serão feitas usando o nome do pacote, portanto, o nome pode ser omitido do identificador. Por exemplo, se você estiver no pacote foo, não precisará do tipo FooFile, que os clientes escreveriam como foo.FooFile. Em vez disso, nomeamos o tipo como File, que os clientes escreverão como foo.File.
