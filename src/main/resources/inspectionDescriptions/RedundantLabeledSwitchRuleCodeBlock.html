Relata regras rotuladas de instruções switch ou expressões switch com blocos de código redundantes.

Exemplo:


  String s = switch (n) {
    case 1 -> { yield Integer.toString(n); }
    default -> "default";
  };

Após a aplicação da correção rápida:


  String s = switch (n) {
    case 1 -> Integer.toString(n);
    default -> "default";
  };
