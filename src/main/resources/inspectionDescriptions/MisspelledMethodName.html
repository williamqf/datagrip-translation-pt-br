Relata casos em que vários métodos de uma classe têm nomes que diferem apenas no uso de maiúsculas e minúsculas. Tais nomes podem ser muito confusos.

Exemplo: 


  public int hashcode() { // Relatado, provavelmente deveria ser hashCode?
    return 0;
  }

A correção rápida para renomear tais métodos está disponível apenas no editor.

Use a opção Ignorar métodos que sobrescrevem/implementam super métodos para ignorar métodos que sobrescrevem ou implementam métodos de uma superclasse.
