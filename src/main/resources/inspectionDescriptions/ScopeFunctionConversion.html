Reporta funções de escopo (let, run, apply, also) que podem ser convertidas entre si.

O uso da função correspondente pode tornar o código mais simples.

A correção rápida substitui uma função de escopo por outra.

Exemplo:


  val x = "".let {
      it.length
  }

Após aplicar a correção rápida:


  val x = "".run {
      length
  }
