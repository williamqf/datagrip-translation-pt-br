Relata declarações import que não estão organizadas de acordo com o estilo de código atual (consulte "Configurações | Editor | Estilo de Código").

Exemplo:


  import java.util.List;
  import java.util.ArrayList;

  public class Example {
    List list = new ArrayList();
  }

Após aplicar a correção rápida "Otimizar imports": 


  import java.util.ArrayList;
  import java.util.List;

  public class Example {
    List list = new ArrayList();
  }
