Relata classes internas que podem ser static.

Uma classe interna static não mantém uma referência implícita à sua instância envolvente. Ao usar o Java 17 ou anterior, isso evita causas comuns de vazamentos de memória e reduz a memória usada por cada instância da classe.

Exemplo:


  public class Outer {
    class Inner { // não static
      public void foo() {
        bar("x");
      }

      private void bar(String string) {}
    }
  }

Após a aplicação da correção rápida:


  public class Outer {
    static class Inner {
      public void foo() {
        bar("x");
      }

      private void bar(String string) {}
    }
  }
