Relata pacotes cujos nomes são muito curtos, muito longos ou não seguem o padrão de expressão regular especificado.

Exemplo:


  package io;

Use as opções para especificar o comprimento mínimo e máximo de um nome de pacote, bem como uma expressão regular para corresponder a nomes de pacotes válidos (a expressão regular segue o formato padrão java.util.regex).
