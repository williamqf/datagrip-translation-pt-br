Relata o uso das formas sem curto-circuito de 'and' e 'or' booleanos (&, |, &= e |=). Embora as versões sem curto-circuito sejam ocasionalmente úteis, na maioria dos casos a forma de curto-circuito (&& e ||) é a pretendida, e esse uso não intencional pode levar a erros difíceis de detectar.

Uma correção rápida é sugerida para usar a versão de curto-circuito.

Exemplo:


  void foo(boolean x, boolean y, boolean z) {
    if (x | y) { x |= z; }
  }

Após a aplicação da correção rápida:


  void foo(boolean x, boolean y) {
    if (x || y) { x = x || z; }
  }
