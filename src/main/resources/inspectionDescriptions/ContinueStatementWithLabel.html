Relata instruções continue rotuladas.

Instruções continue com rótulos tornam a refatoração mais complexa e podem ser confusas.

Exemplo:


  void handle(List<String> strs) {
    outer:
    for (String s: strs) {
      for (char ch : s.toCharArray()) {
        if ('s' == ch) continue outer;
        handleChar(ch);
      }
    }
  }
