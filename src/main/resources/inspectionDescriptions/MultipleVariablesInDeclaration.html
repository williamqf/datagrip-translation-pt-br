Relata múltiplas variáveis declaradas em uma única declaração e sugere a criação de uma declaração separada para cada variável.

Alguns padrões de codificação proíbem tais declarações.

Exemplo:


  int x = 1, y = 2;

Após aplicar a correção rápida:


  int x = 1;
  int y = 2;

Configuração da inspeção:

  • Use a opção Ignorar declarações de loop 'for' para ignorar múltiplas variáveis declaradas na inicialização de instruções de loop 'for', por exemplo: 
    
  for (int i = 0, max = list.size(); i > max; i++) {}
  • Use a opção Avisar apenas para dimensões de array diferentes em uma única declaração para avisar somente quando variáveis com diferentes dimensões de array forem declaradas em uma única declaração, por exemplo: 
    
  String s = "", array[];
