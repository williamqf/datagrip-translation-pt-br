Relata referências nas seguintes anotações ORM de Persistência que não podem ser resolvidas nas fontes de dados configuradas:
  • Table
  • Column
  • [Any]To[Any]
  • JoinColumn / JoinTable
  • NamedQuery / NativeQuery
  • Embeddable

Sugere atualizar a fonte de dados ou atribuir uma fonte de dados diferente.

Exemplo de tabela de banco de dados:


  CREATE TABLE DATA_TABLE (
    ID INTEGER,
    DATA VARCHAR(100)
);

Exemplo de classe de entidade JPA:


  @Entity
  @Table(name = "UNKNOWN_TABLE") // referência não resolvida
  public class JavaEntity {
    @Column(name = "ID") // não é possível resolver a coluna em uma tabela indefinida
    Integer id;
  }

Observação: as fontes de dados podem ser configuradas por meio da ação `Atribuir fontes de dados` ou pelo ícone da calha correspondente da classe de entidade.
