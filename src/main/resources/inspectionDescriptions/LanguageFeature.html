Relata utilizações de recursos avançados da linguagem que não estão habilitados no arquivo atual.

Recursos avançados da linguagem precisam ser habilitados pela opção do compilador correspondente ou pela importação de um sinalizador de recurso explícito no arquivo em que são usados.

Notação de operador pós-fixado:
val s = 123 toString

Chamada reflexiva:
def foo(v: {def bar()}) = v.bar()

Seleção dinâmica de membro:
class Foo extends Dynamic

Conversão implícita:
implicit def toInt(s: String) = s.toInt

Tipo de ordem superior (habilitado por padrão desde Scala 2.13.1):
class Bar[M[A]]

Tipo existencial:
def foo(v: Seq[T] forSome { type T })

Definição de macro:
def assert(s: String) = macro Asserts.assertImpl

O compilador Scala também pode emitir estes avisos quando a opção "-feature" é fornecida.

Para opções do compilador, consulte: Arquivo | Configurações | Compilação, execução, implantação | Compilador | Compilador Scala | Recursos
