Relata qualquer comparação com Double.NaN ou Float.NaN. Tais comparações não fazem sentido, pois NaN não é igual a nada, inclusive a si mesmo. Use os métodos Double.isNaN() ou Float.isNaN() como alternativa.

Exemplo:


  if (x == Double.NaN) {...}

Após a aplicação da correção rápida:


  if (Double.isNaN(x)) {...}
