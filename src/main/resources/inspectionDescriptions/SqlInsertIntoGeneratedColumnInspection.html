Relata instruções INSERT que atribuem valores a colunas geradas. Colunas geradas podem ser lidas, mas seus valores não podem ser gravados diretamente.

Exemplo (PostgreSQL):

CREATE TABLE foo
(
   col1 INT,
   col2 INT GENERATED ALWAYS AS (col1 + 1) STORED
);
INSERT INTO foo(col1, col2) VALUES (1, 2);

Você não pode inserir 2 na coluna col2, pois esta coluna é gerada. Para que este script funcione, você pode alterar 2 para DEFAULT.

INSERT INTO foo(col1, col2) VALUES (1, DEFAULT);
