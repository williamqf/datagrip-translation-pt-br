Relata expressões switch que não abrangem todos os resultados possíveis da correspondência.

O Groovy não exige que as expressões switch sejam exaustivas. Ele funciona como se um ramo default -> null implícito tivesse sido inserido. Se o desenvolvedor esquecer de inserir os ramos case necessários, isso pode resultar em um null inesperado.

Exemplo:


  enum A { X, Y }

  def foo(A a) {
    def x = switch (a) { // relata o switch
      case A.X -> ...
    }
  }
