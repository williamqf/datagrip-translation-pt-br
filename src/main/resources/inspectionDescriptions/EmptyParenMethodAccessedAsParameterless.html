Relata chamadas sem cláusulas de argumento se referindo a métodos com uma cláusula de parâmetro vazia.

A convenção é que um método deve ter parênteses se tiver efeitos colaterais. Embora seja possível omitir cláusulas de argumento em chamadas de método (para adaptar o princípio de acesso uniforme para Java), é recomendado que os usuários adicionem uma cláusula de argumento vazia quando o método invocado representa mais que uma propriedade.

A correção rápida adiciona uma cláusula de argumento vazia.

Exemplo:


  def teste(): Int = ???

  teste

Após as correções rápidas serem aplicadas:


  def teste(): Int = ???

  teste()
