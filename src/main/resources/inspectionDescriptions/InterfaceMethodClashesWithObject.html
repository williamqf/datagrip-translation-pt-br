Relata métodos de interface que conflitam com os métodos protected clone() e finalize() na classe java.lang.Object.

Em uma interface, é possível declarar esses métodos com tipos de retorno que são incompatíveis com os métodos de java.lang.Object. Classes que implementam tal interface não podem ser compiladas. Embora ainda seja possível criar lambdas a partir dela quando a interface for funcional, isso não é recomendado.

Exemplo:


  // Aviso: esta interface não pode ser implementada por nenhuma classe,
  // apenas por um lambda ou referência de método
  interface MyInterface {
    double clone();
  }
