Relata o fallthrough em instruções switch. Embora ocasionalmente útil, o fallthrough geralmente não é intencional e pode levar a erros inesperados.

Exemplo:


switch(n) {
  case 1:
    print 1
  case 2: // "case 1" faz fallthrough para "case 2". As instruções de "case 2" serão executadas imediatamente após "case 1".
    print 2
    break
  default:
    print "Default"
}
