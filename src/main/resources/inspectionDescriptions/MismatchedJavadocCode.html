Reporta partes da especificação do método escritas em inglês que entram em contradição com a declaração do método. Isso inclui:
  • O método especificado para retornar true ou false, mas seu tipo de retorno não é booleano.
  • O método especificado para retornar null, mas é anotado como @NotNull ou seu tipo de retorno é primitivo.
  • O método especificado para retornar lista, mas seu tipo de retorno é definido ou uma matriz.
  • E assim por diante.

Exemplo:


  /**
   * @return true se o usuário for encontrado, false caso contrário
   */
  User findUser(String name);

Observe que falsos positivos são possíveis, pois esta inspeção tenta interpretar uma linguagem humana. No entanto, se a inspeção relatar incorretamente, ainda é possível que a descrição seja confusa e deva ser reescrita.
