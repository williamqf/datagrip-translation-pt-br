Reporta um return com rótulo usado na última expressão dentro de um lambda.

Tais retornos são redundantes e podem ser removidos com segurança.

Exemplo:


fun foo() {
  listOf(1,2,3).find {
    return@find true
  }
}

Após a aplicação da correção rápida:


fun foo() {
  listOf(1,2,3).find {
    true
  }
}
