Relata chamadas a métodos em qualquer uma das classes junit.framework.Assert, org.junit.Assert ou org.junit.Assume em testes JUnit 5.

Embora esses testes funcionem corretamente, a migração para org.junit.jupiter.api.Assertions/org.junit.jupiter.api.Assumptions ajuda a evitar dependências de versões legadas do JUnit.

Exemplo:


  import org.junit.Assert;
  import org.junit.jupiter.api.Test;

  public class MyTest {
    @Test
    public void simpleTest() {
      Assert.assertEquals(4, 2 + 2);
    }
  }

Após a aplicação da correção rápida:


  import org.junit.jupiter.api.Assertions;
  import org.junit.jupiter.api.Test;

  public class MyTest {
    @Test
    public void simpleTest() {
      Assertions.assertEquals(4, 2 + 2);
    }
  }
