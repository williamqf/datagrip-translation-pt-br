Informa sobre o número errado de aspas ao redor dos parâmetros nos valores das propriedades.

Se houver parâmetros nos valores das propriedades, o valor deve ser passado como um padrão para `java.text.MessageFormat`. Isso significa que aspas simples são usadas para escape. Portanto, para que as aspas fiquem visíveis, aspas duplicadas devem ser usadas:

  • '{0}' → {0}
  • ''{0}'' → '$valor$'

Ao usar o formato choice, o formato aninhado será avaliado como o próprio formato; portanto, uma segunda aspas dupla deve ser adicionada. Ao passar `1`:

  • {0, choice, 0#no|#1''{0}'' file} → {0} arquivos
  • {0, choice, 0#no|#1''''{0}'''' file} → '$numero_de_arquivos$' arquivos
