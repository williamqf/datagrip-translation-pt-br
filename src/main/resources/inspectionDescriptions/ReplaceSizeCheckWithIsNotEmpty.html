Relata verificações de tamanho de Collections/Array/String que devem ser substituídas por isNotEmpty().

O uso de isNotEmpty() torna o código mais simples.

A correção rápida substituirá a verificação de tamanho por isNotEmpty().

Exemplo:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.size > 0
  }

Após a aplicação da correção rápida:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.isNotEmpty()
  }
