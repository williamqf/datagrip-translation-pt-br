Relata o uso de context.TODO().

De acordo com a documentação em pkg.go.dev, o context.TODO deve ser usado quando não estiver claro qual Context utilizar ou quando o Context ainda não estiver disponível. O Context pode não estar disponível porque as funções circundantes ainda não foram estendidas para aceitar um parâmetro Context.

Observe que ele é um marcador temporário que deve ser alterado no futuro para um contexto mais significativo (por exemplo, context.Background()).
