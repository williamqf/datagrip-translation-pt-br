Relata chamadas da função hashMapOf ou do construtor HashMap que podem ser substituídas por chamadas do construtor EnumMap.

O uso do construtor EnumMap torna o código mais simples.

A correção rápida substitui a chamada da função por uma chamada do construtor EnumMap.

Exemplo:


  enum class E {
      A, B
  }

  fun getMap(): Map<E, String> = hashMapOf()

Após a aplicação da correção rápida:


  enum class E {
      A, B
  }

  fun getMap(): Map<E, String> = EnumMap(E::class.java)
