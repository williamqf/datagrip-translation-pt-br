Relata comentários Javadoc e tags ausentes.

Exemplo:


  /**
   * Relata "@param" ausente (se configurado).
   */
  public void sample(int param){
  }

A correção rápida adiciona a tag ausente ou o comentário Javadoc ausente:


  /**
   * Relata "@param" ausente (se configurado).
   * @param param
   */
  public void sample(int param){
  }

A inspeção pode ser configurada para ignorar elementos depreciados ou métodos acessores simples, como getField() ou setField().

Você também pode configurar as tags necessárias e a visibilidade mínima exigida para elementos de código específicos, como métodos, campos, classes, pacotes, módulos, etc., usando as opções abaixo.
