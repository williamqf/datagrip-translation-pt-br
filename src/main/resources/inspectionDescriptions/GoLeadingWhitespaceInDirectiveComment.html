Relata espaços à esquerda antes de diretivas Go em comentários.

Se houver um espaço entre // e a diretiva Go, a diretiva Go não será reconhecida.

A correção rápida remove o espaço à esquerda antes da diretiva Go.

Exemplo:


//  go:embed file.txt
var File string

Após a aplicação da correção rápida:


//go:embed file.txt
var File string
