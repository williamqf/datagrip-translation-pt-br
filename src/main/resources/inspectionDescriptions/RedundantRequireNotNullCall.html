Relata chamadas redundantes a requireNotNull ou checkNotNull em expressões que não podem ser nulas.

Exemplo:


  fun foo(i: Int) {
      requireNotNull(i) // este 'i' nunca é nulo, portanto esta chamada 'requireNotNull' é redundante.
      ...
  }

Após a aplicação do ajuste rápido:


  fun foo(i: Int) {
      ...
  }
