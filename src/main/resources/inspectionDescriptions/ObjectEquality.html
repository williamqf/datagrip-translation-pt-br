Reporta códigos que usam == ou != em vez de equals() para testar a igualdade de objetos.

Comparar objetos usando == ou != geralmente é um bug, pois compara objetos por identidade em vez de igualdade. Comparações com null não são reportadas.

Comparações de arrays, String e Number são reportadas por inspeções separadas.

Exemplo:

  if (list1 == list2) {
    return;
  }

Após a aplicação da correção rápida:

  if (Objects.equals(list1, list2)) {
    return;
  }

Use as configurações de inspeção para configurar exceções para esta inspeção.
