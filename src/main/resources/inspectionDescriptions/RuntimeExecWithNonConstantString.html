Reporta chamadas a java.lang.Runtime.exec() que recebem uma string construída dinamicamente como o comando a ser executado.

Strings de execução construídas são uma fonte comum de vulnerabilidades de segurança. Por padrão, esta inspeção ignora constantes de tempo de compilação.

Exemplo:


  String i = getUserInput();
  Runtime runtime = Runtime.getRuntime();
  runtime.exec("foo" + i); // reporta um aviso

Use as configurações da inspeção para considerar qualquer campo static final como uma constante. Tenha cuidado, pois strings como a seguinte serão ignoradas quando esta opção estiver ativada: 


  static final String COMMAND = "ping " + getDomainFromUserInput() + "'";
