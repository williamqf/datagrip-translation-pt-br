Relata chamadas equals() em instâncias de StringBuilder, StringBuffer e do pacote java.util.concurrent.atomic.

O método equals() não é sobrescrito nestas classes, portanto, pode retornar false mesmo que o conteúdo dos dois objetos seja o mesmo. Se a intenção for comparar a igualdade de referência, recomendamos o uso de == para evitar confusão.

Correções rápidas para StringBuilder, StringBuffer, AtomicBoolean, AtomicInteger, AtomicBoolean e AtomicLong estão disponíveis para converter para uma comparação de conteúdo. A correção rápida pode alterar a semântica quando uma das instâncias for nula.

Exemplo:


  public void test(StringBuilder sb1, StringBuilder sb2) {
    boolean result = sb1.equals(sb2); // Suspeito
  }

Após a aplicação da correção rápida:


  public void test(StringBuilder sb1, StringBuilder sb2) {
      boolean result = sb1.toString().equals(sb2.toString());
  }
