Detecta chamadas de função que podem ser aceleradas convertendo o argumento para Set.

Operações como 'minus' ou 'intersect' são mais eficientes quando o argumento é um conjunto (set). Converter explicitamente Iterable<T> ou Array<T> para Set<T> geralmente pode aumentar a eficiência do código.

A correção rápida adiciona uma conversão explícita à chamada da função.

Exemplo:


  fun <T> f(a: Iterable<T>, b: Iterable<T>): Int =
      a.intersect(b).size

Após aplicar a correção rápida:


  fun <T> f(a: Iterable<T>, b: Iterable<T>): Int =
      a.intersect(b.toSet()).size
