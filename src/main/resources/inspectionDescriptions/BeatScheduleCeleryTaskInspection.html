Relata tarefas não registradas no agendamento do Celery Beat.

Exemplo:


  #settings.py

  CELERY_BEAT_SCHEDULE = {
      'task_name': {
          'task': 'example_app.tasks.task_name', # path to a task represented by a class or function
          'schedule': 3.0,
      },
  }


  #example_app/tasks.py

  @app.task()
  def taskName():
      pass

  class TaskName(Task):
      pass

Se uma classe de tarefa não herdar de Task ou se uma função de tarefa não possuir o decorador necessário, ela não será considerada registrada.
