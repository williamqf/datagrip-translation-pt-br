Relata código que acessa propriedades do sistema usando um dos seguintes métodos:
  • System.getProperties(), System.setProperty(), System.setProperties(), System.clearProperties()
  • Integer.getInteger()
  • Boolean.getBoolean()

Embora o acesso às propriedades do sistema não seja, por si só, um risco de segurança, ele é frequentemente encontrado em códigos maliciosos. O código que acessa as propriedades do sistema deve ser examinado cuidadosamente em qualquer auditoria de segurança.
