Relata declarações de variáveis, argumentos ou funções que se sobrepõem a palavras-chave integradas (built-in) ou reservadas.

Se você vir este erro, isso indica que seu código não está tão explícito quanto poderia e pode confundir outros usuários.

Exemplo:

type byte struct{}
type string interface{}

Os tipos byte e string conflitam com aliases de tipos integrados. Portanto, eles serão destacados. Considere renomear tais declarações.
