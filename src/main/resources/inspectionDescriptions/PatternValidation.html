Relata expressões passadas como argumentos para parâmetros @Pattern e retornadas de métodos anotados com @Pattern que não correspondem ao padrão especificado.

Exemplo:


  @Pattern("\\d\\d-\\d\\d\\d") String getId() {
    return "1";
  }

Use a opção Sinalizar expressões constantes que não são de tempo de compilação para fazer com que a inspeção relate expressões com valores desconhecidos e sugira a adição da anotação de substituição (@Subst).
