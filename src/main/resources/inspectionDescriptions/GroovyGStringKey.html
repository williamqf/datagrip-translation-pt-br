Relata instruções que usam objetos groovy.lang.GString como chaves de mapa. Geralmente, objetos GString são mutáveis e provavelmente não devem ser usados como chaves. Além disso, as entradas GString não podem ser acessadas usando um objeto java.lang.String com o mesmo valor.

Exemplo:


def map = [:]
def key = 'foo'
map << ["${key}": 'bar']
assert map[key] == null // resultado 'true' de uma comparação confusa

Novidade na versão 2017.1
