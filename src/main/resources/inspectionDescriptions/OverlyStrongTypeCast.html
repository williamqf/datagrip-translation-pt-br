Relata casts de tipo excessivamente fortes. Por exemplo, realizar o cast de um objeto para ArrayList quando o cast para List seria suficiente.

Observação: Semelhante à inspeção Cast de tipo redundante, se você estiver usando intencionalmente um cast excessivamente forte para gerar uma ClassCastException, a aplicação de uma correção para esta inspeção pode alterar a semântica do programa.

Exemplo:


  interface Super {
    void doSmth();
  }
  interface Sub extends Super { }

  void use(Object obj) {
    // Aviso: ((Super)obj).doSmth() pode ser usado
    ((Sub)obj).doSmth();
  }
