Relata a delegação de interface para uma propriedade var.

Apenas o valor inicial da propriedade é usado para a delegação; qualquer atribuição subsequente não a afetará.

Exemplo:


  class Example(var text: CharSequence): CharSequence by text

A correção rápida substituirá a propriedade por uma propriedade imutável:


  class Example(val text: CharSequence): CharSequence by text

Se você precisar de mutabilidade por algum motivo, existe uma alternativa:


  class Example(text: CharSequence): CharSequence by text {
      var text = text
  }
