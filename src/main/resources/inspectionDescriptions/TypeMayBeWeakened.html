Relata variáveis e tipos de retorno de métodos que podem ser alterados para tipos mais abstratos (mais genéricos). Isso torna o código mais abstrato e, portanto, aumenta a reusabilidade.

Exemplo:


  // O tipo do parâmetro pode ser enfraquecido para java.util.List
  void processList(ArrayList<String> list) {
    if (list.isEmpty()) return;
    System.out.println("Processing");
    for (String s : list) {
      System.out.println("String: " + s);
    }
  }

Ative a caixa de seleção Somente enfraquecer para interface abaixo para relatar problemas apenas quando o tipo puder ser enfraquecido para um tipo de interface.

Ative a caixa de seleção Não sugerir enfraquecimento para variáveis declaradas como 'var' abaixo para evitar o relato de variáveis locais declaradas com a palavra-chave 'var' (Java 10+).

As Classes de parada destinam-se a evitar que uma classe seja enfraquecida além do nível das classes de parada, mesmo que isso seja possível. Em alguns casos, isso pode melhorar a legibilidade.
