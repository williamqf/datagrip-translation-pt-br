Relata o uso de chaves em templates de string para envolver identificadores simples.

Use a correção rápida 'Remover chaves' para remover chaves redundantes.

Exemplo:


  fun redundant() {
     val x = 4
     val y = "${x}" // <== redundante
  }

  fun correctUsage() {
      val x = "x"
      val y = "${x.length}" // <== ok
  }

Após aplicar a correção rápida:


  fun redundant() {
     val x = 4
     val y = "$x"
  }

  fun correctUsage() {
      val x = "x" <== atualizado
      val y = "${x.length}"
  }
