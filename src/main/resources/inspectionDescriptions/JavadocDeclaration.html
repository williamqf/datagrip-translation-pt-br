Relata comentários e tags Javadoc com os seguintes problemas:
  • Nome de tag inválido
  • Descrição de tag incompleta
  • Tags duplicadas
  • Descrição Javadoc ausente

Exemplo:


  /**
   * Nome de tag inválido
   * @poram param description
   */
  public void sample(int param){
  }

Exemplo:


  /**
   * Autorreferência {@link #sample(int)}
   */
  public void sample(int param){
  }

A correção rápida adicionará as tags Javadoc desconhecidas à lista de tags adicionais definidas pelo usuário.
