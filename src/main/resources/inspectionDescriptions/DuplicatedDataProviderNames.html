Relata provedores de dados TestNG com o mesmo nome, o que causaria uma org.testng.TestNGException.

Exemplo:


  public class DuplicatedDataProviders {
    @DataProvider
    public Object[][] intTestData() { // Duplicata 1
      return new Integer[][]{
        new Integer[]{1, 1},
      };
    }

    @DataProvider(name = "intTestData")
    public Object[][] someTestData() { // Duplicata 2
      return new Integer[][]{
        new Integer[]{1, 1},
      };
    }

    @Test(dataProvider = "intTestData")
    public void testIsOK(Integer key, Integer value) {
      assertEquals(key, value);
    }
  }
