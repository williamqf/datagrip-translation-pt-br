Relata classes de teste JUnit 3 que podem ser convertidas em classes de teste JUnit 4.

Exemplo:


  public class MainTestCase extends junit.framework.TestCase {
    public void test() {
      Assert.assertTrue(true);
    }
  }

Após a aplicação da correção rápida:


  public class MainTestCase {
    @org.junit.Test
    public void test() {
      Assert.assertTrue(true);
    }
  }

Esta inspeção relata apenas se o nível de linguagem do projeto ou módulo for 5 ou superior e o JUnit 4 estiver disponível no classpath.
