Relata parâmetros de função não utilizados no corpo da função.

Exemplo:


my_fahrenheit_to_celsius  <- function(f_temp) {
  return((temp - 32) * 5 / 9)
}

Modifique o trecho de código destacado para garantir que o parâmetro seja usado ou removido.

Soluções possíveis:


my_fahrenheit_to_celsius  <- function(f_temp) {
  return((f_temp - 32) * 5 / 9)
}
