Relata instruções e expressões switch aprimoradas. Sugere substituí-las por instruções switch regulares.

Exemplo:


  boolean even = switch (condition) {
    case 1, 3, 5, 7, 9 -> false;
    default -> true;
  };

Após a aplicação da correção rápida:


  boolean even;
  switch (condition) {
    case 1:
    case 3:
    case 5:
    case 7:
    case 9:
      even = false;
      break;
    default:
      even = true;
      break;
}

O switch aprimorado surgiu no Java 14. Esta inspeção ajuda a realizar o downgrade para fins de compatibilidade com versões anteriores do Java.
