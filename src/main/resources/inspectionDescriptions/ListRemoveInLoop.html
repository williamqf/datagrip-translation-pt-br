Reporta chamadas de List.remove(index) dentro de loops que podem ser substituídas por List.subList().clear().

Esta substituição é mais eficiente para a maioria das implementações de List ao remover múltiplos elementos.

Exemplo:


  void removeRange(List<String> list, int from, int to) {
    for (int i = from; i < to; i++) {
        list.remove(from);
    }
  }

Após a aplicação da correção rápida:


  void removeRange(List<String> list, int from, int to) {
    if (to > from) {
      list.subList(from, to).clear();
    }
  }

A correção rápida adiciona automaticamente uma verificação de intervalo para evitar uma possível IndexOutOfBoundsException caso o valor mínimo seja maior que o máximo. Se essa situação for impossível no seu código, a verificação pode ser removida.
