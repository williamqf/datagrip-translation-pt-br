Relata chamadas como s.substring(s.indexOf(x)) que podem ser substituídas por s.substringAfter(x).

O uso de s.substringAfter(x) torna o código mais simples.

A correção rápida substitui a chamada substring por substringAfter.

Exemplo:


  fun foo(s: String) {
      s.substring(s.indexOf('x'))
  }

Após a aplicação da correção rápida:


  fun foo(s: String) {
      s.substringAfter('x')
  }
