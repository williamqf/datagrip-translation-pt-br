Detecta recursos/fontes de dados que não podem ser resolvidos em relação aos provedores conhecidos. Os provedores devem ser especificados na seção required_providers ou devem vir do namespace hashicorp.

Por exemplo, o código abaixo causará um erro: 


terraform {
  required_version = "1.1.3"
}
resource "abbey_demo" "demo" {
  email      = "user@mail.company"
  permission = "user"
}

O Terraform tentará importar o provedor hashicorp/abbey para o recurso abbey_demo.

Para corrigir isso, precisamos especificar as coordenadas exatas do provedor: 


terraform {
  required_version = "1.1.3"
  required_providers {
    abbey = {
      source  = "abbeylabs/abbey"
      version = "0.2.9"
    }
  }
}
resource "abbey_demo" "demo" {
  email      = "user@mail.company"
  permission = "user"
}

Para mais detalhes, consulte a documentação do Terraform.
