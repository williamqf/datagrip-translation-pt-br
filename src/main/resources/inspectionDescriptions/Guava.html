Relata o uso de primitivas funcionais do Guava que podem ser migradas para chamadas da API Java padrão.

Por exemplo, esta inspeção relata o uso de classes e interfaces como FluentIterable, Optional, Function, Predicate ou Supplier.

Exemplo:


  ImmutableList<String> results = FluentIterable.from(List.of(1, 2, 3)).transform(Object::toString).toList();

Após a aplicação da correção rápida:


  List<String> results = List.of(1, 2, 3).stream().map(Object::toString).collect(Collectors.toList());

Esta correção rápida pode alterar a semântica. Alguns iteráveis do Guava com avaliação preguiçosa (lazy-evaluated) podem ser convertidos em iteráveis com avaliação imediata.
