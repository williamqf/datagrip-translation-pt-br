Relata chamadas de método em EL que não correspondem à restrição method-signature em arquivos TLD.

Exemplo de classe Java:



public class ExampleAction {
  public void goodAction() { ... }
  public void badAction(String str) { ... }
}

Exemplo de arquivo JSP:



<jsp:useBean id="example" scope="request" type="org.example.ExampleAction"/>
<f:view>
  <h:commandButton action="#{example.goodAction}"/> // OK: o método aceita zero argumentos e retorna `void`
  <h:commandButton action="#{example.badAction}"/> // Erro: o método `void badAction(String)` retorna o tipo `java.lang.String` em vez de `void`
</f:view>
