Detecta instruções não permitidas antes do bloco 'plugins {}'.

Devido a limitações do DSL de plugin do Gradle, apenas um conjunto restrito de instruções Groovy está disponível antes do bloco 'plugins {}'. As únicas opções são 'buildscript {}', 'pluginManagement {}' e outros blocos 'plugins {}'.
Consulte a documentação do Gradle

Exemplo:


import foo.bar.Baz
plugins {} // 'plugins' é reportado
plugins {
  foo() // 'foo' é reportado
  id 'java'
}
