Reporta declarações de variáveis locais e acessores para componentes de record em instruções 'for' aprimoradas que podem ser substituídos por variáveis de padrão, que geralmente são mais compactas.

Exemplo:


record Record(Integer x, String y) {
}

public static void test(List<Record> records) {
    for (Record record : records) {
        System.out.println(record.y());
        Integer x = record.x;
        System.out.println(x);
    }
}

Pode ser substituído por:


record Record(Integer x, String y) {
}

public static void test(List<Record> records) {
    for (Record(Integer x, String y) : records) {
        System.out.println(y);
        System.out.println(x);
    }
}
  • Use a opção Limite de profundidade de aninhamento para especificar o número máximo de padrões de desconstrução aninhados a serem reportados
  • Use a opção Número máximo de componentes de record a serem desconstruídos para especificar o número máximo de componentes que um record pode conter para ser usado em padrões de desconstrução
  • Use a opção Número máximo de componentes de record não utilizados para especificar o número máximo de componentes não utilizados na instrução for
