Relata os seguintes problemas ao recuperar serviços:
  • Tentativa de recuperar um serviço não registrado.
  • Incompatibilidade ao recuperar serviços: tentativa de obter um serviço de nível de projeto como um serviço de nível de aplicação, ou vice-versa.

Exemplo (Kotlin):


@Service
class MyAppService

@Service(Service.Level.PROJECT)
class MyProjectService(private val project: Project)


// Incorreto:
val projectService = service<MyProjectService>()         // O serviço de nível de projeto é recuperado como um serviço de nível de aplicação
val applicationService = project.service<MyAppService>() // O serviço de nível de aplicação é recuperado como um serviço de nível de projeto


// Correto:
val projectService = project.service<MyProjectService>()
val applicationService = service<MyAppService>();
