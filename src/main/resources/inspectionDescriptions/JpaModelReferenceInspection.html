Relata símbolos não resolvidos nas seguintes anotações JPA e métodos de API:
  • AttributeOverride
  • AssociationOverride
  • MapsId
  • MapKey
  • Métodos da Criteria API

Exemplo:



  @Entity
  @AssociationOverrides({
    @AssociationOverride(name = "missingAttribute", joinColumns = { ... }), // Erro: atributo não resolvido
    @AssociationOverride(name = "existingAttribute", joinColumns = { ... })
  })
  public class JavaEntity {
    @OneToOne
    AnotherEntity existingAttribute;
  }
