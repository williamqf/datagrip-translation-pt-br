Relata expressões de máscara bitwise que são avaliadas obrigatoriamente como true ou false.

Esta inspeção verifica expressões na forma (var & constant1) == constant2 ou (var | constant1) == constant2, onde constant1 e constant2 são constantes de máscara de bits incompatíveis.

Exemplo:

  // Máscara incompatível: como a máscara termina em 00,
  // o resultado pode ser 0x1200, mas não 0x1234
  if ((mask & 0xFF00) == 0x1234) {...}
