Relata instruções try-finally que podem usar o Gerenciamento Automático de Recursos (Automatic Resource Management) do Java 7 (que é menos propenso a erros).

Uma correção rápida pode converter a instrução try-finally em uma instrução try-with-resources.

Exemplo:


  PrintStream printStream = new PrintStream(fileName);
  try {
    printStream.print(true);
  } finally {
    printStream.close();
  }

Após a correção rápida:


  try (PrintStream printStream = new PrintStream(fileName)) {
    printStream.print(true);
  }
