Relata comparações com NULL que podem ser substituídas pelos operadores IS NULL ou IS NOT NULL.

Exemplo (Microsoft SQL Server):

CREATE TABLE foo ( id int );

SELECT * FROM foo WHERE NULL = NULL;
SELECT * FROM foo WHERE NULL != NULL;

NULL = NULL pode ser substituído por IS NULL e a comparação NULL != NULL pode ser substituída por IS NOT NULL. Para realizar essa substituição, você pode usar as correções rápidas Usar o operador IS NULL ou Usar o operador IS NOT NULL.

SELECT * FROM foo WHERE NULL IS NULL;
SELECT * FROM foo WHERE NULL IS NOT NULL;
