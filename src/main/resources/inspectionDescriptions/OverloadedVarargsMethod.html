Relata métodos vararg com o mesmo nome que outros métodos na classe ou em uma superclasse. Métodos sobrecarregados que aceitam um número variável de argumentos podem ser muito confusos, pois muitas vezes não está claro qual sobrecarga está sendo chamada.

Exemplo:


    public void execute(Runnable... r) {} // aviso
    public void execute(Runnable r1, Runnable r2) {}

Use a opção para ignorar métodos sobrecarregados com tipos de parâmetros claramente incompatíveis.
