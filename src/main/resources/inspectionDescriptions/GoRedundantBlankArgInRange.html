Relata variáveis em branco opcionais em loops range.

Ao usar um loop range para iterar sobre um slice, dois valores são retornados para cada iteração. O primeiro valor é o número do índice e o segundo valor é uma cópia do elemento nesse índice. Se você não precisar do segundo valor, poderá omitir esse elemento em vez de usar um identificador em branco.

Exemplo:

for a, _ = range v {} // `for a, _ =` é o mesmo que `for a =`

Para remover o identificador em branco, você pode usar a correção rápida Excluir argumento em branco. Após aplicar a correção rápida, o código ficará assim:

for a = range v {}
