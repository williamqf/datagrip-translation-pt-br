Relata casos em que uma expressão restringida por um concept é atribuída a uma variável declarada como auto e sugere a adição de uma restrição de tipo.

Exemplo:


  template<class T>
  concept Foo = true;

  Foo auto func() { return true; }

  void bar() {
      auto f = func(); // aviso para adicionar a restrição 'Foo' aqui
  }
