Relata chamadas de método para java.util.Collection ou java.util.Map em que o argumento é a própria coleção ou o próprio mapa. Tais situações podem ocorrer em códigos com tipos brutos (raw types) como resultado de copiar e colar.

Exemplo:


  ArrayList list = new ArrayList<>();
  list.add(list); // aviso aqui
  return list.hashCode(); // lança StackOverflowError
