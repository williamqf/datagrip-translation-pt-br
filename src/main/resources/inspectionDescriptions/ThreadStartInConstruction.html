Relata chamadas a start() em java.lang.Thread ou em qualquer uma de suas subclasses durante a construção do objeto.

Embora tais construções sejam ocasionalmente úteis, elas devem ser evitadas devido a problemas de herança. Subclasses de uma classe que inicia uma thread durante a construção do objeto não terão concluído sua própria lógica de inicialização antes que a thread comece.

Esta inspeção não relata se a classe que inicia a thread estiver declarada como final.

Exemplo:


  class MyThread extends Thread {
    MyThread() {
      start();
    }
  }
