Reporta campos não final em tipos enum. Campos não final introduzem um estado mutável global, o que geralmente é considerado indesejável.

Exemplo:


  enum Enum {
    FIRST("first"),
    SECOND("second");

    public String str;

    Enum(String str) {
        this.str = str;
    }
  }

Após aplicar a correção rápida:


  enum Enum {
    FIRST("first"),
    SECOND("second");

    public final String str;

    Enum(String str) {
        this.str = str;
    }
  }
Use a opção "Ignorar campos que não podem ser tornados 'final'" para avisar apenas sobre campos que podem se tornar final usando a correção rápida.
