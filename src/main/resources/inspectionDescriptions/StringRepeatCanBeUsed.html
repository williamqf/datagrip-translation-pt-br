Relata loops que podem ser substituídos por um único método String.repeat() (disponível a partir do Java 11).

Exemplo:


  void append(StringBuilder sb, int count, Object obj) {
    for (int i = 0; i < count; i++) {
      sb.append(obj);
    }
  }

Após a aplicação do ajuste rápido:


  void append(StringBuilder sb, int count, Object obj) {
    sb.append(String.valueOf(obj).repeat(Math.max(0, count)));
  }

Por padrão, esta inspeção pode envolver count com Math.max(0, count) se não for possível provar estaticamente que count não é negativo. Isso é feito para evitar possíveis mudanças semânticas, já que String.repeat() rejeita números negativos. Se necessário, use a opção Adicionar Math.max(0, count) para evitar possíveis mudanças semânticas para desativar este comportamento.

Da mesma forma, a string a ser repetida pode ser envolvida em String.valueOf para evitar um possível NullPointerException se não for conhecido se ela pode ser null.

Esta inspeção relata apenas se o nível de linguagem do projeto ou módulo for 11 ou superior.

Novidade na versão 2019.1
