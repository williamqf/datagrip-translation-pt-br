Relata constantes hexadecimais int negativas em um contexto long. Essas constantes são implicitamente estendidas para long, o que significa que seus bits de ordem superior serão 1 em vez de 0 (por exemplo, 0xFFFF_FFFF se tornará 0xFFFF_FFFF_FFFF_FFFFL). É improvável que isso seja intencional e, mesmo que seja, o uso de uma constante long explícita seria menos confuso.

Exemplo:


  // Aviso: esta é uma constante int -1, que é estendida para long
  // tornando-se 0xFFFF_FFFF_FFFF_FFFFL.
  long mask = 0xFFFF_FFFF;
