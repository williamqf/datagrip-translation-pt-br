Relata objetos Kotlin registrados como extensões de plugin.

O ciclo de vida das extensões é gerenciado pela plataforma IntelliJ. O uso de objetos Kotlin para o registro de extensões pode resultar na criação de instâncias de extensão desnecessárias e impedir que o plugin seja descarregado.

Exemplo

Registro da extensão:


<annotator language="myLang" implementationClass="com.example.MyAnnotator"/>

Implementação da extensão:


// Incorreto:
object MyAnnotator : Annotator {
  ...
}

// Correto:
class MyAnnotator : Annotator {
  ...
}
