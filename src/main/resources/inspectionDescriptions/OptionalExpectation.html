Relata anotações expect opcionais que não possuem uma anotação actual correspondente em alguns módulos de plataforma.

Exemplo:

// Código comum
@OptionalExpectation
expect annotation class JvmName(val name: String)

@JvmName(name = "JvmFoo")
fun foo() { }

// Código JVM
actual annotation class JvmName(val name: String)

Esta inspeção também relata casos em que actual annotation class JvmName é omitida para plataformas não JVM (por exemplo, Native).
