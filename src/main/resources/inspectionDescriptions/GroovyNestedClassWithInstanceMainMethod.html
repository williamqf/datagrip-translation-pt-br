Relata métodos main de instância declarados dentro de classes aninhadas.

A chamada de um método main em uma classe aninhada não é tratada diretamente pelo Groovy; no entanto, isso pode ser feito delegando a execução ao Java. Nesse caso, o código só será executado com sucesso se a versão do Java for recente o suficiente. Considere o seguinte exemplo: 


class Outer {
  static class Nested {
    void main() {
      println 'Hello from nested'
    }
  }
}
O código é executado com sucesso no Java 25, mas falha no Java 17.
