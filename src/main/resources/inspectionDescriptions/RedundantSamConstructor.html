Relata usos de construtores SAM (Single Abstract Method) que podem ser substituídos por lambdas.

Exemplo:


  fun main() {
      foo(Runnable { println("Hi!") })
  }

  fun foo(other: Runnable) {}

Após a aplicação da correção rápida:


  fun main() {
      foo( { println("Hi!") })
  }

  fun foo(other: Runnable) {}
