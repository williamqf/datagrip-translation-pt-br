Reporta expressões return sem rótulo dentro de lambdas inline.

Tais expressões podem ser confusas, pois pode não estar claro a qual escopo o return pertence.

A correção rápida Alterar para return@... pode ser usada para corrigir o código automaticamente.

Exemplo:


  fun test(list: List<Int>) {
      list.forEach {
          // Esta expressão de retorno retorna da função test
          // Pode ser alterada para return@forEach para mudar o escopo
          if (it == 10) return
      }
  }

Após aplicar a correção rápida:


  fun test(list: List<Int>) {
      list.forEach {
          if (it == 10) return@test
      }
  }
