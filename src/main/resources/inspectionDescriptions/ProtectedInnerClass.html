Relata classes aninhadas protected.

Exemplo:


  public class Outer {
    protected static class Nested {} // aviso
    protected class Inner {}         // aviso
    protected enum Mode {}           // o aviso depende da configuração
    protected interface I {}         // o aviso depende da configuração
  }

Configuração da inspeção:

  • Use a opção Ignorar enums internos 'protected' para ignorar enums internos protected.
  • Use a opção Ignorar interfaces internas 'protected' para ignorar interfaces internas protected.
