Relata links de texto simples em comentários Javadoc.

A correção rápida sugere envolver o link com uma tag <a>.

Exemplo:


  class Main {
   /**
    * https://en.wikipedia.org/
    */
    void foo() {}
  }

Após aplicar a correção rápida:


  class Main {
   /**
    * <a href="https://en.wikipedia.org/">https://en.wikipedia.org/</a>
    */
    void foo() {}
  }
