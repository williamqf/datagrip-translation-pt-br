Relata comparações == e === usadas na mesma variável em uma única expressão.

Devido à similaridade, == e === podem ser misturados inadvertidamente, exigindo uma verificação cuidadosa para identificar se == foi usado em vez de ===.

Exemplo:


  if (type === FIELD || type == METHOD || type == ANNOTATION_METHOD || // note que '==' foi usado incorretamente
      type === LAMBDA_EXPRESSION) return
