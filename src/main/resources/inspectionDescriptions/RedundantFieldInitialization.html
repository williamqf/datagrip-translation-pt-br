Relata campos que são inicializados explicitamente com seus valores padrão.

Exemplo:


  class Foo {
    int foo = 0;
    List bar = null;
  }

Após a aplicação da correção rápida:


  class Foo {
    int foo;
    List bar;
  }

Use as configurações da inspeção para relatar apenas inicializações null explícitas, por exemplo: 


  class Foo {
    int foo = 0;     // sem aviso
    List bar = null; // aviso de inicialização de campo redundante
  }
