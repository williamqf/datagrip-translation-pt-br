Relata a criação redundante de File em um dos seguintes construtores quando apenas um caminho String poderia ser utilizado: FileInputStream, FileOutputStream, FileReader, FileWriter, PrintStream, PrintWriter, Formatter.

Exemplo:


InputStream is = new FileInputStream(new File("in.txt"));

Após a aplicação do ajuste rápido:


InputStream is = new FileInputStream("in.txt");

Novidade na versão 2020.3
