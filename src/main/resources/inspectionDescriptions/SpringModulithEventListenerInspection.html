Relata declarações de listeners de eventos transacionais assíncronos que podem ser simplificadas: 

@Component
public class MyEventListener {

    @Async
    @Transactional
    @TransactionalEventListener
    public void on(MyEvent e) {
        // ...
    }
}
Usando @ApplicationModuleListener: 

@Component
public class MyEventListener {

    @ApplicationModuleListener
    public void on(MyEvent e) {
        // ...
    }
}
