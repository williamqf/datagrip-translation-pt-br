Relata chamadas a Object.notify() ou Object.notifyAll() que ocorrem sem que nenhuma mudança de estado detectável tenha sido realizada.

Object.notify() e Object.notifyAll() são geralmente usados para notificar outras threads de que uma mudança de estado ocorreu. Essa mudança de estado deve ocorrer dentro do contexto sincronizado que contém a chamada a Object.notify() ou Object.notifyAll() e deve ocorrer antes da chamada. Embora a ausência de tal mudança de estado não seja necessariamente incorreta, certamente vale a pena verificar.

Exemplo:


  synchronized (this) {
    notify();
  }
  // sem mudança de estado
  synchronized (this) {
    notify(); // a notificação pode ser redundante
  }
