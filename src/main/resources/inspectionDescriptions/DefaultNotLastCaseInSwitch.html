Relata instruções ou expressões switch nas quais a ramificação default está posicionada antes de outro caso. Tais construções podem causar confusão desnecessária. Se possível, uma correção rápida é fornecida para mover a ramificação default para a última posição.

Exemplo:


  switch (n) {
      default:
          System.out.println();
          break;
      case 1:
          break;
  }

Após a aplicação da correção rápida:


  switch (n) {
    case 1:
        break;
    default:
        System.out.println();
        break;
  }
