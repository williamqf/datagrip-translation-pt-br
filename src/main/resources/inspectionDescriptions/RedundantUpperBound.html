Relata um limite superior explícito redundante Any? em parâmetros de tipo. No Kotlin, o limite superior padrão de um parâmetro de tipo é Any?, portanto não há necessidade de especificá-lo explicitamente.

Exemplo:

// Antes
fun <T : Any?> foo(t: T) {}
class C<T : Any?>

Após a aplicação da correção rápida:


fun <T> foo(t: T) {}
class C<T>

A correção rápida remove o limite superior redundante e os dois-pontos que o precedem.
