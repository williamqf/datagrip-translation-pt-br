Relata casos em que um SELECT é executado a partir de uma função ou DBLINK sem um alias tipado (por exemplo, AS t1 (s VARCHAR)).

Este requisito não se aplica a funções escalares.

Exemplo (PostgreSQL):

CREATE FUNCTION produce_a_table() RETURNS RECORD AS $$
SELECT 1;
$$ LANGUAGE sql;
SELECT * FROM produce_a_table() AS s (c1 INT);
SELECT * FROM produce_a_table() AS s (c1);
SELECT * FROM DBLINK('dbname=mydb', 'SELECT proname, prosrc FROM pg_proc') AS t1;

AS s (c1 INT) possui um alias tipado, enquanto AS s (c1) e AS t1 não possuem. Neste caso, a segunda chamada de produce_a_table() e o DBLINK() serão destacados.
