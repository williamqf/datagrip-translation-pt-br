Relata o modificador inner em uma classe como redundante se a classe não referenciar membros de sua classe externa.

Exemplo:


  class Foo {
      inner class InnerClass { // modificador `inner` redundante
          fun hello() {
              println("Hi!")
          }
      }
  }

  class List {
      val objects = Array<Any>(42) { Any() }

      inner class Iterator { // modificador `inner` não redundante
          fun next(): Any {
              return objects[0]
          }
      }
  }

Após a aplicação da correção rápida:


  class Foo {
      class InnerClass { // modificador `inner` redundante
          fun hello() {
              println("Hi!")
          }
      }
  }

  class List {
      val objects = Array<Any>(42) { Any() }

      inner class Iterator { // modificador `inner` não redundante
          fun next(): Any {
              return objects[0]
          }
      }
  }
