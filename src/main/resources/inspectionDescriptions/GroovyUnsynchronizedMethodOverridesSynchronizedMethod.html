Relata métodos não synchronized que sobrescrevem um método synchronized.

Exemplo:


  abstract class Base {
      synchronized void foo() {
          // ...
      }
  }
  class Derived extends Base {
      @Override
      void foo() {
          super.foo()
            // ...
        }
    }

Neste exemplo, o método não synchronized foo() na classe Derived sobrescreve um método synchronized.
