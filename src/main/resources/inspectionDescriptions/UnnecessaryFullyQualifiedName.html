Relata nomes de classe totalmente qualificados que podem ser encurtados.

A correção rápida encurta o nome totalmente qualificado e adiciona uma instrução import se necessário.

Exemplo:


  class ListWrapper {
    java.util.List<Integer> l;
  }

Após a aplicação da correção rápida:


  import java.util.List;
  class ListWrapper {
    List<Integer> l;
  }

Configuração da inspeção:

Use a opção Ignorar em instruções de módulo do Java 9 para ignorar nomes totalmente qualificados dentro das instruções de módulo provides e uses do Java 9.

Em Configurações | Editor | Estilo de código | Java | Importações, configure a inspeção usando as seguintes opções:

  • Use a opção Inserir importações para classes internas se as referências a classes internas devam ser qualificadas com a classe externa.
  • Use a opção Usar nomes de classes totalmente qualificados no Javadoc para permitir nomes totalmente qualificados no Javadoc.
