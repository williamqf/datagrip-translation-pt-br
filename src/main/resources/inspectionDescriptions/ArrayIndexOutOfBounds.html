Relata expressões de acesso a variáveis de array ou ponteiro em que o índice pode estar fora dos limites do array ou do buffer alocado.

Exemplo:


  void foo() {
      int buffer[100];

      for (int i = 0; i <= 100; i ++)
          buffer[i] = 0; // buffer overflow quando i é igual a 100
  }
