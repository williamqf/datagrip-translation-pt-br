Relata usos de javax.swing.border.EmptyBorder e JBUI.Borders.emptyXyz() que podem ser simplificados.

Instâncias de EmptyBorder não reconhecem o DPI e podem causar problemas de layout na UI.

O quick fix substituirá pelo JBUI.Borders.empty() ou simplificará a expressão.

Exemplo:


// Incorreto:
Border border1 = new EmptyBorder(1, 2, 3, 4);
Border border2 = new EmptyBorder(1, 2, 1, 2);
Border border3 = new EmptyBorder(1, 0, 0, 0);

// Correto:
Border border1 = JBUI.Borders.empty(1, 2, 3, 4);
Border border2 = JBUI.Borders.empty(1, 2);
Border border3 = JBUI.Borders.emptyTop(1);
