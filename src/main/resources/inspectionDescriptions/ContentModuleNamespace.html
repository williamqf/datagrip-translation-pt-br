Valida o atributo de namespace nas declarações de módulos de conteúdo no arquivo plugin.xml.

Esta inspeção relata os seguintes problemas:

  • Namespace ausente: um namespace de conteúdo é obrigatório quando o conteúdo do plugin contém pelo menos um módulo com visibilidade não privada
  • Formato de namespace inválido: o namespace deve corresponder ao padrão exigido ([a-zA-Z0-9]+([_-][a-zA-Z0-9]+)*) e o comprimento deve ter entre 5 e 30 caracteres
  • Namespace inconsistente: todos os elementos de conteúdo devem definir o mesmo namespace
