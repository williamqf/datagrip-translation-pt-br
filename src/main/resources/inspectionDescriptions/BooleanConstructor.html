Relata a criação de objetos Boolean.

Raramente é necessário criar novos objetos Boolean e, se feito com frequência, isso pode causar problemas de desempenho. Além disso, o construtor Boolean foi depreciado desde o Java 9 e pode ser removido ou tornar-se inacessível em versões futuras do Java.

Exemplo:


  Boolean b1 = new Boolean(true);
  Boolean b2 = new Boolean(str);

Após a aplicação da correção rápida:


  Boolean b1 = Boolean.TRUE;
  Boolean b2 = Boolean.valueOf(str);
