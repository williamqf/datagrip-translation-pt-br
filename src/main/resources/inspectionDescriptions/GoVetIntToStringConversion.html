Relata conversões de expressões como string(x), em que x é um inteiro, mas não um byte ou rune.

Essas conversões não são recomendadas porque retornam a representação UTF-8 do ponto de código Unicode x, em vez da representação de string decimal de x que se poderia esperar. Além disso, se x representar um ponto de código inválido, a conversão não poderá ser rejeitada estaticamente.

Para conversões que pretendem usar pontos de código, considere substituí-las por string(rune(x)). Caso contrário, strconv.Itoa e suas funções equivalentes retornarão a representação de string do valor na base desejada.

Exemplo:

func main() {
 a := 1
 _ = string(a)
}

Após aplicar a correção rápida Converter inteiro para rune:

func main() {
 a := 1
 _ = string(rune(a))
}
