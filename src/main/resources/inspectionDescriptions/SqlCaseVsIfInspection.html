Relata casos em que CASE e IF são intercambiáveis.

Exemplo (MySQL):

SELECT CASE
WHEN C1 IS NULL THEN 1
ELSE 0
END
FROM dual;

Você pode encurtar o código substituindo a estrutura CASE por IF. Para fazer isso, aplique a ação de intenção Substituir por chamada 'IF'. O código de exemplo será semelhante ao seguinte:

SELECT IF(C1 IS NULL, 1, 0)
FROM dual;

Para reverter de IF para CASE, clique no IF e aplique a ação de intenção Substituir por expressão 'CASE'.
