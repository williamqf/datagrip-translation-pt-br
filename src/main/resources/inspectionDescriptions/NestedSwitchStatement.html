Relata instruções ou expressões switch aninhadas.

Instruções switch aninhadas podem tornar o código excepcionalmente confuso. Essas instruções podem ser extraídas para um método separado.

Exemplo:


  int res = switch (i) {
            case 0 -> 0;
            default -> switch (i) {
                case 100 -> 0;
                default -> i;
            };
        };
