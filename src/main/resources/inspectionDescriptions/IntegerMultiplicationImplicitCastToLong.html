Relata multiplicações de inteiros e deslocamentos à esquerda convertidos implicitamente para long.

Exemplo:


  void f(int i) {
    long val = 65536 * i;
  }

Após a aplicação da correção rápida, o código muda para:


  void x(int i) {
    long val = 65536L * i;
  }

Exemplo:


  void f(int i) {
    long value = i << 24;
  }

Após a aplicação da correção rápida, o código muda para:


  void f(int i) {
    long value = (long) i << 24;
  }

Tais multiplicações geralmente indicam um erro, pois o truncamento por estouro (overflow) pode ocorrer inesperadamente. Converter um literal int em um literal long (65536L) corrige o problema.
