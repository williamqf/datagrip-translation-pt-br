Relata expressões return (uma única linha ou when) que podem ser substituídas pela sintaxe de corpo de expressão.

A sintaxe de corpo de expressão é recomendada no guia de estilo.

A correção rápida Converter para corpo de expressão pode ser usada para corrigir o código automaticamente.

Exemplo:


  fun sign(x: Int): Int {
      return when { // <== pode ser simplificado
          x < 0 -> -1
          x > 0 -> 1
          else -> 0
      }
  }

Após a aplicação da correção rápida:


  fun sign(x: Int): Int = when {
      x < 0 -> -1
      x > 0 -> 1
      else -> 0
  }
