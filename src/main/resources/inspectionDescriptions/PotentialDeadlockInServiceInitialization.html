Relata ações de leitura/gravação e invokeAndWait chamados a partir do escopo de inicialização do serviço:
  • Construtores de serviço e blocos de inicialização (incluindo static)
  • Blocos de inicialização de objetos companion de serviço (Kotlin)
  • Métodos de serviço usados em blocos de inicialização e inicialização de campos
  • Métodos do ciclo de vida de inicialização do PersistentStateComponent:
    • loadState
    • noStateLoaded
    • initializeComponent

Executar ações de leitura/gravação ou chamar invokeAndWait durante a inicialização do serviço pode causar um deadlock.

Exemplos:

Kotlin:


@Service
internal class MyService {
  private val myData = initMyData();

  constructor() {
    val data = runReadAction { // Erro: executando ação de leitura no construtor
      // ler dados
    }
  }

  private fun initMyData(): Data {
    return runWriteAction { // Erro: chamado durante a inicialização da propriedade myData
      // gravar dados
    }
  }

  companion object {
    lateinit var companionData: String
    init {
      companionData = runReadAction { // Erro: executando ação de leitura no bloco init
        // ler dados
      }
    }
  }
}

Java:


@Service
class MyService {
  private static final Data ourData1 = ReadAction.compute(() -> {
    // ler dados
  });
  private static final Data ourData2;

  static {
    ourData2 = ReadAction.compute(() -> { // Erro: executando ação de leitura no bloco de inicialização static
      // ler dados
    });
  }

  private final Data myData2 = initMyData();

  MyService() {
    Data data = WriteAction.compute(() -> { // Erro: executando ação de gravação no construtor
      // gravar dados
    });
  }

  private Data initMyData() {
    return ReadAction.compute(() -> { // Erro: chamado durante a inicialização da propriedade myData
      // ler dados
    });
  }
}

PersistentStateComponent:


@Service
@State(...)
internal class MySettings : PersistentStateComponent<MyService> {
  var stateValue: String? = null

  override fun loadState(state: MySettings) {
    val data = runReadAction { // Erro: executando ação de leitura em loadState
      // ler dados
    }
    // ...
  }

  override fun noStateLoaded() {
    val data = runWriteAction { // Erro: executando ação de gravação em noStateLoaded
      // gravar dados
    }
    // ...
  }

  override fun initializeComponent() {
    val data = runReadAction { // Erro: executando ação de leitura em initializeComponent
      // ler dados
    }
    // ...
  }

}
