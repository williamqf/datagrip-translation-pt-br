Relata tipos explicitamente especificados para variáveis locais que são óbvios e redundantes, como val f: Foo = Foo().

Exemplo:


  class Point(val x: Int, val y: Int)

  fun foo() {
      val t: Boolean = true
      val p: Point = Point(1, 2)
      val i: Int = 42
  }

Após aplicar a correção rápida:


  class Point(val x: Int, val y: Int)

  fun foo() {
      val t = true
      val p = Point(1, 2)
      val i = 42
  }
