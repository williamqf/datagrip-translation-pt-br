Relata expressões booleanas desnecessárias ou excessivamente complexas.

Tais expressões incluem: combinações de && e true, combinações de || e false, comparações de igualdade com literais booleanos ou negação de literais booleanos. Essas expressões podem ser simplificadas.

Exemplo: 


  boolean a = !(x && false);
  boolean b = false || x;
  boolean c = x != true;

Após aplicar a correção rápida:


  boolean a = true;
  boolean b = x;
  boolean c = !x;

Configuração da inspeção:

Use a opção Ignorar constantes nomeadas ao determinar expressões inúteis para ignorar constantes nomeadas ao determinar se uma expressão é inútil.
