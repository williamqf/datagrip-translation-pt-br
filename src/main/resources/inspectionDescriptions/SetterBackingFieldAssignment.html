Relata setters de propriedade que não atualizam o campo de suporte (backing field).

A correção rápida adiciona uma atribuição ao campo de suporte.

Exemplo:


  class Test {
      var foo: Int = 1
          set(value) {
          }
  }

Após a aplicação da correção rápida:


  class Test {
      var foo: Int = 1
          set(value) {
              field = value
          }
  }
