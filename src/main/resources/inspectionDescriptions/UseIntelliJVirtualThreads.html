thoughtful Relata instâncias onde Thread.ofVirtual é utilizado em vez de IntelliJVirtualThreads.ofVirtual.

Thread.ofVirtual é uma API Java genérica para criar threads virtuais que não é adequada para as necessidades da plataforma IntelliJ.

A correção rápida substituirá o uso por IntelliJVirtualThreads.ofVirtual.

Exemplo: 


// Incorreto:
Thread thread = Thread.ofVirtual().start();

// Correto:
Thread thread = IntelliJVirtualThreads.ofVirtual().start();
