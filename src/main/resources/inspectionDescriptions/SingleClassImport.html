Relata instruções import que importam uma única classe (em vez de um pacote inteiro).

Alguns padrões de codificação proíbem tais instruções import.

Você pode configurar o IntelliJ IDEA para detectar e corrigir tais instruções usando o comando Otimizar imports. Vá para Configurações | Editor | Estilo de código | Java | Imports e desmarque a caixa de seleção Usar importação de classe única. Portanto, esta inspeção é útil principalmente para relatórios offline de bases de código que você não pretende alterar.
