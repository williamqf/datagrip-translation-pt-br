Relata atribuições ou modificações em parâmetros de loops for dentro do corpo do loop.

Embora ocasionalmente intencional, essa construção pode ser confusa e geralmente é causada por erros de digitação ou pelo uso da variável errada.

A correção rápida adiciona uma nova declaração de variável.

Exemplo:


  for (String s : list) {
    // Aviso: s é alterado dentro do loop
    s = s.trim();
    System.out.println("String: " + s);
  }

Após aplicar a correção rápida:


  for (String s : list) {
    String trimmed = s.trim();
    System.out.println("String: " + trimmed);
  }

Atribuições em loops for básicos sem uma instrução de atualização não são relatadas. Nesses casos, a atribuição provavelmente é intencional e não pode ser movida facilmente para a parte de atualização do loop for.

Exemplo:


  for (int i = 0; i < list.size(); ) {
    if (element.equals(list.get(i))) {
      list.remove(i);
    } else {
      // Como não há instrução de atualização, a modificação
      // do parâmetro do loop for não é relatada
      i++;
    }
  }

Use a opção Verificar parâmetros de loop 'for' aprimorados para especificar se as modificações dos parâmetros de loop for aprimorados também devem ser relatadas.
