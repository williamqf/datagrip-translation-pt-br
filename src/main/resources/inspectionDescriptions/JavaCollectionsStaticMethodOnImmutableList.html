Relata chamadas de métodos modificadores (mutators) do Java (como fill, reverse, shuffle, sort) em coleções imutáveis do Kotlin.

Isso pode resultar em uma UnsupportedOperationException em tempo de execução.

Exemplo:


  import java.util.Collections

  fun test() {
      val immutableList = listOf(1, 2)
      Collections.reverse(immutableList)
  }

Para corrigir este problema, torne a lista mutável.
