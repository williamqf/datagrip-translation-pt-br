Relata métodos static com a mesma assinatura de um método static da superclasse. Tais métodos podem parecer uma sobrescrita, mas na verdade eles ocultam o método da superclasse, já que métodos static não podem ser sobrescritos em Java.

Exemplo:


  class Parent {
    static void method(){}
  }

  class Example extends Parent {
    static void method(){} //aviso
  }
