Relata anotações que podem ser simplificadas para a forma de elemento único ou para a forma abreviada de marcador (marker shorthand).

Problemas relatados:

  • value= redundante em pares nome-valor de anotação
  • Chaves redundantes em valores de array que contêm apenas um único valor
  • Espaço em branco redundante entre o símbolo @ e o nome da anotação
  • Espaço em branco redundante entre o nome da anotação e a lista de parâmetros
  • Parênteses redundantes em anotações sem parâmetros

Exemplo:


  @interface Foo { String[] value(); }

  @ Foo({"foo"})
  public String name;

Após a aplicação da correção rápida:


  @interface Foo { String[] value(); }

  @Foo("foo")
  public String name;
