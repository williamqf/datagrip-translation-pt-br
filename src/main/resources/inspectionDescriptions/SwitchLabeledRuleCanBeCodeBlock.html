Relata regras de expressões switch ou instruções switch aprimoradas com corpos de expressão. Estas podem ser convertidas em blocos de código.

Exemplo:


  String message = switch (errorCode) {
    case 404 -> "Not found!";
    ...
  };

Após aplicar a correção rápida:


  String message = switch (errorCode) {
    case 404 -> {
                yield "Not found!";
            }
    ...
  };
