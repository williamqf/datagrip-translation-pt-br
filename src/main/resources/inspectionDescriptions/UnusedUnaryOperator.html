Relata operadores unários de tipo numérico não utilizados em expressões.

O uso de operadores unários sem chaves interrompe a expressão anterior. Como resultado, expressões matemáticas que abrangem várias linhas podem ser enganosas.

Exemplo:


  fun main() {
      val result = 1 + 2 * 3
                  + 3              // <== observe que '+ 3' não pertence à variável 'result' e não é utilizado
      println("Result = $result")  // o resultado é '7' em vez do valor esperado '10'
  }
