Relata problemas de assinatura para funções integradas.

Esta inspeção relata números incorretos de argumentos, palavras-chave inválidas, tipos de dados incorretos e outros problemas.

Exemplo (MySQL):

CREATE TABLE foo (a INT, b INT, c INT)

SELECT IFNULL() FROM foo; -- erro
SELECT IFNULL(a) FROM foo; -- erro
SELECT IFNULL(a, b) FROM foo; -- correto
SELECT IFNULL(a, b, c) FROM foo; -- erro

No MySQL, a função IFNULL() aceita estritamente dois argumentos. Portanto, apenas a consulta SELECT IFNULL(a, b) FROM foo; está correta.
