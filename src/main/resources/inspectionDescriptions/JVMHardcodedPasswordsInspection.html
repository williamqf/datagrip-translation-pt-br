Detecta possíveis tokens de segurança ou senhas contidos no código usando análise de entropia e expressões regulares.

Esta inspeção verifica strings como senhas ou tokens de segurança contidos na base de código usando análise de entropia e expressões regulares. Ele destaca os resultados encontrados para permitir que os desenvolvedores identifiquem e protejam contra vulnerabilidades potenciais. A eficácia desta inspeção depende do padrão definido em sua configuração, permitindo que ela se adapte a diferentes ambientes de codificação e requisitos. 


// Exemplo de string que pode ser detectada como um possível token:
String possibleToken = "pa#sw@rd;";

// Exemplo de um padrão de expressão regular usado para detecção:
String regexPattern = "[0-9]+:AA[0-9A-Za-z\\-_]{33}";
