Relata padrões de formatação de data que podem ter sido usados incorretamente.

Os seguintes padrões são relatados:

  • Um "Y" maiúsculo, a menos que "w" esteja por perto. Ele representa o "ano da semana", que é quase sempre igual ao "ano" normal (padrão "y" minúsculo), mas pode apontar para o ano seguinte logo após o fim de dezembro.
  • Um "M" maiúsculo (mês) perto de "H", "K", "h" ou "k" (horas). Provavelmente pretendia-se usar o "m" minúsculo (minutos).
  • Um "m" minúsculo (minutos) perto de "y" (ano) ou "d" (dia do mês). Provavelmente pretendia-se usar o "M" maiúsculo (mês).
  • Um "D" maiúsculo (dia do ano) perto de "M" ou "L" (mês). Provavelmente pretendia-se usar o "d" minúsculo (dia do mês).
  • Um "S" maiúsculo (milissegundos) perto de "m" (minutos). Provavelmente pretendia-se usar o "s" minúsculo (segundos).

Exemplos:
new SimpleDateFormat("YYYY-MM-dd"): provavelmente pretendia-se usar "yyyy-MM-dd".
new SimpleDateFormat("yyyy-MM-DD"): provavelmente pretendia-se usar "yyyy-MM-dd".
new SimpleDateFormat("HH:MM"): provavelmente pretendia-se usar "HH:mm".
