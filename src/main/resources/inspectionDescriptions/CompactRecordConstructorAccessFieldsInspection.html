Se métodos em um construtor de registro compactado acessarem campos, esses chamados de método serão informados.

Essas chamadas parecem suspeitas, pois o campo de registro pode não estar inicializado no momento dessas chamadas.

Exemplo:


public record MyRecord(String name, int id) {
    public MyRecord {
        validateMyFields(); // parece suspeito
    }

    private void validateMyFields() {
        if (this.name.isEmpty()) {
            throw new IllegalArgumentException();
        }
    }
}
