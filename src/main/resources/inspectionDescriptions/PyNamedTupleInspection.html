Relata definições inválidas de typing.NamedTuple.

Exemplo:


import typing


class FullName(typing.NamedTuple):
    first: str
    last: str = ""
    middle: str

A correção consiste em posicionar os campos que possuem valores padrão após os campos sem valores padrão:


import typing


class FullName(typing.NamedTuple):
    first: str
    middle: str
    last: str = ""
