Relata o uso de sintaxe obsoleta desde o Ruby 1.9. Por exemplo, o uso de : em vez de then para if, when e unless de linha única.

Exemplo:

condition = true
if condition: 1 else 0 # -> if condition then 1 else 0
unless condition: 0 else 1 # -> unless condition then 0 else 1

value = 1
case value
when 1: true # -> when 1 then true
end
