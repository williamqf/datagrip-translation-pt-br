Relata o uso da propriedade kotlin.coroutine.coroutineContext em código que depende da biblioteca kotlinx.coroutines.

Quando tanto kotlin.coroutine.coroutineContext quanto kotlinx.coroutines.CoroutineScope.coroutineContext estão presentes no código, isso pode causar confusão e bugs em potencial.

A biblioteca kotlinx.coroutines fornece a função currentCoroutineContext() como uma alternativa mais clara que deve ser preferida, mesmo quando não houver conflitos explícitos.

Para mais detalhes, consulte a documentação de kotlin.coroutine.coroutineContext e kotlinx.coroutines.currentCoroutineContext.

Exemplo:


  suspend fun getCurrentJob(): Job? {
    return coroutineContext[Job]
  }

Após a aplicação da correção rápida:


  suspend fun getCurrentJob(): Job? {
    return currentCoroutineContext()[Job]
  }
