Relata recursos de linguagem obsoletos e estruturas de código desnecessariamente verbosas durante a operação de limpeza de código (Código | Limpeza de código).

A correção rápida substitui automaticamente o uso de recursos de linguagem obsoletos ou estruturas de código desnecessariamente verbosas por uma sintaxe compacta e atualizada.

Também substitui símbolos obsoletos pelos substitutos sugeridos.
