Relata as seguintes divergências de arquivos JSON em relação à especificação da linguagem:
  • Comentários de linha ou de bloco (configurável).
  • Múltiplos valores de nível superior (para arquivos JSON Lines; configurável para outros arquivos).
  • Vírgulas no final em objetos ou arrays (configurável).
  • Strings com aspas simples.
  • Chaves de propriedade que não são strings com aspas duplas.
  • Valores numéricos NaN ou Infinity/-Infinity como literais de ponto flutuante (configurável).
