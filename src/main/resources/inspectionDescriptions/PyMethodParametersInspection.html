Relata métodos que não possuem o primeiro parâmetro, geralmente chamado de self.

Exemplo:


class Movie:

   def show():
       pass

Após a aplicação do ajuste rápido, o código torna-se:


class Movie:

   def show(self):
       pass

A inspeção também relata problemas de nomenclatura em métodos de classe.

Exemplo:


class Movie:
    @classmethod
    def show(abc):
        pass

Como o primeiro parâmetro de um método de classe deve ser cls, o IDE oferece um ajuste rápido para renomeá-lo.
