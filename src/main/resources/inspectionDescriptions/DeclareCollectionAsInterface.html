Relata declarações de variáveis Collection feitas usando classes de coleção como tipos em vez das interfaces apropriadas. Se o enfraquecimento do tipo da variável resultar em erros de compilação, nenhum aviso será emitido.

Exemplo:


  // Aviso: a classe de coleção concreta ArrayList foi usada.
  int getTotalLength(ArrayList<String> list) {
    return list.stream().mapToInt(String::length).sum();
  }

  // Sem aviso, pois o método trimToSize() não está disponível
  // na interface List
  void addData(ArrayList<String> data) {
    data.add("Hello");
    data.add("World");
    data.trimToSize();
  }
Uma correção rápida sugere o uso da interface de coleção apropriada (por exemplo, Collection, Set ou List).
