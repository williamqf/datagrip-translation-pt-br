Reporta usos de definições alfanuméricas sem o modificador `infix` como operador infix.

  • O quick-fix "Encapsular entre aspas invertidas" adiciona símbolos `` ao redor das chamadas.
  • O quick-fix "Converter de expressão infix" converte a chamada infix para a sintaxe de método .method(...).
  • O quick-fix "Converter de tipo infix" converte o tipo infix para a sintaxe de prefixo Type[...].
  • O quick-fix "Converter de padrão infix" converte o padrão infix para a sintaxe de prefixo pattern(...).

Exemplo:


  class C:
    def op(x: Int): Int = ???
  type Or[X, Y]
  case class Pair[T](x: T, y: T)

  val c = C()
  val p = Pair(1, 2)

  c op 2
  val or: Int Or String = ???
  val _ Pair _ = p

Após o quick-fix "Encapsular entre aspas invertidas" ser aplicado:


  class C:
    def op(x: Int): Int = ???
  type Or[X, Y]
  case class Pair[T](x: T, y: T)

  val c = C()
  val p = Pair(1, 2)

  c `op` 2
  val or: Int `Or` String = ???
  val _ `Pair` _ = p

Após o quick-fix "Converter de expressão infix" ser aplicado:


  class C:
    def op(x: Int): Int = ???
  type Or[X, Y]
  case class Pair[T](x: T, y: T)

  val c = C()
  val p = Pair(1, 2)

  c.op(2)
  val or: Int Or String = ???
  val _ Pair _ = p
