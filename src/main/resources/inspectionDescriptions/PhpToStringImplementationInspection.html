Relata tentativas de converter um objeto sem uma implementação do método __toString em uma string.

Antes do PHP 7.4, isso resultava em um erro fatal de nível E_RECOVERABLE_ERROR. No PHP 7.4, um ERROR fatal irrecuperável é lançado. A partir do PHP 8.0, o valor de retorno segue a semântica de tipos padrão do PHP e é coagido para uma string se possível e se o modo de tipagem estrita estiver desativado.

Se a opção Verificar a presença de __toString para cada tipo de expressão estiver ativada, a inspeção verificará todos os tipos possíveis da expressão e relatará se pelo menos um deles não contiver uma implementação do método __toString.

Para obter detalhes, consulte __toString (php.net).
