Relata padrões JavaScript comuns usados para detectar o navegador ou o sistema operacional a partir do qual o script está sendo executado. Além de apontar construções não portáveis, esses padrões de detecção de plataforma geralmente são incompletos e fáceis de serem burlados. Na maioria dos casos, detectar recursos específicos do ambiente é preferível a tentar detectar a plataforma inteira.

Os padrões detectados incluem:

  • document.all
  • document.layers
  • navigator.userAgent
  • navigator.oscpu
  • navigator.appName
  • navigator.appCodeName
  • navigator.platform
