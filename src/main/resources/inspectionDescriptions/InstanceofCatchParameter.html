Relata o uso de expressões instanceof para testar o tipo de um parâmetro em um bloco catch.

Geralmente é melhor testar o tipo de um parâmetro catch usando blocos catch separados em vez de usar instanceof.

Exemplo:


  void foo(Runnable runnable) {
    try {
        runnable.run();
    } catch (Throwable throwable) {
        if (throwable instanceof NoClassDefFoundError) { // Aviso: 'instanceof' no parâmetro 'throwable' do 'catch'
            System.out.println("Class not found!");
        }
    }
  }
