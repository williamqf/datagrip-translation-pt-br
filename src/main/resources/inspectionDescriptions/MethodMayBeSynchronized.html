Relata métodos cujo corpo contém apenas uma instrução synchronized. A expressão de bloqueio para essa instrução synchronized deve ser igual a this para métodos de instância ou [ClassName].class para métodos estáticos.

Para melhorar a legibilidade de tais métodos, o invólucro synchronized pode ser removido e o método pode ser marcado como synchronized.

Exemplo:


  public int generateInt(int x) {
    synchronized (this) {
      return 1;
    }
  }

Após a aplicação do ajuste rápido:


  public synchronized int generateInt(int x) {
    return 1;
  }
