Relata expressões booleanas negadas que podem ser simplificadas.

A correção rápida simplifica a expressão booleana.

Exemplo:


    fun test(n: Int) {
        !(0 == 1)
    }

Após a aplicação da correção rápida:


    fun test(n: Int) {
        0 != 1
    }

Observe que esta ação pode alterar a semântica do código se valores NaN IEEE-754 estiverem envolvidos:


    fun main() {
        println(!(Double.NaN >= 0)) // true
    }

Após a aplicação da correção rápida:


    fun main() {
        println(Double.NaN < 0) // false
    }
