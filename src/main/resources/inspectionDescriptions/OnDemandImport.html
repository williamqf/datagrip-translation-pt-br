Relata quaisquer instruções import que abrangem um pacote inteiro ('* imports').

Alguns padrões de codificação proíbem tais instruções import.

Você pode configurar o IntelliJ IDEA para detectar e corrigir tais instruções usando o comando Otimizar imports. Vá para Configurações | Editor | Estilo de código | Java | Imports, certifique-se de que a opção Usar import de classe única esteja habilitada e especifique os valores nos campos Contagem de classes para usar import com '*' e Contagem de nomes para usar import estático com '*'. Portanto, esta inspeção é útil principalmente para relatórios offline de bases de código que não se pretende alterar.
