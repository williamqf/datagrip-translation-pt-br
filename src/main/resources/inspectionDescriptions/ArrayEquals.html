Reporta chamadas equals() que comparam dois arrays.

Chamar equals() em arrays compara a identidade, o que é equivalente a usar ==. Use Arrays.equals() para comparar o conteúdo de dois arrays ou Arrays.deepEquals() para comparar o conteúdo de arrays multidimensionais.

Exemplo:


  void sample(int[] first, int[] second){
    if (first.equals(second)) return;
  }

Após a aplicação da correção rápida:


  void sample(int[] first, int[] second){
    if (Arrays.equals(first, second)) return;
  }
