Relata instruções if das quais partes comuns podem ser extraídas das ramificações.

Essas partes comuns são independentes da condição e tornam a instrução if mais difícil de entender.

Exemplo:


  if (x > 12) {
    doSomethingBefore();
    doSomethingDifferent1();
    doSomethingAfter();
  } else {
    doSomethingBefore();
    doSomethingDifferent2();
    doSomethingAfter();
  }

Após aplicar a correção rápida:


  doSomethingBefore();
  if (x > 12) {
    doSomethingDifferent1();
  } else {
    doSomethingDifferent2();
  }
  doSomethingAfter();

Funcionalidade atualizada na versão 2018.1
