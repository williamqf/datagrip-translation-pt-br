Relata métodos e campos em classes Serializable e Externalizable que não são adequados para serem anotados com java.io.Serial.

Exemplo:


class Test implements Serializable {
  @Serial // O campo anotado não faz parte do mecanismo de serialização porque não é final
  private static long serialVersionUID = 7874493593505141603L;

  @Serial // O método anotado não faz parte do mecanismo de serialização porque não é private
  void writeObject(ObjectOutputStream out) throws IOException {
  }
}


class Test implements Externalizable {
  @Serial // O método anotado não faz parte do mecanismo de serialização porque está em uma classe Externalizable
  private void writeObject(ObjectOutputStream out) throws IOException {
  }
}

Para informações sobre todos os casos válidos, consulte o Javadoc da classe java.io.Serial.
