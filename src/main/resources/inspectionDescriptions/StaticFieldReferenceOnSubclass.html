Relata acessos a campos static onde a chamada é qualificada por uma subclasse da classe declarante em vez da própria classe declarante.

O Java permite esse tipo de qualificação, mas tal acesso pode indicar uma confusão sutil entre herança e sobrescrita.

Exemplo:


  class Parent {
    static int foo = 0;
  }

  class Child extends Parent { }

  void bar() {
    System.out.println(Child.foo);
  }

Após a aplicação da correção rápida, o resultado será:


  class Parent {
    static int foo = 0;
  }

  class Child extends Parent { }

  void bar() {
    System.out.println(Parent.foo);
  }
