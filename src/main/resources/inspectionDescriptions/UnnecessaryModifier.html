Relata modificadores redundantes e sugere sua remoção. O código resultante será mais curto, mas seu comportamento e significado permanecerão inalterados.

Exemplo 1:


  // No Java 17 e versões posteriores, todo o código é implicitamente strictfp
  strictfp class X {

    // Enums internos são implicitamente static
    static enum Inner {
        A, B, C
    }

    // Records internos são implicitamente static
    static record R() {
    }
  }

Exemplo 2:


  final record R() {
    // Todos os records são implicitamente final
  }

Exemplo 3:


  // Todas as interfaces são implicitamente abstract
  abstract interface Printer {

    // Todos os membros de interface são implicitamente public
    public int size();

    // Todas as classes internas de uma interface são implicitamente static
    static class Inner {}
  }
