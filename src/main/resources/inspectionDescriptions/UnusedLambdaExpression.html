Relata expressões lambda que não são chamadas nem atribuídas.

Expressões lambda que não são chamadas ou atribuídas a uma variável não têm efeito na execução do programa e podem ser não intencionais.

Exemplo:


  fun foo() {
    {} // Aviso: A expressão lambda nunca é chamada
  }

Após a aplicação da correção rápida:


  fun foo() {
    run {} // A lambda agora é chamada através da função 'run'
  }
