Relata colunas, aliases e outros itens de subconsulta que não são referenciados na expressão de consulta externa.

Exemplo (PostgreSQL):

CREATE TABLE for_subquery(id INT);
SELECT a, q FROM (SELECT 1 AS a, 10 AS b, 2 + 3 AS q, id
      FROM for_subquery) x;

Nós referenciamos os aliases a e q da subconsulta. No entanto, o alias b e a coluna id não são referenciados na instrução SELECT externa. Portanto, b e id serão exibidos em cinza.
