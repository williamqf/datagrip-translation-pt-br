Relata classes que implementam dois ou mais traits que contêm métodos com a mesma assinatura.

A chamada de tais métodos pode produzir resultados inesperados.

A correção rápida adiciona um método de sobrescrita explícito.

Exemplo:


  trait T1 {
      def foo() {}
  }

  trait T2 {
      def foo() {}
  }

  class X implements T1, T2 {}

  // chamará T2.foo()
  new X().foo()

Após aplicar a correção rápida:


  class X implements T1, T2 {
      @Override
      Object foo() {
          return T2.super.foo()
      }
  }
