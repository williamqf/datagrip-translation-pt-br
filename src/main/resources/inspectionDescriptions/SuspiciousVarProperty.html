Relata propriedades var com setters e getters padrão que não fazem referência ao backing field.

Tais propriedades não afetam a chamada de seu setter; portanto, seria mais claro alterar essas propriedades para val e remover o inicializador.

A correção rápida Alterar para val e remover o inicializador pode ser usada para corrigir o código automaticamente.

Exemplo:


  // Esta propriedade sempre retorna '1', não importa se a propriedade é 'var'
  var foo: Int = 0
      get() = 1
