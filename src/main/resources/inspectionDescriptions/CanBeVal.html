Relata variáveis locais declaradas com a palavra-chave var que nunca são modificadas.

O Kotlin incentiva o uso da palavra-chave val para declarar variáveis efetivamente imutáveis, garantindo que seus valores nunca mudem.

Exemplo:


  fun example() {
      var primeNumbers = listOf(1, 2, 3, 5, 7, 11, 13)
      var fibonacciNumbers = listOf(1, 1, 2, 3, 5, 8, 13)
      print("Same numbers: " + primeNumbers.intersect(fibonacciNumbers))
  }

A correção rápida substituirá a palavra-chave var por val:


  fun example() {
      val primeNumbers = dummyListOf(1, 2, 3, 5, 7, 11, 13)
      val fibonacciNumbers = dummyListOf(1, 1, 2, 3, 5, 8, 13)
      print("Same numbers: " + primeNumbers.intersect(fibonacciNumbers))
  }
