Relata expressões lambda cujo número de instruções excede o máximo especificado.

Expressões lambda muito longas podem ser confusas; geralmente é melhor extrair as instruções para um método separado.

As seguintes instruções não são contabilizadas:

  • Instruções vazias (ponto e vírgula)
  • Instruções de bloco
  • Instruções de inicialização de loop for, ou seja, int i = ... dentro de uma instrução for(int i = ...;...)
  • Instruções de atualização de loop for, ou seja, i += 2 dentro de uma instrução for(int i = ...;...; i += 2)

Use o campo Limite de instruções de código não comentadas para especificar o número máximo de instruções permitidas em uma expressão lambda.
