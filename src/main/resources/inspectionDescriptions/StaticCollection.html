Relata campos static do tipo Collection. Embora isso não seja necessariamente um problema, coleções estáticas frequentemente levam a vazamentos de memória e, portanto, são proibidas por alguns padrões de codificação.

Exemplo:


  public class Example {
    static List<String> list = new ArrayList<>();

  }

Configuração da inspeção:

  • Use a opção Ignorar coleções ou mapas estáticos fracos para ignorar campos do tipo java.util.WeakHashMap.
