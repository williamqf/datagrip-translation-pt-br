Relata classes com nomes muito curtos, muito longos ou que não seguem o padrão de expressão regular especificado. Alguns estilos de codificação possuem convenções de nomenclatura especiais para classes.

Use as opções de inspeção para especificar o comprimento mínimo/máximo e a expressão regular esperada para nomes de classe. Para ignorar o comprimento do nome, especifique 0.

Para saber mais sobre expressões regulares, consulte o Quick Start guide (regular-expressions.info).
