Relata problemas em arquivos de propriedades incluídos em resource bundles.
  • Relatar traduções ausentes

    Use esta opção para relatar propriedades que estão incluídas em um arquivo de propriedades pai, mas estão ausentes em propriedades herdadas (a menos que seja um dialeto de idioma).

    Exemplo:

    
    # messages.properties
    abc=xxx

    # messages_fr.properties
    # Arquivo vazio
    A propriedade abc será relatada como não traduzida.


  • Relatar propriedades inconsistentes

    Use esta opção para relatar propriedades que estão incluídas em arquivos de propriedades herdados, mas estão ausentes nas propriedades pai (ou propriedades irmãs, se não houver pai).

    Exemplo:

    
    # messages.properties
    # Arquivo vazio

    # messages_fr.properties
    abc=xxx
    Aqui, a tradução da propriedade abc não está disponível para nenhum idioma além do francês; portanto, será relatada como ausente no arquivo de propriedades (padrão) messages.properties.


  • Relatar propriedades sobrescritas com o mesmo valor

    Use esta opção para relatar propriedades que foram copiadas e coladas literalmente em vários arquivos de propriedades.


    Exemplo:

    
    # messages.properties
    abc=xxx

    # messages_fr.properties
    abc=xxx
    A propriedade abc será relatada no arquivo messages_fr.properties como herdada desnecessariamente.


  • Relatar propriedades sobrescritas com placeholders diferentes

    Use esta opção para verificar a consistência dos placeholders em propriedades sobrescritas.

    Exemplo:

    
    # messages.properties
    qwe={0}xxx{1}
    abc={0}yyy{1}

    # messages_fr.properties
    qwe={0}xxx{0}xxx{1}
    abc={0}yyy
    A propriedade abc será relatada como contendo placeholders de formato de mensagem que não correspondem a messages.properties.


  • Relatar propriedades sobrescritas com terminações de valores diferentes

    Use esta opção para verificar a consistência das terminações em propriedades sobrescritas.

    Exemplo:

    
    # messages.properties
    abc=xxxzzz

    # messages_fr.properties
    abc=xxx;
    A propriedade abc será relatada como terminando com um símbolo especial (! / ? / . / : / ;), enquanto o valor pai em messages.properties não termina.
