Relata modificadores finais ausentes para objetos case.

Embora os objetos case internos sejam efetivamente final em Scala, eles não podem ser estendidos e nem sempre são sinalizados como tal no bytecode compilado. Portanto, esta inspeção encoraja você a fazê-lo. Veja boas práticas.

Exemplo:


  object Outer {
    case object Inner
  }

Após a correção rápida ser aplicada:


  object Outer {
    final case object Inner
  }
