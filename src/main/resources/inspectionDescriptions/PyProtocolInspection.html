Relata definições e usos inválidos de protocolos introduzidos no PEP-544.

Exemplo:


from typing import Protocol


class MyProtocol(Protocol):
    def method(self, p: int) -> str:
        pass


class MyClass(MyProtocol):
    def method(self, p: str) -> int: # o tipo de 'method' é incompatível com 'MyProtocol'
        pass


class MyAnotherProtocol(MyClass, Protocol): # todas as bases de um protocolo devem ser protocolos
    pass
