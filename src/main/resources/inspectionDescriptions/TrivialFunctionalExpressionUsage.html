Relata chamadas de método de interface funcional que são invocadas diretamente na definição de uma lambda, referência de método ou classe anônima. Essas chamadas de método podem ser substituídas pelo corpo da implementação da interface funcional.

Exemplo:


  boolean contains(List<String> names, String name) {
    return ((Predicate<String>)x -> {
      return names.contains(x);
    }).test(name);
  }

Após a aplicação da correção rápida, a chamada do método é alterada para:


  boolean contains(List<String> names, String name) {
    return names.contains(name);
  }
