Relata uma chamada ReadScope inválida de uma suspensão ou contexto de ação.

Não é possível chamar a função ReadScope de uma corrotina ou uma ação. Se isso ocorrer, pode levar a uma exceção de tempo de execução.

Exemplo:


  fun ReadScope.kernel(/*...*/) { /*...*/ }

  fun ReadScope.foo() {
      readEditorAction { ac ->
          kernel() // Incorreto
      }
  }
