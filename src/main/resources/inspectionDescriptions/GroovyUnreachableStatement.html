Relata instruções inacessíveis. Isso pode ocorrer se a instrução estiver posicionada após um loop infinito ou após as instruções return, break ou continue.

Exemplo: 


  void foo (int n) {
      if (n < 1) {
          return
          print('This statement is unreachable')
      }
      while (true){
          print ('Hello, world!')
      }
      print('This statement is unreachable too')
  }
