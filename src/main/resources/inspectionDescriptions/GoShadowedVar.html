Relata declarações de variáveis que se sobrepõem a declarações em escopos externos.

Como o significado da variável depende do escopo nesses casos, isso pode causar confusão e levar a consequências inesperadas.

Exemplo:

for i := 0; i < len(nums); i++ {
    for i := 0; i < len(nums); i++ {
    }
  }

A variável i no loop aninhado está sendo sombreada (shadowed). Para remover o sombreamento, considere renomear a variável no loop aninhado.

for i := 0; i < len(nums); i++ {
    for j := 0; j < len(nums); j++ {
    }
  }
