Relata instruções que não têm nenhum efeito.

Exemplo:


class Car:
    def __init__(self, speed=0):
        self.speed = speed
        self.time # não tem nenhum efeito

2 + 3 # não tem nenhum efeito

Neste exemplo, você pode adicionar o campo time à classe Car ou introduzir uma variável para a instrução em questão.
