Relata variáveis de template Go duplicadas declaradas no mesmo escopo.

A duplicação de variáveis reatribui uma variável existente com o mesmo nome. Esta operação pode levar a diversos problemas imprevisíveis.

Exemplo:

{{$v := 0}}{{$v := 1}}{{$v}} é 0. {{/* avaliado como '1 é 0' */}}
{{$v := 0}}{{$w := 1}}{{$v}} é 0. {{/* funciona conforme o esperado */}}
