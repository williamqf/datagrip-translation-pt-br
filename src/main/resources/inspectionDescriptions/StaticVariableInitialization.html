Relata variáveis static que podem não ser inicializadas durante a inicialização da classe.

Exemplo:


  class Foo {
    public static int bar;

    static { }
  }

Observe que esta inspeção usa um algoritmo de fluxo de dados muito conservador e pode relatar incorretamente variáveis static como não inicializadas. Variáveis relatadas como inicializadas sempre serão inicializadas.

Use a opção Ignorar campos primitivos para ignorar campos primitivos não inicializados.
