Relata iterações sobre o keySet() de uma instância de java.util.Map, onde a chave iterada é usada para recuperar um valor do mapa.

Tais iterações podem ser mais eficientes quando substituídas pela iteração de entrySet() ou values() (se as chaves não forem realmente usadas).

Da mesma forma, se o valor for recuperado em um lambda, keySet().forEach(key -> ...) pode ser substituído por forEach((key, value) -> ...).

Exemplo:


  for (Object key : map.keySet()) {
   Object val = map.get(key);
  }

Após a aplicação da correção rápida:


  for (Object val : map.values()) {}
