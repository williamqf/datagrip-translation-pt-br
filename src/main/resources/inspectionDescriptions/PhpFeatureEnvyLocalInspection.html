Relata o code smell "Inveja de Recursos" (Feature Envy). A Inveja de Recursos é definida quando um método chama métodos de outra classe com mais frequência do que chama os métodos da sua própria classe de origem. Isso geralmente indica que a funcionalidade pretendida está na classe errada.

Chamadas para classes de biblioteca, superclasses e classes que contêm ou são contidas pela classe atual não são contabilizadas nesta inspeção.
