Relata chamadas a get() em um Optional sem verificar se há um valor presente.

Chamar Optional.get() em uma instância de Optional vazia lançará uma exceção.

Exemplo:


  void x(List<Integer> list) {
    final Optional<Integer> optional =
      list.stream().filter(x -> x > 10).findFirst();
    final Integer result = optional.get(); // problema aqui
  }
