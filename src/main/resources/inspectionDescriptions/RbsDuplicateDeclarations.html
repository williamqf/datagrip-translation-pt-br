Relata declarações duplicadas de classes, módulos, interfaces, constantes, variáveis globais, aliases de tipo e métodos.
  • Declarações duplicadas geram um DuplicatedDeclarationError.
  • Definições de método duplicadas geram um DuplicatedMethodDefinitionError.
  • Definições de método de interface duplicadas geram um DuplicatedInterfaceMethodDefinitionError.

Uma correção rápida pode ser usada para navegar até as declarações duplicadas.

Exemplo:

# Declaração de interface duplicada
interface _Example
end

# Declaração de interface duplicada
interface _Example
end
