Relata loops contendo uma instrução if que podem ser encerrados com um break sem alterar a semântica. Isso evita iterações redundantes do loop.

Exemplo:


  boolean found = false;
  for (int i = 0; i < arr.length; i++) {
    if (Objects.equals(value, arr[i])) {
      found = true;
    }
  }

Após a aplicação da correção rápida:


  boolean found = false;
  for (int i = 0; i < arr.length; i++) {
    if (Objects.equals(value, arr[i])) {
      found = true;
      break;
    }
  }
