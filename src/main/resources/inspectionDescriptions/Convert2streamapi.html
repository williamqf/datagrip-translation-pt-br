Relata loops que podem ser substituídos por chamadas da Stream API usando expressões lambda.

Essa substituição altera o estilo de imperativo para funcional, tornando o código mais compacto.

Exemplo:


  boolean check(List<String> data) {
    for (String e : data) {
      String trimmed = e.trim();
      if (!trimmed.startsWith("xyz")) {
        return false;
      }
    }
    return true;
  }

Após aplicar a correção rápida:


  boolean check(List<String> data) {
    return data.stream().map(String::trim).allMatch(trimmed -> trimmed.startsWith("xyz"));
  }
