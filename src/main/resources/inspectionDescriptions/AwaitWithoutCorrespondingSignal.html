Relata chamadas a Condition.await() para as quais nenhuma chamada correspondente a Condition.signal() ou Condition.signalAll() pode ser encontrada.

Chamar Condition.await() em uma thread sem um Condition.signal() correspondente pode fazer com que a thread seja desativada até que seja interrompida ou ocorra um "despertar espúrio" (spurious wakeup).

Esta inspeção relata apenas chamadas cujo alvo são campos da classe atual.

Exemplo:


  class Queue {
    private final Condition isEmpty = ...;

    void add(Object elem) {
      // ...
      // isEmpty.signal();
      // ...
    }

    void remove(Object elem) throws InterruptedException {
      // ...
      isEmpty.await(); // 'await()' não contém uma chamada 'signal()'/'signalAll()' correspondente
      // ...
    }
  }
