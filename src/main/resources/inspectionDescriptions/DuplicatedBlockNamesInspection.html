Relata nomes de blocos duplicados em templates Django.

Exemplo:


<!DOCTYPE html>
<html lang="en">
<head>
    <link rel="stylesheet" href="style.css">
    <title>{% block title %}Meu site incrível{% endblock %}</title>
</head>

<body>
    <div id="sidebar">
        {% block title %}
        <ul>
            <li><a href="/">Início</a></li>
            <li><a href="/blog/">Blog</a></li>
        </ul>
        {% endblock %}
    </div>

    <div id="content">
        {% block content %}{% endblock %}
    </div>
</body>
</html>
