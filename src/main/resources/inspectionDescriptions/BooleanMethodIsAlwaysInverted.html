Relata métodos com tipo de retorno boolean que são sempre negados ao serem chamados.

Oferece uma correção rápida para inverter e, opcionalmente, renomear o método. Por motivos de desempenho, nem todos os métodos com problemas são destacados no editor.

Exemplo:


  class C {
    boolean alwaysTrue() {
      return true;
    }

    void f() {
      if (!alwaysTrue()) {
        return;
      }
    }
    boolean member = !alwaysTrue();
  }

Após a aplicação da correção rápida:


  class C {
    boolean alwaysFalse() {
      return false;
    }

    void f() {
      if (alwaysFalse()) {
        return;
      }
    }
    boolean member = alwaysFalse();
  }
