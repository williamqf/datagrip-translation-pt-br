(Gradle 4.9+) Detecta o uso de APIs que interagem imediatamente (eager) com tarefas.

A interação imediata com tarefas acarreta diversas dificuldades:

  • O usuário deve definir manualmente todas as dependências entre as tarefas;
  • Na fase de configuração, todas as tarefas acessadas por meio de APIs imediatas são configuradas, mesmo que não sejam executadas posteriormente. Isso leva à degradação do desempenho.

Eventualmente, as APIs imediatas serão descontinuadas em favor das APIs lazy (de configuração tardia).

Para um guia de migração, consulte a documentação do Gradle.

Exemplo:

task foo { // reporta 'task', sugere substituí-la por 'task.register'
  // ...
}
