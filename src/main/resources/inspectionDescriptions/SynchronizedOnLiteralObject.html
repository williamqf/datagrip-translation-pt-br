Relata blocos synchronized que realizam bloqueio em objetos inicializados com literais.

Literais de string são internados e literais Character, Boolean e Number podem ser alocados a partir de um cache. Devido a isso, outras partes do sistema (usando objetos inicializados pelo mesmo literal) podem conter referências ao mesmo objeto exato. Isso pode resultar em situações inesperadas de deadlock se o objeto de bloqueio for considerado privado.

Exemplo:


  class Main {
    final String mutex = "Mutex";
    void method() {
      synchronized (mutex) {
      }
    }
  }

Use a opção Avisar sobre todos os literais possíveis para relatar qualquer sincronização de objetos String, Character, Boolean e Number.
