Relata chamadas ao operador rangeTo ou .. em vez de ..<.

O uso da função correspondente pode tornar o código mais simples.

A correção rápida substituirá chamadas rangeTo ou .. por ..<.

Exemplo:


  fun foo(a: Int) {
      for (i in 0..a - 1) {

      }
  }

Após a aplicação da correção rápida:


  fun foo(a: Int) {
      for (i in 0..<a) {

      }
  }
