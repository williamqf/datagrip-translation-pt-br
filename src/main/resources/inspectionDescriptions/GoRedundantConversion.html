Relata conversões de tipo que podem ser omitidas.

Exemplo:

var s = string("hello")

O valor "hello" já é do tipo string, portanto a conversão adicional para string é redundante. Para remover a conversão, considere usar a correção rápida Remover conversão.

Após a aplicação da correção rápida:

var s = "hello"

Às vezes, converter expressões de ponto flutuante para tipos de ponto flutuante pode ser intencional (consulte este problema). Nesses casos, o IDE alertará sobre uma possível conversão redundante.
