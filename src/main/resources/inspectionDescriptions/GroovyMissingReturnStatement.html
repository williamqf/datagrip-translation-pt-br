Relata a ausência de instruções return no final de métodos com um tipo de retorno não void. O final do método deve ser alcançável pelo fluxo de execução do método.

Exemplo:


String foo(int a) {
  if (a > 0) {
    return "more than zero"
  }
} // foo(-1) retornará 'null'

int bar(int a) {
  if (a > 0) {
    return a
  }
} // bar(-1) falhará com uma exceção de tempo de execução
