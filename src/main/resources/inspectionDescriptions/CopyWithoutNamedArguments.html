Relata chamadas ao método copy() de classes de dados (data classes) sem argumentos nomeados.

Como todos os argumentos da função copy() são opcionais, pode ser difícil entender quais propriedades estão sendo modificadas. Fornecer explicitamente os nomes dos parâmetros torna o código mais fácil de entender, sem a necessidade de navegar até a declaração da data class.

Exemplo:


  data class User(val name: String, val age: Int)

  fun copyUser(user: User): User {
      return user.copy("John")
  }

A correção rápida fornece nomes de parâmetros para todos os argumentos de copy():


  data class User(val name: String, val age: Int)

  fun copyUser(user: User): User {
      return user.copy(name = "John")
  }
