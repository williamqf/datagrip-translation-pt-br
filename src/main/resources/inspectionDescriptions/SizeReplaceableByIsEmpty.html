Relata comparações de .size() ou .length() com o literal 0 que podem ser substituídas por uma chamada a .isEmpty().

Exemplo:


  boolean emptyList = list.size() == 0;

Após a aplicação do ajuste rápido:


  boolean emptyList = list.isEmpty();

Use a tabela Classes ignoradas para adicionar classes para as quais nenhuma comparação de .size() ou .length() deve ser substituída.

Use a opção Ignorar expressões que seriam substituídas por !isEmpty() para ignorar quaisquer expressões que seriam substituídas por !isEmpty().
