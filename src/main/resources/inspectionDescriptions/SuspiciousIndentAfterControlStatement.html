Relata indentação suspeita de instruções que seguem instruções de controle sem chaves.

Com tal indentação, a instrução parece estar dentro da instrução de controle, mas na verdade ela será executada incondicionalmente após a instrução de controle.

Exemplo:


  class Bar {
      void foo(int i) {
          if (i == 0)
              System.out.println("foo");
              System.out.println("bar"); // aviso
          if (i == 1);
              System.out.println("great"); // aviso
          if (i == 42)
              System.out.println("answer");
                  System.out.println("question"); // aviso
      }
  }
