Relata possíveis escapes de this durante a inicialização do objeto. Um escape ocorre quando this é usado como um argumento de método ou em uma atribuição de objeto dentro de um construtor ou inicializador. Tais escapes podem causar bugs difíceis de detectar, pois o objeto torna-se disponível em um contexto onde a sua inicialização completa não é garantida.

Exemplo:


  class Foo {
    {
      System.out.println(this);
    }
  }
