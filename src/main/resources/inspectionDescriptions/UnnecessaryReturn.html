Relata instruções return no final de construtores e métodos que retornam void. Essas instruções são redundantes e podem ser removidas com segurança.

Esta inspeção não relata em arquivos JSP.

Exemplo:


  void message() {
    System.out.println("Hello World");
    return;
  }

Após a aplicação do ajuste rápido: 


  void message() {
    System.out.println("Hello World");
  }

Use a opção Ignorar no branch 'then' de instruções 'if' com um branch 'else' para ignorar instruções return no branch then de uma instrução if que também tenha um branch else.
