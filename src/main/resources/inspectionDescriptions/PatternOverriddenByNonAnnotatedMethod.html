Reporta métodos sem a anotação @Pattern que sobrescrevem métodos anotados com @Pattern. Isso não impede o destaque de erros no editor, mas o método que sobrescreve não será verificado em tempo de execução.

Uma correção rápida é fornecida para adicionar a anotação @Pattern correspondente à anotação no método da superclasse. Isso garante que a instrumentação de verificação em tempo de execução funcione corretamente.

Exemplo:


  abstract class Parent {
    abstract @Pattern("\\d\\d-\\d\\d\\d") String getId();
  }
  class Child extends Parent {
    @Override String getId() { // Aviso aqui
      return "12-345";
    }
  }
