Relata nomes de parâmetros inconsistentes em chamadas de métodos Java especificados em blocos de comentário.

Exemplo:


  // Java
  public class JavaService {
      public void invoke(String command) {}
  }


  // Kotlin
  fun main() {
      JavaService().invoke(/* name = */ "fix")
  }

A correção rápida corrige o nome do parâmetro no bloco de comentário:


  fun main() {
      JavaService().invoke(/* command = */ "fix")
  }
