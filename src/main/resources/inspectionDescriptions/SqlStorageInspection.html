Relata casos em que o código-fonte de um objeto de banco de dados foi alterado.

A inspeção é acionada ao realizar a introspecção do banco de dados ou de um objeto. A introspecção é executada quando você abre o código-fonte do objeto, executa instruções e realiza a refatoração de código. Além disso, você pode executar a introspecção clicando com o botão direito no objeto e selecionando Atualizar.

A inspeção abrange os seguintes casos:

  • O código-fonte do objeto foi alterado no banco de dados, mas o código no editor não foi atualizado. Funciona para PostgreSQL, Microsoft SQL Server, Oracle e Sybase ASE.
  • Você alterou o código-fonte do objeto e realizou a introspecção do banco de dados, mas o código-fonte já havia sido alterado por outra pessoa.
  • O introspector de banco de dados no IDE foi atualizado e você precisa baixar novas propriedades de objeto que estavam ausentes na versão anterior do introspector.
