Relata campos de logger que não foram declarados como static e/ou final. Garantir que cada logger de classe seja efetivamente uma constante e esteja vinculado à classe simplifica a tarefa de fornecer uma implementação de log uniforme para a aplicação.

Uma correção rápida é fornecida para alterar os modificadores do logger para static final.

Exemplo:


  public class Significant {
    private Logger LOG = Logger.getLogger(Critical.class);
  }

Após aplicar a correção rápida:


  public class Significant {
    private static final Logger LOG = Logger.getLogger(Critical.class);
  }

Configuração da inspeção:

  • Use a tabela Nomes de classes de logger para especificar os nomes das classes de logger. Esta inspeção relatará campos que não sejam static e final e cujo tipo seja equivalente a um dos nomes de classe especificados.
