Relata o uso de funções ou métodos Python marcados como obsoletos que geram um aviso DeprecationWarning ou PendingDeprecationWarning.

Além disso, esta inspeção destaca o uso dos decoradores abc.abstractstaticmethod, abc.abstractproperty e abc.abstractclassmethod.

Exemplo:


class Foo:
    @property
    def bar(self):
        import warnings
        warnings.warn("this is deprecated", DeprecationWarning, 2)
        return 5


foo = Foo()
print(foo.bar)
