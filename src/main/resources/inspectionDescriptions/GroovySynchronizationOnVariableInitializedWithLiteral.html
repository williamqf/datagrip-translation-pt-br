Relata blocos synchronized que realizam o bloqueio em objetos inicializados com literais.

Literais de string são internados e literais Number podem ser alocados a partir de um cache. Por causa disso, é possível que outras partes do sistema (que utilizam objetos inicializados pelo mesmo literal) contenham referências ao exato mesmo objeto. Isso pode resultar em situações inesperadas de deadlock se a string for considerada privada.
