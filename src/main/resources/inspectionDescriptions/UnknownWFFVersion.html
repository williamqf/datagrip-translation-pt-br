Relata o uso de uma versão desconhecida do Watch Face Format no arquivo de manifesto. Essa versão pode ser válida, mas ainda não é suportada na versão atual do Android Studio. Quando uma versão é desconhecida, ferramentas como o preenchimento automático podem não funcionar corretamente.
Uma lista de versões válidas pode ser encontrada em https://developer.android.com/training/wearables/wff.
