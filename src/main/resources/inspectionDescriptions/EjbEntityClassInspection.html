Detecta as seguintes violações de nível de erro da especificação EJB:
  • Métodos Finder e Selector não definidos corretamente
  • Acessores de campos CMP e CMR não definidos corretamente

Exemplo:


public class SessionEJB implements javax.ejb.SessionBean {
  // Erro: o método 'ejbCreateXXX' deve ser public
  protected void ejbCreateExample() { ... }

  // Erro: o método 'ejbCreateAnotherExample' deve retornar 'void'
  public int ejbCreateAnotherExample() { ... }
}
