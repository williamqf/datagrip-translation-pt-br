Relata verificações de igualdade que usam Float.NaN ou Double.NaN (que devem ser substituídas por verificações isNaN()).

De acordo com o padrão IEEE 754, as verificações de igualdade em relação ao NaN sempre retornam false, mesmo para NaN == NaN. Portanto, é muito provável que tal verificação esteja incorreta.

A correção rápida substitui a comparação por uma verificação isNaN() que utiliza uma técnica de comparação diferente e lida corretamente com valores NaN.

Exemplo:


  fun check(value: Double): Boolean {
      return Double.NaN == value
  }

Após a aplicação da correção:


  fun check(value: Double): Boolean {
      return value.isNaN()
  }
