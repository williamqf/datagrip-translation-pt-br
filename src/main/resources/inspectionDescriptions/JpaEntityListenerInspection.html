Relata os seguintes erros de especificação JPA:
  • Classes de listener de entidade sem um construtor público sem argumentos
  • Métodos de listener de entidade com assinaturas incorretas

Exemplo:


  @Entity
  @EntityListeners(value = JavaEntityListener.class)
  class JavaEntity { ... }

  class JavaEntityListener {

    @PostLoad
    void postLoad(String parameter) { ... } // Erro: o método 'postLoad' deve receber um parâmetro do tipo 'Object' aqui
  }
