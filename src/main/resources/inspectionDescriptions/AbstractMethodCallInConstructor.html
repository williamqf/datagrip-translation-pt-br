Relata chamadas a métodos abstract da classe atual durante a construção do objeto.

Um método é chamado durante a construção do objeto se estiver em um dos seguintes locais:

  • Construtor
  • Inicializador de instância não static
  • Inicializador de campo não static
  • Método clone()
  • Método readObject()
  • Método readObjectNoData()

Tais chamadas podem resultar em erros sutis, pois a inicialização do objeto pode ocorrer antes da chamada do método.

Exemplo:


  abstract class Parent {
    abstract void abstractMethod();
  }

  class Child extends Parent {
    Child() {
      abstractMethod();
    }
  }

Esta inspeção compartilha funcionalidades com as seguintes inspeções:

  • Método que pode ser sobrescrito chamado durante a construção do objeto
  • Método sobrescrito chamado durante a construção do objeto

Apenas uma inspeção deve ser habilitada por vez para evitar avisos duplicados.
