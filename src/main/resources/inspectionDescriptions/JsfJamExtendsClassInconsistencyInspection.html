Relata classes com anotações JSF específicas que não implementam a interface necessária:

  • @FacesComponent deve estender <javax/jakarta>.faces.component.FacesComponent
  • @FacesRenderer deve estender <javax/jakarta>.faces.render.Renderer
  • @FacesConverter deve estender <javax/jakarta>.faces.convert.Converter
  • @FacesValidator deve estender <javax/jakarta>.faces.validator.Validator

Exemplo:



@FacesConverter(value="...")  // a classe deve estender 'javax.faces.convert.Converter'
public class FooConverter {
}

@FacesRenderer(rendererType="..", componentFamily="..")  // a classe deve estender 'javax.faces.render.Renderer'
public class FooRenderer {
}

@FacesValidator(value="...")  // a classe deve estender 'javax.faces.validator.Validator'
public class FooValidator {
}
