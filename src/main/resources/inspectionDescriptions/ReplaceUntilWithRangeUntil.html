Relata o uso de until que pode ser substituído pelo operador ..<.

Nenhuma substituição de until por ..< altera a semântica de forma alguma.

Pesquisas de UX mostram que desenvolvedores cometem cerca de 20 a 30% menos erros ao ler códigos contendo ..< em comparação com until.

Exemplo:


  fun main(args: Array<String>) {
      for (index in 0 until args.size) {
          println(index)
      }
  }

Após aplicar a correção rápida:


  fun main(args: Array<String>) {
      for (index in 0..<args.size) {
          println(index)
      }
  }

Esta inspeção relata apenas se o nível de linguagem Kotlin do projeto ou módulo for 1.7 ou superior.
