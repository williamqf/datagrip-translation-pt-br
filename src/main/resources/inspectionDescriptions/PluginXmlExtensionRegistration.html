Relata problemas com o registro de extensões no plugin.xml.

Os seguintes problemas serão relatados:

  • Declaração language ausente. Se a extensão não for direcionada a um idioma específico, use a correção rápida para criar uma declaração explícita para "any language".
  • Inspeção: atributo ausente
  • Intenção: declaração descriptionDirectoryName redundante
  • Serviço: declaração serviceInterface redundante
  • com.intellij.stubElementTypeHolder sem externalIdPrefix, consulte os índices de stub
  • com.intellij.statusBarWidgetFactory sem id
  • com.intellij.applicationConfigurable|projectConfigurable sem displayName/key
