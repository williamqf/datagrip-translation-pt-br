Relata classes Serializable que não implementam os métodos readObject() e writeObject().

Se os métodos readObject() e writeObject() não forem implementados, o algoritmo de serialização padrão será usado, o que pode ser subótimo em muitos ambientes em termos de desempenho e compatibilidade.

Use as seguintes opções para configurar a inspeção:

  • Liste as classes cujos herdeiros não devem ser relatados por esta inspeção. Isso se aplica a classes que herdam Serializable de uma superclasse, mas não se destinam à serialização.
  • Se deve ignorar classes Serializable sem campos não estáticos (non-static).
  • Se deve ignorar classes anônimas Serializable.
