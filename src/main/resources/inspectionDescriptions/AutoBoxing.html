Relata expressões afetadas pela conversão de autoboxing (empacotamento automático de valores primitivos em objetos). Tente evitar o uso de objetos em vez de primitivos. Isso pode afetar significativamente o desempenho.

Exemplo:

Integer x = 42;

A correção rápida torna a conversão explícita:

Integer x = Integer.valueOf(42);

O autoboxing surgiu no Java 5. Esta inspeção ajuda no downgrade para garantir a retrocompatibilidade com versões anteriores do Java.
