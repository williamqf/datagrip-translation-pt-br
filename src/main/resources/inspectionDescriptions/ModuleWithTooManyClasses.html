Relata módulos que contêm classes demais. Módulos excessivamente grandes podem indicar falta de clareza no design. Classes Java, Kotlin e Groovy são contabilizadas.

Use o campo Número máximo de classes para especificar o número máximo de classes que um módulo pode ter.
