Relata acessos a campos que não são qualificados com this ou algum outro qualificador.

Alguns estilos de código exigem que todos os acessos a campos sejam qualificados para evitar confusão com o acesso a variáveis locais ou parâmetros locais.

Exemplo:


  class Foo {
    int foo;

    void bar() {
      foo += 1;
    }
  }

Após a aplicação do ajuste rápido:


  class Foo {
    int foo;

    void bar() {
      this.foo += 1;
    }
  }
