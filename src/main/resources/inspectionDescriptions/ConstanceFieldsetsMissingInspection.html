Detecta chaves ausentes em CONSTANCE_CONFIG_FIELDSETS (uma constante usada para configurar o Django Constance).

Exemplo:


CONSTANCE_CONFIG = {
    'SITE_NAME': ('My Title', 'Website title'),
    'SITE_DESCRIPTION': ('', 'Website description'),
}

CONSTANCE_CONFIG_FIELDSETS = {
    'General Options': ('SITE_NAME'),
}
