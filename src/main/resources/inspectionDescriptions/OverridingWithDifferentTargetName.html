Reporta as definições com uma anotação @targetName que sobrescrevem definições com anotação @targetName, mas possuem nomes externos diferentes.

Veja The @targetName annotation.

A correção rápida substitui o nome externo da anotação @targetName pelo nome externo da definição sobrescrita.

Exemplo:


  trait A:
    @targetName("multiply")
    def *(i: Int): A

  class B extends A:
    @targetName("times")
    override def *(i: Int): A = ???

Depois que a correção rápida é aplicada:


  trait A:
    @targetName("multiply")
    def *(i: Int): A

  class B extends A:
    @targetName("multiply")
    override def *(i: Int): A = ???
