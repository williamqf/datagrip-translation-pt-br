Relata quaisquer arquivos package.html usados para documentar pacotes.

Desde o JDK 1.5, recomenda-se o uso do arquivo package-info.java, pois ele também pode conter anotações de pacote. Dessa forma, o package-info.java torna-se o repositório único para anotações e documentação de nível de pacote.

Exemplo: package.html


<html>
  <body>
    Documentation example.
  </body>
</html>

Após a correção rápida: package-info.java


/**
 * Documentation example.
 */
package com.sample;
