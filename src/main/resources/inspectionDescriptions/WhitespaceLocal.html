Detecta o uso inconsistente de caracteres de indentação nos espaços em branco iniciais, sinalizando especificamente tabulações onde espaços estão configurados e espaços onde tabulações estão configuradas.

O Godot não permite misturar tabulações e espaços na indentação.

Por exemplo, se o seu estilo de código estiver configurado para usar espaços para indentação: 


func _init():
  var x = 1
\tvar y = 2 # Destaque: tabulação encontrada onde espaços são preferidos

Ou, se o seu estilo de código estiver configurado para usar tabulações para indentação: 


func _init():
\tvar x = 1
    var y = 2 # Destaque: espaço encontrado onde tabulações são preferidas
