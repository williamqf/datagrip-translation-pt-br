Reporta pacotes não vazios que estão presentes em múltiplos módulos. Quando um pacote está contido em vários módulos, é fácil criar classes com o mesmo nome em dois módulos diferentes. Módulos que dependem desses módulos enfrentarão conflitos se tentarem usar tais classes. O Java Platform Module System não permite pacotes contidos em múltiplos módulos (também conhecidos como split packages).
