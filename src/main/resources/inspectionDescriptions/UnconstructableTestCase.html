Relata classes de teste JUnit que não podem ser construídas pelo runner de teste padrão JUnit.

Classes de teste JUnit 4 precisam ser public e incluir um construtor public sem parâmetros ou não incluir construtor algum (construtor padrão implícito), e não ter nenhum outro construtor public. Classes de teste JUnit 3 precisam ser public e precisar de um construtor public sem parâmetros ou de um construtor public com um único parâmetro do tipo String (que chama o superconstrutor correspondente). Caso contrário, as classes de teste não podem ser executadas pelo runner de teste padrão JUnit.

Exemplo:


public class MeuTeste {

  private MeuTeste() {} // construtor sem argumentos é private

  @Test
  public void testaAlgo() {
    assertEquals(1, 1);
  }
}
