Relata chamadas ao método java.util.List.get(). Tais chamadas podem ser substituídas por uma forma de acesso por chave mais curta e clara.

Exemplo:


def list = ["foo"]
def str = list.get(0) // list.get(0) pode ser substituído por list[0]

Após a aplicação da correção rápida:


def list = ["foo"]
def str = list[0]
