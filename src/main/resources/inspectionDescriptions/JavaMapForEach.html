Relata chamadas ao método forEach de um Map do Java que podem ser substituídas pelo forEach do Kotlin.

Exemplo:


  fun test(map: HashMap<Int, String>) {
      map.forEach { key, value ->
          foo(key, value)
      }
  }

  fun foo(i: Int, s: String) {}

A correção rápida adiciona parênteses:


  fun test(map: HashMap<Int, String>) {
      map.forEach { (key, value) ->
          foo(key, value)
      }
  }

  fun foo(i: Int, s: String) {}
