Detecta arquivos JSON que podem ser interpretados como parte de uma especificação OpenAPI/Swagger.

Esses arquivos não contêm atributos de especificação explícitos, mas são referenciados a partir de um arquivo de especificação principal próximo.

O exemplo abaixo contém dois arquivos localizados no mesmo diretório. O primeiro é um arquivo de especificação principal comum. O segundo é referenciado pelo primeiro, portanto, sugere-se que seja considerado como uma especificação.

Arquivo de especificação principal openapi.yaml:


openapi: 3.1.0
components:
  schemas:
    CustomSchema:
      description: Custom schema object
      properties:
        foo:
          $ref: 'common.components.json#/components/schemas/CommonSchema'

Candidato a arquivo de especificação common.components.json:


{
  "components": { // destaque 'Marcar arquivo como especificação OpenAPI'
    "schemas": {
      "CommonSchema": {
        "description": "Common schema object reused in several specifications"
      }
    }
  }
}
