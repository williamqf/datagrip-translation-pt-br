Relata uma string com aspas simples no formato de array JSON.

No formato de array JSON, aspas duplas (") devem ser usadas ao redor das palavras, não aspas simples ('). Caso contrário, o build do Docker falhará.

Exemplos:


  # todos os comandos abaixo falharão
  RUN ['/bin/bash', '-c', 'echo hello']
  ADD ['binaryA.jar', 'binary2.jar', 'destination/']
  COPY ['binaryA.jar', 'binary2.jar', 'destination/']

Após a aplicação da correção rápida:


  RUN ["/bin/bash", "-c", "echo hello"]
  ADD ["binaryA.jar", "binary2.jar", "destination/"]
  COPY ["binaryA.jar", "binary2.jar", "destination/"]
