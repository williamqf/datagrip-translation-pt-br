Relata propriedades Boolean não anuláveis em interfaces externas em projetos Kotlin/JS.

As propriedades Boolean em interfaces externas devem ser anuláveis para lidar com casos em que o valor JavaScript pode ser undefined.

Exemplo:

external interface MyInterface {
    var enabled: Boolean // Aviso: deve ser Boolean?
    var visible: Boolean // Aviso: deve ser Boolean?
}

Após aplicar a correção rápida:

external interface MyInterface {
    var enabled: Boolean?
    var visible: Boolean?
}
