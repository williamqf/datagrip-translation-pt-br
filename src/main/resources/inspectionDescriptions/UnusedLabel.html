Relata rótulos que não são o alvo de nenhuma instrução break ou continue.

Exemplo:


  label: for (int i = 0; i < 10; i++) {
    if (i == 3) {
      break;
    }
  }

Após a aplicação da correção rápida, o rótulo será removido:


  for (int i = 0; i < 10; i++) {
    if (i == 3) {
      break;
    }
  }
