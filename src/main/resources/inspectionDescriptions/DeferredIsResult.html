Inspeciona funções que retornam o tipo kotlinx.coroutines.Deferred.

Funções que retornam Deferred devem ter o sufixo Async em seus nomes. Caso contrário, é recomendado marcar a função como suspend e desempacotar o Deferred nela.

Exemplo:


  fun calcEverything(): Deferred<Int> {
      return CompletableDeferred(42)
  }

Após aplicar a correção rápida que adiciona o sufixo Async:


  fun calcEverythingAsync(): Deferred<Int> {
      return CompletableDeferred(42)
  }

Após aplicar a correção rápida que converte a função para suspend:


  suspend fun calcEverything(): Int {
      return CompletableDeferred(42).await()
  }
