Relata chamadas ineficazes aos métodos de String replace(), replaceAll() ou replaceFirst(). Tais chamadas são comprovadamente ineficazes quando o qualificador e a string de busca são constantes em tempo de compilação e a string de busca não é encontrada no qualificador. Isso é código redundante e pode indicar um erro.

Exemplo:


  // A substituição não faz nada
  "hello".replace("$value$", value);
