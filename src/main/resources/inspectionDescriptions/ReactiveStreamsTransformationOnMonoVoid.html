Relata operações de transformação invocadas em Mono with void value dentro de uma cadeia Reactive Streams.

Invocar uma cadeia Reactive Streams em Mono with Void value não terá efeito. A correção rápida sugere substituir map(), flatMap() por then() que retorne outro Mono.

Exemplo:

Mono.when(Mono.just(1), Mono.just(2))
                .map(v -> "New value")
                .doFinally(System.out::println);

Após aplicar a correção rápida:

Mono.when(Mono.just(1), Mono.just(2))
                .then(Mono.fromCallable(() -> "New value"))
                .doFinally(System.out::println);
