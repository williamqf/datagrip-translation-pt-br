Relata referências a tipos genéricos com um número incorreto de argumentos de tipo. É um erro usar um tipo genérico sem passar a quantidade correta de argumentos de tipo.

Uma correção rápida pode ser usada para corrigir a quantidade de argumentos de tipo passados.

Exemplo:

class Example[A]
end

# Muitos argumentos de tipo fornecidos
type example = Example[untyped, untyped]

Após aplicar a correção rápida:

class Example[A]
end

type example = Example[untyped]
