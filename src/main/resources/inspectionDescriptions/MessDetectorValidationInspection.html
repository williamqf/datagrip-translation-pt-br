Relata problemas de estilo de código detectados pelo PHP Mess Detector.

Esta inspeção requer que o PHP Mess Detector esteja instalado corretamente e configurado no IDE em Configurações | PHP | Ferramentas de Qualidade | Mess Detector.

Para saber mais sobre a instalação do PHP Mess Detector, consulte PHP Mess Detector installation (phpmd.org).

Você pode usar conjuntos de regras predefinidos ou adicionar suas próprias regras especificando um arquivo ruleset.xml válido. Para mais detalhes, consulte o índice de regras (phpmd.org) e Como criar um conjunto de regras personalizado (phpmd.org).
