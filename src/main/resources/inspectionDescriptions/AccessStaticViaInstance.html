Relata referências a métodos e campos static por meio de instâncias de classe, em vez da própria classe.

Embora a Especificação da Linguagem Java permita o uso de variáveis de instância para referenciar membros static, isso pode tornar o código confuso, pois os leitores podem interpretar erroneamente que o método depende da instância.

A correção rápida substitui a variável de instância pelo nome da classe.

Exemplo:


  String s1 = s.valueOf(0);

Após a aplicação da correção rápida:


  String s = String.valueOf(0);
