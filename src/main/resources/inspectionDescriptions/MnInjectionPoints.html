Relata pontos de injeção em classes Bean que possuem dependências ausentes ou ambíguas.

Exemplo:


  interface Engine {}

  @Singleton
  class GasEngine implements Engine {}

  @Singleton
  class ElectricEngine implements Engine {}

  @Singleton
  class Car {

    @Inject
    Engine engine; // Dependência ambígua: existem múltiplos Beans correspondentes ao ponto de injeção
  }
