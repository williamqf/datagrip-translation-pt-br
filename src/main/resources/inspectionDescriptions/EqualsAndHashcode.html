Relata classes que sobrescrevem o método equals(), mas não o método hashCode(), ou vice-versa. Isso pode causar problemas ao adicionar a classe a uma Collection ou HashMap.

A correção rápida gera uma implementação padrão para o método ausente.

Exemplo:


class StringHolder {
  String s;

  @Override public int hashCode() {
    return s != null ? s.hashCode() : 0;
  }
}

Após a aplicação da correção rápida:


class StringHolder {
  String s;

  @Override public int hashCode() {
    return s != null ? s.hashCode() : 0;
  }

  @Override
  public boolean equals(Object o) {
    if (this == o) return true;
    if (!(o instanceof StringHolder)) return false;

    StringHolder holder = (StringHolder)o;

    if (s != null ? !s.equals(holder.s) : holder.s != null) return false;

    return true;
  }
}
