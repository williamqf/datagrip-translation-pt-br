Relata sequências de escape de caracteres de line feed (\n) e carriage return (\r) usadas em literais de string, literais de caractere ou blocos de texto. Esses caracteres são frequentemente usados como separadores de linha e, se forem hardcoded, a portabilidade poderá ser afetada.

Exemplo:


  String count = "first\nsecond\rthird";
