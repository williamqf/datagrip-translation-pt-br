Reporta o uso de instruções DELETE ou UPDATE sem uma cláusula WHERE.

Sem uma cláusula WHERE, o DELETE exclui todos os dados da tabela, enquanto o UPDATE sobrescreve os valores de todas as linhas da tabela.

Exemplo (MySQL):

CREATE TABLE t1 (a TEXT, b INT, c BOOLEAN);
update t1 set  a = 'Smith';
delete from t1;
