Relata declarações marcadas com @ApiStatus.ScheduledForRemoval que já deveriam ter sido removidas da versão atual da biblioteca declarante.

Ela compara a versão especificada para a remoção programada com a versão que pode ser definida abaixo.

Especifique a versão como uma string separada por pontos, opcionalmente acrescida dos sufixos alpha, beta, snapshot ou eap.

Exemplos de versões válidas: 1.0, 2.3.1, 2018.1, 7.5-snapshot, 3.0-eap.

A comparação de versões é intuitiva, por exemplo: 1.0 < 2.0, 1.0-eap < 1.0, 2.3-snapshot < 2.3, etc. Para detalhes sobre a lógica de comparação, consulte a implementação do VersionComparatorUtil.
