Relata funções que correspondem a uma das convenções de operador, mas não possuem a palavra-chave operator.

Ao adicionar o modificador operator, você permite que os usuários da função escrevam código Kotlin idiomático.

Exemplo:


  class Complex(val real: Double, val imaginary: Double) {
      fun plus(other: Complex) =
          Complex(real + other.real, imaginary + other.imaginary)
  }

  fun usage(a: Complex, b: Complex) {
      a.plus(b)
  }

A correção rápida adiciona a palavra-chave modificadora operator:


  class Complex(val real: Double, val imaginary: Double) {
      operator fun plus(other: Complex) =
          Complex(real + other.real, imaginary + other.imaginary)
  }

  fun usage(a: Complex, b: Complex) {
      a + b
  }
