Relata atribuições compostas se o tipo do operando do lado direito for incompatível com o tipo da variável.

Durante essas atribuições compostas, ocorre uma coerção implícita, o que pode resultar em uma conversão com perda de dados.

Exemplo:


  long c = 1;
  c += 1.2;

Após a aplicação da correção rápida:


  long c = 1;
  c += (long) 1.2;

Novidade na versão 2023.2
