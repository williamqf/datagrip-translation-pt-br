Relata código que sobrescreve chaves de Map, elementos de Set ou elementos de array em sequências de chamadas add/put, ou que utiliza métodos de fábrica do Java 9 como o Set.of (o que resultará em uma exceção de tempo de execução).

Isso geralmente é causado por erros de copiar e colar.

Exemplo:


  map.put("A", 1);
  map.put("B", 2);
  map.put("C", 3);
  map.put("D", 4);
  map.put("A", 5); // A chave "A" é duplicada, sobrescrevendo a entrada gravada anteriormente

Novidade na versão 2017.3
