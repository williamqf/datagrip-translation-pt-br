Relata métodos de classe cujos nomes são muito curtos, muito longos ou não correspondem ao padrão de expressão regular especificado.

Use o campo Padrão para especificar a expressão regular a ser usada para validar os nomes dos métodos de classe.

Use os campos Comprimento mínimo e Comprimento máximo para especificar o comprimento mínimo e máximo do nome, respectivamente.
