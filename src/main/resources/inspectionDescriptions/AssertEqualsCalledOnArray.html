Relata chamadas JUnit assertEquals() com argumentos do tipo array. Esse método compara a identidade do array, em vez do seu conteúdo. O conteúdo do array deve ser verificado usando o método assertArrayEquals().

Exemplo:


  @Test
  public void testSort() {
    int[] actual = {248, 496, 0, 56};
    Arrays.sort(actual);
    Assert.assertEquals(new int[] {0, 56, 248, 496}, actual);
  }

Após a aplicação do ajuste rápido:


  @Test
  public void testSort() {
    int[] actual = {248, 496, 0, 56};
    Arrays.sort(actual);
    Assert.assertArrayEquals(new int[] {0, 56, 248, 496}, actual);
  }
