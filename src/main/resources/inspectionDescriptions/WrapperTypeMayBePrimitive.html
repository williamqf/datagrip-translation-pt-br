Relata variáveis locais de tipos wrapper usadas principalmente como tipos primitivos.

Em alguns casos, o boxing pode causar uma perda significativa de desempenho, especialmente dentro de loops.

Heurísticas são aplicadas para estimar o número de operações de boxing. Por exemplo, as conversões dentro de loops são consideradas muito mais frequentes.

Exemplo:

  public void example() {
    Integer value = 12;
    needBox(value);
    for (int i = 0; i < 10; i++) {
      // Usos em loops são considerados como ocorrendo com mais frequência
      needPrimitive(value);
    }
  }

  void needPrimitive(int value) {}
  void needBox(Integer value) {}

Após a aplicação da correção rápida:

  public void example() {
    int value = 12;
    needBox(value);
    for (int i = 0; i < 10; i++) {
      // Usos em loops são considerados como ocorrendo com mais frequência
      needPrimitive(value);
    }
  }

  void needPrimitive(int value) {}
  void needBox(Integer value) {}
