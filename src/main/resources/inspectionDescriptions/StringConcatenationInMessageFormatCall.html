Relata a concatenação de strings não constantes usadas como argumentos em chamadas para MessageFormat.format().

Embora ocasionalmente intencional, isso geralmente é um uso incorreto do método de formatação e pode até levar a exceções se a string concatenada contiver caracteres especiais como {.

Além disso, às vezes isso pode ser o resultado de concatenar incorretamente argumentos de formatação de string digitando + onde uma , deveria ter sido inserida.

Exemplo:


  String formatGreeting(String userName, int balance) {
    return MessageFormat.format("Hello, " + userName + "! Your balance is {0}.", balance);
  }

Aqui, userName é interpretado como parte da string de formato, o que pode resultar em uma IllegalArgumentException (por exemplo, se userName for "{"). Esta chamada provavelmente deveria ser substituída por MessageFormat.format("Hello, {0}! Your balance is {1}.", userName, balance).
