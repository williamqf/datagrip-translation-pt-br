Relata expressões binárias que contêm diferentes operadores e não possuem parênteses.

Tais expressões podem ter baixa legibilidade devido às diferentes regras de precedência de operadores.

Exemplo:

  fun foo(b: Boolean?, i: Int?) {
      val x = b ?: i == null // avaliado como `(b ?: i) == null`
      val y = i ?: 0 + 1 // avaliado como `i ?: (0 + 1)`
  }
