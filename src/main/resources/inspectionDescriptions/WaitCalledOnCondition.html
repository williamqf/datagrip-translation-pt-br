Relata chamadas wait() em objetos java.util.concurrent.locks.Condition. Isso provavelmente é um erro de programação, onde se pretendia usar alguma variante do método await().

Exemplo:


  void acquire(Condition released) throws InterruptedException {
    while (acquired) {
      released.wait();
    }
  }

O código correto deve ser semelhante a:


  void acquire(Condition released) throws InterruptedException {
    while (acquired) {
      released.await();
    }
  }
