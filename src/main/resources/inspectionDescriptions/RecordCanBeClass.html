Relata classes record e sugere convertê-las em classes comuns.

Ao aplicar uma correção rápida a este registro, esta inspeção permite mover registros Java para uma base de código que utilize versões anteriores do Java.

Observe que a classe resultante não é exatamente equivalente ao registro original:

  • A classe resultante não estende mais java.lang.Record, portanto instanceof Record retorna false.
  • Métodos de reflexão, como Class.isRecord() e Class.getRecordComponents(), produzem resultados diferentes.
  • A implementação de hashCode() gerada pode produzir resultados diferentes, pois a fórmula para calcular o hashCode de um registro não é especificada intencionalmente.
  • O mecanismo de serialização de registros é diferente do de classes comuns. Para mais detalhes, consulte a Java Object Serialization Specification.

Exemplo:


  record Point(int x, int y) {}

Após aplicar a correção rápida:


  final class Point {
    private final int x;
    private final int y;

    Point(int x, int y) {
      this.x = x;
      this.y = y;
    }

    public int x() { return x; }

    public int y() { return y; }

    @Override
    public boolean equals(Object obj) {
      if (obj == this) return true;
      if (obj == null || obj.getClass() != this.getClass()) return false;
      var that = (Point)obj;
      return this.x == that.x &&
             this.y == that.y;
    }

    @Override
    public int hashCode() {
      return Objects.hash(x, y);
    }

    @Override
    public String toString() {
      return "Point[" +
             "x=" + x + ", " +
             "y=" + y + ']';
    }
  }
