Informa declarações de pacotes que devem ser encadeadas ao pacote de base.

Aplica-se somente quando um pacote de base é definido em Configurações | Linguagens & Frameworks | Scala | Pacotes de Base.

Exemplo:


  package org.example.application.NOME

Após a correção rápida ser aplicada:


  package org.example.application
  package NOME
