Reporta instruções switch que não contêm um rótulo default.

Adicionar um rótulo default garante que todos os cenários possíveis sejam cobertos, facilitando a criação de suposições sobre o estado atual do programa.

Por padrão, esta inspeção não reportará instruções switch se todos os casos para enums ou classes sealed forem cobertos. Se você deseja alterar esse comportamento, use a opção Ignorar instruções switch exaustivas.
