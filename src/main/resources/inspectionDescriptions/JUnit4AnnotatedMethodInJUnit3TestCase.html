Reportando os métodos anotados com JUnit 4 @Test em herdeiros de junit.framework.TestCase. A mistura de APIs do JUnit pode causar problemas ao executar o teste.

Exemplo:


  public class MyTest extends TestCase {
      @Test // O nome não inicia com "test", então será ignorado
      public void wouldBeIgnored() {}
  
      @Test   // O nome inicia com "test"
      @Ignore // Então será executado mesmo com a anotação @Ignore
      public void testWouldBeExecuted() {}
  }

Correção fornecida:

  • Remover a anotação @Ignore e renomear o método de teste para que o nome não inicie com "test".
  • Converter a classe de teste JUnit 3 para JUnit 4.
