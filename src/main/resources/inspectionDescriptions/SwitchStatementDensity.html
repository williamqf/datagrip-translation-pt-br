Relata instruções ou expressões switch onde a proporção de rótulos switch para instruções executáveis é muito baixa.

Tais instruções switch podem ser confusas e talvez devam ser refatoradas.

Exemplo:


    switch (i) { // 1 case, 5 instruções executáveis -> 20% de densidade
        case 1:
            System.out.println("1");
            System.out.println("2");
            System.out.println("3");
            System.out.println("4");
            System.out.println("5");
            break;
    }

Use o campo Densidade mínima de ramificações para especificar a proporção permitida de rótulos switch em relação às instruções executáveis.
