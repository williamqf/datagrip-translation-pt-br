Relata expressões de atribuição composta (+=) em Collection somente leitura.

Expressões de atribuição composta (+=) em Collection somente leitura alocam temporariamente uma nova coleção, o que pode afetar o desempenho.

A correção rápida Alterar tipo para mutável pode ser usada para corrigir o código automaticamente.

Exemplo:


  fun test() {
      var list = listOf(0)
      list += 42 // Aqui uma nova lista é alocada, equivalente a list = list + 42
  }

Após aplicar a correção rápida:


  fun test() {
      val list = mutableListOf(0)
      list += 42
  }
