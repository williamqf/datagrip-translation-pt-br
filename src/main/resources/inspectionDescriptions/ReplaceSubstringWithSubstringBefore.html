Relata chamadas como s.substring(0, s.indexOf(x)) que podem ser substituídas por s.substringBefore(x).

O uso de substringBefore() torna o código mais simples.

A correção rápida substituirá a chamada substring por substringBefore.

Exemplo:


  fun foo(s: String) {
      s.substring(0, s.indexOf('x'))
  }

Após a aplicação da correção rápida:


  fun foo(s: String) {
      s.substringBefore('x')
  }
