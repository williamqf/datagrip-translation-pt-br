Relata chamadas de métodos parametrizados nas quais os tipos de argumento explícitos podem ser omitidos, visto que o compilador os inferirá de forma clara.

O uso de argumentos de tipo redundantes é desnecessário e reduz a legibilidade do código.

Exemplo:


  List<String> list = Arrays.<String>asList("Hello", "World");

Uma correção rápida está disponível para remover argumentos de tipo redundantes:


  List<String> list = Arrays.asList("Hello", "World");
