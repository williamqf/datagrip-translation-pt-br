Reporta chamadas ao método java.util.Map.put(). Tais chamadas podem ser substituídas por uma forma de acesso por chave mais curta e clara.

Exemplo:


def map = ["foo": "bar"]
map.put("foo", "baz") // map.put("foo", "baz") pode ser substituído por map["foo"] = "baz"

Após a aplicação do ajuste rápido:


def map = ["foo": "bar"]
map["foo"] = "baz"
