Reporta chamadas como s.substring(0, s.length - x) que podem ser substituídas por s.dropLast(x).

O uso da função correspondente torna o código mais simples.

A correção rápida substitui a chamada substring por dropLast.

Exemplo:


  fun foo(s: String) {
      s.substring(0, s.length - 5)
  }

Após a aplicação da correção rápida:


  fun foo(s: String) {
      s.dropLast(5)
  }
