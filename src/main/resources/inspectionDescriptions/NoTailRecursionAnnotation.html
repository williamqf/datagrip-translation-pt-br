Relata métodos recursivos finais sem anotação @tailrec que verifica se o método será compilado com otimização de chamada de cauda.

Observe que um método deve ser efetivamente final para que esta inspeção seja exibida.

Exemplo:


  def factorial(n: Long, acc: Long = 1): Long =
    se (n <= 1) acc senão factorial(n - 1, acc * n)

Após a correção rápida ser aplicada:


  @tailrec
  def factorial(n: Long, acc: Long = 1): Long =
    se (n <= 1) acc senão factorial(n - 1, acc * n)
