Relata campos não final acessados em contextos synchronized e não synchronized. Campos volatile e acessos em construtores e inicializadores são ignorados por esta inspeção.

Esse tipo de acesso "parcialmente sincronizado" é frequentemente o resultado de um descuido de codificação e pode levar a estruturas de dados inconsistentes de forma inesperada.

Exemplo:


  public class Program {
    Console console; // Aviso: o campo 'console' é acessado em contextos sincronizados e não sincronizados

    public synchronized void execute() {
      console.print("running");
    }

    public void check() {
      console.check();
    }
  }

Use a opção para especificar se getters e setters simples também contam como acesso.
