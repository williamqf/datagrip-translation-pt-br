Relata propriedades lateinit var que sobrescrevem outras propriedades lateinit var.

Uma instância de subclasse terá dois campos para uma única propriedade, e o campo da superclasse permanecerá sem uso.

Exemplo:


    open class BaseClass {
      open lateinit var name: String
    }

    class RealClass : BaseClass() {
      override lateinit var name: String
    }
