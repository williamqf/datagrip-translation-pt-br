Relata parâmetros de tipo e argumentos de tipo curinga que são declarados explicitamente para estender java.lang.Object.

Tais cláusulas 'extends' são redundantes, pois java.lang.Object é o supertipo de todas as classes.

Exemplo:

  class ClassA<T extends Object> {}

Se a cláusula 'extends Object' precisar ser mantida devido a uma anotação, desative a opção Ignorar quando java.lang.Object estiver anotado. Isso pode ser útil, por exemplo, se você estiver usando um analisador de nulidade e a cláusula 'extends Object' contiver uma anotação @Nullable/@NotNull.

Exemplo:

  class MyClass<T extends @NotNull Object> {}
