Relata uma invocação de construtor Pair que pode ser substituída por uma chamada de função infixa to().

Invocações explícitas de construtores podem adicionar verbosidade, especialmente se forem usadas várias vezes. Substituir chamadas de construtor por to() torna o código mais fácil de ler e manter.

Exemplo:


  val countries = mapOf(
      Pair("França", "Paris"),
      Pair("Espanha", "Madri"),
      Pair("Alemanha", "Berlim")
  )

Após a correção rápida ser aplicada:


  val countries = mapOf(
      "França" to "Paris",
      "Espanha" to "Madri",
      "Alemanha" to "Berlim"
  )
