Relata tentativas de sincronização em instâncias de classes baseadas em valor que, a partir do Java 16, geram avisos em tempo de compilação e lançam exceções em tempo de execução.

Por exemplo, java.lang.Double é anotada com jdk.internal.ValueBased, portanto, o código a seguir gera um aviso em tempo de compilação: 


Double d = 20.0;
synchronized (d) { ... } // aviso do javac

Novo na versão 2021.1
