Relata a sincronização em uma variável local ou parâmetro.

Ao usar esse tipo de sincronização, é muito difícil garantir o funcionamento correto. Esse código pode ser melhorado, por exemplo, controlando o acesso por meio de classes wrapper sincronizadas ou sincronizando em um campo.

Exemplo:


  void bar() {
    final Object lock = new Object();
    synchronized (lock) { }
  }
