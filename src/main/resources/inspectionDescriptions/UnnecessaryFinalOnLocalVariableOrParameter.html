Relata variáveis locais ou parâmetros declarados desnecessariamente como final.

Alguns padrões de codificação desencorajam declarações de variáveis como final por motivos de concisão.

Exemplo:


  class Foo {
    Foo(Object o) {}

    void bar(final Object o) {
      new Foo(o);
    }
  }

Após a aplicação da correção rápida:


  class Foo {
    Foo(Object o) {}

    void bar(Object o) {
      new Foo(o);
    }
  }

Use as opções desta inspeção para definir quais tipos de parâmetros ou variáveis locais devem ser relatados.
