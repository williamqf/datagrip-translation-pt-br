Relata quaisquer chamadas a métodos na classe junit.framework.Assert. Esta classe está obsoleta e essas chamadas podem ser substituídas por chamadas a métodos na classe org.junit.Assert.

Exemplo:


  import org.junit.*;
  public class NecessaryTest {
    @Test
    public void testIt() {
      junit.framework.Assert.assertEquals("expected", "actual");
    }
  }

Após a aplicação da correção rápida, o resultado será assim:


  import org.junit;
  public class NecessaryTest {

    public void testIt() {
      Assert.assertEquals("expected", "actual");
    }
  }
