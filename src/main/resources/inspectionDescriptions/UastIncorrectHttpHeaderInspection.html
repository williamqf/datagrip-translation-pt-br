Relata cabeçalhos HTTP desconhecidos que não correspondem a nenhum dos cabeçalhos publicamente conhecidos. A correção rápida sugere adicionar o cabeçalho à lista de cabeçalhos personalizados para evitar que esta inspeção seja acionada no futuro.

Os cabeçalhos HTTP personalizados com o mesmo nome são listados no grupo "Cliente HTTP" para esta inspeção.
