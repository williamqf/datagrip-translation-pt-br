Relata variáveis cujos nomes são muito curtos, muito longos ou que não correspondem ao padrão de expressão regular especificado. Alguns estilos de codificação possuem convenções de nomenclatura especiais para variáveis.

Use as opções de inspeção para especificar o comprimento mínimo/máximo e a expressão regular esperada para os nomes das variáveis. Para ignorar o comprimento do nome, especifique 0.

Para saber mais sobre expressões regulares, consulte o Quick Start guide (regular-expressions.info).
