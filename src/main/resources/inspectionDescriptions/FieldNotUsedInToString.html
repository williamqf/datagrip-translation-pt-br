Relata campos que não são utilizados no método toString() de uma classe.

Ajuda a descobrir campos adicionados após a última atualização do método toString(). A correção rápida gera novamente o método toString().

No diálogo Gerar | toString(), é possível excluir campos desta inspeção. Se a opção Habilitar getters na geração de código estiver ativada, esta inspeção também verificará problemas nos métodos getter.

Exemplo:


  public class Relevant {
    private String name; // não usado no toString()
    private int index;
    private int length;

    @Override
    public String toString() {
        return "Relevant{" + "index=" + index +
          ", length=" + length + '}';
    }
  }

Após aplicar a correção rápida:


  public class Relevant {
    private String name;
    private int index;
    private int length;

    @Override
    public String toString() {
        return "Relevant{" + "name='" + name + '\'' +
          ", index=" + index + ", length=" + length + '}';
    }
  }
