Relata expressões de cast desnecessárias.

Exemplo:


  static Object toObject(String s) {
    return (Object) s;
  }

Após a aplicação do ajuste rápido:


    static Object toObject(String s) {
    return s;
  }

Use a caixa de seleção abaixo para ignorar casts de esclarecimento, como casts em chamadas de coleção onde um Object é esperado: 


  static void removeFromList(List<String> l, Object o) {
    l.remove((String)o);
  }
