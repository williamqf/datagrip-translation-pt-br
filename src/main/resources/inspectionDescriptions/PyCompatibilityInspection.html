Relata incompatibilidades com as versões especificadas do Python. Habilite esta inspeção se você deseja que seu código seja compatível com uma série de versões do Python (por exemplo, ao criar uma biblioteca).

Para definir o intervalo de versões do Python inspecionadas, selecione as caixas de seleção correspondentes na seção Opções.

Para obter detalhes sobre as versões do Python compatíveis com o IDE, consulte a Ajuda da Web.
