Esta inspeção identifica chamadas de método em que um argumento que pode ser potencialmente null é passado como parâmetro que está anotado com @NotNull, ajudando a detectar NullPointerExceptions potenciais antes que ocorram em tempo de execução.

Exemplo:


def test(@Nullable greeting: String): Unit = {
  sayHello(greeting)
}

def sayHello(@NotNull greeting: String): Unit = {
  println(greeting.toLowerCase)
}
