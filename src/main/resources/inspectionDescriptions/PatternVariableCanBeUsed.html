Relata declarações de variáveis locais que podem ser substituídas por variáveis de padrão, que geralmente são mais compactas.

Exemplo:


  if (obj instanceof String) {
    String str = (String) obj;
    System.out.println(str);
  }

Pode ser substituído por:


  if (obj instanceof String str) {
    System.out.println(str);
  }
