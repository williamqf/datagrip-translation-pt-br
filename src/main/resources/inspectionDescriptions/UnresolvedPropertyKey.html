Reporta argumentos inválidos passados para métodos com parâmetros anotados com @PropertyKey.

Esses argumentos devem ser chaves de propriedade válidas no arquivo de propriedades correspondente. Além disso, esta inspeção verifica se o argumento resourceBundle da anotação @PropertyKey é um bundle de recursos existente.

Use a correção rápida para criar uma nova propriedade ou selecionar uma existente.

Exemplo:


  @PropertyKey(resourceBundle = "myBundle") String value = "invalid.key";
