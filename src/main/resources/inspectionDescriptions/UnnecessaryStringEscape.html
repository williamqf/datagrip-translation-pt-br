Relata caracteres escapados desnecessariamente em literais de String e, opcionalmente, em literais de char.

Caracteres de tabulação escapados (\t) não são relatados porque as tabulações são difíceis de distinguir de espaços, mesmo que não seja necessário escapá-los.

Exemplo:


  String s = "\'Scare\' quotes";
  String t = """
    All you need is\n\tLove\n""";

Após a aplicação da correção rápida:


  String s = "'Scare' quotes";
  String t = """
    All you need is
    \tLove
    """;
