Relata chamadas de log que estão envolvidas por condições de guarda. Esta inspeção pode ser usada para se ajustar a estilos de código personalizados.

Exemplo:


  public class TestObject {
    void test(Object object) {
      if(LOG.isDebugEnabled()){
        LOG.debug("some logging " + expensiveCalculation(1));
      }
    }
  }

Após a aplicação da correção rápida:


  public class TestObject {
    void test(Object object) {
      LOG.debug("some logging " + expensiveCalculation(1));
    }
  }

Esta inspeção suporta os frameworks de log Log4j2 e SLF4J (exceto builders).
