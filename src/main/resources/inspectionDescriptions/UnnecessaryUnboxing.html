Relata o unboxing, que é o desempacotamento explícito de um valor primitivo envolvido (wrapped).

Para o Java 5 e versões posteriores, o unboxing é desnecessário e pode ser removido com segurança.

Exemplo:

  • Integer i = Integer.valueOf(42).intValue(); → Integer i = Integer.valueOf(42);
  • int k = Integer.valueOf(42).intValue(); → int k = Integer.valueOf(42);

    (Relatado apenas se a opção Relatar apenas expressões de unboxing verdadeiramente redundantes estiver desmarcada)

Use a opção Relatar apenas expressões de unboxing verdadeiramente redundantes para relatar apenas o unboxing verdadeiramente redundante, onde o valor desempacotado é imediatamente empacotado (boxing), seja de forma implícita ou explícita. Nesse caso, toda a etapa de unboxing-boxing pode ser removida. Esta inspeção não relata unboxing explícito simples.

Esta inspeção relata apenas se o nível de linguagem do projeto ou módulo for 5 ou superior.
