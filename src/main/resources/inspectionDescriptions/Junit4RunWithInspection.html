Relata quando classes pai e filhas em uma hierarquia de testes JUnit são anotadas com @RunWith. Isso pode causar um comportamento inesperado nos testes.

Exemplo:


  @RunWith(Suite.class)
  @SuiteClasses(MySuiteClass.class)
  public abstract Parent {
  }

  @RunWith(Parameterized.class)
  public MyTest {
  }

Novidade na versão 2024.1
