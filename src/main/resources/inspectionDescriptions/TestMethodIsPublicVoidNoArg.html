Relata quaisquer métodos de teste JUnit que são declarados como static, não são declarados como public, não retornam void ou possuem parâmetros de função. Tais métodos de teste são facilmente criados sem querer, e não serão executados pelo executor de teste JUnit.

Exemplo:


  @Test
  void test() {
    assertEquals(1, 1);
  }

Após aplicação do reparo rápido:


  @Test
  public void test() {
    assertEquals(1, 1);
  }
É possível especificar anotações usando a opção de inspeção. Quaisquer parâmetros de função anotados com uma dessas anotações não serão relatados.
