Relata chaves depreciadas em arquivos YAML.

A depreciação é verificada apenas se houver um esquema JSON associado ao arquivo YAML correspondente.

Observe que o mecanismo de depreciação ainda não foi definido na especificação do esquema JSON e esta inspeção usa a extensão não padrão deprecationMessage.

Exemplo de depreciação de esquema:


  {
    "properties": {
      "SomeDeprecatedProperty": {
        "deprecationMessage": "Baz",
        "description": "Foo bar"
      }
    }
  }

Abaixo está um exemplo com o aviso correspondente:


  SomeDeprecatedProperty: some value
