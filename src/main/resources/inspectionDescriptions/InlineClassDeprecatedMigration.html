Relata classes inline que foram depreciadas no Kotlin 1.5 e versões posteriores, resultando em avisos do compilador. Consulte O que há de novo no Kotlin 1.5.0.

Exemplo:


  inline class Password(val s: String)

Após a aplicação da correção rápida:


  @JvmInline
  value class Password(val s: String)

A inspeção está disponível para o nível de linguagem Kotlin 1.5 ou superior.
