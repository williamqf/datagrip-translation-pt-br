Relata campos injetados ou autowired em componentes Spring.

A correção rápida sugere a injeção de dependência baseada em construtor recomendada em Beans e asserções para campos obrigatórios.

Exemplo:


class MyComponent {
  @Inject MyCollaborator collaborator; // campo injetado

  public void myBusinessMethod() {
    collaborator.doSomething(); // lança NullPointerException
  }
}

Após aplicar a correção rápida:


class MyComponent {

  private final MyCollaborator collaborator;

  @Inject
  public MyComponent(MyCollaborator collaborator) {
    Assert.notNull(collaborator, "MyCollaborator must not be null!");
    this.collaborator = collaborator;
  }

  public void myBusinessMethod() {
    collaborator.doSomething(); // agora esta chamada é segura
  }
}
