Relata declarações de nome @PathVariable redundantes quando coincidem com o nome do parâmetro. A correção rápida remove o atributo de nome explícito da anotação.

Exemplo:


  @GetMapping("/{orderId}")
  public Order getOrder(@PathVariable("orderId") Long orderId)
    return return orderRepository.findOne(orderId);
  }

Após a aplicação da correção rápida, o resultado será:


  @GetMapping("/{orderId}")
  public Order getOrder(@PathVariable Long orderId)
    return orderRepository.findOne(orderId);
  }
