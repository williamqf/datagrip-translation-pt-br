Relata direções de ordenação redundantes, como ASC e DESC, na cláusula ORDER BY.

Exemplo (MySQL):

CREATE TABLE foo(a INT, b INT, c INT);
SELECT * FROM foo ORDER BY a ASC, b DESC, c ASC;

A palavra-chave ORDER BY ordena os registros em ordem crescente por padrão. Portanto, a palavra-chave ASC para as colunas a e c é redundante.
