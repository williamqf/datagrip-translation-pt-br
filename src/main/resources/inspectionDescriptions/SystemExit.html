Relata chamadas a System.exit(), Runtime.exit() e Runtime.halt().

Chamar System.exit() ou Runtime.exit() invoca shutdown hooks e encerra a Java Virtual Machine em execução. Chamar Runtime.halt() encerra forçadamente a JVM sem iniciar os shutdown hooks. Cada um desses métodos deve ser usado com cautela extrema. Chamadas a esses métodos tornam o código de chamada não portável para a maioria dos servidores de aplicação.

Use a opção para ignorar chamadas no método main.
