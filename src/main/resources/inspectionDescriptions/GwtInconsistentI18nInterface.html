Relata inconsistências entre os métodos de interfaces que estendem Constants ou Messages e as propriedades definidas nos arquivos correspondentes.

Os métodos de interface provenientes de interfaces que estendem com.google.gwt.i18n.client.Constants e com.google.gwt.i18n.client.Messages devem corresponder às propriedades de um arquivo de propriedades vinculado.

A correção rápida cria métodos ausentes para as propriedades encontradas no arquivo de propriedades e propriedades ausentes para os métodos definidos na interface.
