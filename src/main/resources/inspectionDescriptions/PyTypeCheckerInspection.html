Relata erros de tipo em expressões de chamada de função, destinos e valores de retorno. Em linguagens de tipagem dinâmica, isso pode ocorrer em um número limitado de casos.

Os tipos dos parâmetros de função podem ser especificados em docstrings ou em anotações de função do Python 3.

Exemplo:


def foo() -> int:
    return "abc" # esperado int, obtido str


a: str
a = foo() # esperado str, obtido int

Uma correção rápida pode ser usada para alterar o tipo problemático:


def foo() -> str:
    return "abc"


a: str
a = foo()
