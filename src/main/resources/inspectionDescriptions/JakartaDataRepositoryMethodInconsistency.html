Relata métodos de repositório Jakarta Data para os quais o builder não consegue gerar uma consulta correspondente.

Exemplo:


interface PersonRepository extends Repository<Person, Long> {
  List<Person> findByEmailAndLastname(String email, String lastname);
  List<Person> findByUnknownAndLastname(EmailAddress emailAddress, String lastname); // não é possível resolver a propriedade 'Unknown'
  List<Person> findByAndLastname(String email, String lastname); // nome da propriedade ausente

  List<Person> findByLastnameOrderByFirstnameAsc(String lastname);
  List<Person> findByLastnameOrderByUnknownDesc(String lastname);  // não é possível resolver a propriedade 'Unknown'
}
