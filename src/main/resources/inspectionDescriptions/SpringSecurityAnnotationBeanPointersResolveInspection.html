Relata Spring Beans não resolvidos referenciados em anotações do Spring Security.

Exemplos:

  • @WithUserDetails(value = "admin", userDetailsServiceBeanName = "unknownUserDetailsService")
      // Se 'unknownUserDetailsService' não estiver definido no modelo Spring, "Não é possível resolver o bean 'unknownUserDetailsService'" será relatado
    
  • @WithUserDetails(value = "admin", userDetailsServiceBeanName = "jpaUserDetailsService")
      // "O bean deve ser do tipo 'org.springframework.security.core.userdetails.UserDetailsService'"
      // será relatado se o bean "jpaUserDetailsService" não for do tipo 'UserDetailsService'
