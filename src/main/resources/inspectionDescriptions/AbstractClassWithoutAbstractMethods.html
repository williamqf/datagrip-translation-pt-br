Relata classes abstract que não possuem nenhum método abstract. Na maioria dos casos, uma classe abstract sem nenhum método abstract não faz sentido, e o modificador abstract pode ser removido da classe. Se a classe foi declarada como abstract para evitar a instanciação, o uso de um construtor private geralmente é uma escolha melhor para essa finalidade.

Exemplo:


  abstract class Example {
    public String getName() {
      return "IntelliJ IDEA";
    }
  }

Use esta opção para ignorar classes utilitárias.
