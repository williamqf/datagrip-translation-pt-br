Relata colunas na cláusula USING que não existem em ambas as tabelas.

Exemplo (MySQL):

CREATE TABLE t1 (i INT, j INT);
CREATE TABLE t2 (k INT, l INT);
SELECT * FROM t1 JOIN t2 USING (j);

Na cláusula USING, o nome da coluna deve existir em ambas as tabelas, e a consulta SELECT unirá automaticamente essas tabelas usando o nome da coluna fornecido. Como não temos a coluna j em t2, podemos reescrever a consulta usando ON. A cláusula ON pode unir tabelas mesmo quando os nomes das colunas não coincidem em ambas as tabelas.

SELECT * FROM t1 JOIN t2 ON t1.j = t2.l;
