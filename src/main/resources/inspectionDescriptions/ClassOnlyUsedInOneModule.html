Relata classes que atendem aos seguintes critérios:
  • Não dependem de nenhuma outra classe em seu módulo
  • Dependem de classes de um módulo diferente
  • São dependências apenas de classes desse outro módulo

Essas classes podem ser movidas para o módulo do qual dependem.
