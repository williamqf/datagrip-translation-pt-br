Relata chamadas a == ou equals() que comparam dois números java.math.BigDecimal. Isso geralmente é um erro, pois dois números java.math.BigDecimal só são iguais se o valor e a escala forem iguais.

Exemplo:


  // a condição é false
  val condition = BigDecimal("2.0") ==
      BigDecimal("2.00")

Após aplicar a correção rápida:


  // a condição é true
  val condition = BigDecimal("2.0").compareTo(
      BigDecimal("2.00")) == 0
