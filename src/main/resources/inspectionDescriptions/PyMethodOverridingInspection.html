Relata inconsistências em assinaturas de métodos sobrescritos.

Exemplo:


class Book:
    def add_title(self):
        pass


class Novel(Book):
    def add_title(self, text):
        pass

O parâmetro do método add_title na classe Novel não corresponde à assinatura do método especificada na classe Book. O IDE propõe aplicar a refatoração "Alterar Assinatura" como uma correção.
