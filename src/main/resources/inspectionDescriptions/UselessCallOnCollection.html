Reporta chamadas filter... da biblioteca padrão em coleções que já foram filtradas.

Várias funções na biblioteca padrão, como filterNotNull() ou filterIsInstance, só fazem sentido quando chamadas em um receiver com um tipo diferente do tipo do resultado. Caso contrário, tais chamadas podem ser omitidas, pois o resultado será o mesmo.

A correção rápida Remover chamada redundante pode ser usada para corrigir o código automaticamente.

Exemplo:


  fun test(list: List<String>) {
      val x = list.filterNotNull() // a correção rápida simplificará para 'list'
      val y = list.filterIsInstance<String>() // a correção rápida simplificará para 'list'
  }
