Relata casos em que equals() é chamado para comparar uma String com uma string vazia. Nesse caso, usar .isEmpty() é melhor porque mostra exatamente o que está sendo verificado.

Exemplo:


  void checkString(String s){
    if ("".equals(s)) throw new IllegalArgumentException();
  }

Após a aplicação da correção rápida:


  void checkString(String s){
    if (s != null && s.isEmpty()) throw new IllegalArgumentException();
  }

"".equals(str) retorna false quando str é null. Por segurança, a correção rápida desta inspeção insere uma verificação de null explícita quando o argumento de equals() puder ser null. Use a opção para fazer com que a inspeção ignore esses casos.
