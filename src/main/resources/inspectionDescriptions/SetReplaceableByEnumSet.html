Relata a instanciação de objetos java.util.Set cujo tipo de conteúdo é uma classe enum. Tais objetos Set podem ser substituídos por objetos java.util.EnumSet.

As implementações de EnumSet são significativamente mais eficientes do que outros conjuntos, pois a estrutura de dados subjacente é um vetor de bits. Use a correção rápida para substituir o inicializador por uma chamada a EnumSet.noneOf(). Esta correção rápida não está disponível quando o tipo da variável é uma subclasse de Set.

Exemplo:


  enum MyEnum { FOO, BAR; }

  Set<MyEnum> enums = new HashSet<MyEnum>();

Após a aplicação da correção rápida:


  enum MyEnum { FOO, BAR; }

  Set<MyEnum> enums = EnumSet.noneOf(MyEnum.class);
