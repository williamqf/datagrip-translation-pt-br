Relata as ProcessCanceledException que estão sendo tratadas de forma incorreta.

ProcessCanceledException e seus sucessores não devem ser capturados, engolidos, registrados ou tratados de forma alguma. Em vez disso, eles devem ser lançados novamente para que a infraestrutura possa tratá-los corretamente.

A inspeção relata as capturas explícitas de ProcessCanceledException ou seus sucessores, bem como as capturas de RuntimeException, Exception e Throwable que cobrem ProcessCanceledException.

Exemplos:


// incorreto:
try {
  // ...
} catch (ProcessCanceledException e) { // A exceção não deve ser engolida
}

// incorreto:
try {
  // ...
} catch (ProcessCanceledException e) {
  LOG.error("Error occurred", e); // A exceção não deve ser registrada
  throw e;
}


// correto:
try {
  // ...
} catch (ProcessCanceledException e) {
  // Outras ações
  throw e;
}
