Relata sobrecargas de método inválidas (...). Uma sobrecarga de método só é válida quando há exatamente uma outra definição não sobrecarregada para o método.

Uma correção rápida pode ser usada para remover a sobrecarga de método inválida.

Exemplo:

class Example
  # Não foi possível encontrar a declaração não sobrecarregada para o método 'call' da classe 'Example'
  def call: -> untyped | ...
end

Após a aplicação da correção rápida:

class Example
  def call: -> untyped
end
