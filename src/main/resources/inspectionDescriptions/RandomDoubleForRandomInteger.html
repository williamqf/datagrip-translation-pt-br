Relata chamadas a java.util.Random.nextDouble() que são usadas para criar números inteiros positivos multiplicando o resultado por um fator e convertendo-o para um número inteiro.

Para gerar números inteiros aleatórios dentro de um intervalo, java.util.Random.nextInt(int) é mais simples e eficiente.

Exemplo:


  int getRandomInt() {
    return (int) ((new Random()).nextDouble() * SIZE);
  }

Após a aplicação da correção rápida:


  int getRandomInt() {
    return (new Random()).nextInt(SIZE);
  }
