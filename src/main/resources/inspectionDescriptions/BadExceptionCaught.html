Reporta cláusulas catch que capturam exceções inadequadas.

Algumas exceções (por exemplo, java.lang.NullPointerException ou java.lang.IllegalMonitorStateException) indicam erros de programação e, portanto, quase certamente não devem ser capturadas em código de produção.

Exemplo:


  try {
    return component.getMousePosition(true) != null;
  } catch (NullPointerException e) { // aviso: exceção proibida 'NullPointerException' capturada
    return false;
  }

Use a lista de Exceções proibidas para especificar quais exceções devem ser reportadas.
