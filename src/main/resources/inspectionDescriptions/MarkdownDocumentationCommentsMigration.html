Relata comentários Javadoc que podem ser convertidos em comentários de documentação Markdown.

Exemplo:


  /**
   * For blubbering of the florz.
   *
   * Similar to {@link com.deeps.Querp querping} class.
   */
  class FlorzBlubber {
  }

Após aplicar a correção rápida:


  /// For blubbering of the florz.
  /// Similar to [querping][com.deeps.Querp] class.
  class FlorzBlubber {
  }
