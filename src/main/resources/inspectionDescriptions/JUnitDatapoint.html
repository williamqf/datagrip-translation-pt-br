Denuncia campos ou métodos que usam a anotação @DataPoint e não são public e/ou static. Classes de teste de teorias com membros @DataPoint com formatação incorreta não podem ser executadas.

Exemplo:


  @RunWith(Theories.class)
  public class SeriousTest {
    @DataPoint
    private String dataPoint = "value";

    @DataPoint("generated")
    private String generatedDataPoint() {
      return "generated value";
    }

    @Theory
    public void theoryMethod(String param) {
      // ...
    }
  }

Uma correção rápida está disponível para corrigir os modificadores:


  @RunWith(Theories.class)
  public class SeriousTest {
    @DataPoint
    public static String dataPoint = "value";

    @DataPoint("generated")
    public static String generatedDataPoint() {
      return "generated value";
    }

    @Theory
    public void theoryMethod(String param) {
      // ...
    }
  }
