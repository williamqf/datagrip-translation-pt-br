Relata chamadas para a função da biblioteca padrão do Kotlin filterIsInstance com argumentos de literal de classe. É mais idiomático usar a versão desta função com um parâmetro de tipo reificado para evitar o uso da sintaxe ::class.java.

Nota: literais de classe genérica não são relatados, pois a sintaxe Class<*, *> na lista de argumentos de tipo pode ser indesejável.

Exemplo:


  fun foo(list: List<*>) {
      list.filterIsInstance(Int::class.java)
  }

Após a aplicação da correção rápida:


  fun foo(list: List<*>) {
      list.filterIsInstance<Int>()
  }
