Relata APIs que podem ser definidas como DumbAware.

Se uma implementação não acessar índices, ela poderá ser marcada para funcionar durante o modo dumb.

Para mais detalhes, consulte a API DumbAware.

Adiciona classes que foram verificadas como não adequadas à "Lista de classes ignoradas". Essas classes serão ignoradas, evitando a adição de comentários ou anotações de supressão. Uma correção rápida pode ser usada para adicionar as classes relatadas a esta lista.
