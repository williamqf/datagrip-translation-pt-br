Relata instruções throw que lançam exceções inadequadas. Por exemplo, uma exceção pode ser inadequada por ser genérica demais, como java.lang.Exception ou java.io.IOException.

Exemplo:


  void setup(Mode mode) {
      if (mode == null)
          throw new RuntimeException("Problem during setup"); // Aviso: exceção proibida 'RuntimeException' lançada
      ...
  }

Use a lista de Exceções proibidas para especificar quais exceções devem ser relatadas.
