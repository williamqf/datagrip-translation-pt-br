Relata chamadas a Throwable.initCause() em que o construtor da exceção também aceita um argumento Throwable cause.

Nesse caso, a chamada a initCause() pode ser removida e seu argumento adicionado à chamada do construtor da exceção.

Exemplo:


  try {
      process();
  }
  catch (RuntimeException ex) {
    RuntimeException wrapper = new RuntimeException("Error while processing");
    wrapper.initCause(ex); // Chamada desnecessária a 'Throwable.initCause()'
    throw wrapper;
  }

Uma correção rápida pode ser usada para passar o argumento da causa para o construtor. Após aplicar a correção rápida:


  try {
      process();
  }
  catch (RuntimeException ex) {
    RuntimeException wrapper = new RuntimeException("Error while processing", ex);
    throw wrapper;
  }
