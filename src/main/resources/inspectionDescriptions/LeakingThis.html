Relata operações inseguras usando this durante a construção do objeto, incluindo:
  • Acesso a propriedades não final durante a inicialização da classe: a partir de um construtor ou da inicialização de propriedades
  • Chamada de funções não final durante a inicialização da classe
  • Uso de this como um argumento de função em construtores de classes não final

Se outras classes herdarem da classe fornecida, elas podem não estar totalmente inicializadas no momento em que a operação insegura for executada.

Exemplo:


  abstract class Base {
      val code = calculate()
      abstract fun calculate(): Int
  }

  class Derived(private val x: Int) : Base() {
      override fun calculate() = x
  }

  fun testIt() {
      println(Derived(42).code) // Esperado: 42, Atual: 0
  }
