Relata expressões de tabela comuns (CTE) não utilizadas em consultas.

Exemplo (PostgreSQL):

CREATE TABLE foo(a INT);

WITH a AS (SELECT 1 AS x FROM foo)
SELECT 1 + 2 FROM foo;

Ao usar WITH, criamos um conjunto de resultados nomeado temporariamente chamado a, também conhecido como uma Expressão de Tabela Comum (CTE). No entanto, não usamos esta CTE posteriormente no código. CTEs não utilizadas são exibidas em cinza.
