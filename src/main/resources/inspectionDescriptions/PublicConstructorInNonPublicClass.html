Relata todos os construtores public em classes que não são public.

Normalmente, não há motivo para criar um construtor public em uma classe com um nível de acesso inferior. No entanto, observe que esta inspeção altera o comportamento de algumas chamadas de reflexão. Em particular, Class.getConstructor() não conseguirá encontrar o construtor atualizado (em seu lugar, deve-se usar Class.getDeclaredConstructor()). Se o seu código ou o código de algum framework utilizado depender da acessibilidade do construtor via getConstructor(), não utilize esta inspeção.

Exemplo:


  class House {
    public House() {}
  }

Após a aplicação da correção rápida (quick fix):


  class House {
    House() {}
  }
