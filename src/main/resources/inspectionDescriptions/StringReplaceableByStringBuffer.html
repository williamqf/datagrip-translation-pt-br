Relata variáveis declaradas como java.lang.String que são concatenadas repetidamente. Tais variáveis podem ser declaradas de forma mais eficiente como java.lang.StringBuffer ou java.lang.StringBuilder.

Exemplo:


  String s = "";
  for (int i = 0; i < names.length; i++) {
    String name = names[i] + (i == names.length - 1 ? "" : " ");
    s = s + name;
  }

Tais loops podem ser substituídos por:


  StringBuilder s = new StringBuilder();
  for (int i = 0; i < names.length; i++) {
    String name = names[i] + (i == names.length - 1 ? "" : " ");
    s.append(name);
  }

Ou até mesmo por:


  String s = String.join(" ", names);

Use esta opção para que esta inspeção relate apenas quando a variável for concatenada dentro de um loop.
