Relata expressões if que possuem condições literais constantes true ou false e que podem ser simplificadas.

Embora ocasionalmente seja intencional, essa estrutura é confusa e geralmente é resultado de erros de digitação ou de refatorações anteriores.

Exemplo:


  fun example() {
      if (true) {
          throw IllegalStateException("Unexpected state")
      }
  }

A correção rápida remove a condição if:


  fun example() {
      throw IllegalStateException("Unexpected state")
  }
