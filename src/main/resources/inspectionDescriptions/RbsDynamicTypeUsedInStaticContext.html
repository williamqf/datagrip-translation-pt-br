Relatar o uso de tipos dinâmicos (instance, class e self) em contextos estáticos. O uso de tipos dinâmicos em contextos estáticos pode ser confuso, pois esse tipo nunca muda.

É possível usar uma correção rápida para substituir o uso de tipos dinâmicos pelo equivalente estático.

Exemplo:

class Exemplo
  # Uso de 'instance' em contexto estático
  INSTANCE: instance
  # Uso de 'class' em contexto estático
  CLASSE: class
end

Após aplicar a correção rápida:

class Exemplo
  INSTANCE: Exemplo
  CLASSE: singleton(Exemplo)
end
