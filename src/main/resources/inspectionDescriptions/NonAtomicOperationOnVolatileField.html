Relata operações não atômicas em campos volatile.

Um exemplo de operação não atômica é a atualização de um campo usando um operador de incremento. Como a operação envolve leitura e escrita, e outras modificações podem ocorrer entre elas, os dados podem ser corrompidos. A operação pode ser tornada atômica ao envolvê-la em um bloco synchronized ou ao usar uma das classes do pacote java.util.concurrent.atomic.

Exemplo:


  private volatile int v = 1;

  void foo() {
    v = 2 * v;
  }
