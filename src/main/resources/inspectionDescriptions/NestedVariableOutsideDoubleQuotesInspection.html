Detecta instâncias em arquivos `.env` onde variáveis aninhadas são usadas fora de aspas duplas e podem não ser analisadas corretamente. Esta inspeção garante que as variáveis aninhadas sejam adequadamente delimitadas para evitar problemas de configuração.

Esta inspeção ajuda a manter a sintaxe correta em arquivos `.env` ao exigir o uso de aspas duplas para variáveis aninhadas, evitando comportamentos inesperados durante a avaliação das variáveis.

Correção rápida: Esta inspeção oferece uma correção rápida para envolver automaticamente as variáveis aninhadas com aspas duplas. Clicar na correção rápida fornecida resolve o problema automaticamente.
