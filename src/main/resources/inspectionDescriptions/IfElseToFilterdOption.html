Inspeciona uma instrução if que avalia Option e sugere a substituição por filter.

Exemplo:


  def condition(x: Int): Boolean = ???
  val value = 3
  if (condition(value)) Some(value) else None

Após a aplicação do reparo rápido:


  def condition(x: Int): Boolean = ???
  val value = 3
  Some(value).filter(condition)
