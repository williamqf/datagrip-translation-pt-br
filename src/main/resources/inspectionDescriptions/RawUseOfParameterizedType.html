Relata classes genéricas onde os parâmetros de tipo foram omitidos. Esse uso bruto (raw) de tipos genéricos, embora válido em Java, anula o propósito dos parâmetros de tipo e pode ocultar erros. Esta inspeção reflete o aviso rawtypes do javac.

Exemplo:


//Aviso: uso bruto da classe parametrizada 'List'
List list = new ArrayList<String>();
//Uma lista de strings foi criada, mas também aceita um número inteiro
list.add(1);


//Sem aviso, pois não é possível fornecer argumentos de tipo durante a criação de um array
IntFunction<List<?>[]> fun = List[]::new;

Configuração da inspeção:

  • Use a opção Ignorar construção de novos objetos para ignorar tipos brutos usados na construção de objetos.
  • Use a opção Ignorar conversões de tipo para ignorar tipos brutos usados em casts de tipo.
  • Use a opção Ignorar locais onde os parâmetros de tipo não compilam para ignorar casos em que os parâmetros de tipo resultariam em erro de compilação (por exemplo, ao criar arrays ou sobrescrever métodos de biblioteca).
  • Use a opção Ignorar tipos de parâmetros de métodos sobrescritos para ignorar parâmetros de tipo usados em parâmetros de métodos sobrescritos.
  • Use a opção Ignorar quando a correção rápida automática não estiver disponível para ignorar casos em que uma correção rápida não pode ser aplicada.
