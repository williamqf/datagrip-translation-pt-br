Relata chamadas a associate() e associateTo() que podem ser substituídas por associateBy() ou associateWith().

Ambas as funções aceitam funções transformadoras aplicadas aos elementos de uma sequência ou coleção determinada (como receptor). O par resultante é então usado para construir o Map final.

Considerando que o transformador faz referência a it, as chamadas a associate[To]() podem ser substituídas por associateBy() ou associateWith(), que oferecem melhor desempenho.

Exemplo:

  fun getKey(i: Int) = 1L
  fun getValue(i: Int) = 1L

  fun test() {
      arrayOf(1).associate { getKey(it) to it }  // 'associate()' substituível
      listOf(1).associate { it to getValue(it) } // 'associate()' substituível
  }

Após a aplicação do ajuste rápido:

  fun getKey(i: Int) = 1L
  fun getValue(i: Int) = 1L

  fun test() {
      arrayOf(1).associateBy { getKey(it) }
      listOf(1).associateWith { getValue(it) }
  }
