Relata o uso de UElement da UAST como PsiElement.

O PsiElement obtido desta forma é ambíguo.

Para obter o PsiElement "físico", use UElementKt.getSourcePsiElement(). Para o PsiElement que "emula" o comportamento de elementos Java (PsiClass, PsiMethod, etc.), use UElementKt.getAsJavaPsiElement().

Consulte UAST - Unified Abstract Syntax Tree na documentação do SDK.
