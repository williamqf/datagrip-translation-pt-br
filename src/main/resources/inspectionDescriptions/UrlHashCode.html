Relata chamadas a hashCode() e equals() em objetos java.net.URL, bem como chamadas que adicionam objetos URL a mapas e conjuntos.

Os métodos equals() e hashCode() de URL podem realizar uma consulta DNS para resolver o nome do host. Isso pode causar atrasos significativos, dependendo da disponibilidade e velocidade da rede e do servidor DNS. O uso de java.net.URI em vez de java.net.URL evitará a consulta DNS.

Exemplo:


  boolean urlEquals(URL url1, URL url2) {
      return url1.equals(url2);
  }
