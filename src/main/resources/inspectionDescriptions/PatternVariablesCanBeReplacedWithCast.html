Relata o instanceof com padrões e sugere convertê-lo em um instanceof comum com cast.

Ao aplicar uma correção rápida, esta inspeção permite mover o instanceof com padrões para uma base de código que utiliza versões anteriores do Java.

Observe que, ao usar expressões complexas antes do instanceof, o resultado pode não ser totalmente equivalente ao instanceof original com padrões. Nesse caso, a expressão será reavaliada.

Exemplo:


  if (object instanceof String txt && txt.length() == 1) {
      System.out.println(txt);
  } else {
      return;
  }
  System.out.println(txt);

Após a aplicação da correção rápida:


  if (object instanceof String && ((String) object).length() ==1) {
      String txt = (String) object;
      System.out.println(txt);
  } else {
      return;
  }
  String txt = (String) object;
  System.out.println(txt);
