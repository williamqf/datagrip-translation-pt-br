Relata combinações de verificações de null e de vazio que podem ser simplificadas em uma única verificação.

A correção rápida substituirá as verificações destacadas por uma chamada de verificação combinada, como isNullOrEmpty().

Exemplo:


  fun test(list: List<Int>?) {
      if (list == null || list.isEmpty()) {
          println("List is empty!")
      } else {
          println(list.joinToString())
      }
  }

Após a aplicação da correção rápida:


  fun test(list: List<Int>?) {
      if (list.isNullOrEmpty()) {
          println("List is empty!")
      } else {
          println(list.joinToString())
      }
  }
