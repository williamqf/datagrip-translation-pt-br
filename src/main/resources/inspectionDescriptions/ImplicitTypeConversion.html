Relata conversões implícitas entre os tipos XPath predefinidos STRING, NUMBER, BOOLEAN e NODESET. Ajuda a escrever scripts XSLT com representação de tipos mais explícita e a evitar erros difíceis de detectar.

Exemplo:


  <xsl:if test="foo" />
não é o mesmo que 

  <xsl:if test="string(foo)" />

O primeiro teste verifica se o elemento "foo" existe (count(foo) > 0), enquanto o segundo teste será "true" apenas se o elemento contiver algum texto (string-length(foo) > 0). Recomenda-se tornar a conversão de tipo mais explícita.

Configure a inspeção usando as seguintes opções:

  • Ativar ou desativar conversões implícitas entre certos tipos
  • Sempre relatar conversões explícitas que não resultam no tipo esperado real, por exemplo, <xsl:if test="number(foo)" />
  • Ignorar a conversão de NODESET para BOOLEAN causada pelo uso da função string() como um atalho para escrever string-length() > 0.
