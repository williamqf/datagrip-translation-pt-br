Reporta chamadas de funções e métodos que não tratam o resultado do tipo error.

A API de tais funções implica que a execução pode falhar e retornar um erro. Chamadas que não tratam o resultado de erro podem indicar que a API está sendo usada incorretamente.

Exemplo:


os.Remove("non-existing") // O erro é ignorado

Após aplicar a correção rápida Tratar erro:

err := os.Remove("non-existing") // O erro é tratado
if err != nil {
  return err
}
