Relata pacotes que contêm menos classes do que o valor mínimo especificado.

Pacotes que contêm subpacotes não são relatados. Pacotes excessivamente pequenos podem indicar um design fragmentado.

Disponível apenas em Código | Inspecionar código ou Código | Analisar código | Executar inspeção por nome, e não é relatado no editor.

Use o campo Número mínimo de classes para especificar o número mínimo permitido de classes em um pacote.
