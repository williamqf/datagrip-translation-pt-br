Relata o uso de Process.waitFor() e Process.onExit() em um contexto de corrotina. O uso desses métodos é proibido em contextos de corrotina.

Use com.intellij.util.io.awaitExit() em seu lugar.

Exemplo:


suspend fun doSomething(process: Process) {
  val exitCode = process.waitFor() // Incorreto
  // ...
}

suspend fun doSomething(process: Process) {
  val exitCode = process.awaitExit() // Correto
  // ...
}
