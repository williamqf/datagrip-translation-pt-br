Relata erros em literais de data e hora. Esta inspeção está disponível no MySQL, Oracle, Db2 e H2.

Exemplo (MySQL):

SELECT TIME '10 -12:13:14' FROM dual;
SELECT TIME ' 12 : 13 : 14 ' FROM dual;
SELECT TIME '12 13 14' FROM dual;
SELECT TIME '12-13-14' FROM dual;
SELECT TIME '12.13.14' FROM dual;
SELECT TIME '12:13:' FROM dual;
SELECT TIME '12:13' FROM dual;
SELECT TIME '12:' FROM dual;

Neste exemplo, as datas ignoram os padrões de literais de data e hora do MySQL. Portanto, elas serão destacadas. Para obter detalhes sobre literais de data e hora no MySQL, consulte Date and Time Literals em dev.mysql.com.

Os seguintes literais de data e hora são válidos para o MySQL.

SELECT TIME '12:13:14' FROM dual;
SELECT TIME '12:13:14.555' FROM dual;
SELECT TIME '12:13:14.' FROM dual;
SELECT TIME '-12:13:14' FROM dual;
SELECT TIME '10 12:13:14' FROM dual;
SELECT TIME '-10 12:13:14' FROM dual;
