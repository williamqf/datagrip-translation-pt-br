Reporta inicializadores de instância que podem ser tornados static sem produzir erros.

Um inicializador de instância pode ser static se não referenciar nenhum membro não static da sua classe. Os inicializadores static são executados quando a classe é resolvida, enquanto os inicializadores de instância são executados sempre que a classe é instanciada.

Esta inspeção não relata inicializadores vazios nem inicializadores em classes anônimas.

Exemplo:


  class A {
     public static String CONSTANT;
     {
        CONSTANT = "Hello";
     }
  }

Após a aplicação da correção rápida:


  class A {
     public static String CONSTANT;
     static {
        CONSTANT = "Hello"; // Agora inicializado apenas uma vez por classe
     }
  }
