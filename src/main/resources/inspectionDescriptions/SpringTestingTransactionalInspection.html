Relata declarações de métodos de ciclo de vida de transação inválidos anotados com @BeforeTransaction e @AfterTransaction em classes de teste anotadas com @Transactional.

Os métodos anotados não devem ter argumentos nem um tipo de retorno.

Exemplo:


  @ContextConfiguration
  @Transactional
  public class AbstractShowcaseTest {
    @BeforeTransaction // O tipo de retorno do método deve ser 'void'
    public boolean setupData() {...}

    @AfterTransaction // Número incorreto de argumentos
    public void disposeData(boolean a) throws Exception {...}
  }
