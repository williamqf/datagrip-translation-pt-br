Relata o uso de true | false que pode ser substituído por bool. A palavra-chave bool é uma forma abreviada que ajuda a melhorar a legibilidade.

Uma correção rápida pode ser usada para converter qualquer combinação de true | false em bool.

Exemplo:

# Pode simplificar 'true | false'
type example = true | false

Após aplicar a correção rápida:

type example = bool
