Esta inspeção identifica chamadas de método onde um argumento que poderia ser null é passado a um parâmetro que não é anotado com @Nullable, ajudando a detectar potenciais NullPointerExceptions antes que ocorram em tempo de execução.

Esta inspeção é mostrada apenas para os métodos direcionados que estão no código-fonte do projeto.

Exemplo:


def test(@Nullable greeting: String): Unit = {
  sayHello(greeting)
}

def sayHello(greeting: String): Unit = {
  println(greeting.toLowerCase)
}
