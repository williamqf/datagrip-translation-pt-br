Relata instruções assert e asserções de frameworks de teste suprimidas por blocos catch envolventes. Tais asserções nunca falharão, pois o AssertionError lançado será capturado e ignorado silenciosamente.

Exemplo 1:


  void javaAssertion() {
    try {
      ...
      assert 1 == 2;
    } catch (AssertionError e) {
      // a asserção é ignorada silenciosamente
    }
  }

Exemplo 2:


  @Test
  void testWithAssertJ() {
    try {
      ...
      assertThat(1).as("test").isEqualTo(2);
    } catch (AssertionError e) {
      // a asserção é ignorada silenciosamente
    }
  }

Exemplo 3:


  @Test
  void testWithJunit() {
    try {
      ...
      assertEquals(1, 2);
    } catch (AssertionError e) {
      // a asserção é ignorada silenciosamente
    }
  }
