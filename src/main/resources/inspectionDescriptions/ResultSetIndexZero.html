Relata tentativas de acesso à coluna 0 de um java.sql.ResultSet ou java.sql.PreparedStatement. Por razões históricas, as colunas de java.sql.ResultSet e java.sql.PreparedStatement são numeradas começando em 1, e não em 0, e o acesso à coluna 0 é um erro comum na programação JDBC.

Exemplo:


  String getName(ResultSet rs) {
    return rs.getString(0);
  }
