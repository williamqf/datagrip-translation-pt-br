Relata variáveis globais que foram definidas, mas nunca utilizadas no código.

Se houver variáveis não utilizadas, o código não será compilado. Para obter detalhes sobre variáveis e importações não utilizadas, consulte Unused imports and variables em go.dev. 

func main() {
 a := 422
}

O código no exemplo não será compilado. Portanto, ele será destacado como um erro. Para esses casos, você pode aplicar duas correções rápidas: Excluir variável e Renomear para _. A primeira correção rápida exclui a variável e a segunda converte a variável em um identificador em branco.

Após aplicar a correção rápida Renomear para _:

func main() {
 _ := 422
}
