Relata strings de formatação que não estão em conformidade com a sintaxe Java padrão.

Por padrão, esta inspeção considera constantes de tempo de compilação como strings de formatação se forem usadas como argumentos para os métodos correspondentes em java.util.Formatter, java.lang.String, java.io.PrintWriter ou java.io.PrintStream.

Exemplo:


  String.format("x = %d, y = %d", 42);

Use as configurações da inspeção para marcar classes e métodos adicionais como relacionados à formatação de strings.

Como alternativa, você pode usar a anotação org.intellij.lang.annotations.PrintFormat para marcar parâmetros de método de string de formatação. Nesse caso, o parâmetro de argumentos de formatação deve seguir imediatamente a string de formatação e deve ser o último parâmetro do método. Exemplo:


  void myFormatMethod(int mode, @PrintFormat String formatString, Object... args) {...}

Métodos anotados desta forma também serão reconhecidos por esta inspeção.
