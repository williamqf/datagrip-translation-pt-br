Relata classes aninhadas public.

Exemplo:


  public class Outer {
    public static class Nested {} // Aviso
    public class Inner {}         // Aviso
    public enum Mode {}           // O aviso depende das configurações
    public interface I {}         // O aviso depende das configurações
  }

Configuração da inspeção:

  • Use a opção Ignorar enums internos 'public' para ignorar enums internos public.
  • Use a opção Ignorar interfaces internas 'public' para ignorar interfaces internas public.
