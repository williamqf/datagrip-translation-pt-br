Reporta o uso de kotlin.coroutines.suspendCoroutine e sugere substituí-lo por kotlinx.coroutines.suspendCancellableCoroutine quando a dependência kotlinx.coroutines estiver presente.

A variante suspendCancellableCoroutine fornece suporte integrado para cancelamento, o que é importante para a concorrência estruturada e o gerenciamento adequado de recursos.

Exemplo:


    suspend fun usage(): String {
        return suspendCoroutine { continuation ->
            continuation.resume("Hello!")
        }
    }

Após a aplicação da correção rápida:


    suspend fun usage(): String {
        return suspendCancellableCoroutine { continuation ->
            continuation.resume("Hello!")
        }
    }
