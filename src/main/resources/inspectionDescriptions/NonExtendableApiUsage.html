Relata classes, interfaces e métodos que estendem, implementam ou sobrescrevem elementos de API marcados com @ApiStatus.NonExtendable.

A anotação @ApiStatus.NonExtendable indica que uma classe, interface ou método não deve ser estendido, implementado ou sobrescrito. É comum realizar o cast de tais interfaces e classes para implementações internas da biblioteca; portanto, se o cliente fornecer uma implementação diferente, uma ClassCastException será lançada. Adicionar novos métodos abstratos a essas classes e interfaces quebrará a compatibilidade com as implementações do cliente.

Exemplo:


  // No código da biblioteca upstream
  @ApiStatus.NonExtendable
  public interface MyService {
    public void perform();
  }

  // No código do cliente downstream
  public class MyServiceImpl implements MyService {
    @Override
    public void perform() {
      // ...
    }
  }
Esta inspeção também detecta os seguintes problemas relacionados à aplicação incorreta da anotação:
  • Anotar uma classe/método que não pode ser estendido/sobrescrito
  • Anotar redundantemente métodos em uma classe que já possui a anotação @ApiStatus.NonExtendable
