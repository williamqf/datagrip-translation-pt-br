Relata classes, interfaces e objetos em projetos Kotlin/JS que estendem tipos React Props ou State, mas não estão marcados como external.

Os tipos de componentes React devem ser interfaces externas para interoperar corretamente com o runtime JavaScript.

Exemplo:

interface MyProps : Props { // Aviso: deve ser uma interface external
    var title: String
}

class MyState : State { // Aviso: deve ser uma interface external
    var count: Int = 0
}

Após a aplicação da correção rápida:

external interface MyProps : Props {
    var title: String
}
