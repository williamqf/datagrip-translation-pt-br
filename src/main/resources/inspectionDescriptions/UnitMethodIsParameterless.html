Relata métodos com tipos de resultado Unit que não têm cláusulas de parâmetros.

Métodos com tipos de resultado Unit são somente executados por seus efeitos colaterais e, por convenção, métodos com efeitos colaterais têm pelo menos uma cláusula de parâmetros (vazia).

O reparo rápido adiciona uma cláusula de parâmetros vazia.

Exemplo:


  def test: Unit = ()

Após o reparo rápido ser aplicado:


  def test(): Unit = ()
