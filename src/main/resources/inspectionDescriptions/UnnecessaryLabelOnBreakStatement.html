Relata instruções break com rótulos desnecessários. Tais rótulos não alteram o fluxo de controle, mas dificultam o acompanhamento do código.

Exemplo:


  label:
  for(int i = 0; i < 10; i++) {
    if (shouldBreak()) break label;
    // faz algo
  }

Após a aplicação da correção rápida:


  label:
  for(int i = 0; i < 10; i++) {
    if (shouldBreak()) break;
    // faz algo
  }
