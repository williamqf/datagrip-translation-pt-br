Relata quaisquer variáveis e parâmetros usados em condições de loop que não são atualizados dentro do loop.

Tais variáveis e parâmetros são geralmente usados incorretamente, pois, se executados, podem causar um loop infinito.

Exemplo:


  void loopDoesNotLoop(boolean b) {
    while (b) {
      System.out.println();
      break;
    }
  }

Configuração da inspeção:

Se a condição puder ser atualizada indiretamente (por exemplo, por meio de um método chamado ou atualizada simultaneamente a partir de outra thread), use a opção Ignorar possíveis alterações não locais para desativar esta inspeção.
