Reporta métodos que referenciam muitas outras classes. Métodos com acoplamento excessivo podem ser muito frágeis e devem ser divididos em métodos menores.

Cada classe referenciada é contada apenas uma vez, independentemente de quantas vezes seja referenciada.

Configuração da inspeção:

  • Use o campo Limite de acoplamento do método para especificar o acoplamento máximo permitido para um método.
  • Use a opção Incluir acoplamento a classes do sistema Java para contar referências a classes nos pacotes java ou javax.
  • Use a opção Incluir acoplamento a classes de biblioteca para contar referências a classes de bibliotecas de terceiros.
