Relata modificações de variáveis com atribuições simples (por exemplo, y = y + x) que podem ser substituídas por uma atribuição com operador.

A correção rápida substituirá a atribuição pelo operador de atribuição correspondente.

Exemplo:


  fun foo() {
      val list = mutableListOf(1, 2, 3)
      list = list + 4
  }

Após aplicar a correção rápida:


  fun foo() {
      val list = mutableListOf(1, 2, 3)
      list += 4
  }
