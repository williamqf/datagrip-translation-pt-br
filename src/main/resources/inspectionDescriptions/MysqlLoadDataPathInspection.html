Relata caminhos que começam com o caractere til em instruções LOAD.

Exemplo (MySQL):

CREATE TABLE table_name (id int);
LOAD DATA LOCAL INFILE '~/Documents/some_file.txt'
INTO TABLE table_name FIELDS TERMINATED BY ',' LINES TERMINATED BY '\n'
IGNORE 1 LINES;

Use o caminho completo do arquivo em vez do caractere til.
