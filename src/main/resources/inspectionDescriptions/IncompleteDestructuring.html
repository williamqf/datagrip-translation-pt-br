Relata declarações de desestruturação incompletas.

Exemplo:


  data class Person(val name: String, val age: Int)
  val person = Person("", 0)
  val (name) = person

A correção rápida completa a declaração de desestruturação usando novas variáveis:


  data class Person(val name: String, val age: Int)
  val person = Person("", 0)
  val (name, age) = person
