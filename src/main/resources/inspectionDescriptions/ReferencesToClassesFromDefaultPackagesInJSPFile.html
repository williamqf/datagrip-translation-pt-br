Relata quaisquer referências a classes no pacote padrão em arquivos JSP.

Exemplo:


  <%!
  Integer countSomeUsefulStuff() {
    ...
    new ClassFromDefaultPackage(...); // Erro
    new com.example.GoodClass(...); // OK
  }
  %>
  <html>
    <head>
      <title>Hello, JSP!</title>
    </head>
  </html>

Nota: Tais referências não serão resolvidas durante a compilação do JSP, pois as classes compiladas do JSP geralmente são colocadas em pacotes que não são o padrão (por exemplo, "org.apache.jsp" para o Tomcat, "com.ibm._jsp" para o WebSphere), e a importação a partir do pacote padrão não é permitida em Java.
