Reporta métodos com três ou mais negações. Tais métodos podem ser confusos.

Exemplo:


  void doSmth(int a, int b, boolean flag1, boolean flag2) {
    if (!flag && !flag2) {
      if (a != b) {
        doOther();
      }
    }
  }

Sem as negações, o método torna-se mais fácil de entender:


  void doSmth(int a, int b, boolean flag1, boolean flag2) {
    if (flag1 || flag2 || a == b) return;
    doOther();
  }

Configuração da inspeção:

  • Use a opção Ignorar negações em métodos 'equals()' para desativar a inspeção em métodos equals().
  • Use a opção Ignorar negações em instruções 'assert' para desativar a inspeção em instruções assert.
