Relatar condições is log enabled for em declarações if que não correspondem ao nível da chamada de log inclusa.

Exemplo: 


  if (LOG.isTraceEnabled()) {
    // Nível de depuração é registrado, mas nível de rastreio é verificado
    LOG.debug("some log message");
  }

Esta inspeção entende as estruturas de registro do java.util.logging, log4j, Log4j 2, Apache Commons Logging e SLF4J.
