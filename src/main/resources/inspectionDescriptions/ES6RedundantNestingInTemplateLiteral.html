Relata instâncias aninhadas de strings ou literais de modelo. Sugere a incorporação das instâncias aninhadas na string de modelo que as contém.

Exemplo: 


let a = `Hello, ${`Brave ${"New"}`} ${"World"}!`

Após a aplicação do ajuste rápido, o código ficará assim:


let a = `Hello, Brave New World!`
