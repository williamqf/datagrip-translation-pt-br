Relata chamadas da função to que podem ser substituídas pela forma infixa.

O uso da forma infixa pode tornar o código mais simples.

A correção rápida substitui o to pela forma infixa.

Exemplo:


  fun foo(a: Int, b: Int) {
      val pair = a.to(b)
  }

Após aplicar a correção rápida:


  fun foo(a: Int, b: Int) {
      val pair = a to b
  }
