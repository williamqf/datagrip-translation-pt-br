Relata referências de bean/método incorretas no parâmetro value das anotações @MockitoBean, @MockitoSpyBean e @TestBean.

Exemplo:


@SpringBootTest
class TestBeanApplicationTests {
   @TestBean(value = "unknownBean", methodName="createUserService")  // Relata 1. "Não foi possível resolver o bean 'unknownBean'" e 2. "Não foi possível resolver o símbolo 'createUserService'"
   UserService userService;

  @TestBean  // Relata "Método factory estático não encontrado"
  UserService userService;
}
