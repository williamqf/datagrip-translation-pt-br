Relata instruções ou expressões switch que contêm o mesmo código em diferentes ramos e sugere a mesclagem de ramos duplicados.

Exemplo:


  switch (n) {
    case 1:
      System.out.println(n);
      break;
    case 2:
      System.out.println(n);
      break;
    default:
      System.out.println("default");
  }

Após a aplicação da correção rápida:


  switch (n) {
    case 1:
    case 2:
      System.out.println(n);
      break;
    default:
      System.out.println("default");
  }
