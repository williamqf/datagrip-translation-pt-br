Relata declarações marcadas com @ApiStatus.ScheduledForRemoval, mas sem @Deprecated.

Exemplo:


  @ApiStatus.ScheduledForRemoval(inVersion = "2017.3")
  public void myLegacyMethod() { }

Após a aplicação da correção rápida (quick fix), o resultado será o seguinte:


  @Deprecated
  @ApiStatus.ScheduledForRemoval(inVersion = "2017.3")
  public void myLegacyMethod() { }
