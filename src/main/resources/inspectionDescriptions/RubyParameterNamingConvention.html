Relata parâmetros cujos nomes são muito curtos, muito longos ou não correspondem ao padrão de expressão regular especificado.

Use o campo Padrão para especificar a expressão regular para validar os nomes dos parâmetros.

Use os campos Comprimento mínimo e Comprimento máximo para especificar o comprimento mínimo e máximo do nome, respectivamente.

Use as caixas de seleção Ignorar... para ignorar nomes de parâmetros de bloco, lambda ou rescue.
