Relata o uso da cláusula ORDER BY sem TOP, OFFSET ou FOR XML em views, funções inline, tabelas derivadas, subqueries e expressões de tabela comuns.

Para obter informações detalhadas sobre o uso de ORDER BY, consulte a cláusula SELECT - ORDER BY (Transact-SQL) em docs.microsoft.com.

Exemplo (Microsoft SQL Server):

CREATE TABLE foo (a INT NOT NULL, b  INT NOT NULL);

SELECT *
FROM (SELECT a, b
FROM foo A
WHERE a < 89
ORDER BY b) ALIAS;

Na subquery, o ORDER BY será realçado como um erro. Você pode adicionar TOP, OFFSET ou FOR XML à subquery. Como alternativa, use a correção rápida Remover elemento para excluir a seção ORDER BY.

Após aplicar a correção rápida:

SELECT *
FROM (SELECT a, b
FROM foo A
WHERE a < 89) ALIAS;
