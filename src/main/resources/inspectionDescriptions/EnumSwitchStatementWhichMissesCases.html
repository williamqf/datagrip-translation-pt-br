Relata instruções switch sobre tipos enum que não são exaustivas.

Exemplo:


  enum AlphaBetaGamma {
    A, B, C;

    void x(AlphaBetaGamma e) {
      switch (e) {

      }
    }
  }

Após a aplicação da correção rápida:


  enum AlphaBetaGamma {
    A, B, C;

    void x(AlphaBetaGamma e) {
      switch (e) {
        case A -> {}
        case B -> {}
        case C -> {}
      }
    }
  }

Use a opção Ignorar instruções 'switch' com ramo 'default' para ignorar instruções switch que possuam um ramo default.
