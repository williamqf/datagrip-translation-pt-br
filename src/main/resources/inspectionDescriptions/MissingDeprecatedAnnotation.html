Relata declarações de módulo, classes, campos ou métodos que possuem a tag Javadoc @deprecated, mas não possuem a anotação @java.lang.Deprecated.

Exemplo:


  /**
   * @deprecated use {@code example()} em seu lugar
   */
  void sample(){ }

Após a aplicação da correção rápida:


  /**
   * @deprecated use {@code example()} em seu lugar
   */
  @Deprecated
  void sample(){ }
