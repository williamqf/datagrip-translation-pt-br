Relata filter ou filterNot sobre Set e sugere usar intersect ou diff em seu lugar.

Exemplo:


  val set = Set(1, 2)
  val others = Set(2, 3)
  set.filter(others.contains(_))
  set.filterNot(others.contains)
  set.filter(x => !others.contains(x))
  set.filterNot(x => !others.contains(x))

Após aplicação de correção rápida:


  val set = Set(1, 2)
  val others = Set(2, 3)
  set.intersect(others)
  set.diff(others)
  set.diff(others)
  set.intersect(others)
