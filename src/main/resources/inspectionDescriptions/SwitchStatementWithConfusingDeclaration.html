Relata variáveis locais declaradas em uma ramificação de uma instrução switch e usadas em outra ramificação. Tais declarações podem ser especialmente confusas.

Exemplo:


    switch(i) {
      case 2:
          int x = 0;
          break;
      case 3:
          x = 3;
          System.out.println(x);
          break;
    }
