Relata entradas de declaração de desestruturação que correspondem a nomes de propriedades diferentes da data class desestruturada.

Exemplo:


  data class Foo(val a: String, val b: Int, val c: String)

  fun bar(f: Foo) {
      val (a, c) = f
  }

A correção rápida altera o nome da variável para corresponder ao nome do campo da classe correspondente:


  data class Foo(val a: String, val b: Int, val c: String)

  fun bar(f: Foo) {
      val (a, b) = f
  }
