Relata implementações do método clone(), chamadas a ele e o uso da interface java.lang.Cloneable.

Alguns padrões de codificação proíbem o uso de clone(), sugerindo o uso de um construtor de cópia ou um método factory static em seu lugar.

Esta inspeção ignora chamadas a clone() em arrays, pois essa é uma maneira correta e compacta de copiar arrays.

Exemplo:


  class Copy implements Cloneable /*aviso*/ {

    public Copy clone() /*aviso*/ {
      try {
        return (Copy) super.clone(); // aviso
      } catch (CloneNotSupportedException e) {
        throw new AssertionError();
      }
    }
  }
