Oferece inspeções gerais para anotações Lombok.

Os avisos e erros relatados por esta inspeção visam corresponder de 1:1 aos avisos e erros relatados pelo processador de anotações Lombok em tempo de compilação, tais como:

  • Anotação aplicada de forma redundante (relatada como um aviso)

    Exemplo:

    
      @ToString
      class Person {
        final String name;
        final int age;

        @Override
        public String toString() {
            return "Person{name='" + name + "', age=" + age + '}';
        }
      }
    A correção rápida sugere a remoção da anotação redundante.
  • Anotação aplicada incorretamente (relatada como um erro)

    Exemplo:

    
      @Value
      record Person(String name, int age) {}
    A correção rápida sugere a remoção da anotação inválida.
