Relata classes utilitárias que podem ser convertidas em um enum.

Algumas diretrizes de estilo de código exigem que as classes utilitárias sejam implementadas como enums para evitar problemas de cobertura de código em construtores private.

Exemplo:


  class StringUtils {
    public static final String EMPTY = "";
  }

Após a aplicação da correção rápida:


  enum StringUtils {
    ;
    public static final String EMPTY = "";
  }
