Relata o uso de APIs que não estão disponíveis no nível de linguagem configurado. Esta inspeção tem três funções:
  • Destaca o uso de classes generificadas quando o nível de linguagem é inferior ao Java 7.
  • Destaca quando um método padrão não é sobrescrito e o nível de linguagem é inferior ao Java 8.
  • Destaca o uso de APIs quando o nível de linguagem é inferior ao nível marcado com a tag @since na documentação.

Use a opção Proibir o uso da API para proibir o uso de APIs em relação ao projeto ou a níveis de linguagem personalizados.
