Usar uma função como .mkString ou .getLines diretamente no Source aberto por fromFile, fromURL ou fromURI vazará o identificador do arquivo subjacente.

Considere usar .close() com try/finally para gerenciar o período de vida do recurso ou use Using do Scala 2.13.

Exemplo:


  Source.fromFile("file.txt").getLines()
