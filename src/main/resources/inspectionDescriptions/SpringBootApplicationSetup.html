Relata @SpringBootApplication no pacote padrão e as anotações @EnableAutoConfiguration ou @ComponentScan redundantes.

A correção rápida remove as anotações redundantes.

Exemplo:


@SpringBootApplication
@ComponentScan // relata 'Declaração redundante: @SpringBootApplication já implica @ComponentScan'
public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}

Após a aplicação da correção rápida:


  @SpringBootApplication
  public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
  }
