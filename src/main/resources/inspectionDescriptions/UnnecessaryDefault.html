Reporta instruções ou expressões switch de enumeração nas quais a ramificação default nunca é alcançada, pois todos os valores possíveis já são cobertos pelas ramificações case.

Tais elementos são redundantes, especialmente para expressões switch, visto que estas não compilam se as ramificações case não cobrirem todas as constantes da enumeração.

O nível de linguagem deve ser configurado como 14 para que as expressões switch sejam reportadas.

A correção rápida fornecida remove a ramificação default.

Exemplo:


  enum E { A, B }
  int foo(E e) {
    return switch (e) {
      case A -> 1;
      case B -> 2;
      default -> 3;
    };
  }

Após a aplicação da correção rápida:


  enum E { A, B }
  int foo(E e) {
    return switch (e) {
      case A -> 1;
      case B -> 2;
    };
  }

Use a opção Reportar apenas expressões switch para reportar apenas ramificações default redundantes em expressões switch.
