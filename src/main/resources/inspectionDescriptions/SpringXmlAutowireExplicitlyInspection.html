Relata o atributo autowire no elemento <bean> e sugere a injeção explícita de propriedades do Bean sempre que possível.

Exemplo:


  public class MyComponent {
    public void setOtherBean(OtherBean bean){...}
  }


  <beans>
    <bean class="beans.OtherBean" id="bar"/>
    <bean autowire="byType" class="beans.MyComponent"/>
    <!-- "Dependência autowired desnecessária" -->
    <!-- Sugere "Tornar a dependência autowired explícita" -->
  </beans>

Após a aplicação da correção rápida:


  <beans>
    <bean class="beans.OtherBean" id="bar"/>
    <bean class="beans.MyComponent" id="foo">
      <property name="otherBean" ref="otherBean"/>
    </bean>
  </beans>
