Relata parâmetros de função não utilizados. 
func main() {
 printAll(
  42,
  "bird",
 )
}

func printAll(
 i int,
 s string,
) {
 fmt.Println(i)
}

Chamamos a função printAll, passando 42 e bird como argumentos. A função printAll aceita dois parâmetros, int e string, mas utiliza apenas o primeiro. Portanto, s string ficará esmaecido.
