Relata a instanciação de classes utilitárias usando a palavra-chave new.

Em classes utilitárias, todos os campos e métodos são static. A instanciação de tais classes é provavelmente desnecessária e indica um erro.

Exemplo:


  class MyUtils {
    public static double cube(double x) {
      return x * x * x;
    }
  }
  class Main {
    public static void main(String[] args) {
      // Instanciação de classe utilitária
      MyUtils utils = new MyUtils();
    }
  }

Para evitar que uma classe utilitária seja instanciada, recomenda-se o uso de um construtor private.
