Esta inspeção visualiza a aplicação das configurações de análise de taint destacando as chamadas de método.

As configurações de taint determinam quais métodos são considerados fontes (sources), receptores (sinks), higienizadores (sanitizers), pass-through ou regras de propagação personalizadas.

Para chamadas de método que referenciam métodos sem configurações de análise de taint definidas, a inspeção fornece uma correção rápida para adicionar a configuração apropriada.
