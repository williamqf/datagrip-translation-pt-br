Relata funções anônimas que podem ser expandidas pela eta e também relata espaços reservados que podem ser removidos de expansões pela eta.

Exemplo:


  def f(x: Any, y: Int): Unit = ()
  val x = f(_, _)
  val y: (Int, Int) => Unit = f _

Após a aplicação da correção rápida:


  def f(x: Any, y: Int): Unit = ()
  val x = f _
  val y: (Int, Int) => Unit = f
