Relata loops for que iteram sobre coleções usando a função withIndex(), mas sem utilizar a variável de índice.

Use a correção rápida "Remover o índice do loop 'for'" para limpar o código.

Exemplo:


  fun foo(bar: List<String>) {
     for ((index : Int, value: String) in bar.withIndex()) { // <== 'index' não é utilizado
         println(value)
     }
  }

Após aplicar a correção rápida:


  fun foo(bar: List<String>) {
      for (value: String in bar) { // <== '.withIndex()' e 'index' foram removidos
          println(value)
      }
  }
