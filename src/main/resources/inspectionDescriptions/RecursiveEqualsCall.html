Relata chamadas recursivas ao método equals(==).

No Kotlin, == compara valores de objetos chamando o método equals internamente. Por outro lado, === compara objetos por referência.

O operador === é comumente usado em implementações do método equals. No entanto, o === pode ser confundido erroneamente com ==, o que pode levar a uma recursão infinita.

Exemplo:


  class X {
      override fun equals(other: Any?): Boolean {
          if (this == other) return true
          return false
      }
  }

Após a aplicação da correção rápida:


  class X {
      override fun equals(other: Any?): Boolean {
          if (this === other) return true
          return false
      }
  }
