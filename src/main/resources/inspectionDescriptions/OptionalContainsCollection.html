Relata tipos java.util.Optional ou com.google.common.base.Optional que possuem parâmetros de tipo de array ou coleção.

Nesses casos, é mais claro usar um array ou uma coleção vazia para representar a ausência de resultados.

Exemplo:


  Optional<List<Integer>> foo() {
    return Optional.empty();
  }

Este código poderia ser assim:


  List<Integer> foo() {
    return List.of();
  }
