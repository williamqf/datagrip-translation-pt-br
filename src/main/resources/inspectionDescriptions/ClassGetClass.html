Relata chamadas ao método getClass() em instâncias de java.lang.Class.

Isso geralmente é um erro, pois o resultado é sempre equivalente a Class.class. Se for um erro, recomenda-se remover a chamada getClass() e usar o qualificador diretamente. Se o comportamento for intencional, é melhor escrever explicitamente Class.class para evitar confusão.

Exemplo:


  void test(Class<?> clazz) {
    String name = clazz.getClass().getName();
  }

Após a aplicação de uma das correções rápidas possíveis:


  void test(Class<?> clazz) {
    String name = clazz.getName();
  }
