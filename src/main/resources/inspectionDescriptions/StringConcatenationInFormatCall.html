Relata a concatenação de strings não constantes usada como um argumento de string de formato.

Embora ocasionalmente intencional, isso geralmente é um uso incorreto de métodos de formatação e pode até levar a problemas de segurança se as variáveis usadas na string concatenada contiverem caracteres especiais como %.

Além disso, às vezes isso pode ser o resultado do erro ao concatenar argumentos de formato de string ao digitar + quando , deveria ter sido digitado.

Exemplo:


  static String formatGreeting(String userName) {
    return String.format("Hello, " + userName);
  }

Aqui, userName será interpretado como parte da string de formato, o que pode levar a uma IllegalFormatException (por exemplo, se userName for "%") ou ao uso de uma quantidade enorme de memória (por exemplo, se userName for "%2000000000%"). Essa chamada provavelmente deve ser substituída por String.format("Hello, %s", userName);.

Esta inspeção verifica chamadas aos métodos de formatação de java.util.Formatter, java.lang.String, java.io.PrintWriter ou java.io.PrintStream.
