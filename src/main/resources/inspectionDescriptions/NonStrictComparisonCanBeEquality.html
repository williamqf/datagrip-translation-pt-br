Relata condições de desigualdade que, com base na análise de fluxo de dados, podem ser satisfeitas por apenas um único valor de operando. Essas condições podem ser substituídas por condições de igualdade para tornar o código mais claro.

Exemplo:


  if (x >= 10) {
    ...
    if (x <= 10) { // pode ser substituído por 'x == 10'
    }
  }
