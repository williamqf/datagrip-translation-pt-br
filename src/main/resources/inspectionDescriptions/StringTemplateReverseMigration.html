Relata expressões de template de string que usam o processador STR e fornece uma correção rápida para migrar de volta para a concatenação de strings simples.

Exemplo:


  String name = "Bob";
  String greeting = STR."Hello, \{name}. You are 29 years old.";

Após aplicar a correção rápida:


  String name = "Bob";
  String greeting = "Hello, " + name + ". You are 29 years old.";

Novidade na versão 2024.2
