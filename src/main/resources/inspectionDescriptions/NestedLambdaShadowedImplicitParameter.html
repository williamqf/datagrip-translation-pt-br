Relata lambdas aninhadas com parâmetros implícitos sombreados.

Exemplo:


fun foo(listOfLists: List<List<String>>) {
  listOfLists.forEach {
    it.forEach {
      println(it)
    }
  }
}

Após a aplicação da correção rápida:


fun foo(listOfLists: List<List<String>>) {
  listOfLists.forEach {
    it.forEach { it1 ->
      println(it1)
    }
  }
}
