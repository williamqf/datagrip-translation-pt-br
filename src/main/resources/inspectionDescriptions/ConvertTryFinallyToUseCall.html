Relata blocos try-finally contendo resource.close() no finally que podem ser convertidos em chamadas resource.use().

O use() é mais legível e menos propenso a erros, pois elimina a necessidade de uma chamada explícita a close().

Exemplo:


  fun example() {
      val reader = File("file.txt").bufferedReader()
      try {
          reader.lineSequence().forEach(::print)
      } finally {
          reader.close()
      }
  }

Após a aplicação do quick fix:


  fun example() {
      File("file.txt").bufferedReader().use { reader ->
          reader.lineSequence().forEach(::print)
      }
  }
