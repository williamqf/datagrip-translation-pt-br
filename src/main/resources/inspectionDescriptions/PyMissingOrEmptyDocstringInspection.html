Relata docstrings ausentes e vazias.

Exemplo de docstring ausente


def demo(a):
    c = a ** 2

Exemplo de docstring vazia


def demo(a):
    """
    """
    c = a ** 2

Após a aplicação da correção rápida, o fragmento de código é alterado para:


def demo(a):
    """

    :param a:
    """
    c = a ** 2

Você deve fornecer alguns detalhes sobre os parâmetros no modelo gerado.
