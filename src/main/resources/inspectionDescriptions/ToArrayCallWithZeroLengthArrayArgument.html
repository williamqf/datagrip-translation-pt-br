Relata chamadas de Collection.toArray() que não seguem o estilo preferido e sugere a aplicação do estilo preferido.

Existem dois estilos para converter uma coleção em um array:

  • Um array pré-dimensionado, por exemplo, c.toArray(new String[c.size()])
  • Um array vazio, por exemplo, c.toArray(new String[0])

Em versões antigas do Java, recomendava-se o uso de um array pré-dimensionado porque a chamada de reflexão necessária para criar um array de tamanho apropriado era muito lenta.

No entanto, desde as atualizações mais recentes do OpenJDK 6, esta chamada foi otimizada, tornando o desempenho da versão com array vazio igual ou, às vezes, superior ao da versão pré-dimensionada. Além disso, passar um array pré-dimensionado é perigoso para coleções concorrentes ou sincronizadas, pois pode ocorrer uma corrida de dados entre as chamadas de size e toArray. Isso pode resultar em um null extra ao final do array se a coleção for reduzida simultaneamente durante a operação.

Use as opções da inspeção para escolher o estilo preferido.
