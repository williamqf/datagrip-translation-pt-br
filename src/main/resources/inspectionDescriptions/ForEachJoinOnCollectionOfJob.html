Relata chamadas forEach { it.join() } em coleções de Job.

Isso pode ser substituído por uma chamada joinAll() mais concisa e totalmente equivalente.

Para mais detalhes, consulte a documentação de kotlinx.coroutines.joinAll.

Exemplo:


  suspend fun waitForJobs(jobs: List<Job>) {
    jobs.forEach { it.join() }
  }

Após a aplicação da correção rápida:


  suspend fun waitForJobs(jobs: List<Job>) {
    jobs.joinAll()
  }
