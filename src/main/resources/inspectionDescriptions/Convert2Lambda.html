Relata classes anônimas que podem ser substituídas por expressões lambda.

Exemplo:


  new Thread(new Runnable() {
    @Override
    public void run() {
      // executar thread
    }
  });

Após a aplicação da correção rápida:


  new Thread(() -> {
    // executar thread
  });

Observe que, se a classe anônima for convertida em um lambda sem estado (stateless), o Java runtime poderá reutilizar o mesmo objeto lambda em chamadas subsequentes. Por outro lado, ao usar classes anônimas, objetos separados são criados a cada vez. Portanto, em casos raros, a aplicação da correção rápida pode resultar em mudanças semânticas, por exemplo, quando uma instância de classe anônima é usada como chave de um HashMap.

Use a opção Relatar quando a interface não estiver anotada com @FunctionalInterface para ignorar casos em que classes anônimas implementam interfaces que não possuem a anotação @FunctionalInterface.
