Relata atributos de canal não resolvidos nas anotações @StreamListener, @SendTo, @Output e @Input.

O uso de nomes de vinculação (binding) inválidos causa erros de 'Bean não encontrado' em tempo de execução.

Exemplo:


  @EnableBinding(Sink.class)
  public class LogSource {
    @StreamListener(Sink.class)
	  public void log1() {
    }

    @StreamListener("invalid")  // relata 'Canal não encontrado'
	  public void log2() {
    }
  }
