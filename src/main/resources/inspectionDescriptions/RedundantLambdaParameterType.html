Relata tipos de parâmetros formais de lambda que são redundantes por poderem ser inferidos a partir do contexto.

Exemplo:


  Map<String, Integer> map = ...
  map.forEach((String s, Integer i) -> log.info(s + "=" + i));

A correção rápida remove esses tipos de parâmetros da lambda.


  Map<String, Integer> map = ...
  map.forEach((s, i) -> log.info(s + "=" + i));
