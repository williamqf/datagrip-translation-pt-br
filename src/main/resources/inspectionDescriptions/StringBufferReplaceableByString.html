Relata o uso de StringBuffer, StringBuilder ou StringJoiner que podem ser substituídos por uma única concatenação de String.

O uso da concatenação de String torna o código mais curto e simples.

Esta inspeção relata apenas quando a substituição sugerida não resultará em uma degradação significativa de desempenho em JVMs modernas. Em muitos casos, a concatenação de String pode ter um desempenho melhor.

Exemplo:


  StringBuilder result = new StringBuilder();
  result.append("i = ");
  result.append(i);
  result.append(";");
  return result.toString();

Após a aplicação do ajuste rápido:


  String result = "i = " + i + ";";
  return result;
