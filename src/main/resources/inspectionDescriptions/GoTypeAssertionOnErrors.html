Relata asserções de tipo ou switches de tipo para erros, como err.(*MyErr) ou switch err.(type), e sugere o uso de errors.As.

A partir do Go 1.13, os erros podem ser envolvidos usando a função fmt.Errorf com o verbo %w. Portanto, uma asserção de tipo ou switch de tipo de erro falhará em um erro envolvido. A maneira preferida de verificar um tipo de erro específico é usar a função errors.As da biblioteca padrão, pois esta função percorre a cadeia de erros envolvidos ao verificar um tipo de erro específico.

Para obter mais informações sobre errors.As, consulte working with Errors in Go 1.13.

A correção rápida substitui asserções de tipo de erro e switches de tipo por chamadas para errors.As.

Exemplo


errFoo, ok := err.(*ErrFoo)


switch err.(type) {
  case *ErrBar:
}

Após a aplicação da correção rápida:


var errFoo *ErrFoo
ok := errors.As(err, &errFoo)

var errBar *ErrBar
switch {
case errors.As(err, &errBar):
}
