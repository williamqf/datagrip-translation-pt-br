Relata construtores desnecessários.

Um construtor é desnecessário se for o único construtor da classe, não possuir parâmetros, tiver o mesmo modificador de acesso que a classe que o contém e não realizar nenhuma inicialização, exceto chamar (explícita ou implicitamente) o construtor da superclasse sem argumentos. Tal construtor pode ser removido com segurança, pois é gerado pelo compilador mesmo que não seja especificado.

Exemplo:


  public class Foo {
    public Foo() {}
  }

Após a aplicação da correção rápida:


  public class Foo {}

Use as configurações de inspeção para ignorar construtores desnecessários que possuam anotações.
