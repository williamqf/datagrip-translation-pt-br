Relata classes das quais muitas outras classes dependem diretamente.

Qualquer modificação em tais classes pode exigir alterações em muitas outras classes, o que pode ter um alto custo.

Somente classes de nível superior são relatadas.

Use o campo abaixo para especificar o número máximo permitido de dependências para uma classe.
