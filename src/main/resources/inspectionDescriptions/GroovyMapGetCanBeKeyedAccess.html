Reporta chamadas ao método java.util.Map.get(). Tais chamadas podem ser substituídas por uma forma de acesso por chave mais curta e clara.

Exemplo:


def map = ["foo": "bar"]
def str = map.get("foo") // map.get("foo") pode ser substituído por map["foo"]

Após a aplicação da correção rápida:


def map = ["foo": "bar"]
def str = map["foo"]
