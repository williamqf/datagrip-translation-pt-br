Relata qualificadores desnecessários em referências. Alguns qualificadores são desnecessários quando uma cláusula use ou um namespace está presente.

Uma correção rápida pode ser usada para remover qualificadores desnecessários.

Exemplo:

use Enumerator::Generator
class Example
  # 'Enumerator::Generator' pode ser encurtado
  def call: -> Enumerator::Generator
end

Após aplicar a correção rápida:

use Enumerator::Generator
class Example
  def call: -> Generator
end
