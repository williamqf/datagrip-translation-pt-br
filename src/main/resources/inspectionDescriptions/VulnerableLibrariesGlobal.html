Relata vulnerabilidades em dependências Gradle e Maven importadas para o projeto. A lista completa de dependências Gradle e Maven é exibida na janela de ferramentas 'Projeto' em 'Bibliotecas Externas'.

Corrigir os problemas relatados ajuda a evitar que seu software seja comprometido por invasores.

Para resolver o problema, você pode atualizar para uma versão na qual a vulnerabilidade foi corrigida (se disponível) ou mudar para uma dependência sem vulnerabilidades.

As correções rápidas disponíveis podem sugerir a atualização para uma versão segura ou a visita a um site para saber mais sobre uma vulnerabilidade específica.
