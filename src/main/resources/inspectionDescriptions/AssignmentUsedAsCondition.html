Relata atribuições usadas como condições em instruções if, while, for ou do, ou em expressões condicionais.

Embora às vezes seja intencional, esse uso é confuso e pode indicar um erro de digitação (por exemplo, o uso de = em vez de ==).

A correção rápida substitui = por ==.

Exemplo:


  void update(String str, boolean empty) {
    // Aviso: 'empty' é reatribuído,
    // não comparado com str.isEmpty()
    if (empty = str.isEmpty()) {
      ...
    }
  }

Após aplicar a correção rápida:


  void update(String str, boolean empty) {
    if (empty == str.isEmpty()) {
      ...
    }
  }
