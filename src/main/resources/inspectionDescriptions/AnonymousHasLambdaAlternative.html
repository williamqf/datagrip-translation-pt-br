Relata classes anônimas que podem ser convertidas em uma chamada de construtor ou de um factory method com argumentos de expressão lambda.

Esta inspeção relata as seguintes categorias:

  • Classes anônimas que estendem ThreadLocal e possuem o método initialValue() (podem ser substituídas por ThreadLocal.withInitial)
  • Classes anônimas que estendem Thread e possuem o método run() (podem ser substituídas por new Thread(Runnable))

Exemplo:


  new Thread() {
    @Override
    public void run() {
      System.out.println("Hello from thread!");
    }
  }.start();

Após a aplicação da correção rápida:


  new Thread(() -> {
    System.out.println("Hello from thread!");
  }).start();
