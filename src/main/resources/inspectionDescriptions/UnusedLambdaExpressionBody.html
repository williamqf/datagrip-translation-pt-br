Relata chamadas com um valor de retorno não utilizado em que a função chamada retorna um lambda a partir de um corpo de expressão.

Se houver um = entre o cabeçalho da função e o bloco do corpo, o código dentro da função não será avaliado, o que pode levar a um comportamento incorreto.

A ação Remover o token = da declaração da função pode ser usada para corrigir o código automaticamente.

Exemplo:


  fun printHello() = { println("Hello") }

  fun main() {
      printHello() // esta função não imprime nada
  }

Após aplicar a correção rápida:


  fun printHello() { println("Hello") }

  fun main() {
      printHello()
  }
