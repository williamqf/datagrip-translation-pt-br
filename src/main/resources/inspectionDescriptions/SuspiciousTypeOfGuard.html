Relata verificações de type guard typeof ou instanceof que não são íntegras. Um type guard typeof x pode não ser íntegro em um dos dois casos a seguir:
  • typeof x nunca corresponde ao valor especificado (por exemplo, typeof x === 'number' quando x é do tipo 'string | boolean')
  • typeof x sempre corresponde ao valor especificado (por exemplo, typeof x === 'string' quando x é do tipo 'string')

Um type guard x instanceof A pode não ser íntegro em um dos dois casos a seguir:
  • O tipo de x não está relacionado a A
  • O tipo de x é A ou um subtipo de A
