Relata nomes duplicados de aliases de coluna na lista SELECT.

Exemplo (Sybase ASE):

CREATE TABLE t1 (a TEXT, b INT, c INT);

SELECT a AS x, b AS x FROM t1;

O alias x é usado para as colunas a e b. Essas atribuições são realçadas como erros porque você não pode usar o mesmo alias para várias colunas no Sybase ASE.
