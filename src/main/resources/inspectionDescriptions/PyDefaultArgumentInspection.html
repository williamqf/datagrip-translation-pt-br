Relata problemas quando valores mutáveis, como listas ou dicionários, são detectados em valores padrão de argumentos.
Os valores padrão dos argumentos são avaliados apenas uma vez no momento da definição da função, o que significa que modificar o valor padrão do argumento afetará todas as chamadas subsequentes a essa função.

Exemplo:


def func(s, cache={}):
    cache[s] = None

Após a aplicação da correção rápida, o código torna-se:


def func(s, cache=None):
    if cache is None:
        cache = {}
    cache[s] = None
