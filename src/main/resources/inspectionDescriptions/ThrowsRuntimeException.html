Relata declarações de exceções não verificadas (java.lang.RuntimeException ou uma de suas subclasses) na cláusula throws de um método.

A declaração de exceções não verificadas não é necessária e pode ser removida ou movida para a tag Javadoc @throws.

Exemplo:


  public class InvalidDataException extends RuntimeException {}

  class TextEditor {
    void readSettings() throws InvalidDataException {} // Aviso: exceção não verificada 'InvalidDataException' declarada na cláusula 'throws'
  }
