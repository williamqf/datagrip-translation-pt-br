Relata o uso de JBColor.namedColor() com chaves não registradas.

Todas as chaves devem ser registradas com os metadados correspondentes em *.themeMetadata.json.

A correção rápida oferece navegação para todos os arquivos candidatos *.themeMetadata.json conhecidos para adicionar as entradas necessárias.

Para mais detalhes, consulte Expondo metadados de tema na documentação do SDK.
