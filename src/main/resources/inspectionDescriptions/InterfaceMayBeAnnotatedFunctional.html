Reporta interfaces que podem ser anotadas com @FunctionalInterface.

Anotar uma interface com @FunctionalInterface indica que a interface é funcional e que não é possível adicionar outros métodos abstract a ela.

Exemplo:


  interface FileProcessor {
    void execute(File file);
  }

Após a correção rápida:


  @FunctionalInterface
  interface FileProcessor {
    void execute(File file);
  }
