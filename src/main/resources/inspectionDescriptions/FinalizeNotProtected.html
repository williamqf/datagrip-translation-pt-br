Relata quaisquer implementações do método Object.finalize() declaradas como public.

De acordo com o contrato de Object.finalize(), apenas o coletor de lixo chama este método. Tornar este método public pode ser confuso, pois implica que o método pode ser usado por outros códigos.

Uma correção rápida está disponível para tornar o método protected, evitando que ele seja chamado por outras classes.

Exemplo:


  class X {
    public void finalize() {
      /* ... */
    }
  }

Após aplicar a correção rápida:


  class X {
    protected void finalize() {
      /* ... */
    }
  }
