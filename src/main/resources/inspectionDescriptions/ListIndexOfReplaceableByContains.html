Relata qualquer expressão List.indexOf() que possa ser substituída pelo método List.contains().

Exemplo:


  boolean hasEmptyString(List<String> list) {
    // Alerta: pode ser simplificado
    return list.indexOf("") >= 0;
  }

A correção rápida fornecida substitui a chamada indexOf por uma chamada contains:


  boolean hasEmptyString(List<String> list) {
    // Correção rápida aplicada
    return list.contains("");
  }
