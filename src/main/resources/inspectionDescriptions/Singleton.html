Relata classes singleton.

Uma classe singleton é declarada de forma que apenas uma instância da classe possa ser instanciada. As classes singleton complicam os testes e sua presença pode indicar uma deficiência no design orientado a objetos.

Exemplo:


  class Singleton {
    private static final Singleton ourInstance = new Singleton();

    private Singleton() {
    }

    public Singleton getInstance() {
        return ourInstance;
    }
  }
