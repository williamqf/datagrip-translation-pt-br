Relata cabeçalhos HTTP desconhecidos que não correspondem a nenhum cabeçalho publicamente conhecido. A correção rápida sugere adicionar o cabeçalho à lista de cabeçalhos personalizados quando a opção Usar cabeçalhos HTTP personalizados estiver ativada. Cabeçalhos HTTP na lista de cabeçalhos personalizados não acionarão a inspeção.
