Relata instruções throw dentro de blocos finally. Embora ocasionalmente intencionais, tais instruções throw podem ocultar exceções lançadas nos blocos try e catch, tornando o processo de depuração mais complexo.

Exemplo:


  class Fascinating {
    void doTask(Task t) {
      try {
        t.activity();
      } finally {
        if (!t.finish()) {
          throw new IllegalStateException();
        }
      }
    }
  }

Use a opção Avisar em todos os pontos onde uma exceção declarada pode ser lançada para também emitir avisos em chamadas de métodos que declaram exceções.
