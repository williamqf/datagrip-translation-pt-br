Relata o uso das anotações @Ignore do JUnit 4 ou @Disabled do JUnit 5. Manter testes anotados com essas anotações por um longo período é considerado um code smell, especialmente se nenhum motivo for especificado.

Exemplo:


  @Ignore
  public class UrgentTest {

    @Test
    public void testIt() {
      Assert.assertEquals("expected", "actual");
    }
  }

Configuração da inspeção:

  • Use a opção Relatar apenas anotações sem motivo para relatar apenas os casos em que nenhum motivo foi especificado no atributo value da anotação.
