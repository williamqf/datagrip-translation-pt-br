Relata chamadas a métodos de manipulação de arrays não genéricos, como Arrays.fill(), nos quais os tipos dos argumentos não coincidem. Tais chamadas não realizam nenhuma tarefa útil e são provavelmente erros.

Exemplo:


  int foo(String[] strings) {
    return Arrays.binarySearch(strings, 1);
  }

Novidade na versão 2017.2
