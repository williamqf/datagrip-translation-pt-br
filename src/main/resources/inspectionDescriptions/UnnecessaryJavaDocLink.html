Relata as tags Javadoc @see, {@link} e {@linkplain} que se referem ao método que possui o comentário, a um supermétodo do método com comentário ou à classe que contém o comentário.

Tais links são desnecessários e podem ser removidos com segurança por meio da correção rápida desta inspeção. Se a tag for o único conteúdo, a correção rápida removerá todo o comentário Javadoc.

Exemplo:


  class Example {
    /**
     * @see Example#method
     */
    public void method() { }
  }

Após a aplicação da correção rápida:


class Example {
  public void method() { }
}

Use a caixa de seleção abaixo para ignorar links inline ({@link} e {@linkplain}) para supermétodos. Embora links para todos os supermétodos sejam adicionados automaticamente pela ferramenta Javadoc, às vezes links inline para supermétodos podem ser necessários no texto de um comentário Javadoc.
