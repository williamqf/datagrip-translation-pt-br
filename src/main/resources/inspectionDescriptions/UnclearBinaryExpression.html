Relata expressões binárias, condicionais ou instanceof compostas por diferentes operadores sem parênteses. Devido às diferentes regras de precedência dos operadores, essas expressões podem ter baixa legibilidade.

Exemplo:


  int n = 3 + 9 * 8 + 1;

Após a aplicação da correção rápida:


  int n = 3 + (9 * 8) + 1;
