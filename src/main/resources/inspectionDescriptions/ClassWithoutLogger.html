Relata classes que não possuem um logger declarado.

Garantir que cada classe tenha um logger dedicado é um passo importante para fornecer uma implementação de logging unificada para a aplicação. Esta inspeção não relata interfaces, enums, anotações, classes internas e classes abstratas.

Exemplo:


  public class NoLoggerDeclared {

    int calculateNthDigitOfPi(int n) {
      // todo
      return 1;
    }
  }

Use a tabela na seção Opções para especificar os nomes das classes de logger. Esta inspeção relatará classes que não declaram um campo de um dos tipos de classe especificados.
