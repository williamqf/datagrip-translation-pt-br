Relata funções com = { ... } e tipos de retorno inferidos.

Exemplo:


  fun sum(a: Int, b: Int) = { a + b } // O tipo de retorno desta função é '() -> Int'.

Existem várias correções rápidas. Após aplicar uma correção rápida:


    fun sum(a: Int, b: Int) = a + b // Remover chaves
    fun sum(a: Int, b: Int): () -> Int = { a + b } // Especificar o tipo de retorno explicitamente
    fun sum(a: Int, b: Int) = run { a + b } // Converter para run { ... }
    fun sum(a: Int, b: Int) = { -> a + b } // Especificar uma assinatura de lambda explícita
