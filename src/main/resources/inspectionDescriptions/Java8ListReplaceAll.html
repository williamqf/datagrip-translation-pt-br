Relata loops que podem ser substituídos por uma única chamada de List.replaceAll().

Exemplo:


  for (int i = 0; i < strings.size(); i++) {
    String str = strings.get(i).toLowerCase();
    strings.set(i, str);
  }

Após a aplicação da correção rápida:


  strings.replaceAll(String::toLowerCase);
