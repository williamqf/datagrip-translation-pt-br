Relata problemas essenciais para este arquivo (por exemplo, erros de sintaxe) nos resultados da execução da inspeção de código em lote. Ao contrário das inspeções, esses problemas geralmente são realçados no editor e não podem ser configurados. Estas opções controlam o escopo das verificações realizadas por esta inspeção:
  • Opção Relatar erros de sintaxe: relata problemas relacionados ao parser.
  • Opção Relatar problemas de anotadores específicos da linguagem: relata problemas encontrados por anotadores configurados para a linguagem relevante. Consulte Suporte a Linguagem Personalizada: Anotadores para obter detalhes.
  • Opção Relatar outros problemas de realce: relata problemas específicos da linguagem do arquivo atual (por exemplo, incompatibilidade de tipos ou exceções não relatadas). Consulte Suporte a Linguagem Personalizada: Realce para obter detalhes.
