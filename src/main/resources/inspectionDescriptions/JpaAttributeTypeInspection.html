Relata incompatibilidades de tipo de propriedade para atributos JPA.

Exemplo:



  @Entity
  public class JavaEntity {

    @OneToOne
    Map<Integer, AnotherEntity> incorrectRelationship; // Erro: o tipo do atributo 'One To One' deve ser Entity, não Map

    @ManyToMany
    Map<Integer, AnotherEntity> correctRelationship;
  }
