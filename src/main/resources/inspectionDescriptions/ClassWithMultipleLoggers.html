Relata classes que declaram múltiplos loggers. Um passo importante para fornecer uma implementação de log uniforme para uma aplicação é garantir que cada classe tenha um único logger dedicado.

Exemplo:


  public class Critical {
    protected static final Logger LOG = Logger.getLogger(Critical.class);

    protected static final Logger myLogger = Logger.getLogger(getClass());
  }
