Relata chamadas ao método openSession() se o recurso org.hibernate.Session retornado não for fechado com segurança.

Por padrão, esta inspeção assume que o recurso pode ser fechado por qualquer método que contenha 'close' ou 'cleanup' em seu nome.

Exemplo:


  void doHibernateQuery(SessionFactory factory) {
    Session session = factory.openSession(); // aviso
    session.createQuery("...");
  }

Configure a inspeção usando as seguintes opções:

  • Se é permitido abrir recursos org.hibernate.Session dentro de blocos try. Este estilo é menos desejável, pois é mais verboso do que abrir o recurso antes do bloco try.
  • Se o recurso pode ser fechado por qualquer chamada de método que passe o recurso como um argumento.
