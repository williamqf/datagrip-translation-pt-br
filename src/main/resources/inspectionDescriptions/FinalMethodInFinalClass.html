Relata métodos final em classes final.

Como classes final não podem ser herdadas, marcar um método como final é redundante e pode ser confuso.

Exemplo:

record Bar(int a, int b) {
  public final int sum() { 
     return a + b;
  }
}

Após a aplicação da correção rápida:

record Bar(int a, int b) {
  public int sum() { 
     return a + b;
  }
}

Como mostrado neste exemplo, uma classe pode ser marcada como final de forma explícita ou implícita.
