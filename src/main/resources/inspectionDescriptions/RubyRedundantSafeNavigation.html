Relata o uso do operador de navegação segura (&.) com receptores não nulos que podem ser ignorados.

Use a lista de Receptores ignorados para especificar chamadas de método usadas como receptores que devem ser ignorados (por exemplo, Foo#call ou Foo::call para métodos de instância ou singleton, respectivamente).

Exemplo:

value = 1
if value&.even?
  # ...
end
