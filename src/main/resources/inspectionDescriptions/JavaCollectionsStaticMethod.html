Relata chamadas de métodos estáticos de Collections do Java que podem ser substituídas por funções da Kotlin stdlib.

Exemplo:


  import java.util.Collections

  fun test() {
      val mutableList = mutableListOf(1, 2)
      Collections.fill(mutableList, 3)
  }

A correção rápida substituirá a chamada de método estático de Collections do Java pela chamada de método correspondente da Kotlin stdlib:


  import java.util.Collections

  fun test() {
      val mutableList = mutableListOf(1, 2)
      mutableList.fill(3)
  }
