Relata links TMS possivelmente inválidos.

Sugere a presença de erros de digitação no ID e em links para elementos TMS que podem ter sido removidos.

Causas possíveis:

  • A anotação de link TMS não fornece o atributo value ou ele está vazio

  • O ID especificado não foi encontrado entre os IDs de elementos TMS carregados
