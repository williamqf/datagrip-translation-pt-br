Relata objetos StringBuilder, StringBuffer ou StringJoiner cujos conteúdos são lidos mas não escritos, ou escritos mas não lidos.

Tais leituras e escritas inconsistentes não fazem sentido e podem indicar código morto, código incompleto ou erros.

Exemplo:


  public void m1() {
    StringBuilder sb = new StringBuilder();
    sb.append("a");
  }
