Relata anotações de "cache" incorretas: @Cacheable, @CacheEvict, @CachePut, @CacheConfig, etc.

Exemplo:


@org.springframework.stereotype.Component
public class MyCacheManager implements CacheManager {...}

public class MyConfiguration {
   @Cacheable(value = "a",
       cacheResolver ="myCacheManager") // Relata "O Bean deve ser do tipo 'org.springframework.cache.interceptor.CacheResolver'"
   public String getCache(String isbn) {...}

   @Cacheable(value = "abc",
   private String getAbc() // Relata "Anotações de cache devem ser definidas em métodos públicos"
   {...}
}
