Relata instâncias em que uma variável pode ser nil ou ter um valor inesperado porque não foi verificado se o erro associado não é nil, como em v, err := foo().

A correção rápida removerá a função do escopo desta inspeção.

Exemplo:


func _() {
    file, err := os.Open("file.txt")
    // A verificação de erro foi omitida aqui
    name := file.Name()
    print(name, err)
}

No exemplo fornecido, file pode ser nil ou ter um valor inesperado quando err não for nil.
