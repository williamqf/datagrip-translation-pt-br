Relata variáveis de caminho não utilizadas no Kotlin router DSL, bem como usos de variáveis de caminho não declaradas.

As variáveis de caminho devem ser declaradas no caminho da rota antes do acesso. A tentativa de acessar uma variável de caminho não declarada resultará em um erro de tempo de execução.

Exemplo:


@Configuration
class RouterConfiguration {
    @Bean
    fun myRouter() = router {
        GET("/test/{var}") { ServerResponse.ok().body("${it.pathVariable("bar")}") }
    }
}

Após aplicar a correção:


@Configuration
class RouterConfiguration {
    @Bean
    fun myRouter() = router {
        GET("/test/{var}") { ServerResponse.ok().body("${it.pathVariable("var")}") }
    }
}
