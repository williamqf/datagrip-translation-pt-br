Relata os operadores until e rangeTo em loops for que podem ser substituídos por Collection.indices ou pela iteração sobre a coleção.

O uso de açúcar sintático pode tornar o código mais simples.

A correção rápida substituirá o intervalo manual pela construção correspondente.

Exemplo:


  fun main(args: Array<String>) {
      for (index in 0..args.size - 1) {
          println(args[index])
      }
  }

Após a aplicação da correção rápida:


  fun main(args: Array<String>) {
      for (element in args) {
          println(element)
      }
  }
