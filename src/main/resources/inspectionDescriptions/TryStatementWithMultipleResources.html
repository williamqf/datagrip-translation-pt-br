Relata instruções try com múltiplos recursos que podem ser divididas automaticamente em várias instruções try-with-resources.

Essa transformação é útil para refatorações posteriores (por exemplo, para extrair instruções try aninhadas em métodos separados).

Exemplo:


  try (FileInputStream in = new FileInputStream("in.txt");
     FileOutputStream out = new FileOutputStream("out.txt")) {
     /*leitura e escrita*/
  }

Após a aplicação da correção rápida:


  try (FileInputStream in = new FileInputStream("in.txt")) {
    try (FileOutputStream out = new FileOutputStream("out.txt")) {
      /*leitura e escrita*/
    }
  }
