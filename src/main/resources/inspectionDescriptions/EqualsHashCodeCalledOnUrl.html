Relata chamadas hashCode() e equals() nos objetos java.net.URL.

Os métodos equals() e hashCode() do URL podem executar pesquisa de DNS para resolver o nome do host. Isso pode causar atrasos significativos, dependendo da disponibilidade e velocidade da rede e dos servidores DNS. Usar java.net.URI em vez de java.net.URL evitará pesquisa de DNS.

Exemplo:


  int equalsHashCode(URL url1, URL url2) {
    return url1.hashCode() == url2.hashCode();
  }
