Relata cabeçalhos de sintaxe wiki desbalanceados e o texto ignorado após tais cabeçalhos em documentos do Scaladoc.

No exemplo a seguir, ==== é marcado:


  /**
   * == I'm a wiki style header ====
   */
  def teste(i: Int): Unit = {

  }

No exemplo a seguir, Texto ignorado é marcado:


  /**
   * == I'm a wiki style header == Texto ignorado
   */
  def teste(i: Int): Unit = {

  }
