Relata casts inseguros com return em atribuições de variáveis ou argumentos de valor que podem ser substituídos por um cast seguro.

A correção rápida substitui o cast inseguro por um cast seguro.

Exemplo


    fun test(x: Any): Any? {
        val y = x as String ?: return null
    }

Após aplicar a correção rápida:


  fun test(x: Any): Any? {
      val y = x as? String ?: return null
  }
