Relata cadeias longas de transformação de Reactive Streams.

Cada chamada de método de operador, como map() ou filter(), cria objetos para esses operadores. Chamar cadeias longas de operadores para cada elemento do stream em cada assinatura pode resultar em sobrecarga de desempenho. Para evitar isso, combine cadeias longas de chamadas em uma única chamada de operador, sempre que possível.

Exemplo:


Flux.just(1, 2, 3)
        .map(it -> it + 1)
        .map(it -> it + 2)
        .map(it -> it + 3)

Após a aplicação da correção rápida:


Flux.just(1, 2, 3)
        .map(it -> it + 1 + 2 + 3)
