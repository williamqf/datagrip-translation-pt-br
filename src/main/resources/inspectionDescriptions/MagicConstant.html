Relata expressões que podem ser substituídas por constantes "mágicas".

Exemplo 1:


  // Literal simples "2" usado; aviso:
  Font font = new Font("Arial", 2)

Exemplo 2:


  // Constante predefinida usada; bom:
  Font font = new Font("Arial", Font.ITALIC)

Se possível, a correção rápida insere a constante predefinida correspondente.

O comportamento desta inspeção é controlado pela anotação org.intellij.lang.annotations.MagicConstant. Alguns métodos da biblioteca Java padrão são pré-anotados, mas você também pode usar esta anotação em seu código.
