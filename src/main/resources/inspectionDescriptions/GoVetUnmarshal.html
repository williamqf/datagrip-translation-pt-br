Reporta chamadas para json.Unmarshal e funções semelhantes se o argumento passado para armazenar o resultado não for um ponteiro ou uma interface.

Essas chamadas falharão e retornarão um erro.

Para obter detalhes sobre o Unmarshal, consulte func Unmarshal no go.dev.

Exemplo:

var animals []Animal
err := json.Unmarshal(jsonData, animals) // sempre retorna um erro

Após aplicar o ajuste rápido Prefixar com '&':

var animals []Animal
err := json.Unmarshal(jsonData, &animals)
