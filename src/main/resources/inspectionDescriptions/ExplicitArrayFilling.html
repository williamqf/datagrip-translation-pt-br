Relata loops que podem ser substituídos por chamadas a Arrays.setAll() ou Arrays.fill().

Esta inspeção sugere substituir o loop por Arrays.setAll() se o nível de linguagem do projeto ou módulo for 8 ou superior. Você pode substituir o loop por Arrays.fill() em qualquer nível de linguagem.

Exemplo:


  for (int i=0; i<array.length; i++) {
     array[i] = calc(i);
  }

Após aplicar a correção rápida:


  Arrays.setAll(array, this::calc);
