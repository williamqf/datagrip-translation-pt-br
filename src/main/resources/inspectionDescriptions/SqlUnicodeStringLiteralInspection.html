Relata literais de string que usam caracteres nacionais sem o prefixo N.

Sem o prefixo N, a string será convertida para a página de código padrão do banco de dados. Esta página de código padrão pode não reconhecer determinados caracteres. Para obter mais informações, consulte a página "nchar e nvarchar" (Transact-SQL) no docs.microsoft.com.

Exemplo (Microsoft SQL Server):

SELECT 'abcde' AS a;
SELECT N'abcde' AS b;
SELECT 'абвгд' AS c;
SELECT N'абвгд' AS d;

Como SELECT 'абвгд' AS c; não possui o prefixo N, a parte 'абвгд' será realçada.
