Relata struct tags que não estão em conformidade com as convenções do Go para struct tags.

De acordo com essas convenções, uma string de tag é uma concatenação de pares key:"value" opcionalmente separados por espaços. Cada chave é uma string não vazia composta por caracteres de não controle que não sejam espaço (U+0020 ' '), aspas (U+0022 '"') e dois-pontos (U+003A ':'). Cada valor é delimitado por caracteres U+0022 '"' e utiliza a sintaxe de literal de string do Go.

Além disso, esta inspeção verifica se os campos com tags são exportados.

Exemplo de tags válidas:

type Example struct {
  Field int `json:"field" xml:"demo"`
}
