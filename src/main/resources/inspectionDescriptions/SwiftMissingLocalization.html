Reporta o uso de strings localizadas que não foram adicionadas aos arquivos de strings.

Por padrão, esta inspeção é acionada se uma chave de localização não estiver definida em nenhum dos arquivos de strings disponíveis. Use a caixa de seleção abaixo para avisar apenas se esta chave não estiver definida em todos os idiomas.

Sugere a adição dos pares chave-valor ausentes aos arquivos de strings.

Exemplo: 


  let presentation = "Do It"

Após aplicar a correção rápida: 


  let presentation = NSLocalizedString("Do It", comment: ""))
