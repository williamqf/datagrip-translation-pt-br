Relata declarações de campo TokenSet na classe ParserDefinition que referenciam tipos de elementos que não são da plataforma.

As instâncias de ParserDefinition de todos os idiomas são criadas na inicialização da aplicação. Declarar um TokenSet que referencia tipos de elementos de linguagem que não são da plataforma pode levar à criação e ao registro de todos os tipos de elementos de linguagem na classe container dos tipos referenciados, mesmo que o projeto não contenha nenhum arquivo desse idioma.

Exemplo:


// Container de tipos de elemento:
public interface MyLangTokenTypes {
  IElementType COMMENT = new MyLangTokenType("COMMENT");
  IElementType TYPE1 = new MyLangTokenType("TYPE1");
  IElementType TYPE2 = new MyLangTokenType("TYPE2");
  // mais tipos...
}


// Incorreto:

public class MyLangParserDefinition implements ParserDefinition {
  // Este campo causa a inicialização e o registro de todos os tipos em MyLangTokenTypes:
  private static final TokenSet COMMENTS = TokenSet.create(MyLangTokenTypes.COMMENT);

  @NotNull
  @Override
  public TokenSet getCommentTokens() {
    return COMMENTS;
  }
  ...
}


// Correto:

public final class MyLangTokenSets {
  public static final TokenSet COMMENTS = TokenSet.create(MyLangTokenTypes.COMMENT);
}

public class MyLangParserDefinition implements ParserDefinition {
  @NotNull
  @Override
  public TokenSet getCommentTokens() {
    // Os tipos são referenciados e registrados apenas quando este método é chamado:
    return MyLangTokenSets.COMMENTS;
  }
  ...
}

// Correto (Kotlin):

// Declarações de nível superior não são carregadas até que o método getCommentTokens() seja chamado:
private val COMMENTS = TokenSet.create(MyLangTokenTypes.COMMENT);

class MyLangParserDefinition : ParserDefinition {
  override getCommentTokens(): TokenSet {
    return COMMENTS;
  }
  ...
}

// Correto:

public class MyLangParserDefinition implements ParserDefinition {
  // TokenSet da plataforma permitido:
  private static final TokenSet COMMENTS1 = TokenSet.EMPTY;
  // TokenType da plataforma permitido:
  private static final TokenSet COMMENTS2 = TokenSet.create(TokenType.WHITE_SPACE);

  @NotNull
  @Override
  public TokenSet getCommentTokens() {
    ...
  }
  ...
}
