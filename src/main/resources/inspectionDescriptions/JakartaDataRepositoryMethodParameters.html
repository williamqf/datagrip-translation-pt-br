Reporta tipos de parâmetro de método de repositório incorretos.

Exemplo:


  public class Person {
    private int id;
    @Column
    private String lastname;

    @Column
    private int age;
  }

  public interface CustomerRepository extends Repository<Person, Integer> {
    List<Person> findByAddress(int lastname);  // deve ser do tipo 'String'
    List<Person> deleteByAge(String str);   // deve ser do tipo 'int'
  }
