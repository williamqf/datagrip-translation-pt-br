Relata o uso de interfaces vazias como tipos ou restrições de tipo.

Uma interface vazia representa o conjunto de todos os tipos. O Go 1.18 introduziu o alias mais explícito any para interface{}, que é equivalente a interface{} em todos os aspectos.

A inspeção é acionada se uma interface vazia for usada como um tipo ou restrição de tipo. Aliases de interfaces vazias e interfaces que incorporam exclusivamente outras interfaces vazias não acionam a inspeção.

Considere usar o alias mais explícito any em seu lugar.
