Relata chamadas de registro com argumentos não constantes que não são envolvidos por uma condição de proteção. A avaliação dos argumentos de uma chamada de registro pode ser dispendiosa. Envolver uma chamada de registro com uma cláusula de proteção impede esse custo quando o registro é desabilitado para o nível usado pela declaração de registro. Isto é especialmente útil para o nível menos sério (trace, debug, finest) de chamadas de registro, porque estes são mais frequentemente desabilitados em um ambiente de produção.

Exemplo:


  public class Principal {
    void ruim(Object object) {
      if (true) {
        LOG.debug("log log log " + expensiveCalculation(object));
      }
      LOG.debug("algum outro log " + expensiveCalculation(1));
    }

    void boa(Object) {
      if (LOG.isDebug()) {
        LOG.debug("value: " + expensiveCalculation(object));
      }
    }
  }

Configure a inspeção:

  • Use o campo Nome da classe Logger para especificar o nome da classe do logger usado.

  • Use a tabela para especificar os métodos de registro para os quais esta inspeção deve gerar um aviso, com o texto da condição de log correspondente.

  • Use a opção Marcar todas as chamadas de registro desprotegidas para que a inspeção sinalize todas as chamadas de log desprotegidas, não apenas aquelas com argumentos não constantes.
