Relata classes, métodos e campos depreciados que ainda estão sendo usados no código.

Exemplo:


  class MyCode {
  @Deprecated
  void oldMethod() {}// Aviso: "Membro depreciado ainda em uso"

  void newMethod() {
    oldMethod(); // Uso esquecido
  }
  }

Usos dentro de elementos depreciados são ignorados.

Nota: Por motivos de desempenho, membros não privados são verificados apenas se seus nomes aparecerem raramente no projeto.
