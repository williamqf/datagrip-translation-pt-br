Relata expressões if-then que podem ser convertidas em expressões de acesso seguro (?.).

Exemplo:


  fun bar(x: String) = ""

  fun foo(a: String?) {
     if (a != null) bar(a) else null
  }

A correção rápida converterá a expressão if-then em uma expressão de acesso seguro (?.):


  fun bar(x: String) = ""

  fun foo(a: String?) {
     a?.let { bar(it) }
  }
