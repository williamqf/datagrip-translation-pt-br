Relata classes ou arquivos que contêm vários métodos main sobrecarregados no Groovy 5.

Quando uma classe ou arquivo possui vários métodos main com assinaturas diferentes, pode não estar claro qual deles serve como o ponto de entrada da aplicação, pois a invocação do método main ocorre no tempo de execução com base na presença de argumentos. Considere o seguinte exemplo: 


class A {
    void main(String[] args) {
        println "1"
    }

    void main() {
        println "2"
    }
}
Se nenhum argumento for passado, main() será chamado; caso contrário, main(String[]) será chamado.
