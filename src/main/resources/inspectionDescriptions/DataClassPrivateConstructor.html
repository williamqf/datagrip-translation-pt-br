Relata um construtor primário private em uma classe data.

As classes data possuem o método de fábrica copy(), que pode ser usado de forma semelhante a um construtor. Para fornecer segurança suficiente, o construtor não deve ser marcado como private.

Exemplo:


  data class User private constructor(val name: String)

A correção rápida alterará o modificador de visibilidade do construtor para public:


  data class User(val name: String)
