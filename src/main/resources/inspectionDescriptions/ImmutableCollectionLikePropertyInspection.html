Relata o uso de coleções imutáveis em entidades JPA.

Detecta propriedades anotadas com @OneToMany ou @ManyToMany que possuem tipos de coleção imutáveis (List, Set, Map). Sugere o uso de tipos de coleção mutáveis (MutableList, MutableSet, MutableMap) para garantir a compatibilidade com o Hibernate.

Trecho de código: 


@Entity
open class User(
  @Id
  val id: Long = 0L,
  @OneToMany
  val orders: List = listOf(),
)
Após a aplicação da correção rápida: 

@Entity
open class User(
  @Id
  val id: Long = 0L,
  @OneToMany
  val orders: MutableList = mutableListOf(),
)
