Relata variáveis de instância que podem ser seguramente definidas como static. Se um campo for declarado como final e inicializado com uma constante, ele pode ser um campo static.

Exemplo:


  public final String str = "sample";

Esta inspeção não relata campos final que podem ser escritos implicitamente. Use o botão "Anotações" para modificar a lista de anotações que pressupõem a escrita implícita de campos.
