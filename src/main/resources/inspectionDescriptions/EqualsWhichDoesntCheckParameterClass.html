Relata métodos equals() que não verificam o tipo de seu parâmetro.

Se o tipo do parâmetro no método equals() não for verificado, erros potenciais podem ocorrer quando o objeto for usado em coleções não tipadas.

Exemplo:


  class MyClass {
    int x;

    @Override
    public boolean equals(Object obj) {
      // Se obj não for MyClass, o método equals deve retornar false
      return ((MyClass)obj).x == x;
    }
  }
