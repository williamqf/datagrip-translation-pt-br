Relata métodos booleanos cujos nomes não começam com uma palavra interrogativa.

Esta inspeção ignora métodos booleanos que sobrescrevem métodos de biblioteca.

Exemplo:

boolean empty(List<String> list) {
  return list.isEmpty();
}

A correção rápida para renomear tais métodos está disponível apenas no editor.

Configuração da inspeção:

  • Use a lista Prefixos de nome de método booleano para especificar as palavras interrogativas aceitáveis para iniciar nomes de métodos booleanos.
  • Use a opção Ignorar métodos com tipo de retorno 'java.lang.Boolean' para ignorar métodos com o tipo de retorno java.lang.Boolean.
  • Use a opção Ignorar métodos booleanos em @interface para ignorar métodos booleanos de tipos de anotação (@interface).
  • Use a opção Ignorar métodos que sobrescrevem/implementam um super método para ignorar métodos que possuem um super método.
