thoughtful Relata instruções inacessíveis em rotinas SQL.

Exemplo (Microsoft SQL Server):

CREATE FUNCTION foo() RETURNS INT AS
BEGIN
    THROW;
    RETURN 1;
END;

No Microsoft SQL Server, a instrução THROW lança uma exceção e transfere a execução para o bloco CATCH de uma estrutura TRY...CATCH. Portanto, a parte RETURN 1; nunca será executada.
