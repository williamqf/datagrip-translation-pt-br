Relata funções forEach dentro de funções de escopo, como also ou apply, que podem ser simplificadas.

A correção rápida Converter chamada forEach para onEach pode ser usada para corrigir o código automaticamente.

Exemplo:


  fun test(list: List<Int>) {
      val x = list.also { it.forEach { it + 4 } }.toString()
      val y = list.apply { forEach { println(it) } }
  }

Após aplicar a correção rápida:


  fun test(list: List<Int>) {
      val x = list.onEach { it + 4 }.toString()
      val y = list.onEach { println(it) }
  }
