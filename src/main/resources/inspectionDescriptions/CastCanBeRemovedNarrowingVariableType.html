Relata casts de tipo que podem ser removidos se o tipo da variável for restringido ao tipo do cast.

Exemplo:


  Object x = "  string  ";
  System.out.println(((String)x).trim());

Aqui, alterar o tipo de x para String torna o cast redundante. A correção rápida sugerida atualiza o tipo da variável e remove todos os casts redundantes dessa variável: 


  String x = "  string  ";
  System.out.println(x.trim());
