Relata construtores redundantes declarados em records Java.

Exemplo 1:


  record Point(int x, int y) {
    public Point {} // pode ser removido
  }
  
  record Point(int x, int y) {
    public Point(int x, int y) { // pode ser removido
      this.x = x;
      this.y = y;
    }
  }

A correção rápida remove o construtor redundante.

Exemplo 2:


   // pode ser convertido para um construtor compacto
  record Range(int from, int to) {
    public Range(int from, int to) {
      if (from > to) throw new IllegalArgumentException();
      this.from = from;
      this.to = to;
    }
  }

A correção rápida converte este código em um construtor compacto.
