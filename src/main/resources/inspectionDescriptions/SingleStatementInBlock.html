Relata instruções de fluxo de controle que contêm apenas uma instrução em um bloco de código e sugere a remoção das chaves do corpo da instrução de fluxo de controle.

Exemplo:


  if (x > 0) {
    System.out.println("x is positive");
  }

Após aplicar a correção rápida:


  if (x > 0) System.out.println("x is positive");
