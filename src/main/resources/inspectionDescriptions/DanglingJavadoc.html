Relata comentários Javadoc que não pertencem a nenhuma classe, método ou campo. A ferramenta Javadoc ignora comentários Javadoc suspensos e não os inclui na documentação HTML gerada.

Exemplo:


  class A {
    /**
     * Comentário suspenso
     */
    /**
     * Javadoc de método
     */
     public void m(){}
  }

É possível excluir completamente tais comentários ou convertê-los em comentários de bloco por meio de uma correção rápida. Após a aplicação da correção rápida:


  class A {
    /*
      Comentário suspenso
    */
    /**
     * Javadoc de método
     */
    public void m(){}
  }

Use a opção Ignorar comentários de cabeçalho de arquivo no formato JavaDoc para ignorar comentários no início de arquivos Java. Estes são geralmente mensagens de copyright.
