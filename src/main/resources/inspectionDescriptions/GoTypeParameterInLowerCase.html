Relata parâmetros de tipo declarados com letras minúsculas.

Os exemplos na documentação oficial do Go usam parâmetros de tipo em letras maiúsculas. Esta inspeção segue esta regra de capitalização para parâmetros de tipo.

func PrintSlice[t any](s []t) {
    for _, v := range s{
        print(v)
    }
}

O parâmetro de tipo t está declarado em minúsculas e, portanto, será relatado.
