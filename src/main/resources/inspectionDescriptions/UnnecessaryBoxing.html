Relata o boxing explícito, ou seja, o empacotamento de valores primitivos em objetos.

No Java 5 e versões posteriores, o boxing manual explícito é desnecessário e pode ser removido com segurança.

Exemplos:

  • Integer i = new Integer(1); → Integer i = Integer.valueOf(1);
  • int i = Integer.valueOf(1); → int i = 1;

Use a opção Relatar apenas expressões de boxing verdadeiramente redundantes para relatar apenas o boxing que for verdadeiramente redundante, onde o valor sofrer unboxing imediato, implícita ou explicitamente. Nesse caso, toda a etapa de boxing-unboxing pode ser removida. Esta inspeção não relata o boxing explícito simples.

Esta inspeção apenas relata se o nível de linguagem do projeto ou módulo for 5 ou superior.
