Relata instruções for aprimoradas.

Exemplo:


  for (int x: Arrays.asList(1, 2, 3)) {
    System.out.println(x);
  }

Após a aplicação da correção rápida:


  for (Iterator<Integer> iterator = Arrays.asList(1, 2, 3).iterator(); iterator.hasNext(); ) {
    final int x = iterator.next();
    System.out.println(x);
  }

A instrução for aprimorada surgiu no Java 5. Esta inspeção ajuda a realizar o downgrade para compatibilidade com versões anteriores do Java.
