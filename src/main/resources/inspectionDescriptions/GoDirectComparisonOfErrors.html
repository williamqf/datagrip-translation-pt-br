Relata a comparação direta de erros, como o uso de ==, e sugere o uso de errors.Is.

A partir do Go 1.13, os erros podem ser envolvidos usando a função fmt.Errorf e o verbo %w. Portanto, a comparação direta de erros usando uma verificação de igualdade falhará para erros envolvidos. A maneira preferencial de verificar erros específicos é usar a função errors.Is da biblioteca padrão, pois esta função percorre a cadeia de erros envolvidos ao verificar um erro específico.

Para obter detalhes sobre o errors.Is, consulte Working with Errors in Go 1.13.

A correção rápida substituirá a comparação direta de erros por uma chamada a errors.Is.

Exemplo:


if err == MyError {
  fmt.Println("MyErr")
}

Após a aplicação da correção rápida:


if errors.Is(err, MyError) {
  fmt.Println("MyErr")
}
