Relata definições incorretas dos seguintes itens relacionados a EJBs de entidade:
  • A classe não implementa as interfaces esperadas (EJBLocalObject, EJBObject, etc.)
  • Modificadores de classe EJB incorretos
  • A classe não define um construtor público sem argumentos
  • Uso de classe em vez de interface ou interface em vez de classe

Exemplo:


public final class SessionEJB implements javax.ejb.SessionBean {
  private SessionEJB() {} // Erro: a classe deve definir um construtor público sem argumentos
}
