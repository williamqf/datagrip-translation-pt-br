Relata operandos de string vazia em concatenações de strings. A concatenação com uma string vazia pode ser usada para converter objetos que não são String ou primitivos em String, mas o uso de uma chamada de método String.valueOf() pode ser mais claro.

Uma correção rápida é sugerida para simplificar a concatenação.

Exemplo:


  void foo(int x, int y) {
    String s = "" + x + " ; " + y;
  }

Após a aplicação da correção rápida:


  void foo(int x, int y) {
    String s = x + " ; " + y;
  }

Use a opção Relatar apenas quando a string vazia puder ser removida sem outras alterações para ignorar situações em que a remoção da string vazia exigiria a adição de uma conversão String.valueOf() de outro operando.
