Relata quaisquer referências a variáveis locais não Serializable externas a lambdas, classes locais e classes anônimas Serializable.

Quando uma variável local é referenciada a partir de uma classe anônima, seu valor é armazenado em um campo implícito dessa classe. O mesmo ocorre para classes locais e lambdas. Se o tipo da variável não for Serializable, a serialização falhará.

Exemplo:


  interface A extends Serializable {
      abstract void foo();
  }
  class B {}
  class C {
      void foo() {
          B b = new B();
          A a = new A() {
              @Override
              public void foo() {
                  System.out.println(b); // aviso
              }
          };
      }
  }
