Reporta verificações null que podem ser substituídas por chamadas de métodos estáticos de Objects ou Stream.

Exemplo:


  if (message == null) {
    application.messageStorage().save(new EmptyMessage());
  } else {
    application.messageStorage().save(message);
  }

Após a aplicação da correção rápida:


  application.messageStorage()
    .save(Objects.requireNonNullElseGet(message, () -> new EmptyMessage()));

Use a opção Não avisar quando a substituição for mais longa que o original para ignorar casos em que o comprimento da substituição exceder o do código original.

Novidade na versão 2017.3
