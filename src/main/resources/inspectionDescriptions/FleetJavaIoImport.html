Relata importações suspeitas dos pacotes Java/Kotlin IO nos módulos das camadas comum/workspace do Fleet.

Os seguintes prefixos de importação são problemáticos:

  • java.io.File
  • java.nio
  • kotlin.io.path

Essas APIs são específicas de plataforma. A menos que você realmente deseje executar algo junto ao processo JVM local (por exemplo, por motivos de desempenho), use abstrações independentes de plataforma por meio da API do Fleet.
