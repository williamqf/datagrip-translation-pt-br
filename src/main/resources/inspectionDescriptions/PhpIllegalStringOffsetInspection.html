Relata o uso de offsets que não sejam integer em expressões de acesso a strings, como $str[42].

A partir do PHP 5.4, os offsets de string devem ser inteiros ou strings semelhantes a inteiros. Como o uso de uma string como offset de string é considerado uma má prática, esta inspeção também relata tais usos.

Para obter mais detalhes, consulte Acesso e modificação de string por caractere (php.net).
