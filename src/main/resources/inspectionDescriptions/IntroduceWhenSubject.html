Relata expressões when que podem ser simplificadas ao introduzir um argumento de sujeito.

Exemplo:


  fun test(obj: Any): String {
      return when {
          obj is String -> "string"
          obj is Int -> "int"
          else -> "unknown"
      }
  }

A correção rápida introduz um argumento de sujeito:


  fun test(obj: Any): String {
      return when (obj) {
          is String -> "string"
          is Int -> "int"
          else -> "unknown"
      }
  }
