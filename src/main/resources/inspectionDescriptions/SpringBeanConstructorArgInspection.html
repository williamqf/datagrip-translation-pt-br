Relata um Spring <bean> que utiliza injeção de dependência baseada em construtor.

A inspeção é acionada quando não é possível encontrar um construtor ou método de fábrica adequado para o <bean> usando as tags <constructor-arg> configuradas e a estratégia autowire definida.

Exemplo:


    public class MyComponent {
    // construtor
    public MyComponent(MyBean bean) {}
    // método de fábrica
    public static MyComponent getInstance(String name, int port) {
    }


    <beans>
      <bean class="beans.MyComponent" id="foo"> <!-- Relata "Não foi possível encontrar um construtor correspondente na classe 'MyComponent'" -->
          <constructor-arg ref="myBean"/>
         <constructor-arg value="123"/>
      </bean>

      <bean class="beans.MyComponent" factory-method="getInstance"> <!-- Relata "Não foi possível encontrar um método de fábrica correspondente na classe" -->
         <constructor-arg value="123"/>
      </bean>
    </beans>
