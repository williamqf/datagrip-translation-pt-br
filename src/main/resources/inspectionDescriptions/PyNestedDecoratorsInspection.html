Relata problemas com decoradores aninhados. Esta inspeção destaca casos em que classmethod ou staticmethod são aplicados antes de outro decorador.

Exemplo:


def innocent(f):
    return f


class A:
    @innocent  # O decorador não receberá o objeto chamável esperado
    @classmethod
    def f2(cls):
        pass

    @innocent  # O decorador não receberá o objeto chamável esperado
    @staticmethod
    def f1():
        pass

A correção rápida sugerida pela IDE é remover este decorador.
