Relata operadores de subtração unários desnecessários. Tais expressões podem ser difíceis de entender e podem conter erros.

Exemplo:

void unaryMinus(int i) {
    int x = - -i;
  }

As seguintes correções rápidas são sugeridas aqui:

  • Remover os operadores - antes da variável i:

    void unaryMinus(int i) {
    int x = i;
  }

  • Substituir os operadores - pelo operador de decremento prefixado:

    void unaryMinus(int i) {
    int x = --i;
  }

Outro exemplo:

void unaryMinus(int i) {
    i += - 8;
  }

Após aplicar a correção rápida:

void unaryMinus(int i) {
    i -= 8;
  }
