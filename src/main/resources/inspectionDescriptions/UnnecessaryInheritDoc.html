Relata comentários Javadoc que contêm apenas a tag {@inheritDoc}. Como o Javadoc copia os comentários da superclasse quando nenhum comentário está presente, um comentário contendo apenas {@inheritDoc} não agrega nada.

Além disso, relata o uso de {@inheritDoc} em locais inválidos (por exemplo, em campos).

Sugere a remoção de comentários Javadoc desnecessários.

Exemplo:


  class Example implements Comparable<Example> {
    /**
     * {@inheritDoc}
     */
    @Override
    public int compareTo(Example o) {
      return 0;
    }
  }

Após a aplicação da correção rápida:


  class Example implements Comparable<Example> {
    @Override
    public int compareTo(Example o) {
      return 0;
    }
  }
