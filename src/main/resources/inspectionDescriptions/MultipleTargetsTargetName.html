Relata o uso da anotação @targetName em múltiplos alvos.

A correção rápida remove a anotação @targetName.

Exemplo:


  // 1
  val list: List[Int] = (1 to 5).toList
  @targetName("head_and_tail")
  val (head, tail) = (list.head, list.tail)

  // 2
  enum E:
    case A
    @targetName("b_and_c")
    case B, C
    case D

Depois da correção rápida ser aplicada:


  // 1
  val list: List[Int] = (1 to 5).toList
  val (head, tail) = (list.head, list.tail)

  // 2
  enum E:
    case A
    case B, C
    case D
