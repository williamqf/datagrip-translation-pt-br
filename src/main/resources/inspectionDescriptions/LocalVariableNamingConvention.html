Relata variáveis locais cujos nomes são muito curtos, muito longos ou não seguem o padrão de expressão regular especificado.

Exemplo: int X = 42; é relatado se esta inspeção estiver ativada com as configurações padrão, nas quais os nomes das variáveis devem começar com uma letra minúscula.

Configurar inspeção:

Use os campos na seção Opções para especificar o comprimento mínimo, o comprimento máximo e a expressão regular esperada para os nomes das variáveis locais. Especifique 0 para não verificar o comprimento do nome. A expressão regular deve ser especificada no formato padrão java.util.regex.

Use as caixas de seleção para ignorar parâmetros de loops for e seções catch.
