Relata casos de acesso impróprio a propriedades:
  • Definição de uma propriedade somente leitura
  • Leitura de uma propriedade somente escrita
  • Exclusão de uma propriedade que não pode ser excluída

Exemplo:


class MyClass:
    @property
    def read_only(self): return None

    def __write_only_setter(self, value): pass

    write_only = property(None, __write_only_setter)


a = MyClass()
a.read_only = 10 # Não é possível definir a propriedade
del a.read_only # Não é possível excluir a propriedade
print(a.write_only) # Não é possível ler a propriedade
