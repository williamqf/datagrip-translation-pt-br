Relata concatenações de String que podem ser simplificadas ao serem substituídas por templates de string.

Exemplo:


  String name = "Bob";
  String greeting = "Hello, " + name + ". You are " + 29 + " years old.";

Após aplicar a correção rápida:


  String name = "Bob";
  String greeting = STR."Hello, \{name}. You are 29 years old.";
