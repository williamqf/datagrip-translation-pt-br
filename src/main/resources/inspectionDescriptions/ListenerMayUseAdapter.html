Relata classes que implementam um listener em vez de estender o adaptador correspondente.

Após substituir a implementação do listener pela extensão do adaptador, um quick fix pode ser usado para remover quaisquer métodos vazios redundantes.

Use a opção Avisar apenas se métodos de implementação vazios forem encontrados para configurar a inspeção para avisar mesmo se nenhum método vazio for encontrado.
