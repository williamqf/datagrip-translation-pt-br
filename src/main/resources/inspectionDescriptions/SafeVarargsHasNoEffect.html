Relata operações de escrita dentro de métodos @SafeVarargs onde o parâmetro vararg é reatribuído a outro objeto. Nesse caso, o array recém-atribuído não está relacionado aos argumentos de aridade variável no local de chamada e pode não ser mais seguro.

Exemplo:


    @SafeVarargs
    public final <T> void test(T[] tt, T... t) {
        t = tt;
        System.out.println(t[0]);
    }
