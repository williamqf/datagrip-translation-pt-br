Relata versões de dependências que foram retiradas (retracted).

Os usuários devem evitar o uso de versões retiradas, pois elas podem conter vulnerabilidades, bugs críticos ou outros problemas potencialmente prejudiciais. O motivo da retirada é explicado no aviso de retirada fornecido.

Para resolver o problema, atualize a dependência para a versão mais recente que não tenha sido retirada. Observe que a retirada não impede você de usar a versão em questão.

As correções rápidas disponíveis podem sugerir a atualização para a versão mais recente ou a leitura do aviso de retirada.

Para mais detalhes, consulte a diretiva retract no go.dev.
Os dados de retirada são fornecidos pelo comando go list -m -u.
