Reporta declarações de equal() com um único parâmetro. Geralmente, trata-se de um erro de digitação onde, na verdade, equals() era o pretendido.

Uma correção rápida para renomear o método para equals é sugerida.

Exemplo:


  class Main {
    public boolean equal(Object obj) {
      return true;
    }
  }

Após a aplicação da correção rápida:


  class Main {
    public boolean equals(Object obj) {
      return true;
    }
  }
