Relata o uso de tags <code> em comentários Javadoc. A partir do Java 5, essas tags podem ser substituídas pela construção {@code ...}. Isso permite o uso de colchetes angulares < e > nos comentários, em vez de entidades de caracteres HTML.

Exemplo:


  /**
   * @return empty <code>ArrayList&lt;Integer&gt;</code>
   */
  List<Integer> getList(){ ... }

Após a aplicação da correção rápida:


  /**
   * @return empty {@code ArrayList<Integer>}
   */
  List<Integer> getList(){ ... }
