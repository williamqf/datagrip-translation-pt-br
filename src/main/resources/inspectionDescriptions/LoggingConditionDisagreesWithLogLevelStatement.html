Relata condições is log enabled for em instruções if que não correspondem ao nível de log das chamadas de log contidas nela.

Exemplo: 


  if (LOG.isTraceEnabled()) {
    // O nível debug foi registrado, mas o nível trace foi verificado
    LOG.debug("some log message");
  }

Esta inspeção reconhece os frameworks de log java.util.logging, Log4j, Log4j2, Apache Commons Logging e SLF4J.
