Denuncia métodos sem cláusulas de parâmetro que sobrescrevem métodos com cláusulas de parâmetro vazias.

A convenção é que um método deve ter parênteses se tiver efeitos colaterais. De acordo com o princípio da substituição de Liskov, quando um método sobrescrito tem uma cláusula de parâmetro, indicando um efeito colateral, o método de sobrescrita também deve ser declarado como um método com efeitos colaterais.

A correção rápida adiciona uma cláusula de parâmetro vazia.

Exemplo:


  class Base {
    def x() = 1
  }
  class Impl extends Base {
    override def x = 2
  }

Depois que as correções rápidas são aplicadas:


  class Base {
    def x() = 1
  }
  class Impl extends Base {
    override def x() = 2
  }
