Relata o uso de membros static não qualificados pelo nome da classe.

Se o membro static estiver na mesma classe, o uso é válido, mas pode ser confuso.

Exemplo:


  class Foo {
    static void foo() {}
    static int x;

    void bar() {
      foo();
      System.out.println(x);
    }

    static void baz() { foo(); }
  }

Após a aplicação da correção rápida:


  class Foo {
    static void foo() {}
    static int x;

    void bar() {
      Foo.foo();
      System.out.println(Foo.x);
    }

    static void baz() { Foo.foo(); }
  }

Use as configurações de inspeção para alternar o relato dos seguintes itens:

  • Acesso a campos static
    void bar() { System.out.println(x); }

  • Chamadas a métodos static
    void bar() { foo(); }
    static void baz() { foo(); }

Você também pode configurar a inspeção para relatar o uso de membros static apenas em contextos não estáticos. No exemplo acima, static void baz() { foo(); } não seria relatado.
