Relata classes EJB que possuem métodos que retornam this ou o recebem como um parâmetro.

Exemplo:


@Stateless
public class ExampleBean {
  private AnotherBean anotherBean;

  public ExampleBean setAnotherBean(AnotherBean anotherBean) {
    this.anotherBean = anotherBean;
    return this; // Erro: não deve retornar `this` aqui
  }
}
