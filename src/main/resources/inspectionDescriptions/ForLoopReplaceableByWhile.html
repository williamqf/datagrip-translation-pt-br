Relata loops for que não contêm componentes de inicialização e atualização e sugere convertê-los em loops while. Isso torna o código mais fácil de ler.

Exemplo:


  for(; exitCondition(); ) {
    process();
  }

Após aplicar a correção rápida:


  while(exitCondition()) {
    process();
  }

A correção rápida também está disponível para outros loops for, portanto, você pode substituir qualquer loop for por um loop while.

Se você deseja ignorar loops for com condições triviais ou inexistentes, use a opção Ignorar loops 'infinite' for sem condições.
