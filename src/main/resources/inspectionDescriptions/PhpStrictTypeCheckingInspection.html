Relata violações das regras de verificação de tipo estrita.

No modo estrito, apenas variáveis que correspondem exatamente ao tipo declarado são aceitas. O modo de verificação de tipo estrita pode ser habilitado das seguintes formas:

  • Para cada arquivo, se a diretiva declare(strict_types=1) for especificada. A diretiva deve ser a primeira instrução no arquivo, e o modo de bloco é explicitamente proibido.
  • Para todos os arquivos, se a opção de inspeção Habilitar para todos os arquivos estiver ativada.

Para mais detalhes, consulte Strict typing (php.net).
