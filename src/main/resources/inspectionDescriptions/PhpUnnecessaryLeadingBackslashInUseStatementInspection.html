Relata instruções use que possuem uma \ inicial desnecessária.

Para nomes com namespace (nomes de namespace totalmente qualificados que contêm um separador de namespace, como Foo\Bar, ao contrário de nomes globais como FooBar), uma barra invertida inicial não é necessária nem recomendada, pois os nomes de importação devem ser totalmente qualificados e não são processados em relação ao namespace atual.

Para mais detalhes, consulte Using namespaces: Aliasing/Importing (php.net).
