Relata parâmetros de tipo declarados para estender uma classe final.

Como as classes final não podem ser estendidas, sugere-se substituir o parâmetro de tipo pelo tipo da classe final especificada.

Exemplo:


  void foo() {
    List<? extends Integer> list; // Aviso: a classe Integer é uma classe final
  }

Após aplicar a correção rápida:


  void foo() {
    List<Integer> list;
  }
