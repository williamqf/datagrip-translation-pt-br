Relata construtores runBlocking alcançáveis a partir de uma corrotina.

As corrotinas compartilham threads para execução. Quando o construtor runBlocking é chamado a partir de uma corrotina, ele bloqueia a thread subjacente e impede que outras corrotinas usem esse recurso. Isso pode causar problemas de desempenho e, em casos graves, privação de threads (thread starvation).

Esta inspeção fornece uma pilha de chamadas potencial de uma primitiva de corrotina (função suspend ou construtor) para um construtor runBlocking.

Exemplo: 


  suspend fun main() {
      foo() 
   }

  fun foo() {
      runBlocking { suspendFunction() }
  }

Solução possível: 


  suspend fun main() {
      foo() 
   }

  suspend fun foo() {
      suspendFunction()
  }

Use a opção Explorar funções com sobrescritas para configurar a análise de funções virtuais:

  • Use a opção Não para pular a análise de funções que possuam sobrescritas.
  • Use a opção Sim, excluindo sobrescritas para explorar apenas funções no tipo do chamado (callee) definido.
  • Use a opção Sim, incluindo sobrescritas para explorar a função e todas as suas sobrescritas.
