Relata propriedades, constantes e métodos declarados sem visibilidade definida.

De acordo com a PSR-12, a visibilidade deve ser definida para:

  • Todas as propriedades,
  • Todas as constantes, se a versão mínima do PHP do seu projeto suportar visibilidade de constante (PHP 7.1 ou superior),
  • Todos os métodos.

Para mais detalhes, consulte PSR-12: Extended Coding Style (php-fig.org).
