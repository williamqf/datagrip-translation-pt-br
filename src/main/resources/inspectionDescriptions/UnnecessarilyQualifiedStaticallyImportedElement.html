Relata o uso de membros importados estaticamente que são qualificados pelo nome da classe que os contém.

Essa qualificação é desnecessária e pode ser removida, pois membros importados estaticamente podem ser acessados diretamente pelo nome do membro.

Exemplo:


  import static foo.Test.WIDTH;

  class Bar {
    void bar() {
      System.out.println(Test.WIDTH);
    }
  }

Após a aplicação do ajuste rápido:


  import static foo.Test.WIDTH;

  class Bar {
    void bar() {
      System.out.println(WIDTH);
    }
  }
