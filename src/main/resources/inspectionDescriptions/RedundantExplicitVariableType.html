Relata tipos de variáveis locais redundantes.

Esses tipos podem ser inferidos a partir do contexto e, portanto, podem ser substituídos por var.

Exemplo:


  void test(InputStream s) {
    try (InputStream in = s) {}
  }

Após a aplicação da correção:


  void test(InputStream s) {
    try (var in = s) {}
  }
