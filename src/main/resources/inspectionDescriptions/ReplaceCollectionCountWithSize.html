Relata chamadas a Collection<T>.count().

Esta chamada de função pode ser substituída por .size.

A forma .size garante que a operação seja O(1) e não aloque objetos extras, enquanto count() pode ser confundido com Iterable<T>.count(), que é O(n) e realiza alocação.

Exemplo:

  fun foo() {
      var list = listOf(1,2,3)
      list.count() // 'count()' substituível
  }

Após a aplicação da correção rápida:

  fun foo() {
      var list = listOf(1,2,3)
      list.size
  }
