Relata o acesso por reflexão a campos e métodos que não existem ou não são visíveis.

Exemplo:


  Field stringHashField() throws NoSuchFieldException {
    return String.class.getField("hash");
  }

Após a aplicação da correção rápida:


  Field stringHashField() throws NoSuchFieldException {
    return String.class.getDeclaredField("hash");
  }

Para classes final, fica claro se existe um campo ou método com o nome especificado na classe.

Para classes não final, pode haver subclasses com campos ou métodos que usam esse nome, portanto, podem ocorrer falsos positivos. Utilize as configurações desta inspeção para eliminar tais falsos positivos em qualquer local ou em classes específicas.
