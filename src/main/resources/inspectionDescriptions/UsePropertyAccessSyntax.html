Relata chamadas de métodos Java get e set que podem ser substituídas por propriedades sintéticas do Kotlin.

A correção rápida Usar sintaxe de acesso à propriedade pode ser usada para corrigir automaticamente o código.

Exemplo:


  // Java:
  public class JavaClassWithGetter {
      private final String expr = "result";

      // ...

      public String getExpr() {
          return expr;
      }
  }


  // Kotlin:
  fun test(j: JavaClassWithGetter) {
      // ...
      j.getExpr() // <== a correção rápida simplifica a expressão para 'j.expr'
  }
