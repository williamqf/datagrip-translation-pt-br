Reporta o uso de APIs da plataforma IntelliJ introduzidas em versões posteriores à versão especificada no atributo @since-build de <idea-version> no plugin.xml.

O uso dessas APIs pode tornar o plugin incompatível com versões mais antigas do IDE.

Para evitar possíveis problemas ao executar o plugin em versões mais antigas do IDE, aumente o since-build adequadamente ou remova o uso desta API.

Consulte a seção Build Number Ranges na documentação do SDK de plugins da plataforma IntelliJ para obter mais detalhes.

Configuração da inspeção:

Se os atributos @since/until-build de <idea-version> não estiverem especificados no plugin.xml, defina explicitamente Since/Until.
