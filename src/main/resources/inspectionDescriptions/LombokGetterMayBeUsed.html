Relata métodos getter padrão que podem ser substituídos pela anotação @Getter do lombok.

Exemplo:


  import java.util.Date;

  public class MyClass {
    /**
     * Data de processamento.
     */
    private Date processDate;

    /**
     * Retorna a data.
     *
     * @return A data
     */
    public Date getProcessDate() {
      return processDate;
    }
  }

Após a aplicação da correção rápida/limpeza:


  import lombok.Getter;
  import java.util.Date;

  @Getter
  public class MyClass {
    /**
     * Data de processamento.
     * -- GETTER --
     * Retorna a data.
     *
     * @return A data
     */
    private Date processDate;
  }

Relata apenas se a biblioteca lombok estiver configurada. Para remover o lombok, consulte o site do lombok.
