Detecta referências a provedores de chaves de criptografia inexistentes em configurações de blocos encryption do OpenTofu. Por exemplo, para o seguinte código: 

  
    terraform {
      encryption {
        key_provider "pbkdf2" "foo" {
          # Parâmetros de configuração
        }
        method "aes_gcm" "yourname" {
          keys = key_provider.pbkdf2.unknown
        }
        method "aes_gcm" "yourname2" {
          keys = key_provider.pbkdf2.foo
        }
      }
    }
Devemos verificar se a referência do provedor de chaves está correta. Neste caso, a referência unknown deve ser destacada.

Para mais detalhes, consulte https://opentofu.org/docs/language/state/encryption/#methods.
