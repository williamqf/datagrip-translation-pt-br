Relata métodos getter ou setter que acessam um campo cujo nome não é o esperado para o nome do método. Por exemplo, quando getY() retorna o campo x. Isso geralmente pode indicar um erro de copiar e colar.

Exemplo:

class Point {
    private int x;
    private int y;

    public void setX(int x) { // Aviso: o setter 'setX()' atribui ao campo 'y'
      this.y = x;
    }

    public int getY() { // Aviso: o getter 'getY()' retorna o campo 'x'
      return x;
    }
  }
