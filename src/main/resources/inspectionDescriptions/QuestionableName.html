Relata variáveis, métodos ou classes com nomes suspeitos em vez de nomes verdadeiramente descritivos. Tais nomes não auxiliam na compreensão do código e provavelmente foram criados de forma temporária e posteriormente esquecidos.

Exemplo:


  int aa = 42;

A correção rápida para renomear é sugerida apenas no editor.

Use a opção para listar os nomes que devem ser relatados.
