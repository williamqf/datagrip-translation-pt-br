Relata comentários que não começam com o nome do elemento exportado.

De acordo com Comment Sentences em github.com/golang, é uma convenção colocar o nome do elemento exportado no início do comentário.

Exemplo:

// Representa uma requisição para executar um comando.
type Request struct {}

O comentário começa com uma descrição da struct em vez do nome da struct. Para seguir a regra da convenção, você pode aplicar a correção rápida Adicionar prefixo ao comentário. Após a aplicação da correção rápida, o comentário será semelhante a este:

// Request representa uma requisição para executar um comando.
type Request struct {} // melhor
