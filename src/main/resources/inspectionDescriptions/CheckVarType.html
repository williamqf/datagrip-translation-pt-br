Relata variáveis sem um tipo especificado

Quando o código não consegue resolver o tipo contido na variável, ele deve ser especificado manualmente.

Exemplo:


  var int_val = 1
  val string_val = ""

Após a aplicação da correção rápida:


  var int_val: int = 1
  val string_val: String = ""

Ou quando a sintaxe curta é utilizada:


  var int_val := 1
  val string_val := ""
