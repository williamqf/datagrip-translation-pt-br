Relata referências às classes NilClass, TrueClass e FalseClass em assinaturas de tipo. O uso dessas classes é desnecessariamente verboso.

Uma correção rápida pode ser usada para substituir o uso de NilClass, TrueClass ou FalseClass por seus respectivos literais.

Exemplo:

# 'NilClass' é referenciado em vez de 'nil', ...
type example = NilClass | TrueClass | FalseClass

Após a aplicação da correção rápida:

type example = nil | true | false
