Relata classes anônimas.

Substituir ocasionalmente classes anônimas por classes internas pode melhorar a legibilidade e a manutenibilidade do código. Alguns padrões de código desencorajam o uso de classes anônimas.

Exemplo:


  class Example {
    public static void main(String[] args) {
      new Thread() {
        public void run() {
          work()
        }

        private void work() {}
      }.start();
    }
  }

Após aplicar a correção rápida:


  class Example {
    public static void main(String[] args) {
      new MyThread().start();
    }

    private static class MyThread extends Thread {
      public void run() {
        work();
      }

      private void work() {}
    }
  }
