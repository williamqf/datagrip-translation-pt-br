Relata pontos e vírgula redundantes (;) que podem ser removidos com segurança.

O Kotlin não exige o uso de ponto e vírgula no final de cada instrução ou expressão. É sugerida uma correção rápida para remover os pontos e vírgula redundantes.

Exemplo:


  val myMap = mapOf("one" to 1, "two" to 2);
  myMap.forEach { (key, value) ->  print("$key -> $value")};

Após a aplicação da correção rápida:


  val myMap = mapOf("one" to 1, "two" to 2)
  myMap.forEach { (key, value) ->  print("$key -> $value")}

Existem dois casos em que o ponto e vírgula é necessário:

  1. Várias instruções em uma única linha devem ser separadas por ponto e vírgula: 
    
  map.forEach { val (key, value) = it; println("$key -> $value") }
  2. Uma classe enum que também declara propriedades ou funções exige um ponto e vírgula após a lista de constantes de enumeração: 
    
  enum class Mode {
      SILENT, VERBOSE;

      fun isSilent(): Boolean = this == SILENT
  }
