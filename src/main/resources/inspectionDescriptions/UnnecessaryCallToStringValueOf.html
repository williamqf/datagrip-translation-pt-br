Relata chamadas desnecessárias a métodos static que convertem seus parâmetros em strings (por exemplo, String.valueOf() ou Integer.toString()). Tais chamadas são desnecessárias quando usadas em concatenação de strings.

Exemplo:


  System.out.println("Number: " + Integer.toString(count));

Após aplicar a correção rápida:


  System.out.println("Number: " + count);
Além disso, tais chamadas são desnecessárias quando usadas como argumentos para métodos de biblioteca que realizam sua própria conversão de string. Alguns exemplos de métodos de biblioteca que realizam sua própria conversão de string são:
  • Classes java.io.PrintWriter, java.io.PrintStream
    • print(), println()
  • Classes java.lang.StringBuilder, java.lang.StringBuffer
    • append()
  • Classe org.slf4j.Logger
    • trace(), debug(), info(), warn(), error()

Use a opção Relatar chamadas que podem ser substituídas por concatenação com uma string vazia para também relatar situações onde a concatenação com uma string vazia pode ser usada em vez de uma chamada a String.valueOf().
