Relata instruções que não terminam com ponto e vírgula ou quebra de linha.

Selecione a opção "Terminar instruções com ponto e vírgula" em Editor | Estilo de código | JavaScript ou TypeScript - Pontuação para relatar todas as instruções que não terminam com ponto e vírgula, mesmo quando uma quebra de linha é utilizada. De acordo com alguns estilos de codificação, o uso de pontos e vírgulas é preferível a quebras de linha para manter a consistência com outras linguagens.
