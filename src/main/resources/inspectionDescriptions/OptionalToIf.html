Relata cadeias de chamadas Optional que podem ser substituídas por uma sequência de instruções if.

Exemplo:


  return Optional.ofNullable(name)
    .map(this::extractInitials)
    .map(initials -> initials.toUpperCase(Locale.ENGLISH))
    .orElseGet(this::getDefault);

Após a aplicação da correção rápida:


  if (name != null) {
    String initials = extractInitials(name);
    if (initials != null) return initials.toUpperCase(Locale.ENGLISH);
  }
  return getDefault();

Esta inspeção é útil para realizar o downgrade visando a retrocompatibilidade com versões anteriores do Java.
