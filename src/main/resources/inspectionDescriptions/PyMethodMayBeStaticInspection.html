Relata qualquer método que não exija a criação de uma instância da classe e que possa ser definido como estático.

Exemplo:


class MyClass(object):
    def my_method(self, x):
        print(x)

Se a correção rápida Criar função a partir de método for aplicada, o código mudará para:


def my_method(x):
    print(x)


class MyClass(object):
    pass

Se a correção rápida Tornar o método estático for selecionada, o decorador @staticmethod será adicionado:


class MyClass(object):
    @staticmethod
    def my_method(x):
        print(x)
