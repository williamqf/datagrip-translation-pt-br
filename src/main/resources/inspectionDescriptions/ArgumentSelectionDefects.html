Relata incompatibilidades entre os nomes dos identificadores dos argumentos e os nomes dos parâmetros formais. Isso ajuda a detectar erros de programação, por exemplo, quando você troca acidentalmente argumentos do mesmo tipo. O artigo de pesquisa original pode ser encontrado aqui.

Exemplo:


  const User* get_User(int64_t company_id, int64_t user_id) {
    return new User{company_id, user_id, "foo"};
  }

  void call_User(int64_t company_id, int64_t user_id) {
    const User* user = get_User(user_id, company_id); // Problema potencial aqui
    //...
  }
