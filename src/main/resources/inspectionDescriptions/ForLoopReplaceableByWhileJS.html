Relata loops for que não contêm componentes de inicialização nem de atualização. Sugere-se substituir o loop por uma instrução while mais simples.

Exemplo:


  for(; exitCondition(); ) {
    process();
  }

Após a aplicação da correção rápida, o resultado será o seguinte:


  while(exitCondition()) {
    process();
  }

Utilize a caixa de seleção abaixo se desejar que esta inspeção ignore loops for com condições triviais ou inexistentes.
