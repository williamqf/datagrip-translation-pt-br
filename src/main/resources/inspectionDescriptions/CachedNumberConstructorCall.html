Relata a instanciação de novos objetos Long, Integer, Short ou Byte com argumentos primitivos long, integer, short ou byte.

Recomenda-se o uso do método estático valueOf() introduzido no Java 5. Por padrão, este método armazena em cache objetos com valores entre -128 e 127 (inclusive).

Exemplo:


  Integer i = new Integer(1);
  Long l = new Long(1L);

Após a aplicação da correção rápida, o código torna-se:


  Integer i = Integer.valueOf(1);
  Long l = Long.valueOf(1L);

Esta inspeção relata apenas se o nível de linguagem do projeto ou módulo for 5 ou superior.

Use a opção Ignorar novas expressões numéricas com um argumento String para ignorar chamadas a construtores numéricos que contenham um argumento String.

Use a opção Relatar apenas quando o construtor for @Deprecated para relatar apenas chamadas a construtores depreciados. Os construtores de Long, Integer, Short e Byte foram depreciados desde o JDK 9.
