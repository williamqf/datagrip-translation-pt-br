Relata chamadas toString() em objetos de classes que estendem Number. Tais chamadas são geralmente incorretas em contextos de internacionalização, e alguma formatação específica de localidade deve ser usada em seu lugar.

Exemplo:


  void print(Double d) {
    System.out.println(d.toString());
  }
Uma maneira possível de corrigir esse problema é: 

  void print(Double d) {
    System.out.printf("%f%n", d);
  }
Isso formatará o número usando a localidade padrão, que é definida durante a inicialização da JVM com base no ambiente do host.
