Relata chamadas equals() em constantes de enumeração.

Tais chamadas podem ser substituídas por uma comparação de identidade (==), pois duas constantes de enumeração são iguais apenas se possuírem a mesma identidade.

Uma correção rápida pode ser usada para alterar a chamada para uma comparação.

Exemplo:


  boolean foo(MyEnum value) {
    return value.equals(MyEnum.FOO);
  }

Após a aplicação da correção rápida:


  boolean foo(MyEnum value) {
    return value == MyEnum.FOO;
  }
