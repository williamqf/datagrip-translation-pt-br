Relata parâmetros de método que possuem sempre o mesmo valor constante.

Exemplo:


  static void printPoint(int x, int y) { // x é sempre 0
    System.out.println(x + ", " + y);
  }

  public static void main(String[] args) {
    printPoint(0, 1);
    printPoint(0, 2);
  }

A correção rápida faz o inline do valor constante. Isso pode simplificar a implementação do método.

Use a opção Ignorar quando nenhuma correção rápida estiver disponível para suprimir a inspeção nos seguintes casos:

  • O parâmetro é modificado dentro do método
  • O valor do parâmetro passado é uma referência a um campo inacessível (apenas Java)
  • O parâmetro é um vararg (apenas Java)

Use a opção Visibilidade máxima do método para controlar a visibilidade máxima dos métodos a serem relatados.

Use o campo Contagem mínima de usos do método para relatar o parâmetro para especificar o número mínimo de usos do método com o mesmo valor de parâmetro.
