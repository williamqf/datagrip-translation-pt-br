Relata assinaturas incompatíveis para os métodos __new__ e __init__.

Exemplo:


class MyClass(object):
    def __new__(cls, arg1):
        return super().__new__(cls)

    def __init__(self):
        pass

Se __new__ e __init__ tiverem argumentos diferentes, não será possível instanciar MyClass.

O IDE sugere a aplicação da refatoração "Alterar Assinatura" como uma correção.
