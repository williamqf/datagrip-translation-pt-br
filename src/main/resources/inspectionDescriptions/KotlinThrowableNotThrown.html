Relata a instanciação de Throwable ou de suas subclasses quando o Throwable criado nunca é realmente lançado.

O código relatado indica erros que são difíceis de capturar em testes.

Além disso, esta inspeção relata chamadas de métodos que retornam uma instância de Throwable ou de suas subclasses quando a instância de Throwable resultante não é lançada.

Exemplo:


  fun check(condition: Boolean) {
      if (!condition) /* o 'throw' está ausente aqui */ IllegalArgumentException("condition is not met");
  }

  fun createError() = RuntimeException()

  fun foo() {
      /* o 'throw' está ausente aqui */ createError()
  }
