Relata o uso de tipos que foram descontinuados e podem desaparecer em versões futuras do DBMS.

Tipos relatados:

  • LONG no Oracle (consulte a página Deprecated and Desupported Features no docs.oracle.com).
  • TEXT, NTEXT e IMAGE no Microsoft SQL Server (consulte a página Deprecated Database Engine Features in SQL Server 2016 no docs.microsoft.com).

Exemplo (Oracle):

CREATE TABLE ot.foo(
a NUMBER GENERATED BY DEFAULT AS IDENTITY,
b LONG NOT NULL
);
