Relata comparações em que os operandos esquerdo e direito representam a mesma expressão. Embora às vezes comparar uma expressão consigo mesma possa ser intencional, geralmente é fruto de um descuido.

Exemplo:


  // possivelmente left.getLength() == right.getLength() era o pretendido
  boolean result = left.getLength() == left.getLength();

Para ignorar comparações que podem produzir efeitos colaterais, use a opção Ignorar condições com efeitos colaterais. Desativar esta opção pode levar a falsos positivos, por exemplo, quando o mesmo método retorna valores diferentes em chamadas subsequentes.

Exemplo:


  native int unknownMethod();
  
  ...
  
  if (unknownMethod() > unknownMethod()) {
    System.out.println("Got it");
  }

Devido aos potenciais efeitos colaterais em unknownMethod() (no exemplo), o aviso só será acionado se a opção Ignorar condições com efeitos colaterais estiver desativada.
