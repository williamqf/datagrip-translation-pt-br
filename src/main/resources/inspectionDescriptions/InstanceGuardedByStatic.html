Relata anotações @GuardedBy em campos ou métodos de instância onde a proteção é um campo static. Proteger um membro não static com um bloqueio static pode levar a uma contenção excessiva de bloqueio, pois o acesso ao campo bloqueado em qualquer instância de objeto impedirá o acesso simultâneo ao campo em todas as instâncias do objeto.

Exemplo:


    private static ReadWriteLock lock = new ReentrantReadWriteLock(); // campo de proteção static
    private Object state;

    @GuardedBy("lock")
    public void bar() {
        state = new Object();
    }

Anotações @GuardedBy suportadas:

  • net.jcip.annotations.GuardedBy
  • javax.annotation.concurrent.GuardedBy
  • org.apache.http.annotation.GuardedBy
  • com.android.annotations.concurrency.GuardedBy
  • androidx.annotation.GuardedBy
  • com.google.errorprone.annotations.concurrent.GuardedBy
