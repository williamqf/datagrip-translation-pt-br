Relata chamadas a Objects.equals(a, b) onde o primeiro argumento é estaticamente conhecido como não sendo nulo.

Tais chamadas podem ser substituídas com segurança por a.equals(b) ou a == b se ambos os argumentos forem tipos primitivos.

Exemplo:


  String defaultName = "default";
  boolean isDefault = Objects.equals(defaultName, name);

Após a aplicação da correção rápida:


  String defaultName = "default";
  boolean isDefault = defaultName.equals(name);

Novidade na versão 2018.3
