Relata métodos não booleanos cujos nomes começam com uma palavra interrogativa. Esses nomes de métodos podem ser confusos.

Métodos não booleanos que sobrescrevem métodos de biblioteca são ignorados por esta inspeção.

Exemplo: 


  public void hasName(String name) {
    assert names.contains(name);
  }

A correção rápida para renomear esses métodos está disponível apenas no editor.

Configuração da inspeção:

  • Use a lista de Prefixos de nomes de métodos booleanos para especificar as palavras interrogativas que devem ser usadas apenas para métodos booleanos.
  • Use a opção Ignorar métodos com tipo de retorno 'java.lang.Boolean' para ignorar métodos com o tipo de retorno java.lang.Boolean.
  • Use a opção Ignorar métodos que sobrescrevem/implementam métodos super para ignorar métodos que possuem um supermétodo.
