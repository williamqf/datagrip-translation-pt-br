Reporta variáveis static com nomes muito curtos, muito longos ou que não seguem o padrão de expressão regular especificado.

Esta inspeção não verifica constantes (ou seja, variáveis de tipos imutáveis declaradas como static final).

Use os campos fornecidos abaixo para especificar o comprimento mínimo, o comprimento máximo e a expressão regular esperados para os nomes de variáveis static.

A expressão regular deve ser especificada no formato padrão java.util.regex.
