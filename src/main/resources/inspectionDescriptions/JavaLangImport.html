Relata instruções import que referenciam o pacote java.lang.

As classes do pacote java.lang são sempre importadas implicitamente, portanto, tais instruções import são redundantes e confusas.

O IntelliJ IDEA pode detectar e corrigir automaticamente essas instruções por meio do comando Otimizar importações; essa inspeção é útil principalmente para relatórios offline de bases de código que não se pretende alterar.
