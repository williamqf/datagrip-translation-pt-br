Relata variáveis locais com parâmetros de tipo.

Um parâmetro de tipo para uma variável local não faz sentido, pois não pode ser especializado.

Exemplo:


  fun main() {
      val <T> x = ""
  }

Após a aplicação da correção rápida:


  fun main() {
      val x = ""
  }

Esta inspeção apenas relata se o nível de linguagem Kotlin do projeto ou módulo for 1.4 ou superior.
