Relata expressões do tipo char usadas em expressões de adição ou subtração.

Esse tipo de código não é necessariamente um problema, mas pode causar bugs (por exemplo, se uma string for esperada).

Exemplo: int a = 'a' + 42;

Após a aplicação da correção rápida: int a = (int) 'a' + 42;

Para o contexto de String: 

int i1 = 1;
int i2 = 2;
System.out.println(i2 + '-' + i1 + " = " + (i2 - i1));

Após aplicar a correção rápida:

System.out.println(i2 + "-" + i1 + " = " + (i2 - i1));
