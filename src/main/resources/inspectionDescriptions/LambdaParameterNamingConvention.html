Relata parâmetros lambda cujos nomes sejam muito curtos, muito longos ou não sigam o padrão de expressão regular especificado.

Exemplo: Function<String, String> id = X -> X; seria relatado se a inspeção estivesse ativada com as configurações padrão (onde os nomes de parâmetros devem começar com uma letra minúscula).

Configurar inspeção:

Use os campos na seção Opções para especificar o comprimento mínimo, o comprimento máximo e a expressão regular esperada para os nomes de parâmetros lambda. Especifique 0 para não verificar o comprimento do nome.

A expressão regular deve ser especificada no formato padrão java.util.regex.
