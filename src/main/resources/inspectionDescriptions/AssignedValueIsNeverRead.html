Relata atribuições em que o valor atribuído nunca é usado no código subsequente.

Exemplo:


  fun foo(): Int = 42

  fun example() {
      var local = 0
      print(local)
      local = 42  // O valor atribuído nunca é lido
  }
  • Após a correção rápida, a atribuição é excluída:

  fun foo(): Int = 42

  fun example() {
      var local = 0
      print(local)
  }
  • Se a expressão do lado direito (RHS) puder ter efeitos colaterais, a correção rápida também sugerirá manter a expressão RHS para preservar esses efeitos colaterais: 

  fun foo(): Int = 42

  fun example() {
      var local = 0
      print(local)
      foo()  // Esta chamada de função é mantida devido aos seus efeitos colaterais
  }
