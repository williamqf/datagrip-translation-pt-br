Informa o uso inválido da anotação ClassVar.

Exemplo:


from typing import ClassVar


class Cat:
    color: ClassVar[str] = "white"
    weight: int

    def __init__(self, weight: int):
        self.weight = weight


Cat.color = "black"  # OK
my_cat = Cat(5)
my_cat.color = "gray"  # Erro, definindo variável de classe em uma instância
