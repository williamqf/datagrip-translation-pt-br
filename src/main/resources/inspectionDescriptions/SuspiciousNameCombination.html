Reporta atribuições e chamadas de função em que o nome da variável de destino ou do parâmetro da função não corresponde ao nome do valor atribuído a ela.

Exemplo 1:


  int x = 0;
  int y = x; // x é usado como coordenada y

Exemplo 2:


  int x = 0, y = 0;
  // x é usado como coordenada y, y é usado como coordenada x
  Rectangle rc = new Rectangle(y, x, 20, 20);

Configuração da inspeção:

Use a área Grupos de nomes para especificar nomes que não devem ser usados juntos: se o nome do parâmetro ou o nome do destino da atribuição contiver uma palavra de um grupo e o nome da variável atribuída ou passada contiver uma palavra de outro grupo, um erro será relatado.

Use a área Ignorar métodos para especificar métodos que não devem ser verificados, mas que podem ter nomes suspeitos. Por exemplo, os parâmetros de Integer.compare() são nomeados como x e y, mas não têm relação com coordenadas.
