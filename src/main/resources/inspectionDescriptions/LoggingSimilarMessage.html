Relata chamadas de log do SLF4J ou Log4j2 em uma classe que possuam mensagens de log semelhantes, como logger.info("message: {}", key). Pode ser impossível distinguir entre essas chamadas, o que dificulta entender de onde vem uma determinada mensagem de log.

Exemplo (Java):


  private static void request1(String text) {
      log.info("Message: {}", text); // chamada semelhante
      doSomething1();
  }

  private static void request2(int i) {
      log.info("Message: {}", i); // chamada semelhante
      doSomething2();
  }
  • Use a opção Comprimento mínimo da sequência semelhante para definir o comprimento mínimo de uma sequência semelhante a partir do qual as chamadas serão relatadas.
  • Use a opção Não relatar chamadas no nível de log 'error' para ignorar mensagens quando o nível de log for `error` e houver uma exceção presente. Ocultar o aviso pode ser útil, pois o local da chamada ainda pode ser localizado usando o stack trace.
