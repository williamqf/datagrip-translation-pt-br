Relata chamadas a Math.random() que são convertidas imediatamente para int via cast.

A conversão de um double entre 0.0 (inclusive) e 1.0 (exclusivo) para int sempre resultará em zero. Antes de converter o valor para int, ele deve ser multiplicado por algum fator para obter um valor entre zero (inclusive) e o fator de multiplicação (exclusivo). Outra solução possível é usar o método nextInt() de java.util.Random.

Exemplo:

  int r = (int)Math.random() * 10;

Após a aplicação da correção rápida:

  int r = (int)(Math.random() * 10);
