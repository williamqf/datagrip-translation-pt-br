Relata loops while que podem ser escritos de forma mais eficiente como loops do-while. Existem loops while em que o código imediatamente antes do loop é idêntico ao código dentro do corpo do loop. A substituição por um loop do-while remove o código duplicado. Uma correção rápida também é oferecida no editor para loops while sem esse código duplicado, mas sem realce.

Exemplo:


  foo();
  while (x) {
      foo();
  }

Pode ser substituído por:


  do {
    foo();
  } while (x);
