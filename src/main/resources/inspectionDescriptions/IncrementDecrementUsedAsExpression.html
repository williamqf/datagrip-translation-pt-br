Relata expressões de incremento ou decremento aninhadas em outras expressões. Tais expressões podem ser confusas e violam o princípio de design geral de que cada construção deve realizar apenas uma tarefa.

A correção rápida extrai a operação de incremento ou decremento em uma instrução de expressão separada.

Exemplo:


  int i = 10;
  while (i-- > 0) {
    System.out.println(i);
  }

Após a aplicação da correção rápida:


  int i = 10;
  while (i > 0) {
    i--;
    System.out.println(i);
  }
  i--;
