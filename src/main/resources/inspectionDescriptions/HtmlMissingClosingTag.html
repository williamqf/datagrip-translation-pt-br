Reporta elementos HTML que não possuem uma tag de fechamento. Alguns estilos de codificação exigem que os elementos HTML contenham tags de fechamento, mesmo onde elas são opcionais.

Exemplo:


  <html>
    <body>
      <p>Behold!
    </body>
  </html>

Após a aplicação do ajuste rápido:


  <html>
    <body>
      <p>Behold!</p>
    </body>
  </html>
