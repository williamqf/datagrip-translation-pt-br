Relata chamadas de método que são usadas como argumentos para outra chamada de método.

Essa correção rápida introduz uma variável, tornando o código mais simples e fácil de depurar.

Exemplo:


  public int y() { return 1; }
  public int f(int x) { return 2 * x; }

  public void foo() {
      int x = f(y());
  }

Após a aplicação da correção rápida:


  public int y() { return 1; }
  public int f(int x) { return 2 * x; }

  public void foo() {
      int y = y();
      int x = f(y);
  }

Use as opções de inspeção para alternar os relatos para:

  • Chamadas de método em inicializadores de campo
  • Chamadas para métodos estáticos (static)
  • Chamadas para getters simples
