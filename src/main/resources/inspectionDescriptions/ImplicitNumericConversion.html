Relata conversões implícitas entre tipos numéricos.

Conversões numéricas implícitas não são um problema por si só, mas se forem inesperadas, podem dificultar o rastreamento de erros.

Exemplo:


  double m(int i) {
    return i * 10;
  }

Após aplicar a correção rápida:


  double m(int i) {
    return (double) (i * 10);
  }

Configuração da inspeção:

  • Use a opção Ignorar conversões de alargamento para ignorar conversões implícitas que não podem causar perda de dados (por exemplo, int->long).
  • Use a opção Ignorar conversões de/para 'char' para ignorar conversões entre char e outros tipos. A inspeção ainda relatará conversões entre números de ponto flutuante e outros tipos.
  • Use Ignorar conversões de constantes e literais para fazer com que a inspeção ignore conversões a partir de literais e constantes de tempo de compilação.
