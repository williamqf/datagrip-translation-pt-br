Relata o uso inválido da sintaxe de parâmetros de tipo da PEP 695.

Procura os seguintes problemas em definições de funções e classes, bem como em instruções de alias de tipo de novo estilo:

  • Extensão de typing.Generic em classes genéricas de novo estilo
  • Extensão de typing.Protocol parametrizado em classes genéricas de novo estilo
  • Uso de limites superiores e restrições genéricas com parâmetros de tipo para ParamSpec e TypeVarTuple
  • Mistura de variáveis de tipo tradicionais e de novo estilo
  • Uso de variáveis de tipo tradicionais em aliases de tipo de novo estilo

Exemplos: 


  from typing import Generic

  class Example[T](Generic[T]): ... # Uma classe com uma lista de parâmetros de tipo não deve estender 'Generic'


  class Example[T: (list[S], str)]: ... # Tipos genéricos não são permitidos dentro de restrições e limites de parâmetros de tipo


  from typing import TypeVar

  K = TypeVar("K")

  class ClassC[V]:
      def method2[M](self, a: M, b: K) -> M | K: ... # Não é permitido misturar TypeVars tradicionais e de novo estilo
