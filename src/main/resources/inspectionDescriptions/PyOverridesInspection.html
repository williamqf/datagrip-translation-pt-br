Reporta quando um método decorado com @override não possui um método correspondente em suas superclasses.

Exemplo:


from typing import override

class Parent:
    def foo(self) -> int:
        return 1

    def bar(self, x: str) -> str:
        return x

class Child(Parent):
    @override
    def foo(self) -> int:
        return 2

    @override # método da superclasse ausente para a função de sobrescrita
    def baz(self) -> int:
        return 1
