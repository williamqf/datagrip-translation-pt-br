Relata expressões de bloqueio de instrução synchronized compostas por referências a campos não final. Tais instruções dificilmente possuem uma semântica útil, pois diferentes threads podem obter bloqueios diferentes, mesmo ao operar no mesmo objeto.

Exemplo:


  private Object o;
  public void foo() {
    synchronized (o) // Sincronização em um campo não final
    { }
  }
