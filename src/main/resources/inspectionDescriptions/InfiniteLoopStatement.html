Relata instruções for, while ou do que só podem ser encerradas lançando uma exceção. Embora tais instruções possam estar corretas, elas geralmente são resultantes de erros de codificação.

Exemplo:


  for (;;) {
  }

Use a opção Ignorar quando colocado em Thread.run para ignorar instruções de loop infinito em Thread.run. Isso pode ser útil para threads daemon.

Exemplo:


  new Thread(() -> {
    while (true) {
    }
  }).start();
