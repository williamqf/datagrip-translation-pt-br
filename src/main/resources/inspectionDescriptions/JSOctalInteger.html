Relata literais inteiros octais depreciados que usam o prefixo 0 em vez de 0o.
Esses literais não são permitidos no código ECMAScript moderno e causam erros no modo strict.
Para aplicar esta inspeção nos níveis de linguagem ES5 e ES3, marque a caixa de seleção "Avisar sobre literais octais obsoletos no código ES5" abaixo.
