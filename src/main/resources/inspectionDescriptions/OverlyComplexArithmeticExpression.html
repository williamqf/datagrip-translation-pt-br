Relata expressões aritméticas com excesso de termos. Tais expressões podem ser difíceis de entender e podem conter erros.

Parâmetros, referências de campo e outras expressões primárias contam como um termo.

Exemplo: 

int calc(int a, int b) {
        return a + a + a + b + b + b + b; // Esta linha contém 7 termos e será relatada.
    }

Use o campo abaixo para especificar o número de termos permitidos em uma expressão aritmética.
