Relata expressões when com ramos constantes true ou false.

A correção rápida Simplificar 'when' pode ser usada para corrigir o código automaticamente.

Exemplo:


  fun redundant() {
      when { // <== redundante; a correção rápida simplificará a expressão when para "println("true")"
          true -> println("true")
          else -> println("false")
      }
  }
