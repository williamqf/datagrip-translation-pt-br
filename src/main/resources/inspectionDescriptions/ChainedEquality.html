Reporta comparações de igualdade em cadeia.

Tais comparações podem ser confusas: a == b == c significa (a == b) == c, mas é possível que a == b && a == c fosse a intenção original.

Exemplo:


  boolean chainedEquality(boolean a, boolean b, boolean c) {
    return a == b == c;
  }

Você pode usar parênteses para tornar a comparação menos confusa:


  boolean chainedEquality(boolean a, boolean b, boolean c) {
    return (a == b) == c;
  }
