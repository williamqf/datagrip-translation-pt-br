Relata chamadas da função StringBuilder.append(CharArray, offset, len) na plataforma JVM que devem ser substituídas por chamadas da função StringBuilder.appendRange(CharArray, startIndex, endIndex).

A função append comporta-se de forma diferente nas plataformas JVM, JS e Native, portanto, recomenda-se o uso da função appendRange.

Exemplo:


  fun f(charArray: CharArray, offset: Int, len: Int): String {
      return buildString {
          append(charArray, offset, len)
      }
  }

Após a aplicação do quick fix:


  fun f(charArray: CharArray, offset: Int, len: Int): String {
      return buildString {
          appendRange(charArray, offset, offset + len)
      }
  }
