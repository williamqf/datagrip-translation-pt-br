Relata chamadas de asserção non-null (!!) que podem ser substituídas pelo operador Elvis e return (?: return).

As asserções non-null podem levar a exceções NPE (NullPointerException) inesperadas. É uma boa prática evitar o uso de !!.

A correção rápida substituirá a asserção non-null por return ou return null.

Exemplo:


  fun test(number: Int?) {
      val x = number!!
  }

Após aplicar a correção rápida:


  fun test(number: Int?) {
      val x = number ?: return
  }
