Reporta chamadas a Condition.signal() ou Condition.signalAll() para as quais nenhuma chamada correspondente a Condition.await() pode ser encontrada.

Esta inspeção reporta apenas chamadas que visam campos da classe atual.

Exemplo:


  class Queue {
    private final Condition isEmpty = ...;

    void add(Object elem) {
      // ...
      isEmpty.signal(); // Alerta: chamada a 'signal()' sem um 'await()' correspondente
      // ...
    }

    void remove(Object elem) throws InterruptedException {
      // ...
      //      isEmpty.await();
      // ...
    }
  }
