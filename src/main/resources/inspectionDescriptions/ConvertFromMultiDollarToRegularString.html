Reporta todas as strings com prefixos de interpolação.

A correção rápida da inspeção remove o prefixo de interpolação, atualiza as entradas de interpolação e escapa caracteres de cifrão inseguros.

Exemplo:


val foo = $$"""
{
    "foo": $ref,
    "bar": {
        "baz": "$20"
        "boo": "$${100 + 200}"
    }
}
"""

Após aplicar a correção rápida:


val foo = """
{
    "foo": ${'$'}ref,
    "bar": {
        "baz": "$20"
        "boo": "${100 + 200}"
    }
}
"""
