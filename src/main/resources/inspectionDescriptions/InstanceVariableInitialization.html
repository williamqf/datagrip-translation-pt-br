Relata variáveis de instância que podem não ter sido inicializadas durante a inicialização do objeto.

Exemplo:


  class Foo {
    public int bar;

    static { }
  }

Observe que esta inspeção utiliza um algoritmo de fluxo de dados muito conservador e pode relatar incorretamente variáveis de instância como não inicializadas. As variáveis relatadas como inicializadas serão sempre inicializadas.

Use a opção Ignorar campos primitivos para ignorar campos primitivos não inicializados.
