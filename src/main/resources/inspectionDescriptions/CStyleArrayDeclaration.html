Relata declarações de array escritas em sintaxe de estilo C, em que os colchetes do array são colocados após o nome da variável ou após a lista de parâmetros do método. A maioria dos estilos de código prefere declarações de array no estilo Java, em que os colchetes do array são colocados após o nome do tipo.

Exemplo:


  public String process(String value[])[] {
    return value;
  }

Após a aplicação da correção rápida:


  public String[] process(String[] value) {
    return value;
  }

Configuração da inspeção:

Use a opção Ignorar declarações no estilo C em variáveis para relatar apenas declarações de array no estilo C em tipos de retorno de método.
