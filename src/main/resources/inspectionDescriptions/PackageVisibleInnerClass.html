Relata classes aninhadas declaradas sem nenhum modificador de acesso (também conhecido como package-private).

Exemplo:


  public class Outer {
    static class Nested {}  // Aviso
    class Inner {}          // Aviso
    enum Mode {}            // O aviso depende das configurações
    interface I {}          // O aviso depende das configurações
  }

Configuração da inspeção:

  • Use a opção Ignorar enums internos com visibilidade de pacote para ignorar enums internos package-private.
  • Use a opção Ignorar interfaces internas com visibilidade de pacote para ignorar interfaces internas package-private.
