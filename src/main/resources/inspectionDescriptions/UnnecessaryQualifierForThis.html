Relata qualificações desnecessárias de this ou super.

O uso de qualificadores em this ou super para remover a ambiguidade de referências de código pode facilmente tornar-se desnecessário por meio de refatorações automáticas e deve ser removido para fins de clareza.

Exemplo:


  class Foo {
    void foo() {}
  }

  class Bar extends Foo {
    void bar() {
      Bar.super.foo();
    }
  }

Após a aplicação da correção rápida:


  class Foo {
    void foo() {}
  }

  class Bar extends Foo {
    void bar() {
      super.foo();
    }
  }
