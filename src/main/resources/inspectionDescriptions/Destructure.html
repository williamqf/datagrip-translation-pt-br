Relata declarações que podem ser desestruturadas.

Exemplo:


  data class My(val first: String, val second: Int, val third: Boolean)

  fun foo(list: List<My>) {
      list.forEach { my ->
          println(my.second)
          println(my.third)
      }
  }

A correção rápida desestrutura a declaração e introduz novas variáveis com os nomes da classe correspondente:


  data class My(val first: String, val second: Int, val third: Boolean)

  fun foo(list: List<My>) {
      list.forEach { (_, second, third) ->
          println(second)
          println(third)
      }
  }
