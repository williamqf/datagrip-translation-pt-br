Relata classes declaradas como final. Não relata classes final que estendem classes ou interfaces sealed. Tais classes não podem ser herdadas e podem indicar falta de design orientado a objetos. Alguns padrões de codificação desencorajam classes final.

Exemplo:


  public final class Main {
  }

Após aplicar a correção rápida:


  public class Main {
  }
