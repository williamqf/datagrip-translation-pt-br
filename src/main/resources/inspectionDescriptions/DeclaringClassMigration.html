Relata chamadas à propriedade 'declaringClass' em Enum que causam erros de compilação a partir da versão 1.9.

'Enum.getDeclaringClass' é uma função Java "oculta" que normalmente não é visível por meio de resolução. No entanto, ela é visível através de propriedades sintéticas, o que é um bug de frontend.

Mais detalhes: KT-49653 Depreciação e remoção da propriedade sintética Enum.declaringClass

A correção rápida substitui a chamada por 'declaringJavaClass'.

Exemplo:


  fun <E: Enum<E>> foo(values: Array<E>) {
    EnumSet.noneOf(values.first().declaringClass)
  }

Após aplicar a correção rápida:


  fun <E: Enum<E>> foo(values: Array<E>) {
    EnumSet.noneOf(values.first().declaringJavaClass)
  }

Esta inspeção relata apenas se o nível de linguagem Kotlin do projeto ou módulo for 1.7 ou superior.
