Relata loops for que possuem uma variável de índice incrementada manualmente.

Loops for com uma variável de índice incrementada manualmente podem ser simplificados usando a função withIndex().

A correção rápida Usar withIndex() em vez de incremento de índice manual pode ser usada para corrigir o código automaticamente.

Exemplo:


  fun foo(list: List<String>): Int? {
      var index = 0
      for (s in list) { // <== pode ser simplificado
          val x = s.length * index
          index++
          if (x > 0) return x
      }
      return null
  }

Após a aplicação da correção rápida:


  fun foo(list: List<String>): Int? {
      for ((index, s) in list.withIndex()) {
          val x = s.length * index
          if (x > 0) return x
      }
      return null
  }
