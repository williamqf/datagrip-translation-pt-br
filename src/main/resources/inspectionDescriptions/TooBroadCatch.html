Relata blocos catch com parâmetros mais genéricos do que as exceções lançadas pelo bloco try correspondente.

Exemplo:


  try  {
    File file = new File(pathToFile);
    return file.getAbsolutePath();
  } catch (Exception ex) { // aviso: o 'catch' de 'Exception' é muito abrangente, ocultando a exceção 'RuntimeException'
    return defaultFilePath;
  }

Após a aplicação da correção rápida:


  try  {
    File file = new File(pathToFile);
    return file.getAbsolutePath();
  } catch (RuntimeException ex) {
    return defaultFilePath;
  }

Configuração da inspeção:

  • Use a opção Avisar apenas para RuntimeException, Exception, Error ou Throwable para que esta inspeção avise apenas sobre as exceções mais comuns.
  • Use a opção Ignorar exceções que ocultam outras, mas são lançadas por si mesmas para ignorar qualquer exceção que oculte outras, mas que ainda possa ser lançada e, portanto, não seja tecnicamente muito abrangente.
