Relata expressões de deslocamento (shift) iguais ou superiores à largura do tipo inteiro.

Em tais expressões, todos os bits do operando à esquerda são deslocados, resultando em um valor constante. Um valor constante significa que o deslocamento está incorreto ou que a expressão de deslocamento é redundante.

Exemplo:


  func shift(i int8) {
    fmt.Println(i << 8) // Sempre imprime 0
  }
