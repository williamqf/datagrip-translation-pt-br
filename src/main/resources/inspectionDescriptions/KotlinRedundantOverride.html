Relata declarações de sobrescrita redundantes.

Se a sobrescrita não modificar a semântica da assinatura herdada, por exemplo, ao alterar a visibilidade, a sobrescrita poderá ser omitida.

Exemplo:


  open class Foo {
      open fun singleExpression() {
      }
  }

  class Bar : Foo() {
      override fun singleExpression() = super.singleExpression()
  }

Após a aplicação do ajuste rápido:


  class Bar : Foo() {
  }
