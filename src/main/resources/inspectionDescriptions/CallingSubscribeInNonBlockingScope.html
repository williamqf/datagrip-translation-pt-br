Relata chamadas de subscribe() em métodos "reativos".

Métodos que retornam tipos Publisher (incluindo Flux e Mono) não devem chamar o método subscribe() diretamente, pois isso pode quebrar a cadeia de chamadas reativas.

Em vez de usar subscribe(), considere utilizar operadores de composição, como flatMap(), zip(), then(), etc.

Exemplo:


  Flux<String> stringFlux(){
    Flux<String> flux = Flux.just("abc");
    flux.subscribe(); // <- chamada 'subscribe' bloqueante em um contexto não bloqueante
    return flux;
  }
