Relata classes sealed que não possuem uma lista permits explícita. Classes seladas que possuem subclasses apenas no mesmo arquivo não precisam ter uma lista permits explícita. Por padrão, esta inspeção não realça no editor, fornecendo apenas correções.

Exemplo:


  sealed class A {}
  final class B extends A {}

Após aplicar a correção rápida:


  sealed class A permits B {}
  final class B extends A {}

Novidade na versão 2020.3
