Relata parâmetros incompatíveis em docstrings. Por exemplo, b é destacado porque tal parâmetro não existe na função add.


    def add(a, c):
    """
    @param a:
    @param b:
    @return:
    """
    pass

A inspeção não avisa sobre parâmetros ausentes se nenhum deles for mencionado na docstring:


def mult(a, c):
    """
    @return:
    """
    pass
