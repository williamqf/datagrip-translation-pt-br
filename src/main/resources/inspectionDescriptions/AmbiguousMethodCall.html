Relata chamadas para métodos da superclasse a partir de uma classe anônima, interna ou local se um método com a mesma assinatura existir no código que envolve a classe. Nesses casos, pode parecer que o método do código circundante está sendo chamado, quando na verdade a chamada é para o método da superclasse.

Para esclarecer a intenção do código, recomenda-se adicionar um qualificador super explícito à chamada do método.

Exemplo:


  class Parent {
    void ambiguous(){}
  }

  class Example {
    void ambiguous(){}

    class Inner extends Parent {
      void example(){
        ambiguous(); //aviso
      }
    }
  }

Após a aplicação da correção rápida:


  class Parent {
    void ambiguous(){}
  }

  class Example {
    void ambiguous(){}

    class Inner extends Parent {
      void example(){
        super.ambiguous();
      }
    }
  }
