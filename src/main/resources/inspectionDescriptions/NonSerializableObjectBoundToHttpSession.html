Relata objetos de classes que não implementam java.io.Serializable sendo usados como argumentos para javax.servlet.http.HttpSession.setAttribute() ou javax.servlet.http.HttpSession.putValue().

Tais objetos não serão serializados se a HttpSession for passivada ou migrada, o que pode resultar em bugs difíceis de diagnosticar.

Esta inspeção assume que objetos dos tipos java.util.Collection e java.util.Map são Serializable, a menos que os parâmetros de tipo sejam não-Serializable.

Exemplo:


  void foo(HttpSession session) {
      session.setAttribute("foo", new NonSerializable());
  }
  static class NonSerializable {}
