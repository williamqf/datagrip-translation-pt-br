Relata destinos de anotação sem sentido em superclasses a partir do Kotlin 1.4.

Destinos de anotação como @get: não têm sentido em superclasses e são proibidos.

Exemplo:


  interface Foo

  annotation class Ann

  class E : @field:Ann @get:Ann @set:Ann @setparam:Ann Foo

Após a aplicação da correção rápida:


  interface Foo

  annotation class Ann

  class E : Foo

Esta inspeção relata apenas se o nível de linguagem Kotlin do projeto ou módulo for 1.4 ou superior.
