Sugere a substituição da inicialização de um campo que utiliza atribuição pela inicialização na declaração do campo.

Reporta apenas quando a atribuição do campo está localizada em um inicializador de instância ou static, e quando parece seguro combiná-la com a declaração do campo. Em outros casos (como atribuições dentro de construtores), a correção rápida é oferecida sem destaque, pois a correção pode alterar a semântica.

Exemplo:


  class MyClass {
    static final int intConstant;
    
    static {
      intConstant = 10;
    }
  }
A correção rápida moverá o valor atribuído para o inicializador do campo e removerá o inicializador de classe, se possível: 

  class MyClass {
    static final int intConstant = 10;
  }
