Relata campos em subclasses de java.lang.Exception que não foram declarados como final.

Os dados em objetos de exceção não devem ser modificados, pois isso pode levar à perda do contexto do erro para depuração e registro posteriores.

Exemplo:


  public class EditorException extends Exception {
    private String message; // Aviso: campo não final 'message' na classe de exceção
  }
