Relata declarações (classes, métodos, campos) anotadas com @ApiStatus.Obsolete.

Às vezes, não é possível remover a API atual, embora ela possa não funcionar corretamente, pois existe uma API mais recente, melhor ou mais genérica. Desta forma, esta é uma variante mais fraca da anotação @Deprecated. A API anotada não deve ser usada em novo código, mas o adiamento da migração do código existente é permitido, portanto, esse uso não é considerado um aviso.
