Relata variáveis de padrão com o mesmo nome que campos da classe envolvente. Devido a essa nomenclatura, você pode usar incorretamente a variável de padrão quando a intenção era usar o campo de mesmo nome.

Uma correção rápida sugere renomear a variável.

Exemplo:


  class Pointless {
    Point p = new Point();

    public void test(Object a) {
      if (a instanceof Point p) {
        System.out.print("a is a point (" + p.x + ", " + p.y + ')');
      } else {
        System.out.print("p is a point (" + p.x + ", " + p.y + ')');
      }
    }
  }
