Relata construtores em classes anotadas com @Singleton, a menos que sejam declarados como não estritos.

Exemplo:


  @Singleton
  class Foo{
      Foo(){
      }
  }

Existem duas correções rápidas possíveis: remover o construtor ou declarar o @Singleton como não estrito.

Após aplicar a correção rápida:


  @Singleton
  class Foo{
  }

ou:


  @Singleton(strict = false)
  class Foo{
      Foo(){
      }
  }
