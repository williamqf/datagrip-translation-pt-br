Relata atribuições ou modificações em parâmetros lambda. Embora ocasionalmente seja intencional, essa construção pode ser confusa e geralmente é causada por erros de digitação ou pelo uso de uma variável incorreta.

A correção rápida adiciona a declaração de uma nova variável.

Exemplo:


  list.forEach(s -> {
    s = s.trim();
    System.out.println("String: " + s);
  });

Após a aplicação da correção rápida:


  list.forEach(s -> {
    String trimmed = s.trim();
    System.out.println("String: " + trimmed);
  });

Use a opção Ignorar se a atribuição for uma transformação do parâmetro original para ignorar atribuições que modificam o valor do parâmetro com base em seu valor anterior.
