Relata comparações com null precedendo chamadas de equals() que utilizam um argumento constante.

Exemplo:


  if (s != null && s.equals("literal")) {}

Após a aplicação da correção rápida:


  if ("literal".equals(s)) {}

Use as configurações da inspeção para relatar chamadas de equals() com argumentos não constantes quando for comprovado que o argumento de equals() não é null.
