Relata instruções if com ramificações em excesso. Tais instruções podem ser confusas e geralmente indicam um nível insuficiente de abstração de design.

Exemplo:


if (a) {
  print "foo"
} else if (b) {
  print "bar"
} else if (c) {
  print "baz"
} else if (d) {
  print "Too many branches"
}

Use o campo Número máximo de ramificações para especificar o número máximo de ramificações esperado.
