Relata operações de comparação com valores constantes no lado errado.

Algumas convenções de codificação especificam que as constantes devem estar em um lado específico da comparação, seja no lado esquerdo ou no lado direito.

Exemplo:


  boolean compare(int x) {
    return 1 > x; // constante '1' no lado esquerdo da comparação
  }

Após a aplicação da correção rápida (quick fix):


  boolean compare(int x) {
    return x < 1;
  }

Use as configurações de inspeção para escolher o lado da constante nas comparações e se deseja emitir um aviso quando o literal null estiver no lado errado.

Novidade na versão 2019.2
