Relata comparações encadeadas que podem ser simplificadas.

Exemplo:


  def do_comparison(x):
      xmin = 10
      xmax = 100
      if x >= xmin and x <= xmax:
          pass

O IDE sugere simplificar if x >= xmin and x <= xmax. Após a aplicação do ajuste rápido, o código se torna:


  def do_comparison(x):
      xmin = 10
      xmax = 100
      if xmin <= x <= xmax:
          pass
