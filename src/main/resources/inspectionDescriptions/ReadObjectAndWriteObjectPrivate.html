Relata classes Serializable nas quais os métodos readObject ou writeObject não estão declarados como private. Não há razão para que a visibilidade desses métodos seja superior a private.

Uma correção rápida é sugerida para tornar o método correspondente private.

Exemplo:


  public class Test implements Serializable {
    public void readObject(ObjectInputStream stream) {
      /* ... */
    }
  }

Após a aplicação da correção rápida:


  public class Test implements Serializable {
    private void readObject(ObjectInputStream stream) {
      /* ... */
    }
  }
