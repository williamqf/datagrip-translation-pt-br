Relata parâmetros de tipo que podem ter variância in ou out.

O uso de variância in e out permite uma inferência de tipo mais precisa e uma semântica de código mais clara no Kotlin.

Exemplo:


  class Box<T>(val obj: T)

  fun consumeString(box: Box<String>) {}
  fun consumeCharSequence(box: Box<CharSequence>) {}

  fun usage(box: Box<String>) {
      consumeString(box)
      consumeCharSequence(box) // erro de compilação
  }

A correção rápida adiciona o modificador de variância correspondente:


  class Box<out T>(val obj: T)

  fun consumeString(box: Box<String>) {}
  fun consumeCharSequence(box: Box<CharSequence>) {}

  fun usage(box: Box<String>) {
      consumeString(box)
      consumeCharSequence(box) // OK
  }
