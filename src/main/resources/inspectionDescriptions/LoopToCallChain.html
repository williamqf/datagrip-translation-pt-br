Relata loops for que podem ser substituídos por uma sequência de operações da biblioteca padrão (stdlib), como map, filter, etc.

Exemplo:


fun foo(list: List<String>): List<Int> {
  val result = ArrayList<Int>()
  for (s in list) {
     if (s.length > 0)
       result.add(s.hashCode())
     }
  return result
}

Após a aplicação da correção rápida:


fun foo(list: List<String>): List<Int> {
  val result = list
    .filter { it.length > 0 }
    .map { it.hashCode() }
  return result
}
