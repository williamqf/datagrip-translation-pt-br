Reporta tabelas com duas colunas que possuem incremento automático. Nos dialetos MySQL, Microsoft SQL Server e Db2, uma tabela pode ter apenas um campo com a opção de incremento automático e esse campo deve ser uma chave.

Exemplo (MySQL):

CREATE TABLE my_table
(
    id INT AUTO_INCREMENT,
    c2 INT AUTO_INCREMENT,
);

A restrição AUTO_INCREMENT de c2 será realçada porque c1 já possui essa restrição. Para corrigir o aviso, você pode tornar id a chave primária e remover o AUTO_INCREMENT de c2.

CREATE TABLE my_table
(
    id INT AUTO_INCREMENT PRIMARY KEY,
    c2 INT,
);
