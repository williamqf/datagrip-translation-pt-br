Relata componentes Swing que não fornecem contexto de acessibilidade.

Esta informação é utilizada por leitores de tela. Se não for fornecida, usuários com deficiência visual não conseguirão acessar o componente.

Exemplo:


  ListCellRenderer<String> renderer = (list, val, index, sel, cell) -> {
    JPanel panel = new JPanel();
    return panel;
  };

Para corrigir este problema, você deve chamar setAccessibleName() no JPanel retornado ou sobrescrever o seu método getAccessibleContext().

O texto retornado deve refletir o propósito do componente. Por exemplo, no caso de um ListCellRenderer, este seria o texto do item de menu.
