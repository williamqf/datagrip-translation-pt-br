Relata a criação de novas instâncias de classes anotadas com @groovy.lang.Singleton. Essa construção pode causar a exceção de tempo de execução Can't instantiate singleton.

Exemplo:


  @Singleton
  class Foo{
  }
  
  Foo foo = new Foo()

Após a aplicação da correção rápida:


  @Singleton
  class Foo{
  }
  
  Foo foo = Foo.instance
