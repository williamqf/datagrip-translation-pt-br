Relata literais true redundantes em condições de loop for.

No Go, o loop for possui uma forma com uma condição implícita. Isso significa que você pode criar um loop infinito escrevendo for { }. Portanto, em casos com uma condição true (for true { }), o true é considerado redundante e pode ser omitido.

Para obter detalhes sobre loops for no Go, consulte For no go.dev.

Exemplo:

for true {
  println("ping")
}
