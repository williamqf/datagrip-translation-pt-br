Relata verificações de comprimento de array desnecessárias que precedem uma iteração de array. Quando o comprimento do array é zero, a iteração é ignorada, portanto não há necessidade de verificar o comprimento explicitamente.

Exemplo:


  void f(String[] array) {
    if (array.length != 0) { // verificação desnecessária
      for (String str : array) {
        System.out.println(str);
      }
    }
  }

Uma correção rápida é sugerida para remover ou desembrulhar a verificação de comprimento:


  void f(String[] array) {
    for (String str : array) {
      System.out.println(str);
    }
  }
