Esta inspeção detecta potenciais desreferenciamentos de nil, considerando todas as chamadas de função. A análise é interprocedural e considera as chamadas em todo o projeto, incluindo chamadas entre diferentes pacotes.

Exemplo:


package a

func CreateUser(name string) *User {
  if isValid(name) {
    return &User{Name: name}
  }
  return nil
}


package main

func main() {
	user := CreateUser("NilDereference")
	print(user.Name) // potencial desreferenciamento de nil
}

No exemplo fornecido, user é um ponteiro para a struct User, mas pode ser nil porque CreateUser pode retornar nil. Tentar desreferenciar user para acessar Name sem verificar primeiro se user é nil resultará em um erro em tempo de execução.
