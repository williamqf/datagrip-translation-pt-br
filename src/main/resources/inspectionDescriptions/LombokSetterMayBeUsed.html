Relata métodos setter padrão que podem ser substituídos pela anotação @Setter do lombok.

Exemplo:


  import java.util.Date;

  public class MyClass {
    /**
     * A data do processo.
     */
    private Date processDate;

    /**
     * Define a data.
     *
     * @param param A data
     */
    public void setProcessDate(Date param) {
      processDate = param;
    }
  }

Após a aplicação da correção rápida/limpeza:


  import lombok.Setter;
  import java.util.Date;

  @Setter
  public class MyClass {
    /**
     * A data do processo.
     * -- SETTER --
     * Define a data.
     *
     * @param param A data
     */
    private Date processDate;
  }

Relata apenas quando a biblioteca lombok está configurada. Para remover o lombok (unlombok), consulte o site do lombok.
