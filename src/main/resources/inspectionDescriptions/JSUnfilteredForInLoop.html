Relata loops for-in não filtrados.

O uso dessa estrutura processa não apenas as propriedades do próprio objeto, mas também as propriedades do protótipo do objeto. Isso pode ser inesperado em casos específicos, como em métodos utilitários que copiam ou modificam todas as propriedades, ou onde o protótipo de Object pode ter sido modificado incorretamente. Por exemplo, o código a seguir imprimirá 42 e myMethod: 


Object.prototype.myMethod = function myMethod() {};
let a = { foo: 42 };
for (let i in a) {
  console.log(a[i]);
}

Sugere-se substituir o loop inteiro pelo método Object.keys() ou adicionar uma verificação hasOwnProperty(). Após a aplicação da correção rápida, o código ficará assim:


for (let i in a) {
  if (a.hasOwnProperty(i)) {
    console.log(a[i]);
  }
}
