Relata supressões com nomes de diagnóstico antigos, como @Suppress("HEADER_WITHOUT_IMPLEMENTATION").

Alguns diagnósticos do Kotlin 1.2 e versões anteriores estão agora obsoletos, tornando essas supressões redundantes.

Exemplo:


@Suppress("HEADER_DECLARATION_WITH_BODY")
expect fun connection() {
  // ...
}

Após aplicar a correção rápida:


@Suppress("EXPECTED_DECLARATION_WITH_BODY")
expect fun connection() {
  // ...
}
