Relata variáveis locais desnecessárias que não aumentam a compreensibilidade de um método, incluindo:
  • Variáveis locais retornadas imediatamente.
  • Variáveis locais que são atribuídas imediatamente a outra variável e não são mais utilizadas.
  • Variáveis locais que possuem sempre o mesmo valor que outra variável local ou parâmetro.

Exemplo:


  boolean yes() {
    boolean b = true;
    return b;
  }

Após a aplicação da correção rápida:


  boolean yes() {
    return true;
  }

Configuração da inspeção:

  • Use a opção Ignorar variáveis retornadas ou lançadas imediatamente para ignorar variáveis que são retornadas ou lançadas de imediato. Alguns estilos de codificação sugerem o uso de tais variáveis para fins de clareza e facilidade de depuração.
  • Use a opção Ignorar variáveis com anotações para ignorar variáveis anotadas.
