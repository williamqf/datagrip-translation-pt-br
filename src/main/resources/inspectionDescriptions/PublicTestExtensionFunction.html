Relata funções de extensão public em Driver/Finder/IdeaFrameUI em classes de teste dentro do módulo intellij.driver.tests.

As funções de extensão public no módulo intellij.driver.tests criam problemas de manutenibilidade porque elas:

  • Ficam visíveis para todos os testes em todos os contextos
  • Podem levar a dependências de teste pouco claras e dificultar a manutenção

Soluções recomendadas:

  • Mover a função para um companion object private
  • Relocar para um Page Object apropriado:
    • Page Objects genéricos para funções amplamente aplicáveis
    • Page Objects específicos para funções dependentes de contexto
