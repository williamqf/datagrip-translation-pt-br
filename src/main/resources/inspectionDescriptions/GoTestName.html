thoughtful Relata nomes de testes, benchmarks e exemplos mal formatados.

De acordo com o pacote testing no go.dev, os nomes devem seguir convenções especiais para que a ferramenta go possa processá-los corretamente.

Exemplo:

func Testfoo(*testing.T) {} // a ferramenta 'go' não executará este teste

Após aplicar a correção rápida Renomear para:

func TestFoo(*testing.T) {}
