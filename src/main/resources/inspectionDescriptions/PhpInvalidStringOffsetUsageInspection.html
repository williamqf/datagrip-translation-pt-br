Relata o uso de offsets de string inválidos que resultarão em erros de tempo de execução.

Os seguintes usos são relatados:

  • Usar um offset de string como um objeto ou array
  • Remover um offset de string via unset
  • Passar um offset de string por referência
  • Criar uma referência para ou a partir de um offset de string
  • Incrementar ou decrementar um offset de string
  • Usar atribuição com operadores compostos, como += ou .=, em um offset de string
  • Atribuir uma string vazia ou uma string com comprimento maior que 1 a um offset de string
