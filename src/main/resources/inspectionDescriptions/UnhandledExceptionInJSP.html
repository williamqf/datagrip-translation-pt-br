Relata exceções lançadas por scriptlets JSP de nível superior.

Exemplo:


  <%!
  String getFormattedDate() {
    ...
    throw new IllegalArgumentException(...); // erro
  }
  %>
  <html>
  <head>
    <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
    <title>Hello, JSP!</title>
  </head>
  <body>
  <i>Today is <%= getFormattedDate() %></i>
  </body>
  </html>

Nota: Tais páginas JSP serão compiladas porque o container de servlet envolve todo o código JSP em blocos try {} catch() {}, mas produzirão exceções de tempo de execução quando implantadas no servidor.
