Relata constantes que são definidas, mas nunca usadas no código. 
func main() {
 const i = 100
}

Diferente de variáveis e imports não utilizados, este código será compilado. Constantes não utilizadas podem aumentar a base de código e tornar a compilação do programa mais lenta. Para remover constantes não utilizadas, considere usar a correção rápida Excluir constante.
