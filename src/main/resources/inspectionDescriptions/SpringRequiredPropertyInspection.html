Relata propriedades Bean setter @Required que não foram injetadas ou vinculadas automaticamente (autowired).

Exemplo:


<beans>
  <bean class="beans.MyComponent" id="foo"/>
  <!-- Relata "Propriedade obrigatória ausente: 'port'" -->
</beans>


public class MyComponent {
   @Required
   public void setPort(int port) // Relata "A propriedade obrigatória 'port' não foi injetada"
   {...}
}
