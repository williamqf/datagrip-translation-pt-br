Relata propriedades que são explicitamente atribuídas a parâmetros do construtor primário.

As propriedades podem ser declaradas diretamente no construtor primário, reduzindo a quantidade de código e melhorando a legibilidade do código.

Exemplo:


  class User(name: String) {
      val name = name
  }

A correção rápida unirá o parâmetro e a declaração da propriedade no parâmetro do construtor primário:


  class User(val name: String) {
  }
