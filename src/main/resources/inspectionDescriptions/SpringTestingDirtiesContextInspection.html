Relata configurações 'mode' incorretas na anotação @DirtiesContext.

Exemplo:


  @ContextConfiguration
  @DirtiesContext(methodMode = MethodMode.AFTER_METHOD, // Definir o modo de método em uma classe de teste anotada não faz sentido. Para controle em nível de classe, use classMode em seu lugar.
                  hierarchyMode = DirtiesContext.HierarchyMode.CURRENT_LEVEL) // O hierarchyMode deve ser usado quando o contexto é configurado como parte de uma hierarquia por meio de @ContextHierarchy
  public class MyTest {
    @DirtiesContext(classMode = DirtiesContext.ClassMode.AFTER_CLASS, // Definir o modo de classe em um método de teste anotado não faz sentido. Para controle em nível de método, use methodMode em seu lugar.
                     hierarchyMode = DirtiesContext.HierarchyMode.CURRENT_LEVEL) // O hierarchyMode deve ser usado quando o contexto é configurado como parte de uma hierarquia por meio de @ContextHierarchy
    public void testFoo() {...}
  }
