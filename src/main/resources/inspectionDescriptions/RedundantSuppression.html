Relata as ocorrências dos seguintes elementos que podem ser removidos com segurança, pois as inspeções que eles afetam não são mais aplicáveis neste contexto:
  • anotação @SuppressWarnings, ou
  • comentário de linha // noinspection, ou
  • comentário JavaDoc /** noinspection */

Exemplo:


public class C {
 // O símbolo já é private,
 // mas a anotação ainda permanece
  @SuppressWarnings({"WeakerAccess"})
 private boolean CONST = true;
 void f() {
    CONST = false;
  }
}
