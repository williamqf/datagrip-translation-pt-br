Relata trechos de código Java idênticos a métodos static existentes e sugere a reutilização desses métodos. Reutilizar métodos existentes torna o código mais curto e legível.

Exemplo:


  static List<String> readFileAndTrim(Path path) throws IOException {
    List<String> lines = Files.readAllLines(path);
    return lines.stream().map(String::trim).toList();
  }
  
  static List<String> readFileAndTrim(String path) throws IOException {
    Path p = Path.of(path);
    List<String> lines = Files.readAllLines(p);
    return lines.stream().map(String::trim).toList();
  }
Aqui, o segundo método é muito semelhante ao primeiro, o qual pode ser reutilizado em sua implementação. Após a aplicação da correção rápida, o resultado será o seguinte: 

  static List<String> readFileAndTrim(Path path) throws IOException {
    List<String> lines = Files.readAllLines(path);
    return lines.stream().map(String::trim).toList();
  }

  static List<String> readFileAndTrim(String path) throws IOException {
    Path p = Path.of(path);
    return readFileAndTrim(p);
  }
