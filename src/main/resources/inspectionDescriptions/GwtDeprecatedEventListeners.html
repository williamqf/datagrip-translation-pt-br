Relata código que usa interfaces de listener de eventos depreciadas no GWT 1.6.

As interfaces *Listener do pacote com.google.gwt.user.client.ui (por exemplo, com.google.gwt.user.client.ui.ClickListener) foram depreciadas desde a versão 1.6 do GWT.

A correção rápida migra o código para usar as interfaces *Handler correspondentes.
