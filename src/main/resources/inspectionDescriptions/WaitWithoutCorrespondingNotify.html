Relata chamadas a Object.wait() para as quais não foram encontradas chamadas Object.notify() ou Object.notifyAll() correspondentes.

Esta inspeção relata apenas chamadas que contêm qualificadores que referenciam campos da classe atual.

Exemplo:


  public class Foo {
    public Object foo = new Object();

    void bar() throws InterruptedException {
      this.foo.wait();
    }
  }
