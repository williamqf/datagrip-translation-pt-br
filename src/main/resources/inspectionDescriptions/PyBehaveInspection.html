Relata o uso incorreto do framework Python Behave BDD:

  • O arquivo de definição de etapa não contém from behave import *.
  • O diretório que contém as definições de etapa não se chama steps.

Se um desses casos ocorrer, o IDE realçará a etapa correspondente no arquivo .feature. Você pode aplicar uma correção rápida para criar uma definição de etapa ou renomear o diretório que contém essa etapa.

Para obter mais detalhes, consulte Criar definição de etapa.
