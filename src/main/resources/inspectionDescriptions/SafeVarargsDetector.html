Relata todos os métodos que possuem aridade variável (varargs) e que podem ser anotados com @SafeVarargs. A anotação @SafeVarargs suprime avisos não verificados (unchecked) sobre a criação de arrays parametrizados no local de chamada.

Exemplo:


  public class Foo<T> {
    private List<T> list = new ArrayList<>();

    public final void safeVarargs(T... elements) {
      Collections.addAll(list, elements);
    }
  }

Após a aplicação da correção rápida:


  public class Foo<T> {
    private List<T> list = new ArrayList<>();

    @SafeVarargs
    public final void safeVarargs(T... elements) {
      Collections.addAll(list, elements);
    }
  }

Esta anotação não é suportada em JVMs do Java 1.6 ou anterior.
