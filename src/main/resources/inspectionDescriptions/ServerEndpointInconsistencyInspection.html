Relata erros de declaração de endpoint WebSocket.

De acordo com a especificação JSR 356, um endpoint WebSocket deve ter exatamente um método @OnMessage para cada tipo de mensagem: texto, binário e pong.

Classes de endpoint anotadas com javax.websocket.server.ServerEndpoint podem ter apenas um método com cada uma destas três anotações de ciclo de vida:

  • @OnOpen
  • @OnClose
  • @OnError

Classes de endpoint que estendem javax.websocket.Endpoint devem implementar os métodos de ciclo de vida correspondentes em vez de usar anotações.
