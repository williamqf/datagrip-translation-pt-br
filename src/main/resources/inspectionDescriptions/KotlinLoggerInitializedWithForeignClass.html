Relata instâncias de Logger inicializadas usando um literal de classe que não é a classe na qual o Logger reside.

Isso pode acontecer ao copiar e colar o código de outra classe. Isso pode resultar no registro de eventos em categorias inesperadas e em uma filtragem incorreta.

Use as opções de inspeção para especificar as classes e métodos de fábrica de logger reconhecidos por esta inspeção.

Exemplo:


class AnotherService
class MyService {
    private val logger = LoggerFactory.getLogger(AnotherService::class.java)
}

Após aplicar a correção rápida:


class MyService {
    private val logger = LoggerFactory.getLogger(MyService::class.java)
}
