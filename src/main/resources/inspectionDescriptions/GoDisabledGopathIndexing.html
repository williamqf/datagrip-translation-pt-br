Relata a indexação do GOPATH desativada, o que pode impedir que as referências de código sejam resolvidas corretamente.

O GOPATH armazena sua base de código e todos os arquivos necessários para o desenvolvimento. Além disso, inclui os pacotes que você baixa e instala. Se a indexação do GOPATH estiver desativada, apenas o projeto e os pacotes vendor serão indexados. Isso pode melhorar o desempenho geral, mas impossibilita o uso de pacotes no GOPATH.
