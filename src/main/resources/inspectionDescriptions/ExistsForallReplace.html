Relata negação dupla em chamadas exists ou forall.

A correção rápida remove a dupla negação.

Exemplo:


  def condição(x: Int): Boolean = ???
  !Seq(1, 2).exists(x => !condição(x))
  !Seq(1, 2).forall(x => !condição(x))

Após a correção rápida ser aplicada:


  def condição(x: Int): Boolean = ???
  Seq(1, 2).forall(x => condição(x))
  Seq(1, 2).exists(x => condição(x))
