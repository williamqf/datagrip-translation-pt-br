Relata referências não resolvidas no código.

Em linguagens dinamicamente tipadas, isso é possível em um número limitado de casos.

Se o tipo da referência for desconhecido, seus atributos não serão realçados como não resolvidos, mesmo que devessem ser:


def print_string(s):
  print(s.abc())

Neste fragmento de código, s é sempre uma string e abc deveria ser realçado como não resolvido. No entanto, o tipo de s é inferido como Any e nenhum aviso é relatado.

O IDE fornece ações de correção rápida para adicionar referências ausentes on-the-fly.
