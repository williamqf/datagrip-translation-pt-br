Relata definições e usos inválidos de classes abstratas.

Exemplo:


from abc import abstractmethod, ABC


class Figure(ABC):

    @abstractmethod
    def do_figure(self):
        pass


class Triangle(Figure):  # nem todos os métodos abstratos foram definidos na classe 'Triangle'
    def do_triangle(self):
        pass


Triangle()  # não é possível instanciar a classe abstrata 'Triangle'

Após a aplicação do ajuste rápido, o IDE implementa os métodos abstratos para a classe Triangle:


from abc import abstractmethod, ABC


class Figure(ABC):

    @abstractmethod
    def do_figure(self):
        pass


class Triangle(Figure):
    def do_figure(self):
        pass

    def do_triangle(self):
        pass


Triangle()
