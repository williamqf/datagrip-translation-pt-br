Relata chamadas String.format que podem ser substituídas por templates de string.

O uso de templates de string torna o código mais simples.

A correção rápida substitui a chamada por um template de string.

Exemplo:


  fun main() {
      val id = "abc"
      val date = "123"
      val s = String.format("%s_%s_%s", id, date, id)
  }

Após aplicar a correção rápida:


  fun main() {
      val id = "abc"
      val date = "123"
      val s = "${id}_${date}_$id"
  }
