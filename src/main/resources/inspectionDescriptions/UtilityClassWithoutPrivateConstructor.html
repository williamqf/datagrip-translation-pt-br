Relata classes utilitárias que não possuem um construtor private.

Classes utilitárias declaram todos os campos e métodos como static. Criar um construtor private em classes utilitárias evita que sejam instanciadas acidentalmente.

Use a opção Ignorar se anotado por para especificar anotações especiais. Esta inspeção ignorará classes marcadas com uma dessas anotações.

Use a opção Ignorar classes com apenas o método main para ignorar classes que não possuem outros métodos além do método main.
