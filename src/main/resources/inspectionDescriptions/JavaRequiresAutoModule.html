Relata o uso de módulos automáticos em diretivas requires.

Módulos automáticos não são confiáveis porque podem depender de tipos no classpath e seu nome e pacotes exportados podem mudar quando forem convertidos em um módulo explícito.

Corresponde às opções do Javac -Xlint:requires-automatic e -Xlint:requires-transitive-automatic. A primeira opção aumenta a conscientização sobre quando módulos automáticos são usados. A segunda alerta os autores de módulos que estabelecer a legibilidade implícita para um módulo automático colocará os usuários desse módulo em risco.

Exemplo:


  //module-info.java
  module org.printer {
    requires transitive drivers.corp.org; // relatado quando 'drivers.corp.org' for um módulo automático
  }

Use a opção Realçar apenas dependências transitivas para alertar somente sobre dependências transitivas.
