Relata códigos que utilizam loops para copiar o conteúdo de um array para uma coleção.

Uma abordagem mais curta e possivelmente mais rápida (dependendo da implementação da coleção) é usar Collection.addAll(Arrays.asList()) ou Collections.addAll().

Apenas loops sem instruções adicionais internas são relatados.

Exemplo:


  void addAll(List<String> list, String[] arr) {
    for (int i = 0; i < arr.length; i++) {
      String s = arr[i];
      list.add(s);
    }
  }

Após aplicar a correção rápida (quick fix):


  void addAll(List<String> list, String[] arr) {
    Collections.addAll(list, arr);
  }
