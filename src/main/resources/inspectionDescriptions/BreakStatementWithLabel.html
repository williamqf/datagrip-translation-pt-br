Relata instruções break com rótulo.

Instruções break com rótulo complicam a refatoração e podem ser confusas.

Exemplo:


  void handle(List<String> strs) {
    outer:
    for (String s: strs) {
      for (char ch : s.toCharArray()) {
        if ('s' == ch) break outer;
        handleChar(ch);
      }
    }
  }
