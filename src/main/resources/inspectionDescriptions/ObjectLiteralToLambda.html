Relata literais de objeto anônimo que implementam uma interface Java com um único método abstrato que podem ser convertidos em uma chamada usando uma expressão lambda.

Exemplo:


class SomeService {
  val threadPool = Executors.newCachedThreadPool()
    
  fun foo() {
    threadPool.submit(object : Runnable {
      override fun run() {
        println("hello")
      }
    })
  }
}

Após a aplicação da correção rápida:


  fun foo() {
    threadPool.submit { println("hello") }
  }
