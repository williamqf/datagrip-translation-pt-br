Relata vulnerabilidades em dependências Gradle, Maven, NPM, PyPI e Go declaradas no projeto. A lista completa de dependências do Gradle e do Maven é exibida em "Bibliotecas Externas" na janela de ferramentas "Projeto".

Corrigir os problemas relatados ajuda a evitar que seu software seja comprometido por invasores.

Para resolver problemas, você pode atualizar para uma versão na qual a vulnerabilidade foi corrigida (se disponível) ou mudar para uma dependência sem vulnerabilidades.

As correções rápidas disponíveis podem sugerir a atualização para uma versão segura ou a visita a um site para saber mais sobre uma vulnerabilidade específica.
