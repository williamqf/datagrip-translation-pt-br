Relata classes que estendem diretamente java.lang.Throwable.

Estender diretamente java.lang.Throwable é geralmente considerado uma má prática. Normalmente, estender java.lang.RuntimeException, java.lang.Exception ou, em casos especiais, java.lang.Error é suficiente.

Exemplo:


  class EnigmaThrowable extends Throwable {} // Aviso: a classe 'EnigmaThrowable' estende diretamente 'java.lang.Throwable'
