Relata caracteres em arquivos de propriedades que não são suportados pelo conjunto de caracteres ISO-8859-1 ao usar o Java 1.8 ou anterior. Esta inspeção verifica se os arquivos de propriedades contêm caracteres que não podem ser mapeados para ISO-8859-1 e sugere convertê-los em sequências de escape Unicode para garantir a compatibilidade.

Configuração da inspeção:

Use a correção rápida Converter para sequência de escape para garantir a compatibilidade com a codificação ISO-8859-1:

Exemplo:


  key=Java + ☕ = 🍀

Após aplicar a correção rápida:


  key=Java + \u2615 = \uD83C\uDF40
