Relata o uso de self como um qualificador quando não é exigido por uma closure de escape.

Sugere a remoção do self redundante.

Exemplo: 


  struct URL {
      func loadAsync() {
          print(self.path)

          DispatchQueue.global().async {
              self.loadSync()
          }
      }
  }

Após aplicar a correção rápida: 


  struct URL {
      func loadAsync() {
          print(path)

          DispatchQueue.global().async {
              loadSync()
          }
      }
  }
