Relata chamadas de String.equals() com um CharSequence como argumento.

O método String.equals() só pode retornar true para argumentos do tipo String. Para comparar o conteúdo de uma String com um argumento CharSequence que não seja uma String, use o método contentEquals().

Exemplo:


  boolean equals(String s, CharSequence ch) {
    return s.equals(ch);
  }

Após a aplicação da correção rápida:


  boolean equals(String s, CharSequence ch) {
    return s.contentEquals(ch);
  }

Novidade na versão 2017.3
