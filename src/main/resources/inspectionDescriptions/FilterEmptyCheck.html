Informa verificações de ausência em coleções filtradas.

A correção rápida usa exists ou forall para verificar a ausência.

Exemplo:


  var x: Seq[Int]
  x.filter(p).size == 0
  x.filter(p).length > 0
  x.filterNot(p).isEmpty
  x.filterNot(p).nonEmpty

Depois que a correção rápida for aplicada:


  var x: Seq[Int]
  !x.exists(p)
  x.exists(p)
  x.forall(p)
  !x.forall(p)
