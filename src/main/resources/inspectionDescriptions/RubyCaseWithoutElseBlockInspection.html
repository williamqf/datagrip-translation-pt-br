Relata instruções case que não contêm um bloco else. Um bloco else deve estar sempre presente para especificar o resultado padrão quando nenhuma correspondência for encontrada.

Exemplo:

# Má prática
case status
when :active
  perform_action
end
Uma correção rápida está disponível para inserir o bloco else. Após a aplicação da correção rápida, o resultado será o seguinte: 
# Boa prática
case status
when :active
  perform_action
else
  # digite o código aqui
end
