Relata classes cujos nomes são muito curtos, muito longos ou não seguem o padrão de expressão regular especificado.

Para cada tipo de classe, use os campos de entrada fornecidos para especificar o comprimento mínimo, o comprimento máximo e a expressão regular esperada para o nome da classe.

As expressões regulares devem ser especificadas no formato padrão java.util.regex.
