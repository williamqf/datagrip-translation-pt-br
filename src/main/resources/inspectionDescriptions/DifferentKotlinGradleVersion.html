Relata que a versão do plugin Gradle não é adequadamente suportada pelo plugin atual do IDE.

Isso pode causar inconsistências entre o IDE e o build do Gradle em relação a relatórios de erros ou comportamento do código.

Exemplo:


  dependencies {
    classpath "org.jetbrains.kotlin:kotlin-gradle-plugin:0.0.1"
  }

Para corrigir este problema, altere a versão do plugin Kotlin Gradle para corresponder à versão do Kotlin incluída no plugin do IDE.
