Relata atribuições onde o lado direito é uma divisão que não deveria ser truncada para um inteiro.

Embora ocasionalmente intencional, essa construção geralmente indica um erro.

Exemplo:


  int x = 18;
  x *= 3/2; // x não será alterado devido ao resultado da divisão inteira

Este código deve ser substituído por: 


  int x = 18;
  x *= 3.0/2;

Nas opções de inspeção, é possível desativar avisos para divisões suspeitas, mas possivelmente corretas, como quando o dividendo não pode ser calculado estaticamente. 


  void calc(int d) {
     int x = 18;
     x *= d/2;
  }
