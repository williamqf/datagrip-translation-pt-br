Relata chamadas aos seguintes métodos em objetos java.util.Properties:
  • put()
  • putIfAbsent()
  • putAll()
  • get()

Por razões históricas, java.util.Properties herda de java.util.Hashtable, mas o uso desses métodos não é recomendado para evitar a poluição de valores de propriedade com tipos diferentes de String.

Chamadas para java.util.Properties.putAll() não são relatadas quando os parâmetros de chave e valor no mapa são ambos do tipo String. Tais chamadas são seguras e não há alternativa melhor.

Exemplo:


  Object f(Properties props) {
    props.put("hello", "world");
    props.putIfAbsent("hello", "world");
    props.putAll(new HashMap<>());
    return props.get("Hello");
  }

Após a aplicação da correção rápida:


  Object f(Properties props) {
    props.setProperty("hello", "world");
    props.putIfAbsent("hello", "world");
    props.putAll(new HashMap<>());
    return props.getProperty("hello");
  }
