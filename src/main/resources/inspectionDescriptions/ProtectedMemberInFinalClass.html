Relata membros protected em classes final sem o modificador protected.

Como as classes final não podem ser herdadas, marcar um método como protected pode ser confuso. É recomendável declarar tais membros como private ou com visibilidade de pacote.

Exemplo:

record Bar(int a, int b) {
  protected int sum() { 
     return a + b;
  }
}

Após a aplicação da correção rápida:

record Bar(int a, int b) {
  int sum() { 
     return a + b;
  }
}

Como mostrado neste exemplo, uma classe pode ser marcada como final de forma explícita ou implícita.
