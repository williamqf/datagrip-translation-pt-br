Relata casts de array suspeitos. Um cast de array é considerado suspeito quando um array é convertido para um tipo de array mais específico. Embora tais casts sejam válidos em tempo de compilação, eles podem falhar em tempo de execução com uma ClassCastException.

Exemplo:


  Number[] numbers = new Number[]{1L, 2L, 4L};
  Long[] longs = (Long[])numbers;
