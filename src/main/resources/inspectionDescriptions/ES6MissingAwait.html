Relata chamadas de função async sem o prefixo await esperado dentro de funções async. Essas chamadas retornam uma Promise e o fluxo de controle continua imediatamente.

Exemplo:


async function bar() { /* ... */ }
async function foo() {
    bar(); // ruim
}

Após aplicar a correção rápida, o prefixo await será adicionado:


async function bar() { /* ... */ }
async function foo() {
    await bar(); // bom
}

Com a caixa de seleção "Relatar promises em instruções return" marcada, a adição de await em instruções return também será sugerida.
Embora isso geralmente não seja necessário, traz dois benefícios principais:
  • Ao envolver o código com try-catch, você não esquecerá de adicionar o await.
  • O await explícito ajuda o runtime V8 a fornecer stack traces assíncronos.
