Relata arquivos no Python 2 que contêm caracteres não ASCII e não possuem uma declaração de codificação no topo.

Exemplo:


class A(object):
# №5
    def __init__(self):
        pass

Neste exemplo, a IDE relata símbolos não ASCII no comentário e a ausência de uma declaração de codificação. Aplique a correção rápida sugerida para adicionar a declaração de codificação ausente:


# coding=utf-8
class A(object)
# №5
    def __init__(self):
        pass
