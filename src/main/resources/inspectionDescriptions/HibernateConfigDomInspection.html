Relata erros de configuração XML do Hibernate, incluindo:

  • Referências a classes que não podem ser instanciadas
  • Referências a classes que não estendem as classes obrigatórias
  • Referências a classes com escopo inadequado
  • Tags e valores de atributos vazios
  • Valores de tags e atributos que não correspondem aos padrões exigidos (por exemplo, identificadores Java)
  • Tags que não contêm tags filhas ou atributos obrigatórios
  • Tags que definem objetos com nomes duplicados
