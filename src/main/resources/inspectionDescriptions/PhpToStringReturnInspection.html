Relata métodos __toString que não retornam uma string.

Antes do PHP 7.4, isso resultaria em um erro fatal de nível E_RECOVERABLE_ERROR. No PHP 7.4, um ERROR fatal e irrecuperável é lançado. A partir do PHP 8.0, os valores de retorno seguem a semântica de tipos padrão do PHP e, se possível, são convertidos para string quando o modo de tipos estritos estiver desativado.

Consulte __toString (php.net) para obter detalhes.
