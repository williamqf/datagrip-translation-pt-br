Relata loops while que iteram sobre coleções e que podem ser substituídos pelo loop for aprimorado (sintaxe de iteração foreach).

Exemplo:


  Iterator it = c.iterator();
  while(it.hasNext()) {
    Object obj = it.next();
    System.out.println(obj);
  }

Pode ser substituído por:


  for (Object obj : c) {
    System.out.println(obj);
  }
