Relata classes cujo número de métodos excede o limite máximo especificado.

Classes com muitos métodos geralmente tentam "fazer demais". Considere dividir essas classes em várias classes menores.

Configuração da inspeção:

  • Use o campo Limite de contagem de métodos para especificar o número máximo de métodos permitidos em uma classe.
  • Use a opção Ignorar métodos getter e setter simples para ignorar getters e setters simples na contagem de métodos.
  • Use a opção Ignorar métodos que sobrescrevem/implementam métodos super para ignorar métodos que sobrescrevem ou implementam métodos de uma superclasse.
