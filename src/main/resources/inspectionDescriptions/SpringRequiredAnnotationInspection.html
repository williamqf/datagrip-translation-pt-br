Relata propriedades de setter do Bean @Required que não foram injetadas nem configuradas por autowiring.

Exemplo:


<beans>
  <bean class="beans.MyComponent" id="foo"/>
  <!-- Relata "Propriedade obrigatória ausente: 'port'" -->
</beans>


public class MyComponent {
   @Required
   public void setPort(int port) // Relata "Propriedade obrigatória 'port' não injetada"
   {...}
}
