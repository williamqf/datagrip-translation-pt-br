Relata campos de coleção public static que podem ser modificados.

Embora esses campos sejam frequentemente usados para armazenar coleções de valores constantes, eles ainda representam um risco de segurança porque, mesmo que o campo seja declarado como final, seu conteúdo ainda pode ser modificado.

Exemplo:


  public static final List<String> EVENTS = new ArrayList<>();

Use a tabela na seção Opções para especificar métodos que retornam coleções imutáveis. Campos de coleção public static inicializados com esses métodos não serão relatados.
