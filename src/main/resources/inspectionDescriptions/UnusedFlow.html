Relata um Flow de Corrotina do Kotlinx que foi criado, mas nunca consumido ou passado para outra função.

O código que define um Flow não é executado, a menos que o Flow seja coletado usando um operador terminal.

Exemplo:


fun foo() {
    flowOf(1).onEach { println(it) }
}
