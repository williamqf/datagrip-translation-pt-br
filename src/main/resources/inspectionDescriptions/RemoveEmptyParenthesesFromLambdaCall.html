Relata parênteses vazios redundantes em chamadas de função em que o único parâmetro é um lambda fora dos parênteses.

Use a correção rápida 'Remover parênteses desnecessários de chamada de função com lambda' para limpar o código.

Exemplo:


  fun foo() {
      listOf(1).forEach() {  }
  }

Após a aplicação da correção rápida:


  fun foo() {
      listOf(1).forEach {  }
  }
