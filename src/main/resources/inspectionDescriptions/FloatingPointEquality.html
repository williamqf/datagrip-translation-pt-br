Relata valores de ponto flutuante comparados usando os operadores == ou !=.

Os valores de ponto flutuante são inerentemente imprecisos e a comparação para igualdade exata raramente é a semântica desejada.

Esta inspeção ignora comparações com literais de zero e infinito.

Exemplo: 


  void m(double d1, double d2) {
    if (d1 == d2) {}
  }
