Relata classes comuns que podem ser convertidas em arquivos-fonte compactos

Exemplo:


  public class Sample {
      public static void main(String[] args) {
          System.out.println("Hello, world!");
      }
  }

Após a aplicação da correção rápida:


  void main() {
      IO.println("Hello, world!");
  }

Configuração da inspeção:

  • Selecione Substituir 'System.out.print()' e 'System.out.println()' por métodos na classe 'java.lang.IO' para converter essas chamadas em métodos de IO.
