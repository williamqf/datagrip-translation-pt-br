Relata construtores de cópia que não copiam todos os campos da classe.

Campos final com inicializadores e campos transient são considerados desnecessários de serem copiados.

Exemplo:


  class Point {

    private int x;
    private int y;

    Point(int x, int y) {
      this.x = x;
      this.y = y;
    }

    Point(Point other) {
      // os campos x e y não foram inicializados
    }
  }
