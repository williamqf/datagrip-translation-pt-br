Relata dependências da biblioteca kotlinx.coroutines no Gradle que devem ser atualizadas para compatibilidade com o Kotlin 1.3 ou superior.

Exemplo:


  dependencies {
      implementation 'org.jetbrains.kotlinx:kotlinx-coroutines-core:0.0.1'
  }

A correção rápida alterará a versão da biblioteca kotlinx.coroutines para uma versão compatível com o Kotlin 1.3:


  dependencies {
      implementation 'org.jetbrains.kotlinx:kotlinx-coroutines-core:0.27.0-eap13'
  }
