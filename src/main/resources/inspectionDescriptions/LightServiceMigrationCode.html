Relata classes que podem ser marcadas como serviços leves usando a anotação @com.intellij.openapi.components.Service em vez de serem registradas como serviços no plugin.xml.

Serviços que não se destinam a ser sobrescritos não precisam ser registrados no arquivo plugin.xml. Em vez disso, você pode usar a anotação @Service na classe de serviço. Para serviços de nível de projeto, especifique @Service(Service.Level.PROJECT).

Requisitos:

  • IntelliJ Platform 2019.3 ou posterior.
  • A classe de serviço deve ser final.
  • serviceInterface não foi especificado.
  • Se o serviço de nível de aplicação for com.intellij.openapi.components.PersistentStateComponent, o roaming deve estar desativado (roamingType = RoamingType.DISABLED).
  • Nenhum dos seguintes atributos foi especificado: os, client, overrides, id, preload, configurationSchemaKey.

Consulte Serviços na documentação do SDK de Plugins da plataforma IntelliJ para obter mais detalhes.
