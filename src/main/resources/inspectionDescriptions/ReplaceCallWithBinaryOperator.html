Relata chamadas de função que podem ser substituídas por operadores binários (especialmente operadores relacionados à comparação).

Exemplo:

  fun test(): Boolean {
      return 2.compareTo(1) > 0 // 'compareTo()' substituível
  }

Após a aplicação da correção rápida:

  fun test(): Boolean {
      return 2 > 1
  }
