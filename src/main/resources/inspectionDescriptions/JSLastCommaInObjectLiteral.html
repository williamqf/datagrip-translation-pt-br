Relata o uso de vírgulas finais em literais de objeto.

Este aviso é relatado apenas quando a versão da linguagem JavaScript está definida como ECMAScript 5.1.

As vírgulas finais em literais de objeto são permitidas pela especificação, mas alguns navegadores podem lançar um erro quando uma vírgula final é usada.

Você pode configurar as opções de formatação para vírgulas finais em Estilo de código | JavaScript ou TypeScript | Pontuação.
