Denuncia as anotações @Scope, @Role e @DataModel que especificam escopo não suportado para um determinado componente.

A tabela abaixo lista os escopos que são suportados para cada tipo de componente (em negrito o escopo padrão):

A variável de contexto @DataModel tem restrições diferentes para seu escopo: se nenhum escopo for especificado, o escopo será o mesmo que o do componente que o envolve (EVENT se o componente estiver no escopo STATELESS). Ou, especifique escopo PAGE para a variável de contexto @DataModel.
