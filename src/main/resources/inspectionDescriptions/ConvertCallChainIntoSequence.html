Relata cadeias de chamadas em uma Collection que devem ser convertidas em uma Sequence.

Cada função de transformação de Collection (por exemplo, map() ou filter()) cria uma nova Collection (geralmente uma List ou um Set) internamente. No caso de múltiplas chamadas subsequentes e uma grande quantidade de itens na Collection, o tráfego de memória pode ser significativo. Nesses casos, é preferível usar uma Sequence.

Exemplo:


  class Entity(val key: String, val value: String)

  fun getValues(lines: List<String>) = lines
      .filter { it.isNotEmpty() }
      .map { it.split(',', limit = 2) }
      .filter { it.size == 2 }
      .map { Entity(it[0], it[1]) }

A correção rápida envolverá a cadeia de chamadas com asSequence() e toList():


  class Entity(val key: String, val value: String)

  fun getValues(lines: List<String>) = lines
      .asSequence()
      .filter { it.isNotEmpty() }
      .map { it.split(',', limit = 2) }
      .filter { it.size == 2 }
      .map { Entity(it[0], it[1]) }
      .toList()
