Relata chamadas a java.util.Set.removeAll() com um argumento java.util.List.

Tais chamadas podem ser lentas quando o tamanho do argumento é maior ou igual ao tamanho do conjunto e o conjunto é uma subclasse de java.util.AbstractSet. Nesse caso, List.contains() é chamado para cada elemento no conjunto, executando uma busca linear.

Exemplo:


  public void check(String... ss) {
    // possível complexidade O(n^2)
    mySet.removeAll(List.of(ss));
  }

Após a aplicação da correção rápida:


  public void check(String... ss) {
    // complexidade O(n)
    List.of(ss).forEach(mySet::remove);
  }
