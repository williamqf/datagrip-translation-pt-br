Relata instruções e expressões switch com poucos rótulos case e sugere reescrevê-las como instruções if e else if.

Exemplo (mínimo de ramificações == 3):


  switch (expression) {
    case "foo" -> foo();
    case "bar" -> bar();
  }

Após a aplicação do ajuste rápido:


  if ("foo".equals(expression)) {
    foo();
  } else if ("bar".equals(expression)) {
    bar();
  }

Não relata expressões switch exaustivas (Java 14+) ou instruções switch com padrão (pattern switch) sem uma ramificação 'default' (Java 17 preview). Isso ocorre porque a verificação de exaustividade em tempo de compilação seria perdida se o switch fosse convertido em if (o que pode ser um resultado indesejado).

Configuração da inspeção:

Use o campo Número mínimo de ramificações para especificar a quantidade mínima esperada de rótulos case.

Use a opção Não relatar instruções switch com padrões para evitar o relato de instruções e expressões switch que possuam ramificações de padrão. Por exemplo:


  String result = switch(obj) {
    case String str -> str.trim();
    default -> "none";
  };

Pode ser preferível manter um switch com apenas uma ramificação de padrão em vez de usar uma instrução instanceof.
