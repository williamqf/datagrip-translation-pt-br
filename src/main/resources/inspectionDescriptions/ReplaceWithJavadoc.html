Relata comentários comuns pertencentes a campos, métodos ou classes que podem ser substituídos por comentários Javadoc.

Exemplo:


  public class Main {
    /*
    * Hello,
    */
    // World!
    void f() {
    }
  }

Após aplicar a correção rápida:


  public class Main {
    /**
     * Hello,
     * World!
     */
    void f() {
    }
  }
