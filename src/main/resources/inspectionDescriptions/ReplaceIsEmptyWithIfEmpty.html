Relata chamadas a isEmpty, isBlank, isNotEmpty ou isNotBlank em instruções if para atribuir valores padrão.

A correção rápida substituirá a condição if por uma chamada a ifEmpty ou ifBlank.

Exemplo:


  fun test(list: List<Int>): List<Int> {
      return if (list.isEmpty()) {
          println()
          foo()
      } else {
          list
      }
  }

Após a aplicação da correção rápida:


  fun test(list: List<Int>): List<Int> {
      return list.ifEmpty {
          println()
          foo()
      }
  }

Esta inspeção relata apenas se a versão da linguagem Kotlin do projeto ou módulo for 1.3 ou superior.
