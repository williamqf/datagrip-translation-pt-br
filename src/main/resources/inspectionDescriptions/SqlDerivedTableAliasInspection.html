Relata tabelas derivadas sem um alias.

Exemplo (MySQL):

CREATE TABLE table1 (id INT, name VARCHAR(20), cats FLOAT);
CREATE TABLE table2 (id INT, age INTEGER);

SELECT id AS ID, name, cats, age
FROM (SELECT table1.id, name, cats, age
FROM table1
JOIN table2 ON table1.id = table2.id);

De acordo com Derived Tables no dev.mysql.com, o alias é obrigatório. Você pode usar a correção rápida Introduzir alias para adicionar um alias.

Após aplicar a correção rápida:

SELECT id AS ID, name, cats, age
FROM (SELECT table1.id, name, cats, age
FROM table1
JOIN table2 ON table1.id = table2.id);
