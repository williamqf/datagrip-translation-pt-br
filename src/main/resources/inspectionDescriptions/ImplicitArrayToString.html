Relata arrays usados em concatenações de String ou passados como parâmetros para métodos de java.io.PrintStream, como System.out.println().

Geralmente, espera-se que o conteúdo do array seja utilizado, em vez do próprio objeto array.

Exemplo:


  void print(Object[] objects) {
    System.out.println(objects);
  }

Após a aplicação da correção rápida:


  void print(Object[] objects) {
    System.out.println(Arrays.toString(objects));
  }
