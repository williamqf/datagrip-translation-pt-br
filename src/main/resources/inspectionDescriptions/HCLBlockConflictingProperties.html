Relata propriedades conflitantes em blocos HCL.

Exemplo de problema:


provider "kubernetes" {
  version = "~> 1.25"
  config_path = "~/.kube/config"
  version = "~> 1.24"
}

Neste exemplo, a propriedade version é declarada duas vezes dentro do bloco do provedor Kubernetes. Isso é um erro.

Exemplo de correção:


provider "kubernetes" {
  version = "~> 1.25"
  config_path = "~/.kube/config"
}
Suportado por: Plugin Terraform e HCL
