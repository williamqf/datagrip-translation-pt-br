Relata prefixos de cifrão redundantes em expressões de template de string.

Tais prefixos podem ser removidos com segurança.

Exemplo: 


fun example() {
    $"Exemplo: um prefixo de comprimento um é sempre redundante"
    $$"Exemplo: a string não contém valores interpolados"
    $$"Exemplo: os sinais de cifrão $$ aqui são caracteres comuns $$"
}

Após a aplicação da correção rápida:


fun example() {
    "Exemplo: um prefixo de comprimento um é sempre redundante"
    "Exemplo: a string não contém valores interpolados"
    "Exemplo: os sinais de cifrão $$ aqui são caracteres comuns $$"
}
