Relata interfaces na lista implements de uma classe ou na lista extends de uma interface que já foram implementadas por uma superclasse ou estendidas por uma superinterface. Tais declarações são desnecessárias e podem ser removidas com segurança.

Exemplo:


  class X implements One, Two {
  }
  interface One {}
  interface Two extends One {}

Após a aplicação da correção rápida:


  class X implements Two {
  }
  interface One {}
  interface Two extends One {}

Use esta opção para não relatar Serializable ou Externalizable na lista extends ou implements.
