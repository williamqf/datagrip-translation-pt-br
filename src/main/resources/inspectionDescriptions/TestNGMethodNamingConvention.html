Relata métodos de teste TestNG cujos nomes são muito curtos, muito longos ou não seguem o padrão de expressão regular especificado.

Quando esta inspeção está ativada, a inspeção Convenção de nomenclatura de métodos de instância ignora automaticamente os métodos de teste TestNG.

Use os campos para especificar o comprimento mínimo, o comprimento máximo e a expressão regular esperada para os nomes dos métodos de teste TestNG. Especifique 0 para não verificar o comprimento do nome. A expressão regular é especificada no formato padrão java.util.regex.
