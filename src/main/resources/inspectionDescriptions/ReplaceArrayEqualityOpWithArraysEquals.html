Relata o uso dos operadores == ou != em arrays que devem ser substituídos por contentEquals().

Os operadores == e != comparam referências de array em vez de seus conteúdos.

Exemplo:

  fun test() {
      val a = arrayOf(1, 2, 3)
      val b = arrayOf(1, 2, 3)
      println(a == b) // comparação de referência
  }

Após a aplicação da correção rápida:

  fun test() {
      val a = arrayOf(1, 2, 3)
      val b = arrayOf(1, 2, 3)
      println(a.contentEquals(b))
  }
