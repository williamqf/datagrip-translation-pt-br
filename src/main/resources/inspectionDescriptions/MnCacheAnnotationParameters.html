Relata anotações do Micronaut Cache cujo valor do atributo parameters contém nomes de parâmetros de método não definidos.

As seguintes anotações são suportadas:

  • io.micronaut.cache.annotation.Cacheable
  • io.micronaut.cache.annotation.CachePut
  • io.micronaut.cache.annotation.CacheInvalidate

Exemplo:


  @CachePut(parameters = "myParameter") // Erro: Não é possível resolver o símbolo 'myParameter'
  public String cachedMethod(String name_is_not_equal_to_myParameter) {
    return "...";
  }
