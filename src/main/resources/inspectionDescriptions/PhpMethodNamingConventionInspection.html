Relata métodos com nomes muito curtos, muito longos ou que não seguem o padrão de expressão regular especificado. Alguns estilos de código possuem convenções de nomenclatura especiais para métodos.

Use as opções de inspeção para especificar os comprimentos mínimo/máximo e a expressão regular esperada para nomes de métodos. Para ignorar o comprimento do nome, especifique 0.

Para saber mais sobre expressões regulares, consulte o guia Quick Start (regular-expressions.info).
