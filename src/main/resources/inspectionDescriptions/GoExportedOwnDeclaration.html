Relata variáveis ou constantes exportadas em uma lista de declarações separadas por vírgulas.

Exemplo:

const C1, C3, C2, C44, C9, C11, C6 = 1, 2, 3, 1, 3, 2, 1

Esta declaração dificulta saber o valor de cada constante. Para tornar esta declaração mais fácil de ler, você pode aplicar a correção rápida Extrair para declaração própria. Após aplicar a correção rápida a cada constante, a declaração ficará assim:

const (
 C3  = 2
 C2  = 3
 C44 = 1
 C9  = 3
 C11 = 2
 C6  = 1
 C1  = 1
)
