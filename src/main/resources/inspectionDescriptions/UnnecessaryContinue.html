Relata instruções continue que são as últimas instruções alcançáveis em um loop. Essas instruções continue são desnecessárias e podem ser removidas com segurança.

Exemplo:


  for (String element: elements) {
    System.out.println();
    continue;
  }

Após a aplicação da correção rápida:


  for (String element: elements) {
    System.out.println();
  }

Esta inspeção não analisa arquivos JSP.

Use a opção Ignorar no ramo 'then' de instruções 'if' com um ramo 'else' para ignorar instruções continue localizadas no ramo then de uma instrução if-else completa.

Exemplo:


  for (String element: elements) {
    if(element.isEmpty()) {
      continue;
    } else {
      //...
    }
  }
