O relatório encontra os seguintes problemas na navegação do arquivo index.page.xml do Seam File Navigation:

  • Referências a classes que não podem ser instanciadas.
  • Referências a classes que não estendem as classes necessárias.
  • Referências a classes com escopo inadequado.
  • Tags e valores de atributos vazios.
  • Tags e valores de atributos que não correspondem ao padrão necessário (por exemplo, identificadores Java).
  • Tags que não contêm as tags filhas ou os atributos necessários.
  • Tags que definem objetos com nomes duplicados.
