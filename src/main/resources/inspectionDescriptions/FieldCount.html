Relata classes onde o número de campos excede o valor máximo especificado.

Classes com um grande número de campos geralmente tentam processar coisas demais. Considere dividir esse tipo de classe em várias classes menores.

Configuração da inspeção:

  • Use o campo Limite de contagem de campos para especificar o número máximo de campos permitidos em uma classe.
  • Use a opção Incluir campos constantes na contagem para indicar se os campos constantes devem ser contados.
  • Por padrão, apenas objetos static final imutáveis são contados como constantes. Use a opção Contar campos 'static final' como constantes para contar qualquer campo static final como constante.
  • Use a opção Incluir constantes enum na contagem para especificar se as constantes enum em uma classe enum devem ser contadas.
