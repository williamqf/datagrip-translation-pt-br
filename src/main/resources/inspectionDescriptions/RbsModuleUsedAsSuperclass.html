Relata módulos usados em cláusulas de superclasse. Cláusulas de superclasse podem referenciar apenas classes.

Uma correção rápida pode ser usada para remover cláusulas de superclasse inválidas.

Exemplo:

# O módulo 'Kernel' é usado como uma superclasse
class Example < Kernel
end

Após a aplicação da correção rápida:

class Example
end
