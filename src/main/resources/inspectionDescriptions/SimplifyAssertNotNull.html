Reporta chamadas assert que verificam se o valor de uma variável declarada não é nulo.

O uso de !! ou ?: pode tornar o código mais simples.

A correção rápida substituirá o assert no inicializador da variável pelos operadores !! ou ?:.

Exemplo:


  fun foo(p: Array<String?>) {
      val v = p[0]
      assert(v != null, { "Should be not null" })
  }

Após a aplicação da correção rápida:


  fun foo(p: Array<String?>) {
      val v = p[0] ?: error("Should be not null")
  }
