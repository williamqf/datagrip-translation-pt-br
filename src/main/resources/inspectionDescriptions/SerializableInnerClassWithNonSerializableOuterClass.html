Relata classes internas não estáticas que implementam Serializable e que são declaradas em classes que não implementam Serializable.

Devido a uma referência implícita à classe externa, é improvável que tal classe seja serializada corretamente.

Exemplo:


  class A {
      class Main implements Serializable {
    }
  }

Configure a inspeção usando as seguintes opções:

  • Listar as classes cujas subclasses não devem ser relatadas por esta inspeção. Isso se aplica a classes que herdam Serializable de uma superclasse, mas não são destinadas à serialização.
  • Se deve ignorar classes anônimas Serializable.
