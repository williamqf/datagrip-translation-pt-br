Relata símbolos e referências de função não resolvidos.

Exemplo:


my_fahrenheit_to_celsius(40)

A função my_fahrenheit_to_celsius não está definida, portanto, o IDE exibirá um aviso. Você deve adicionar a declaração ausente.

Possível solução: 


my_fahrenheit_to_celsius  <- function(f_temp) {
  return((f_temp - 32) * 5 / 9)
}

my_fahrenheit_to_celsius(40)
