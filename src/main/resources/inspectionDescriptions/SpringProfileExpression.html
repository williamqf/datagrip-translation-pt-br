Relata expressões @Profile incorretas:

  • O perfil do Spring não deve estar vazio
  • Em expressões de perfil do Spring, os operadores '&' e '|' não devem ser misturados sem o uso de parênteses

Exemplos:


  // Configurado corretamente
  @Configuration
  @Profile("a & (b | c)")
  public class MyConfiguration {...}

  // Vazio 
  @Configuration
  @Profile() // relata "A expressão de perfil deve conter texto"
  public class MyConfiguration {...}

  // Operadores misturados sem parênteses 
  @Configuration
  @Profile("a & b | c") // relata "Expressão de perfil malformada"
  public class MyConfiguration {...}
