Relata o modificador de visibilidade public que pode ser omitido em propriedades onde a visibilidade do set foi reduzida. As propriedades do PHP são implicitamente públicas para leitura, a menos que especificado de outra forma.

A visibilidade assimétrica é suportada no PHP 8.4 e versões superiores. Para mais detalhes, consulte PHP RFC: Asymmetric Visibility v2.
