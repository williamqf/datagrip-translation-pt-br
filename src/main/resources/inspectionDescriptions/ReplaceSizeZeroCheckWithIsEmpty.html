Relata verificações de size == 0 em Collections/Array/String que devem ser substituídas por isEmpty().

O uso de isEmpty() torna o código mais simples.

A correção rápida substitui a verificação de tamanho por isEmpty().

Exemplo:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.size == 0
  }

Após aplicar a correção rápida:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.isEmpty()
  }
