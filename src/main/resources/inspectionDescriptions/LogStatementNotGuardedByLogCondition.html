Relata chamadas de log que não estão envolvidas por condições de guarda. A avaliação dos argumentos da chamada pode ter um custo elevado. Envolver a chamada de log com uma cláusula de guarda evita esse custo quando o log está desativado para o nível utilizado na instrução de log. Isso é particularmente útil para os níveis de log menos graves (trace, debug, finest), pois essas chamadas são frequentemente desativadas em ambientes de produção.

Exemplo:


  public class TestObject {
    void test(Object object) {
      LOG.debug("some logging " + expensiveCalculation(1));
    }
  }

Após a aplicação da correção rápida:


  public class TestObject {
    void test(Object object) {
      if(LOG.isDebugEnabled()){
        LOG.debug("some logging " + expensiveCalculation(1));
      }
    }
  }

Esta inspeção oferece suporte aos frameworks de log Log4j2 e SLF4J (exceto builders).
