Relata variáveis de tipo não utilizadas em contêineres, métodos e aliases de tipo no projeto.

As variáveis de tipo de contêiner são verificadas apenas se o contêiner tiver exatamente uma declaração.

Uma correção rápida pode ser usada para remover variáveis de tipo não utilizadas.

Exemplo:

# A variável de tipo 'A' da classe 'Example' não é utilizada
class Example[A]
end

Após aplicar a correção rápida:

class Example
end
