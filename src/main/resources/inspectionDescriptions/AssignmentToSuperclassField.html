Relata atribuições ou modificações em campos declarados em uma superclasse a partir de um construtor de subclasse.

É preferível inicializar os campos de uma superclasse em seu próprio construtor e delegar para esse construtor na subclasse. Isso também permitirá declarar um campo como final caso ele não seja alterado após a construção.

Exemplo:


  class Super {
    int x;
  }
  class Sub extends Super {
    Sub(int _x) {
      // Aviso: x é declarado em uma superclasse
      x = _x;
    }
  }

Para evitar esse problema, declare o construtor da superclasse:


  class Super {
    final int x;

    Super(int _x) {
      x = _x;
    }
  }
  class Sub extends Super {
    Sub(int _x) {
      super(_x);
    }
  }
