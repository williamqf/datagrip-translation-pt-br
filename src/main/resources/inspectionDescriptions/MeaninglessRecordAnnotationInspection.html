Relata anotações usadas em componentes de record que não têm efeito.

Isso pode acontecer nos dois casos a seguir:

  • A anotação relatada tem o alvo METHOD, mas o acessor correspondente está definido explicitamente.
  • A anotação relatada tem o alvo PARAMETER, mas o construtor canônico está definido explicitamente.

Exemplo:


@Target(ElementType.METHOD)
@interface A { }
  
// A anotação não aparecerá no bytecode,
// pois deveria ser propagada para o acessor, mas o acessor está definido explicitamente
record R(@A int x) {
  public int x() { return x; }
}
