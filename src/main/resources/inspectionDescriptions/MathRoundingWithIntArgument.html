Relata chamadas aos métodos round(), ceil(), floor() e rint() de Math e StrictMath com um int como argumento.

Esses métodos podem ser chamados quando se espera que os argumentos sejam long ou double, o que pode produzir resultados inesperados.

Esta inspeção fornece uma correção capaz de simplificar tais expressões (exceto round) para uma conversão direta para double.

Exemplo:


  int i = 2;
  double d1 = Math.floor(i);

Após a aplicação da correção rápida:


  int i = 2;
  double d1 = i;
