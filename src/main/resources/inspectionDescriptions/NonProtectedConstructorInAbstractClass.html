Relata construtores public em classes abstract.

Como o construtor de uma classe abstract só pode ser chamado a partir dos construtores de suas subclasses, declará-lo como public pode ser confuso.

A correção rápida tornará esses construtores protected.

Exemplo:


  public abstract class Foo {
    public Foo () { // Aviso: o modificador 'public' está presente
        /* ... */
    }
  }

Após aplicar a correção rápida:


  public abstract class Foo {
    protected Foo () {
        /* ... */
    }
  }

Configuração da inspeção:

Use a opção Ignorar em classes não públicas abaixo para ignorar construtores public em classes não públicas.
