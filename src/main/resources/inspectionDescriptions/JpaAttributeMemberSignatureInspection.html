Relata os seguintes erros de especificação JPA:
  • Múltiplos atributos configurados para um método ou campo
  • Atributo ilegal usado em um objeto Embeddable (JPA 1.0)
  • Metadados de atributo de persistência não especificados no campo ou no getter da propriedade
  • Nenhum setter ou getter fornece acesso baseado em propriedade

Exemplo:


  @Entity
  @Table(name = "TABLE")
  public class JavaEntity {

    private Integer id; // Erro: tanto o getter quanto o setter devem estar presentes

    public Integer getId() { ... }



    private String data;

    public String getData() { ... }

    @Basic // Erro: anotações de persistência devem estar no campo ou no método getter
    public void setData(String data) { ... }
  }
