Reporta cadeias de duas chamadas que podem ser substituídas por uma única chamada.

Isso ajuda a evitar a execução de código redundante.

A correção rápida substitui a cadeia de chamadas por uma única chamada.

Exemplo:


  fun main() {
      listOf(1, 2, 3).filter { it > 1 }.count()
  }

Após a aplicação da correção rápida:


  fun main() {
      listOf(1, 2, 3).count { it > 1 }
  }
