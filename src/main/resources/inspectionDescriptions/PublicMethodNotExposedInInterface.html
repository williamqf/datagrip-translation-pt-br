Relata métodos public em classes que não são expostos em uma interface.

Expor todos os métodos public por meio de interfaces é importante para manter o baixo acoplamento e pode ser necessário para certos estilos de programação baseada em componentes.

Exemplo:


interface Person {
  String getName();
}

class PersonImpl implements Person {
  private String name;

  // ok: o método é exposto na interface
  @Override
  public String getName() {
    return name;
  }

  // aviso: o método é public
  // mas não é exposto na interface
  public void setName() {
    this.name = name;
  }
}

Use a lista Ignorar se anotado por para especificar anotações especiais. Esta inspeção ignorará métodos anotados com uma dessas anotações.

Use a opção Ignorar se a classe envolvente não implementar uma interface que não seja de biblioteca para ignorar métodos em classes que não implementam nenhuma interface no projeto.
