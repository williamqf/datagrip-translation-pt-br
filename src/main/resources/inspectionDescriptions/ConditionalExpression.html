Relata o uso do operador condicional ternário e sugere convertê-lo em instruções if/else.

Alguns padrões de código proíbem o uso de operadores condicionais.

Exemplo:


  Object result = (condition) ? foo() : bar();

Após a aplicação da correção rápida:


  Object result;
  if (condition) {
    comp = foo();
  }
  else {
    comp = bar();
  }

Configuração da inspeção:

Use a opção Ignorar para atribuições simples e retornos para ignorar atribuições e retornos simples e permitir a seguinte estrutura:


  String s = (foo == null) ? "" : foo.toString();

Em contextos onde a substituição automática por uma instrução if não é possível (por exemplo, quando uma expressão condicional é usada em uma chamada de construtor super()), use a opção Ignorar locais onde a instrução if não é possível para ignorar as expressões condicionais.
