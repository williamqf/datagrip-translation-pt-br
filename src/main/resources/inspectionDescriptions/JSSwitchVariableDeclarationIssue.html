Relata variáveis declaradas em uma cláusula case de uma instrução switch, mas usadas em outra cláusula case da mesma instrução. Para variáveis de escopo de bloco, isso resultará no lançamento de um ReferenceError. Para variáveis var, isso indica um erro potencial.

Se este padrão for usado intencionalmente, desative a inspeção para variáveis var.
