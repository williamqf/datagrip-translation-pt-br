Relata configurações incorretas de contexto do Spring.

  • Arquivos e diretórios não resolvidos no atributo locations e nos atributos de alias correspondentes da meta-anotação @ContextConfiguration
  • Arquivos de contexto da aplicação padrão ausentes

Para mais detalhes, consulte @ContextConfiguration.

Exemplo:


  @ContextConfiguration(locations = "classpath:META-INF/unknown-context.xml") // Relata "Não foi possível resolver o arquivo 'unknown-context.xml'"
  class MyTests {...}
