Relata o uso de elementos depreciados.

Exemplo:

// Depreciado: use io.SeekStart, io.SeekCurrent e io.SeekEnd.
const (
 SEEK_SET int = 0 // busca em relação ao início do arquivo
 SEEK_CUR int = 1 // busca em relação ao deslocamento atual
 SEEK_END int = 2 // busca em relação ao final
)

De acordo com as Constants no go.dev, SEEK_SET, SEEK_CUR e SEEK_END estão depreciados.
