Relata chamadas para o método java.util.List.set(). Tais chamadas podem ser substituídas por uma forma de acesso indexado mais curta e clara.

Exemplo:


def list = ["foo"]
list.set(0, "bar") // list.set(0, "bar") pode ser substituído por list[0] = "bar"

Após a aplicação da correção rápida:


def list = ["foo"]
list[0] = "bar"
