Reporta classes locais.

Uma classe local é uma classe aninhada nomeada declarada dentro de um bloco de código. Classes locais não são comuns e, portanto, podem ser confusas. Além disso, alguns padrões de código desencorajam o uso de classes locais.

Exemplo:


  class Example {
    void test() {
      class Local { // aqui
      }
      new Local();
    }
  }

Após a aplicação da correção rápida:


  class Example {
    void test() {
      new Local();
    }

    private static class Local { // aqui
    }
  }
