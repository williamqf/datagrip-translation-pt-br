Reporta os seguintes atributos não resolvidos em anotações relacionadas a EntityGraph:
  • O valor em @NamedAttributeNode
  • O subgráfico nomeado em @NamedAttributeNode
  • O nome em @NamedEntityGraph

Exemplo:



  @Entity
  @NamedEntityGraph(name = "entity-graph", ...)
  public class JavaEntity {
    public void foo() {
      Persistence.createEntityManagerFactory("")
        .createEntityManager()
        .getEntityGraph("unknown-entity-graph"); // referência não resolvida
    }
  }
