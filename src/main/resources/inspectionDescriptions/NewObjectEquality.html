Relata códigos que aplicam == ou != a um objeto recém-alocado em vez de chamar equals().

Uma referência a um objeto recém-alocado não pode apontar para um objeto existente, portanto, a comparação sempre resultará em false. Esta inspeção também pode relatar objetos recém-criados retornados de métodos simples.

Exemplo:


  void test(Object obj) {
    if (new Object() == obj) {...}
  }

Após a aplicação da correção rápida:


  void test(Object obj) {
    if (new Object().equals(obj)) {...}
  }
