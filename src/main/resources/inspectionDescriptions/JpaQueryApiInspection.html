Relata símbolos não resolvidos em queries nomeadas e nativas, bem como em métodos de API relacionados:
  • Query nomeada desconhecida
  • Parâmetro de query desconhecido

Exemplo:


  @Entity
  @NamedQuery(name = "SelectByConditionQuery", query = "...")
  public class JavaEntity { ... }

  class Main {
    public static void executeQuery() {
      Persistence.createEntityManagerFactory("")
        .createEntityManager()
        .createNamedQuery("SelectAllQuery") // Erro: Nome de query desconhecido
        ...
    }
  }
