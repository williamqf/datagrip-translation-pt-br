Relata quando headOption ou lastOption é emulado.

Exemplo:


  var seq: Seq[Int]
  if (seq.nonEmpty) Some(seq.head) else None
  seq.lift(0)

  if (seq.size == 0) None else seq.lastOption
  seq.lift(seq.size - 1)

Após a aplicação da correção rápida:


  var seq: Seq[Int]
  seq.headOption
  seq.headOption

  seq.lastOption
  seq.lastOption
