Relata constantes cujos nomes sejam muito curtos, muito longos ou não correspondam ao padrão de expressão regular especificado. Alguns estilos de código possuem convenções de nomenclatura especiais para constantes.

Use as opções de inspeção para especificar os comprimentos mínimo/máximo e a expressão regular esperada para nomes de constantes. Para ignorar o comprimento do nome, especifique 0.

Para saber mais sobre expressões regulares, consulte o Quick Start guide (regular-expressions.info).
