Relata expressões de atribuição aninhadas em outras expressões.

Tais expressões podem ser confusas e violam o princípio de design geral de que qualquer construção deve realizar apenas uma tarefa.

Exemplo:


  String userName;
  // Aviso: o resultado da atribuição a 'userName' é utilizado
  String message = "Hello " + (userName = "Alice") + "!"
  System.out.println(message);
  System.out.println("Goodbye " + userName);
