Relatórios mapeando uma coleção para valores booleanos e utilizando contains sobre ela.

A correção rápida usa .exists() ou !.forall() ao invés.

Exemplo:


  def p(x: Int): Boolean = ???
  var seq: Seq[Int]
  seq.map(p).contains(true)
  seq.map(p).contains(false)

Depois de aplicar a correção rápida:


  def p(x: Int): Boolean = ???
  var seq: Seq[Int]
  seq.exists(p)
  !seq.forall(p)
