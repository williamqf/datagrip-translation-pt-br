Relata linhas em branco em comentários Javadoc.

Linhas em branco no Javadoc podem indicar a intenção de dividir o texto em parágrafos diferentes. No entanto, a ferramenta Javadoc e o IntelliJ IDEA as ignoram ao renderizar os comentários de documentação.

A correção rápida sugere substituir a linha em branco por uma tag de parágrafo (<p>).

Exemplo:


  class Main {
    /**
     * Não faz nada.
     *
     * Realmente não faz nada
     */
    void foo() {}
  }

Após a aplicação da correção rápida:


  class Main {
    /**
     * Não faz nada.
     * <p>
     * Realmente não faz nada
     */
    void foo() {}
  }
