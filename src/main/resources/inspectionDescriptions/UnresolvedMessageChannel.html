Relata atributos de canal incorretos em anotações de método de endpoint: @Gateway, @ServiceActivator, @Filter, etc.

Exemplo:


  @Component
  public class MyComponent {}
     @Gateway(requestChannel = "requestChannel", replyChannel = "replyChannel")
     public void gateway() {...}

     @Gateway(
        requestChannel ="simpleBean",   // O Bean deve ser de um dos seguintes tipos: org.springframework.integration.MessageChannel,org.springframework.messaging.MessageChannel
        replyChannel = "unknownChannel")   // Canal não encontrado
     public void errors() {...}

     @Bean
     public MyBean simpleBean() {...}
  }
