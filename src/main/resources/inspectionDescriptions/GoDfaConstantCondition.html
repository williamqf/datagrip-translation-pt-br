Relata condições no código que são sempre avaliadas como true ou false.

Exemplo:


func _(file *os.File) {
    if file == nil {
        return
    }
    if file == nil {
    }
}

No exemplo fornecido, a segunda condição if file == nil é redundante. Ela sempre será avaliada como false porque repete uma verificação que já foi realizada no código.
