Relata referências a classes internas que não estão qualificadas com o nome da classe envolvente.

Exemplo:


  import foo.Foo.Bar;

  class Foo {
    class Bar {}
  }

  class Baz {
    void f(Bar bar) {}
  }

Após a aplicação da correção rápida:


  class Foo {
    class Bar {}
  }

  class Baz {
    void f(Foo.Bar bar) {}
  }

Use as configurações de inspeção para ignorar referências a classes internas dentro da mesma classe, de modo que essas classes não precisem de importação.
