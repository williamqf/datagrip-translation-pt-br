Relata as propriedades Range.start e Range.endInclusive com boxing.

Essas propriedades podem ser substituídas pelas propriedades first e last sem boxing para evitar chamadas redundantes.

A correção rápida substitui as propriedades start e endInclusive pelas propriedades first e last correspondentes.

Exemplo:


  fun foo(range: CharRange) {
      val lastElement = range.endInclusive
  }

Após a aplicação da correção rápida:


  fun foo(range: CharRange) {
      val lastElement = range.last
  }
