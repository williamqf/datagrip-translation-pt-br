Relata campos não final e não private acessados em um contexto sincronizado.

Não há garantia de que campos não private sejam sempre acessados de forma sincronizada e esse tipo de acesso "parcialmente sincronizado" pode resultar inesperadamente em estruturas de dados inconsistentes.

Exemplo:


  class Bar {
    public String field1;
  }
  public Bar myBar;

  synchronized public void sample() {
   myBar.field1 = "bar";
  }
