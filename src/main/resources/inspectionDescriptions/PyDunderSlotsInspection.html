Relata usos inválidos de classes que contêm definições __slots__.

Exemplo de acesso a atributo não definido:


class Foo:
    __slots__ = ['foo', 'bar']

    def __init__(self):
        self.x = 3  # erro: 'x' não está definido em __slots__

Exemplo de conflito de atributos:


class A:
    __slots__ = ("x",)
    x = 42  # erro: conflito com "x" listado em __slots__

Exemplo de slots=True:


from dataclasses import dataclass

@dataclass(slots=True)  # erro: __slots__ também está definido em Foo
class Foo:
    __slots__ = ['a']
