Relata alocações de arrays de comprimento zero conhecido quando há uma constante disponível na classe do tipo do elemento do array. Como arrays de comprimento zero são imutáveis, você pode reutilizar a mesma instância de array para economizar memória.

Exemplo:


  class Item {
    // Constante pública de array de comprimento zero reutilizável 
    public static final Item[] EMPTY_ARRAY = new Item[0];
  }
  class EmptyNode {
    Item[] getChildren() {
      // Criação desnecessária de array de comprimento zero
      return new Item[0];
    }
  }

Após a aplicação da correção rápida:


  class EmptyNode {
    Item[] getChildren() {
      return Item.EMPTY_ARRAY;
    }
  }
