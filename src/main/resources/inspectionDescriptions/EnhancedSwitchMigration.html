Relata instruções switch que podem ser substituídas automaticamente por uma instrução ou expressão switch aprimorada.

Exemplo:


  double getPrice(String fruit) {
    // A instrução switch pode ser substituída por um 'switch' aprimorado
    switch (fruit) {
      case "Apple":
        return 1.0;
      case "Orange":
        return 1.5;
      case "Mango":
        return 2.0;
      default:
        throw new IllegalArgumentException();
    }
  }

Após a aplicação do ajuste rápido:


  double getPrice(String fruit) {
    return switch (fruit) {
      case "Apple" -> 1.0;
      case "Orange" -> 1.5;
      case "Mango" -> 2.0;
      default -> throw new IllegalArgumentException();
    };
  }

  • Use a opção Avisar apenas quando puder ser convertido em uma expressão para não emitir avisos sobre a conversão em uma instrução switch.
  • Use a opção Número máximo de instruções em um branch para converter em uma expressão switch para avisar sobre a conversão em uma expressão apenas quando o número de instruções em cada branch for menor que o valor especificado.
