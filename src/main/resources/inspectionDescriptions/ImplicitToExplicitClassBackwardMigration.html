Relata arquivos de origem compactos e sugere substituí-los por uma classe convencional.

Exemplo (no arquivo Sample.java):


  void main() {
      String a = "Hello, world!";
  }

Após a aplicação da correção rápida:


public class Sample {
    void main() {
        String a = "Hello, world!";
    }
}

Esta inspeção é útil para realizar o downgrade visando a compatibilidade retroativa com versões anteriores do Java.
