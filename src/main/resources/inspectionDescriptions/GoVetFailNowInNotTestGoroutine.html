Relata chamadas para testing.T.FailNow e métodos semelhantes em uma goroutine dentro de arquivos de teste.

Métodos como FailNow chamam runtime.Goexit e interrompem a goroutine que faz a chamada, em vez do teste. Portanto, eles devem ser chamados apenas a partir da goroutine que executa o teste ou o benchmark.

Para mais detalhes sobre o FailNow, consulte func (*T) FailNow em go.dev.

Exemplo:

func TestFoo(t *testing.T) {
  go func() {
    t.Fatal("oops") // sai da goroutine, não do TestFoo
  }()
}

Após aplicar a correção rápida Substituir por 'Error' e 'return':

func TestFoo(t *testing.T) {
  go func() {
    t.Error("oops")
    return
  }()
}
