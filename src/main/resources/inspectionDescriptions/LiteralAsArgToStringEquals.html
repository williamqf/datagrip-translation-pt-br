Relata chamadas a String.equals() ou String.equalsIgnoreCase() que contêm argumentos literais de string.

Alguns padrões de codificação especificam que literais de string devem ser o qualificador de equals() em vez do argumento, a fim de minimizar NullPointerException.

Uma correção rápida está disponível para trocar o literal e a expressão.

Exemplo:


  boolean isFoo(String value) {
    return value.equals("foo");
  }

Após aplicar a correção rápida:


  boolean isFoo(String value) {
    return "foo".equals(value);
  }
