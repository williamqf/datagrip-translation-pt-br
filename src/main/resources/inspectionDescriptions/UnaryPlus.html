Relata o uso do operador unário +. O sinal de adição unário é geralmente uma operação nula e sua presença pode representar um erro de codificação. Por exemplo, em combinação com um operador de incremento (como em +++) ou um operador de igualdade (como em =+).

Exemplo:


  void unaryPlus(int i) {
    int x = + +i;
  }

As seguintes correções rápidas são sugeridas:

  • Remover o operador + antes da variável i:

    
  void unaryPlus(int i) {
    int x = i;
  }

  • Substituir o operador + pelo operador de incremento de prefixo:

    
  void unaryPlus(int i) {
    int x = ++i;
  }

Use a caixa de seleção abaixo para relatar um sinal de adição unário usado com uma expressão binária ou outra expressão unária. Isso significa que a inspeção não relatará casos em que uma expressão de adição unária é usada em uma expressão de inicializador de array ou como um argumento de método.
