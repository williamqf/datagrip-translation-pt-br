Relata o retorno de campos de arrays ou tipos mutáveis (como Collection, Date, Map, Calendar, etc.) ou atribuições de parâmetros de método a campos desses tipos.

Como esse tipo é mutável, essa construção pode levar a modificações inesperadas no estado do objeto originadas de fora da classe proprietária. Embora essa construção possa ser útil por motivos de desempenho, ela é inerentemente propensa a erros.

Os seguintes tipos mutáveis são relatados:

  • java.util.Date
  • java.util.Calendar
  • java.util.Collection
  • java.util.Map
  • com.google.common.collect.Multimap
  • com.google.common.collect.Table

A correção rápida adiciona uma chamada ao método .clone() para campos de array ou utiliza um wrapper de coleção não modificável.

Exemplo:


  import java.util.*;
  class Log {
    private String[] messages = {"one", "two", "three"};
    private Map<String, String> map = new HashMap<>();

    String[] getMessages() {
      return messages; // Aviso: retornando o campo String[] 'messages'
    }

    Map<String, String> mapping() {
      return map; // Aviso: retornando o campo Map<String, String> 'map'
    }
  }

Após aplicar a correção rápida:


  import java.util.*;
  class Log {
    String[] messages = {"one", "two", "three"};
    private Map<String, String> map = new HashMap<>();

    String[] getMessages() {
      return messages.clone();
    }

    Map<String, String> mapping() {
      return Collections.unmodifiableMap(map);
    }
  }

Use a opção Ignorar atribuições e retornos em métodos privados para ignorar atribuições e retornos dentro de métodos private.
