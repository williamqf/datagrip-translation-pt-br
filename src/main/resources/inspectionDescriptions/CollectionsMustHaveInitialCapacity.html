Relata tentativas de instanciar um novo objeto Collection sem especificar uma capacidade inicial.

Se uma capacidade inicial não for especificada, a capacidade padrão será usada, o que raramente é a escolha ideal. Deixar de especificar uma capacidade inicial para uma coleção pode causar problemas de desempenho se for necessário realocar espaço e copiar a memória quando a capacidade inicial for excedida. Esta inspeção verifica as alocações das classes listadas nas configurações da inspeção.

Exemplo:


  new HashMap<String, String>();

Configure a inspeção usando as seguintes opções:

  • Listar as classes de coleção que devem ser verificadas.
  • Se os inicializadores de campo devem ser ignorados.
