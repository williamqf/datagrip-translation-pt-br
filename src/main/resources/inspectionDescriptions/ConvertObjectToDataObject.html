Relata declarações object que podem ser convertidas em data object.

O data object gera automaticamente toString, equals e hashCode.

Esta inspeção sugere converter um object em um data object nos dois casos a seguir:

  • Quando um toString personalizado retorna o nome da classe
  • Quando o object herda de uma class ou interface selada (sealed)

Exemplo:


    object Foo {
        override fun toString(): String = "Foo"
    }

Após a aplicação da correção rápida (quick fix):


    data object Foo

Esta inspeção é relatada apenas se o nível da linguagem Kotlin do projeto ou módulo for 1.9 ou superior.
