Relata chamadas de função map.put que podem ser substituídas pelo operador de indexação ([]).

O uso de açúcar sintático pode tornar o código mais simples.

A correção rápida substitui a chamada put por uma atribuição.

Exemplo:


  fun foo(map: MutableMap<Int, String>) {
      map.put(42, "foo")
  }

Após a aplicação da correção rápida:


  fun foo(map: MutableMap<Int, String>) {
      map[42] = "foo"
  }
