Relata referências de método não resolvidas nos parâmetros initMethod e destroyMethod da anotação @Bean.

Exemplo:

Se a classe MyBean não contiver o método 'public void doInit(){...}', o erro 'Não é possível resolver o método' será relatado na expressão 'doInit'. 

  public class MyBean {...}

  @Component
  public class MyComponent {
     @Bean(initMethod="doInit" )
     public MyBean myBean() {...}
  }

Neste exemplo, se MyBean não definir o método doInit(), a inspeção relatará a referência de método não resolvida.
