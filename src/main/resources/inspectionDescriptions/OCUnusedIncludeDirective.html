Relata diretivas include que não são utilizadas ou são desnecessárias e podem ser removidas com segurança.
Detectar as que não são utilizadas de forma alguma
Uma diretiva include é considerada não utilizada se as declarações provenientes dela nunca forem usadas na unidade de tradução.
Detectar as que não são utilizadas diretamente
Uma diretiva include é considerada não utilizada se as declarações provenientes dela não forem usadas diretamente no arquivo. Isso funciona melhor com a política "Include What You Use".
Detectar as que são desnecessárias
Detecta o conjunto mínimo de diretivas include necessário para compilar o arquivo. Todas as outras diretivas #include são consideradas não utilizadas.
