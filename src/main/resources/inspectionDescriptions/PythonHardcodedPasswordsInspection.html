Detecta possíveis tokens de segurança ou senhas contidas no código usando análise entrópica e expressões regulares.

Esta inspeção usa análise entrópica e expressões regulares para escanear strings no código-base que se parecem com tokens de segurança ou senhas. Ela destaca os resultados desses exames para que os desenvolvedores possam identificar e proteger vulnerabilidades potenciais. A eficácia desta inspeção depende dos padrões definidos em sua configuração, o que a torna adaptável a diferentes ambientes de programação e requisitos variados. 


// Exemplo de string que pode ser detectada como um possível token:
possibleToken = "pa#sw@rd;"

// Exemplo de um padrão de expressão regular usado para detecção:
regexPattern = '/[0-9]+:AA[0-9A-Za-z\-_]{33}/'
