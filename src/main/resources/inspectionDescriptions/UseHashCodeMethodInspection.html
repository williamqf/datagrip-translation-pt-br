Relata operações bitwise que podem ser substituídas por chamadas aos métodos Long.hashCode() ou Double.hashCode(). Esta inspeção detecta a construção (int)(x ^ (x >>> 32)), onde x é uma variável do tipo long ou o resultado de uma chamada anterior a Double.doubleToLongBits(). Essa substituição pode encurtar o código e melhorar a legibilidade.

Exemplo:


  int result = (int)(var ^ (var >>> 32));

Após a aplicação da correção rápida (quick fix):


  int result = Long.hashCode(var);

Esta inspeção reporta apenas se o nível de linguagem do projeto ou módulo for 8 ou superior.

Novidade na versão 2024.1
