Entity para SharedEntity de referência inválida relatada.

Não é possível referenciar Entity a partir de SharedEntity. Só é possível referenciar SharedEntity a partir de Entity se a propriedade for anulable ou tiver a anotação @CascadeDeleteBy. Caso contrário, uma exceção de tempo de execução será lançada.

Exemplo:


  interface A: Entity {
    var x: SharedEntity
    var y: SharedEntity
  }

A correção rápida permite gerar a anotação ou tornar o tipo anulável


  interface A: Entity {
    var x: SharedEntity?
    @CascadeDeleteBy
    var y: SharedEntity
  }
