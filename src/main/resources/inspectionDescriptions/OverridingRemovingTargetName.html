Informa as definições sem uma anotação @targetName que sobrescrevem definições com anotação @targetName.

Consulte A anotação @targetName.

O quick-fix adiciona a anotação @targetName com o nome externo da definição sobreposta.

Exemplo:


  trait A:
    @targetName("multiplicar")
    def *(i: Int): A

  class B extends A:
    override def *(i: Int): A = ???

Depois que o quick-fix for aplicado:


  trait A:
    @targetName("multiplicar")
    def *(i: Int): A

  class B extends A:
    @targetName("multiplicar")
    override def *(i: Int): A = ???
