Relata propriedades val de nível superior em objetos que poderiam ser declaradas como const para obter melhor desempenho e interoperabilidade com Java.

Exemplo:


  object A {
      val foo = 1
  }

Após a aplicação da correção rápida:


  object A {
      const val foo = 1
  }
Para autores de bibliotecas: O uso de const para símbolos de API pública é perigoso porque, se a versão da biblioteca em tempo de compilação for diferente da versão em tempo de execução, dois valores diferentes poderão ser utilizados. Use com cautela.
