Relata discrepâncias entre os parâmetros declarados e os argumentos reais, bem como argumentos incorretos (como argumentos nomeados duplicados) e ordem incorreta de argumentos.

Exemplo:


class Foo:
    def __call__(self, p1: int, *, p2: str = "%"):
        return p2 * p1


bar = Foo()
bar.__call__() # parâmetro não preenchido
bar(5, "#") # argumento inesperado

O fragmento de código correto é o seguinte:


class Foo:
    def __call__(self, p1: int, *, p2: str = "%"):
        return p2 * p1


bar = Foo()
bar.__call__(5)
bar(5, p2="#")
