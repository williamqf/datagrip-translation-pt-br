Relata classes abstract que possuem apenas um sucessor direto. Embora tais classes possam proporcionar uma clareza de design admirável, elas aumentam desnecessariamente a pegada (footprint) total da aplicação em ambientes com restrição de memória ou largura de banda. Considere mesclar a classe abstrata com seu sucessor.

Esta inspeção destina-se ao Java ME e a outros ambientes com restrição severa de recursos. A aplicação dos resultados desta inspeção sem a devida consideração pode afetar negativamente a clareza do código e o design.

Exemplo:


  abstract class Base {} // Será relatada

  class Inheritor extends Base {}
