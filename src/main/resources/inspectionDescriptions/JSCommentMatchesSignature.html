Relata incompatibilidades entre o nome e o número de parâmetros nos comentários JSDoc e os parâmetros reais da função. Sugere a atualização dos parâmetros no comentário JSDoc.

Exemplo:


/**
 * @param height Height in pixels
 */
function sq(height, width) {} // width não está documentado

Após aplicar a correção rápida:


/**
 * @param height Height in pixels
 * @param width
 */
function sq(height, width) {}
