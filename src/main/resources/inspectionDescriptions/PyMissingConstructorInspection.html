Relata casos em que uma chamada ao construtor super está ausente em uma classe.

Exemplo:


class Fruit:
    def __init__(self):
        pass


class Pear(Fruit):
    def __init__(self):
        pass

A classe Pear deve conter uma chamada ao super dentro do método __init__.

Após a aplicação do ajuste rápido, o código se torna:


class Fruit:
    def __init__(self):
        pass


class Pear(Fruit):
    def __init__(self):
        super().__init__()
