Relata expressões afetadas pela conversão de unboxing (desempacotamento automático de um objeto para um valor primitivo). Evite usar objetos em vez de primitivos. Isso pode afetar significativamente o desempenho.

Exemplo:

int x = new Integer(42);

A correção rápida tornará a conversão explícita:

int x = new Integer(42).intValue();

O autounboxing surgiu no Java 5. Esta inspeção ajuda no downgrade para retrocompatibilidade com versões anteriores do Java.
