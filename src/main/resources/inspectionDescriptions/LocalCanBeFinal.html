Relata variáveis locais ou parâmetros que poderiam ter o modificador final adicionado às suas declarações.

Exemplo:


  ArrayList<Integer> list = new ArrayList();
  fill(list);
  return list;

Após a aplicação da correção rápida:


  final ArrayList<Integer> list = new ArrayList();
  fill(list);
  return list;

Use as opções desta inspeção para definir quais tipos de parâmetros ou variáveis locais devem ser relatados.
