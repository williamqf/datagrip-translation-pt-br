Relata elementos incorretos em arquivos faces-config.xml do Java Server Faces:

  • Referências a classes que não podem ser instanciadas
  • Referências a classes que não estendem as classes necessárias
  • Referências a classes com escopos inadequados
  • Tags e valores de atributos vazios
  • Valores de tags e atributos que não correspondem ao padrão exigido (por exemplo, identificadores Java)
  • Tags que não contêm as tags filhas ou atributos necessários
  • Tags que definem objetos com nomes duplicados

Exemplo:


  <faces-config >
     <application>
        <action-listener>java.lang.String</action-listener>  <!-- 'java.lang.String' não pode ser atribuído a 'javax.faces.event.ActionListener'  -->
    </application>
    <navigation-rule>
        <from-view-id>myView.xhtml</from-view-id>
        <navigation-case>
            <to-view-id>unknown.xhtml</to-view-id>  <!-- Não é possível resolver 'unknown.xhtml'  -->
        </navigation-case>
    </navigation-rule>
  </faces-config>
