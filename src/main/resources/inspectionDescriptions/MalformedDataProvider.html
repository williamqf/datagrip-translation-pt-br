Relata referências a métodos de provedor de dados inexistentes ou inacessíveis.

Exemplo:


public class InstanceDataProviderFromForeignClass {
  // O método data() não existe na classe A
  @Test(dataProvider = "data", dataProviderClass = A.class)
  public void test() {
  }
}
class A { }

Após aplicar a correção rápida:


// O método data() necessário é criado na classe A
class A {
  @DataProvider
  public Object[][] data() {
    return new Object[][]{};
  }
}
