Reporta casos em que coleções Scala de categorias diferentes são comparadas.

Coleções Scala de categorias diferentes (Set, Seq, Map ou Iterator) não podem ser iguais e não devem ser comparadas com arrays também. Esta inspeção sugere converter uma das coleções na mesma categoria.

Exemplo:


  seq == iterator
  seq != array

Depois que a correção rápida for aplicada:


  seq.toIterator == iterator or seq == iterator.toSeq
  seq != array.toSeq
