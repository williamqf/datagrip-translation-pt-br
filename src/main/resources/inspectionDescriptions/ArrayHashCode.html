Relata cálculos de hash code incorretos para arrays.

Para calcular corretamente o hash code de um array, faça o seguinte:

  • Use Arrays.hashcode() para arrays lineares
  • Use Arrays.deepHashcode() para arrays multidimensionais
Estes métodos também devem ser usados com Objects.hash() quando a sequência de valores de entrada contiver um array, por exemplo, Objects.hash(string, Arrays.hashcode(array)):
