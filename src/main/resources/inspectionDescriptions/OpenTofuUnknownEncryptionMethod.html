Detecta referências a métodos de criptografia inexistentes em configurações de bloco encryption do OpenTofu. Para o seguinte código: 

    
      terraform {
        encryption {
          key_provider "pbkdf2" "foo" {
            # Configuration parameters
          }
          method "aes_gcm" "yourname2" {
            keys = key_provider.pbkdf2.foo
          }
        }
      }
Devemos verificar se o nome do método está correto.

Para mais detalhes, consulte https://opentofu.org/docs/language/state/encryption/#methods.
