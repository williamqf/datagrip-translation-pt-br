Relata valores de variáveis que nunca são usados após serem atribuídos. Isso pode ocorrer quando:
  • A variável nunca é lida após ser atribuída.
  • O valor é sobrescrito por outra atribuição antes de ser acessado para leitura.
  • O inicializador da variável é redundante (devido a um dos dois motivos acima).
  • A variável em si nunca é usada.

Exemplo:


  class C {
    int field;

    void foo() {
      field = unused_value;
    }

    void bar() {
      use(field);
      foo();
    }
  };
