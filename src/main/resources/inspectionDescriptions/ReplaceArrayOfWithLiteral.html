Reporta chamadas arrayOf que podem ser substituídas por literais de array [...].

Exemplo:

  annotation class MyAnnotation(val strings: Array<String>)

  @MyAnnotation(arrayOf("alpha", "beta", "omega")) // 'arrayOf()' substituível
  class MyClass

Após a aplicação da correção rápida:

  annotation class MyAnnotation(val strings: Array<String>)

  @MyAnnotation(["alpha", "beta", "omega"])
  class MyClass
