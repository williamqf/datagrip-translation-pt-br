Relata as seguintes violações da especificação EJB:
  • O método não declara que lança java.rmi.RemoteException
  • O tipo de retorno ou os argumentos do método são incompatíveis com RMI/IIOP
  • A interface de endpoint do serviço EJB possui declarações de constantes

Exemplo:


public interface EntityRemote extends javax.ejb.EJBObject{
  public int businessExample(String value) throws java.lang.RuntimeException; // Erro: deve declarar 'RemoteException' em vez disso
}
