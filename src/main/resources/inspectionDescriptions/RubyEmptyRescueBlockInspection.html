Relata blocos rescue vazios. Embora ocasionalmente sejam intencionais, blocos rescue vazios geralmente indicam um erro de codificação e podem dificultar a depuração.

Use a caixa de seleção Ignorar blocos 'rescue' com comentários para definir se blocos rescue contendo apenas comentários devem ser ignorados.

Use a caixa de seleção Ignorar parâmetros 'rescue' chamados '_' para definir se blocos rescue que utilizam _ como parâmetro devem ser ignorados.
