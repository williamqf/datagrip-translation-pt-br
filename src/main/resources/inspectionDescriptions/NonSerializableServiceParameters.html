Relata métodos de serviços remotos que possuem tipos de parâmetros, tipos de retorno ou tipos de exceções lançadas que não são serializáveis.

A correção rápida marca o tipo de referência como serializável estendendo a interface correspondente.

Desative a opção Relatar interfaces para ignorar as verificações de tipos de interface. Isso não causará problemas se a classe real da instância passada para o método for serializável.
