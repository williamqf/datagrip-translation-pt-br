Relata casos em que as cláusulas except não estão listadas na ordem correta, do mais específico para o mais genérico, ou situações em que uma classe de exceção é capturada duas vezes.

Se a ordem não for corrigida, o manipulador mais específico pode não conseguir capturar certas exceções.

Exemplo:


try:
    call()
except ValueError:
    pass
except UnicodeError:
    pass

O IDE sugere mover a cláusula para cima. Após a aplicação da correção rápida, o código muda para:


try:
    call()
except UnicodeError:
    pass
except ValueError:
    pass
