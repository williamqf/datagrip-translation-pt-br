Relata operações de conversão de tipo (cast) que podem ser substituídas por uma variável local ou de padrão existente com o mesmo valor.

Exemplo:


  void foo(Object obj) {
    String s = (String) obj;
    System.out.println(((String) obj).trim());
  }

Após a aplicação do ajuste rápido:


  void foo(Object obj) {
    String s = (String) obj;
    System.out.println(s.trim());
  }
