Relata diretivas requires redundantes em arquivos module-info.java do Java Platform Module System. Uma diretiva requires é redundante quando o módulo A requer o módulo B, mas o código no módulo A não importa nenhum pacote ou classe de B. Além disso, todos os módulos possuem uma dependência declarada implicitamente do módulo java.base, portanto, a diretiva requires java.base; é sempre redundante.

A correção rápida remove diretivas requires redundantes. Se a dependência removida reexportar módulos que são realmente usados, a correção adicionará diretivas requires para esses módulos.

Esta inspeção relata apenas se o nível de linguagem do projeto ou módulo for 9 ou superior.
