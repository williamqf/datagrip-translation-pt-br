Relata declarações de método com sintaxe de procedimento.

Não é recomendado que desenvolvedores usem sintaxe do tipo procedimento para métodos. É inconsistente, pode levar a erros e será removido no Scala 3.

A correção rápida converte a declaração da sintaxe de procedimento para a sintaxe de função.

Exemplo:


  trait Test {
    def method()
  }

Depois que a correção rápida é aplicada:


  trait Test {
    def method(): Unit
  }

Dica: Você pode usar Analisar / Executar Inspeção por Nome (Ctrl+Alt+Shift+I) para aplicar essa inspeção ao projeto inteiro.
