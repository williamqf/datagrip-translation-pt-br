Relata construtores primários vazios quando estão implicitamente disponíveis.

O construtor primário é redundante e pode ser omitido com segurança quando não possui anotações ou modificadores de visibilidade. Use a correção rápida "Remover construtor primário vazio" para limpar o código.

Exemplo:


  class MyClassA constructor() // Redundante, pode ser substituído por 'class MyClassA'

  annotation class MyAnnotation
  class MyClassB @MyAnnotation constructor() // Necessário devido à anotação

  class MyClassC private constructor() // Necessário devido ao modificador de visibilidade
