Relata o uso redundante de await, como await await, ou a espera por resultados que não são promises.

Quando a opção "Relatar para promises" é selecionada, sugere a remoção do await antes de uma promise quando aplicável (em instruções return e quando Promise.resolve/reject estiver presente).

A remoção do await em tais contextos causa dois problemas.

  • Envolver o código com try-catch e esquecer de adicionar o await alterará a semântica do código sem que você perceba.
  • O uso de um await explícito pode impedir que o runtime V8 forneça rastreamentos de pilha assíncronos.
