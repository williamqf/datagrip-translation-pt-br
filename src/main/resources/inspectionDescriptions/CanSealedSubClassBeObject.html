Relata herdeiros diretos de uma sealed class que não possuem estado nem o método equals() sobrescrito.

É altamente recomendável sobrescrever o equals() para garantir a estabilidade da comparação ou converter a class em um object para obter o mesmo efeito.

Exemplo:


  sealed class Receiver {
      class Everyone : Receiver()
      class User(val id: Int) : Receiver()
  }

A correção rápida converterá a class em um object:


  sealed class Receiver {
      object Everyone : Receiver()
      class User(val id: Int) : Receiver()
  }
