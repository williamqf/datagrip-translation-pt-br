Relata referências de método ou expressões lambda que apontam para um método do seu próprio tipo de interface funcional, permitindo a substituição pelo qualificador correspondente para remover alocações de objetos desnecessárias.

Exemplo:


  SwingUtilities.invokeLater(r::run);
  SwingUtilities.invokeAndWait(() -> r.run());

Após a aplicação do ajuste rápido:


  SwingUtilities.invokeLater(r);
  SwingUtilities.invokeAndWait(r);

Esta inspeção relata apenas se o nível de linguagem do projeto ou módulo for 8 ou superior.
