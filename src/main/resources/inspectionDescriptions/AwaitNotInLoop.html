Relata chamadas a java.util.concurrent.locks.Condition.await() que não são feitas dentro de um loop.

await() e métodos relacionados são geralmente usados para suspender uma thread até que uma determinada condição se torne verdadeira. As threads podem ser despertadas por diferentes motivos, portanto, a condição deve ser verificada após o retorno da chamada a await(). O uso de um loop é uma maneira simples de realizar essa verificação.

Exemplo:


  void acquire(Condition released) throws InterruptedException {
    released.await();
  }

O código adequado deve ser semelhante a:


  void acquire(Condition released) throws InterruptedException {
    while (acquired) {
      released.await();
    }
  }
