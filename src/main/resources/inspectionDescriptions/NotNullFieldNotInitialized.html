Relata campos anotados como não nulos que não foram inicializados no construtor.

Exemplo:

public class MyClass {
  private @NotNull String value;

  public void setValue(@NotNull String value) {
    this.value = value;
  }

  public @NotNull String getValue() {
    return value;
  }
}

Tais campos podem violar as restrições de não nulidade. No exemplo acima, o parâmetro de setValue está anotado como não nulo, mas getValue pode retornar null se o setter não for chamado.
