Relata classes utilitárias que possuem um construtor public.

Uma classe utilitária declara todos os campos e métodos como static. Criar um construtor public em tal classe é confuso e pode levar à instanciação acidental da classe.

Exemplo:


  public final class UtilityClass {
    public UtilityClass(){
    }
    public static void foo() {}
  }

Após a aplicação da correção rápida:


  public final class UtilityClass {
    private UtilityClass(){
    }
    public static void foo() {}
  }
