Reporta declarações exportadas que não possuem comentários de documentação. De acordo com Doc Comments no github.com/golang, todos os nomes exportados de nível superior devem possuir comentários de documentação.

Além disso, para detalhes sobre sentenças de comentário, consulte Comment Sentences no github.com/golang.

Para adicionar um comentário, você pode aplicar a correção rápida Adicionar comentário.
