Relata instruções que podem causar efeitos colaterais enquanto a fonte de dados está no modo somente leitura.

Para mais detalhes sobre como ativar o modo somente leitura, consulte Ativar o modo somente leitura para uma conexão na documentação do IDE.

A correção rápida Desativar modo somente leitura desativa o modo somente leitura da fonte de dados correspondente.

Exemplo:


db.my_collection.insertOne()
