Relata chamadas suspeitas a Collection.toArray().

Os seguintes tipos de chamadas são considerados suspeitos:

  • O tipo do argumento do array não é o mesmo que o tipo do array para o qual o resultado é convertido.
  • O tipo do argumento do array não corresponde ao parâmetro de tipo na declaração da coleção.

Exemplo:


void m1(List list) {
  Number[] ns = (Number[]) list.toArray(new String[0]);
}

void m2(List<Number> list) {
  Number[] ns = list.toArray(new String[0]);
}
