Relata expressões que podem ser substituídas por uma chamada a java.util.Objects#equals.

Exemplo:


  void f(Object a, Object b) {
    boolean result = a != null && a.equals(b);
  }

Após aplicar a correção rápida:


  void f(Object a, Object b) {
    boolean result = Objects.equals(a, b);
  }

Substituir expressões como a != null && a.equals(b) por Objects.equals(a, b) altera ligeiramente a semântica. Use a opção Realçar expressões como 'a != null && a.equals(b)' para ativar ou desativar este comportamento.
