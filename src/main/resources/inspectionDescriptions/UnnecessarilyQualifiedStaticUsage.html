Relata o uso de membros static que estão qualificados desnecessariamente pelo nome da classe.

Quando um membro static está disponível na classe envolvente ou em uma superclasse da classe envolvente, a qualificação pela classe não é necessária. Essa qualificação pode ser removida com segurança.

Exemplo:


  class Foo {
    static void foo() {}
    static int x;

    void bar() {
      Foo.foo();
      System.out.println(Foo.x);
    }

    static void baz() { Foo.foo(); }
  }

Após a aplicação da correção rápida:


  class Foo {
    static void foo() {}
    static int x;

    void bar() {
      foo();
      System.out.println(x);
    }

    static void baz() { foo(); }
  }

Use as opções de inspeção para alternar o que é relatado:

  • Acesso a campos static:
    void bar() { System.out.println(Foo.x); }

  • Chamadas a métodos static:
    void bar() { Foo.foo(); }

Além disso, você pode configurar a inspeção para relatar apenas o uso de membros static em um contexto static. Nesse caso, apenas static void baz() { Foo.foo(); } será relatado.
