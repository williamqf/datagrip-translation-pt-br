Relata chamadas suspeitas a System.arraycopy().

Tais chamadas são consideradas suspeitas nos seguintes casos:

  • A origem ou o destino não são tipos de array
  • Os tipos da origem e do destino são diferentes
  • O comprimento do bloco copiado é maior que src.length - srcPos
  • O comprimento do bloco copiado é maior que dest.length - destPos
  • Quando a origem e o destino são o mesmo array, os intervalos sempre se sobrepõem

Exemplo:


  void foo() {
    int[] src = new int[] { 1, 2, 3, 4 };
    System.arraycopy(src, 0, src, 1, 2); // Aviso: copiando para o mesmo array com intervalos que se sobrepõem
  }
