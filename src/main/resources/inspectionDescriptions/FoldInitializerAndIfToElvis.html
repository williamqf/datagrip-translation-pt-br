Relata expressões if que verificam se uma variável é null imediatamente após a inicialização, as quais podem ser convertidas para o operador elvis no inicializador.

Exemplo:


  fun test(foo: Int?, bar: Int): Int {
      var i = foo
      if (i == null) {
          return bar
      }
      return i
  }

A correção rápida converterá a expressão if com o inicializador em uma expressão elvis:


  fun test(foo: Int?, bar: Int): Int {
      var i = foo ?: return bar
      return i
  }
