Relata a negação de isEmpty() e isNotEmpty() para coleções e String, ou de isBlank() e isNotBlank() para String.

O uso das funções correspondentes torna o código mais simples.

A correção rápida substitui a chamada negada pela chamada correspondente na biblioteca padrão.

Exemplo:


  fun main() {
      val list = listOf(1,2,3)
      if (!list.isEmpty()) {
          // faz algo
      }
  }

Após aplicar a correção rápida:


  fun main() {
      val list = listOf(1,2,3)
      if (list.isNotEmpty()) {
          // faz algo
      }
  }
