Reporta expressões when que possuem apenas um ramo else e podem ser simplificadas.

A correção rápida Simplificar expressão pode ser usada para corrigir o código automaticamente.

Exemplo:


  fun redundant() {
      val x = when { // <== redundante, a correção rápida simplificará a expressão when para "val x = 1"
          else -> 1
      }
  }
