Relata inicializações perigosas de array (como $arr[] = value).

Essa prática não é recomendada porque, se $arr já contiver algum valor (por exemplo, uma string de uma variável de requisição), esse valor permanecerá e [] pode, na verdade, representar o operador de acesso a string. É melhor sempre inicializar as variáveis por meio de atribuição direta.

Para obter mais detalhes, consulte Creating/modifying with square bracket syntax (php.net).
