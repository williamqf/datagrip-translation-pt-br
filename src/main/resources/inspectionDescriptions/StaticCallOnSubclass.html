Relata chamadas de método static que são qualificadas por uma subclasse da classe declarante em vez da própria classe declarante.

O Java permite esse tipo de qualificação de classe, mas essas chamadas podem indicar uma confusão sutil entre herança e sobrescrita.

Exemplo:


  class Parent {
    public static void print(String str) {}
  }
  class Child extends Parent {}

  Child.print("Hello, world!");

Após a aplicação da correção rápida:


  Parent.print("Hello, world!");
