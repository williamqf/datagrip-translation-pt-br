Relata um arquivo module-info.java vazio, indicando dependências de módulo não resolvidas. Adiciona automaticamente as instruções requires necessárias ao inspecionar as importações. Para suprimir este aviso, você pode escrever qualquer comentário dentro do corpo da instrução do módulo, como mostrado abaixo: 

module module.name {
    // sem dependências
}
Correção rápida: Preencher dependências do módulo preencherá os requires ausentes com base nas importações do código-fonte. Novidade na 2024.1
