Relata casts seguros com return que podem ser substituídos por verificações de tipo if.

O uso da construção correspondente pode simplificar o código.

A correção rápida substituirá o cast seguro por uma verificação de tipo if.

Exemplo:


  fun test(x: Any) {
      x as? String ?: return
  }

Após a aplicação da correção rápida:


  fun test(x: Any) {
      if (x !is String) return
  }

Da mesma forma, ele detecta casts inseguros no formato:


    fun foo(o: Any) {
    o as String ?: return
}

E os converte em verificações de tipo:


    fun foo(o: Any) {
    if (o !is String) return
}
