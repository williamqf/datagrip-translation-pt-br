Reporta a comparação .length ou .size e sugere substituir por .sizeIs.

Exemplo:


  var seq: Seq[Int]
  seq.length == n
  seq.length != n
  seq.length < n
  seq.length > n

  seq.size == n
  seq.size != n
  seq.size < n
  seq.size > n

Após a correção rápida ser aplicada:


  var seq: Seq[Int]
  seq.sizeIs == n
  seq.sizeIs != n
  seq.sizeIs < n
  seq.sizeIs > n

  seq.sizeIs == n
  seq.sizeIs != n
  seq.sizeIs < n
  seq.sizeIs > n
