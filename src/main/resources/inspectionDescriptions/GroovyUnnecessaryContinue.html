Relata instruções continue que são as últimas instruções acessíveis em um loop. Essas instruções continue são desnecessárias e podem ser removidas com segurança.

Exemplo:


  for(int i in array) {
      println(i)
      continue
  }

Após a aplicação do quick fix:


  for(int i in array) {
      println(i)
  }

Para mais detalhes, consulte a mesma inspeção em Java.
