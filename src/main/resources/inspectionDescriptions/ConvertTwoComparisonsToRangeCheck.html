Relata duas comparações consecutivas que podem ser convertidas em uma verificação de intervalo.

A verificação em relação a um intervalo torna o código mais simples ao remover a duplicação do alvo do teste.

Exemplo:


  fun checkMonth(month: Int): Boolean {
      return month >= 1 && month <= 12
  }

A correção rápida substituirá a verificação baseada em comparação por uma verificação de intervalo:


  fun checkMonth(month: Int): Boolean {
      return month in 1..12
  }
