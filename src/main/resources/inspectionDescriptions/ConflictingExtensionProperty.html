Relata propriedades de extensão que conflitam com propriedades sintéticas geradas automaticamente a partir de métodos get ou set do Java.

Essas propriedades devem ser removidas ou renomeadas para evitar que o código seja quebrado por alterações futuras no compilador.

A correção rápida exclui a propriedade de extensão.

Exemplo:


  val File.name: String
      get() = getName()

A correção rápida adiciona a anotação @Deprecated:


  @Deprecated("Is replaced with automatic synthetic extension", ReplaceWith("name"), level = DeprecationLevel.HIDDEN)
  val File.name: String
      get() = getName()
