Relata métodos clone() que não possuem o modificador final.

O método clone() pode ser usado para instanciar objetos sem o uso de um construtor. Portanto, permitir a sobrescrita do método clone() pode levar à corrupção do objeto ou até mesmo a vulnerabilidades de segurança. Isso pode ser evitado tornando o método clone() ou a própria classe envolvente como final.

Exemplo:


  class Main implements Cloneable {
      @Override
      protected Object clone() throws CloneNotSupportedException {
        return super.clone();
      }
  }
