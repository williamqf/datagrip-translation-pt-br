Relata chamadas de função com argumentos de tipo que podem ser inferidos automaticamente. Tais argumentos de tipo são redundantes e podem ser omitidos com segurança.

Use a correção rápida "Remover argumentos de tipo explícitos" para limpar o código.

Exemplo:


  // O tipo 'String' pode ser inferido aqui
  fun foo(): MutableList<String> = mutableListOf<String>()

  // O tipo 'String' não pode ser inferido aqui, o argumento de tipo é obrigatório.
  fun bar() = mutableListOf<String>()

Após aplicar a correção rápida:


  fun foo(): MutableList<String> = mutableListOf() <== Atualizado

  fun bar() = mutableListOf<String>()
