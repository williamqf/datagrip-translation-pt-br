Reporta chamadas a métodos como Assert.assertEquals(), Assert.assertTrue(), entre outros, que podem ser migrados para chamadas ao estilo declarativo Hamcrest Assert.assertThat().

Exemplo:


  public class SubstantialTest {
    @Test
    public void testContents(Collection<String> c, String s) {
      Assert.assertTrue(c.contains(s));
      Assert.assertEquals(c, s);
      Assert.assertNotNull(c);
      Assert.assertNull(c);
      Assert.assertFalse(c.contains(s));
    }
  }

Uma correção rápida é fornecida para realizar a migração:


  public class SubstantialTest {
    @Test
    public void testContents(Collection<String> c, String s) {
      assertThat(c, hasItem(o));
      assertThat(o, is(c));
      assertThat(c, notNullValue());
      assertThat(c, nullValue());
      assertThat(c, not(hasItem(o)));
    }
  }

Esta inspeção requer que a biblioteca Hamcrest esteja disponível no classpath.

Utilize a opção Importar estaticamente métodos matcher para especificar se deseja que a correção rápida importe estaticamente os métodos matcher do Hamcrest.
