Relata atribuições ou modificações em parâmetros de método.

Embora às vezes intencional, essa construção pode ser confusa e, por isso, é proibida em alguns projetos Java.

A correção rápida adiciona a declaração de uma nova variável.

Exemplo:


  void printTrimmed(String s) {
    s = s.trim();
    System.out.println(s);
  }

Após a aplicação da correção rápida:


  void printTrimmed(String s) {
    String trimmed = s.trim();
    System.out.println(trimmed);
  }

Use a opção Ignorar se a atribuição for uma transformação do parâmetro original para ignorar atribuições que modificam o valor do parâmetro com base no valor anterior.
