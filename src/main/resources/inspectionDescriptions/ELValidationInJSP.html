Relata possíveis problemas de EL, como referências não resolvidas e localizações de EL inválidas.

Exemplo:


Considere a função `exampleFun(String)` definida no namespace atual. 

<jsp:root xmlns:example="urn:jsptld:exampleNamespace.tld">
  ${ example:exampleFun("1") + example:unknownFun("2") } // Erro: Não é possível resolver a função 'unknownFun'
  ${ example:exampleFun("1", "2") } // Erro: O número de parâmetros formais e reais é diferente
</jsp:root>
