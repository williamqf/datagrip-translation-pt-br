Relata beans gerenciados que possuem campos de instância públicos e não estão no escopo padrão @Dependent.

De acordo com a especificação CDI, escopos de contexto (@RequestScoped, @ApplicationScoped, @SessionScoped e ConversationScoped) não são permitidos para beans gerenciados que possuam campos de instância públicos.

Exemplo:


  @ApplicationScoped // Aviso: Beans com escopos que não sejam o padrão não podem ter campos públicos
  class Car {
    public String engine;
  }
