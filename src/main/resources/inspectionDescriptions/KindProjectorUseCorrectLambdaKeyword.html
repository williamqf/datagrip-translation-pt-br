Relatórios usando `λ`.

Ao usar a sintaxe de função do plugin Kind Projector, use `Lambda` ao invés de `λ`. Você pode alterar suas configurações de estilo de código para usar `λ` ao invés de `Lambda` em Configurações | Editor | Estilo de Código | Scala | Outros.

Esta inspeção roda quando o plugin Kind Projector está habilitado.
