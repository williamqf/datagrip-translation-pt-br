Relata classes que sobrescrevem o método clone(), mas não implementam a interface Cloneable. Isso geralmente indica um erro de programação.

Use a opção Avisar apenas sobre métodos clone 'public' para ignorar métodos que não sejam public.

Para classes projetadas para herança, você pode optar por não implementar a interface Cloneable, mas sobrescrever o método clone() declarando-o como protected e decidindo se deseja implementar a interface Cloneable nas subclasses.
