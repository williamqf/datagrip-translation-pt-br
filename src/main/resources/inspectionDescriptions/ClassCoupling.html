Relata classes que referenciam muitas outras classes.

Classes com acoplamento excessivo podem ser muito frágeis e devem ser divididas em classes menores.

Configuração da inspeção:

  • Use o campo Limite de acoplamento de classe para especificar o acoplamento máximo permitido para uma classe.
  • Use a opção Incluir acoplamento com classes do sistema Java para especificar se as referências a classes do sistema (aquelas nos pacotes java. ou javax.) devem ser contabilizadas.
  • Use a opção Incluir acoplamento com classes de biblioteca para especificar se as referências a quaisquer classes de biblioteca devem ser contabilizadas.
