Relata chamadas desnecessárias a close() no final de blocos try-with-resources e sugere a sua remoção.

Exemplo:


  try(MyAutoCloseable ac = new MyAutoCloseable()) {
    foo();
    ac.close();
  }

Após a aplicação do ajuste rápido:


  try(MyAutoCloseable ac = new MyAutoCloseable()) {
    foo();
  }
