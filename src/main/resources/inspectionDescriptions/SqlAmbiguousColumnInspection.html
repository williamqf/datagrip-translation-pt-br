Reporta colunas que possuem o mesmo nome, mas pertencem a tabelas diferentes.

Exemplo (MySQL):

CREATE TABLE foo(id INT PRIMARY KEY);
CREATE TABLE bar(id INT PRIMARY KEY);

SELECT foo.id, bar.id FROM foo, bar WHERE id > 0;

A coluna id aparece nas tabelas foo e bar. Você precisa qualificar o nome da coluna para que a consulta esteja correta.

SELECT foo.id, bar.id FROM foo, bar WHERE foo.id > 0;
