Relata as seguintes violações da especificação EJB:
  • A classe não implementa as interfaces esperadas (EJBLocalObject, EJBObject, etc.)
  • A classe não define um construtor público sem argumentos
  • Uso de uma classe em vez de uma interface ou de uma interface em vez de uma classe
  • Os métodos EJB são static
  • A classe de Bean de Sessão sem estado define múltiplos métodos ejbCreate
  • O tipo de retorno do método da classe Session Bean está incorreto
  • O EJB define o método finalize

Exemplo:


public class SessionEJB implements java.io.Serializable { // Erro: a classe deve estender `SessionBean`
  private SessionEJB() {} // Erro: a classe deve definir um construtor público sem argumentos
}
