Relata extensões que são instanciadas pela plataforma IntelliJ, mas declaradas como objetos Kotlin.

O ciclo de vida da extensão é gerenciado pela plataforma IntelliJ. O uso de objetos Kotlin para o registro de extensões pode levar à criação de instâncias de extensão desnecessárias e impedir que o plugin seja descarregado.

Exemplo

Registro da extensão:


<annotator language="myLang" implementationClass="com.example.MyAnnotator"/>

Implementação da extensão:


// Errado:
object MyAnnotator : Annotator {
  ...
}

// Correto:
class MyAnnotator : Annotator {
  ...
}
