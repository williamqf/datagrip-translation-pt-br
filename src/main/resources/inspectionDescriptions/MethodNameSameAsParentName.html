Relata métodos com o mesmo nome que uma superclasse da classe do método, pois tais nomes de métodos podem ser confusos.

Esta inspeção não verifica interfaces ou superclasses profundas na hierarquia.

Exemplo: 


  class Parent {}
  class Child extends Parent {
     public Parent Parent() {
       return null;
     }
  }

A correção rápida para renomear tais métodos está disponível apenas no editor.
