Relata os parâmetros de classe que sombreiam uma var de superclasse.

Exemplo:

class Animal {
  var name: String = ""
}

class Cat(name: String = "") extends Animal {
  // Torna `name` uma `val` private[this], que sombreia a `var`
  def uso = name
}

Ao estender uma classe com uma var não private, o usuário pode esperar que um parâmetro de classe da subclasse sobrescreva a var da superclasse, mas na verdade irá apenas sombreá-la. Isso pode às vezes levar a erros não triviais.

Observe que o destaque só aparece se o valor for realmente usado.
