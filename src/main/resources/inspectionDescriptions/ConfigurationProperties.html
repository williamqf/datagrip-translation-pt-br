Relata prefixos inválidos definidos na anotação @ConfigurationProperties:

  • Prefixo ausente
  • Prefixo vazio
  • Prefixo duplicado
  • Prefixo em uma notação diferente de kebab-case

A inspeção é acionada em classes anotadas com @ConfigurationProperties que não estão registradas via @EnableConfigurationProperties, marcadas como componentes Spring ou detectadas via @ConfigurationPropertiesScan.

Quando a versão do Spring Boot do projeto ou módulo é definida como 2.2.0 ou superior, esta inspeção é acionada apenas em classes anotadas com @ConfigurationProperties que não são detectadas via @ConfigurationPropertiesScan.

Novidade na versão 2018.3
