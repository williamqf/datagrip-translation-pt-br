Relata chamadas ao método values() de classes enum a partir do Kotlin que podem ser substituídas pela leitura da propriedade entries.

O uso de Enum.entries pode melhorar o desempenho do código.

A correção rápida substitui values() por entries.

Mais detalhes: KT-48872 Fornecer substituição moderna e eficiente para Enum.values()

Observação: o tipo da propriedade entries é diferente do tipo de retorno do método values() (EnumEntries<T> herda de List<T>, em vez de Array<T>). Portanto, em alguns casos, a correção rápida inserirá uma conversão adicional .toTypedArray() para evitar a quebra do código, mas na maioria dos casos a substituição será feita sem essa conversão (por exemplo, em loops for).

Exemplo:


    enum class Version {
        V1, V2
    }

    Version.values().forEach { /* .. */ }
    val firstVersion = Version.values()[0]
    functionExpectingArray(Version.values())

Após a aplicação da correção rápida:


    enum class Version {
        V1, V2
    }

    Version.entries.forEach { /* .. */ }
    val firstVersion = Version.entries[0]
    functionExpectingArray(Version.entries.toTypedArray())
