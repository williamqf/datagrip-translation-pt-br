Relata métodos ou campos de serialização definidos em uma classe record. Os métodos de serialização incluem writeObject(), readObject(), readObjectNoData(), writeExternal() e readExternal(), bem como o campo serialPersistentFields. Esses membros não são usados para serialização ou desserialização de registros e, portanto, são desnecessários.

Exemplo:


  record R1() implements Serializable {
    // Este campo é ignorado durante a serialização do registro
    @Serial
    private static final ObjectStreamField[] serialPersistentFields = new ObjectStreamField[0];

    // Este método é ignorado durante a serialização do registro
    @Serial
    private void writeObject(ObjectOutputStream out) throws IOException {
    }
  }


  record R2() implements Externalizable {
    // Este método é ignorado durante a serialização do registro
    @Override
    public void writeExternal(ObjectOutput out) throws IOException {
    }

    // Este método é ignorado durante a serialização do registro
    @Override
    public void readExternal(ObjectInput in) throws IOException, ClassNotFoundException {
    }
  }
