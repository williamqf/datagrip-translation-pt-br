Relata chamadas de java.lang.String.concat().

Se o método for chamado em uma constante com um argumento constante, tais chamadas podem ser substituídas pelo operador + para melhorar a clareza e o desempenho.

Exemplo:


  String foo(String name) {
    return name.concat("foo");
  }

Após a aplicação da correção rápida:


  String foo(String name) {
    return name + "foo";
  }
