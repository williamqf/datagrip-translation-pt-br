Relata expressões literais de função que podem ser substituídas por referências de função.

Substituir um lambda por uma referência de função geralmente torna o código mais conciso e fácil de entender.

Exemplo:


  fun Int.isEven() = this % 2 == 0

  fun example() {
      val numbers = listOf(1, 2, 4, 7, 9, 10)
      val evenNumbers = numbers.filter { it.isEven() }
  }

Após aplicar a correção rápida:


  fun Int.isEven() = this % 2 == 0

  fun example() {
      val numbers = listOf(1, 2, 4, 7, 9, 10)
      val evenNumbers = numbers.filter(Int::isEven)
  }
