ChangeScope relata as propriedades não atribuídas de Entity ao criar com as funções new ou getOrCreate.

Todas as propriedades não anuladas de Entity devem ser atribuídas ou anotadas com @Many. Caso contrário, uma exceção de tempo de execução será lançada.

Exemplo:


  interface A: Entity {
    var x: Int
    var y: Int?
    @Many var z: Int
    var p: Int
  }

  fun ChangeScope.foo() {
    new(A::class) {
      this.p = 0
    }
  }

A correção rápida gera atribuições para propriedades obrigatórias ausentes


  fun ChangeScope.foo() {
    new(A::class) {
      this.x = x
      this.p = 0
    }
  }
