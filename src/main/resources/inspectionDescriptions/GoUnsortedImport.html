Reporta imports não ordenados.

Todos os programas Go devem ser formatados da mesma maneira, com regras de formatação estabelecidas pela ferramenta gofmt. Essas regras exigem que os imports sejam ordenados.

Exemplo de ordenação incorreta:

import (
 "net"
 "errors"
 "fmt"
)

Você pode aplicar a correção rápida Ordenar imports para corrigir o problema de ordenação. Após aplicar a correção rápida, a ordenação será a seguinte:

import (
 "errors"
 "fmt"
 "net"
)
