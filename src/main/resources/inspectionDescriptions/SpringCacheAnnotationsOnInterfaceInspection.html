Relata anotações @Cache* em interfaces.

Você deve usar anotações @Cache* apenas em classes concretas (e métodos de classes concretas). Anotar interfaces (ou métodos de interface) com @Cache* requer o uso de proxies baseados em interface. Como as anotações Java não são herdadas de interfaces, ao usar proxies baseados em classe (proxy-target-class="true") ou aspectos baseados em weaving (mode="aspectj"), a infraestrutura de proxy e weaving não reconhecerá as configurações de cache. Como resultado, o objeto não será envolvido em um proxy de cache.
