Relata uma propriedade data de um componente Vue que não é uma função. Sugere envolver um literal de objeto com uma função.

Ao definir um componente, data deve ser declarada como uma função que retorna o objeto de dados inicial, porque a mesma definição será usada para criar inúmeras instâncias. Se um objeto simples ainda for usado para data, esse objeto será compartilhado por referência entre todas as instâncias criadas! Com uma função data, toda vez que uma nova instância é criada, podemos simplesmente chamá-la para retornar uma nova cópia dos dados iniciais.
