Relata instâncias em que ponteiros que podem ser nil são desreferenciados.

Exemplo:


type Person struct{ Name string }

func _(person *Person) {
	if person != nil || person.Name != "Peter" {
	}
}

No exemplo fornecido, person é um ponteiro para uma estrutura Person, mas é nil, pois não há atribuição de valor para ele. Tentar desreferenciar person para acessar Name sem checar se person é nil resulta em um erro de tempo de execução.
