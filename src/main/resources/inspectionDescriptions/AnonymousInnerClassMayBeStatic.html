Relata classes anônimas que podem ser substituídas com segurança por classes internas static. Uma classe anônima pode ser uma classe interna static se não referenciar explicitamente sua instância envolvente ou classes locais do método circundante.

Classes internas static não mantêm uma referência implícita para sua instância envolvente. Isso evita causas comuns de vazamentos de memória e reduz o consumo de memória por instância de classe.

A partir do Java 18, apenas classes anônimas serializáveis mantêm uma referência implícita para sua instância envolvente se essa referência não for usada. Portanto, se o nível de linguagem do módulo for Java 18 ou superior, esta inspeção relatará apenas classes serializáveis.

A correção rápida extrai a classe anônima em uma classe interna static nomeada.

Exemplo:


  void sample(){
    Thread thread = new Thread(new Runnable() {
      @Override
      public void run() {
      }
    });
  }

Após aplicar a correção rápida:


  void sample() {
    Thread thread = new Thread(new Task());
  }

  private static class Task implements Runnable {
    @Override
    public void run() {
    }
  }
