Relata chamadas de construtor em supertipos de função que causarão erros de compilação a partir da versão 1.9.

Tipos de motivação:

  • A implementação não cumpre a especificação ou documentação publicada

Mais detalhes: KT-46344: Nenhum erro na chamada de construtor de superclasse em interface funcional na lista de supertipos

A correção rápida remove a chamada do construtor.

Exemplo:


  abstract class A : () -> Int()

Após aplicar a correção rápida:


  abstract class A : () -> Int

Esta inspeção relata apenas se o nível da linguagem Kotlin do projeto ou módulo for 1.7 ou superior.
