Relata instruções if que podem ser substituídas por condições usando os operadores &&, ||, ==, != ou ?:.

O resultado é geralmente mais curto, mas nem sempre mais claro, portanto não é recomendado aplicar esta correção em todos os casos.

Exemplo:


  if (condition) return true; else return foo;

Após a aplicação da correção rápida:


  return condition || foo;

Configuração da inspeção:

  • Use a opção Não sugerir o operador '?:' para desativar avisos quando o operador ?: for sugerido. Neste caso, apenas sugestões para &&, ||, == e != serão realçadas. A correção rápida ainda estará disponível no editor.
  • Use a opção Ignorar instruções 'if' encadeadas para desativar avisos para cadeias if-else. A correção rápida ainda estará disponível no editor.
