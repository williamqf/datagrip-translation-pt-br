Relata instruções import não utilizadas no código Python.

Esta inspeção detecta instruções import que não são usadas no código e podem ser removidas com segurança. Remover imports não utilizados ajuda a manter o código limpo e reduz o risco de conflitos de nomes.

Exemplo:


import os           # Import não utilizado
import sys         # Import utilizado
from math import pi # Import não utilizado

print(sys.version)

Esta inspeção fornece uma correção rápida para otimizar imports que remove todas as instruções import não utilizadas.

Observe que alguns imports podem ser usados indiretamente (por exemplo, para efeitos colaterais) e não devem ser removidos. Se certos imports forem necessários devido a efeitos colaterais, você pode suprimir esta inspeção para eles.
