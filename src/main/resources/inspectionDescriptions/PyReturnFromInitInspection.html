Relata instruções return com um valor de retorno dentro do método __init__ de uma classe.

Exemplo:


class Sum:
    def __init__(self, a, b):
        self.a = a
        self.b = b
        self.sum = a + b
        return self.sum

Construtores não devem retornar nenhum valor. O método __init__ deve apenas inicializar os valores dos membros da instância do novo objeto.

A correção rápida proposta pelo IDE é remover a instrução return.
