Relata elementos incorretos no arquivo <validation-config> do Bean Validation:

  • Referências a classes que não podem ser instanciadas
  • Referências a classes que não estendem as classes necessárias
  • Referências a classes com escopo inadequado
  • Tags e valores de atributos vazios
  • Valores de tags e atributos que não correspondem aos padrões exigidos (por exemplo, identificadores Java)
  • Tags que não contêm as subtags ou atributos obrigatórios
  • Tags que definem objetos com nomes duplicados

Exemplo:


<validation-config>
  <default-provider>foo.UnknownProvider</default-provider>  <!-- Não é possível resolver a classe ou o pacote 'UnknownProvider' -->
  <message-interpolator>my.Interpolator</message-interpolator>
  <constraint-mapping>META-INF/ConstraintMappings.xml</constraint-mapping>
  <constraint-mapping>META-INF/unknown.xml</constraint-mapping>
</validation-config>
