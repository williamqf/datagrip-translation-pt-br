Relata classes anotadas com @com.intellij.openapi.components.Service que não são final.

Se uma classe for concreta, sugere torná-la final.

Exemplo:


  // MyService.kt
  @Service(Service.Level.APP)
  open class MyService

Após a aplicação da correção rápida:


  // MyService.kt
  @Service(Service.Level.APP)
  class MyService

Se for uma classe abstrata ou interface, sugere remover a anotação @Service.

Exemplo:


  // MyService.java
  @Service(Service.Level.APP)
  abstract class MyService {}

Após a aplicação da correção rápida:


  // MyService.java
  abstract class MyService {}
