Relata referências não resolvidas em comentários Javadoc.

No exemplo a seguir, o parâmetro someParam será destacado porque está ausente:


class A {
    /**
     * @param someParam descrição
     **/
    void foo() {
    }
}

Desative a opção Relatar símbolos inacessíveis para ignorar tags que referenciam parâmetros de método, classes, campos e métodos ausentes.
