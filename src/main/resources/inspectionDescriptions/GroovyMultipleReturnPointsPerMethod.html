Relata métodos com muitos pontos de retorno. Métodos com pontos de retorno excessivos podem ser confusos e difíceis de refatorar.

Exemplo:


  int foo(int a) {
    if (a > 0) {
      return a
    }
    if (a < 0) return -a
    return 0
  }

Utilize o campo fornecido abaixo para especificar o número máximo aceitável de pontos de retorno em um método.
