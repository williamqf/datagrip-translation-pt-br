Relata funções que não possuem uma instrução RETURN.

Exemplo (Oracle):

CREATE FUNCTION foo RETURN int AS
BEGIN
END;

A função foo deve retornar um valor inteiro, mas o corpo da função não retorna nada. Para corrigir o erro, adicione uma instrução RETURN (por exemplo, RETURN 1;).

CREATE FUNCTION foo RETURN int AS
BEGIN
 RETURN 1;
END;
