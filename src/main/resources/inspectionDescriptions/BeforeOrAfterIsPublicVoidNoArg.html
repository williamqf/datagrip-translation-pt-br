Métodos JUnit 4 @Before ou @After não declarados como public, que não retornam void ou recebem parâmetros são relatados. Tais métodos podem ser criados inadvertidamente, mas não serão executados pelo executor de testes JUnit.

Esta inspeção fornecerá uma correção da assinatura do método.

Exemplo:


  @Before private int foo(int arg) { ... }

Após aplicação da correção rápida:


  @Before public void foo() { ... }
