Relata blocos de código que são semanticamente redundantes no programa e que podem ser substituídos pelo seu conteúdo.

Blocos de código que servem como corpo de instruções if, do, while ou for não serão relatados por esta inspeção.

Exemplo:


  void foo() {
    { // desnecessário
      int result = call();
      analyze(result);
    } // desnecessário
  }

Configuração da inspeção:

Utilize a opção Ignorar ramos da instrução 'switch' para ignorar blocos de código usados como ramos de uma instrução switch.
