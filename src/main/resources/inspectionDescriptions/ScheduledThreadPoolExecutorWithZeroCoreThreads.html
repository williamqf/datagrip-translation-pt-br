Relata quaisquer instâncias de java.util.concurrent.ScheduledThreadPoolExecutor que tenham o corePoolSize definido como zero por meio do método setCorePoolSize ou do construtor do objeto.

Um ScheduledThreadPoolExecutor com zero threads core não executa nada.

Exemplo:


  void foo(int corePoolSize) {
      if (corePoolSize != 0) return;
      ThreadPoolExecutor executor = new ScheduledThreadPoolExecutor(corePoolSize); // aviso
      executor.setCorePoolSize(corePoolSize); // aviso
  }
