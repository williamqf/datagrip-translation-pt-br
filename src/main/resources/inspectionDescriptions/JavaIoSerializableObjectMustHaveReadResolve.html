Relata um object (incluindo data object) que implementa java.io.Serializable, mas não implementa readResolve

Exemplo:


  import java.io.Serializable

  object Foo : Serializable

A correção rápida implementa o método readResolve:


  import java.io.Serializable

  object Foo : Serializable {
      private fun readResolve() = Foo
  }
