Relata expressões lambda que contêm uma única referência chamável (callable reference).

Substituir uma lambda por uma referência chamável sem alterar as chaves para parênteses é um erro comum.

Exemplo:


  val numbers = listOf(1, 2, 3)
  val result = numbers.map { it::toString }

Após a aplicação da correção rápida:


  val numbers = listOf(1, 2, 3)
  val result = numbers.map(Int::toString) // o tipo real entra aqui
