Detecta o uso de métodos de configuração obsoletos.

Métodos de configuração podem se tornar obsoletos por um destes dois motivos:
  • A API do Gradle evoluiu, portanto, a escolha de métodos antigos deve ser evitada;
  • Um plugin fornece uma versão melhor do método de configuração obsoleto.
Na maioria das vezes, haverá uma alternativa para substituir o método obsoleto.

Exemplo:


plugins {
  id 'java'
}

dependencies {
  archive 'org.gradle.api:gradle:1.0' // reporta 'archive'
}
