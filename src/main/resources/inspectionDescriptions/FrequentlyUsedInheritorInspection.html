Relata classes ou interfaces que podem ser substituídas por implementações ou extensões de classes ou interfaces mais específicas e comumente usadas.

Para que esta inspeção funcione, a superclasse deve estar nos arquivos de origem do projeto e o projeto deve usar o sistema de compilação do IntelliJ IDEA.

Exemplo:


  class MyInheritor implements A {} // B sugerido para a referência a A

  interface A {}

  abstract class B implements A {}

  abstract class C1 extends B {}
  abstract class C2 extends B {}
  abstract class C3 extends B {}
  abstract class C4 extends B {}
  abstract class C5 extends B {}

Por padrão, esta inspeção não destaca problemas no editor; em vez disso, fornece apenas correções rápidas.
