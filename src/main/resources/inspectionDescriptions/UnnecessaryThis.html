Relata qualificadores this desnecessários.

O uso de this para desambiguar referências de código é desencorajado em muitos estilos de codificação e pode facilmente tornar-se desnecessário por meio de refatoração automática.

Exemplo:


  class Foo {
    int x;
    void foo() {
      this.x = 2;
    }
  }

Após a aplicação da correção rápida:


  class Foo {
    int x;
    void foo() {
      x = 2;
    }
  }

Use as configurações de inspeção para ignorar atribuições a campos. Por exemplo, this.x = 2; não será relatado, mas int y = this.x; será.
