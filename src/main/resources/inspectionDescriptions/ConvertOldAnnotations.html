Relata a anotação org.testng.annotations.Configuration do TestNG.

É preferível usar as anotações modernas @BeforeXXX/@AfterXXX para configurar a suíte de testes.

Exemplo:


  public class Test {
    @Configuration(beforeSuite = true, afterTest = true)
    public void afterBefore(){
    }
  }

Após a aplicação da correção rápida:


  public class Test {
    @AfterTest()
    @BeforeSuite()
    public void afterBefore(){
    }
  }
