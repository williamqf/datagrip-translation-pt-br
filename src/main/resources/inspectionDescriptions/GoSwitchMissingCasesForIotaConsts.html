Relata valores não tratados em uma instrução switch quando o identificador iota está presente em uma declaração const. Para tratar esses valores, considere usar uma cláusula case ou default.

A palavra-chave iota no Go cria uma sequência de inteiros em um bloco const. Embora o compilador Go não exija o tratamento de todos os valores iota, a ausência de alguns deles pode indicar um bug.

Para mais detalhes sobre iota, consulte Iota em go.dev.

Exemplo:

const (
  a = iota
  b
  c = -5
)

func main() {
  v := 5
  switch v {
  case a:
    break;
  case b:
    break;
  }
}

O IDE aciona a inspeção porque c não é tratado pela instrução switch. Observe que c está no mesmo bloco const, mesmo que não use iota.

As opções de correção rápida são Criar cláusula 'case' para valores e Criar cláusula 'default'. Após aplicar a correção rápida Criar cláusula 'default':

const (
  a = iota
  b
  c
)

func main() {
  v := 5
  switch v {
  case a:
    break;
  default:
  }
}
