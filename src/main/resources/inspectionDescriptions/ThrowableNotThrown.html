Relata a instanciação de Throwable ou de suas subclasses quando o Throwable criado nunca é realmente lançado. Além disso, esta inspeção relata chamadas de método que retornam uma instância de Throwable ou de suas subclasses quando o resultado da chamada de método não é lançado.

Chamadas para métodos anotados com a anotação @CanIgnoreReturnValue do Error Prone ou do AssertJ não são relatadas.

Exemplo:


    void check(String s) {
        if (s == null) {
            new NullPointerException("s");
        }
        // ...
    }
