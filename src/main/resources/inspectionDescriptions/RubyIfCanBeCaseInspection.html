thoughtful Relata instruções if que podem ser substituídas por instruções case.

Use o campo Número mínimo de ramificações para especificar o número mínimo de ramificações when para a instrução case resultante.

Exemplo:

fruit = gets

if fruit == 'apple' # Instrução 'if' simplificável
  puts 'This is apple!'
elsif fruit == 'peach'
  puts 'This is peach!'
elsif fruit == 'orange'
  puts 'This is orange!'
elsif fruit == 'banana'
  puts 'This is banana!'
end
