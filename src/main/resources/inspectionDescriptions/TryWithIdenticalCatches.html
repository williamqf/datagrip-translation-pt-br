Relata seções catch idênticas em uma única instrução try.

Agrupar essas seções em um único bloco multi-catch reduz a duplicação de código e evita situações em que uma seção catch é atualizada enquanto a outra não é.

Exemplo:


    try {
        doSmth();
    }
    catch (IOException e) {
        LOG.error(e);
    }
    catch (URISyntaxException e) {
        LOG.error(e);
    }

Uma correção rápida está disponível para tornar o código mais compacto:


    try {
        doSmth();
    }
    catch (IOException | URISyntaxException e) {
        LOG.error(e);
    }
