Relata dicas ausentes ou redundantes na criação do UastHintedVisitorAdapter.

O UastHintedVisitorAdapter recebe dicas que permitem otimizar a execução do visitor UAST ao acessar apenas elementos das classes especificadas na lista de dicas.

Por exemplo, se o visitor implementar o método visitCallExpression, mas não incluir UCallExpression nas dicas, o método não será executado.

Por exemplo, se o visitor não implementar o método visitCallExpression, mas incluir UCallExpression nas dicas, o método herdado será executado desnecessariamente.

Exemplos


return UastHintedVisitorAdapter.create(
  language,
  object : AbstractUastNonRecursiveVisitor() {
    override fun visitForExpression(node: UForExpression): Boolean {
      inspectLoopExpression()
      return true
    }

    override fun visitDoWhileExpression(node: UDoWhileExpression): Boolean { // não usado em tempo de execução
      inspectLoopExpression()
      return true
    }
    //...
  },
  arrayOf(UForExpression::class.java) // Erro: faltando UDoWhileExpression::class.java
)


UastHintedVisitorAdapter.create(
  language,
  object : AbstractUastNonRecursiveVisitor() {
    override fun visitForExpression(node: UForExpression): Boolean {
      inspectLoopExpression()
      return true
    }
    //...
  },
  arrayOf(
    UForExpression::class.java,
    UDoWhileExpression::class.java // Erro: visita a UDoWhileExpression não implementada
  )
)
