Relata instruções que podem modificar o banco de dados durante uma conexão somente leitura.

Para habilitar o modo somente leitura para uma conexão, clique com o botão direito na fonte de dados na janela de ferramentas Banco de Dados (Exibir | Janelas de Ferramentas | Banco de Dados) e selecione Propriedades. Na caixa de diálogo Fontes de Dados e Drivers, clique na guia Opções e marque a caixa de seleção Somente leitura.

Exemplo (MySQL):

CREATE TABLE foo(a INT);
INSERT INTO foo VALUES (1);

Como as instruções CREATE TABLE e INSERT INTO resultam em modificações no banco de dados, essas instruções serão realçadas no modo de conexão somente leitura.
