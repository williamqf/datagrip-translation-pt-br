Relata chamadas ao operador rangeTo ou .. em vez de chamadas a until.

O uso da função correspondente torna o código mais simples.

A correção rápida substitui chamadas rangeTo ou .. por until.

Exemplo:


  fun foo(a: Int) {
      for (i in 0..a - 1) {

      }
  }

Após a aplicação da correção rápida:


  fun foo(a: Int) {
      for (i in 0 until a) {

      }
  }
