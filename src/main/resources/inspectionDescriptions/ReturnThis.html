Relata métodos que retornam this.

Embora tais retornos sejam válidos, eles raramente são necessários e geralmente indicam que o método se destina a ser usado como parte de uma cadeia de chamadas de métodos semelhantes (por exemplo, buffer.append("foo").append("bar").append("baz")). Muitos padrões de codificação desaprovam tais cadeias.

Exemplo:


  public Builder append(String str) {
    // [...]
    return this;
  }
