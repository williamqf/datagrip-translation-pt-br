Relata chamadas de métodos Java que podem ser substituídas por funções Kotlin, como System.out.println().

A substituição desse código remove dependências do Java e torna o código Kotlin mais idiomático.

A correção rápida substitui a chamada do método Java pela chamada Kotlin equivalente.

Exemplo:


  import java.util.Arrays

  fun main() {
      val a = Arrays.asList(1, 3, null)
  }

Após a aplicação da correção rápida:


  fun main() {
      val a = listOf(1, 3, null)
  }
