Relata variáveis static lidas antes da inicialização.

Esta inspeção ignora verificações de igualdade com null.

Exemplo:


  class Foo {
    public static int bar;

    public static void main(String[] args) {
      System.out.println(bar);
    }
  }

Observe que esta inspeção usa um algoritmo de fluxo de dados muito conservador e pode relatar incorretamente variáveis static como não inicializadas. Variáveis relatadas como inicializadas serão sempre inicializadas.

Use a opção Ignorar campos primitivos para ignorar campos primitivos não inicializados.
