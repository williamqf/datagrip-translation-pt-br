Relata variáveis que possuem nomes "padrão" que não correspondem aos seus tipos. Esses nomes podem ser confusos. Os nomes padrão para tipos específicos são os seguintes:
  • i, j, k, m, n - int
  • f - float
  • d - double
  • b - byte
  • c, ch - char
  • l - long
  • s, str - String

A correção rápida para renomear é sugerida apenas no editor.

Use a opção para ignorar nomes de parâmetros que sejam idênticos aos nomes de parâmetros em um supermétodo direto.
