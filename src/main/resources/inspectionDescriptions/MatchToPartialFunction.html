Relata instruções que podem ser convertidas em funções anônimas com equiparação de padrões.

Exemplo:


  iterable.map(valor => valor match para {
      Alguns(valor) => whenValue(valor)
      Nenhum => whenNothing()
  })

Após a aplicação da correção rápida:


  iterable.map {
      Alguns(valor) => whenValue(valor)
      Nenhum => whenNothing()
  }
