Relata regras de definição de Bean incorretas.

Uma classe Java de nível superior pode ser um Bean simples se:

  • Não for um tipo parametrizado.
  • Não for uma classe interna não estática (non-static).
  • For uma classe concreta ou estiver anotada com @Decorator.
  • Possuir um construtor apropriado: a classe possui um construtor sem parâmetros ou declara um construtor anotado com @Initializer.
  • Não implementar nenhuma das seguintes interfaces: javax.servlet.Servlet, javax.servlet.Filter, javax.servlet.ServletContextListener, javax.servlet.http.HttpSessionListener, javax.servlet.ServletRequestListener, javax.ejb.EnterpriseBean.
  • Não estender javax.faces.component.UIComponent.
