Relata problemas de configuração em classes Seam com anotações

Exemplo:


@Name("authenticator")
class SeanComponent {
  @Create // Não é permitido usar métodos estáticos
  private static void create() {...}

   @Destroy // Não é permitido usar métodos estáticos
   private static void destroy() {...}

   @DataModel // Não é um getter
   private Map method() {...}

   @Factory // Deve ser public
   private Map getUniverse2() {...}

   @Unwrap // Deve ser sem parâmetros
   private static Map createUniverse(int days) { ... }
}
