Relata tipos de parâmetro explicitamente especificados em setters de propriedade.

O tipo do parâmetro do setter sempre corresponde ao tipo da propriedade, portanto, não é necessário que ele seja explícito. A correção rápida 'Remover especificação de tipo explícita' permite modificar o código adequadamente.

Exemplo:


  fun process(x: Int) {}

  var x: Int = 0
      set(value: Int) = process(value) // <== a especificação 'Int' pode ser omitida com segurança

Após a aplicação da correção rápida:


  fun process(x: Int) {}

  var x: Int = 0
      set(value) = process(value)
