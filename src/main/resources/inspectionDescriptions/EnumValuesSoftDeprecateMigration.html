Relata chamadas ao método values() em classes de enumeração do Kotlin que podem ser substituídas pela leitura da propriedade entries.

Usar Enum.entries pode melhorar o desempenho do código.

Essa correção rápida substitui values() por entries.

Mais detalhes: KT-48872 Fornece uma substituição moderna e de alto desempenho para Enum.values()

Observação: o tipo da propriedade entries é diferente do tipo de retorno do método values() (EnumEntries<T> herda de List<T>, não de Array<T>). Portanto, em alguns casos a correção rápida insere uma conversão adicional .toTypedArray() para não quebrar o código, mas na maioria dos casos simples a substituição é feita sem essa conversão (por exemplo, em um loop for).

Exemplo:


    enum class Version {
        V1, V2
    }

    Version.values().forEach { /* .. */ }
    val firstVersion = Version.values()[0]
    functionExpectingArray(Version.values())

Após aplicar a correção rápida:


    enum class Version {
        V1, V2
    }

    Version.entries.forEach { /* .. */ }
    val firstVersion = Version.entries[0]
    functionExpectingArray(Version.entries.toTypedArray())
