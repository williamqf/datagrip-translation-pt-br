Relata referências a variáveis que são verificadas quanto à nulidade em uma condição de instrução if ou expressão condicional, mas não são utilizadas dentro dessa instrução if.

Isso geralmente significa que a verificação é desnecessária ou que a variável não foi referenciada dentro da instrução if por engano.

Exemplo:


  void test(Integer i) {
    if (i != null) { // aqui 'i' não é utilizado dentro da instrução 'if'
      System.out.println();
    }
  }
