Relata código do lado do cliente em um módulo GWT que utiliza código do lado do cliente de outro módulo que não foi herdado no módulo atual.

O código do lado do cliente de um módulo GWT só pode utilizar o código do lado do cliente de outro módulo GWT se houver uma tag inherits correspondente no arquivo gwt.xml.

A correção rápida adiciona a tag inherits necessária ao arquivo gwt.xml.
