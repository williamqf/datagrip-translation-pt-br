Relata referências redundantes a Companion.

Exemplo:


  class A {
      companion object {
          fun create() = A()
      }
  }
  fun test() {
      val s = A.Companion.create()
  }

Após a aplicação da correção rápida:


  class A {
      companion object {
          fun create() = A()
      }
  }
  fun test() {
      val s = A.create()
  }
