Relata chamadas de função toString que podem ser substituídas por um template de string.

O uso de templates de string torna o código mais simples.

A correção rápida substitui o toString por um template de string.

Exemplo:


  fun test(): String {
      val x = 1
      return x.toString()
  }

Após a aplicação da correção rápida:


  fun test(): String {
      val x = 1
      return "$x"
  }
