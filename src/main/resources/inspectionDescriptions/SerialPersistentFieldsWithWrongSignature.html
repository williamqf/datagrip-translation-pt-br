Relata classes Serializable cujo campo serialPersistentFields não está declarado como private static final ObjectStreamField[].

Se o campo serialPersistentFields não for declarado com esses modificadores, o comportamento da serialização será exatamente como se o campo não tivesse sido declarado de forma alguma.

Exemplo:


  class List implements Serializable {
    private List next;

    ObjectStreamField[] serialPersistentFields = {new ObjectStreamField("next", List.class)};

  }
