Relata o uso de propriedades imutáveis em entidades JPA validadas.

Detecta propriedades imutáveis anotadas com @NotNull. Sugere torná-las mutáveis para garantir a compatibilidade com o Hibernate.

Trecho de código incorporado: 


@Entity
open class User(
  @Id
  val id: Long = 0L,
  @NotNull
  val name: String = "",
)
Após a aplicação da correção rápida: 

@Entity
open class User(
  @Id
  val id: Long = 0L,
  @NotNull
  var name: String = "",
)
