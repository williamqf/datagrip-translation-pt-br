Relata instâncias em que uma string não vazia é passada como o parâmetro $escape para métodos e funções relacionados a CSV.
  • SplFileObject::setCsvControl()
  • SplFileObject::fputcsv()
  • SplFileObject::fgetcsv()
  • fputcsv()
  • fgetcsv()
  • str_getcsv().

Passar uma string não vazia como o parâmetro $escape está descontinuado a partir do PHP 8.4.
