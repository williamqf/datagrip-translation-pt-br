Relata tentativas de criar colunas NOT NULL sem um valor DEFAULT.

Exemplo (Microsoft SQL Server):

CREATE TABLE foo (a INT, b  INT)

ALTER TABLE foo ADD c INT NOT NULL;

Por padrão, as colunas contêm valores NULL. No exemplo, usamos a restrição NOT NULL para forçar a coluna a não aceitar valores NULL. Se proibirmos o uso de valores NULL, devemos definir um valor DEFAULT que o SQL possa usar ao criar novos registros. 

ALTER TABLE foo ADD c INT NOT NULL DEFAULT 42;

Você pode usar o ajuste rápido Adicionar valor DEFAULT para adicionar rapidamente um valor DEFAULT.
