Relata chamadas de método em coleções parametrizadas onde o tipo do argumento real não é compatível com o tipo do elemento da coleção.

Exemplo:


  List<Integer> list = getListOfElements();
  list.remove(""); // remove é realçado

Nas configurações da inspeção, é possível desativar avisos para códigos que podem estar corretos, como mostrado abaixo: 


  public boolean accept(Map<Integer, String> map, Object key) {
    return map.containsKey(key);
  }
