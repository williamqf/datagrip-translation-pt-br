Relata instruções switch aninhadas em outras instruções switch. Tais instruções switch aninhadas são confusas e podem levar a comportamentos inesperados.

Exemplo:


switch (outer) {
  case 1:
    switch (inner) {
      case 1:
        print "inner 1"
        break
      default:
        print "inner default"
    }
    break
  default:
    print "default"
}
