Relata campos que não têm a inicialização garantida após um objeto ser desserializado pelo método readObject().

Esta inspeção não relata campos transient.

Nota: esta inspeção usa um algoritmo de fluxo de controle muito conservador e pode relatar incorretamente campos como não inicializados.

Exemplo:


class DataObject implements Serializable {
  String s; // s não é inicializado em readObject
  int i;

  private void readObject(ObjectInputStream stream) throws IOException {
    i = stream.readInt();
  }
}
