Relata a ausência de caracteres de continuação no comando RUN.

Na forma shell do comando RUN, você deve usar uma '\' (barra invertida) para continuar uma única instrução RUN na próxima linha. Caso contrário, o build do Docker falhará.

Exemplos:


  # o comando abaixo falhará
  RUN /bin/bash -c 'source $HOME/.bashrc;
      echo $HOME'

Após a aplicação da correção rápida:


  RUN /bin/bash -c 'source $HOME/.bashrc; \
      echo $HOME'
