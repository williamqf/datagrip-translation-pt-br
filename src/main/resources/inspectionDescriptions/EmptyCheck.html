Relata diferentes maneiras de verificar o vazio das coleções e Options e sugere substituí-las por .isEmpty, .nonEmpty ou .isDefined.

Exemplo:


  !Seq(1).isEmpty
  Seq(1).size == 0
  Seq(1).size != 0
  !(Seq(1).length > 0)
  Seq(1).exists(_ => true)
  import scala.Function.const
  Seq(1).exists(const(true))
  !Option(0).isEmpty
  Option(0) == None
  Option(0) != None

Após o reparo rápido ter sido aplicado:


  Seq(1).nonEmpty
  Seq(1).isEmpty
  Seq(1).nonEmpty
  Seq(1).isEmpty
  Seq(1).nonEmpty
  import scala.Function.const
  Seq(1).nonEmpty
  Option(0).isDefined
  Option(0).isEmpty
  Option(0).isDefined
