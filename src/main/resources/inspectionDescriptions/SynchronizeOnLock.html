Relata blocos synchronized que bloqueiam uma instância de java.util.concurrent.locks.Lock. Essa sincronização é quase certamente não intencional e as versões apropriadas de .lock() e .unlock() devem ser usadas em seu lugar.

Exemplo:


  final ReentrantLock lock = new ReentrantLock();

  public void foo() {
    synchronized (lock) {}
  }
