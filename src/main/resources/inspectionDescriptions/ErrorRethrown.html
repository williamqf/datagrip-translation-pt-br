Relata instruções try que capturam java.lang.Error ou qualquer uma de suas subclasses e não lançam o erro novamente.

Instruções que capturam java.lang.ThreadDeath não serão relatadas.

Exemplo:


  try {
    executeTests(request);
  }
  catch (OutOfMemoryError ex) { // Aviso: o erro 'ex' não é lançado novamente
    return false;
  }
