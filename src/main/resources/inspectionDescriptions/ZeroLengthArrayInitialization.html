Relata alocações de matrizes com comprimentos conhecidos de 0.

Como os comprimentos das matrizes em Java não são modificáveis, é quase sempre possível compartilhar matrizes de comprimento 0, em vez de alocar repetidamente novas. Esse compartilhamento pode fornecer otimizações úteis no tempo de execução do programa ou pegada.

Observe que a inspeção não relata matrizes de comprimento 0 alocadas como campos finais estáticos, já que essas matrizes são supostamente usadas para implementar compartilhamento de matriz.
