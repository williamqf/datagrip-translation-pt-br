Relata o acesso a um campo de superclasse a partir de uma classe anônima, interna ou local se uma variável local, parâmetro ou campo com o mesmo nome existir no código circundante. Nesses casos, pode parecer que um elemento do código circundante está sendo acessado, quando, na realidade, o campo da superclasse é o que está sendo acessado.

Para esclarecer a intenção do código, recomenda-se adicionar um qualificador super explícito ao acesso ao campo.

Exemplo:


    class First {
      protected String ambiguous;
    }
    class Second {
      void foo(String ambiguous) {
        new First() {
          {
            System.out.println(ambiguous); // Acessa o campo, não o parâmetro
          }
        };
      }
    }

Após a aplicação da correção rápida:


    class First {
      protected String ambiguous;
    }
    class Second {
      void foo(String ambiguous) {
        new First() {
          {
            System.out.println(super.ambiguous);
          }
        };
      }
    }
