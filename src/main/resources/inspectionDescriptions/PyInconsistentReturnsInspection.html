Destaca instruções return inconsistentes em uma função. De acordo com a PEP 8, todas as instruções return em uma função devem retornar uma expressão ou nenhuma delas deve retornar.

Recomendação da PEP 8: Todas as instruções return em uma função devem retornar uma expressão ou nenhuma delas deve retornar. Se qualquer instrução return retornar uma expressão, quaisquer instruções return que não retornem um valor devem declarar isso explicitamente como return None, e uma instrução return explícita deve estar presente no final da função (se acessível): 


# Correto:

def foo(x):
    if x >= 0:
        return math.sqrt(x)
    else:
        return None

def bar(x):
    if x < 0:
        return None
    return math.sqrt(x)


# Incorreto:

def foo(x):
    if x >= 0:
        return math.sqrt(x)

def bar(x):
    if x < 0:
        return
    return math.sqrt(x)
