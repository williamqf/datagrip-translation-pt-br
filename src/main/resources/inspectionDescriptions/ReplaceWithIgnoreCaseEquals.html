Relata comparações que não diferenciam maiúsculas de minúsculas que podem ser substituídas por equals(..., ignoreCase = true).

Ao usar equals(), as strings podem ser comparadas sem a necessidade de alocar strings extras com toLowerCase() ou toUpperCase().

A correção rápida substituirá comparações que não diferenciam maiúsculas de minúsculas usando toLowerCase() ou toUpperCase() por equals(..., ignoreCase = true).

Nota: a semântica pode ser alterada para algumas localidades.

Exemplo:


  fun main() {
      val a = "KoTliN"
      val b = "KOTLIN"
      println(a.toLowerCase() == b.toLowerCase())
  }

Após a aplicação da correção rápida:


  fun main() {
      val a = "KoTliN"
      val b = "KOTLIN"
      println(a.equals(b, ignoreCase = true))
  }
