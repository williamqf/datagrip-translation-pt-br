Relata comparações com chamadas String.indexOf() que podem ser substituídas por uma chamada ao método String.contains().

Exemplo:


  boolean b = "abcd".indexOf('e') >= 0;

Após a aplicação da correção rápida:


  boolean b = "abcd".contains('e');

Esta inspeção relata apenas se o nível de linguagem do projeto ou módulo for 5 ou superior.
