Relata perfis inativos em contextos de aplicação Spring XML.

Exemplo:


  <!-- O perfil ativo 'DEV' foi selecionado para este contexto de aplicação -->
  <beans profile="DEV">
    <bean class="MyFactory" factory-method="createBean"/>
  </beans>
  <!-- O conteúdo desta tag é destacado como 'unused' -->
  <beans profile="TESTS">
    <bean class="MyTestFactory" factory-method="createBean"/>
  </beans>
  <!-- Fim da seção 'unused' -->
  </beans>
