Relata construtores que não começam com uma chamada ao construtor "super" ou a outro construtor da mesma classe.

Pode-se considerar que tais construtores começam implicitamente com uma chamada a super(). Alguns padrões de codificação preferem chamadas explícitas a super().

Exemplo:


  class Foo {
    Foo() {}
  }

Após a aplicação da correção rápida:


  class Foo {
    Foo() {
      super();
    }
  }

Use as configurações de inspeção para ignorar classes que estendem diretamente de Object. Exemplo: 


  class Foo {
    Foo() {} // Não relatado
  }

  class Bar extends Foo {
    Bar() {} // Chamada implícita a 'super()'
  }
