Relata conversões potencialmente incorretas de uintptr para unsafe.Pointer.

Uma conversão de uintptr para unsafe.Pointer é inválida se uma palavra do tipo uintptr que contém um valor de ponteiro estiver armazenada na memória, pois essa palavra é invisível para a cópia de pilha e para o coletor de lixo (garbage collector).

Exemplo de uso inválido:

nums := []int8{42, 24}
ptr := unsafe.Pointer(&nums[0])
addr := uintptr(ptr) // o endereço é armazenado em uma variável local
ptr = unsafe.Pointer(addr + uintptr(1))

Exemplo de uso válido:

nums := []int8{42, 24}
ptr := unsafe.Pointer(&nums[0])
ptr = unsafe.Pointer(uintptr(ptr) + uintptr(1))
