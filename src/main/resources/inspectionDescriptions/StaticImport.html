Relata instruções import static.

Tais instruções import não são suportadas em JVMs do Java 1.4 ou anterior.

Configuração da inspeção:

  • Utilize a tabela abaixo para especificar as classes que a inspeção deve ignorar quando usadas em instruções import static.
  • Use a caixa de seleção Ignorar importação estática de campo único para ignorar instruções import static de campo único.
  • Use a caixa de seleção Ignorar importação estática de método único para ignorar instruções import static de método único.
