Relata propriedades cujos nomes são muito curtos, muito longos ou não correspondem ao padrão de expressão regular especificado. Alguns estilos de código possuem convenções de nomenclatura especiais para propriedades.

Use as opções da inspeção para especificar o comprimento mínimo/máximo e a expressão regular esperada para os nomes de propriedades. Para ignorar o comprimento do nome, especifique 0.

Para saber mais sobre expressões regulares, consulte o Quick Start guide (regular-expressions.info).
