Relata padrões de formato de mensagem incorretos ou índices incorretos para espaços reservados (placeholders).

Os seguintes erros são relatados:

  • Índices não resolvidos ou negativos
  • Chaves não fechadas
  • Aspas sem par. Nesse caso, uma parte do padrão pode não ser utilizada
  • Número de aspas possivelmente incorreto
  • Limite inferior incorreto para padrões choice aninhados
  • Índice de espaço reservado incorreto. Nesse caso, o espaço reservado pode não ser substituído ou o argumento pode não ser utilizado

Exemplo:


  MessageFormat.format("{wrong}", 1); // índice incorreto
  MessageFormat.format("{0", 1); // chaves não correspondentes
  MessageFormat.format("'{0}", 1); // aspas sem par
  MessageFormat.format("It''''s {0}", 1); // imprimirá "It''s" em vez de "It's"
  MessageFormat.format("{0}", 1, 2); // o argumento com índice '1' não é utilizado no padrão

Use a tabela Métodos MessageFormat personalizados para especificar chamadas de método cujos argumentos devem ser verificados como padrões de MessageFormat. A tabela contém pares de nomes de classe totalmente qualificados e expressões regulares de nomes de método para corresponder aos nomes das classes e chamadas de método. O nome da classe também corresponde a subclasses.
