Relata chamadas como "abc".substring(0, 1) que podem ser substituídas por "abc"[0].

Obter elementos por índice torna o código mais simples.

A correção rápida substitui a chamada substring pelo operador de indexação.

Exemplo:


  fun foo() {
      "abc".substring(0, 1)
  }

Após a aplicação da correção rápida:


  fun foo() {
      "abc"[0]
  }
