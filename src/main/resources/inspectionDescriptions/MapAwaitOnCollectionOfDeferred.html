Relata chamadas map { it.await() } em coleções de Deferred.

Isso pode ser substituído por uma chamada awaitAll() mais concisa e potencialmente mais eficiente.

Para mais detalhes, consulte a documentação de kotlinx.coroutines.awaitAll.

Exemplo:


  suspend fun waitForResult(results: List<Deferred<String>>): List<String> {
    return results.map { it.await() }
  }

Após a aplicação da correção rápida:


  suspend fun waitForResult(results: List<Deferred<String>>): List<String> {
    return results.awaitAll()
  }
