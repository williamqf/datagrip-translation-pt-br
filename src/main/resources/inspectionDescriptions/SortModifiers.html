Relata modificadores que não seguem a ordem sugerida no guia de estilo.

A correção rápida Reordenar modificadores pode ser usada para corrigir automaticamente o código.

Exemplo:


  private inline fun correctOrder(f: () -> Unit) {} // <== Ok

  infix private fun Int.wrongOrder(expr: Int) {} // <== Ordem incorreta, a correção rápida corrigirá os modificadores para "private infix"
