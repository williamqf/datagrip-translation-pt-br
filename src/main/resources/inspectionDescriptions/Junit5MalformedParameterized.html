Inspeciona testes parametrizados com fontes de parâmetros com formato incorreto:

  • MethodSource com alvo desconhecido ou o método não é static, no-arg.
  • Tipos ValueSource e EnumSource não podem ser convertidos em parâmetros de método.
  • Fonte não definida.

Exemplo:


  class Test {
    @MethodSource("parameters")
    @ParameterizedTest
    void foo(String param) {}
  }

Após a aplicação das correções rápidas:


  class Test {
    private static Stream<Arguments> parameters() {
        return Stream.empty();
    }

    @MethodSource("parameters")
    @ParameterizedTest
    void foo(String param) {}
  }
