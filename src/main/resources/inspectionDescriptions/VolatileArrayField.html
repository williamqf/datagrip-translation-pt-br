Relata campos de array declarados como volatile. Tais declarações podem ser confusas, pois o acesso ao array em si segue as regras de campos volatile, mas o acesso ao conteúdo do array não.

Exemplo:


  class Data {
    private volatile int[] idx = new int[0];
  }

Se esse tipo de acesso volátil ao conteúdo do array for necessário, considere utilizar as classes do pacote java.util.concurrent.atomic: 


  class Data {
    private final AtomicIntegerArray idx = new AtomicIntegerArray(new int[0]);
  }
