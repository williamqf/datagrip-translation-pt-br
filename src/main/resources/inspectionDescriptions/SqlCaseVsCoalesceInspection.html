Relata casos em que as chamadas CASE e COALESCE podem ser intercambiadas. Esta inspeção possui as seguintes ações de intenção: Substituir pela chamada 'COALESCE' e a oposta, Substituir pela expressão CASE.

Exemplo (MySQL):

SELECT
  -- este CASE pode ser substituído por COALESCE
	CASE
		WHEN C1 IS NOT NULL THEN C1
		ELSE 0
		END
FROM dual;

Neste exemplo, a instrução CASE pode ser substituída por SELECT COALESCE(C1, 0), que produz o mesmo resultado.

Se você preferir usar expressões CASE, selecione a opção Preferir expressão CASE em vez da função COALESCE na página de inspeção.
