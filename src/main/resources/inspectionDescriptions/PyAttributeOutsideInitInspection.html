Relata definições de atributos de instância fora do método __init__.

Exemplo:


    class Book:
    def __init__(self):
        self.author = 'Mark Twain'

    def release(self):
        self.year = '1889'

Após a aplicação do ajuste rápido, o exemplo de código torna-se: 


    class Book:
    def __init__(self):
        self.year = '1889'
        self.author = 'Mark Twain'

    def release(self):
        pass
