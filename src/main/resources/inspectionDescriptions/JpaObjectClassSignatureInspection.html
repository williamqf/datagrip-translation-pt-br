Relata classes de objetos de persistência definidas incorretamente nos seguintes casos:
  • A classe não é uma classe de nível superior
  • A classe é final
  • A classe foi anotada várias vezes
  • A classe está definida como uma entidade de persistência e um ouvinte de entidade ao mesmo tempo

Exemplo:



  @Entity
  @EntityListeners(value = JavaEntity.class) // Erro: as anotações @Entity e @EntityListener apontam para a mesma classe
  public final class JavaEntity { // Erro: entidades de persistência não devem ser final
    ...
  }
