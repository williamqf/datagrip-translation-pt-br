Relata operações redundantes em objetos java.time:
  • Criação de objetos de data/hora do pacote java.time do JDK quando uma chamada de método mais simples poderia ser usada ou a criação poderia ser evitada.
  • Chamadas de método java.time que usam java.time.temporal.ChronoField e java.time.temporal.ChronoUnit como argumentos quando podem ser substituídas por chamadas de métodos mais específicos.
  • Comparações java.time com chamadas compareTo() que podem ser substituídas por chamadas isAfter(), isBefore() ou isEqual().

Exemplos:

- Antes: 

LocalDateTime now = LocalDateTime.now();
return LocalDateTime.from(now);

Após a aplicação da correção rápida:


LocalDateTime now = LocalDateTime.now();
return now;
- Antes: 

LocalTime localTime = LocalTime.now();
int minute = localTime.get(ChronoField.MINUTE_OF_HOUR);

Após a aplicação da correção rápida:


LocalTime localTime = LocalTime.now();
int minute = localTime.getMinute();
- Antes: 

  LocalDate date1 = LocalDate.now();
  LocalDate date2 = LocalDate.now();
  boolean t = date1.compareTo(date2) > 0;

Após a aplicação da correção rápida:


  LocalDate date1 = LocalDate.now();
  LocalDate date2 = LocalDate.now();
  boolean t = date1.isAfter(date2);
