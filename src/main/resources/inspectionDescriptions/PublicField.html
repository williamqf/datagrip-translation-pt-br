Relata campos public. Não relata constantes (campos marcados como static e final).

Exemplo:


  class Main {
    public String name;
  }

Após aplicar a correção rápida:


  class Main {
    private String name;

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }
  }

Configuração da inspeção:

  • Use a lista Ignorar se anotado por para especificar as anotações a serem ignoradas. Esta inspeção ignorará os campos que contenham qualquer uma dessas anotações.
  • Use a opção Ignorar campos 'public final' de enums para ignorar campos public final do tipo enum.
