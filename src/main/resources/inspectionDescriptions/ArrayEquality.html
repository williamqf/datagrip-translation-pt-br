Relata os operadores == e != usados para testar a igualdade de arrays. Na maioria dos casos, a intenção é testar a igualdade do conteúdo do array, o que pode ser feito por meio do método java.util.Arrays.equals().

Uma correção rápida é sugerida para substituir == por java.util.Arrays.equals().

Exemplo:


  void foo(Object[] x, Object[] y) {
    boolean comparison = x == y;
  }

Após a aplicação da correção rápida:


  void foo(Object[] x, Object[] y) {
    boolean comparison = Arrays.equals(x, y);
  }
