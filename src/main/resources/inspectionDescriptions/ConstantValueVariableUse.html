Relata todos os usos de variáveis conhecidas como constantes.

Isso ocorre se o uso (leitura) de uma variável estiver cercado por instruções if, while ou for que contenham uma condição == comparando essa variável a uma constante. Nesse caso, o uso da variável conhecida como constante pode ser substituído pela constante real.

Exemplo:


    private static void foo(double number) {
        if (number == 1.0) {
            f(number);
        }
    }
    private static void f(double number) {}

Após a aplicação da correção rápida:


    private static void foo(double number) {
        if (number == 1.0) {
            f(1.0);
        }
    }
    private static void f(double number) {}
