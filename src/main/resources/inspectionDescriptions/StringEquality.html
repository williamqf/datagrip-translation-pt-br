Relata o código que usa == ou != para comparar strings.

Esses operadores determinam a igualdade de referência em vez de comparar o conteúdo. Na maioria dos casos, as strings devem ser comparadas usando equals(), que realiza uma comparação caractere por caractere quando as strings são objetos diferentes.

Exemplo:


  void foo(String s, String t) {
    final boolean b = t == s;
  }

Se for conhecido que t não é nulo, é seguro aplicar a correção rápida "insegura" e obter um resultado semelhante ao seguinte:


  void foo(String s, String t) {
    final boolean b = t.equals(s);
  }
