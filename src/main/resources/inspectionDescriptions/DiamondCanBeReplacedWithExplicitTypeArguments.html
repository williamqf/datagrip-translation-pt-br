Relata a instanciação de classes genéricas nas quais o símbolo <> (diamond) é usado em vez de parâmetros de tipo.

A correção rápida substitui <> (diamond) por parâmetros de tipo explícitos.

Exemplo:

List<String> list = new ArrayList<>()

Após a aplicação da correção rápida:

List<String> list = new ArrayList<String>()

O operador diamond surgiu no Java 7. Esta inspeção ajuda a fazer o downgrade para compatibilidade reversa com versões anteriores do Java.
