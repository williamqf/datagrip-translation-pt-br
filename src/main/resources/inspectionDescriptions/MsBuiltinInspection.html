Relata o truncamento de argumentos de string na função ISNULL.

A sintaxe de ISNULL é ISNULL(check_expression, replacement_value).

De acordo com o ISNULL no docs.microsoft.com, o replacement_value será truncado se for mais longo que o check_expression.

Exemplo (Microsoft SQL Server):

DECLARE @name1 VARCHAR(2) = NULL;
DECLARE @name2 VARCHAR(10) = 'Example';
DECLARE @name3 VARCHAR(2) = 'Hi';

  -- `@name2` é VARCHAR(10) e será truncado
SELECT ISNULL(@name1, @name2);

  -- `@name3` é VARCHAR(2) (como `@name1`) e não será truncado
SELECT ISNULL(@name1, @name3);
