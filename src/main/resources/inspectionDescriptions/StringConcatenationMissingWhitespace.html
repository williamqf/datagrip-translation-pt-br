Relata concatenações de strings com espaço ausente, ou seja, quando o lado esquerdo termina com uma letra ou dígito Unicode e o lado direito começa com uma letra ou dígito Unicode.

Exemplo:


  String sql = "SELECT column" +
               "FROM table";

Use a opção Ignorar concatenações com strings variáveis para relatar apenas quando ambos os lados da concatenação forem literais.
