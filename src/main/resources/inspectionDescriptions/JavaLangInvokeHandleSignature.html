Relata chamadas de métodos de fábrica MethodHandle e VarHandle que não correspondem a nenhum método ou campo.

Também relata argumentos de MethodHandle.invoke() e métodos semelhantes que não correspondem à assinatura de MethodHandle, bem como argumentos de VarHandle.set() que não correspondem ao tipo de VarHandle.

Exemplo:

  MethodHandle mh = MethodHandles.lookup().findVirtual(
      MyClass.class, "foo", MethodType.methodType(void.class, int.class));
  // Este argumento deve ser um valor int
  mh.invoke(myObj, "abc");


  // Este argumento deve ser String.class
  VarHandle vh = MethodHandles.lookup().findVarHandle(
      MyClass.class, "text", int.class);


  VarHandle vh = MethodHandles.lookup().findVarHandle(
      MyClass.class, "text", String.class);
  // Este argumento deve ser um valor String
  vh.set(myObj, 42);
