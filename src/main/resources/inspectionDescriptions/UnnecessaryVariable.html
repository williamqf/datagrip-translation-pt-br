Relata variáveis locais usadas apenas na instrução return subsequente ou que sejam cópias exatas de outras variáveis.

Tais variáveis podem ser incorporadas (inline) com segurança para tornar o código mais claro.

Exemplo:


  fun sum(a: Int, b: Int): Int {
      val c = a + b
      return c
  }

Após aplicar a correção rápida:


  fun sum(a: Int, b: Int): Int {
      return a + b
  }

Configuração da inspeção:

Use a opção Relatar variáveis retornadas imediatamente para relatar variáveis que são retornadas logo em seguida. Essas variáveis podem, em alguns casos, aumentar a legibilidade do código quando recebem nomes descritivos, e é por isso que esta opção é desativada por padrão.
