Reporta elementos XML obsoletos na versão atual do contexto de aplicação baseado em XML do Spring Integration.

Para mais detalhes, consulte o guia de migração.

Exemplo:


<beans ... >
   <int:channel id="failChannel"
                dispatcher="failover"/> <!-- Este atributo é obsoleto, use o subelemento dispatcher em seu lugar -->
   <int:poller default="true">
     <int:cron-trigger  expression="5"/>  <!-- Este elemento foi removido no Spring Integration 2.1, use o atributo 'cron' em seu lugar -->
     <int:interval-trigger interval="3"/> <!-- Este elemento foi removido no Spring Integration 2.1, use o atributo interval-trigger em seu lugar -->
  </int:poller>
  <ftp:inbound-channel-adapter session-factory="ftpSessionFactory"
                               cache-sessions="false" <!-- Este atributo é obsoleto, use CachingSessionFactory em seu lugar -->
                               channel="channel"/>
</beans>
