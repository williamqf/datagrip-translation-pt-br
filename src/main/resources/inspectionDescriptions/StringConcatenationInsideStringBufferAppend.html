Relata concatenações de String usadas como argumentos para StringBuffer.append(), StringBuilder.append() ou Appendable.append().

Tais chamadas podem ser vantajosamente transformadas em chamadas de append encadeadas no StringBuffer/Builder/Appendable existente, economizando o custo de alocação adicional de StringBuffer/Builder. Esta inspeção ignora concatenações de String avaliadas em tempo de compilação, casos em que a conversão apenas prejudicaria o desempenho.

Exemplo:


  void bar(StringBuilder builder, String name) {
    builder.append("Hello," + name);    //aviso
    builder.append("Hello," + "world"); //sem aviso
  }
