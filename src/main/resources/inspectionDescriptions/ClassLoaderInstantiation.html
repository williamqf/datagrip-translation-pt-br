Relata a instanciação da classe java.lang.ClassLoader.

Embora geralmente seja inofensiva, qualquer instanciação de ClassLoader deve ser examinada cuidadosamente em qualquer auditoria de segurança.

Exemplo:

    Class<?> loadExtraClass(String name) throws Exception {
      try(URLClassLoader loader =
          new URLClassLoader(new URL[]{new URL("extraClasses/")})) {
        return loader.loadClass(name);
      }
    }
