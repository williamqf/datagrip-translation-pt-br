Relata importações totalmente qualificadas que podem ser relative ao pacote base (Settings | Languages & Frameworks | Scala | Base Packages).

Aplicável somente quando a opção Exceto para o pacote base é habilitada em Settings | Editor | Code Style | Scala | Imports.

Exemplo:


  package org.example.application
  package ui

  import org.example.application.model.Data

Após a correção rápida ser aplicada:


  package org.example.application
  package ui

  import model.Data
