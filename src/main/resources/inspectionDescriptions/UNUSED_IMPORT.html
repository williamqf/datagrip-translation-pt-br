Relata instruções import redundantes.

Se as classes e pacotes importados no arquivo de origem não forem utilizados, as instruções import regulares não são necessárias. O mesmo se aplica a campos e métodos static importados que não são usados no arquivo de origem.

Exemplo:


  import java.util.ArrayList;
  public class Example {
    public static void main(String[] args) {
      System.out.println("Hello World!");
    }
  }

Após a aplicação da correção rápida: 


  public class Example {
    public static void main(String[] args) {
      System.out.println("Hello World!");
    }
  }
