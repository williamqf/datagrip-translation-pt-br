Relata usos de APIs (classes, campos e métodos) obsoletas marcadas para remoção com @Deprecated(forRemoval=true).

O código que utiliza APIs marcadas para remoção pode resultar em erros de tempo de execução em versões futuras da API. É por isso que a severidade recomendada para esta inspeção é Erro.

Se desejar usar o mesmo destaque de código de uma depreciação comum, você pode alterar a severidade para Aviso.
