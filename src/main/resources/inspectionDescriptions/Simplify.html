Reporta instruções que podem ser simplificadas, como condições constantes, ramos if idênticos, expressões booleanas redundantes, etc. Use as caixas de seleção abaixo para controlar as opções de análise:
  • Simplificar "if (true)", "while(false)": sugere substituir as instruções if/while por literais booleanos.
  • Simplificar "condition == true": sugere substituir instruções de igualdade por literais booleanos.
  • Simplificar "condition1 ? true : condition2;": sugere simplificar operadores ternários em que uma das expressões é um literal booleano.
  • Simplificar "if (condition1) return true; return condition2;": sugere simplificar o if onde um literal booleano é retornado se a condição for verdadeira, caso contrário, outra condição é retornada.
