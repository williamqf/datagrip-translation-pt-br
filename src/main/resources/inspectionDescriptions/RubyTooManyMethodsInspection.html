Relata classes/módulos com muitos métodos. Isso geralmente significa que eles estão tentando "fazer demais" e podem precisar ser refatorados em vários componentes menores.

Use o campo Contagem de métodos para especificar o número máximo de métodos permitidos para uma classe/módulo.

Use a caixa de seleção Ignorar classes de teste para ignorar classes de teste.
