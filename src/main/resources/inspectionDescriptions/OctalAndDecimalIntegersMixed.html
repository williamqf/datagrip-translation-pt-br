Relata literais inteiros octais e decimais misturados em um único inicializador de array. Isso pode acontecer quando você copia uma lista de números para um inicializador de array. Alguns números no array podem estar preenchidos com zeros e o compilador os interpretará como octais.

Exemplo:

int[] elapsed = {1, 13, 052};

Após aplicar a correção rápida para remover zeros à esquerda:

int[] elapsed = {1, 13, 52};

Se for um número octal (por exemplo, após o inline de uma variável), você pode usar outra correção rápida para converter o octal em decimal:

int[] elapsed = {1, 13, 42};
