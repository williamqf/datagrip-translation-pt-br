Relata classes abstract que podem ser convertidas em interfaces.

É preferível usar interfaces em vez de classes, pois o Java não suporta herança múltipla de classes, enquanto uma classe pode implementar várias interfaces.

Uma classe pode ser convertida em uma interface se não possuir uma superclasse (além de Object) e contiver apenas campos public static final, métodos public abstract e classes internas public.

Exemplo:


abstract class Example {
    public static final int MY_CONST = 42;
    public abstract void foo();
}

class Inheritor extends Example {
    @Override
    public void foo() {
        System.out.println(MY_CONST);
    }
}

Após a aplicação da correção rápida:


interface Example {
    int MY_CONST = 42;
    void foo();
}

class Inheritor implements Example {
    @Override
    public void foo() {
        System.out.println(MY_CONST);
    }
}

Configuração da inspeção:

A opção Relatar classes com métodos não abstratos ao usar o Java 8 pode ser usada para relatar apenas classes cujos métodos static e não abstratos possam ser convertidos em métodos default (aplicável apenas quando o nível de linguagem for 8 ou superior).
