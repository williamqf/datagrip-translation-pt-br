Reporta chamadas de cadeias da Stream API, Iterable.forEach() e Map.forEach() que podem ser convertidas automaticamente em loops clássicos. Esta inspeção ajuda a fazer o downgrade para compatibilidade com versões anteriores do Java.

Exemplo:


  String joinNonEmpty(List<String> list) {
    return list.stream() // A stream pode ser convertida em um loop
      .filter(s -> !s.isEmpty())
      .map(String::trim)
      .collect(Collectors.joining(", "));
  }

Após a aplicação da correção rápida:


  String joinNonEmpty(List<String> list) {
    StringJoiner joiner = new StringJoiner(", ");
    for (String s : list) {
      if (!s.isEmpty()) {
        String trim = s.trim();
        joiner.add(trim);
      }
    }
    return joiner.toString();
  }

Observe que, às vezes, esta inspeção pode levar a pequenas alterações semânticas. Deve-se ter atenção especial com o curto-circuito, pois o número real de elementos lidos não é especificado quando ocorre um curto-circuito na stream.

Configurar inspeção:

Use a opção Iterar sobre fontes de stream desconhecidas via Stream.iterator() para sugerir a conversão de streams cujas fontes não são reconhecidas. Neste caso, um iterador será criado a partir da stream. Por exemplo, se a caixa de seleção estiver marcada, a conversão será sugerida aqui:


  List<ProcessHandle> handles = ProcessHandle.allProcesses().collect(Collectors.toList());

Neste caso, o resultado é o seguinte:


  List<ProcessHandle> handles = new ArrayList<>();
  for (Iterator<ProcessHandle> it = ProcessHandle.allProcesses().iterator(); it.hasNext(); ) {
    ProcessHandle allProcess = it.next();
    handles.add(allProcess);
  }
