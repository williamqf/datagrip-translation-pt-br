Relata campos não static e não final cujos nomes estão inteiramente em letras maiúsculas.

Tais campos podem causar confusão por violarem as convenções de nomenclatura comuns e, frequentemente, são usados de forma incorreta.

Exemplo: 


  public static int THE_ANSWER = 42; // aviso aqui: modificador final ausente

A correção rápida para renomear esses campos está disponível apenas no editor.
