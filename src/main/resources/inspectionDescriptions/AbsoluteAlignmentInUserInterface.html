Relata o uso de constantes de alinhamento absoluto no AWT e Swing. Aplicativos internacionalizados usam alinhamento relativo, pois ele segue as configurações de orientação de componentes da localidade.

Exemplo:


  JPanel panel = new JPanel(new BorderLayout(2, 2));
  JLabel label = new JLabel("Hello World");
  panel.add(label, BorderLayout.NORTH);

Após aplicar a correção rápida:


  JPanel panel = new JPanel(new BorderLayout(2, 2));
  JLabel label = new JLabel("Hello World");
  panel.add(label, BorderLayout.PAGE_START);
