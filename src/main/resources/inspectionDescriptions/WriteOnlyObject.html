Relata objetos que foram modificados, mas nunca consultados.

Esta inspeção depende de contratos de mutação de métodos que podem ser inferidos ou pré-anotados para certos métodos de biblioteca. Esta inspeção não relata coleções, mapas e builders de string, pois outras inspeções mais precisas relatam esses tipos.

Exemplo:


  AtomicReference<String> ref = new AtomicReference<>();
  ref.set("hello"); // ref não é mais usado

Use a opção Ignorar construtores impuros para controlar se os objetos criados por construtores ou métodos de pureza desconhecida devem ser processados. Desmarcar essa opção pode causar alguns falsos positivos se uma referência de objeto for intencionalmente vazada durante a construção.
