Relata quaisquer instruções break desnecessárias.

Uma instrução break é desnecessária se nenhuma outra instrução for executada caso ela seja removida.

Exemplo:


  switch (e) {
    case A -> {
        System.out.println("A");
        break; // Relata 'break' como desnecessário
    }
    default -> {
        System.out.println("Default");
        break; // Relata 'break' como desnecessário
    }
  }
