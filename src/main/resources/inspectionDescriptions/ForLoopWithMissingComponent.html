Relata loops for nos quais faltam as cláusulas de inicialização, condição ou atualização. Alguns estilos de codificação proíbem tais loops.

Exemplo:


  for (int i = 0;;i++) {
    // corpo
  }

Use a opção Ignorar iteração de coleção para ignorar loops que utilizam iteradores. Esta é a maneira padrão de iterar em coleções onde o loop for não possui uma cláusula de atualização.
