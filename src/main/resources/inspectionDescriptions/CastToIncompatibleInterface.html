Relata expressões de conversão de tipo (cast) nas quais a expressão convertida possui um tipo de classe ou interface que não estende nem implementa o tipo da classe ou interface de destino, e não possui subclasses que o façam.

Tais construções provavelmente estão incorretas e lançarão uma java.lang.ClassCastException em tempo de execução.

Exemplo:


  interface A {}
  interface Z {}
  static class C {}

  void x(C c) {
    if (c instanceof Z) {
      A a = ((A)c); // cast para interface incompatível 'A'
    }
  }
