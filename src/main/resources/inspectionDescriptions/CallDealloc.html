Relate uma mensagem dealloc enviada diretamente. Mensagens desse tipo nunca devem ser enviadas diretamente. Em vez disso, o método dealloc do objeto deve ser chamado indiretamente através do método de protocolo release NSObject (se a mensagem release fizer com que a contagem de retenção do receptor se torne 0). Para obter mais informações sobre o uso desses métodos, consulte o Memory Management Programming Guide (Guia de Programação de Gerenciamento de Memória).
