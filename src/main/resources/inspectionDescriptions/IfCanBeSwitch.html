Relata instruções if que podem ser substituídas por instruções switch.

A substituição geralmente resulta em um código mais curto e claro.

Exemplo:


  void test(String str) {
    if (str.equals("1")) {
      System.out.println(1);
    } else if (str.equals("2")) {
      System.out.println(2);
    } else if (str.equals("3")) {
      System.out.println(3);
    } else {
      System.out.println(4);
    }
  }

Após a aplicação da correção rápida:


  void test(String str) {
    switch (str) {
      case "1" -> System.out.println(1);
      case "2" -> System.out.println(2);
      case "3" -> System.out.println(3);
      default -> System.out.println(4);
    }
  }

Esta inspeção reporta apenas se o nível de linguagem do projeto ou módulo for 7 ou superior.

Use o campo Número mínimo de ramificações de condição 'if' para especificar o número mínimo de ramificações de condição if que uma instrução if deve ter para ser relatada. Observe que a ramificação else final (sem if) não é contada.

Use a opção Sugerir switch para números para ativar sugestões de instruções switch para números e caracteres primitivos e empacotados (boxed).

Use a opção Sugerir switch para enums para ativar sugestões de instruções switch para constantes enum.

Use a opção Sugerir apenas para expressões null-safe para sugerir apenas instruções switch que não possam introduzir uma NullPointerException.
