Relata instruções de atribuição no formato x = atomic.AddUint64(&x, 1).

Tais operações não são atômicas, o que é um mau uso comum da API sync/atomic. Para torná-la atômica, remova a atribuição para usar uma chamada direta: atomic.AddUint64(&x, 1). Nesse caso, o valor de x será atualizado de forma atômica por meio do endereço.

Exemplo:

import (
 "sync/atomic"
)

type Counter uint64

func AtomicTests() {
 x := uint64(1)
 x = atomic.AddUint64(&x, 1)         // Erro "Atribuição direta a um valor atômico"
 _, x = 10, atomic.AddUint64(&x, 1)  // Erro "Atribuição direta a um valor atômico"
 x, _ = atomic.AddUint64(&x, 1), 10  // Erro "Atribuição direta a um valor atômico"
}
