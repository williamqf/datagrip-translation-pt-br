Relata funções exportadas que possuem tipos de retorno não exportados.

Ao visualizar a documentação no go doc, tipos não exportados podem ser difíceis de usar.

Exemplo:

type hidden struct{}

func Exported() hidden { // Função exportada com o tipo de retorno não exportado 'hidden'
 return hidden{}
}

Você pode aplicar a correção rápida Exportar para exportar o tipo. Após aplicar a correção rápida, o nome do tipo começará com letra maiúscula:

type Hidden struct{}

func Exported() Hidden { // Corrigido
 return Hidden{}
}
