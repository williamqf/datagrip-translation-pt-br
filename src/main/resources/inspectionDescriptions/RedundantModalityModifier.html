Relata modificadores de modalidade que coincidem com a modalidade padrão do elemento (final para a maioria dos elementos, open para membros com override).

Exemplo:


  final class Foo

  open class Bar : Comparable<Bar> {
      open override fun compareTo(other: Bar): Int = 0
  }

Após a aplicação da correção rápida:


  class Foo

  open class Bar : Comparable<Bar> {
      override fun compareTo(other: Bar): Int = 0
  }
