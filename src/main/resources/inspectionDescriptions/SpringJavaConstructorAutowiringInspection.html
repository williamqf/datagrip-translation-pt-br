Relata o uso incorreto de @Autowired nos construtores de Spring Bean: vários Construtores @Autowired, nenhum Construtor @Autowired correspondente, etc.

Exemplo:


@Component
public class MyComponent {
	@Autowired
	public MyComponent(BarBean bean) {...} // relata "Apenas um construtor @Autowired é permitido"

	@Autowired
	public MyComponent(FooBean bean) {...} // relata "Apenas um construtor @Autowired é permitido"
}

@Component
public class MyFactory {  // relata "Nenhum construtor @Autowired correspondente"
	public MyFactory(String str) {...}
	public MyFactory(int count) {...}
}
