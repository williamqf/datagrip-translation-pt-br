Relata casos em que nenhum dialeto SQL foi atribuído aos arquivos SQL.

Por exemplo, ao abrir um novo arquivo SQL sem atribuir um dialeto a ele, você verá uma notificação sugerindo o dialeto mais adequado. Clique no link Usar <dialect> para usar o dialeto sugerido. Como alternativa, clique no link Alterar dialeto para para selecionar outro dialeto.
