Relata objetos não Serializable usados como argumentos para java.io.ObjectOutputStream.write(). Tais chamadas resultarão em uma exceção em tempo de execução.

Esta inspeção assume que objetos dos tipos java.util.Collection e java.util.Map são Serializable, a menos que seus tipos declarados não sejam Serializable.

Exemplo:


  public class IWantToSerializeThis {
    public static void main(String[] args) throws IOException {
      try(var stream = new ObjectOutputStream(Files.newOutputStream(Paths.get("output")))) {
        // Aviso -- falhará com NotSerializableException
        stream.writeObject(new IWantToSerializeThis());
      }
    }
  }
