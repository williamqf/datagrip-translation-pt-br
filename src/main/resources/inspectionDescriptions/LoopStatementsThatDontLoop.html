Relata quaisquer instâncias de instruções for, while e do cujo corpo seja executado no máximo uma vez. Geralmente, isso indica um erro.

Use a opção Ignorar loops for aprimorados para ignorar loops foreach. Eles às vezes são usados para realizar uma ação apenas no primeiro item de um iterável de forma compacta.

Exemplo:


  for (String s : stringIterable) {
    doSomethingOnFirstString(s);
    break;
  }
