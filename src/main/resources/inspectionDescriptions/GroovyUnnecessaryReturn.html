Relata instruções return ao final de construtores e métodos que retornam void. Tais instruções são desnecessárias e podem ser removidas com segurança.

Exemplo: 


  void foo (String s){
      print(s)
      return
  }

Após a aplicação da correção rápida:


  void foo (String s){
      print(s)
  }

Para mais informações, consulte a mesma inspeção em Java.
