Relata chamadas explícitas às funções get ou set que podem ser substituídas pelo operador de indexação [].

O Kotlin permite implementações personalizadas para um conjunto predefinido de operadores para tipos. Para sobrecarregar um operador, você pode marcar a função correspondente com o modificador operator: 


  operator fun get(index: Int) {}
  operator fun set(index: Int, value: Int) {}
As funções acima correspondem ao operador de indexação.

Exemplo:

  class Test {
      operator fun get(i: Int): Int = 0
  }

  fun test() {
      Test().get(0) // 'get()' substituível
  }

Após a aplicação da correção rápida:

  class Test {
      operator fun get(i: Int): Int = 0
  }

  fun test() {
      Test()[0]
  }
