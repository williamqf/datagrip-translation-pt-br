Relata o uso de variáveis static não final durante a inicialização da classe.

Nesses casos, a semântica do código pode se tornar dependente da ordem de criação da classe. Além disso, isso pode levar ao uso de uma variável antes de sua inicialização, o que geralmente resulta em erros difíceis e confusos.

Exemplo:


  class Foo {
    public static int bar = 0;

    static {
      System.out.println(bar);
    }
  }
