Relata chamadas encadeadas que podem ser simplificadas usando o operador de navegação segura (&.). Um grupo de chamadas encadeadas é um padrão comum usado para verificar a existência de nil em cada componente da chamada.

Exemplo:

# Prática ruim
if user && user.profile && user.profile.thumbnails
end
Uma correção rápida pode ser usada para simplificar a expressão com &.. Após a aplicação da correção rápida, o resultado será o seguinte: 
# Boa prática
if user&.profile&.thumbnails
end
