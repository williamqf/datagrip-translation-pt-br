Relata chamadas de construtor de corrotina que violam a concorrência estruturada ao aceitar um CoroutineContext que contém um Job.

Passar um contexto com um Job para construtores de corrotinas (como launch, async, produce, promise) ou para withContext quebra a relação pai-filho entre as corrotinas, anulando assim o propósito da concorrência estruturada.

Exemplo:


coroutineScope.launch(Job()) {
  // Esta corrotina não possui uma relação de pai com o receptor `coroutineScope`
}
