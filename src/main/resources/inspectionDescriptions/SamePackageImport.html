Relata instruções import que referenciam o mesmo pacote que o arquivo que as contém.

Arquivos no mesmo pacote são sempre importados implicitamente, portanto, tais instruções import são redundantes e confusas.

Como o IntelliJ IDEA pode detectar e corrigir automaticamente tais instruções por meio do comando Otimizar Importações, esta inspeção é útil principalmente para relatórios offline de bases de código que você não pretende alterar.
