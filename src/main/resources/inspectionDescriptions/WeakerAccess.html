Relata campos, métodos ou classes cujos modificadores de acesso podem ser restringidos.

Exemplo:


  class Sample {
    void foo() {
        bar("foo", "foo");
    }
    void bar(String x, String y) { } // pode ser private
  }

Após a aplicação da correção rápida:


  class Sample {
    void foo() {
        bar("foo", "foo");
    }
    private void bar(String x, String y) { }
  }

Use as opções de inspeção para definir as regras das sugestões de alteração de modificadores.
