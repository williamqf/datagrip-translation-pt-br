Relata modificadores non-sealed desnecessários usados em métodos, campos ou variáveis.

Este modificador é válido apenas para classes, interfaces e traits.

Exemplo:


  non-sealed boolean foo() {} // o modificador é desnecessário
  non-sealed Object bar // o modificador é desnecessário

  // o modificador é obrigatório, portanto não está realçado
  non-sealed class A {}
